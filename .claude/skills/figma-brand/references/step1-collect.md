# Step 1: 자료 수집 (대화형)

브랜드명을 받은 후, 아래 항목을 **하나씩** 대화형으로 수집한다.
한 번에 여러 개를 묻지 않는다. 사용자가 "없어" 또는 "모르겠어"라고 하면 스킵하고 다음으로 넘어간다.

## 수집 항목

### 필수 (반드시 받아야 함)
1. **로고 파일** — "로고 파일 있어? 이미지로 첨부하거나 파일 경로를 알려줘."
2. **피그마 파일 URL** — "브랜드 가이드를 넣을 피그마 파일 URL을 줘."

### 선택 (없으면 웹 리서치로 보완)
3. **공식 웹사이트 URL** — "공식 웹사이트 URL이 있어? (카페24, 스마트스토어도 OK)"
4. **인스타그램** — "인스타그램 URL이나 피드 스크린샷이 있어?"
5. **브랜드 한줄 소개** — "브랜드를 한 문장으로 설명하면? (없으면 웹에서 찾아볼게)"
6. **주력 제품 라인업** — "주요 제품 이름과 카테고리를 알려줘. (없으면 웹에서 찾아볼게)"
7. **브랜드 컬러/폰트 정보** — "혹시 지정된 브랜드 컬러나 폰트가 있어? (없으면 로고에서 추출할게)"

## 수집 완료 조건

최소한 **로고 + 피그마 URL**이 있으면 진행 가능.
웹사이트 URL도 없으면 브랜드명으로 WebSearch하여 공식 사이트를 찾는다.

## 수집 결과 정리

수집이 끝나면 내부적으로 아래 구조로 정리한다 (사용자에게 보여주지 않음):

```
brand_info = {
  name: "위타민",
  name_en: "weetamin",
  logo: [첨부 이미지 또는 경로],
  website: "https://weetamin.co.kr",
  instagram: "@weetamin" 또는 스크린샷,
  tagline: "약사가 연구하고 만드는 영양제",
  products: ["장당당", "간당당", ...],
  colors: null,  // 없으면 null
  fonts: null,    // 없으면 null
  figma_url: "https://www.figma.com/design/...",
  figma_file_key: "VdlNsuuyWXEUyTuAYXLXPY"
}
```
