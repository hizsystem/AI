# Meta 최적화 진단

현재 광고 성과를 진단하고 에이전트별 개선 액션을 도출하는 스킬.

## 실행 흐름

### 1단계: 현황 입력
사용자에게 현재 지표 입력 요청:
- 기간
- 광고비
- 노출, 도달
- 클릭, CTR
- 전환, CPA
- ROAS (있으면)

### 2단계: 퍼널 진단
meta-growth-analyst 에이전트가 분석:
```
[노출] → [클릭] → [랜딩] → [전환]
 CPM      CTR      CVR     CPA
```
각 단계의 벤치마크 대비 상태 진단 (양호/주의/심각)

### 3단계: 에이전트별 액션 플랜

| 문제 영역 | 담당 에이전트 | 액션 |
|----------|-------------|------|
| CPM 높음 / 도달 부족 | 🎯 미디어바이어 | 오디언스 확장, 입찰 변경 |
| CTR 낮음 / 클릭 부족 | 🎨 크리에이티브 PD | 소재 리프레시, 후킹 강화 |
| CVR 낮음 / 전환 부족 | 📈 그로스 분석가 | 랜딩페이지, 제안 최적화 |
| 전체 관리 / 보고 | 📋 어카운트 매니저 | 진단 결과 보고, 일정 조율 |

### 산출물 저장
```
clients/{client-name}/meta/optimizations/YYYY-MM-DD-{issue}.md
```
