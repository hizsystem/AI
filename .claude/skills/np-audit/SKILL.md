# 네이버 플레이스 진단 리포트

소상공인 매장의 네이버 플레이스 최적화 상태를 100점 만점으로 진단하고 리포트를 생성한다.

## 모드 분기

사용자가 모드를 지정하지 않으면 물어본다:

- `/np-audit lite` — 무료 진단 리포트 1장 (15분). 영업 도구용.
- `/np-audit full` — 유료 풀 진단서 (1~2시간). 계약 후 첫 산출물.
- `/np-audit` — "Lite(무료 1장)와 Full(풀 진단서) 중 어떤 모드로 진단할까요?"

---

## 공통: 데이터 수집

### Step 1. 기본 정보 확인

사용자에게 다음을 요청한다:

```
진단할 매장 정보를 알려주세요:
1. 네이버 플레이스 URL (필수)
2. 매장명 (URL에서 파악 가능하면 생략)
3. 업종 (예: 국밥, 카페, 미용실)
4. 상권/지역 (예: 센텀, 홍대, 강남)
```

### Step 2. 플레이스 데이터 수집

플레이스 URL에서 수집 가능한 데이터를 WebFetch로 시도한다.
수집이 안 되는 항목은 사용자에게 직접 확인을 요청한다.

**자동 수집 시도 항목:**
- 매장명, 주소, 카테고리
- 리뷰 수
- 메뉴 정보 (개수, 사진 유무)
- 편의시설
- AI 브리핑 활성화 여부

**수동 확인 요청 항목 (체크리스트로 제시):**

```
아래 항목을 확인해서 알려주세요 (O/X 또는 상세):

[S등급 — 매출 직결]
S1. 리뷰 수: ___개
S1. 영수증 리뷰 유도 장치(엽서/카드/이벤트) 있나요? O/X
S2. 플레이스 광고 집행 중인가요? O/X / 키워드 최적화까지 하고 있나요?
S3. 예약 기능 켜져 있나요? O/X
S3. 알림받기 쿠폰 세팅되어 있나요? O/X
S3. 톡톡 마케팅 메시지 활용 중인가요? O/X

[A등급 — 노출 순위]
A1. 대표 사진 퀄리티 좋은가요? (맛있겠다/가보고싶다 느낌) O/X
A1. 메뉴별 사진 전부 있나요? O/X
A1. 영상 소식 올려져 있나요? O/X
A2. 대표 키워드 세팅되어 있나요? O/X (있으면 키워드 목록도)
A2. 키워드에 지역명 포함? O/X
A3. 최근 30일 소식 발행한 적 있나요? O/X
A3. 이벤트/쿠폰 연동되어 있나요? O/X
A4. 블로그 체험단 운영 중/했었나요? O/X

[B등급 — 기본기]
B1. 영업시간 등록되어 있나요? O/X
B1. 업주 인증 되어 있나요? O/X
B1. 편의시설 등록되어 있나요? O/X
B2. 상세 소개문 몇 자 정도인가요? (미작성/500자미만/1000자/1500자+)
B3. AI 브리핑 켜져 있나요? O/X
B3. 매장명에 지역명 포함? O/X

[보너스]
X1. 인스타그램 연동? O/X
X1. 당근마켓 비즈프로필? O/X
X1. 자사 블로그 연동? O/X
X2. 네이버 커넥트(포스) 등록? O/X
```

## 채점 엔진 (100점 만점)

수집된 데이터로 아래 기준에 따라 채점한다.

### S등급 — 50점 (매출 직결)

**S1. 영수증 리뷰 & 별점 (20점)**
- 리뷰 수: 0개→0점, ~100→4점, ~300→8점, ~500→12점, 1000+→16점
- 키워드 포함 리뷰 유도 장치(엽서/카드/이벤트): 있음→4점, 없음→0점

**S2. 플레이스 광고 (15점)**
- 미집행→0점
- 집행 중→10점
- 키워드 최적화 + A/B 테스트까지→15점

**S3. 예약 & 쿠폰 (15점)**
- 예약 기능 ON→5점, OFF→0점
- 알림받기 쿠폰 세팅→5점, 미세팅→0점
- 톡톡 마케팅 메시지 활용→5점, 미활용→0점

### A등급 — 30점 (노출 순위 결정)

**A1. 대표 사진 & 영상 (8점)**
- 대표 사진 퀄리티 좋음→3점, 보통/나쁨→1점, 없음→0점
- 메뉴별 사진 완비→2점, 일부→1점, 없음→0점
- 영상 소식 업로드→3점, 없음→0점

**A2. 대표 키워드 (8점)**
- 키워드 세팅 유무: 있음→3점, 없음→0점
- 지역명 포함: 포함→2점, 미포함→0점
- 유입 키워드와 정합성: 좋음→3점, 보통→1점, 확인 불가→0점

**A3. 소식 & 이벤트 (7점)**
- 최근 30일 소식 발행: 있음→4점, 없음→0점
- 이벤트/쿠폰 연동: 있음→3점, 없음→0점

**A4. 블로그 체험단 (7점)**
- 블로그 리뷰 존재: 있음→3점, 없음→0점
- 키워드 타겟팅: 하고 있음→2점, 안 함→0점
- 블로거 품질(4개 지표 중 3개 충족): 좋음→2점, 보통→1점, 확인 불가→0점

### B등급 — 15점 (기본기)

**B1. 기본 정보 완성도 (6점)**
- 영업시간 등록→2점, 누락→0점
- 업주 인증→2점, 미인증→0점
- 편의시설 등록→1점, 미등록→0점
- 길안내 상세→1점, 미작성→0점

**B2. 상세 소개문 (5점)**
- 미작성→0점
- 500자 미만→1점
- 500~1000자→3점
- 1500자 이상→5점

**B3. AI 브리핑 & 기타 (4점)**
- AI 브리핑 켜기→2점, 꺼짐→0점
- 카테고리 최적화→1점
- 매장명 지역명 포함→1점

### 보너스 — 5점 (매출 부스터)

**X1. 외부 채널 연계 (3점)**
- 인스타그램 연동→1점
- 당근마켓 비즈프로필→1점
- 자사 블로그 연동→1점

**X2. 네이버 커넥트 (2점)**
- 등록→2점, 미등록→0점

### 등급 판정

| 점수 | 등급 | 리포트 헤드라인 |
|------|------|----------------|
| 80~100 | A등급 | "잘 관리되는 매장입니다. 미세 최적화로 1등을 유지하세요." |
| 60~79 | B등급 | "기본기는 있지만 빈틈이 있습니다. 몇 가지만 개선하면 바로 효과가 납니다." |
| 40~59 | C등급 | "핵심 기능을 활용하지 못하고 있습니다. 지금 놓치고 있는 매출이 있습니다." |
| 0~39 | D등급 | "기본 세팅부터 시작하면 빠르게 올라갑니다. 가장 효과가 큰 3가지부터 시작하세요." |

## Full 모드: 추가 데이터 수집 + 풀 진단서

Lite 채점을 먼저 수행한 후, 추가 데이터를 요청한다:

```
Full 진단을 위해 추가 데이터가 필요합니다:

1. 에드로그 데이터 (캡처 또는 텍스트)
   - N1(유사도), N2(관련성), N3(랭킹) 수치
   - 주요 키워드 순위

2. 경쟁사 정보
   - 같은 상권 경쟁 매장 2~3곳 플레이스 URL
   - (또는 경쟁사 이름만 알려주시면 제가 찾겠습니다)

3. 키워드 데이터
   - 경쟁사 소스코드에서 추출한 keywordlist (있으면)
   - 마피아넷 키워드 분석 결과 (있으면)

4. 스마트플레이스 내부 데이터 (있으면)
   - 유입 키워드 통계
   - 광고 성과 데이터
```
