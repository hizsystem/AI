# 콘텐츠 제작 파이프라인

3개 에이전트를 순차 호출하여, 컨텍스트 기반 브랜드 정합성 높은 콘텐츠를 제작하는 스킬.

## 실행 흐름

### 0단계: 브리핑 수집 + 컨텍스트 자동 로딩

**사용자에게 확인:**
- **클라이언트/브랜드명**: 누구를 위한 콘텐츠인가
- **콘텐츠 유형**: SNS 피드 / 릴스 캡션 / 블로그 / 뉴스레터 / 광고 카피 / 랜딩페이지
- **플랫폼**: 인스타그램 / 링크드인 / 블로그 / 이메일 / 페이스북 / 유튜브
- **목적**: 인지 / 참여 / 전환 / 리텐션
- **타겟**: 누가 볼 것인가
- **특별 요청**: 참고 레퍼런스, 키메시지, 프로모션 정보 등

사용자 입력에서 파악 가능한 정보는 질문하지 않는다.

**컨텍스트 자동 로딩** (범용 라우팅 룰 적용):
1. `clients/{name}/context/` 존재 → 브랜드 가이드라인 + 제품 스펙 읽기
2. `context/` 없음 → `clients/{name}/strategy.md` → `brief.md` 순서로 읽기
3. 아무것도 없으면 → 브랜드 톤, 타겟, 핵심 메시지를 사용자에게 직접 질문

### 1단계: 방향 설정 (✦ 브랜드 전략가)

`brand-strategist` 에이전트 호출:
- 로딩된 컨텍스트 기반으로 콘텐츠 방향 설정
- **핵심 메시지**: 이 콘텐츠가 전달할 하나의 메시지
- **브랜드 적합성**: 톤앤매너, 비주얼 방향 가이드
- **차별화 포인트**: 경쟁사 콘텐츠와의 차이점
- **전략적 위치**: 전체 콘텐츠 전략에서의 역할

### 2단계: 초안 제작 (✍ 콘텐츠 크리에이터)

`content-creator` 에이전트 호출 (1단계 방향 + 컨텍스트 전달):
- **2-3개 시안** 제작, 각각 다른 접근:
  - A안: 직접적/문제 해결형
  - B안: 스토리텔링/감성형
  - C안: 데이터/증거 기반형
- 각 시안에 포함: 헤드라인/훅, 본문, CTA, 해시태그/키워드
- 각 시안의 전략적 의도 설명

### 3단계: 검수 파이프라인

**3-1. 브랜드 크로스체크** (✦ 브랜드 전략가):
- 브랜드 톤앤매너 일치 여부
- 타겟 페르소나의 언어인지
- 전략적 메시지 정확성
- 피드백 사항 정리

**3-2. 규제 검수** (해당 시에만 자동 실행):
- 클라이언트가 식품/건강기능식품/화장품/의료 업종인 경우 → `food-regulation-advisor` 에이전트 호출
- 식품표시광고법, 건강기능식품법, 화장품법 등 관련 법적 리스크 점검
- 수정 필요 사항 표시 (표현 대체안 포함)

> 규제 검수 대상 판별: 컨텍스트에 식품/건기식/화장품/의료 키워드가 있거나, 클라이언트별 오버라이드에 규제 검수가 명시된 경우

**3-3. 피드백 반영** (✍ 콘텐츠 크리에이터):
- 3-1, 3-2 피드백을 반영하여 시안 수정
- 수정 불가 사항은 사유와 함께 보고

### 4단계: 최종 산출물

최종 콘텐츠를 플랫폼별 포맷으로 제공:
- 최종 카피 (선택된 시안 또는 전체 시안)
- 플랫폼별 최적화 버전
- 비주얼 가이드 (이미지 방향 제안)
- 게시 추천 시간대
- 성과 측정 기준

## 에이전트 호출 방식

```
0단계: 컨텍스트 로딩 (파일 읽기)
  ↓
1단계: ✦ 브랜드 전략가 (방향 설정)
  ↓
2단계: ✍ 콘텐츠 크리에이터 (시안 제작)
  ↓
[병렬] 3-1. ✦ 브랜드 전략가 (크로스체크)
       3-2. ⚖️ 규제 검수 (해당 시)
  ↓ 통합
3-3. ✍ 콘텐츠 크리에이터 (피드백 반영)
  ↓
4단계: 최종 산출물 저장
```

## 산출물 저장

```
clients/{client-name}/content/
└── YYYY-MM-DD-{content-type}-{topic}.md
```

## 플랫폼별 포맷 가이드

| 플랫폼 | 길이 | 특징 |
|--------|------|------|
| 인스타그램 | 150자 내외 (캡션 2,200자) | 감성적, 비주얼 중심, 해시태그 15-20개 |
| 링크드인 | 1,300자 이내 | 전문적, 인사이트 중심, 스토리텔링 |
| 블로그 | 2,000-3,000자 | SEO 최적화, 구조화된 헤딩, 내부 링크 |
| 뉴스레터 | 500-800자 | 개인적 톤, 큐레이션, 명확한 CTA |
| 광고 카피 | 25자(헤드라인) / 90자(설명) | 후킹, 베네핏, 긴급성 |

## 빠른 실행 모드

사용자 입력이 충분히 구체적이면 (클라이언트 + 유형 + 목적이 명확) 0단계 질문을 건너뛰고 바로 컨텍스트 로딩 → 1단계부터 시작한다.
