# Meta 캠페인 설계

캠페인 목표부터 소재 방향까지 구조를 자동 설계하는 스킬.

## 실행 흐름

### 1단계: 목표 확인
사용자에게 질문:
- 캠페인 목표: 전환 / 트래픽 / 인지도 / 리드
- 월 예산
- 기간
- 타겟 특성

### 2단계: 오디언스 설계
meta-media-buyer 에이전트가 타겟 구조 설계:
- 코어 오디언스 (관심사/행동/인구통계)
- 유사 오디언스 (소스 및 비율)
- 리타게팅 오디언스 (소스 및 기간)

### 3단계: 예산 배분
- 전체 예산 중 테스트/스케일링 비율
- 광고세트별 일예산
- 스케일링 기준

### 4단계: 크리에이티브 브리프
meta-creative-pd 에이전트가 소재 방향 제안:
- 시안 3개 (각각 다른 후킹 유형)
- 포맷 추천 (피드/릴스/캐러셀)
- 카피 초안

### 산출물 저장
```
clients/{client-name}/meta/campaigns/YYYY-MM-DD-{campaign-name}.md
```
