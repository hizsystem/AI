# 캠페인 기획 파이프라인

5개 에이전트가 5단계로 협업하여, 시장 분석부터 실행 콘텐츠까지 완성하는 스킬.

## 실행 흐름

### 0단계: 브리핑 수집 + 컨텍스트 자동 로딩

**사용자에게 확인:**
- **클라이언트/브랜드명**: 누구를 위한 캠페인인가
- **캠페인 목적**: 인지/참여/전환/리텐션/런칭
- **예상 예산**: (없으면 규모감이라도)
- **기간**: 캠페인 운영 기간
- **타겟**: 핵심 타겟 오디언스
- **특별 요청**: 채널 제한, 필수 포함 요소 등

사용자 입력에서 파악 가능한 정보는 질문하지 않는다.

**컨텍스트 자동 로딩** (범용 라우팅 룰 적용):
1. `clients/{name}/context/` 존재 → 브랜드 가이드라인 + 제품 스펙 + 시장 환경 읽기
2. `context/` 없음 → `clients/{name}/strategy.md` → `brief.md` → `research.md` 순서로 읽기
3. 아무것도 없으면 → 핵심 정보를 사용자에게 직접 질문

### 1단계: 시장 분석 (🔍 시장 조사관)

`market-researcher` 에이전트 호출:
- **시장 타이밍**: 왜 지금 이 캠페인인가 (시즌, 트렌드, 이벤트)
- **경쟁사 캠페인**: 최근 경쟁사 캠페인 분석
- **타겟 인사이트**: 타겟의 현재 관심사, Pain Point
- **기회 영역**: 경쟁사가 놓치고 있는 빈 공간

### 2단계: 크리에이티브 전략 (✦ 브랜드 전략가 + ✍ 콘텐츠 크리에이터)

`brand-strategist` + `content-creator` 에이전트 **병렬 호출** (1단계 결과 전달):
- **캠페인 컨셉**: 한 줄 컨셉 + 2-3개 방향 제시
- **핵심 메시지**: 캠페인이 전달할 하나의 메시지
- **톤앤매너**: 이 캠페인만의 톤 (브랜드 톤 안에서의 변주)
- **비주얼 방향**: 참고 이미지, 컬러, 무드 제안
- **카피 시안**: 헤드라인 2-3개, 바디카피 방향

### 3단계: 미디어 플래닝 (📊 퍼포먼스 분석가 + 💰 세일즈 전략가)

`performance-analyst` + `sales-strategist` 에이전트 **병렬 호출** (1-2단계 결과 전달):
- **채널 선정**: 어디서 집행할 것인가 + 이유
- **예산 배분**: 채널별 예산 비율
- **타겟팅**: 플랫폼별 타겟 세그먼트
- **KPI 설정**: 채널별 목표 지표 (CPM, CPC, CPA, ROAS)
- **성과 예측**: 예상 도달, 클릭, 전환

### 4단계: 실행 계획 (📋 프로젝트 매니저)

`project-manager` 에이전트 호출 (전체 결과 통합):
- **캠페인 일정**: 준비 → 론칭 → 운영 → 마무리 타임라인
- **산출물 목록**: 필요한 크리에이티브, 카피, 랜딩페이지 등
- **제작 일정**: 산출물별 제작 마감일
- **운영 계획**: 일별/주별 모니터링 및 최적화 포인트
- **리스크 관리**: 예상 리스크와 대응 방안

### 5단계: 규제 검수 (해당 시)

클라이언트가 식품/건강기능식품/화장품/의료 업종인 경우:
- `food-regulation-advisor` 에이전트 호출
- 카피/메시지의 법적 리스크 점검
- 수정 필요 사항 표시 (대체 표현 포함)

## 에이전트 호출 방식

```
0단계: 컨텍스트 로딩 (파일 읽기)
  ↓
1단계: 🔍 시장 조사관 (시장 분석)
  ↓
[병렬] 2단계: ✦ 브랜드 전략가 + ✍ 콘텐츠 크리에이터 (크리에이티브)
  ↓
[병렬] 3단계: 📊 퍼포먼스 분석가 + 💰 세일즈 전략가 (미디어)
  ↓
4단계: 📋 프로젝트 매니저 (실행 계획)
  ↓
5단계: ⚖️ 규제 검수 (해당 시)
```

## 산출물 저장
