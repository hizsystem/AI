# 바이럴 콘텐츠 해부 분석

바이럴 해부학자 에이전트를 호출하여 터지는 콘텐츠의 구조를 분석하고, 재현 가능한 공식을 추출하는 스킬.

## 실행 흐름

### 0단계: 분석 대상 확인
사용자 입력 유형에 따라 분기:

**A. URL 입력** (특정 ��텐츠 분석)
- 릴스/틱톡/쇼츠 URL 제공 시, 해당 콘텐츠 1개를 심층 분석

**B. 카테고리 입력** (카테고리 바이럴 패턴 분석)
- "카페", "식품", "뷰티" 등 카테고리 제공 시, 해당 카테고리의 바이럴 사례를 조사하여 분석
- 사례 수: 기본 3개 (서로 다른 메커니즘)

**C. 자사 콘텐츠 입력** (성과 진단)
- 자사 콘텐츠의 지표(조회수, 완주율, 저장, 공�� 등)를 입력받아 개선점 분석

### 1단계: 사례 수집 (B/C 유형일 때)
- 카테고리 바이럴 사례 웹 리서치 (조회수, 인게이지먼트, 콘텐츠 유형 수집)
- 서로 다른 바이럴 메커니즘을 가진 사례를 선별

### 2단계: 5단계 해부 프레임워크 적용
`viral-analyst` 에이전트 호출, 각 사례에 대해:

#### (1) 구조 해체 (Structure Breakdown)
- 훅 유형 분류 (10종 중)
- 스크립트 프레임워크 식별 (6종 중)
- 리텐션 장치 분석 (패턴 인터럽트 간격, 오픈 루프)
- 엔딩 전략 분류 (루프/클리프행어/CTA)

#### (2) 감정 매핑 (Emotional Mapping)
```
Primary Emotion → Secondary Response → Sharing Trigger
```
- 고각성 감정 체크 (경외감/웃음/놀라움/분노·공감/관련성)

#### (3) STEPPS 채점
| 요소 | 점수 (0-2) | 해당 장치 |
|------|-----------|----------|
| Social Currency | | |
| Triggers | | |
| Emotion | | |
| Public | | |
| Practical Value | | |
| Stories | | |
| **총점** | **/12** | |

→ 6점 이상: 바이럴 잠재력 높음 / 9점 이상: 매우 높음

#### (4) 성과 시그널 분석
- 어떤 지표가 특히 높았는지 (저장/공유/댓글/완주율)
- 왜 그 지표가 높았는지 구조적 근거

#### (5) 재현 공식 추출
```
[훅 유형] + [프레임워크] + [감정 트리거] + [엔딩 전략] = 공식명
```
- 재현 시 핵심 변수 3가지
- 제작 난이도 (낮/중/높)
- 재현 가능 횟수 (제한/높음/무한)

### 3단계: 비교 분석 & 추천 (복수 사례일 때)
- 사례 간 재현 공식 비교 테이블
- 브랜드 적합도 평가
- **최종 추천**: 재현 가능성이 가장 높은 공식 + 이유

### 4단계 (선택): 즉시 기획 전환
분석 결과를 바탕으로 `/viral-plan`으로 연결 가능:
- "이 공식으로 우리 브랜드에 맞는 콘텐츠를 기획해줘"
- 바이럴 해부 결과가 숏폼 CD의 입력으로 자동 전달

## 트렌드 분류 매트릭스

사례 분석 시 트렌드 수명주기도 함께 판단:

| 단계 | 신호 | 액션 |
|------|------|------|
| 상승기 | 사용 수 급증, 대형 크리에이터 미참여 | 즉시 제작 |
| 정점 | 모든 크리에이터 참여, 밈화 | 차별화 변형만 |
| 포화 | 패러디/비판 등장 | 참여 금지 |
| 쇠퇴 | 사용 수 급감 | 완전 폐기 |
