# 바이럴 숏폼 콘텐츠 기획 파이프라인

5개 전담 에이전트를 순차 호출하여, 전략부터 편집 디렉션까지 프로덕션 레디 산출물을 생성하는 스킬.

## 실행 흐름

### 0단계: 브리핑 수집
사용자에게 질문:
- **브랜드/계정명**: 누구를 위한 콘텐츠인가
- **타겟 플랫폼**: Reels / TikTok / Shorts / 전체
- **카테고리**: 카페, 식품, 뷰티, 라이프스타일 등
- **콘��츠 수**: 몇 개 컨셉을 기획할 것인가 (기본 3개)
- **촬영 여건**: 장비, 인력, 편집 수준
- **특별 요청**: 참고 레퍼런스, 원하는 포맷, 시리즈 여부 등

### 1단계: 알고리즘 트래커 (📡)
`algorithm-tracker` 에이전트 호출:
- 해당 플랫폼의 현재 알고리즘 최적 전략 제시
- 데스 시그널 회피 가이드
- 최적 포스팅 가이드 (빈도, 시간, 길이)
- 공략할 핵심 시그널 우선순위

### 2단계: 바이럴 해부학자 (🔥)
`viral-analyst` 에이전트 호출:
- 해당 카테고리의 바이럴 패턴 분석
- 재현 가능한 공식 추출
- STEPPS 기반 바이럴 잠재력 평가 기준 설정
- 추천 훅 유형 + 프레임워크 + 감정 트리거 조합

### 3단계: 숏폼 크리에이티브 디렉터 (🎬)
`shortform-cd` 에이전트 호출:
- 콘텐츠 필러 설계 (3-5개)
- 크리에이티브 브리프 작성 (요청 수만큼)
- 각 컨셉별 STEPPS 점수 + 바이럴 잠재력 근거
- 시리즈화 방향 + 배칭 전략
- 콘텐츠 캘린더 초안

### 4단계: 씬 & 스크립트 라이터 (📝)
`scene-scriptwriter` 에이전트 호출:
- 각 브리프별 프로덕션 레디 스크립트
- 3중 훅 설계 (비주얼 + 텍스트 + 버���)
- 초 단위 씬 분할 테이블
- 리텐션 장치 배치 (패턴 인터럽트, 오픈 루프)
- 훅 변형 2-3안 추가 제시
- 무음 시청 대응 검증

### 5단계: 에디트 디렉터 (🎵)
`edit-director` 에이전트 호출:
- 편집 디렉션 시트 (음악/컷리듬/캡션/이펙트)
- 패턴 인터럽트 맵 (초 단위)
- 루프/엔딩 설계
- 커버프레임 추천
- 촬영 여건에 맞는 편집 실행 가이드 (CapCut 등)

## 에이전트 호출 방식

- 1-2단계는 **병렬 호출** 가능 (알고리즘 전략과 ���이럴 분석은 독립적)
- 3단계는 1-2단계 결과를 통합하여 호출
- 4-5단계는 **병렬 호출** 가능 (스크립트와 편집 디렉션은 같은 브리프 기반)

```
[병렬] 📡 알고리즘 트래커 + 🔥 바이럴 해부학자
         ↓ 통합
       🎬 숏폼 CD (브리프 생성)
         ↓
[병렬] 📝 씬 라이터 + 🎵 에디트 디렉터
```
