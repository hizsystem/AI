# Meta 광고 온보딩

신규 클라이언트의 Meta 광고를 시작하기 위한 온보딩 스킬.

## 실행 흐름

### 1단계: 비즈니스 파악
다음 정보를 수집한다:
- 업종 및 제품/서비스
- 타겟 고객 (연령, 성별, 관심사, 지역)
- 경쟁사 3개
- 월 광고 예산
- 핵심 KPI (리드/매출/앱설치)

### 2단계: 현황 진단
체크 항목:
- Meta Business Manager 계정 유무
- 광고 계정 상태
- 페이스북 페이지 / 인스타그램 비즈니스 계정
- 픽셀 설치 여부
- 기존 광고 이력

### 3단계: 셋업 체크리스트
meta-account-manager 에이전트의 온보딩 체크리스트를 기반으로 점검.

### 4단계: 초기 캠페인 설계
meta-media-buyer 에이전트가 첫 캠페인 구조를 제안:
- 캠페인 구조 (목표, 예산 방식)
- 광고세트 3개 (코어/유사/리타게팅)
- 소재 방향 → meta-creative-pd에게 전달

### 5단계: 산출물 저장
```
clients/{client-name}/meta/onboard-brief.md
```
