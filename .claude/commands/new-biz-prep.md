# New Business 상담 준비

상담 신청 폼 데이터와 추가 자료를 기반으로, 4개 전담 에이전트가 병렬로 분석하여
**컨설팅 수주를 위한 완벽한 상담 준비 패키지**를 생성합니다.

기존 `/pre-meeting`의 강화판으로, 글로벌 컨설팅사 방법론(McKinsey, SPIN Selling, Challenger Sale, Trusted Advisor)을 적용합니다.

사용자 입력: $ARGUMENTS

---

### 신청 경로별 접근 톤 가이드 (상담 코치 참고)

| 신청 경로 | 추정 상태 | 접근 톤 |
|----------|----------|--------|
| **광고** | 문제 인식 명확, 솔루션 탐색 중 | 전문성 강조 + 빠른 진단 |
| **카카오톡 커뮤니티** | 커뮤니티 내 추천/입소문 | 커뮤니티 맥락 공유 + 사례 중심 |
| **지인추천** | 신뢰도 높은 상태 | 소개자 언급 + 편안한 톤 |
| **오프라인 강의 참석** | 브랜드라이즈 전문성 경험 | 강의 내용 연결 + 깊은 진단 |
| **@brandrise 인스타그램** | 콘텐츠에 공감 | 특정 콘텐츠 언급 + 맞춤 솔루션 |
| **메일/레터** | 구독자, 지속적 관심 | 뉴스레터 내용 연결 + 장기 관점 |
| **스레드** | 가벼운 관심에서 시작 | 캐주얼 톤 + 실질적 가치 빠르게 |

### 예산별 수주 전략 자동 조정 (수주 전략가 참고)

| 예산 구간 | 전략 방향 | 3-Tier 조정 |
|----------|----------|------------|
| **100만원 정도** | 스타터 패키지 / 단발성 진단 | Essential만 제시, 또는 파일럿 제안 |
| **300만원 이하** | 선택과 집중 1개 채널 | Essential 중심 + Professional 티저 |
| **500만원 이하** | 전략+실행 1세트 | 3-Tier 표준 구성 |
| **1,000만원 이하** | 통합 마케팅 가능 | 3-Tier 표준 + Premium 강조 |
| **5,000만원 이하** | 풀서비스 가능 | Professional~Premium 중심 |
| **1억원 이상** | 엔터프라이즈급 | Premium + 커스텀 제안 |

### 희망 서비스 vs 진짜 니즈 갭 분석 (니즈 분석가 참고)

| 희망 서비스 | 흔한 갭 패턴 |
|-----------|-----------|
| **브랜드 마케팅** | 실제로는 포지셔닝이 아니라 세일즈 퍼널이 없는 것이 문제인 경우 多 |
| **퍼포먼스 마케팅** | 광고 이전에 브랜드 메시지/타겟이 정의 안 된 경우 多 |
| **콘텐츠 마케팅** | 콘텐츠 전략 없이 "올릴 콘텐츠"만 필요하다고 생각하는 경우 多 |
| **IMC 마케팅** | 각 채널별 역할 분담 없이 "다 해야 할 것 같다"는 막연함 多 |
| **잘 모르겠음** | ★ 가장 큰 수주 기회 — 니즈 분석이 핵심 가치가 됨 |

### 2차 데이터: 사전 메일 회신 (있을 수도, 없을 수도 있음)

상담 확정 후 대표님에게 사전 준비 메일을 발송합니다. **회신율은 약 50%** — 회신 여부 자체가 중요한 시그널입니다.

**사전 메일 요청 항목:**

| # | 요청 항목 | 에이전트 활용 | 미회신 시 대체 |
|---|---------|-------------|-------------|
| 1 | **사업소개서 / 메인 제품 소개** | 🔎 프로파일러: 기업 이해도 대폭 상승 | 웹 리서치로 보완 |
| 2 | **현재 운영 중인 마케팅 채널** | 🔎 프로파일러: 디지털 프레즌스 진단 보완 | SNS/광고 라이브러리 리서치 |
| 3 | **주요 고객 타겟 정의** | 🧠 니즈분석: JTBD 정교화 | 업종 기반 추정 |
| 4 | **자사 제품/서비스의 가장 큰 강점 1가지** | 💼 수주: Challenger Teaching 소재 | 리뷰/기사에서 추출 |
| 5 | **최근 마케팅 성과 좋았던/아쉬웠던 사례** | 🧠 니즈분석: Pain Hierarchy L2 핵심 입력 | 상담에서 직접 질문 |
| 6 | **향후 6개월 목표 매출** | 💼 수주: ROI 프레이밍 + 가격 앵커링 | 투자단계/업종 기반 추정 |
| 7 | **현재 가장 시급한 과제 1가지** | 🧠 니즈분석: SPIN Problem 핵심 | 폼 페인포인트로 대체 |
| 8 | **이번 컨설팅을 통해 기대하시는 결과** | 🎤 상담코치: Phase 4-5 방향 설정 | 폼 "기대 성과"로 대체 |
| 9 | **진행했던 광고 리포트 또는 주요 수치 자료** | 💼 수주: 데이터 기반 진단 가능 | 상담에서 요청 |

**회신 여부에 따른 상담 전략 분기:**

| 상황 | 의미 | 상담 전략 조정 |
|------|------|--------------|
| **메일 회신 O (상세)** | 준비된 대표, 의지 高, 체계적 사고 | 데이터 기반 깊은 진단 → Professional/Premium 추천 가능 |
| **메일 회신 O (간략)** | 관심은 있으나 시간 부족 또는 정리 안 됨 | 상담에서 함께 정리해주는 톤, "제가 도와드릴게요" |
| **메일 회신 X** | 바쁘거나 아직 정리 안 됨, 또는 온도 낮음 | 상담 초반에 상황 확인 시간 더 배분 (Phase 2를 15분으로 확대), 가벼운 톤 |

### 3차 데이터: 기타 추가 자료 (사용자가 별도 전달)

## 조사 프레임워크 (5단계 모두 수행):

### 1단계: 기업 기초 프로필

### 2단계: 디지털 프레즌스 진단
- 웹사이트: 구조, 메시징, UX/UI 수준
- SNS: 채널별 팔로워, 게시빈도, 콘텐츠 톤앤매너, 인게이지먼트
- 미디어 노출: 언론 보도, 인터뷰
- 광고: Meta 광고 라이브러리 검색, 네이버 검색광고 여부

### 3단계: 대표자/창업팀 프로필
대표 이력, 인터뷰/기고문에서 드러나는 철학, 의사결정 스타일 추정

### 4단계: 시장·경쟁 환경
TAM/SAM/SOM 추정, 직접 경쟁사 3-5개, 시장 트렌드, 벤치마크 브랜드

### 5단계: 초기 가설 수립
- 브랜드 성숙도 (5단계: 미정립/초기/성장/확립/재정립필요)
- 마케팅 성숙도 (5단계: 미착수/시행착오/체계화초기/체계화/최적화)
- 핵심 과제 가설 + 가설 근거

WebSearch를 적극 활용하세요.
검색어: "{회사명}", "{회사명} 대표", "{대표자명}", "{회사명} 투자", "{회사명} 후기"
