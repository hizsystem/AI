# MarketingOS 도움말

MarketingOS 에이전트 시스템에 오신 것을 환영합니다.

## 사용 가능한 커맨드

| 커맨드 | 용도 | 설명 |
|--------|------|------|
| `/marketingos` | 도움말 | 이 도움말을 표시합니다 |
| `/new-client` | 온보딩 | 신규 클라이언트 프로젝트를 시작합니다 |
| `/proposal` | 제안서 | 클라이언트 맞춤 제안서를 생성합니다 |
| `/campaign` | 캠페인 | 마케팅 캠페인을 기획합니다 |
| `/content` | 콘텐츠 | 콘텐츠를 기획하고 제작합니다 |
| `/report` | 보고서 | 성과 보고서를 생성합니다 |
| `/review` | 리뷰 | 6명 에이전트의 멀티 관점 피드백을 받습니다 |

## 에이전트 팀

6명의 전문 에이전트가 각자의 관점에서 협업합니다:

- **(🔍) 시장 조사관** - 데이터와 팩트로 시장을 분석합니다
- **(✦) 브랜드 전략가** - 브랜드 포지셔닝과 메시지를 설계합니다
- **(✍) 콘텐츠 크리에이터** - 채널별 콘텐츠를 제작합니다
- **(💰) 세일즈 전략가** - 가격 전략과 제안서를 만듭니다
- **(📋) 프로젝트 매니저** - 일정과 리소스를 관리합니다
- **(📊) 퍼포먼스 분석가** - 성과를 측정하고 인사이트를 도출합니다

## 빠른 시작

1. **새 클라이언트가 들어왔다면** → `/new-client`
2. **제안서를 써야 한다면** → `/proposal`
3. **콘텐츠를 만들어야 한다면** → `/content`
4. **아이디어에 대한 피드백이 필요하다면** → `/review`
5. **성과를 정리해야 한다면** → `/report`

## 클라이언트 프로젝트 구조

새 클라이언트 프로젝트는 `clients/{client-name}/` 하위에 관리됩니다.
`/new-client` 커맨드가 자동으로 폴더 구조를 생성합니다.

위 커맨드 중 하나를 실행하여 시작하세요.
