# 네이버플레이스 키워드 진단 (NP Keyword)

에드로그 데이터를 기반으로 키워드 선점 전략을 수립합니다.
사용자가 에드로그 스크린샷(히든 키워드, 순위 체크, 순위 비교분석)을 공유하면 4단계로 분석하여
**키워드 전략 리포트 + 대표키워드 교체 제안 + 4주 키워드 액션 플랜**을 생성합니다.

사용자 입력: $ARGUMENTS

---

## 사전 준비

$ARGUMENTS에서 매장명 또는 클라이언트 slug를 파싱한다.

사용자에게 에드로그 스크린샷을 요청한다:

```
키워드 진단에 필요한 에드로그 스크린샷을 공유해주세요.

필수 (최소 2개):
1. 플레이스 히든 키워드 — 기본 정보 + 키워드 순위표
2. 플레이스 히든 지수 — N1/N2/N3 점수 (목표 키워드 1~2개)

선택 (있으면 더 정밀한 분석 가능):
3. 플레이스 순위 비교분석 — 목표 키워드 상위 업체들과 비교
4. 플레이스 추세 분석 — 경쟁사 순위 변동 히스토리
5. 플레이스 유입 통계 — 채널별/키워드별 유입수 (유료)
```

스크린샷 없이 구두 데이터만 제공해도 진행 가능하나, 정확도가 떨어진다는 점을 안내한다.

---

## Step 1: 히든 키워드 분류 — "유지 / 기회 / 장기"

에드로그 히든 키워드 데이터에서 아래 정보를 추출한다:

- 매장명, 대표키워드 5개, 카테고리
- 방문자리뷰수, 블로그리뷰수, 저장수
- 히든 키워드 목록 (최대 32개)
- 각 키워드별: 월 검색량(조회수), 현재 순위, 과거 순위 추이

### 분류 기준

| 분류 | 조건 | 의미 |
|------|------|------|
| **유지** | 1~3위 + 순위 안정 | 이미 선점 완료. 유지만 하면 됨 |
| **핵심 기회** | 검색량 1,000건+ AND (순위 급상승 중 OR 10위 이내 진입) | 지금 밀면 선점 가능한 골든타임 |
| **기회** | 검색량 100~999건 AND 순위 개선 중 (20위 이내) | 세팅 최적화로 올릴 수 있음 |
| **장기 도전** | 검색량 높지만 순위 50위+ OR 업체수 대비 경쟁 과다 | 당장은 어렵고, 축적 필요 |
| **부적합** | 우리 브랜드/업종과 무관한 키워드 | 노출돼도 전환 안 됨. 무시 |

### 출력 형식

```
### 키워드 분류표

| 키워드 | 월 검색량 | 현재 순위 | 2주 전 | 분류 | 비고 |
|--------|----------|----------|--------|------|------|
```

**추가 분석**:
- 검색량 상위 5개 키워드 중 우리 순위 50위 밖인 것 = 장기 도전 목록
- 순위 변동폭이 가장 큰 키워드 3개 = 모멘텀 키워드 (상승이면 기회, 하락이면 주의)

---

## Step 2: N1/N2 진단 — "왜 이 순위인지"

히든 지수 또는 순위 체크 스크린샷에서 N1/N2/N3 값을 추출한다.

### 진단 매트릭스

| N1 (유사도) | N2 (관련성) | 진단 | 우선 액션 |
|------------|-----------|------|----------|
| 0.5 이상 | 0.5 이상 | 양호 — 순위 유지/소폭 개선 가능 | 현 상태 유지 |
| **0.5 미만** | 0.5 이상 | **키워드 정합성 문제** — 세팅으로 해결 가능 | 대표키워드 교체, 리뷰/블로그 키워드 통일 |
| 0.5 이상 | **0.5 미만** | **축적 데이터 부족** — 투자로 해결 | 체험단 확대, 외부유입(메타광고), 저장 유도 |
| **0.5 미만** | **0.5 미만** | **둘 다 부족** — 세팅 먼저, 투자 다음 | 키워드 재세팅 → 체험단/광고 순서 |

### N1 세부 진단 (키워드 정합성)

N1이 낮을 때 체크할 5가지:

| # | 체크 포인트 | 확인 방법 |
|---|-----------|----------|
| 1 | 대표키워드에 목표 키워드 포함? | 대표키워드 5개 목록 확인 |
| 2 | 영수증 리뷰에 목표 키워드 자연 등장? | 리뷰 키워드 패턴 확인 |
| 3 | 블로그 체험단 키워드와 일치? | 체험단 가이드 키워드 확인 |
| 4 | 메뉴명에 관련 키워드 포함? | 메뉴 리스트 확인 |
| 5 | 소개문에 관련 문맥 포함? | 소개문 텍스트 확인 |

### N2 세부 진단 (축적 데이터)

N2가 낮을 때 체크할 4가지:

| # | 체크 포인트 | 기준 |
|---|-----------|------|
| 1 | 블로그 리뷰 수 | 상위권 평균 대비 비교 |
| 2 | 저장 수 | 상위권 평균 대비 비교 |
| 3 | 외부 유입 | 인스타/당근 → 플레이스 유입 경로 있는지 |
| 4 | 체류시간 | 영상/사진 10장+ 등 체류시간 부스터 |

---

## Step 3: 경쟁사 비교 — "상위권과 뭐가 다른지"

순위 비교분석 스크린샷이 있으면 아래 분석을 수행한다. 없으면 Step 2의 N1/N2 진단만으로 액션을 도출한다.

### 비교 항목

목표 키워드의 상위 1~5위 업체와 우리를 비교:

```
### 경쟁사 비교 ({목표 키워드})

| 순위 | 업체명 | 방문리뷰 | 블로그 | 저장수 | N1 | N2 |
|------|--------|---------|-------|-------|-----|-----|
| 1 | | | | | | |
| ... | | | | | | |
| {n} | {우리 매장} | | | | | |
```

### 격차 분석

| 지표 | 우리 | 상위 3위 평균 | 배수 차이 | 판정 |
|------|------|-------------|----------|------|
| 블로그 리뷰 | | | | 심각/부족/적정 |
| 저장수 | | | | 심각/부족/적정 |
| 방문자 리뷰 | | | | 심각/부족/적정 |
| N2 | | | | 근소/부족/심각 |

**병목 식별**: 가장 배수 차이가 큰 지표 = 핵심 병목. 이것부터 해결.

---

## Step 4: 액션 플랜 — "구체적으로 뭘 하라"

Step 1~3 결과를 종합하여 3가지 산출물을 생성한다.

### 4-1. 대표키워드 5개 재구성 제안

```
### 대표키워드 교체 제안

| 슬롯 | 현재 | 제안 | 교체 근거 |
|------|------|------|----------|
| 1 | | | |
| 2 | | | |
| 3 | | | |
| 4 | | | |
| 5 | | | |

교체 원칙:
- 지역+업종 조합 2~3개 (핵심)
- 목적형 키워드 1개 (점심, 해장, 모임 등)
- 기회 키워드 1개 (히든 키워드에서 발굴)
- 롱테일 최대 2개 (과도하면 네이버 반려)
- 업종 카테고리와 불일치하는 키워드 제거
```

### 4-2. 키워드별 N1/N2 액션 매핑

```
### 키워드별 액션

| 키워드 | 분류 | N1 액션 | N2 액션 |
|--------|------|---------|---------|
| {핵심 기회 1} | 핵심 기회 | 대표키워드 추가, 리뷰 키워드 유도 | 체험단 "키워드" 지정, 인스타 광고 |
| {핵심 기회 2} | 핵심 기회 | ... | ... |
| {기회 1} | 기회 | ... | ... |
```

### 4-3. 4주 키워드 액션 플랜

```
### 4주 키워드 액션 플랜

Week 1 — 세팅 최적화 (N1↑)
- [ ] 대표키워드 5개 교체 (담당: 우리)
- [ ] 메뉴명 키워드 추가 (담당: 사장님)
- [ ] 소개문에 목표 키워드 문맥 추가 (담당: 우리)
- [ ] 영수증 리뷰 가이드에 목표 키워드 반영 (담당: 우리)

Week 2 — 콘텐츠 투입 (N2↑)
- [ ] 블로그 체험단 {n}명 섭외 — "{키워드}" 지정 (담당: 우리)
- [ ] 인스타 광고 → 플레이스 링크 랜딩 세팅 (담당: 우리)
- [ ] 영상 소식 업로드 (담당: 사장님)

Week 3 — 모니터링 + 보정
- [ ] 에드로그 순위 체크 — 변동 확인 (담당: 우리)
- [ ] N1/N2 변화 확인 → 키워드 교체 판단 (담당: 우리)
- [ ] 유입 키워드 통계에서 실제 유입 확인 (담당: 우리)

Week 4 — 평가 + 차기 전략
- [ ] 목표 키워드 순위 변동 리포트 (담당: 우리)
- [ ] 기회 키워드 재발굴 — 히든 키워드 재확인 (담당: 우리)
- [ ] 대표키워드 미세 조정 (담당: 우리)
```

## 산출물 저장

파일 저장: `clients/{store-slug}/keyword-strategy.md`

```markdown
# {매장명} 키워드 전략 리포트

> 분석일: {날짜} | 데이터: 에드로그 히든 키워드
> 대표키워드: {현재 5개}
> 히든 키워드: {n}개

## 1. 키워드 분류표
(Step 1 결과)

## 2. N1/N2 진단
(Step 2 결과)

## 3. 경쟁사 비교
(Step 3 결과 — 데이터 있을 때만)

## 4. 대표키워드 교체 제안
(Step 4-1 결과)

## 5. 키워드별 액션
(Step 4-2 결과)

## 6. 4주 키워드 액션 플랜
(Step 4-3 결과)
```

저장 후 반드시 절대 경로를 표시한다.

## 주의사항

- 에드로그 스크린샷이 없으면 사용자에게 다시 요청한다. 구두 데이터로도 진행 가능하나 "스크린샷 기반이 훨씬 정확합니다"를 안내
- N1/N2 기준값 0.5는 교육 내용 기반. 실제로는 키워드마다 상위권 평균이 다를 수 있으므로, 비교분석 데이터가 있으면 상위권 평균 기준으로 판단한다
- 대표키워드 교체는 즉시 반영되지만, N1 반영에 1~2주 소요될 수 있음을 안내
- 롱테일 키워드는 5개 중 최대 2개까지만 권장 (과도하면 네이버가 반려)
- 업종 카테고리와 키워드 불일치 시 N1이 하락할 수 있음을 명시 (예: 국밥집에 "우동" 키워드)
- 띄어쓰기 버전이 검색 키워드임을 항상 기억 ("부산국밥" = 브랜드명, "부산 국밥" = 지역+업종)
