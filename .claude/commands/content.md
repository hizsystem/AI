# 콘텐츠 제작

콘텐츠를 기획하고 제작합니다. 콘텐츠 크리에이터와 브랜드 전략가가 협업합니다.

사용자 입력: $ARGUMENTS

---

## 콘텐츠 제작 5단계

### 1단계: 컨텍스트 파악

먼저 다음을 확인합니다:

- **콘텐츠 유형**: SNS / 블로그 / 뉴스레터 / 광고 / 랜딩페이지 / 기타
- **플랫폼**: 인스타그램 / 링크드인 / 블로그 / 이메일 / 페이스북 / 유튜브
- **목적**: 인지 / 참여 / 전환 / 리텐션
- **타겟**: 누가 볼 것인가
- **브랜드 톤**: (clients/ 폴더에 전략 문서가 있으면 참조)

사용자 입력에서 위 정보를 파악하고, 부족한 부분만 질문합니다.

### 2단계: 콘텐츠 기획 (✦ 브랜드 전략가)

브랜드 전략가 관점에서 방향을 설정합니다:

- **핵심 메시지**: 이 콘텐츠가 전달할 하나의 메시지
- **브랜드 적합성**: 브랜드 톤앤매너와의 일치 확인
- **경쟁사 차별화**: 비슷한 콘텐츠와의 차이점
- **전략적 위치**: 전체 콘텐츠 전략에서의 역할

### 3단계: 초안 제작 (✍ 콘텐츠 크리에이터)

2-3개 시안을 제작합니다. 각 시안은 다른 접근 방향입니다:

- **A안**: (예: 직접적/문제 해결형)
- **B안**: (예: 스토리텔링/감성형)
- **C안**: (예: 데이터/증거 기반형)

각 시안에 포함할 것:
- 헤드라인/훅
- 본문
- CTA
- 해시태그/키워드 (해당 시)
- 시안 선택 이유 설명

### 4단계: 크로스체크 (✦ 브랜드 전략가 → ✍ 콘텐츠 크리에이터)

브랜드 전략가가 초안을 리뷰합니다:

- 브랜드 톤앤매너 일치 여부
- 타겟 페르소나의 언어인지
- 경쟁사와 차별화되는지
- 전략적 메시지가 정확히 전달되는지

피드백을 반영하여 콘텐츠 크리에이터가 수정합니다.

### 5단계: 최종 산출물

최종 콘텐츠를 플랫폼별 포맷으로 제공합니다:

- 최종 카피 (선택된 시안 기준)
- 플랫폼별 최적화 버전 (길이, 포맷, 해시태그)
- 비주얼 가이드 (이미지 방향 제안)
- 게시 추천 시간대
- 성과 측정 기준

---

## 플랫폼별 포맷 가이드

| 플랫폼 | 길이 | 특징 |
|--------|------|------|
| 인스타그램 | 150자 내외 (캡션 2,200자) | 감성적, 비주얼 중심, 해시태그 15-20개 |
| 링크드인 | 1,300자 이내 | 전문적, 인사이트 중심, 스토리텔링 |
| 블로그 | 2,000-3,000자 | SEO 최적화, 구조화된 헤딩, 내부 링크 |
| 뉴스레터 | 500-800자 | 개인적 톤, 큐레이션, 명확한 CTA |
| 광고 카피 | 25자(헤드라인) / 90자(설명) | 후킹, 베네핏, 긴급성 |

## 작성 가이드

1. 사용자 입력이 구체적이면 바로 3단계(시안 제작)부터 시작합니다
2. 클라이언트 폴더에 전략 문서가 있으면 반드시 참조합니다
3. 항상 2-3개 시안을 제시하고 각각의 전략적 의도를 설명합니다
4. 완성된 콘텐츠는 `clients/{client-name}/content/` 에 저장합니다
