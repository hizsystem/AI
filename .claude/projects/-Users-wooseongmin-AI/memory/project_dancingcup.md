**Why:** 위드런 통합 어카운트 (미례국밥 500 + 댄싱컵 500 + 신메뉴 200 = 월 1,200만원). 바이럴 콘텐츠 마케팅 실험장으로서의 가치. 프랜차이즈 본사 마케팅 새 카테고리 레퍼런스.

**How to apply:**
- 킥오프: 2026-04-15(수) 부산, 미례국밥 촬영 후
- 실무 접점: 변미혜 팀장 / 대표: 조기원
- 여름 신메뉴 출시: 2026-05-12
- 전략: "두 바퀴" — 딥라떼(정체성) + 트렌드 캐치(화제성). 대표가 직접 잡은 방향, 우리는 수용+실행.
- 우리 스콥: 인스타+틱톡 집중 (콘텐츠 직접 제작). NP는 가이드만, 오프라인은 댄싱컵 내부.
- 킥오프 덱: clients/dancingcup/kickoff-deck.html
- 킥오프 전 보완 필요: 콘텐츠 방향 슬라이드, 바이럴 실행 구조(크루/체험단/외주), KPI 근거(블랙키위 대기), 핵심 한문장, PPL 전략
- 미례국밥 레퍼런스는 킥오프 덱에서 뺌
- 바이럴 콘텐츠 전담 팀 5명 에이전트 세팅 완료 (CLAUDE.md)

- 동시 프로젝트 3개 상한선 도달 (휴닉+미례국밥+댄싱컵)
