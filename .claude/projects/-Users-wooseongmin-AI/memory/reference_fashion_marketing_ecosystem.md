# 패션 브랜드 마케팅 생태계 레퍼런스

## 패션만의 5가지 마케팅 문법

1. **"누가 입었느냐"가 마케팅** — 셀럽/인플루언서 착용 = 브랜드 포지셔닝
2. **시즌 캘린더가 지배** — SS(3~4월 피크), FW(9~10월 피크), 6~8주 전 기획 시작
3. **자사몰 마진 vs 플랫폼 인지도** — 자사몰 62% vs 무신사 42% vs 백화점 37% 마진
4. **FW가 SS보다 매출 1.5~2배** — 아우터 단가 효과
5. **재고가 수익을 결정** — 시즌 내 60~70% 판매율이 양호

## 성장 단계별 마케팅 투자

| 단계 | 연매출 | 마케팅 비율 | 인스타 기준 |
|------|--------|-----------|-----------|
| 론칭기 | 0~10억 | 0~3% | ~5K |
| 초기 성장 | 10~50억 | 5~7% | 30~50K |
| 성장기 | 50~300억 | 7~15% | 100K+ |
| 성숙기 | 300억+ | 10~20% | - |

## 플랫폼 수수료 구조

- 무신사: 평균 27.6% (실질 9.4%)
- 29CM: 평균 28.0% (객단가 23만원, 거래액 1조)
- W컨셉: 평균 28.3%
- 자사몰: PG 3~5%만

## 벤치마크 브랜드

### 포터리 (POTTERY) — CRM 교과서
- 재구매율 ~50% (업계 평균 13~30%), 단골 매출 70%, 자사몰 70%+
- 도구: 데이터라이즈(CRM) + 채널톡(상담) + 카카오 알림톡
- 전략: 무신사=신규획득 → 자사몰=전환+CRM. 할인율 비노출 VIP 등급제(sVIP/VIP/VVIP)
- 오프라인 = CRM 접점 (수선 서비스, 브랜드 경험)

### ALO Yoga — 글로벌 라이프스타일 브랜딩 교과서
- 연매출 10억 달러+, 인스타 378만, 매장 169개
- 콘텐츠: 라이프스타일 30% / 셀럽 25% / 릴스 25% / UGC 10% / 프로모 10%
- CRM: ALO Access 무료 3-Tier 로열티 + Pro Program(강사 25% 할인) + Alo Moves 앱(200만 유저)
- 핵심: "제품이 아닌 라이프스타일에 소속감을 판다", 매장=미디어=커뮤니티

### 세터 (SATUR)
- 월 300만 광고비로 연 1,100억. 콘텐츠가 광고를 대체
- 뉴스레터: 매주 같은 요일·시간, 2년 반복 → 습관 형성
- 무신사 코디숍 콘텐츠 800개 아카이빙

### 성공 패턴 4가지
1. 대표=인플루언서 (마뗑킴)
2. 레거시 리브랜딩 (마리떼 3억→1,500억)
3. 오프라인 중심 확장 (세터)
4. 투자 레버리지 (하고엘앤에프)

## 대표님 필수 질문 3가지 (상담 시)
1. "어느 채널에서 매출이 제일 잘 나오세요?" → 자사몰 비중 = CRM 가능성
2. "월 광고비 얼마에 어느 채널이 가장 큰지?" → 투자 효율 + 스케일업 여지
3. "주목하고 있는 브랜드가 어디인지?" → 대표 머릿속 벤치마크 = 프로젝트 방향
