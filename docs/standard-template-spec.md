# 표준 클라이언트 Figma 템플릿 사양

고벤처포럼의 3구역 시스템을 기반으로 모든 클라이언트 파일에 적용할 표준 구조.

## 파일 구조

### 필수 페이지

| 페이지 | 용도 | 비고 |
|--------|------|------|
| COVER | 프로젝트 커버 | 클라이언트명 + 계약 기간 + 로고 |
| Instagram | 피드 콘텐츠 작업 | 3구역 시스템 적용 |
| 카드뉴스 | 카드뉴스 작업 | 슬라이드 단위 |
| 스토리/릴스 | 스토리/릴스 작업 | 1080x1920 포맷 |
| Brand Assets | 브랜드 에셋 모음 | 로고, 컬러, 폰트 레퍼런스 |

### 선택 페이지 (클라이언트 니즈에 따라)

| 페이지 | 용도 |
|--------|------|
| 광고 소재 | Meta/Google 광고용 |
| 프레젠테이션 | 제안서/보고서 |
| 인쇄물 | 명함, 브로셔, 포스터 |
| 무드보드 | 초기 방향성 설정용 |

## 3구역 시스템 (Instagram 페이지)

고벤처포럼에서 검증된 구조를 표준화.

```
┌─────────────────────────────────────────────────────┐
│  COMPONENT (수정 가능)                                │
│  ┌───────────────┐  ┌───────────────┐               │
│  │ Instagram     │  │ Instagram     │               │
│  │ Grid          │  │ Post          │               │
│  │ (그리드 편집)   │  │ (포스트 편집)   │               │
│  └───────────────┘  └───────────────┘               │
├─────────────────────────────────────────────────────┤
│  PREVIEW (수정 불가 -- 자동 반영)                       │
│  ┌───────────────┐  ┌───────────────┐               │
│  │ Grid          │  │ Post          │               │
│  │ Preview       │  │ Preview       │               │
│  └───────────────┘  └───────────────┘               │
├─────────────────────────────────────────────────────┤
│  INFO AREA                                          │
│  ┌────────┐  ┌─────────────┐  ┌──────────────────┐ │
│  │Profile │  │Post Header  │  │Post Description  │ │
│  └────────┘  └─────────────┘  └──────────────────┘ │
└─────────────────────────────────────────────────────┘
```

### 구역별 규칙

| 구역 | 편집 권한 | 역할 |
|------|----------|------|
| COMPONENT | 디자이너만 | 실제 디자인 수정 영역. 여기서 변경하면 PREVIEW에 자동 반영 |
| PREVIEW | 수정 불가 | 인스턴스로 연결. 실제 Instagram 화면 미리보기 |
| INFO AREA | 디자이너만 | 프로필 정보, 포스트 헤더, 설명 텍스트 등 메타 정보 |

### 안내 텍스트 (파일에 포함)

```
- 넣고 싶은 이미지 복사 -> 목표 위치 ctrl+클릭 -> 붙여넣기(ctrl+V)
- ctrl+클릭 하여 원하는 오브젝트를 선택할 수 있습니다
- 피드 영역 선택한 상태에서 더블클릭 -> 게시글 하나 선택됩니다
- 이 상태로 드래그하여 순서 바꿀 수 있습니다
- SHIFT + SPACE BAR 입력 시 프리뷰 재생 가능합니다
```

## 카드뉴스 페이지 구조

브랜드라이즈에서 검증된 구조를 표준화.

```
카드뉴스 페이지 (1080x1350 포맷)
├── 타이틀장        # 시리즈 제목 + 브랜드 로고
├── 세부 제목장     # 세부 주제 + 번호
├── 내용장 (복수)   # 본문 콘텐츠
└── 마지막장        # CTA + Instagram 핸들 + 이미지 그리드
```

### 마지막장 표준 레이아웃

```
┌──────────────────────────┐
│  주제 카테고리 (상단)      │
│                          │
│  ┌──────┐  ┌──────┐     │
│  │ img1 │  │ img2 │     │
│  └──────┘  └──────┘     │
│  ┌──────┐  ┌──────┐     │
│  │ img3 │  │ img4 │     │
│  └──────┘  └──────┘     │
│                          │
│  CTA 메시지 (하단)        │
│  @instagram_handle       │
└──────────────────────────┘
```
