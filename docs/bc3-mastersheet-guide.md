## 새 프로젝트 온보딩 체크리스트

프로젝트 수주 확정 시 아래 순서대로 진행:

### Step 1: 내부 마스터시트 생성

1. Google Sheets에서 **새 스프레드시트** 생성
2. 이름: `[2026] {프로젝트명} 내부 마스터시트`
3. 확장 프로그램 → Apps Script → `bc3-master-template.gs` 코드 붙여넣기
4. **CONFIG 섹션 수정** (프로젝트 정보 입력):
   - `projectName`, `clientName`, `industry`
   - `contractPeriod`, `contractAmount`
   - `startMonth`, `endMonth`, `monthlyRetainer`
   - `teamMembers`, `clientContacts`
   - `parts` (비용 마진 탭의 Part 구성)
5. `setupMasterSheet()` 실행
6. 팀원에게 편집 권한 공유

### Step 2: 공유 마스터시트 생성

1. 새 스프레드시트 생성
2. 이름: `{프로젝트명} x 브랜드라이즈 | 프로젝트 마스터시트`
3. 수동으로 4탭 생성:
   - MAP (월간 운영 플랜)
   - 대시보드 (KPI 주간 트래킹)
   - 스케줄링 (간트 차트)
   - 링크 & 정산

### Step 3: 총괄 시트 업데이트

1. `bc3-summary-setup.gs`의 `PROJECTS` 배열에 새 프로젝트 추가
2. `setupSummarySheet()` 재실행
   - ⚠️ 기존 수동 입력 데이터가 초기화됨 — 백업 후 실행
3. 또는 수동으로 총괄 시트에 행 추가 (안전)

### Step 4: 초기 데이터 입력

- [ ] Summary 탭: 계정정보, 팀구성 확인
- [ ] 2026 예상 매출: 계약서 기준 월별 매출 입력
- [ ] 2026 예상 지출: 예상 비용 항목별 입력
- [ ] 월별 비용 마진: Part별 항목 + 견적 입력
- [ ] 인플루언서 관리: 해당 시 초기 후보 리스트

### Step 5: 연동 확인

- [ ] 총괄 Dashboard에서 해당 프로젝트 행 확인
- [ ] 마스터시트 링크 연결
- [ ] (선택) IMPORTRANGE 수식으로 자동 연동

## 내부 마스터시트 — 표준 6탭 구조

### 탭 1: Summary
- 프로젝트 개요 (프로젝트명/클라이언트/기간/금액)
- 핵심 수치 요약 (예상매출/지출/수익률 — 자동 계산)
- 팀구성
- 계정 정보 ⚠️ 내부 전용
- 주요 링크 & 정산 (월별 운영플랜/견적서/보고서/입금/세금계산서)
- 시트 색인

### 탭 2: 비용 상세
- 월별 경비 집행 내역
- 열: 월/NO/구분/항목/비용(VAT별도)/비용(VAT포함)/결제방식/결제날짜/세금계산서/발급날짜/결제자/비고
- 구분 드롭다운: 진행비, 광고비, 인플루언서, 체험단, 촬영, 디자인/인쇄, 출장비, 툴비용, 기타

### 탭 3: 월별 비용 마진 ★핵심
- 미례국밥 구조 기반
- 열: 월/구분/항목/내용/견적(VAT별도)/예상비용/실지출/마진/수익률/비고
- Part별 소계 → 월 합계 구조
- **이 탭의 월 합계행이 총괄 시트로 연동됨 — 구조 변경 금지**

### 탭 4: 인플루언서 관리
- 씨딩/체험단/인플루언서 전체 관리
- 열: NO/월/유형/이름/팔로워/플랫폼/카테고리/협찬내용/비용/결제/컨택~업로드일/URL/성과/CPV/상태/비고
- CPV(조회당 비용) 자동 계산

### 탭 5: 2026 예상 매출
- 세금계산서 기준 월별 매출 예측
- 행 3이 합계행 (Named Range: `revenue_total`, `revenue_{월}`)
- 리테이너 자동 입력 (CONFIG에서 설정)

### 탭 6: 2026 예상 지출
- 카테고리별: 인건비/외주비/광고비/인플루언서·체험단/경비/툴·구독
- 행 3이 합계행 (Named Range: `expense_total`, `expense_{월}`)

## BC3 총괄 대시보드 — 4탭 구조

### 탭 1: Dashboard
- KPI (인당 3억 × 5명 = 15억)
- 프로젝트별 현황 (예상매출/실매출/예상지출/실지출/수익률)
- 각 프로젝트 마스터시트 링크

### 탭 2: 월별 수익 요약
- 1T 구조 그대로
- 총 매출 (프로젝트별 월별) → 총 지출 (프로젝트별 월별) → 팀 인건비 → 수익률
- 수익률 = (매출 - 지출 - 인건비) / 매출

### 탭 3: 월별 캐시플로우
- 입금 예정 / 실입금 / 미수금 / 실지출 / 순 캐시플로우
- "이번 달 입금 됐나?" 한눈에 확인

### 탭 4: 팀 리소스
- 팀원 × 프로젝트 배정 매트릭스
- Lead/Support 표시
- 프로젝트별 인력 현황

## 주간 체크 루틴

**매주 월요일 (10분):**

1. **총괄 Dashboard** → KPI 달성률 확인
2. **월별 수익 요약** → 이번 달 수익률 확인
3. **캐시플로우** → 미수금 체크, 입금 누락 확인

**매월 1일 (30분):**

1. 각 프로젝트 **월별 비용 마진** → 전월 실지출 마감
2. 총괄 시트 데이터 업데이트 (IMPORTRANGE 아닌 경우)
3. 예상 매출/지출 → 다음 달 예측 조정

## 절대 규칙

1. **내부 마스터시트의 탭 이름/열 순서를 변경하지 마라** → 총괄 IMPORTRANGE 깨짐
2. **비용 마진 탭의 월 합계행을 삽입/삭제하지 마라** → Named Range 깨짐
3. **계정 정보가 있는 시트는 공유하지 마라** → 보안 사고
4. **새 프로젝트는 반드시 템플릿으로 생성하라** → 표준 유지
5. **Apps Script 원본은 Git에 보관하라** → 버전 관리
