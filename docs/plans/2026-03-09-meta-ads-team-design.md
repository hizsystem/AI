## 1. 개요

Meta 광고 전담 에이전트 4명 + 스킬 4개 + 뉴비즈니스 마스터시트 대시보드 재설계.
클라이언트 대행 + 자체 마케팅 병행 목적.

## 2. 에이전트 (4명)

| 에이전트 | 파일 | 역할 |
|---------|------|------|
| (🎯) Meta 미디어바이어 | `meta-media-buyer` | 캠페인 구조, 타겟, 예산, 입찰 전략 |
| (🎨) Meta 크리에이티브 PD | `meta-creative-pd` | 소재 기획, 카피, A/B 테스트, 피로도 관리 |
| (📈) Meta 그로스 분석가 | `meta-growth-analyst` | 지표 분석, 퍼널 진단, 스케일링, 어트리뷰션 |
| (📋) Meta 어카운트 매니저 | `meta-account-manager` | 온보딩, 리포팅, 클라이언트 관리, 일정 |

## 3. 스킬 (4개)

| 커맨드 | 기능 |
|--------|------|
| `/meta-onboard` | 신규 클라이언트 Meta 광고 온보딩 (브리프 → 픽셀 체크 → 초기 캠페인) |
| `/meta-campaign` | 캠페인 구조 자동 설계 (목표 → 오디언스 → 예산 → 소재 방향) |
| `/meta-report` | 성과 보고서 생성 (일간/주간/월간, 내부/클라이언트용) |
| `/meta-optimize` | 최적화 진단 (ROAS 분석 → 병목 → 에이전트별 액션) |
