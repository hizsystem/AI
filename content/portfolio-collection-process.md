# 포트폴리오 정보 취합 프로세스

> 목적: 각 담당자로부터 최소 부담으로 포트폴리오 원재료를 수집하는 시스템
> 작성일: 2026-03-12

---

## 전체 프로세스 요약

```
[STEP 1] 취합 폼 배포 (담당자 5분)
     ↓
[STEP 2] 자료 등급 분류 (PM 30분)
     ↓
[STEP 3] 보충 인터뷰 (담당자 20분)
     ↓
[STEP 4] 콘텐츠 초안 작성 (PM/AI)
     ↓
[STEP 5] 팩트체크 + 공개동의 (담당자 10분)
     ↓
[STEP 6] 발행
```

**원칙: 담당자 총 소요시간 35분 이내**

---

## STEP 1. 취합 폼

### 배포 방식 (택 1)

| 방식 | 장점 | 추천 상황 |
|------|------|----------|
| **Notion 데이터베이스** | 한 곳에서 관리, 필터/정렬 가능 | 팀이 Notion 쓰는 경우 |
| **Google Forms → Sheets** | 모바일에서도 쉽게 입력 | 담당자가 바쁜 경우 |
| **슬랙 스레드 + 템플릿** | 가장 진입장벽 낮음 | 빠르게 시작할 때 |

### 폼 질문 설계

> 설계 원칙: 서술형 최소화, 선택형/단답형 위주

#### Part C. 자료 현황 체크 (1분)

| # | 질문 | 입력 형태 |

| 17 | 클라이언트명 공개 가능? | 가능 / 업종만 공개 / 불가 |

| 20 | 클라이언트 동의 필요? | 이미 받음 / 받아야 함 / 확인 필요 |

## STEP 2. 등급 분류 기준

폼 회수 후 PM이 아래 기준으로 분류:

### A급 → 즉시 콘텐츠화

```
✅ 성과 수치가 명확하다
✅ Before/After 자료가 있다
✅ 클라이언트 공개 가능하다 (또는 업종 공개라도)
```

### B급 → 인터뷰 후 콘텐츠화

```
⚠️ 수치는 애매하지만 스토리가 좋다
⚠️ 자료가 일부만 있다 ("찾아봐야 함" 항목 있음)
⚠️ 공개 동의를 받아야 한다
```

### C급 → 보류

```
❌ 자료가 거의 없다
❌ 공개 불가능하다
❌ 성과도 스토리도 약하다
```

### 분류 결과 정리표

| 프로젝트 | 카테고리 | 등급 | 다음 액션 | 담당자 |
|----------|---------|------|----------|--------|
| OO 팝업 | 팝업 | A | 콘텐츠 제작 | 김OO |
| XX CRM | CRM | B | 인터뷰 필요 | 박OO |
| ... | ... | ... | ... | ... |

**목표: 8개 카테고리 × 최소 1개 A~B급 확보**

## STEP 3. 보충 인터뷰 (B급 프로젝트)

### 진행 방식

- **시간**: 20분 (슬랙 허들 or 대면)
- **녹음**: 반드시 녹음 (AI 정리 활용)
- **인터뷰어**: PM (또는 콘텐츠 담당)

### 인터뷰 질문지 (7개)

```
[도입 - 1분]
"폼에 적어준 내용 기반으로 몇 가지만 더 여쭤볼게요."

[질문 - 15분]

1. 처음 상황
   "클라이언트가 처음 왔을 때 상황이 어땠어요?
    어떤 문제/니즈가 있었어요?"

2. 우리의 접근
   "그래서 우리가 제일 먼저 뭘 했어요?
    왜 그 방법을 선택했어요?"

3. 실행 과정
   "진행하면서 제일 어려웠던 건 뭐였어요?
    어떻게 해결했어요?"

4. 핵심 결과
   "결과적으로 뭐가 달라졌어요?
    클라이언트 반응은 어땠어요?"

5. 자랑 포인트
   "이 프로젝트에서 우리가 잘한 거 하나만 꼽으면?"

6. 배운 점
   "다시 한다면 다르게 할 것 있어요?"

7. 자료 확인
   "혹시 당시 캡처나 사진 남아있는 거 있어요?
    카톡/슬랙/드라이브 어디든."

[마무리 - 1분]
"감사합니다. 초안 나오면 팩트체크 부탁드릴게요."
```

## STEP 5. 팩트체크 + 공개동의

### 담당자 팩트체크 (10분)

콘텐츠 초안 완성 후 담당자에게 전달:

```
[팩트체크 요청 메시지 템플릿]

OO님, 포트폴리오 콘텐츠 초안 나왔습니다!

📄 초안 링크: [링크]

아래 3가지만 확인 부탁드려요 (10분):
1. ❌ 틀린 내용 있나요?
2. ❌ 공개되면 안 되는 내용 있나요?
3. ✅ 이대로 발행해도 될까요?

답변 기한: O월 O일까지
```

### 클라이언트 공개동의 (필요 시)

```
[클라이언트 동의 요청 메시지 템플릿]

안녕하세요, [클라이언트명] 담당자님.

저희가 함께 진행했던 [프로젝트명] 프로젝트를
저희 회사 포트폴리오 콘텐츠로 소개하고 싶습니다.

▸ 게재 채널: 자사 인스타그램 / 블로그
▸ 공개 범위: 업종 + 성과 (구체 수치는 범위로 표기)
▸ 초안 미리보기: [링크]

게재 전 최종본 한 번 더 확인 드리겠습니다.
동의해 주시면 감사하겠습니다.
```

## 카테고리별 자료 수집 체크리스트

### 팝업
- [ ] 공간 사진 (시공 전/후)
- [ ] 방문자 수 데이터
- [ ] 현장 사진 (줄/체험/전시)
- [ ] SNS 바이럴 캡처 (스토리 태그 등)

### 행사
- [ ] 행사 현장 사진
- [ ] 참석자 수/만족도 설문
- [ ] 프로그램 스케줄
- [ ] 미디어 노출 캡처

### 메타광고
- [ ] 광고 관리자 대시보드 캡처 (ROAS, CPA, CTR)
- [ ] 월별 성과 추이
- [ ] 주요 소재 캡처 (블러 가능)
- [ ] 캠페인 구조 변경 전/후

### CRM
- [ ] 자동화 플로우 캡처
- [ ] 재구매율/LTV 변화 데이터
- [ ] 메시지 오픈률/전환률
- [ ] 세그먼트 구조

### 체험단
- [ ] 체험단 리뷰 캡처 (블로그/인스타)
- [ ] 검색 결과 캡처 (전/후)
- [ ] 리뷰 수 집계
- [ ] 가이드라인 샘플

### 인플루언서 씨딩
- [ ] 협업 인플루언서 콘텐츠 캡처
- [ ] 도달/참여율 데이터
- [ ] 비용 대비 성과 (CPE)
- [ ] 인플루언서 리스트 (비공개용)

### 공구
- [ ] 공구 매출 데이터
- [ ] 셀러 콘텐츠 캡처
- [ ] 실시간 반응 캡처 (댓글, DM)
- [ ] 판매 수량/시간대별 추이

### 브랜드 리뉴얼
- [ ] 리뉴얼 전/후 비교 이미지
- [ ] 디자인 시안 발전 과정
- [ ] 리뉴얼 후 매출/반응 변화
- [ ] 소비자 피드백 캡처
