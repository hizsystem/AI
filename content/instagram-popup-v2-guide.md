# 인스타그램 카드뉴스 수정안 — 팝업 카테고리 (Option A 큐레이션형)

> 시리즈명: "요즘 브랜드는 이렇게 ___한다"
> 첫 번째: 팝업편
> 계정: @brandrise_kr

---

## 전체 구조 비교

### Before (현재 10장)
```
타이틀 → 비비고(3장) → 네스카페(2장) → 스벅(1장) → 리솔츠(1장) → 뉴욕(1장) → 도쿄(1장) → CTA
= 사진 나열 중심, 역할/성과 부재
```

### After (수정안 9장)
```
타이틀 → 프로젝트 4개(각 1장) → 글로벌(1장) → 수치 요약(1장) → 역할 범위(1장) → CTA
= 프로젝트 압축 + 수치/역할 추가
```

---

## 슬라이드별 상세 가이드

---

### 1장. 타이틀 (훅) — 유지, 소폭 수정

**현재:** 좋음. 유지.

**수정 포인트:**

| 요소 | 현재 | 수정 |
|------|------|------|
| 메인 카피 | 요즘 브랜드는 이렇게 팝업을 만든다 | ✅ 유지 |
| 서브 카피 | 눈에 띄는 팝업 콘텐츠 모음 | → **"기획부터 운영까지, 우리가 함께한 팝업들"** |
| 배경 이미지 | 비비고×세븐틴 현장 | ✅ 유지 |

**이유:** 서브 카피에 "우리가 함께한"을 넣어야 포트폴리오임이 인지됩니다. 현재 "콘텐츠 모음"이면 남의 팝업 큐레이션으로 오해할 수 있습니다.

**카피 시안:**
```
메인: 요즘 브랜드는 이렇게
      팝업을 만든다

서브: 기획부터 운영까지, 우리가 함께한 팝업들

@brandrise_kr
```

---

### 2장. 세븐틴 × 비비고 — 3장 → 1장으로 압축

**현재:** 홍대점, 명동점, 세빛섬 각 1장씩 (3장)

**수정:** 대표 사진 1장 + 텍스트 오버레이

**레이아웃:**
```
┌─────────────────────────┐
│ [홍대점]                  │  ← 주황 태그 유지
│                           │
│   (홍대점 대표 사진)       │  ← 가장 임팩트 있는 1장
│                           │
│                           │
│ ┌───────────────────────┐│
│ │ 세븐틴 X 비비고         ││
│ │ 10주년 기념 팝업        ││
│ │                        ││
│ │ 홍대·명동·세빛섬 3개소  ││  ← 스케일 표현
│ │ 동시 운영               ││
│ └───────────────────────┘│
│               @brandrise_kr│
└─────────────────────────┘
```

**카피:**
```
세븐틴 X 비비고
10주년 기념 팝업

홍대·명동·세빛섬 3개소 동시 운영
```

**왜 압축?** 3장을 써도 "와 여러 지점이다"라는 정보 이상을 주지 못합니다. 1장에 "3개소 동시 운영"이라고 쓰면 같은 임팩트를 텍스트 한 줄로 전달합니다.

---

### 3장. 네스카페 돌체구스토 네오 — 2장 → 1장으로 압축

**수정:** 런칭쇼 현장 대표 사진 1장 (영등포 타임스퀘어 전경이 스케일감 가장 좋음)

**레이아웃:**
```
┌─────────────────────────┐
│ [영등포 타임스퀘어]        │  ← 주황 태그
│                           │
│  (런칭쇼 전경 사진)       │
│                           │
│                           │
│ ┌───────────────────────┐│
│ │ 네스카페 돌체구스토      ││
│ │ 네오 런칭쇼             ││
│ │                        ││
│ │ 영등포·하남 2개소 동시   ││
│ └───────────────────────┘│
│               @brandrise_kr│
└─────────────────────────┘
```

**카피:**
```
네스카페 돌체구스토
네오 런칭쇼

영등포·하남 2개소 동시
```

### 4장. 스타벅스 Coffee at Home — 유지, 카피 보강

**수정:** 기존 하남 스타필드 사진 활용 + 텍스트 보강

**카피:**
```
스타벅스 Coffee at Home
by NESCAFE

하남 스타필드 체험형 팝업
```

---

### 5장. 리솔츠 국내외 팝업 — 유지, 카피 보강

**수정:** 기존 사진 활용 + 글로벌 포인트 강조

**카피:**
```
자사브랜드 리솔츠
국내·뉴욕·도쿄 팝업

3개국 동시 브랜드 경험 설계
```

**변경:** 기존 7~9장에 흩어진 뉴욕/도쿄를 여기서 리솔츠와 묶습니다.

---

### 6장. 글로벌 팝업 모아보기 — 신규 (기존 8,9장 통합)

**레이아웃:** 2분할 또는 3분할 그리드

```
┌─────────────────────────┐
│  [뉴욕]    │   [도쿄]    │  ← 주황 태그 2개
│            │             │
│ (뉴욕 팝업 │ (도쿄 팝업   │
│  사진)     │  사진)       │
│            │             │
│ ┌───────────────────────┐│
│ │ 서울을 넘어             ││
│ │ 뉴욕, 도쿄까지          ││
│ └───────────────────────┘│
│               @brandrise_kr│
└─────────────────────────┘
```

**카피:**
```
서울을 넘어
뉴욕, 도쿄까지
```

**효과:** 짧은 카피가 글로벌 스케일을 강조합니다.

### 7장. 수치 요약 — 신규 추가 (핵심 장)

**이 장이 "사진 갤러리"와 "포트폴리오"를 가르는 결정적 차이입니다.**

**레이아웃:** 텍스트 중심, 다크 배경 또는 브랜드 컬러 배경

```
┌─────────────────────────┐
│                           │
│     (브랜드 컬러 배경)     │
│                           │
│        팝업 실적           │
│                           │
│     OO+  프로젝트 수행     │  ← 큰 숫자
│                           │
│     OO만+ 누적 방문자      │
│                           │
│     O개 도시              │
│     서울·뉴욕·도쿄         │
│                           │
│               @brandrise_kr│
└─────────────────────────┘
```

**카피 예시:**
```
팝업 실적

15+  프로젝트 수행
10만+ 누적 방문자
3개국 5개 도시

*수치는 실제 데이터로 교체
```

**디자인 팁:**
- 숫자를 크게 (볼드, 36pt+)
- 설명 텍스트는 작게 (16pt)
- 배경은 사진이 아닌 단색 또는 그라데이션 → 숫자에 집중

### 8장. 역할 범위 — 신규 추가 (두 번째 핵심 장)

**목적:** "이 회사가 어디까지 해주는 건지"를 명확히

**레이아웃:**

```
┌─────────────────────────┐
│                           │
│     우리가 하는 일         │
│                           │
│  ① 컨셉 기획              │
│     브랜드에 맞는 공간 설계 │
│                           │
│  ② 공간 디자인 & 시공      │
│     동선·연출·그래픽 전체   │
│                           │
│  ③ 현장 운영              │
│     스태프·물류·고객 관리   │
│                           │
│  ④ 성과 분석              │
│     방문자·SNS·매출 리포팅  │
│                           │
│               @brandrise_kr│
└─────────────────────────┘
```

**카피:**
```
우리가 하는 일

① 컨셉 기획
   브랜드에 맞는 공간 경험 설계

② 공간 디자인 & 시공
   동선·연출·그래픽

③ 현장 운영
   스태프·물류·고객 관리

④ 성과 분석
   방문자·SNS·매출 리포팅
```

**디자인 팁:**
- 7장과 같은 톤의 배경 (단색/그라데이션)
- 번호 or 아이콘으로 시각 구분
- 한 줄씩 간결하게

### 9장. CTA (마지막) — 수정

**현재:** "팝업, 마케팅 전문가와 함께 제대로 하세요"

**수정:**

```
┌─────────────────────────┐
│                           │
│     (다크 배경)            │
│                           │
│   다음 팝업,               │
│   함께 만들어요            │
│                           │
│   컨셉부터 운영까지        │
│   원스톱으로 진행합니다     │
│                           │
│   ┌─────────────────┐    │
│   │ DM으로 무료 상담 →│    │  ← 주황 버튼
│   └─────────────────┘    │
│                           │
│               @brandrise_kr│
└─────────────────────────┘
```

**카피:**
```
다음 팝업,
함께 만들어요

컨셉부터 운영까지
원스톱으로 진행합니다

[DM으로 무료 상담 →]
```

**변경 이유:** "마케팅 전문가와 함께 제대로 하세요"는 약간 훈계 톤. "함께 만들어요"가 더 자연스럽고, 8장에서 역할 범위를 봤기 때문에 "컨셉부터 운영까지"가 구체적으로 와닿습니다.

## 전체 흐름 요약

| 장 | 역할 | 체류 목적 |
|---|------|----------|
| 1 | 훅 (멈추게 하기) | "트렌드 알려주는 콘텐츠네" |
| 2 | 프로젝트 ① 비비고×세븐틴 | "오 이 팝업 이 회사가 했어?" |
| 3 | 프로젝트 ② 네스카페 | "이것도?" |
| 4 | 프로젝트 ③ 스타벅스 | "대기업도 맡기네" |
| 5 | 프로젝트 ④ 리솔츠 + 글로벌 | "자체 브랜드도 있구나" |
| 6 | 글로벌 확장 | "해외까지 하네" |
| **7** | **수치 요약** | **"진짜 많이 했네, 실력 있겠다"** |
| **8** | **역할 범위** | **"기획부터 분석까지 다 해주는구나"** |
| 9 | CTA | "연락해볼까" |

**보는 사람의 심리 흐름:**
```
예쁘다 → 이 회사가 했어? → 대기업도? → 해외도? → 숫자로 확인 → 뭘 해주는지 이해 → 연락
```
