## 세그먼트 3: 중기 (매출 10-30억)

### 프로파일
- **기업 규모**: 직원 10-30명, 마케팅 담당 1-3명
- **현재 상태**: 매출은 나오지만 브랜드 자산이 없음. 퍼포먼스에 의존
- **예산**: 월 마케팅 예산 500만-2,000만원
- **의사결정자**: 대표 또는 마케팅 리드

### Pain Point (✅ 딥 리서치로 검증 완료 — segment3-mid-growth.md)
1. **팀은 있는데 전략이 없다** — 주니어 마케터 1-3명이 실행만, 전략적 질문에 답할 사람 없음
2. **매출은 나는데 브랜드가 안 쌓인다** — CAC 상승 악순환, 광고 끄면 매출 0
3. **10억 → 30억 스케일업 벽** — 단일채널→멀티채널, 전술→전략 전환 필요
4. **대행사 한계 체감** — 월 200-500만원에 배정되는 주니어 AE, ROI 불투명
5. **CMO 채용 불가** — 연봉 8천만-1.5억, 채용 3-6개월, 온보딩 3개월

### 제공 서비스
- 브랜드 전략 리빌딩
- 퍼포먼스 마케팅 최적화
- 콘텐츠 마케팅 시스템 구축
- 통합 CMO 서비스

## 세그먼트 4: 중중기 (매출 30-100억)

### 프로파일
- **기업 규모**: 직원 30-100명, 마케팅팀 3-10명
- **현재 상태**: 채널은 여러 개 운영 중이나 통합 전략 부재
- **예산**: 월 마케팅 예산 2,000만-1억원
- **의사결정자**: 대표, CMO, 마케팅 이사

### Pain Point (✅ 딥 리서치로 검증 완료 — segment4-expansion.md)
1. **마케팅팀은 있는데 전략적 방향이 없다** — 5-10명이 각자 실행, KPI가 비즈니스와 연결 안 됨
2. **대행사 3-4곳 통합 불가** — 채널별 분리 운영, 메시지 불일치, 통합 대시보드 없음
3. **30억 → 100억 스케일업 벽** — 기존 성장 방식 한계, 멀티채널+브랜드 전환 필요
4. **CMO 채용 딜레마** — 연봉 1-2억, 평균 재임 4.1년(C-레벨 중 최단), 문화 핏 리스크
5. **브랜드 부재의 대가** — 가격 경쟁에 갇힘, 충성 고객 없음, 리포지셔닝 시점 판단 불가

### 제공 서비스
- 브랜드 전략 리빌딩 + 비주얼 리뉴얼
- 멀티채널 퍼포먼스 최적화
- 콘텐츠 마케팅 시스템 + 팀 역량 강화
- 통합 CMO 서비스 (전략-실행 원스톱)
