## 1. REAL WHY — 타겟의 진짜 감정

### 핵심 감정: "막막함" + "외로움"

초기 스타트업 대표의 마케팅 관련 감정은 한 마디로 **"뭘 해야 하는지 모르겠고, 물어볼 데도 없다"**이다.

```
감정 지도:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
막막함 ████████████████████ (40%) — 방향을 모른다
외로움 ███████████████      (30%) — 혼자 다 해야 한다
불안함 ██████████           (20%) — 돈 쓰고 실패할까 봐
조급함 █████                (10%) — 경쟁사는 앞서가는데
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

## 2. Pain Point TOP 5 + Hidden Pain

| 순위 | Pain Point | 핵심 발언 | 감정 |
|------|-----------|----------|------|
| 1 | 뭘 어디서부터 해야 할지 모르겠다 | "채널은 많은데 우리한테 맞는 게 뭔지..." | 막막함 |
| 2 | 대표가 직접 해야 하는데 본업도 버겁다 | "밤 11시에 인스타 만들고 아침에 투자자 미팅" | 번아웃 |
| 3 | 돈 써봤는데 효과를 모르겠다 | "대행사가 우리 사업을 이해 못 한다" | 배신감 |
| 4 | 브랜딩, 지금 해야 하나? | "PMF도 못 찾았는데 브랜딩이 사치 아닌가" | FOMO |
| 5 | 좋은 제품인데 아무도 모른다 | "기술은 자신 있는데 알리는 법을 모르겠다" | 좌절 |

### Hidden Pain (대표가 말 안 하는 것)
- 마케팅 모르니까 **사기당할까 불안** (호구 잡히는 느낌)
- 정부지원사업 끝나면 **마케팅비 절벽** (갑자기 0원)
- 경쟁사 마케팅 보면 **극심한 비교 스트레스**
- 성과가 안 나올 때 **제품 문제인지 마케팅 문제인지 진단 불능**

## 3. Purchase Trigger — 결제하는 순간

### 전환율 TOP 3 (높은 순서)

| 순위 | Trigger | 왜 전환율이 높은가 | 최적 메시지 |
|------|---------|------------------|-----------|
| 1 | **투자 유치 직후** | 예산 확보 + 빠른 실행 압박 | "투자금의 첫 3개월이 전체 ROI를 결정합니다" |
| 2 | **런칭 D-30** | 데드라인 = 즉시 결정 | "런칭 D-30. 지금 시작해도 됩니다" |
| 3 | **대표 번아웃** | 감정적 구매 + 절박함 | "더 이상 혼자 안 하셔도 됩니다" |

### 구매 장벽 TOP 3 + 해결책

| 장벽 | 해결책 |
|------|--------|
| 가격 부담 | 가격 투명 공개 + "풀타임 마케터 vs brandrise" 비교표 |
| 성과 불확실성 | 파일럿 프로젝트 + 주간 데이터 리포팅 + 리스크 리버설 |
| 신뢰 부족 | 스타트업 특화 포트폴리오 + 대표 개인 브랜딩 + 무료 진단 |

## 4. 콘텐츠 전략 — 무엇을 어디에

### 채널 우선순위

| 순위 | 채널 | 역할 | 최적 포맷 | 발행 |
|------|------|------|----------|------|
| 1 | **링크드인** | 신뢰 구축 + 리드 | 캐러셀, 1500자 텍스트 | 화~목 아침 8-9시, 주 3회 |
| 2 | **인스타그램** | 인지도 + 레퍼런스 | 릴스, 카드뉴스 | 수/금 저녁 7-8시, 주 2-4회 |
| 3 | **유튜브** | 전문성 증명 | 10-15분 사례 분석 | 화 저녁 7시, 격주 |
| 4 | **뉴스레터** | 리드 너처링 | 인사이트 + 사례 | 월 아침 8시, 주 1회 |

### 반응 높은 주제 TOP 5

| 순위 | 주제 | 반응 유형 |
|------|------|----------|
| 1 | 구체적 숫자 포함 성장 사례 | 저장 + 댓글 폭발 |
| 2 | 저비용 고효율 마케팅 방법 | 저장 + 공유 |
| 3 | 실패 경험 + 교훈 | 댓글 + 공감 |
| 4 | Before/After 성과 비교 | 저장 + 문의 |
| 5 | 바로 쓸 수 있는 도구/템플릿 | 저장 + 리드 확보 |

### 콘텐츠 제작 원칙

```
모든 콘텐츠 = Pain → Value → Proof

1. Pain (고객의 고통을 건드린다)
   "마케팅 예산을 태워도 성과가 안 나는 스타트업 대표님"

2. Value (해결 방법을 제시한다)
   "3개월 만에 ROAS 340%를 만든 전략이 있습니다"

3. Proof (증거를 보여준다)
   "실제 클라이언트 A사의 Before/After 데이터"
```

### 소비 패턴 핵심

- **스낵형 소비**: 5-10분 단위로 끊어서 본다 → 첫 2초가 생사
- **저장 후 읽기**: 저장율이 KPI → 저장할 가치가 있어야 한다
- **동료 추천 신뢰**: 같은 창업자가 공유한 콘텐츠 신뢰도 2-3배
- **실행 가능성 필터**: "우리 팀에서 바로 써먹을 수 있나?"가 최종 판단

## 5. 경쟁사 대비 포지셔닝

### 경쟁사 공통 약점

| 약점 | 구체적 현상 |
|------|-----------|
| 사업 이해도 부족 | 템플릿식 결과물, "대행사는 공동창업자가 아니다" |
| 전략-실행 분리 | 코칭형은 전략만, 대행형은 실행만 |
| ROI 불투명 | 노출수만 보고, 매출 기여 안 보여줌 |
| 가격-품질 양극화 | 싸면 품질 낮고, 좋으면 감당 불가 |
| 지식 이전 없음 | 외주 끝나면 내부에 남는 게 없음 |

### brandrise 포지션

```
전략(코칭형) + 실행(에이전시형) + 사업 이해(인하우스)
= "스타트업 전담 CMO 팀"
```

### 핵심 차별화 메시지

> **"마케팅 외주가 아닙니다. 당신의 스타트업에만 집중하는 CMO 팀입니다."**

### 경쟁사 경험자별 전환 메시지

| 경험 | 전환 카피 |
|------|----------|
| 에이전시 실망 | "동시 3개 프로젝트만 진행합니다. 그래서 당신의 사업을 압니다." |
| 크몽/숨고 저가 외주 | "5만원짜리 10개보다 제대로 된 전략 1개가 매출을 바꿉니다." |
| 마케팅 강의만 | "배운 건 많은데 실행할 시간이 없다면, 결과물을 드립니다." |
| 코칭만 | "숙제를 내드리는 게 아니라, 결과물을 드립니다." |
| 마케터 채용 실패 | "채용 6개월 + 온보딩 3개월. brandrise는 2주 안에 시작합니다." |

## 6. 메타 광고 소재 가이드 (이 문서 기반)

### 훅 카피 공식

```
[Pain 직격형]  "마케팅 예산을 태워도 성과가 안 나는 스타트업 대표님"
[숫자 증거형]  "3개월 만에 ROAS 340%를 만든 방법"
[공포 + 숫자]  "시리즈A 스타트업 10곳 중 7곳이 하는 마케팅 실수"
[상황 공감형]  "마케팅 전담 인력 없이도 가능합니다"
[질문 유도형]  "좋은 제품인데 왜 안 팔리는지 모르겠다면"
```

### 소재 유형 우선순위

1. **Before/After 카드뉴스** — 성과 대비, 저장율 최고
2. **UGC 스타일 릴스** — 광고 같지 않은 자연스러움
3. **데이터 증거형** — 구체적 숫자가 클릭률 2-3배
4. **체크리스트/가이드형** — 저장 유도, 리드 확보

### A/B 테스트 우선순위

1. Before/After 이미지 vs UGC 스타일 영상
2. 숫자 중심 카피 vs 스토리 중심 카피
3. 대표 얼굴 등장 vs 그래픽 중심
4. 15초 릴스 vs 30초 릴스
