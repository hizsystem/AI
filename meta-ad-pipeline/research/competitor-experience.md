# 초기 스타트업 대표 -- 경쟁사 이용 경험 리서치

> 조사일: 2026-02-27
> 목적: brandrise의 타겟(시드~시리즈A 스타트업 대표)이 기존에 어떤 마케팅/브랜딩 서비스를 써봤고, 무엇이 만족/불만족이었는지 파악하여 차별화 전략 수립

## 1. 경쟁사 카테고리별 분석

### 1-1. 코칭형: 김용훈 그로스연구소 (levigrowth.com)

**개요**
- 15년차 CMO 출신, 굿닥(헬스케어 O2O) 코스닥 상장 경험
- 그로스해킹/그로스 마케팅 기반 스타트업 컨설팅
- 주간 미팅 + 숙제 기반 코칭 모델

**가격대**
- 월 66만원 내외 (1:1 컨설팅 기준)
- 교육 프로그램: SEO 최적화, 데이터 마케팅, 메타 광고 효율 최적화 등 개별 클래스 운영
- 정부지원사업 연계 컨설팅도 진행

**핵심 가치 제안**
- "데이터 기반 그로스해킹으로 스타트업 성장 가속"
- 여러 회사와 동시 계약하여 주 1회 방문 컨설팅
- 실제 CMO 경험 기반의 실전 노하우 전달

**사용자 만족 포인트**
- 실전 경험에서 나온 구체적 액션 아이템 제공
- 마케팅 "사고방식" 자체를 바꿔주는 효과
- 데이터를 보는 시각, 실험 설계 방법론 학습 가능
- 대표가 직접 마케팅 역량을 키울 수 있음

**사용자 불만 포인트**
- 주 1회 미팅만으로는 실행 속도가 느림 -- "숙제를 줘도 바쁜 대표가 실행할 시간이 없다"
- 코칭은 결국 '대표가 직접 해야 하는' 모델 -- 실행 리소스 부족한 초기 스타트업에 한계
- 여러 회사를 동시에 코칭하므로 우리 사업에만 집중하지 않음
- 월 66만원이 "배우기만 하는 데 쓰기엔 부담"이라는 시각 존재

### 1-2. 에이전시형: 미더(meader.co.kr), 오픈피알(openpr.kr), 원포인트(1point.kr)

#### 미더 (Meader)

**개요**
- 2015년 창업, 브랜딩 + 마케팅 통합 에이전시
- 비영리단체, 사회적기업, 스타트업 대상 홍보 전문
- "더 좋은 이야기가 더 멀리 전해지도록" 슬로건

**가격대**
- 러닝 개런티(running guarantee) 계약 -- 월정액 지속 계약 구조
- 일회성 콘텐츠 제작부터 채널 운영대행까지 범위에 따라 가격 상이
- 스타트업 대상 맞춤 패키지 제공 (상담 후 견적)

**핵심 가치 제안**
- 디테일한 분석 기반 맞춤 브랜딩 + 전략 마케팅
- 지속가능한 성장 지원 (러닝 개런티)
- 사업 초창기 마케팅 전략 수립 + 브랜드 아이덴티티 확립

**사용자 만족 포인트**
- 스타트업/소셜벤처 업종 이해도가 높음
- 브랜딩과 마케팅을 한 곳에서 해결 가능

**사용자 불만 포인트**
- 소규모 에이전시 특성상 동시에 여러 클라이언트 진행 시 리소스 분산
- 비영리/사회적기업 중심이라 B2B SaaS 등 테크 스타트업과의 핏이 약할 수 있음

#### 오픈피알 (OpenPR)

**개요**
- 2007년부터 운영, IT기업/스타트업 전문 PR 에이전시
- IT/마케팅 전문지 기자 출신 "애디터(Additor)"들이 PReview(PR+Review) 진행
- 클라우드, AI, HR테크, 이커머스 등 기술 분야 전문성

**가격대**
- 월정액 PR 리테이너 계약 (월 200~500만원 추정)
- 건별 보도자료/리뷰 콘텐츠 제작 가능

**핵심 가치 제안**
- 기자 출신이 쓰는 PR 콘텐츠 -- 언론 노출 효과
- IT/SaaS 업종 전문성
- 폭넓은 미디어 네트워크

**사용자 만족 포인트**
- 기술 제품을 쉽고 친절하게 설명하는 콘텐츠 품질
- 업종 이해도가 높아 커뮤니케이션 비용 낮음
- 언론 보도 성과가 가시적

**사용자 불만 포인트**
- PR 중심이라 퍼포먼스 마케팅(광고 집행, 전환 최적화)은 별도
- 스타트업에게 월 200만원 이상의 PR 비용은 부담
- PR 성과의 직접적 매출 기여 측정이 어려움

#### 원포인트 (One Point)

**개요**
- 탑티어 마케터 파트타임 매칭 서비스
- 업계 상위 10% 전문가를 3단계 검증 후 선별
- "풀타임 정규직처럼 협업, 필요한 시간만큼만 고용"

**가격대**
- 시간제/파트타임 고용 -- 풀타임 대비 비용 절감
- 마케터 등급과 시간에 따라 가격 책정 (상담 후 견적)

**핵심 가치 제안**
- 풀타임 채용의 부담 없이 시니어급 마케터 확보
- 이직 부담이 없어 우수 인재 확보 가능
- 에이전시의 "수수료 기반" 구조적 한계 해결

**사용자 만족 포인트**
- 내부 팀원처럼 밀착 협업 가능
- 검증된 시니어 마케터 매칭
- 고용 리스크 없이 전문 인력 활용

**사용자 불만 포인트**
- 결국 한 명의 프리랜서에 의존하게 되는 구조
- 매칭된 마케터와의 핏이 안 맞을 경우 교체 비용/시간 발생
- 파트타임 특성상 풀타임만큼의 몰입도 확보 어려움

### 1-3. 플랫폼형: 크몽, 숨고, 라우드소싱

#### 크몽 (Kmong)

**개요**
- 국내 최대 프리랜서 마켓, 마케팅 서비스만 91,000개 이상
- 최저 5,000원부터 시작하는 SNS/퍼포먼스/언론홍보/SEO 서비스
- "불만족 시 100% 환불" 정책 운영

**가격대**
- 마케팅 컨설팅: 5만~100만원
- 블로그 마케팅: 1만~50만원
- SNS 관리: 10만~100만원/월
- 종합 광고대행: 30만~300만원/월
- 브랜딩 패키지: 440만원~ (프리미엄)

**핵심 가치 제안**
- 다양한 가격대, 포트폴리오 비교 후 선택
- 에스크로 결제로 안전 거래
- 리뷰 시스템으로 품질 검증

**사용자 만족 포인트**
- 저렴한 가격으로 빠르게 시작 가능
- 리뷰/평점으로 전문가 선별 가능
- 소규모 단건 의뢰에 적합

**사용자 불만 포인트** (실제 리뷰/뉴스 기반)
- **품질 편차 극심**: "저렴한 가격에 맞춰 찍어내기식 결과물" -- 낮은 마진 보전을 위해 자재비/인력 최소화
- **소비자 피해 급증**: 한국소비자원 집계, 크몽/숨고/탈잉 피해구제 신청 3년간 498건 (2022년 93건 -> 2024년 249건, 2.5배 급증)
- **계약 불이행 34.7%**: 가장 많은 피해 유형, 연락 두절, 직접 결제 유도 등
- **피해 구제율 37.8%**: 플랫폼이 적극적으로 분쟁 해결에 나서지 않음
- **"몇 푼 아끼려다 수천만원 날려"** (한국경제 보도): 시세보다 30% 저렴한 견적에 끌렸다가 중도 중단/연락 두절 사례
- **커뮤니케이션 어려움**: 무형 서비스 특성상 요구사항 전달 오류 빈번, "심플하면서도 화려한" 같은 주관적 요청으로 인한 결과물 불일치

#### 숨고 (Soomgo)

**개요**
- 생활 서비스 중심 전문가 매칭 플랫폼
- 견적서 발송 방식 (건당 3,000~4,000원 수수료)
- 마케팅뿐 아니라 인테리어, 과외 등 범용 서비스

**가격대**
- 브랜딩/네이밍 컨설팅 평균: 약 25만원
- 마케팅 컨설팅: 상담 후 견적
- 크몽 대비 기본 견적 단가가 높은 편

**사용자 만족 포인트**
- 여러 전문가의 견적을 한 번에 비교 가능
- 원하는 조건에 맞는 전문가 필터링

**사용자 불만 포인트**
- 크몽과 동일한 품질 편차 문제
- 불공정 약관으로 공정위에 적발 -- 피해 책임 회피
- "일 맡겼더니 나 몰라라" (언론 보도)
- 마케팅 전문성보다는 범용 서비스 중심이라 깊이 부족

#### 라우드소싱 (Loud Sourcing)

**개요**
- 국내 최대 디자인 의뢰 플랫폼 (23만명 디자이너)
- 콘테스트 방식(공모전형) + 디자이너 매칭 + 1:1 의뢰
- 주식회사 스터닝 운영

**가격대**
- 로고 디자인: 30만~200만원
- 브랜딩 패키지: 100만~500만원
- 콘테스트: 최소 30만원부터

**핵심 가치 제안**
- 한 번의 의뢰로 20~30개 시안 확보 (콘테스트)
- 98.7% 만족도 (자체 발표)
- 앱 평점 4.3/5 (706명 평가, 2024년 기준)

**사용자 만족 포인트**
- "여러 디자인을 볼 수 있어 고르기 어려울 정도" (실제 후기)
- 기존 에이전시 대비 합리적 가격에 다양한 시안
- 수정 가능성이 높은 시스템

**사용자 불만 포인트**
- 디자인 중심이라 마케팅 전략/실행까지는 커버하지 못함
- 콘테스트 방식에서 선정되지 않은 디자이너의 노력 낭비 논란
- 로고/CI만으로는 브랜딩이 완성되지 않음 -- 전략 부재

### 1-4. SaaS형: 채널톡, 스티비

#### 채널톡 (Channel.io)

**개요**
- 올인원 고객 소통 플랫폼 (채팅, CRM, 마케팅 자동화)
- 최대 98% 고객유지율 기록
- B2B SaaS/이커머스 스타트업 주요 사용

**가격대**
- 무료 플랜: 기본 채팅
- 유료 플랜: 월 3만~20만원 (기능/MAU에 따라)
- 엔터프라이즈: 별도 상담

**사용자 만족 포인트**
- 도입 후 고객 CS 만족도 즉시 상승
- 마케팅 자동화(타겟 팝업, 이벤트 트리거) 기능
- "친절하고 전문적인 CS"라는 고객 평가
- 개발자 없이도 세팅 가능

**사용자 불만 포인트**
- 채팅/CRM 도구이지 마케팅 전략을 세워주지는 않음
- MAU 기반 과금이라 성장할수록 비용 급증
- 마케팅 "실행"보다는 "인프라" 성격

#### 스티비 (Stibee)

**개요**
- 이메일 마케팅/뉴스레터 제작 및 발송 서비스
- 디자이너/개발자 없이 이메일 제작 가능
- A/B 테스트, 자동화, 통계 분석 기능

**가격대**
- 무료 플랜: 구독자 500명까지
- 스탠다드: 월 1만~10만원 (구독자 수에 따라)
- 프로: 월 3만~30만원

**사용자 만족 포인트**
- 직관적 에디터로 빠른 뉴스레터 제작
- 발송 통계 데이터 확인 용이
- 한국어 인터페이스, 국내 결제 지원

**사용자 불만 포인트**
- 이메일 마케팅 한 채널에 특화 -- 통합 마케팅 도구가 아님
- 구독자 기반 구축 자체가 어려움 (도구는 있지만 독자 확보 전략은 별도)
- 초기 스타트업은 이메일 마케팅보다 급한 채널이 많음

### 1-5. 교육형: 패스트캠퍼스, 클래스101

#### 패스트캠퍼스 (Fast Campus)

**개요**
- 직무 교육 1위 플랫폼
- 디지털 마케팅 초격차 패키지 등 다수 강좌
- 실무자 대상 체계적 커리큘럼

**가격대**
- 단일 강좌: 10만~30만원
- 패키지(시그니처): 30만~80만원
- 올인원 구독: 월 4만원~

**사용자 만족 포인트**
- 전문 강사의 핵심 정리
- 출퇴근 시간 활용 가능
- 체계적 커리큘럼
- 팀 동료와 함께 수강 가능

**사용자 불만 포인트**
- **"배운 것을 실행할 시간/역량 부족"**: 스타트업 대표가 강의를 듣고 직접 실행하기엔 현실적 한계
- 마케팅 "이론"은 알게 되지만 "우리 사업에 맞는 전략"으로 전환이 어려움
- 일반론 위주라 업종별 특수성 반영 부족
- 강의 퀄리티 편차 (강사마다 다름)

#### 클래스101

**개요**
- 취미/전문 교육 플랫폼
- 마케팅 분야 다수 강좌 보유
- 2025년 첫 연간 흑자 달성 (영업이익 39억원)

**가격대**
- 단일 강좌: 5만~20만원
- 구독 모델: 월 1.5만원~

**사용자 만족 포인트**
- 접근성 높은 가격
- 다양한 마케팅 주제 (SNS, 브랜딩, 콘텐츠 등)
- 전문가의 실전 노하우 공유

**사용자 불만 포인트**
- 취미 플랫폼 이미지가 강해 "비즈니스 마케팅" 깊이 부족
- 수강 후 실행 전환율이 낮음
- 1:1 피드백이나 맞춤 컨설팅이 없음

## 2. 불만 패턴 TOP 5

### 1위: "우리 사업을 이해하지 못한다" (업종 이해도 부족)

가장 보편적이고 깊은 불만. 에이전시, 프리랜서 할 것 없이 공통.

> "마케팅 대행사는 공동창업자가 아니기 때문에 사업의 전략과 방향성은 사장님이 지휘해야 한다." -- 토스페이먼츠

> "초기 스타트업이 마케팅 대행사와 함께하지만 만족하는 경우가 드물다. 대행사의 주인 의식 수준은 예산에 비례할 수밖에 없다." -- 원포인트

- 대행사는 동시에 수십 개 클라이언트를 관리하므로, 개별 사업의 PMF(Product-Market Fit), 고객 페르소나, 기술적 특성을 깊이 이해할 여력이 없음
- 결과적으로 범용적인/템플릿화된 마케팅 시안만 받게 됨
- B2B SaaS, 딥테크, 바이오 등 전문 업종일수록 이 불만이 극대화

### 2위: "돈은 나가는데 성과가 안 보인다" (ROI 불투명)

> "모든 마케팅을 외부 대행사에 맡겨버리면, 정작 내부에는 어떤 데이터도 남지 않고, 결국 다음 캠페인 때도 외주에 의존해야 하므로 비용 구조는 점점 무거워진다." -- Eagler Lab

- 월 100~500만원을 대행사에 쓰면서도 MAU/전환율/매출 기여가 불명확
- 대행사 보고서는 "노출수, 클릭수" 중심이지 비즈니스 임팩트(매출, LTV, CAC)를 말해주지 않음
- 특히 브랜딩/PR 분야는 성과 측정 자체가 어려워 "돈 낭비" 인식 발생

### 3위: "품질이 들쭉날쭉하다" (결과물 품질 편차)

> 한국소비자원 집계: 전문가 매칭 플랫폼 피해구제 신청 3년간 498건 (2022년 93건 -> 2024년 249건, 2.5배 급증). 피해 유형 1위는 계약 불이행/불완전 이행(34.7%)

> "몇 푼 아끼려다 수천만원 날려" -- 한국경제 보도

- 크몽/숨고 등 플랫폼에서 저가 견적에 끌렸다가 중도 중단, 연락 두절, 품질 미달 경험
- 피해 구제율 37.8%에 불과 -- 플랫폼이 책임 회피
- 에이전시도 담당자(AE)의 역량에 따라 결과물 천차만별

### 4위: "소통이 안 된다" (커뮤니케이션 비용)

- 무형 서비스 특성상 요구사항 전달 오류 빈번
- 피드백 반영까지의 리드타임이 길어 프로젝트 지연
- 에이전시는 수정 횟수 제한, 추가 수정 시 별도 비용 청구
- 대표의 비전/톤앤매너를 정확히 전달하는 것 자체가 노력

### 5위: "결국 다 직접 해야 한다" (자립 불가 구조)

> "사업 초기 방향성과 마케팅 노하우를 기본적으로 알고 계신 뒤에 마케팅 대행사를 찾는 것이 가장 좋다." -- 토스페이먼츠

- 코칭/교육을 받아도 실행할 시간/인력 부족
- 대행사에 맡겨도 전략적 방향은 대표가 설정해야 함
- SaaS 도구를 도입해도 운영할 마케터가 없음
- "마케팅을 할 줄 아는 사람이 없는 게 근본 문제"라는 인식

## 3. 만족 패턴 TOP 5

### 1위: "내 사업을 진짜 이해하고 함께 고민해줬다" (파트너십 경험)

- 담당자가 우리 업종, 고객, 경쟁사를 직접 공부해온 경우
- 단순 "시안 납품"이 아니라 비즈니스 맥락에서의 제안
- 마치 "CMO가 내부에 있는 것 같은" 경험

### 2위: "바로 쓸 수 있는 결과물을 줬다" (실행 가능성)

- 전략 문서가 아니라 바로 게시할 수 있는 콘텐츠
- 광고 세팅부터 크리에이티브까지 원스톱
- "교과서적 전략서"가 아닌 "내일 당장 실행할 수 있는 투두리스트"

### 3위: "숫자로 성과를 보여줬다" (데이터 기반 성과 증명)

- 투자 대비 매출 1000% 발생 같은 구체적 ROI 제시
- 주간/월간 리포트에서 비용 -> 전환 -> 매출 흐름을 투명하게 공유
- A/B 테스트 결과를 바탕으로 점진적 개선 제안

### 4위: "합리적 가격에 전문적 퀄리티를 얻었다" (가성비)

- 라우드소싱 콘테스트로 로고 20~30개 시안을 30만원에 확보
- 원포인트로 시니어 마케터를 파트타임으로 활용하여 풀타임 대비 비용 절감
- 채널톡/스티비 등 SaaS로 월 수만원에 마케팅 인프라 구축

### 5위: "내부 역량이 함께 성장했다" (학습 효과)

- 코칭형 서비스에서 마케팅 사고방식 자체를 배움
- 대행사가 내부 팀 교육까지 해준 경우
- 결과물뿐 아니라 프로세스와 방법론을 이전받은 경험

## 4. 가격 민감도

### 스타트업 마케팅 예산 현실

| 단계 | 월 마케팅 예산 | 구성 |
|------|-------------|------|
| 프리시드/시드 | 0~100만원 | 대부분 대표가 직접, 또는 무료 도구만 |
| 시드 (투자 후) | 100~500만원 | 콘텐츠 마케팅 중심, 소규모 광고 테스트 |
| 시리즈A | 500~2,000만원 | 퍼포먼스 마케팅 확대, 채널 다각화 |

### "비싸다"고 느끼는 기준선

| 서비스 유형 | 가격 구간 | 스타트업 대표 인식 |
|------------|----------|----------------|
| 코칭/컨설팅 | 월 30만원 이하 | "한번 들어볼 만" |
| | 월 30~70만원 | "효과가 확실하면 OK" (김용훈 그로스연구소 구간) |
| | 월 100만원 이상 | "배우기만 하는 데 이 돈은 부담" |
| 에이전시/대행 | 월 100만원 이하 | "이 가격에 좋은 에이전시를 만나기 어려움" |
| | 월 100~300만원 | "효과가 보이면 적정, 안 보이면 돈 낭비" |
| | 월 500만원 이상 | 시드 스타트업에겐 "감당 불가" |
| 플랫폼 (크몽/숨고) | 건당 5~50만원 | "싸서 시도하기 쉬움, 하지만 품질 불안" |
| | 건당 100만원 이상 | "이 가격이면 차라리 에이전시를" |
| SaaS 도구 | 월 0~3만원 | "무조건 도입" |
| | 월 3~10만원 | "ROI 확인 후 유지 결정" |
| | 월 20만원 이상 | "아직 이른 것 같다" |
| 브랜딩 패키지 | 100~500만원 | 스타트업이 시도하는 구간 (프리랜서/소규모) |
| | 500~3,000만원 | "투자금 있으면 고려" (중견 에이전시) |
| | 3,000만원 이상 | 시리즈A+ 이후에나 가능 |

### 가격 대비 가치를 느끼는 구간

- **월 100~300만원**에서 "내부 마케터 1명 뽑은 효과"를 체감할 때
- **건당 30~100만원**에서 바로 쓸 수 있는 고퀄리티 콘텐츠/디자인을 받았을 때
- **월 50~100만원** 코칭에서 "매출이 실제로 올랐을 때"
- 공통: **"내가 직접 할 때 드는 시간 비용"과 비교**하여 절약되는 시간이 명확할 때 가치를 인정

## 5. brandrise 차별화 기회

### 경쟁사 공통 약점 5가지

| 약점 | 설명 |
|------|------|
| 1. 사업 이해도 부족 | 대행사는 클라이언트 사업을 깊이 모른 채 범용 마케팅 실행 |
| 2. 전략과 실행의 분리 | 코칭형은 전략만, 대행형은 실행만 -- 연결이 끊김 |
| 3. ROI 불투명 | "노출수"만 보고하고, 비즈니스 임팩트(매출/고객 확보)를 증명하지 않음 |
| 4. 가격-품질 양극화 | 싸면 품질 낮고, 좋으면 스타트업이 감당 불가 |
| 5. 지식 이전 없음 | 외주 끝나면 내부에 남는 게 없음 -- 영구적 의존 구조 |

### brandrise가 가져갈 수 있는 포지션

**"스타트업 전담 CMO 팀" 포지션**

기존 경쟁사가 채우지 못하는 빈틈:
- 코칭형(김용훈) = 전략은 주지만 실행은 대표 몫
- 에이전시형(미더, 오픈피알) = 실행은 하지만 사업 이해 부족
- 플랫폼형(크몽, 숨고) = 저렴하지만 품질 리스크
- SaaS형(채널톡, 스티비) = 도구만 있고 전략/콘텐츠 없음
- 교육형(패캠, 클래스101) = 배우지만 실행 불가

**brandrise의 차별화 공식:**

```
전략(코칭형의 장점) + 실행(에이전시형의 장점) + 사업 이해(인하우스의 장점)
= 스타트업 전담 CMO 팀
```

### 전환 메시지 -- 경쟁사 경험자 타겟

| 이전 경험 | 전환 메시지 |
|----------|-----------|
| 에이전시 써봤는데 성과 없었던 대표 | "대행사는 여러분의 사업을 이해할 시간이 없습니다. brandrise는 동시 3개 프로젝트만 진행합니다." |
| 크몽/숨고로 저가 외주 맡겨봤던 대표 | "5만원짜리 콘텐츠 10개보다, 제대로 된 전략 1개가 매출을 바꿉니다." |
| 마케팅 강의만 들었던 대표 | "배운 건 많은데 실행할 시간이 없다면, brandrise가 대신 실행하고 결과로 보여드립니다." |
| 코칭 받았지만 실행 못한 대표 | "전략 + 실행 + 리포팅. 숙제를 내드리는 게 아니라, 결과물을 드립니다." |
| PR만 해봤던 대표 | "보도자료만으로는 고객이 오지 않습니다. 유입부터 전환까지 전체 퍼널을 설계합니다." |
| 마케터 채용에 실패한 대표 | "시니어 마케터 채용에 6개월, 온보딩에 3개월. brandrise는 2주 안에 실행을 시작합니다." |

### 핵심 차별화 메시지 (1줄)

> **"마케팅 외주가 아닙니다. 당신의 스타트업에만 집중하는 CMO 팀입니다."**

### 구체적 차별화 포인트

1. **동시 프로젝트 3개 제한** -- 사업 이해도를 구조적으로 보장
2. **전략 + 실행 일체형** -- 코칭과 대행의 장점만 결합
3. **AI 에이전트 활용** -- 1인 운영이지만 6명 전문가 팀의 산출물 품질
4. **ROI 중심 리포팅** -- 노출수가 아니라 매출/고객 확보 기여로 성과 증명
5. **지식 이전 포함** -- 프로젝트 종료 후에도 내부 역량으로 남는 구조
6. **스타트업 눈높이 가격** -- 월 100~300만원대에서 CMO급 서비스 제공
