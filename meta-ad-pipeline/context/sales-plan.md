**왜 돈을 쓸어담는가 (핵심 분석):**

1. **"숙제 모델"의 위력**: 교육이 아닌 **강제 실행 장치**. 대표가 직접 안 해도 되는 대행과 달리, 대표가 직접 하게 만들되 옆에서 코칭. 결과적으로 대표의 마케팅 역량이 실제로 올라감 → "돈 안 쓸 이유가 없는" 가치를 체감
2. **주간 리듬**: 월 1회 미팅은 흐지부지되지만, 주간 미팅은 긴장감 유지 + 빠른 피드백 루프
3. **낮은 진입 → 높은 전환**: 1,000원 라이브로 맛보기 → 교육 과정 → 컨설팅으로 자연스러운 업셀
4. **개인 브랜드 파워**: 김용훈 개인의 이력이 곧 신뢰. "이 사람한테 배우면 된다"

**데이터라이즈(datarise)가 가져갈 것:**
- 주간 미팅 + 숙제 모델 (Section 3에서 반영)
- 저가 진입점 퍼널 (1,000원~무료 → 유료)
- "대표가 직접 성장하는" 메시지 (대행만 하는 곳과 차별화)

**데이터라이즈(datarise)의 차별화 (vs 김용훈):**
- 김용훈 = **코칭** (대표가 직접 하게 만듦) → 데이터라이즈(datarise) = **코칭 + 실행** (안 되면 대신 해줌)
- 김용훈 = 그로스/퍼포먼스 중심 → 데이터라이즈(datarise) = **브랜딩 + SNS + 퍼포먼스 통합**
- 김용훈 = 일반 스타트업 → 데이터라이즈(datarise) = **청창사/신창사 특화** (정부 자금 활용법 포함)

#### 간접 경쟁 (교육/커뮤니티 플레이어)

| 경쟁사 | 모델 | 강점 | 데이터라이즈(datarise)와의 차이 |
|--------|-----|------|---------------------|
| 언더독스 | 콘텐츠+커뮤니티+교육 | 창업 생태계 영향력, 대규모 커뮤니티 | 교육만 제공, 실행 대행 안 함 |
| EO | 미디어+콘텐츠 | 유튜브 구독자 다수, 콘텐츠 퀄리티 | 미디어 플랫폼이지 대행사 아님 |
| 오픈놀 | 캠프+멘토링 | 프로그램 기획력, 정부사업 경험 | 마케팅 특화 아님 |
| 마케팅학교/캠퍼스 | 교육 콘텐츠 | 인스타 팔로워, 교육 콘텐츠 | 이론 중심, 실행 안 해줌 |

#### 핵심 기회 (Gap)

> **교육만 하는 곳은 많지만, "배운 걸 대신 실행까지 해주는 곳"은 거의 없다.**
> **대형 에이전시는 스타트업을 안 받고, 소형은 전략이 없다.**
> **김용훈처럼 "주간 케어"하는 곳은 있지만, 실행 대행까지 해주진 않는다.**
> 데이터라이즈(datarise)는 이 빈틈을 파고든다: **주간 코칭 + 전략 + 실행 대행의 올인원 솔루션**

## 3. 서비스 패키지 & 가격 전략

> **핵심 벤치마크**: 김용훈 그로스 연구소 월 66만 원 주간 케어 모델
> **전략**: "주간 미팅 + 숙제" 모델을 도입하되, 데이터라이즈(datarise)만의 "실행까지 해주는" 차별화를 더함

### 3.1 서비스 철학: "코칭 + 실행" 하이브리드

```
김용훈 모델: 코칭만 (대표가 직접 하게 만듦) → 월 66만
대행사 모델: 실행만 (대표는 모름) → 월 300만+
─────────────────────────────────────────────
데이터라이즈(datarise): 코칭 + 실행 (대표도 배우고, 안 되는 건 해줌) → 월 66~132만
```

**왜 하이브리드인가?**
- 코칭만: 대표가 시간이 없으면 숙제를 안 함 → 효과 반감
- 실행만: 대표가 마케팅을 이해 못 함 → 에이전시 의존성 높아짐
- **코칭+실행**: 대표는 방향을 잡고, 실무는 우리가 처리 → 가장 높은 만족도

### 3.2 3-Tier 패키지 설계 (v2: 주간 케어 모델)

#### Tier 1: 코칭 (월 66만 원) ← 김용훈과 동일 가격대

- **대상**: 마케팅 아직 아무것도 안 한 초기 스타트업
- **핵심 컨셉**: "매주 만나서 마케팅 체력을 키워드리겠습니다"
- **포함 서비스**:
  - **주간 미팅** (60분, 온/오프 선택) — 지난주 숙제 리뷰 + 이번주 과제 부여
  - 마케팅 진단 리포트 (첫 달)
  - 브랜드 포지셔닝 워크숍 (첫 달 1회)
  - 인스타그램/블로그 콘텐츠 기획안 (월 8개 — 대표가 직접 제작)
  - 카카오 채널로 수시 질의응답
- **미포함**: 콘텐츠 제작, 광고 집행 (대표가 직접)
- **김용훈 대비**: 동일 가격, 동일 주간 케어 + **청창사 특화 진단 + 정부 자금 활용 가이드**

> "월 66만 원에 매주 만나서 케어해주는 마케팅 파트너. 김용훈 연구소에서 검증된 모델입니다."

#### Tier 2: 코칭+실행 (월 132만 원) ← 메인 상품

- **대상**: 제품/서비스 출시 완료, 시간이 없는 대표
- **핵심 컨셉**: "전략은 같이, 실행은 저희가"
- **포함 서비스**:
  - Tier 1 전체 (주간 미팅 포함) +
  - 인스타그램/블로그 콘텐츠 **제작** (월 12개)
  - Meta 광고 세팅 + 운영 (광고비 별도)
  - 경쟁사 모니터링 리포트 (월 1회)
  - 월간 성과 대시보드
- **김용훈 대비**: **"숙제를 안 해도 되는 버전"** — 코칭은 동일하되, 못하는 부분은 대신 실행
- **목적**: 가장 많이 팔릴 상품. 가격 대비 압도적 가치

> 132만 원 = 주간 케어(66만) + 콘텐츠 실행(66만). "코칭만 받을 때보다 2배 빠른 성장"

#### Tier 3: 풀서비스 (월 220만 원)

- **대상**: 올리브영 입점, 쿠팡 런칭 등 채널 확장 단계
- **핵심 컨셉**: "마케팅 팀장을 월 220만에 고용하는 효과"
- **포함 서비스**:
  - Tier 2 전체 +
  - 올리브영/쿠팡 입점 마케팅 전략 + 실행
  - 인플루언서 마케팅 (월 3-5건)
  - 카카오/네이버 광고 추가 운영
  - 주간 성과 리포트 (미팅 시 리뷰)
- **목적**: 고수익 상품. "마케팅 담당자 1명 월급보다 싸다"

### 3.3 가격 전략 근거

| 요소 | 시장 기준 | 데이터라이즈(datarise) | 포지셔닝 |
|------|----------|------------|---------|
| 김용훈 주간 코칭 | **월 66만 원** | Tier 1: 66만 (동일) | "같은 가격, 더 많은 케어" |
| SNS 광고대행 평균 | 월 35~100만 원 | Tier 2에 포함 | 코칭+실행 통합 |
| 대행사 월 리테이너 | 300~500만 원+ | Tier 3: 220만 | "절반 가격, 대표 직접 소통" |
| 마케팅 담당자 월급 | 300~400만 원 | Tier 3: 220만 | "사람 뽑는 것보다 싸다" |

> **핵심 원칙: "일단 클라이언트를 모은다. 마진보다 레퍼런스가 먼저."**

### 3.4 주간 케어 모델 상세 설계

> 김용훈 모델의 핵심을 차용하되, 데이터라이즈(datarise) 버전으로 진화

#### 주간 미팅 프로토콜 (60분)

```
[0-10분] 지난 주 숙제 리뷰
  - 인스타 포스팅 했는지? 반응은?
  - 블로그 글 올렸는지? 유입 수는?
  - 광고 성과 데이터 함께 확인

[10-30분] 이번 주 핵심 이슈 논의
  - 경쟁사 동향 공유
  - 새로운 기회/문제 식별
  - 전략 방향 조정

[30-50분] 이번 주 과제 부여 & 실행 계획
  - 구체적인 to-do 리스트 (3-5개)
  - 각 과제의 기대 결과
  - Tier 2의 경우: 대행 항목 확인

[50-60분] Q&A + 다음 주 예고
  - 자유 질문
  - 다음 주 미팅 일정 확인
```

#### 숙제 예시 (4주 사이클)

| 주차 | 코칭 Tier 숙제 (대표 직접) | 코칭+실행 Tier (데이터라이즈(datarise)가 대행) |
|------|--------------------------|----------------------------------|
| 1주 | 경쟁사 3곳 인스타 분석해오기 | 경쟁사 분석 리포트 작성해드림 |
| 2주 | 이번 주 인스타 3개 직접 올리기 | 12개 콘텐츠 제작 + 대표는 1개만 직접 |
| 3주 | 고객 5명에게 후기 요청하기 | 후기 요청 메시지 작성 + 대표가 전송 |
| 4주 | 블로그 1개 직접 써보기 | 블로그 4개 작성 + 대표는 검수만 |

> **핵심**: Tier 1(코칭)은 "대표가 직접 성장", Tier 2(코칭+실행)는 "성장하면서 결과도 나옴".
> 대표가 Tier 1을 하다가 "시간이 없어요"하면 자연스럽게 Tier 2로 업셀.

```

  ↓
[관심] 무료 웨비나/가이드 → 이메일/카톡 수집
```

### 4.4 Phase 3: 고려 → 전환 (Week 4-8)

#### A. 무료 마케팅 진단 (핵심 전환 장치)

**진행 방식** (30분 화상/대면):
1. 현재 마케팅 현황 체크 (5분)
2. 경쟁사 대비 포지셔닝 분석 (10분)

4. 데이터라이즈(datarise) 서비스 소개 (5분)

> **핵심**: 진단 자체가 가치 있어야 한다. "이 30분 상담만으로도 방향이 잡혔다"는 경험을 주면, 유료 전환은 자연스럽다.

#### B. 제안서 전략

- /proposal 커맨드로 자동 생성
- 포함 요소: 현황 분석 + 전략 방향 + 예상 ROI + 가격 + 타임라인
- 가격 제시: 3-Tier 중 권장 상품 강조 + "첫 달 30% 할인" 프로모션

### 4.5 Phase 4: 충성 → 레퍼럴 (Week 8+)

#### A. 클라이언트 성공 시스템

|----------|------|

**가장 ROI 높은 채널: 프로그램 내부 강의**

1. **외부 강사 지원**: 청창사/신창사의 마케팅 교육 외부 강사로 지원
   - 주제: "스타트업 마케팅 실전 가이드"
   - 대상: 현재 재학 중인 입교생

   - 기존 클라이언트를 통한 소개로 카페/단톡방 입성
   - 정기적으로 유용한 마케팅 정보 공유 (광고 X, 가치 제공 O)

3. **KOSMES/중진공 연계**: 후속지원 프로그램의 마케팅 파트너로 등록
   - 졸업기업 대상 마케팅/판로지원 연계사업에 참여

**차별화할 것:**
- 일반적인 마케팅 팁이 아닌 **"청창사 대표 전용"** 콘텐츠

### "폭발적 유기 수요"를 위한 핵심 공식

```
[킬러 콘텐츠] × [타겟 커뮤니티 침투] × [무료 가치 제공] × [사회적 증거]
= 자발적 수요 폭발
```

- **킬러 콘텐츠**: 청창사 대표만을 위한 구체적이고 실용적인 마케팅 정보

- **무료 가치 제공**: 웨비나, PDF 가이드, 무료 진단 (받는 사람이 "이걸 무료로?" 느끼게)
