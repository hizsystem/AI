## 릴스 1: 세그먼트 3 공감형 (15초)

**타겟**: 매출 10-30억 기업 대표
**톤**: 공감 + 위기감
**목적**: 인지도 (스크롤 멈춤 → 프로필 방문)

### 스크립트

```
[0-3초] 훅 (텍스트 오버레이 + 나레이션)
화면: 다크 배경 + 큰 텍스트 타이핑 효과
나레이션: "마케팅팀이 있는데 전략이 없다고요?"
텍스트: "마케팅팀이 있는데\n전략이 없다고요?"

[3-8초] 공감 (빠른 텍스트 전환)
화면: 체크리스트가 하나씩 나타남
나레이션: "주니어 마케터가 실행만 돌리고,
         대표가 직접 방향을 정하고,
         CAC는 매분기 올라가고..."
텍스트:
  ✓ 실행만 도는 마케팅
  ✓ 대표가 직접 전략
  ✓ CAC 매분기 상승

[8-12초] 전환
화면: 배경 전환 (다크 → 오렌지 그라데이션)
나레이션: "정상입니다. 매출 10-30억 기업의 80%가 같은 상황이에요."
텍스트: "매출 10-30억 기업의\n80%가 같은 상황"

[12-15초] CTA
화면: brandrise 로고 + CTA
나레이션: "필요한 건 마케터가 아니라, 전략을 세우는 CMO 팀입니다."
텍스트: "무료 마케팅 진단 →\nbrandrise"
```

### 제작 노트
- BGM: 차분한 일렉트로닉 (긴장감 있는)
- 편집: 텍스트 타이핑 효과 → 체크리스트 팝업 → 배경 전환
- 자막: 전체 나레이션 자막 필수 (소리 없이 보는 유저 대비)
- 비율: 9:16 세로형

### 메타 광고 설정
- 목적: 인지도/도달
- 타겟: 30-50세, CEO/대표/마케팅리드, 관심사 스타트업/마케팅전략
- 예상 조회당 비용: 10-30원

## 릴스 2: 세그먼트 4 전문가형 (30초)

**타겟**: 매출 30-100억 기업 대표/마케팅 이사
**톤**: 전문성 + 데이터
**목적**: 인지도 + 리드 확보

### 스크립트

```
[0-3초] 훅 (강렬한 텍스트)
화면: 블랙 배경, 숫자가 크게
나레이션: "대행사 3곳에 월 1,600만원."
텍스트: "대행사 3곳\n월 1,600만원"

[3-7초] 문제 제기
화면: 대행사별 보고서 숫자가 나열됨
나레이션: "각 대행사의 숫자는 좋습니다.
         ROAS 320%, 클릭률 4.2%, 팔로워 +2천.
         그런데 매출은?"
텍스트:
  A대행사: ROAS 320% ✓
  B대행사: 클릭률 4.2% ✓
  C대행사: 팔로워 +2,000 ✓
  → 매출은? ❌

[7-12초] 원인 분석
화면: 구조도 애니메이션 (분리 → 파편화)
나레이션: "문제는 대행사가 아닙니다.
         각자의 채널만 최적화하고,
         전체 전략을 세우는 컨트롤타워가 없는
         구조가 문제입니다."
텍스트: "대행사가 아니라\n구조가 문제"

[12-18초] 솔루션
화면: 파편화 → 통합으로 전환 애니메이션
나레이션: "해답은 대행사를 바꾸는 게 아니라,
         마케팅의 구조를 바꾸는 겁니다.
         전략부터 실행까지, 한 팀으로."
텍스트:
  대행사 4곳 → 1팀 통합
  비용 30% 절감
  성과 2배

[18-24초] 증명
화면: Before/After 숫자 비교
나레이션: "통합 CMO 팀을 도입한 기업의 결과.
         마케팅 비용 동일, 매출 2.3배.
         CMO 1명 비용으로 CMO 팀을 쓰세요."
텍스트: "마케팅 비용 동일\n매출 2.3배"

[24-30초] CTA
화면: brandrise 로고 + 서비스 요약
나레이션: "30분 전략 상담으로 시작하세요.
         현재 마케팅 체계를 진단하고,
         통합 전략 로드맵을 제안합니다."
텍스트: "30분 전략 상담 예약\nbrandrise\n@brandrise_kr"
```

### 제작 노트
- BGM: 프로페셔널한 코퍼레이트 음악 (신뢰감)
- 편집: 모션 그래픽 중심, 숫자/그래프 애니메이션
- 자막: 전체 나레이션 자막 + 핵심 숫자 크게
- 비율: 9:16 세로형
- 나레이션: 남성, 30-40대, 차분하고 신뢰감 있는 톤

### 메타 광고 설정
- 목적: 리드 생성 (상담 예약)
- 타겟: 33-53세, CEO/CMO/마케팅이사, 관심사 경영전략/마케팅ROI
- 예상 조회당 비용: 20-50원
- 리타겟팅: 15초 이상 시청자 → 싱글 이미지 광고로 리타겟
