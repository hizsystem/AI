# 메타(Meta) 광고 셋업 가이드 — 인스타그램 콘텐츠 광고 편

> 인스타그램 콘텐츠를 메타 광고로 집행하기 위한 전체 셋업 프로세스
> 실제 brandrise 셋업 경험 기반 (2026-03-03~04)
> 대상: 인스타그램은 운영 중이나 메타 광고는 처음인 스타트업/소규모 브랜드

---

## 전체 소요 시간

| 구분 | 예상 시간 | 비고 |
|------|---------|------|
| 전체 셋업 | 1~2일 | 신규 계정 제한 대기 시간 포함 |
| 순수 작업 시간 | 30~60분 | 대기 없이 순수 클릭 작업만 |

---

## 사전 준비물

```
✅ 인스타그램 프로페셔널 계정 (비즈니스 또는 크리에이터)
✅ 이메일 주소 (비즈니스용 권장)
✅ 결제 수단 (신용카드/체크카드)
✅ 광고에 사용할 콘텐츠 (이미지/영상)
```

---

## Step 1: 페이스북 개인 계정 생성

**왜 필요한가?**
메타 생태계의 모든 비즈니스 도구는 페이스북 개인 계정을 기반으로 작동한다. 개인 계정은 외부에 노출되지 않는 "관리자 인증용"이다.

**방법:**
```
facebook.com 접속 > 새 계정 만들기
- 실명 입력 (메타 정책상 필수)
- 비즈니스 이메일로 가입 권장
```

**⚠️ 주의: 신규 계정 제한**
- 가입 직후 비즈니스 기능(페이지 생성, 비즈니스 계정 연결 등)이 **일시 차단**될 수 있다
- "페이지를 만드는 동안 오류가 발생했습니다" → 정상 현상
- **해결: 1시간~24시간 대기 후 재시도**
- 대기 중 프로필 사진 설정, 기본 정보 입력, 친구 추가 등을 해두면 계정 신뢰도가 올라가 제한이 빨리 풀린다

---

## Step 2: 페이스북 페이지 생성

**왜 필요한가?**
메타 광고는 반드시 페이스북 페이지를 "광고주 신원"으로 요구한다. 인스타그램만으로는 광고 집행 불가.

**방법:**
```
facebook.com > 메뉴(≡) > 페이지 > 새 페이지 만들기

- 페이지 이름: 브랜드명 (예: Brandrise)
- 카테고리: 비즈니스에 맞는 카테고리 선택
  (예: 컨설팅 대행사, 마케팅 에이전시, 소프트웨어 등)
- 소개: 명사형 중심의 간결한 한 줄
  (예: "AI 브랜드 성장 대시보드")
- 프로필 사진: 브랜드 로고 또는 심볼
```

**💡 팁: 소개 문구 작성법**
- 255자 제한이므로 한 줄이 효과적
- "~합니다" 동사형보다 "~솔루션", "~대시보드" 명사형 종결이 전문적
- 인스타 바이오와 동일한 문구로 통일 권장

**⚠️ 주의**
- 비즈니스 스위트(business.facebook.com)에서 만들려면 비즈니스 계정 연결이 필요
- 신규 계정이면 **facebook.com에서 직접** 만드는 게 더 확실하다

---

## Step 3: Meta Business Suite 연결

**왜 필요한가?**
페이지, 인스타, 광고 계정을 한 곳에서 관리하는 컨트롤 타워. 없으면 광고 계정 생성 및 체계적 관리가 불가.

**방법:**
```
business.facebook.com 접속
> 비즈니스 포트폴리오에 페이지 추가
> 설정 > 계정 > 페이지 > "추가" > 생성한 페이지 선택
```

**기존 비즈니스 포트폴리오가 있는 경우:**
- 신규 브랜드의 자산(페이지, 인스타, 광고계정)을 기존 포트폴리오에 넣으면 통합 관리 가능
- 단, 하나의 자산은 하나의 포트폴리오에만 속할 수 있음 (1:1 소유권)
- 다른 포트폴리오에서 먼저 제거 후 추가해야 함

---

## Step 4: 인스타그램 계정 연결 ⭐ 가장 까다로운 단계

**왜 필요한가?**
인스타 콘텐츠를 광고 소재로 사용하고, 인스타 피드/스토리/릴스 광고를 집행하려면 반드시 연결 필요.

### 방법 A: 인스타그램 앱에서 연결 (✅ 가장 확실)

```
인스타그램 앱 > 해당 브랜드 계정으로 전환
> 프로필 > ≡ (햄버거 메뉴)
> 설정 및 개인정보 > 계정 유형 및 도구
> "Facebook에 공유" 또는 "연결된 계정"
> Facebook > 해당 페이스북 페이지 선택
```

### 방법 B: Meta Business Suite에서 연결

```
business.facebook.com > 비즈니스 포트폴리오
> 설정 > 계정 > Instagram 계정 > "+ 추가"
> Instagram 계정 소유권 요청 > 로그인
```

### ⚠️ 자주 발생하는 문제와 해결

| 문제 | 원인 | 해결 |
|------|------|------|
| 소유권 요청 버튼이 로딩만 됨 | 인스타 계정이 이미 다른 포트폴리오에 속함 | 기존 포트폴리오에서 제거 후 재시도 |
| "삭제할 수 없습니다" | 페이스북 페이지와 인스타가 연결 상태 | "페이지 연결 해제" 먼저 클릭 후 삭제 |
| 비즈니스 포트폴리오에서 계속 실패 | Meta Business Suite 경로의 버그/딜레이 | **인스타 앱에서 직접 연결 (방법 A)** |
| 프로페셔널 계정이 아닙니다 | 개인 계정 상태 | 인스타 > 설정 > 프로페셔널 계정으로 전환 |

**💡 핵심 교훈: Meta Business Suite에서 안 되면, 인스타그램 앱에서 직접 연결하는 것이 가장 확실하다.**

---

## Step 5: 광고 계정 생성

**방법:**
```
Meta Business Suite > 설정 > 광고 계정 > "추가" > "새 광고 계정 만들기"

- 광고 계정 이름: 브랜드명 (예: Brandrise Ad Account)
- 시간대: (GMT+09:00) Asia/Seoul
- 통화: KRW (한국 원)
```

**⚠️ 시간대와 통화는 생성 후 변경 불가 — 반드시 확인 후 생성**

---

## Step 6: 결제 수단 등록

```
Meta Business Suite > 설정 > 결제
> 결제 수단 추가 > 신용카드/체크카드 등록
```

---

## Step 7: 첫 광고 집행

```
adsmanager.facebook.com 접속
> "+ 만들기" > 캠페인 목표 선택
> 대표 계정 설정:
  - Facebook 페이지: 브랜드 페이지 선택
  - Instagram 계정: 브랜드 인스타 선택
> 광고 소재: 콘텐츠 이미지 업로드
> 타겟/예산/일정 설정
> 게시
```

## 전체 구조 요약

```
메타 생태계 구조:

👤 페이스북 개인 계정 (관리자 인증)
 └─ 🏢 비즈니스 포트폴리오 (회사 단위 컨트롤 타워)
     ├─ 📄 페이스북 페이지 (브랜드 얼굴, 광고주 신원)
     ├─ 📷 인스타그램 계정 (콘텐츠 + 광고 노출 채널)
     └─ 💳 광고 계정 (예산, 결제, 성과 데이터)
```
