# brandrise 광고 채널 전략 — 세그먼트 3-4

> 작성일: 2026-02-27
> 근거: TARGET-INSIGHT-MID.md, segment3-mid-growth.md, segment4-expansion.md
> 타겟: 매출 10-100억 기업 (세그먼트 3-4)

## 전략 요약

### 핵심 원칙
1. **콘텐츠 마케팅 → 광고 → 리드** 순서 (콘텐츠 자산이 먼저)
2. **메타 + 링크드인** 투 채널 집중 (분산 X)
3. **리타겟팅 퍼널** 설계 (인지 → 관심 → 전환)
4. **데이터 기반 최적화** — 2주 단위 성과 리뷰

### 왜 이 채널인가? (리서치 근거)

| 채널 | 세그먼트 3 영향도 | 세그먼트 4 영향도 | 선택 이유 |
|------|:---:|:---:|----------|
| **메타 (인스타/페이스북)** | 높음 | 중간 | 카드뉴스/릴스 소재 활용, 넓은 도달, 정교한 타겟팅 |
| **링크드인** | 높음 | 매우 높음 | B2B 의사결정자 직접 타겟, 85%가 효과적 채널로 꼽음 |
| 유튜브 | 높음 | 중-높음 | 2차 확장용 (1차 채널 검증 후) |
| 구글 검색 | 중간 | 중간 | 인텐트 캡처용 (브랜드 인지 후 검색하는 유저) |
| 뉴스레터 | 높음 | 높음 | 오가닉 채널 — 유료 광고 아닌 자체 미디어 |

## Phase 1: 런칭 (1-2주차) — 콘텐츠 자산 확보 + 테스트

### 목표
- 광고 소재 성과 데이터 확보 (CTR, CPC 벤치마크)
- 타겟 오디언스 반응 검증
- 최적 소재/메시지 조합 발견

### 메타 광고

#### 캠페인 구조

```
캠페인 1: 인지도 (Awareness)
├── 광고 세트 A: 세그먼트 3 (매출 10-30억)
│   ├── 소재 1: 카드뉴스 캐러셀 "마케팅팀이 있는데 왜 전략이 없을까?"
│   ├── 소재 2: 싱글 이미지 "광고를 끄면 매출도 멈추나요?"
│   └── 소재 3: 릴스 15초 (공감형)
│
└── 광고 세트 B: 세그먼트 4 (매출 30-100억)
    ├── 소재 1: 카드뉴스 캐러셀 "대행사 3곳 쓰는데 왜 매출은 그대로일까?"
    ├── 소재 2: 싱글 이미지 "대행사 3곳 → 1팀"
    └── 소재 3: 릴스 30초 (전문가형)

캠페인 2: 전환 (Conversion) — 2주차부터
├── 광고 세트: 리타겟팅
│   ├── 카드뉴스 3장+ 본 사람 → CMO 비교 싱글 이미지
│   ├── 릴스 15초+ 시청자 → CMO 비교 싱글 이미지
│   └── 웹사이트 방문자 → 리드 광고 (무료 진단)
```

#### 타겟팅 설정

**세그먼트 3:**
| 항목 | 설정 |
|------|------|
| 연령 | 30-50세 |
| 직책 관심사 | CEO, 대표이사, 마케팅 리드, 공동창업자, 사업 개발 |
| 관심사 | 스타트업, 마케팅 전략, 브랜딩, 퍼포먼스 마케팅, 사업 성장, 시리즈A |
| 행동 | 소기업 경영자, 비즈니스 페이지 관리자 |
| 제외 | 대학생, 취업 준비생 |

**세그먼트 4:**
| 항목 | 설정 |
|------|------|
| 연령 | 33-53세 |
| 직책 관심사 | CEO, CMO, 마케팅 이사, VP Marketing, COO |
| 관심사 | 경영 전략, 마케팅 ROI, 브랜드 전략, 기업 성장, 시리즈B |
| 행동 | 소기업 경영자, 비즈니스 페이지 관리자 |
| 제외 | 대학생, 프리랜서, 1인 기업 |

| 항목 | 일 예산 | 월 예산 | 비고 |
|------|--------|--------|------|
| 캠페인 1 (인지도) | 5만원 | 150만원 | Seg3: 3만, Seg4: 2만 |

#### 예상 KPI (Phase 1)

| 지표 | 목표 |
|------|------|

| 랜딩페이지 방문 | 1,000-2,000/월 |
| 리드 (상담 신청) | 10-30건/월 |
| CPL (리드당 비용) | 8,000-24,000원 |

### 링크드인 (오가닉 우선)

#### 전략: 대표 개인 브랜딩 → 인바운드

링크드인은 **유료 광고보다 오가닉 콘텐츠가 더 효과적**인 채널.
대표 개인 계정으로 전문성 있는 콘텐츠를 발행하여 인바운드 리드를 유도.

#### 주간 콘텐츠 플랜

| 요일 | 콘텐츠 유형 | 예시 |
|------|-----------|------|
| 월 | 인사이트 포스트 | "매출 10-30억 기업이 겪는 마케팅의 구조적 문제" |
| 수 | 프레임워크/방법론 | "퍼포먼스 → 브랜드 전환 3단계 프레임워크" |
| 금 | 케이스 스터디/데이터 | "CAC 17,000원 → 6,200원. 전략 전환의 힘" |

#### 포스트 구조 (Pain → Value → Proof)

```
훅 (첫 2줄): 타겟의 Pain 직격
  "마케팅팀 3명인데 전략이 없다.
   이건 팀의 문제가 아닙니다."

본문 (5-8줄): Value 제시
  - 문제의 구조적 원인
  - 해결 방향 (프레임워크)
  - 구체적 데이터/사례

CTA (마지막 2줄): 행동 유도
  "이 상황이 익숙하시다면 DM 주세요.
   30분이면 현재 마케팅 구조를 진단해드립니다."

해시태그 (3-5개):
  #마케팅전략 #CMO #브랜드전략 #스타트업성장 #brandrise
```

## Phase 2: 최적화 (3-4주차) — 데이터 기반 조정

### 목표
- Phase 1 데이터 기반 소재/타겟 최적화
- 상위 성과 소재 스케일업
- 하위 성과 소재 교체

### 최적화 체크리스트

- [ ] CTR 상위 20% 소재 → 예산 2배 증가
- [ ] CTR 하위 20% 소재 → 중단 또는 카피 수정
- [ ] 세그먼트 3 vs 4 반응 비교 → 우선순위 재조정
- [ ] 리타겟팅 전환율 확인 → CPL 기준으로 판단
- [ ] 링크드인 포스트 인게이지먼트 상위 주제 → 메타 광고 소재로 전환

### 소재 교체 주기
- 싱글 이미지: 2주마다 A/B 테스트
- 카드뉴스: 월 1회 신규 시리즈
- 릴스: 월 2회 신규 (성과 좋으면 유지)

## Phase 3: 스케일업 (2개월차~) — 채널 확장

### 조건
- Phase 1-2에서 CPL < 20,000원 달성 시 스케일업
- 리드 → 상담 전환율 30% 이상 확인 시

### 확장 채널

#### 구글 검색 광고 (인텐트 캡처)
- 키워드: "마케팅 전략 컨설팅", "CMO 서비스", "마케팅 조직 재편", "대행사 교체"
- 랜딩페이지: 세그먼트별 전용 (Seg3용, Seg4용)
- 예산: 월 100-200만원

#### 링크드인 유료 광고
- InMail 광고: CMO/마케팅 이사 직접 타겟
- 피드 광고: 케이스 스터디 기반 캐러셀
- 예산: 월 100-300만원

#### 유튜브 (콘텐츠 마케팅)
- 포맷: 10-15분 심층 콘텐츠 ("매출 30억에서 100억 가는 마케팅 전략")
- 목적: SEO + 신뢰 구축 (리드 직접 전환보다 브랜딩)
- 예산: 제작비 위주 (유료 광고는 선택적)

## 측정 대시보드

### 주간 리뷰 지표

| 카테고리 | 지표 | 목표 |
|---------|------|------|
| **도달** | 노출수, 도달수 | 주 15,000+ |
| **반응** | CTR, 영상 조회율 | CTR 2%+ |
| **비용** | CPC, CPM, CPL | CPL 20,000원 이하 |
| **전환** | 랜딩페이지 방문, 리드 수 | 주 리드 5건+ |
| **질** | 상담 전환율, 계약 전환율 | 상담 30%, 계약 20% |

### 월간 리뷰 포인트

1. 소재별 성과 비교 → 상위 소재 스케일업
2. 세그먼트별 반응 비교 → 예산 재배분
3. 채널별 CPL 비교 → 채널 효율 판단
4. 리드 품질 분석 → 타겟팅 정밀도 조정
5. 콘텐츠 주제별 반응 → 다음 달 콘텐츠 기획 반영
