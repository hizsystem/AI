## 1. 기업 프로필

**기본 정보**
- 브랜드: 슈퍼세이브(SuperSave) | 법인명: 승만
- 대표: 신주리 | 매출 규모: 10~50억
- 웹사이트: paceandproof.com (D2C 쇼핑몰)
- 카테고리: 건강기능식품 / 자연식품

**주력 제품 — 곰보배추 시럽**
- 가격: 34,800원
- 타겟: 비염/환절기 건강 관리
- 브랜드 스토리: "두 아들의 엄마가 만든 우리집 상비식품"
- 포지셔닝: 엄마의 진심 + 자연 원료 → 가족 건강

**현재 마케팅 현황**
- 마케팅팀: 1~2명
- 데이터 관리: 기초 수준 (체계적 분석 미흡)
- 주요 채널: D2C 자사몰 중심

**경쟁사**

| 경쟁사 | 포지션 | 슈퍼세이브와의 차이 |
|--------|--------|-------------------|
| 광동제약 (비염S) | 대기업 비염 OTC | 슈퍼세이브는 자연식품, 광동은 의약품 |
| 종근당건강 | 건강기능식품 대기업 | 슈퍼세이브는 니치 원료(곰보배추), 종근당은 범용 |
| 곰보배추즙 소형 제조사 | 원료 직판매 | 슈퍼세이브는 시럽 제형 차별화 + 브랜드 스토리 |

## 2. 시장 환경

**시장 규모**
- 국내 건강기능식품 시장: 5.96조 원 (2024), 연평균 7.8% 성장
- 자연식품/원물 건강식품: 약 1.2조 원 규모
- 올리브영 헬스케어 카테고리: 연 30%+ 성장 (뷰티→헬스 확장 중)

**핵심 트렌드**
- "이너뷰티" 열풍 → 건강기능식품의 뷰티 카테고리 편입
- 30-40대 "엄마 소비자" 타겟 건강식품 급성장
- D2C → 올리브영/카카오 선물하기 멀티채널 확장이 스케일업 공식
- SNS 기반 "후기 마케팅"이 건강식품 구매 결정의 핵심

**올리브영 입점 핵심 정보**
- 수수료: 50~55% (카테고리별 상이)
- 입점 조건: 차별화된 콘텐츠 + SNS 바이럴 실적 + MD 미팅
- 입점 전략: 온라인몰 먼저 → 실적 기반 오프라인 확장
- 핵심: "MD가 원하는 숫자" = SNS 언급량 + 자사몰 판매 실적

**카카오 선물하기 입점 핵심 정보**
- 수수료: 20~30%
- 입점 조건: 사업자등록 + 상품 심사 + 브랜드 스토리
- 핵심: 선물하기에 적합한 패키지 디자인 + 가격대(2~5만원) 필수

**효과적 마케팅 채널 Top 3**
1. 인스타그램/SNS — 건강식품 후기 UGC가 구매 전환의 핵심 (전환율 3~5배)
2. 네이버 검색 — "곰보배추 효능" 등 건강 정보 검색 → 블로그/체험단
3. 올리브영/카카오 — 채널 자체가 신뢰 보증 역할 (입점 = 브랜드 인증)

**벤치마크**
- 닥터브로너스: 자연주의 스토리텔링 → 올리브영 상위 안착
- VB프로그램(비비랩): D2C → 올리브영 온라인 → 오프라인 순차 확장 성공
- 정관장 에브리타임: 카카오 선물하기 건강식품 1위, 패키지 디자인이 핵심

## 3. 핵심 고민 분석

**설문 원문**
- 브랜드 전략, 마케팅 실행, 데이터 관리, 퍼포먼스 — 전 영역에서 도움 필요
- 올리브영, 카카오 선물하기 입점 희망
- 궁극적으로 글로벌 진출 목표

**근본 원인 진단**

| 가설 | 설명 |
|------|------|
| 포지셔닝 불명확 | "곰보배추 시럽"이라는 제품은 있지만, 소비자가 "왜 이 브랜드?"를 선택할 이유가 불명확. "엄마의 마음"이라는 스토리가 마케팅에 체계적으로 녹아들지 않음 |
| 채널 전략 부재 | 자사몰 매출 10~50억은 탄탄하나, 올리브영/카카오 입점을 위한 사전 준비(SNS 바이럴, 패키지 최적화, MD 피칭 자료)가 체계적이지 않음 |
| 패키징 최적화 부재 | 올리브영 진열대/카카오 선물하기 썸네일에 최적화된 패키지 디자인이 필요. 현재 자사몰 중심 디자인으로는 채널별 차별화 어려움 |
| 데이터/퍼포먼스 기초 부족 | 마케팅팀 1~2명으로 데이터 기반 의사결정 어려움. 광고 효율 측정, 고객 분석 체계 미비 |

**D2C → 멀티채널 확장의 전형적 패턴**
- 자사몰 10억+ 매출은 PMF(Product-Market Fit) 검증 완료 신호
- 다음 단계는 "어디에 나갈 것인가"가 아니라 "어떤 모습으로 나갈 것인가" (브랜딩 + 패키징)
- 올리브영 수수료 50%+를 감당하려면 브랜드 프리미엄이 필수 → 포지셔닝 선행

## 5. 미팅 전략

**핵심 질문 3개**
1. "현재 자사몰 매출에서 신규 vs 재구매 비율은? 가장 많이 구매하는 고객층은 누구인가?"
   → 핵심 고객 세그먼트 파악 + 올리브영 타겟과의 일치 여부 확인
2. "올리브영/카카오 입점을 위해 지금까지 시도한 것은? MD 미팅 경험이 있는가?"
   → 현재 입점 준비 수준 + 빠르게 도울 수 있는 영역 파악
3. "곰보배추 시럽의 재구매율은? 고객이 가장 많이 언급하는 구매 이유는?"
   → 제품력 검증 + 마케팅 메시지 소스 확보

**Quick Win 제안: 4주 올리브영 입점 준비 프로젝트**
- 1주차: 고객 데이터 분석 + 포지셔닝 재정의 ("엄마의 상비식품" → 채널별 메시지 최적화)
- 2주차: 올리브영 입점용 브랜드 스토리 + 패키지 컨셉 방향 제안
- 3주차: SNS 콘텐츠 3종 (엄마 후기 UGC 가이드 + 체험단 기획)
- 4주차: 올리브영 MD 피칭 자료 초안 + 카카오 선물하기 상품 구성 제안
- 산출물: 포지셔닝 스테이트먼트 + 채널별 메시징 맵 + SNS 콘텐츠 3종 + MD 피칭 자료

**미팅 오프닝 포지셔닝:**
> "슈퍼세이브는 자사몰에서 10억 이상 매출을 만들어낸 검증된 브랜드입니다. '두 아들의 엄마가 만든 상비식품'이라는 강력한 스토리도 있고요. 지금 필요한 건 이 브랜드 자산을 올리브영과 카카오에서도 통하는 언어로 바꾸는 것입니다."

**예상 패키지**: Professional (포지셔닝 + 콘텐츠 + 채널 입점 전략)
