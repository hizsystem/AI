# 미례국밥 마케팅 제안서 — A안: 바이럴 임팩트형

> **"돼지우동, 서울에서 터뜨리겠습니다"**
> 제안일: 2026-03-10 | 브랜드라이즈

---

## 현재 상황

**잘 하고 계신 것**
- 리뷰 900건+ → 부산 내 신뢰도 확보
- 댄싱컵 100~120개 가맹점 → 프랜차이즈 노하우 보유
- "돼지우동" 시그니처 → 차별화 자산

**놓치고 계신 것**
- 브랜드 스토리 부재 → "왜 미례인가?" 답이 없는 상태
- 인스타그램이 의무 채널로 운영됨
- 바이럴 의존 → 돈 안 쓰면 멈추는 구조

---

## A안 전략: 바이럴로 먼저 터뜨리고, 브랜드는 기본만 잡는다

> 돼지우동의 "의외성"을 무기로 서울 2030 타겟에 빠르게 인지도를 만들고,
> 그 트래픽이 네이버플레이스와 인스타그램으로 흘러들어오게 설계합니다.

---

## 서비스 구성

### 1. 브랜드 전략 — 150만원

> 바이럴이 터졌을 때 "검색해서 들어온 사람"이 실망하지 않을 최소한의 기반

| 산출물 | 내용 |
|--------|------|
| 채널 컨셉 정리 | 미례국밥 인스타그램 톤앤매너, 컬러, 방향 1안 |
| 피드 가이드라인 | 포스팅 카테고리, 그리드 구성 가이드 |
| 프로필 리뉴얼 | 프로필 사진, 소개문구, 하이라이트 구성 |

### 2. 바이럴 콘텐츠 — 200만원

> A안의 핵심. "국밥집인데 우동이 제일 잘 나가요" 의외성 바이럴

| 산출물 | 내용 |
|--------|------|
| 숏폼 콘텐츠 기획·디렉팅 3편 | 릴스/숏츠 콘셉트 기획 + 촬영 디렉팅 + 편집 방향 |
| 인플루언서 시딩 기획 2건 | 서울 맛집 인플루언서 선정 + 가이드라인 + 성과관리 |

### 3. 광고 운영 — 100만원

> 바이럴 콘텐츠를 광고로 증폭

| 산출물 | 내용 |
|--------|------|
| 메타 광고 세팅·운영 | 릴스 도달 광고 + 전환 광고 미디어믹스 |
| 타겟팅 | 서울 2030 맛집 관심층 |
| 리포팅 | 월 1회 성과 보고 |

### 4. 네이버플레이스 셋업 — 50만원

> 바이럴 → 검색 유입의 착지 지점

| 산출물 | 내용 |
|--------|------|
| 기본 정보 최적화 | 영업시간, 브레이크타임, 주차, 단체석, 포장 등 전 항목 점검 |
| 키워드 세팅 | 지역+업종 조합 10~15개 |
| 메뉴판 정비 | 전 메뉴 등록 + 사진 + 가격 |
| 대표 사진·소개란 | 사진 퀄리티 업 가이드 + 브랜드 카피 |
| 스마트콜·인스타 연동 | 외부 트래픽 시그널 확보 |

### (번외) 프랜차이즈 대시보드

> pace&proof 상품으로 별도 제안 가능. 가맹 문의 트래킹, 문의자 관리 대시보드.

## A안의 특징

- **바이럴 기획 비중 60%** (콘텐츠 + 광고) → 빠른 인지도 확보에 집중
- 브랜드 전략은 "검색했을 때 실망 안 할 수준"의 기본만
- 네이버플레이스 1회 셋업 → 바이럴 트래픽의 착지점
- **1개월 안에 눈에 보이는 숫자**(조회수, 도달)를 만들 수 있음
- 리스크: 바이럴 부진 시 브랜드 자산이 상대적으로 적음
