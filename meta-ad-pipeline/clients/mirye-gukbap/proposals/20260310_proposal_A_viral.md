# 미례국밥 마케팅 제안서 — A안: 바이럴 임팩트형

> **"돼지우동, 서울에서 터뜨리겠습니다"**
> 제안일: 2026-03-10 | 브랜드라이즈

---

## 현재 상황

**잘 하고 계신 것**
- 리뷰 900건+ → 부산 내 신뢰도 확보
- 댄싱컵 100~120개 가맹점 → 프랜차이즈 노하우 보유
- "돼지우동" 시그니처 → 차별화 자산

**놓치고 계신 것**
- 브랜드 스토리 부재 → "왜 미례인가?" 답이 없는 상태
- 인스타그램이 의무 채널로 운영됨
- 바이럴 의존 → 돈 안 쓰면 멈추는 구조

---

## A안 전략: 콘텐츠 제작까지 한번에, 바이럴로 먼저 터뜨린다

> 돼지우동의 "의외성"을 무기로 서울 2030 타겟에 빠르게 인지도를 만들고,
> 그 트래픽이 네이버플레이스와 인스타그램으로 흘러들어오게 설계합니다.
> 숏폼 기획부터 제작까지 포함 — 별도 촬영팀 구할 필요 없습니다.

## 서비스 구성

### 1. 브랜드 전략 — 150만원

> 바이럴이 터졌을 때 "검색해서 들어온 사람"이 실망하지 않을 채널 기반

| 산출물 | 내용 |
|--------|------|
| 채널 컨셉 + 무드보드 | 미례국밥 인스타 비주얼 방향 설정 (톤앤매너, 컬러, 무드) |
| 피드 가이드라인 | 포스팅 카테고리, 그리드 구성, 톤앤매너 가이드 |
| Figma 피드 템플릿 | 인스타 피드 템플릿 (내부에서 베리에이션 가능) |
| 콘텐츠 캘린더 1개월 | 주제별 포스팅 기획안 + 해시태그 전략 |

### 2. 바이럴 콘텐츠 기획·제작 — 250만원

> A안의 핵심. "국밥집인데 우동이 제일 잘 나가요" 의외성 바이럴.
> 기획부터 촬영·편집까지 올인원으로 제작합니다.

| 산출물 | 내용 |
|--------|------|
| 숏폼 콘텐츠 2건 (기획→촬영→편집) | 릴스/숏츠 콘셉트 기획 + 촬영 + 편집 완료본 납품 |
| 콘텐츠 전략 방향 | 바이럴 시나리오, 후킹 포인트, CTA 설계 |
| 채널 배포 전략 | 릴스·숏츠·틱톡 최적 포맷별 편집 가이드 |

### 3. 광고 운영 — 50만원

> 바이럴 콘텐츠를 광고로 증폭 (광고비 별도)

| 산출물 | 내용 |
|--------|------|
| 메타 광고 세팅·운영 | 릴스 도달 광고 + 전환 광고 미디어믹스 |
| 타겟팅 | 서울 2030 맛집 관심층 |
| 리포팅 | 주 1회 성과 보고 (노출, 도달, CTR, CPC) |

### 4. 네이버플레이스 셋업 — 50만원

> 바이럴 → 검색 유입의 착지 지점. 1회 셋업.

| 산출물 | 내용 |
|--------|------|
| 기본 정보 최적화 | 영업시간, 브레이크타임, 주차, 단체석, 포장 등 전 항목 점검 |
| 키워드 세팅 | 지역+업종 조합 10~15개 |
| 메뉴판 정비 | 전 메뉴 등록 + 사진 + 가격 |
| 대표 사진·소개란 | 사진 퀄리티 업 가이드 + 브랜드 카피 |
| 스마트콜·인스타 연동 | 외부 트래픽 시그널 확보 |

### (번외) 프랜차이즈 대시보드

> pace&proof 상품으로 별도 제안 가능. 가맹 문의 트래킹, 문의자 관리 대시보드.

## A안의 특징

- **바이럴 콘텐츠 비중 50%** → 빠른 인지도 확보에 올인
- **숏폼 제작 포함** → 별도 촬영팀 없이 완결. 기획~납품까지 원스톱
- 브랜드 전략은 Figma 템플릿+캘린더까지 → 계약 후에도 자체 운영 가능
- 네이버플레이스 1회 셋업 → 바이럴 트래픽의 검색 착지점
- **별도비용이 광고비뿐** → 총 비용 예측이 명확
- 리스크: 인플루언서 없이 자체 콘텐츠+광고만으로 도달 확보
