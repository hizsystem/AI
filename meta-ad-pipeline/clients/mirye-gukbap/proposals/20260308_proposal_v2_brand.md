# 미례국밥 마케팅 제안서 — B안: 브랜드 빌딩 + 바이럴 병행형

> **"부산의 맛집에서, 전국이 찾는 브랜드로"**
> 제안일: 2026-03-08 | 브랜드라이즈

---

## 1. 현재 상황 진단

### 잘 하고 계신 것
- 리뷰 900건+ → 부산 내 온라인 신뢰도 확보 완료
- 댄싱컵 100~120개 가맹점 → 프랜차이즈 운영 노하우 보유
- "돼지우동"이라는 독보적 시그니처 → 차별화 자산

### 놓치고 계신 것
- **브랜드 스토리가 없음** → "왜 미례인가?"에 답할 수 없는 상태
- 인스타그램이 **홍보 채널이 아닌 의무 채널**로 운영됨
- 체험단/인플루언서에 의존 → 돈 안 쓰면 멈추는 구조
- 홈페이지가 아직 없음 → 프랜차이즈 문의자가 갈 곳이 없음

### 핵심 문제
> 미례국밥은 "맛있는 국밥집"은 되었지만, "가맹하고 싶은 브랜드"는 아직 아닙니다.
> 단순 바이럴만으로는 가맹 문의가 와도 "이 브랜드를 믿어도 되나?" 확인할 곳이 없습니다.

## 2. 제안 전략: 브랜드 빌딩 + 바이럴 투트랙

### A 트랙: 브랜드 빌딩 (지속 자산)
> 검색했을 때 "이 브랜드 제대로네" 하고 느끼게 만드는 구조

### B 트랙: 돼지우동 바이럴 (즉시 임팩트)
> "국밥집인데 우동이 제일 잘 나가요" → 의외성 바이럴

### 왜 두 트랙이 필요한가

```
  바이럴만 할 경우:
  ─────────────────────
  조회수 ↑ → 관심 ↑ → "근데 이 브랜드 뭐야?" → 정보 없음 → 이탈

  브랜드 빌딩만 할 경우:
  ─────────────────────
  브랜드 정리 ↑ → 콘텐츠 ↑ → ... 아무도 안 봄 → 효과 없음

  두 트랙 병행:
  ─────────────────────
  바이럴 → 관심 → 검색 → "브랜드 괜찮네" → 가맹 문의
  ▲ B트랙                ▲ A트랙             ▲ 전환
```

## 6. 왜 이 방식인가 — 댄싱컵의 교훈

댄싱컵이 100~120개 가맹점까지 성장한 건, 단순히 "싸고 맛있다"가 아니라 **"댄싱컵이라는 브랜드를 알고 가맹 신청한 사람들"** 덕분입니다.

미례국밥도 마찬가지입니다:
- 바이럴로 "와 이거 맛있겠다" → **맛집 인지**
- 브랜드로 "이 가게 브랜드 관리를 잘 하네" → **가맹 신뢰**

이 두 가지가 함께 있어야 **"여기서 창업하면 되겠다"**는 확신이 생깁니다.

## 7. 브랜드라이즈가 하는 일

| 영역 | 내용 |
|------|------|
| **브랜드 전략** | 브랜드 스토리, 톤앤매너, 비주얼 가이드라인 |
| **콘텐츠 전략/운영** | 인스타 콘텐츠 캘린더, 피드 기획, 숏폼 디렉팅 |
| **바이럴** | 인플루언서/유튜버 섭외, 가이드라인, 성과 관리 |
| **광고** | 메타/유튜브 광고 세팅, 최적화, 리포팅 |
| **가맹 마케팅** | 가맹 LP 카피, 타겟 광고, SEO 콘텐츠 |
| **리포팅** | 월간 성과 보고 + 전략 조정 + 대표님 보고용 요약 |

> 변미혜 팀장님은 현장 운영, 인플루언서 방문 응대, 메뉴 관련 소통에 집중하세요.
> 온라인 전략, 콘텐츠 제작, 광고 운영, 브랜드 관리는 저희가 책임집니다.
