# 미례국밥 마케팅 제안서 — B안: 바이럴 + 시딩 확장형

> **"콘텐츠 + 인플루언서 + 플레이스, 삼각 편대로 터뜨립니다"**
> 제안일: 2026-03-10 | 브랜드라이즈

---

## 현재 상황

**잘 하고 계신 것**
- 리뷰 900건+ → 부산 내 신뢰도 확보
- 댄싱컵 100~120개 가맹점 → 프랜차이즈 노하우 보유
- "돼지우동" 시그니처 → 차별화 자산

**놓치고 계신 것**
- 브랜드 스토리 부재 → "왜 미례인가?" 답이 없는 상태
- 인스타그램이 의무 채널로 운영됨
- 네이버플레이스 등록만 해놓고 관리가 안 되는 상태

---

## B안 전략: A안의 바이럴에 인플루언서 시딩 + 플레이스 관리까지

> A안의 숏폼 제작 + 광고 구조를 그대로 가져가면서,
> 인플루언서 시딩 1건으로 초기 신뢰도를 확보하고,
> 네이버플레이스는 셋업뿐 아니라 경쟁분석·리뷰관리·운영교육까지 포함합니다.

---

## 서비스 구성

### 1. 브랜드 전략 — 150만원

> A안과 동일. 바이럴이 터졌을 때 채널 기반이 되는 자산.

| 산출물 | 내용 |
|--------|------|
| 채널 컨셉 + 무드보드 | 미례국밥 인스타 비주얼 방향 설정 (톤앤매너, 컬러, 무드) |
| 피드 가이드라인 | 포스팅 카테고리, 그리드 구성, 톤앤매너 가이드 |
| Figma 피드 템플릿 | 인스타 피드 템플릿 (내부에서 베리에이션 가능) |
| 콘텐츠 캘린더 1개월 | 주제별 포스팅 기획안 + 해시태그 전략 |

### 2. 바이럴 콘텐츠 기획·제작 — 300만원

> A안 숏폼 2건 제작 + 인플루언서 시딩 1건 기획·관리.
> 자체 콘텐츠와 인플루언서 콘텐츠가 동시에 퍼지는 구조.

| 산출물 | 내용 |
|--------|------|
| 숏폼 콘텐츠 2건 (기획→촬영→편집) | 릴스/숏츠 콘셉트 기획 + 촬영 + 편집 완료본 납품 |
| 인플루언서 시딩 1건 (기획·관리) | 서울 맛집 인플루언서 선정 + 촬영 가이드라인 + 성과 관리 |
| 콘텐츠 전략 방향 | 바이럴 시나리오, 후킹 포인트, CTA 설계 |
| 채널 배포 전략 | 릴스·숏츠·틱톡 최적 포맷별 편집 가이드 |

> **인플루언서 시딩이 추가되면?**
> 자체 숏폼이 "이 집 뭐야?" 호기심을 만들고,
> 인플루언서가 "가봤는데 진짜임" 신뢰를 더합니다.
> 두 콘텐츠가 같은 시기에 나가면 알고리즘 시너지로 도달이 2~3배 됩니다.

### 3. 광고 운영 — 50만원

> A안과 동일. 자체 숏폼 + 인플루언서 콘텐츠를 광고로 증폭 (광고비 별도)

| 산출물 | 내용 |
|--------|------|
| 메타 광고 세팅·운영 | 릴스 도달 광고 + 전환 광고 미디어믹스 |
| 타겟팅 | 서울 2030 맛집 관심층 |
| 리포팅 | 주 1회 성과 보고 (노출, 도달, CTR, CPC) |

### 4. 네이버플레이스 셋업 + 운영 교육 — 100만원

> 셋업만 하면 한 달 뒤 다시 방치됩니다.
> 경쟁 분석, 리뷰 관리, 운영 교육까지 해야 진짜 효과가 지속됩니다.

| 산출물 | 내용 |
|--------|------|
| 기본 정보 최적화 | 영업시간, 브레이크타임, 주차, 단체석, 포장 등 전 항목 점검 |
| 키워드 세팅 | 지역+업종 조합 10~15개 |
| 메뉴판 정비 | 전 메뉴 등록 + 사진 + 가격 |
| 대표 사진·소개란 | 사진 퀄리티 업 가이드 + 브랜드 카피 |
| 스마트콜·인스타 연동 | 외부 트래픽 시그널 확보 |
| 경쟁 매장 비교 분석 | 같은 상권 국밥집 3곳 플레이스 비교 리포트 |
| 리뷰 관리 가이드 | 응답 템플릿 + 영수증 리뷰 유도 POP 제안 |
| 담당자 운영 교육 1회 | 네이버플레이스 관리법, 게시물 작성, 리뷰 응대 실습 |

### (번외) 프랜차이즈 대시보드

> pace&proof 상품으로 별도 제안 가능. 가맹 문의 트래킹, 문의자 관리 대시보드.

## B안의 특징

- **A안 전체 포함 + 인플루언서 시딩 1건 + 플레이스 강화**
- 자체 숏폼 + 인플루언서 동시 발행 → 알고리즘 시너지
- 네이버플레이스 셋업 + 경쟁분석 + 리뷰관리 + 교육 → 계약 후에도 자체 관리 가능
- **+100만원으로 인플루언서 채널 + 플레이스 교육까지 확보**
- 별도비용은 광고비 + 인플루언서 섭외비(마이크로 기준 30~50만)
- 리스크: 총 비용이 A안 대비 약 130~180만원 증가
