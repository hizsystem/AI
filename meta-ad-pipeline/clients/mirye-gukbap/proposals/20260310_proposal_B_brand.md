# 미례국밥 마케팅 제안서 — B안: 브랜드 체계 구축형

> **"부산의 맛집에서, 가맹하고 싶은 브랜드로"**
> 제안일: 2026-03-10 | 브랜드라이즈

---

## 현재 상황

**잘 하고 계신 것**
- 리뷰 900건+ → 부산 내 신뢰도 확보
- 댄싱컵 100~120개 가맹점 → 프랜차이즈 노하우 보유
- "돼지우동" 시그니처 → 차별화 자산

**놓치고 계신 것**
- 브랜드 스토리 부재 → "왜 미례인가?" 답이 없는 상태
- 인스타그램이 의무 채널로 운영됨
- 홈페이지 없음 → 가맹 문의자가 확인할 곳 없음

---

## B안 전략: 브랜드 기반을 먼저 잡고, 바이럴은 핵심만 한다

> "돈 쓴 보람"이 계약 이후에도 남는 구조.
> 브랜드 가이드라인, 콘텐츠 템플릿, 네이버플레이스 최적화 →
> 계약이 끝나도 내부에서 계속 활용 가능한 자산을 만듭니다.

## 서비스 구성

### 1. 브랜드 전략 — 250만원

> B안의 핵심. 검색했을 때 "이 브랜드 제대로네" 하고 느끼게 만드는 구조

| 산출물 | 내용 |
|--------|------|
| 브랜드 스토리 정리 | "왜 미례인가" — 창업 스토리, 철학, 차별화 포인트 |
| 채널 컨셉 + 무드보드 | 인스타 비주얼 방향 3안 제시 → 1안 확정 |
| 피드 가이드라인 | 그리드 구성, 포스팅 카테고리, 톤앤매너 |
| 피드 템플릿 5종 | Canva/Figma 템플릿 (내부 디자이너가 베리에이션 가능) |
| 샘플 피드 9장 | 인스타 첫 화면 완성 |
| 콘텐츠 캘린더 1개월 | 주제별 포스팅 기획안 + 해시태그 전략 |

### 2. 바이럴 콘텐츠 — 100만원

> 핵심 1편에 집중. 돼지우동 의외성 바이럴

| 산출물 | 내용 |
|--------|------|
| 숏폼 콘텐츠 기획·디렉팅 1편 | 릴스/숏츠 콘셉트 기획 + 촬영 디렉팅 + 편집 방향 |
| 인플루언서 시딩 기획 1건 | 서울 맛집 인플루언서 선정 + 가이드라인 + 성과관리 |

### 3. 광고 운영 — 80만원

> 숏폼 + 인플루언서 콘텐츠를 광고로 증폭

| 산출물 | 내용 |
|--------|------|
| 메타 광고 세팅·운영 | 릴스 도달 + 전환 미디어믹스 |
| 타겟팅 | 서울 2030 + 예비 창업자(30~50대) |
| 리포팅 | 월 1회 성과 보고 |

### 4. 네이버플레이스 셋업 — 70만원

> 셋업 + 경쟁 분석 + 리뷰 관리까지 포함

| 산출물 | 내용 |
|--------|------|
| 기본 정보 최적화 | 전 항목 점검 (영업시간, 주차, 단체석, 포장 등) |
| 키워드 세팅 | 지역+업종 조합 10~15개 |
| 메뉴판 정비 | 전 메뉴 등록 + 사진 + 가격 |
| 대표 사진·소개란 | 사진 퀄리티 업 가이드 + 브랜드 카피 |
| 스마트콜·인스타 연동 | 외부 트래픽 시그널 확보 |
| 경쟁 매장 비교 분석 | 같은 상권 국밥집 3곳 플레이스 비교 리포트 |
| 리뷰 관리 가이드 | 응답 템플릿 + 영수증 리뷰 유도 POP 제안 |

### (번외) 프랜차이즈 대시보드

> pace&proof 상품으로 별도 제안 가능. 가맹 문의 트래킹, 문의자 관리 대시보드.

## B안의 특징

- **브랜드 전략 비중 50%** → 계약 이후에도 남는 자산 (가이드라인+템플릿+캘린더)
- 바이럴은 핵심 1편에 올인 → 공수 효율화
- 네이버플레이스는 셋업 + 경쟁분석 + 리뷰관리까지 → 검색 기반 강화
- **내부 디자이너가 템플릿으로 계속 운영** 가능한 구조
- 리스크: 첫 달 바이럴 임팩트는 A안보다 약함
