## 1. 기업 프로필

**기본 정보**
- 설립: 2024년 3월 (1년차) | 대표: 윤창진 | 팀: 7명 (개발 중심)
- 본사: 서울 양천구 목동동로 309, 14층
- 매출: ~1억 (프리랜서 매칭), 칼리지스 출시 2주 만에 고객사 20개 확보
- 마케팅 예산: 월 300만원 이하
- 상담 경로: 지인추천

**주력 서비스 — 칼리지스(Kolleges)**
- 교육 플랫폼 원스톱 구축 SaaS (개발 없이)
- LMS + 결제 + 커뮤니티 + 시험/과제 + CRM + 디지털 인증서
- 블록체인 기반 디지털 배지 (Open Badge 3.0 국제표준)
- 수료증 SNS 공유 → 바이럴 마케팅 기능 내장

**레퍼런스**
- 포항공과대학교 (POSTECH)
- 베트남 FPT대학교
- 고려대/연세대/가천대 학회 등
- 초기 고객 20개사

**경쟁사**

| 경쟁사 | 포지션 | 칼리지스와의 차이 |
|--------|--------|----------------|
| Teachable/Thinkific | 해외 강의 플랫폼 빌더 | 칼리지스는 인증 시스템 + 한국 시장 최적화 |
| 클래스101 비즈니스 | 기업교육 솔루션 | 칼리지스는 자체 플랫폼 구축, 클래스101은 마켓플레이스 |
| 라이브클래스 | 교육자 플랫폼 구축 | 가장 직접적 경쟁. 칼리지스는 블록체인 인증 차별화 |
| 인프런 | IT/코딩 특화 | 마켓플레이스 모델. 칼리지스는 화이트라벨 플랫폼 |

## 2. 시장 환경

**시장 규모**
- 국내 SaaS 시장: 1조 1,430억원 (2025), 연평균 14.9% 성장
- 글로벌 EdTech: 1,871억 달러 (2025) → 7,246억 달러 (2035), 연 14.5%
- 글로벌 LMS: 24억 달러 (2025) → 104억 달러 (2034), 연 16.1%

**핵심 트렌드**
- 하이브리드 학습 확산 → 통합 LMS 수요 증가
- 디지털 배지/학습 인증 표준화 가속
- B2B SaaS: ABM(계정 기반 마케팅) + LinkedIn이 표준 채널
- PLG(Product-Led Growth): 무료 체험 → 유료 전환 모델 확산

**B2B SaaS 효과적 채널 Top 3**
1. 기존 네트워크 & 입소문 (93개 B2B기업 설문 1위)
2. 콘텐츠 마케팅 (블로그/SEO — CAC 가장 효율적)
3. 웨비나/PLG (무료 체험 전환율: 부분유료 12% > 무료평가 5%)

## 3. 핵심 고민 분석

**설문 원문**
- 신규 고객 확보 / 매출 전환율 개선
- 콘텐츠 기반 고객 유입 / SNS 운영
- 현 마케팅 문제 진단을 원함

**근본 원인 진단**

| 가설 | 설명 |
|------|------|
| 포지셔닝 부재 | "블록체인 기반 교육 플랫폼"은 기술 언어. 고객이 원하는 건 "복잡한 교육을 간단하게" 또는 "졸업생 경쟁력 강화"인데 이 가치가 전달되지 않음 |
| 타겟 세분화 불명확 | 대학/기업/강사 모두를 타겟 → 초기에 "모두를 위한" 메시지는 아무에게도 닿지 않음 |
| 콘텐츠 & 신뢰 자산 부재 | POSTECH, FPT 레퍼런스를 마케팅 자산으로 활용 못하고 있음. 사례 콘텐츠 제로 |

**B2B SaaS 초기 전형적 패턴**
- 지인추천으로 20개 확보 → 재현 가능한 마케팅 시스템으로 전환 못하면 성장 정체
- 좋은 프로덕트 ≠ 좋은 포지셔닝. 시장에 "왜 이것인가"를 설명하지 못하면 경쟁사에 밀림
- 월 300만원으로 모든 채널 불가 → "어디에 집중?"이 생존 결정

| 우선순위 | 서비스 | 이유 |
|---------|--------|------|
| 1 | 브랜드 전략 (포지셔닝/메시징) | 20개 고객사 분석 → "왜 칼리지스?"를 고객 언어로 재정의. 타겟 세그먼트별 메시징 분리 |

| 3 | GTM 전략 (PLG + 파트너십) | 무료 체험 최적화 + 교육기관 네트워크 파트너십 |
| 4 | 퍼포먼스 마케팅 | 포지셔닝 확립 후. 월 300만원으로는 포지셔닝 없이 광고 낭비 위험 |

## 5. 미팅 전략

**핵심 질문 3개**
1. "초기 20개 고객사 프로필은? (대학? 기업? 강사?) 어떤 경로로 칼리지스를 알게 되었나?"
   → 성공 세그먼트 역추적 → 마케팅 타겟 결정
2. "경쟁사(라이브클래스, Teachable 등) 대비 고객이 칼리지스를 선택한 가장 큰 이유는?"
   → 실제 차별점 vs 인지된 차별점 확인
3. "현재 마케팅 활동 중 가장 효과 있었던 것은? 그걸 반복/확대할 수 있다고 보는가?"
   → 이미 작동하는 루프를 파악하여 최적화

**Quick Win 제안: 4주 포지셔닝 + 초기 콘텐츠 전략**
- 1주차: 고객 3명 심층 인터뷰 + 경쟁사 분석 → 포지셔닝 스테이트먼트
- 2~3주차: 타겟별 콘텐츠 3종 (대학/기업/강사 각 1개)
- 4주차: Google Ads 초기 캠페인 (교육LMS/디지털배지 키워드)
- 산출물: 포지셔닝 + 메시징 맵 3종 + 블로그 3편 + 광고 크리에이티브 3종

**미팅 오프닝 포지셔닝:**
> "칼리지스는 2주 만에 20개 고객사를 확보한 강력한 프로덕트입니다. POSTECH, FPT대학 레퍼런스도 있고요. 지금 필요한 건 이 입소문 성장을 재현 가능한 마케팅 시스템으로 전환하는 것입니다."

**예상 패키지**: Essential (포지셔닝 + 콘텐츠 기초)
