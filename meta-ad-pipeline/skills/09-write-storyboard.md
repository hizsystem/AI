# Skill 09: write-storyboard

> 카드뉴스용 슬라이드별 스토리보드 작성

## 호출 에이전트
copywriter

## 프로세스
1. 리서치 결과에서 핵심 포인트 7-10개 추출
2. 슬라이드 구조 설계:
   - 1장: 표지 (후킹 제목)
   - 2-8장: 본문 (포인트당 1장)
   - 9장: 요약/정리
   - 10장: CTA
3. 각 슬라이드별 헤드라인 + 본문 + 비주얼 노트

## 스토리보드 규칙
- 표지: 궁금증 유발 또는 충격적 사실
- 본문: 1슬라이드 1메시지
- 텍스트: 슬라이드당 최대 50자
- CTA: 행동 유도 + 프로필 링크 안내

## 산출물
