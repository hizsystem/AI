# Skill 04: plan-visual

> 카피를 기반으로 비주얼 레이아웃 설계

## 호출 에이전트
designer

## 프로세스
1. design-system/tokens.json 참조
2. 카피 길이에 맞는 레이아웃 패턴 선택
3. 색상 테마 결정 (다크/라이트/커스텀)
4. 제품 사진 배치 방식 결정
5. 비주얼 명세서 작성

## 레이아웃 패턴 라이브러리
- **hero-text**: 대형 텍스트 중심 (Before/After 없을 때)
- **split-ba**: 좌우 Before/After (제품 사진 있을 때)
- **card-grid**: 카드 그리드 (성공사례 모음)
- **chart**: 차트/그래프 (ROAS 등 숫자 중심)
- **stat-cards**: 숫자 카드 3개 (바이럴 등)

## 산출물
outputs/{project}/visual-plan.json
