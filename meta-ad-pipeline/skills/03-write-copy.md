# Skill 03: write-copy

> 광고/콘텐츠 카피 2-3안 작성

## 호출 에이전트
copywriter

## 프로세스
1. feedback/feedback.md 읽기 (이전 피드백 반영)
2. 리서치 결과 기반 카피 방향 3가지 설정
   - A안: 문제 해결형 (페인포인트 자극)
   - B안: 성과 증명형 (숫자/사례 중심)
   - C안: 감성 자극형 (스토리텔링)
3. 각 안별 헤드라인 + 서브카피 + CTA 작성
4. 인스타 캡션 + 해시태그 30개

## 규칙
- 헤드라인: 15자 이내, 스크롤 멈춤 유도
- 서브카피: 30자 이내
- CTA: 행동 동사 + 혜택 ("무료 진단받기 →")
- 해시태그: 대형(100만+) 10개 + 중형(1만+) 10개 + 니치 10개

## 산출물
outputs/{project}/copy.json
