3가지 구조 중 하나를 선택해주세요

A

탭 네비게이션 (추천)

상단에 캘린더 | 리포트 | KPI 탭. 한 화면에 하나의 기능에 집중. 심플하고 직관적.

┌─────────────────────────────────────┐
│ 🌱 VEGGIET   [캘린더] [리포트] [KPI]    │
├─────────────────────────────────────┤
│                                     │
│    선택한 탭의 콘텐츠가            │
│    여기에 풀스크린으로 표시        │
│                                     │
└─────────────────────────────────────┘
✓ 가장 심플   ✓ 모바일 대응 쉬움   ✓ 기존 캘린더 재사용 100%
B

사이드바 네비게이션

왼쪽에 사이드바 메뉴 (캘린더, 리포트, KPI, 설정). 오른쪽에 메인 콘텐츠. 확장성 좋음.

┌──────┬──────────────────────────────┐
│ 🌱   │  VEGGIET 콘텐츠 캘린더       │
│──────│──────────────────────────────│
│ 📅   │                              │
│ 📊   │    메인 콘텐츠 영역            │
│ 📈   │                              │
│ ⚙️   │                              │
└──────┴──────────────────────────────┘
✓ 확장성 좋음   △ 모바일에서 사이드바 처리 필요   △ 약간 복잡한 느낌
C

대시보드 홈 + 서브페이지

메인 홈에 KPI 요약 + 이번 주 콘텐츠 미리보기. 각 영역 클릭하면 상세 페이지로 이동.

┌─────────────────────────────────────┐
│ 🌱 VEGGIET Dashboard               │
├──────────────────┬──────────────────┤
│ 팔로워: 8,234    │ 이번주 콘텐츠   │
│ ER: 4.2%         │ ☑ 레시피 피드   │
│ [상세보기→]       │ ☐ 릴스          │
├──────────────────┴──────────────────┤
│ 📋 주간 코치 코멘트                 │
│ "이번 주 릴스 초반 훅이..."        │
└─────────────────────────────────────┘
✓ 한눈에 현황 파악   ✓ 코치 코멘트 눈에 띔   △ 개발 공수 가장 큼

추천: A (탭 네비게이션)

휴닉팀이 사용할 코칭 도구라서 심플함이 가장 중요합니다. 기존 캘린더 컴포넌트를 그대로 재사용하면서 리포트/KPI 탭만 추가하면 빠르게 완성 가능. 나중에 필요하면 C안의 대시보드 홈을 추가할 수 있습니다.
