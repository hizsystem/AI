베지어트/빈커 브랜드 전환 — 어디에 놓을까요?

투트랙 운영을 위한 브랜드 스위처 위치

A

헤더 좌측 브랜드 스위처 (추천)

로고 옆에 드롭다운으로 브랜드 전환. 탭은 공통.

┌─────────────────────────────────────────┐
│ 🌱 VEGGIET ▾   [캘린더] [리포트] [KPI]   │
├─────────────────────────────────────────┤
│  ┌──────────┐                            │
│  │ 🌱 VEGGIET│ ← 클릭하면 드롭다운     │
│  │ 🫘 VINKER │                           │
│  └──────────┘                            │
│                                         │
│    선택한 브랜드의 캘린더 표시            │
└─────────────────────────────────────────┘
✓ 공간 효율적   ✓ 브랜드 컬러로 구분 가능   ✓ 탭 구조 그대로 유지
B

상단 브랜드 탭 + 하위 기능 탭

최상단에 브랜드 탭, 그 아래에 기능 탭. 2단 구조.

┌─────────────────────────────────────────┐
│  [🌱 VEGGIET]   [🫘 VINKER]             │
│─────────────────────────────────────────│
│  [캘린더]  [리포트]  [KPI]                 │
├─────────────────────────────────────────┤
│                                         │
│    콘텐츠 영역                          │
│                                         │
└─────────────────────────────────────────┘
✓ 브랜드 구분 명확   ✓ 전환이 눈에 보임   △ 헤더가 높아짐 (탭 2줄)
C

URL 분리 (/veggiet, /vinker)

완전 별도 페이지. 각 브랜드가 독립적인 대시보드. 네비게이션에서 서로 링크.

/huenic/veggiet         /huenic/vinker
┌──────────────────┐   ┌──────────────────┐
│ 🌱 VEGGIET       │   │ 🫘 VINKER        │
│ ■■■■■ 초록 테마   │   │ ■■■■■ 보라 테마   │
│ 캘린더/리포트/KPI │   │ 캘린더/리포트/KPI │
│ (국내 전략 중심)   │   │ (해외 전략 중심)   │
└──────────────────┘   └──────────────────┘
✓ 브랜드 완전 분리   ✓ 각 브랜드 테마 적용 가능   ✓ 공유 링크가 명확

추천: A (헤더 드롭다운) 또는 C (URL 분리)

A는 가장 심플하고 빠르게 구현 가능. C는 "베지어트 링크", "빈커 링크"를 각각 공유할 수 있어서 실무적으로 편리. 베지어트와 빈커가 톤앤매너/언어가 완전히 다르므로 C가 장기적으로 더 적합할 수 있습니다.
