Account Identity: Name & Bio Direction

Which identity direction feels right for a "brand interpretation photographer"?

LUMIÈRE
STUDIO

A. Lumière Studio

@lumiere.studio (or similar)

Bio: "How brands should feel, not just look.
Visual interpretation — Beauty · Object · Living
Inquiries → DM"

Tone: French-inspired, high-end studio feel. "빛(lumière)"이 사진의 본질을 암시. 글로벌 브랜드 태그 시 자연스러움.

OBJEKT
by [your initial]

B. Objekt

@objekt.studio (or similar)

Bio: "Every object tells a brand story.
Visual direction & still life
Seoul — Open for projects"

Tone: 독일어 "오브젝트". 사물/제품 중심 포지셔닝 명확. 미니멀하고 건조한 감성. 유럽 디자인 커뮤니티 친화.

THE BRAND LENS
VISUAL INTERPRETATION

C. The Brand Lens

@thebrandlens (or similar)

Bio: "Seeing brands differently.
If we shot the campaign — this is how it'd look.
Beauty · Object · Lifestyle"

Tone: 컨셉 자체가 이름. "브랜드를 렌즈로 해석한다"는 걸 직관적으로 전달. 다만 너무 직설적일 수 있음.

STILL FORM
— STUDIO —

D. Still Form

@stillform.studio (or similar)

Bio: "Form follows brand.
Still life with intention.
Collaboration → DM"

Tone: "정물(still life)" + "형태(form)"의 결합. 제품 사진 전문가 느낌 강함. 바우하우스적 절제. "Form follows function"의 변주.

Note: 이름은 출발점이야. 맘에 드는 방향을 고르면 그걸 기반으로 다듬을 수 있어. 또는 아예 다른 키워드가 있으면 말해줘.
