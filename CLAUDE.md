# Naver Place Team — MarketingOS

소상공인(식당, 카페, 바 등) 네이버플레이스 마케팅 전담 워크트리. 초기 세팅, 키워드 최적화, 리뷰 관리, 연계 채널 운영, 다매장 코치형 관리를 담당한다.

## 운영 모델

**코치형 관리**: 직접 실행이 아닌 "주간 미션 → 체크 → 피드백" 루프
- 매장 사장님/스태프가 1차 실행
- 에이전트팀이 미션 설계, 진행 점검, 피드백 제공
- 목표: 매장 10개+ 동시 운영

- 스마트플레이스 등록/최적화 (카테고리, 영업시간)
- [S] 네이버 예약/톡톡/스마트콜 연동 + 알림받기 쿠폰 세팅
- [S] 영수증 리뷰 이벤트 세팅 (가이드 엽서 제작: 방문목적 + 키워드 + 사진 + 별점5)
- [S] 플레이스 광고 세팅 (일 5,000원~, 키워드 태그, 제외 검색어)
- [A] 대표 사진 10장 + 5~10초 영상 촬영 → 소식에 업로드 (체류시간↑)
- [A] 대표 키워드 세팅 (지역명 필수 + 경쟁사 소스코드/마피아넷/자동완성 활용)
- [B] AI 브리핑 노출 켜기
- 상세설명 1,500자 이상 작성 ("누구에게+공감포인트" 공식)

### Week 2: A등급 세팅 + 체류시간 최적화
- [A] 블로그 체험단 섭외 (4개 지표 기준, 목적형 키워드 지정, 띄어쓰기 방식 전달)
- [A] 소식 & 이벤트 발행 (쿠폰 연동, 이벤트 이미지)

- 유입 키워드 통계 확인 → 대표 키워드 교체 판단
- **새로오픈 기능 켜기** (Week 1 세팅 완료 확인 후)

### Week 3: 리뷰 마케팅 + 외부 채널

## 최적화 3P 전략 (전자책 기반)

매출 = 유입량 x 방문율 x 객단가. 체류시간이 모든 것의 열쇠.

### 1. 초기 세팅 (Pin)

**매장이름 전략:**
- 매장이름에 지역이름을 넣으면 해당 지역 검색 노출에 유리
- 단, 키워드 스터핑은 페널티 대상 (예: "강남맛집 OO식당" 금지)

**업종선정 전략:**
- 세분화된 업종이 경쟁이 적어 유리 (일식당 < 초밥, 한식 < 한정식)
- 업종 변경 시 해당 키워드 검색량과 점유율을 반드시 분석
- 같은 업종이라도 키워드에 따라 노출 비율이 크게 다름

**새로오픈 전략 (강의 기반 수정):**
- 새로 오픈한 매장에 알고리즘 가점 (새로오픈 배지)
- **핵심: 새로오픈 기능은 세팅 완료 후 최대한 늦게 켜기** (강의 강조)
- 플레이스 세팅/사진/키워드/리뷰이벤트/광고 전부 준비 완료 → 그 다음 새로오픈 공개
- 마케팅팀에서 조율 후 한 달 뒤 오픈 기능 켜는 방식도 유효
- 폐업 후 재오픈이 이전보다 나은 경우도 있음 (새 사업자등록증 필요)
- 새로오픈 90일간 가점이 있으므로, 켜는 시점을 전략적으로 결정

**오픈 전 필수 세팅 (새로오픈 켜기 전에 완료할 것):**
- 알림받기 쿠폰 세팅
- 영수증 리뷰 이벤트 (가이드 엽서/카드 제작, 방문 목적 4가지 + 키워드 포함)
- 대표 이미지 + 5~10초 영상 세팅
- 네이버예약 활성화 → 알림받기 연동
- 네이버톡톡 활성화 → 스마트콜 050번호 발급
- 플레이스 광고 세팅 (일 5,000원~)
- 블로그 체험단 1차 섭외 완료
- 인스타그램/블로그 연동 준비
- AI 브리핑 노출 켜기

### 2. 체류시간 전략 (Core)

체류시간 ↑ → 예약율 ↑ → 상위노출 (최적화 후 예약율 100% 증가, 체류시간 1200% 증가 사례)

**썸네일에 올인:**
- 대표사진이 클릭을 결정. 배경 컬러를 차별화하면 눈에 띔
- 음식: 5-6장 이상 통일감 있게, 테두리도 효과적
- 대표님/직원 프로필 사진 활용 → 신뢰도 상승
- 전문가 느낌의 프로필 사진은 허들을 낮춤

**10초만 머물게 하는 3가지:**
1. 영상: 매장 분위기, 조리 과정, 사장님 인사 (10초 영상 → 자동 클립 등록)
2. 쿠폰: "100%"와 "공짜" 단어가 시선을 사로잡음. 플레이스쿠폰 → 알림받기 유도 → 재방문율 상승
3. 소식 이미지: 이벤트/계절감 담은 이미지, 공지사항으로 가장 처음 보여짐

**상세설명 글쓰기 공식:**
1. "누구에게" + "공감포인트"로 시작 (고객 주목 끌기)
2. 고객의 문제를 우리가 해결해준다고 설명
3. 어떻게 해결해줄지 구체적으로 (1,2,3,4,5 목록화)
4. 다른 정보(이벤트, 리뷰, 영상, 가격)로 유도
- 리뷰 활용: 실제 고객 리뷰에서 키워드/감성을 가져와 상세설명에 반영

### 3. 대표키워드 황금전략 (Key)

**핵심 원리**: 네이버가 이미 아는 키워드(메뉴명)는 삽입 불필요. **네이버가 모르는 키워드**를 대표키워드에 삽입해야 추가 노출.

**황금키워드 찾는 6단계:**
1. 매장 관련 키워드를 찾는다
2. 키워드로 지역명과 함께 검색해서 매장이 노출되는지 확인
3. (나온다면 시스템이 위치 정보로 이미 인식한 것 → 삽입 불필요)
4. 매장이 노출되지 않는 키워드를 검색해본다
5. 미노출인 키워드를 대표키워드에 삽입하고 노출 확인
6. 노출되면 독점, 안 되면 다른 키워드를 반복 시도

**업종별 황금키워드 (음식점/카페/바):**
- 상황: 점심, 저녁, 회식장소, 단체회식, 데이트코스, 가족모임, 연말모임, 생일
- 분위기: 핫플, 포토존, 실내데이트, 드라이브코스
- 특수: 현지인, 직장인점심, 점심특선
- 주변 관광지/명소를 대표키워드에 넣기 (예: 장미공원 근처 카페)
- 계절: 크리스마스, 봄꽃놀이 등
- 대표키워드 최대 5개, 입력 후 실제 노출 확인 필수

### 4. 리뷰 마케팅 전략 (Star)

**리뷰 = 마케팅 공간** (평가가 아닌 마케팅 도구로 활용)

**리뷰 활용 3단계:**
1. 타겟고객 정하기 — 리뷰마다 내가 궁금해하는 부분에 시선이 감 (커플 vs 가족 vs 직장인)
2. 주제 정하고 내용 정리 — 한 번에 하나씩만 홍보
3. 티내지 않기 — 욕심내서 정보 나열하면 불편. 자연스럽게

**답글 쓰는 공식:**
1. **감사표시**: 구체적으로 칭찬 (예: "정성껏 찍어주셨네요" vs "감사합니다")
2. **친근감 표시**: "나는 고객님을 기억하고 있습니다" 느낌 (예: "OO 좋아하셨죠")
3. **마케팅 (티내지 않고)**:
   - (1) 타겟고객이 중요하게 생각하는 부분 어필
   - (2) 구체적으로 얻을 수 있는 이익 어필
   - (3) 리뷰를 보완해서 추가 정보 노출

**컨셉 전략:**
- 나만의 컨셉을 만들면 타겟고객이 모인다
- 컨셉은 검색어와 연결: "디저트카페" → "디카페인디저트", "케이크아이스크림" 등
- 리뷰/블로그/SNS를 통해 컨셉을 강화
- 장기투숙 고객 = 이벤트/쿠폰으로 재방문 유도

## 세팅 우선순위 (S/A/B 등급 — 2026.03 강의 기반)

### S등급 — 반드시 먼저 (매출 직결)
1. **영수증 리뷰 & 이벤트**: 키워드 포함 리뷰가 쌓여야 상위 노출. 4월 6일부터 별점 제도 부활 → 5점 유도 필수
2. **플레이스 광고**: 일 예산 5,000원~3만원. 광고 자체보다 유입 고객의 체류시간/클릭이 지수를 높임
3. **예약 & 쿠폰**: 켜두는 것 자체가 지수에 영향. 예약 매장 별도 노출 혜택

### A등급 — 필수 (노출 순위 결정)
4. **대표 사진**: 영상 소식 업로드 시 메인에 영상 노출 → 체류시간 증가
5. **대표 키워드**: 알고리즘 봇이 읽는 메타태그. 지역명 필수. 유입 키워드 통계로 교체 판단
6. **소식 & 이벤트**: 2026년부터 지수 상승 효과 증가
7. **블로그 체험단**: 4개 지표(시작연도, 발행수, 일일방문자, 이웃수) 중 3개 이상 충족 기준으로 섭외

### B등급 — 있으면 좋은 것
8. **상세 소개문**: 1,500자 이상 (최대 2,000자), 키워드 밀도 높게
9. **AI 브리핑**: 노출 켜기. 격주 단위 데이터 수집
10. **주차 & 편의시설**: 타겟 고객에게 유효한 정보

### 절대 금지 (페널티)
- 길찾기 트래픽 조작, AI 블로그 대량 배포, 저장하기 트래픽 작업

### 보너스 — 매출 부스터
- **톡톡 무료 메시지**: 매월 1일, 알림받기+예약 방문 고객 최대 ~2,900명에게 무료 발송
- **외부 플랫폼 → 플레이스 유입**: 인스타/당근 광고 랜딩을 플레이스 링크로 → N2 상승 (네이버가 선호)
- **네이버 커넥트(포스)**: 영수증 없이 카드 결제 후 즉시 리뷰 가능. 별도 표시/검색 노출
- **자사 블로그 연동**: 스튜디오, 네일, 미용실 등에 효과적

## 알고리즘 — 히든 지수 N1/N2/N3 (2026.03 강의)

에드로그(Adlog) 툴에서 측정 가능. N1 + N2 = 원인 → N3 = 결과.

| 지수 | 이름 | 측정 기준 | 진단 |
|------|------|---------|------|
| N1 | 유사도 | 대표키워드 ↔ 영수증리뷰 ↔ 블로거키워드 ↔ 소개/메뉴 키워드 일치도 | 낮으면 → 키워드 정합성 개선 |
| N2 | 관련성 | 리뷰수, 예약수, 저장수, 외부 유입, 체류시간, 클릭수 | 낮으면 → 광고/소식/외부채널 강화 |
| N3 | 랭킹 | 해당 키워드 플레이스 순위 (0.5 이상 = 상위 노출 가능권) | 결과값 |

### 에드로그 툴
- 월 2만원, 본사 계정 1개로 전 지점 관리 가능
- 키워드별 순위 트래킹, 히든 지수 분석, 경쟁사 비교, 체류시간 확인
- 등록 시 네이버 모바일 버전 주소창 링크 사용

### 키워드 발굴 3가지 방법 (강의)
1. **경쟁사 소스코드**: 경쟁사 플레이스 → 우클릭 → 프레임 소스보기 → "keyword" 검색
2. **마피아넷 (무료)**: 키워드 분석 → 연관 키워드 30개 → 적합한 것 선택
3. **자동완성 키워드**: 네이버 검색창 자동완성 = 고객 궁금증 → 소개글/체험단 원고에 답 반영

### 리뷰 답글 4요소 (답글도 알고리즘이 읽음)
1. 핵심 키워드 언급
2. 방문 목적 언급 (데이트, 회식, 가족 등)
3. 주차 및 편의시설 PR
4. 매장 이용 꿀팁

### 영수증 리뷰 유도 4단계
1. 방문 목적 적기 ("OO하러 왔어요")
2. 느낀 점 솔직하고 구체적으로
3. 추천 / 상위노출 원하는 내용
4. 별점 5점 (2026.04.06부터 별점 부활)

### 최신 변경사항 (2026)
- 4월 6일: 별점 제도 부활
- 예약 가능 매장 별도 노출로 변경

## 핵심 KPI

### 매장별 추적 지표
- **히든 지수**: N1(유사도), N2(관련성), N3(랭킹) — 에드로그 기준
- **검색 순위**: 주요 키워드 3개 기준 순위 변동 (에드로그 트래킹)
- **리뷰**: 월 신규 리뷰 수 (영수증 + 블로그) + 별점 평균
- **저장 수**: 월 신규 저장 수
- **노출 수**: 스마트플레이스 통계 기준
- **전환**: 전화 수, 길찾기 수, 예약 수, 주문 수
- **광고**: 일 예산 대비 클릭수, 전환 키워드, A/B 테스트 결과

- 리뷰 증가율 상위/하위 매장
