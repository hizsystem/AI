## 1. 현재 계정 분석

### 브랜드 기본 정보
- **제품**: 100% 식물성(Plant-Based) 프로틴 쉐이크
- **라인업**: 멀티그레인, 검은콩(Black Bean), 말차(Matcha) 등
- **유통**: 올리브영 입점
- **바이오**: "지속 가능한 식단 관리의 시작 베지어트"
- **하이라이트**: recipe / review / Q&A / new / brand

### 현재 피드 톤 진단

| 요소 | 현재 상태 | 문제점 |
|------|----------|--------|
| **컬러** | 그린 + 베이지 + 어스톤 일색 | "건강식품 매대" 느낌. 차별화 없음 |
| **피사체** | 제품 컷 + 원재료 클로즈업 + 자연 풍경 | 사람이 거의 없음 → 감정 연결 부재 |
| **카피** | "Good Anywhere" / "For Your Wellness" / "wellness away" | 웰니스 키워드 과다. 라이프스타일 요소 없음 |
| **레이아웃** | 제품 중심 플랫레이, 자연 배경 | 포맷 다양성 부족 (카드뉴스, 텍스트 게시물 등 거의 없음) |
| **시리즈** | 명확한 시리즈 구분 없음 | 피드 스크롤 시 어떤 콘텐츠인지 구분 안 됨 |

### 잘 되고 있는 점 (살릴 것)
- ✅ "Good Anywhere" 컨셉 자체는 좋음 (장소 불문 라이프스타일)
- ✅ 제품 촬영 퀄리티는 준수
- ✅ 올리브영 입점 → 신뢰도 소재 있음
- ✅ 하이라이트 카테고리가 이미 있음 (recipe, review 등)
- ✅ 로고/BI 작업은 완료된 상태 (하단 로고 변형 컬러 다양)

### 문제 요약
> **"좋은 제품인데, 누가 왜 먹는지가 안 보인다"**
>
> 제품과 원재료는 많이 보여주는데,
> 이걸 마시는 **사람의 일상**이 안 보인다.
> 그래서 "건강기능식품"처럼 느껴지고, "라이프스타일 브랜드"로 안 느껴진다.

## 2. 채널 컨셉 제안

### 컨셉 스테이트먼트

> **"바쁜 하루 사이, 나를 위한 한 끼의 여유"**
>
> 베지어트는 건강식품이 아니라,
> 바쁜 일상 속에서 자기 관리를 포기하지 않는 사람들의 루틴이다.

### 무드 키워드 5개

| # | 키워드 | 설명 |
|---|--------|------|
| 1 | **Effortless** | 힘 안 들이고 자연스러운. 억지 건강이 아닌 가벼운 루틴 |
| 2 | **Warm Minimal** | 차가운 미니멀이 아닌, 따뜻한 톤의 깔끔함 |
| 3 | **Daily Ritual** | 거창한 건강관리가 아닌, 매일의 작은 습관 |
| 4 | **Real People** | 스톡 이미지 같은 모델이 아닌, 실제 일상 속 사람 |
| 5 | **Urban Nature** | 산속이 아닌, 도시 생활 속의 자연스러움 |

### 컬러 방향

**현재 → 제안:**

| 현재 | 제안 | 이유 |
|------|------|------|
| 그린 일색 | **웜 베이지** (메인) | 라이프스타일 온기 |
| 어스톤 | **소프트 그린** (서브) | 식물성 아이덴티티 유지하되 톤 다운 |
| 없음 | **테라코타 or 머스타드** (포인트) | 피드에 리듬감. 웰니스 탈피 |

```
메인:  #E8DDD3 (웜 베이지)
서브:  #B5C4A1 (소프트 세이지 그린)
포인트: #C4703F (테라코타) 또는 #D4A843 (머스타드)
텍스트: #2C2C2C (차콜)
배경:  #F5F1EC (오프 화이트)
```

### 톤앤매너 가이드

| 항목 | 가이드 |
|------|--------|
| **말투** | 친구에게 추천하듯 편하게. "~해보세요"보다 "~하는 거야" |
| **사진 톤** | 자연광, 약간 따뜻한 톤. 과도한 보정 없이 일상적으로 |
| **텍스트** | 짧고 직관적. 한 게시물에 핵심 메시지 1개만 |

### DO / DON'T

| DO ✅ | DON'T ❌ |
|-------|---------|
| 사람이 등장하는 일상 장면 | 제품만 단독으로 놓인 스튜디오 컷 반복 |
| 출근길, 점심시간, 운동 후 등 상황 중심 | 산, 계곡, 숲 등 자연 풍경 배경 |
| "간편하다, 맛있다, 습관이다" 메시지 | "건강하다, 영양소, 단백질 몇g" 스펙 나열 |
| 카드뉴스, 릴스, 캐러셀 등 포맷 다양화 | 제품 플랫레이만 반복 |
| 올리브영 입점 사실을 자연스럽게 노출 | 세일/할인 중심의 프로모션 톤 |
| 유저 리뷰/후기를 콘텐츠로 활용 | 브랜드가 일방적으로 말하는 톤 |
