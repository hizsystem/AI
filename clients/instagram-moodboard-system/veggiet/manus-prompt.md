# Manus 프롬프트 — 베지어트 인스타그램 레퍼런스 수집

아래 프롬프트를 Manus에 그대로 복사-붙여넣기 하세요.

---

## 프롬프트

```
베지어트(veggiet)라는 한국 식물성 프로틴 브랜드의 인스타그램 무드보드를 만들고 있어.
현재 방향은 "웰니스 브랜드"에서 "라이프스타일 브랜드"로 전환하는 것이야.

채널 컨셉: "바쁜 하루 사이, 나를 위한 한 끼의 여유"
무드 키워드: Effortless, Warm Minimal, Daily Ritual, Real People, Urban Nature
컬러 방향: 웜 베이지 + 소프트 그린 + 테라코타

아래 6개 콘텐츠 시리즈별로 인스타그램 레퍼런스를 수집해줘.

### 시리즈 1: 「오늘의 한 끼」
- 사람이 일상 공간(주방, 책상, 카페)에서 음료/식품을 마시거나 먹는 장면
- 자연광, 따뜻한 톤, 자연스러운 구도
- 찾을 계정 방향: 라이프스타일 F&B 브랜드, 프로틴/쉐이크 브랜드
- 5-10개 게시물

### 시리즈 2: 「루틴 레시피」
- 탑뷰(위에서 아래) 앵글의 레시피 릴스
- 미니멀한 배경, 핸드모션, 3스텝 이내 구성
- 찾을 방향: 음료/스무디/프로틴 레시피 릴스
- 5개 게시물 또는 릴스

### 시리즈 3: 「한 장 정리」
- 깔끔한 카드뉴스(캐러셀) 디자인
- 웜톤 배경, 미니멀 타이포, 정보형 콘텐츠
- 찾을 방향: 건강/식품/라이프스타일 브랜드의 인포그래픽 게시물
- 5개 게시물

### 시리즈 4: 「솔직 리뷰」
- 유저 생성 콘텐츠(UGC)를 브랜드 콘텐츠로 가공한 사례
- 유저 사진 + 브랜드 프레임/템플릿 조합
- 찾을 방향: F&B 브랜드의 리뷰/후기 콘텐츠
- 3-5개 게시물

### 시리즈 5: 「베지어트 TMI」
- 브랜드 비하인드 스토리 (생산, 개발, 팀 일상 등)
- 캐주얼하고 인간적인 톤
- 찾을 방향: D2C 브랜드의 behind-the-scenes 콘텐츠
- 3-5개 게시물

### 시리즈 6: 「Good Anywhere」
- 다양한 도시 장소(회의실, 카페, 공항, 공원 등)에서 제품이 자연스럽게 놓인 장면
- 장소가 주인공, 제품은 자연스럽게 존재
- 찾을 방향: "product in context" 스타일의 라이프스타일 콘텐츠
- 3-5개 게시물

---

각 시리즈별로 다음 형식으로 정리해줘:

1. 게시물 URL
2. 스크린샷
3. 왜 이 게시물이 레퍼런스로 적합한지 (1-2줄)
4. 비주얼 특징 (컬러, 구도, 톤)

마지막에 전체적인 비주얼 패턴 분석도 해줘:
- 공통 컬러 팔레트
- 가장 많이 사용되는 레이아웃
- 성공하는 콘텐츠의 공통점
```

## Manus 결과물 활용법

Manus가 리포트를 만들어주면:
1. 수집된 게시물 URL/스크린샷 → HTML 무드보드 비주얼 무드 섹션에 반영
2. 비주얼 패턴 분석 → 촬영/디자인 가이드에 반영
3. 시리즈별 레퍼런스 → 피그마 무드보드 Page 4~N에 배치
