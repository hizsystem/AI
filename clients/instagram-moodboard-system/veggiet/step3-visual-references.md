## 2. 시리즈별 비주얼 레퍼런스 가이드

### 시리즈 A: 「오늘의 한 끼」 — 채널의 메인 엔진

**비주얼 공식: 사람 + 일상 공간 + 제품 (자연스럽게)**

| 요소 | 가이드 |
|------|--------|
| 조명 | 자연광 필수. 창가, 아침 빛 선호 |
| 공간 | 주방, 책상, 카페, 헬스장 로비 등 실제 공간 |
| 사람 | 전신보다 부분 (손, 반신). 얼굴 없어도 됨 |
| 제품 | 프레임의 1/3 이하. 주인공은 "상황" |
| 색감 | 웜 톤. 약간 노출 오버. 과보정 금지 |

**레퍼런스 수집 키워드** (Pinterest/인스타에서 검색):
```
- "morning routine protein shake aesthetic"
- "desk lunch minimal photography"
- "post workout shake lifestyle"
- "food in context lifestyle photography"
- "korean cafe interior aesthetic"
```

**Chamberlain Coffee에서 참고할 구체적 포인트:**
- 소파에서 커피 마시는 장면 → 소파에서 쉐이크 마시는 장면
- 차에서 커피 들고 있는 손 → 출근길에 쉐이크 들고 있는 손
- 주방에서 만드는 과정 → 주방에서 쉐이크 만드는 과정

---

### 시리즈 B: 「루틴 레시피」 — 릴스

**비주얼 공식: 탑뷰 + 핸드모션 + 3스텝 이내**

| 요소 | 가이드 |
|------|--------|
| 앵글 | 탑뷰(위에서 아래) 메인. 간혹 45도 |
| 배경 | 대리석, 우드, 리넨 — 1가지로 통일 |
| 동작 | 붓기 → 섞기 → 완성. 3스텝 이내 |
| 시간 | 15-30초 최대 |
| 사운드 | ASMR (붓는 소리, 섞는 소리) or 트렌딩 오디오 |

**레퍼런스 수집 키워드:**
```
- "top view recipe reel minimal"
- "protein shake recipe aesthetic reel"
- "3 step recipe instagram reel"
- "food asmr reel aesthetic"
```

**2025-2026 릴스 트렌드 반영:**
- "3스텝 레시피"가 가장 높은 저장율 (인지심리학적으로 3단계가 기억 한계)
- 과도한 편집보다 자연스럽고 진짜 같은 느낌이 알고리즘에 유리
- 변환(transformation) 과정을 보여주는 것이 핵심

---

### 시리즈 C: 「한 장 정리」 — 카드뉴스

**비주얼 공식: 웜 배경 + 깔끔 타이포 + 핵심 1개**

| 요소 | 가이드 |
|------|--------|
| 배경색 | #F5F1EC (오프화이트) 또는 #E8DDD3 (웜베이지) |
| 텍스트 | #2C2C2C (차콜). 볼드 산세리프 (Pretendard 등) |
| 포인트 | #B5C4A1 (소프트 그린) 또는 #C4703F (테라코타) |
| 장수 | 4-6장. 커버 + 본문 3-4장 + CTA 1장 |
| 커버 | 한 줄 질문형 ("아침 거르는 사람, 대신 뭐 해?") |

**카드뉴스 레이아웃 구조:**
```
[1] 커버: 질문형 헤드라인 (큰 글씨, 배경 이미지 or 단색)
[2] 본문 1: 문제 제기 / 공감
[3] 본문 2: 핵심 정보 (리스트 or 비교표)
[4] 본문 3: 실전 팁 or 추천
[5] CTA: "저장해두고 내일 해봐" + 팔로우 유도
```

**레퍼런스 수집 키워드:**
```
- "instagram carousel infographic minimal warm"
- "korean instagram card news design"
- "wellness brand carousel post design"
- "instagram slide post typography clean"
```

**Ritual에서 참고할 구체적 포인트:**
- 성분 설명을 일상 언어 + 깔끔한 인포그래픽으로
- 과학적 팩트를 "소화 가능한 크기"로 쪼개서 전달

### 시리즈 D: 「솔직 리뷰」 — UGC 활용

**비주얼 공식: 유저 사진(있는 그대로) + 브랜드 프레임**

| 요소 | 가이드 |
|------|--------|
| 유저 사진 | 보정 최소화. 리얼함이 핵심 |
| 프레임 | 브랜드 컬러 보더 or 하단 텍스트 밴드 |
| 텍스트 | 유저 코멘트 직접 인용. 따옴표 활용 |
| 포맷 | 캐러셀: 유저 사진 → 리뷰 텍스트 → 제품 링크 |

**레퍼런스 수집 키워드:**
```
- "ugc content template instagram brand"
- "customer review post design instagram"
- "user generated content food brand"
```

### 시리즈 E: 「베지어트 TMI」 — 브랜드 비하인드

**비주얼 공식: 캐주얼 사진 + 이야기체 캡션**

| 요소 | 가이드 |
|------|--------|
| 사진 | 폰카 느낌 OK. 일부러 "완벽하지 않게" |
| 내용 | 사무실, 생산현장, 회의, 패키지 시안 등 |
| 캡션 | 길어도 OK. 이야기 중심. "사실은~" 톤 |
| 포맷 | 싱글 or 캐러셀 2-3장 |

**레퍼런스 수집 키워드:**
```
- "brand behind the scenes instagram"
- "founder story post instagram"
- "small brand bts content"
```

### 시리즈 F: 「Good Anywhere」 — 기존 컨셉 리뉴얼

**비주얼 공식: 다양한 도시 장소 + 제품 자연스럽게 배치**

| 요소 | 가이드 |
|------|--------|
| 장소 | 회의실, 카페, 비행기, 공원, 서점 등 도시 일상 |
| 구도 | 장소가 주인공. 제품은 자연스럽게 존재 |
| 텍스트 | "Good Anywhere." 하단에 작게 |
| 색감 | 각 장소의 자연스러운 색감 유지 |

**현재 vs 리뉴얼:**
```
현재: 산, 계곡, 숲 배경 → 등산/야외 브랜드 같음
리뉴얼: 회의실, 카페, 비행기 → 도시인의 일상 브랜드
```

**레퍼런스 수집 키워드:**
```
- "product in context urban lifestyle"
- "coffee brand city lifestyle photography"
- "good anywhere brand campaign"
```
