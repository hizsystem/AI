# 브랜드 & 마케팅 전략

> (✦) 브랜드 전략가의 Why-Who-What-How-Where 프레임워크로 작성
> 작성일: 2026-02-24 | 클라이언트: 탭샵바 (TAP & SHOP & BAR)

---

## 전략 요약 (Executive Summary)

탭샵바는 "와인 바"가 아니다. **일상에서 와인을 만나는 가장 쉬운 방법**이다.

커피가 1990년대 한국에서 "특별한 음료"에서 "매일의 루틴"으로 바뀌었듯, 탭샵바는 와인을 "격식의 술"에서 "퇴근 후 아메리카노 대신 고르는 한 잔"으로 바꾸는 브랜드다. 경쟁 상대는 다른 와인바가 아니라 스타벅스 앞에서 "오늘 뭐 마시지?" 고민하는 그 순간이다.

이 전략의 핵심은 **세 가지 포기**에서 시작한다:
- 와인 전문가를 위한 브랜드라는 포지션을 **포기**한다
- 고급스럽고 격식 있는 톤을 **포기**한다
- "모든 연령대를 위한"이라는 욕심을 **포기**한다

좋은 브랜드는 선택과 포기의 결과이며, 탭샵바는 "와인을 처음 좋아하게 되는 곳"이라는 하나의 약속에 집중한다.

## 1. Why - 존재 이유

### 미션

**"와인과 사람 사이의 모든 벽을 허문다"**

가격의 벽 (보틀 단위 → 잔 단위 수천 원), 지식의 벽 (라벨 해독 → 직접 맛보고 선택), 분위기의 벽 (격식 → 캐주얼)을 동시에 제거한다.

### 비전

**"와인이 아메리카노만큼 자연스러운 일상이 되는 세상"**

3년 후, "퇴근하고 탭샵바 갈래?"가 "커피 한잔할래?"만큼 자연스러운 말이 되는 것. 이것이 탭샵바가 그리는 미래다.

### 핵심 가치

| 가치 | 의미 | 브랜드에서의 발현 |
|------|------|------------------|
| **Accessible (접근성)** | 누구나, 부담 없이, 바로 | 잔 단위 수천 원 시음, 설명 없이 탭을 눌러 맛보는 경험, 아울렛가 보틀 구매 |
| **Playful (재미)** | 탐험하듯, 놀이하듯 | 80종 셀프 탭 앞에서 "이것도 저것도" 시도하는 놀이, 와인 놀이터 |
| **Honest (솔직함)** | 있는 그대로, 꾸밈없이 | "몰라도 괜찮아"라는 태도, 비싼 와인이 좋은 게 아니라 내 입에 맞는 와인이 좋은 것 |

**가치의 우선순위**: Accessible > Playful > Honest
접근성이 무너지면 나머지 두 가치가 성립하지 않는다. 가격이 비싸지거나, 분위기가 어려워지거나, 매장에 가기 힘들어지면 탭샵바가 아니다.

### 브랜드 Why

> "한국에서 와인은 아직도 '특별한 날의 어려운 술'이다. 우리는 이것이 부당하다고 생각한다. 와인은 원래 유럽에서 물처럼 마시던 일상의 음료다. 그 일상성을 한국에 되돌려놓는 것, 그래서 누군가의 '두 번째 아메리카노'가 되는 것. 이것이 탭샵바가 존재하는 이유다."

**왜 이 Why가 강력한가:**
1. **적(enemy)이 명확하다** -- 와인을 어렵게 만드는 모든 관행과 싸운다
2. **고객 편에 서 있다** -- "몰라도 괜찮아, 여기서 시작하면 돼"
3. **경쟁사가 카피할 수 없다** -- 셀프 탭 80종 + 아울렛가 1,000종이라는 물리적 인프라가 뒷받침한다
4. **확장 가능하다** -- "와인의 일상화"라는 미션은 매장 확대, 리테일 확장, 커뮤니티 구축 모두를 품는다

### 타겟 우선순위

```
[1순위] 탐험가 지은 ─── 와인 입문자, 호기심 높은 2030 여성
         → 브랜드의 핵심 전도사(evangelist). 이들이 움직이면 나머지가 따라온다.
         → 콘텐츠의 주인공, SNS 확산의 엔진

[2순위] 여유파 민준 ─── 퇴근 후 루틴을 찾는 2030 직장인
         → 재방문율(리텐션)의 핵심. 주 1-2회 방문하는 단골이 매출을 만든다.
         → "두 번째 아메리카노" 메시지의 직접 타겟

[3순위] 데이트 커플 ─── 새로운 경험을 원하는 2030 커플
         → 주말 매출과 SNS 바이럴의 동력.
         → 비주얼 콘텐츠의 소재, 첫 방문 유입 채널
```

**의도적으로 배제하는 타겟:**
- 와인 전문가/애호가 (이들의 기대 수준은 탭샵바의 Why와 충돌한다)
- 40대 이상 접대/비즈니스 목적 (톤앤매너가 달라진다)
- 최저가만 찾는 가격 민감 고객 (가치가 아닌 가격으로 연결되면 브랜드가 약해진다)

## 3. What - 가치 제안

### 포지셔닝 스테이트먼트

> **퇴근 후 "오늘 뭐 마시지?" 고민하는 2030 직장인에게,
> 탭샵바는 아메리카노 대신 고를 수 있는 일상의 와인 한 잔이다.
> 80종 셀프 탭으로 몇 천 원부터 내 취향을 직접 찾을 수 있으니까.**

**포지셔닝 맵 (경쟁 구도)**

```
            고급/격식
               |
               |
     소믈리에       프렌치 와인바
     와인바들        (레스토랑형)
               |
               |
 일상적 ───────┼─────── 특별한 날
               |
          ★탭샵바★
               |        이마트 와인장터
     편의점      |       (연 1회 행사)
     와인        |
               |
            캐주얼/접근성
```

탭샵바의 고유한 영역: **"캐주얼하면서 일상적"** -- 이 사분면에 자리 잡은 와인 브랜드는 한국 시장에 아직 없다. 편의점 와인은 일상적이지만 경험이 없고, 와인바는 경험이 있지만 일상적이지 않다. 탭샵바는 이 빈 공간을 차지한다.

### 핵심 메시지

**태그라인 (주력 안):**

> **"오늘의 한 잔, 탭에서 시작"**

- "오늘의" -- 특별한 날이 아닌 매일의 일상성
- "한 잔" -- 보틀이 아닌 잔 단위의 접근성
- "탭에서 시작" -- 셀프 탭이라는 차별화된 경험, 와인 여정의 시작점

**보조 태그라인:**
- "두 번째 아메리카노" -- 상황 설명용, 광고 캠페인용
- "와인 놀이터" -- 매장 내 경험 설명용, 셀프 탭 존 네이밍
- "마셔보면 알아" -- CTA(행동 유도)용, 퍼포먼스 광고용

### 메시지 계층구조

메시지는 3개 층위로 구성한다. 상위 메시지가 흔들리면 하위도 무너지므로, 일관성이 핵심이다.

```
┌─────────────────────────────────────────────────────┐
│  Level 1: 브랜드 에센스 (한 단어)                       │
│  → "일상"                                            │
│  고객이 탭샵바를 한 단어로 떠올릴 때 이것이어야 한다         │
├─────────────────────────────────────────────────────┤
│  Level 2: 핵심 약속 (한 문장)                           │
│  → "와인을 가장 쉽고 재미있게 만나는 곳"                   │
│  모든 커뮤니케이션의 기준점                               │
├─────────────────────────────────────────────────────┤
│  Level 3: 증거 메시지 (3가지 기둥)                       │
│                                                     │
│  기둥 1: 접근성     기둥 2: 재미       기둥 3: 솔직함     │
│  "잔 단위 수천 원"  "80종을 직접 탭"   "몰라도 괜찮아"    │
│  "아울렛가 보틀"    "내 취향 발견"     "있는 그대로"      │
│  "동네 가까이"      "와인 놀이터"      "가격도 솔직하게"   │
└─────────────────────────────────────────────────────┘
```

**메시지 DO & DON'T:**

| DO (이렇게 말한다) | DON'T (이렇게 말하지 않는다) |
|---|---|
| "마셔보면 알아" | "소믈리에가 엄선한" |
| "오늘 뭐 마실래?" | "오늘의 추천 빈티지" |
| "잔 단위로 부담 없이" | "프리미엄 와인을 합리적 가격에" |
| "몰라도 괜찮아" | "와인의 세계로 초대합니다" |
| "퇴근하고 한 잔" | "특별한 날을 위한" |
| "와인 놀이터" | "와인 감정 공간" |
| "아메리카노 대신" | "격조 있는 한 잔" |

## 4. How - 전달 방식

### 톤앤매너

**브랜드 보이스: "와인 잘 아는 재미있는 친구"**

이 친구는 와인을 많이 알지만 절대 잘난 척하지 않는다. "이거 마셔봐, 진짜 맛있어"라고 말하지, "이 와인의 떼루아가..."라고 말하지 않는다. 전문성은 쉬운 말 뒤에 숨어 있다.

**톤 스펙트럼:**

```
격식 ──────────────────────────────── 캐주얼
                              ★ [여기]

진지 ──────────────────────────────── 유머
                         ★ [여기]

전문적 ────────────────────────────── 친근
                              ★ [여기]

권위적 ────────────────────────────── 동등
                                ★ [여기]
```

**톤앤매너 가이드:**

| 원칙 | 설명 | 예시 |
|------|------|------|
| **반말 톤** | SNS에서는 반말 기본. 격식 없는 친구처럼 | "이거 진짜 맛있어" (O) / "추천드립니다" (X) |
| **쉬운 말** | 와인 용어는 최소화, 쓸 때는 반드시 번역 | "산미(새콤한 맛)가 좋아서 치킨이랑 찰떡" |
| **짧은 문장** | 한 문장에 한 메시지. 길어지면 잘라 | 15자 이내 권장 |
| **감탄보다 공감** | "대단하죠!"보다 "그치?" | "첫 잔이 제일 떨리지. 근데 여긴 괜찮아" |
| **행동 유도** | 모든 콘텐츠는 "그래서 뭐 하라고?"가 명확 | "오늘 퇴근하고 탭 한번 눌러봐" |

**금지어 목록:**
- 엄선, 프리미엄, 럭셔리, 고급, 격조
- 소믈리에 추천, 와인 전문가, 빈티지 셀렉션
- ~해드립니다, ~모시겠습니다, 고객님
- 최고의, 최상의, 완벽한

### 비주얼 방향

**비주얼 원칙: "편의점의 접근성 + 카페의 감성"**

| 요소 | 방향 | 레퍼런스 느낌 |
|------|------|-------------|
| **컬러** | 메인 - 와인 레드 계열 (딥 버건디 아닌 밝은 코랄 레드), 서브 - 웜 베이지, 포인트 - 차콜 | 무인양품의 따뜻함 + 스타벅스 리저브의 감성 |
| **서체** | 한글 - 둥글고 친근한 고딕 (프리텐다드, 스포카 한 산스 등), 영문 - 소문자 중심 라운드 산세리프 | 배달의민족 한나체의 친근함 (그러나 더 절제된) |
| **사진 톤** | 자연광, 따뜻한 색감, 사람이 있는 장면 중심, 손에 잔이 들린 컷 | 와인 자체보다 "와인을 즐기는 사람"이 주인공 |
| **일러스트** | 심플한 라인 드로잉, 와인 관련 아이콘, 귀여운 캐릭터 요소 | 과하지 않은 포인트 일러스트 |
| **영상 톤** | 짧고 리듬감 있게 (15-30초), 탭 누르는 소리/와인 따르는 ASMR, 브이로그 스타일 | 셀프 탭 체험 영상이 핵심 비주얼 자산(asset) |

**비주얼 DO & DON'T:**

| DO | DON'T |
|---|---|
| 사람 손에 들린 와인잔 | 와인잔만 단독으로 놓인 스틸컷 |
| 웃고 있는 친구들 사이의 자연스러운 컷 | 정장 입고 와인 테이스팅하는 장면 |
| 셀프 탭에서 직접 따르는 동적 장면 | 소믈리에가 서빙하는 격식 있는 장면 |
| 밝고 따뜻한 조명의 매장 인테리어 | 어둡고 무거운 와인 셀러 분위기 |
| 안주와 함께 놓인 캐주얼한 테이블 세팅 | 정찬 테이블 세팅 |

### 콘텐츠 전략

**콘텐츠 철학: "와인을 가르치지 말고, 와인과 노는 법을 보여줘라"**

교육 콘텐츠가 아닌 공감 콘텐츠. "이건 카베르네 소비뇽이고 특징은..."이 아니라 "월요일 퇴근하고 이거 한 잔 했는데 진짜 살 것 같았어"가 탭샵바의 콘텐츠다.

**콘텐츠 기둥 (Content Pillar, 콘텐츠를 분류하는 핵심 주제):**

| 기둥 | 비중 | 핵심 메시지 | 콘텐츠 유형 |
|------|------|-----------|-----------|
| **일상 속 와인** | 40% | "와인은 특별한 게 아니야" | 퇴근 후 한 잔 브이로그, 요일별 와인 추천, 상황별 페어링 |
| **탭 체험** | 30% | "직접 눌러봐, 그게 제일 빨라" | 셀프 탭 체험 영상, 신규 입고 와인 소개, 탭 앞 리액션 |
| **와인 쉽게** | 20% | "몰라도 괜찮아, 같이 알아가자" | 초보 와인 용어 해설, 취향 찾기 가이드, Q&A |
| **매장 & 사람** | 10% | "여기가 그런 곳이야" | 매장 분위기, 스태프 소개, 고객 후기, 이벤트 |

**콘텐츠 포맷별 가이드:**

| 포맷 | 길이 | 핵심 | 빈도 |
|------|------|------|------|
| 인스타 릴스(Reels) | 15-30초 | 셀프 탭 체험, 한 잔 추천, 리액션 | 주 3-4회 |
| 인스타 피드 | 이미지 1-5장 | 와인 추천 카드뉴스, 매장 사진 | 주 2-3회 |
| 인스타 스토리 | 즉석 | 오늘의 매장, 실시간 소통, 투표 | 매일 |
| 블로그/브런치 | 1,000-1,500자 | 와인 입문 가이드, 매장 스토리 | 월 2-4회 |
| 유튜브 쇼츠 | 30-60초 | 릴스 확장판, 매장 투어 | 주 2-3회 |

### KPI 측정 원칙

1. **허영 지표(Vanity Metrics)에 속지 않기**: 팔로워 수보다 참여율, 조회수보다 저장/공유가 중요하다
2. **매장 연결이 최종 목표**: 모든 온라인 활동은 결국 "매장에 발을 들이게 하는 것"으로 귀결되어야 한다
3. **주간 체크, 월간 리뷰**: 숫자는 매주 확인하되, 전략 수정은 월 단위로 판단한다
4. **비교 기준은 자기 자신**: 경쟁사 벤치마크보다 "지난달 대비 성장"에 집중한다

## 부록: 브랜드 검증 체크리스트

모든 콘텐츠, 캠페인, 커뮤니케이션을 내보내기 전에 이 5가지를 확인한다:

- [ ] **탭샵바답나?** -- 다른 와인바가 해도 어색하지 않은 콘텐츠라면, 탭샵바답지 않다
- [ ] **"일상"이 느껴지나?** -- 특별한 날이 아닌 평범한 화요일에도 어울리는 톤인가
- [ ] **초보자가 주눅 들지 않나?** -- 와인을 하나도 모르는 친구에게 보여줘도 편안한가
- [ ] **행동을 유도하나?** -- "그래서 뭐 하라고?"에 대한 답이 있는가
- [ ] **3초 안에 핵심이 보이나?** -- 스크롤하다 멈출 만한 첫 인상인가
