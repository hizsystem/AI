## 1. 탭샵바플레이스 — 여의도점 벚꽃 데이트 코스

**기획의도**: 여의도 벚꽃축제 시기에 맞춘 데이트 코스 소개. "벚꽃 보고 → 와인 한 잔"이라는 자연스러운 동선. 여의도 = 벚꽃 명소라는 시즌 연결이 핵심.

**포맷**: 릴스 15~30초

```
[0~3초] 여의도 벚꽃길 걷는 커플 (or 벚꽃 풍경)
[3~8초] "벚꽃 보고 어디 가지?" 자막
[8~13초] 매장 입구 → 문 열기
[13~20초] 나란히 앉아 탭에서 와인 따르기 → 잔 부딪히기
[20~25초] 테이블에 와인 + 안주 + 둘이 즐기는 장면
[25~30초] 로고 + 위치
```

자막: "벚꽃 보고" → "와인 한 잔." → "여의도 데이트 코스 완성." → 📍 tap.shop.bar 여의도점

## 7. 탭샵바가이드 — 셀프탭 이용 가이드

**기획의도**: #탭샵바가이드 시리즈 첫 번째 콘텐츠. "처음 왔는데 어떻게 해요?"에 대한 답. 한 번 만들면 하이라이트에 고정해서 계속 쓸 수 있는 자산. 이후 "와인 고르는 법", "페어링 기본기", "탭샵바 100% 즐기는 법" 등으로 시리즈 확장 가능.

**포맷**: 릴스 15~30초 (영상 파일 업로드)

```
[0~3초] "탭샵바 처음이에요?" 자막
[3~8초] 입구에서 들어오는 장면
[8~12초] 탭 존 앞에 서는 장면 → "여기서 골라요"
[12~18초] 카드 터치 → 와인 따르기 → "이렇게 따르면 돼요"
[18~24초] 잔 들고 자리에 앉기 → "끝. 이게 다예요."
[24~30초] 로고 + "몰라도 괜찮아, 탭이 다 알려줘요"
```

**캡션**:
```
탭샵바 처음이면
이것만 알면 돼요.

1. 마시고 싶은 와인 앞에 서기
2. 카드 대고 잔 대기
3. 버튼 누르기

끝.

소믈리에한테 물어볼 필요 없고
메뉴판 읽을 필요도 없어요.

몰라도 괜찮아요.
탭이 다 알려줘요.

📍Place
tap.shop.bar 전 매장
```

**하이라이트 고정**: "가이드" 하이라이트에 상단 고정 → 프로필 방문자 전환 도구로 활용.
