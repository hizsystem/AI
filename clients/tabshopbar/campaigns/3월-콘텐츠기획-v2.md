## 핵심 메시지

### 탭샵바만의 한 줄

> **"비싼 와인, 유명한 와인도 한 잔만 트라이해볼 수 있는 곳"**

셀프 탭 = 병 단위가 아니라 잔 단위. 10만원짜리 와인도 부담 없이 한 잔.
이 메시지가 모든 콘텐츠의 바닥에 깔려야 한다.

### 3월 목표
- **톤 확립**: "탭샵바 피드는 이런 느낌이야" 브랜드팀 합의
- **시리즈 런칭**: 5개 꼭지 중 2~3개 시리즈의 첫 콘텐츠 발행
- **첫 달 부담 최소화**: 기존 소재 + 저공수 위주

## 3월 콘텐츠 플랜 (6건)

### 1. 매장 무드 — 톤세팅용 첫 피드

| 항목 | 내용 |
|------|------|
| **시리즈** | #탭샵바플레이스 (프리뷰) |
| **포맷** | 단일 이미지 1장 |
| **업로드** | 3/10 (월) |
| **공수** | ★☆☆☆☆ |

**콘셉트**: 사진 한 장으로 "여기 가보고 싶다" 만들기. 피드 전체 톤의 기준점.

**비주얼**: 매장 조명 아래 와인잔 들린 손 / 셀프 탭에서 따르는 순간 / 1인칭 시점

**캡션 시안**:
```
퇴근하고 한 잔.
```
대안: `수요일엔 와인.` / `오늘 탭에서 제일 맛있었던 거.`

**포인트**: 이 1장이 앞으로의 모든 콘텐츠 기준. 브랜드팀 합의 필수.

### 6. 합정점 공간 릴스 (시리즈 1호)

| 항목 | 내용 |
|------|------|
| **시리즈** | #탭샵바플레이스 |
| **포맷** | 릴스 (15~30초) |
| **업로드** | 3월 말 or 4월 초 |
| **공수** | ★★★☆☆ |

**콘셉트**: #탭샵바플레이스 시리즈의 첫 번째. 합정점을 시네마틱하게 소개.

**릴스 구성**:
```
[0~3초] 입구 외관 → 문 열림
[3~10초] 내부 공간 → 셀프 탭 존
[10~20초] 탭에서 와인 따르는 손 → 잔에 와인 차오름
[20~25초] 자리에 앉아 한 잔 (1인칭)
[25~30초] 로고 + 위치 정보
```

**자막 오버레이**:
```
"10만원짜리 와인도"
"여기선 한 잔만."
"탭에서 골라, 직접 따라."
📍 탭샵바 합정점
```

**BGM**: 잔잔한 재즈 or 로파이 (저작권 프리)
