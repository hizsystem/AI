## 와인을 매력적으로 만드는 법 — 타겟별 프레임

같은 와인이라도 타겟이 "끌리는 포인트"가 다릅니다.
4월 프로모션인 **뉴질랜드 쇼비뇽 블랑**으로 예를 들면:

| 타겟 | 와인에 끌리는 포인트 | 쇼비뇽 블랑을 이렇게 말하면 끌림 | 이렇게 말하면 안 끌림 |
|------|-------|-------|-------|
| **탐험가 지은** (25-32 여성) | **발견 + 공유** — "이거 아는 사람 별로 없는데" "친구한테 추천해야지" | "뉴질랜드에서 온 와인인데, 패션프루트 스무디에 라임 짠 맛이에요. 이거 마셔보면 화이트 와인 인생이 바뀜" | "뉴질랜드의 대표 품종으로 가격 대비 만족도가 높습니다" |
| **여유파 민준** (28-35 남성) | **루틴 업그레이드** — "맥주 대신 이거" "가성비 좋네" | "퇴근하고 맥주 대신 이거 한 잔. X,XXX원인데 하루가 달라짐. 시원하게 마시면 맥주보다 나음" | "탭으로 한 잔부터 시도해볼 수 있습니다" |
| **커플 수아&재현** (24-30 커플) | **분위기 완성** — "여기서 이거 마시면 사진 예쁘겠다" | "이 와인 잔에 따르면 노란빛이 조명 아래서 반짝거려요. 데이트 테이블에 놓으면 완성" | "80종 셀프 탭으로 맛볼 수 있는 곳" |

**원칙**: 각 콘텐츠는 하나의 타겟을 주력으로 설계하되, 다른 타겟도 자연스럽게 반응할 수 있는 보편성을 유지합니다.

- 잔에 따라진 와인 + 보틀이 함께 있는 테이블 컷

**선행 확인**: 쇼비뇽 블랑 브랜드명, 잔당 가격, 프로모션 기간, 추천 페어링 메뉴

### 2. 탭샵바플레이스 — 여의도점 소개 릴스

| 항목 | 내용 |
|------|------|
| **카테고리** | #탭샵바플레이스 |
| **포맷** | 릴스 15~30초 |
| **주력 타겟** | 여유파 민준 (퇴근 후 루틴) |
| **목적** | 여의도 직장인 타겟 유입 + "퇴근하고 여기 가볼까" |

**콘셉트**: 여의도점은 금융/오피스 상권. "퇴근 후 가볍게 한 잔"이 가장 자연스러운 지점. 도산점(데이트 무드)이나 합정점(동네 바)과 다른, **직장인의 저녁 루틴** 앵글로.

**릴스 구성**:
```
[0~3초] 여의도 빌딩숲 or IFC → 퇴근 시간 풍경
[3~8초] 매장 입구 → 문 열기
[8~15초] 내부 바 좌석 → 탭에서 와인 따르기
[15~22초] 바에 앉아 한 잔 — 혼자 or 동료와
[22~28초] 창밖 여의도 야경 + 자막
[28~30초] 로고 + 위치
```

**자막 오버레이**:
```
"퇴근하고"
"한 잔."
"이게 루틴이에요."
📍 tap.shop.bar 여의도점
```


**왜 "여의도 + 민준"인가**:
- 여의도 = 직장인 밀집 상권. 이 지점의 핵심 타겟은 "퇴근 후 한잔" 니즈가 있는 민준.
- 상암점 릴스("상암에 이런 와인바가?")와 같은 구조지만, 여의도는 "퇴근 루틴"에 초점.
- 도산점 릴스는 나중에 "데이트 무드"(수아&재현)로 차별화.

**비주얼 디렉션**:
- 상암점 릴스와 같은 포맷 유지 (입구→내부→탭→좌석)
- 여의도 차별화: 빌딩숲 퇴근 풍경으로 시작, 바 좌석 중심 (테이블보다 바)
- 혼자 앉아 있는 직장인 or 동료 2명의 자연스러운 장면
- BGM: 재즈 or 로파이 (상암점과 같은 계열)

**선행 확인**: 여의도점 촬영 일정 (짐벌 필요), 매장 내 바 좌석 앵글 확인

| **목적** | 페어링 시리즈 런칭 + 이달의탭 연계 + 저장 유도 |

- 커플은 "이거 같이 시킬까?" 하는 데이트 매뉴얼.
- → 타겟 구분 없이 먹히는 카테고리. 다만 캡션 톤은 지은(공유)에 맞추면 나머지도 따라옴.

**선행 확인**: 쇼비뇽 블랑과 궁합 좋은 메뉴 2~3개 (박창규 확인)

### 4. 탭샵바씬 — 추카추카추 (4월 에피소드)

| 항목 | 내용 |
|------|------|
| **카테고리** | #탭샵바씬 |
| **포맷** | 릴스 15~20초 |
| **주력 타겟** | 전체 (알고리즘 도달용) |
| **목적** | 셀럽 인지도 탑승 → 채널 노출 극대화 |

**콘셉트**: 추카추카추 유튜브 롱폼에서 탭샵바 노출 장면을 컷편집한 숏폼. 강승윤→키키에 이어 4월 에피소드가 있으면 같은 포맷으로 연속성 유지.

**캡션 구조** (3월 톤 유지):
```
[후킹] — 셀럽 이름 + 궁금증 ("잠깐, 이번에는 ○○님이?")
[상황] — "탭샵바에서 ○○" (와인 고르기, 파티, 공연 등)
[연결] — "와인 마시기 좋은 공간에 음악까지"
[CTA] — "이 분위기, 탭샵바에서 한 번 느껴보세요."
[정보] — 매장명, Full ver 유튜브 링크
```

**3월 추카추카추 캡션에서 개선할 점**:

강승윤 편의 "압도적인 규모와 인테리어! / 이 정도면 자랑 좀 해도 되죠"가 자기 자랑 톤이었던 점. 키키 편에서는 "와인 마시기 좋은 공간에 음악까지 더해진 순간"으로 개선됨. 이 방향 유지.

다만 한 가지 더 — 추카추카추 콘텐츠의 핵심은 "셀럽이 탭샵바에 왔다"가 아니라 **"셀럽도 편하게 와서 즐기는 곳"**이 메시지여야 합니다. "찾아주셨습니다 ✨"는 약간 접대 톤. 대신:

```
// AS-IS
"이번에는 키키님이 탭샵바를 찾아주셨습니다 ✨"

// TO-BE
"이번에는 키키님이 탭샵바에서 와인 한 잔 했어요."
```

"찾아주셨습니다" → 탭샵바가 을. "한 잔 했어요" → 탭샵바가 당연히 가는 곳.

**선행 확인**: 4월 추카추카추 에피소드 촬영 여부/일정. 없으면 이 슬롯은 다른 #탭샵바씬 콘텐츠(대관 행사 등)로 대체.

### 5. 탭샵바뉴 or 추가 콘텐츠 (유동)

| 항목 | 내용 |
|------|------|
| **카테고리** | #탭샵바뉴 (신메뉴 있을 경우) or 추가 #이달의탭 |
| **포맷** | 카드뉴스 or 단일 이미지 |
| **주력 타겟** | 상황에 따라 |

**옵션 A — 신메뉴 있을 경우 (#탭샵바뉴)**:

신메뉴 사진 + 짧은 소개 + 페어링 추천. 3월의 "이번 달 탭샵바 테이블에 새로 올라온 것들" 포맷.

```
[캡션 방향]
새로 나온 [메뉴명],
쇼비뇽 블랑이랑 같이 먹으면 큰일 나요.
(좋은 쪽으로)

이달의탭 프로모션이랑 같이 즐겨보세요.
```

→ 이달의탭(#1) + 페어링(#3) + 뉴(#5)가 서로 연결되면서 "쇼비뇽 블랑 + 이 안주" 메시지가 한 달 내내 반복됨. 이게 기억에 남음.

**옵션 B — 신메뉴 없을 경우**:

이달의탭 리마인드 콘텐츠 (프로모션 마감 전 "아직 안 마셔본 사람?")
or 상암점 오픈 콘텐츠 (3월 딜레이분 소화)

## 캡션 톤 가이드 (실제 운영 기준)

3월 발행/기획 캡션을 분석해서 정리한 **실제 톤**:

### 현재 톤 (유지할 것)
- 존댓말 베이스 ("~할 수 있습니다", "~해보세요", "~이에요")
- 짧은 행 — 한 줄에 하나의 메시지
- 이모지 절제 (🍷 정도만)
- 마지막에 Place + "wine is second americano" 시그니처

### 개선할 것
| AS-IS | TO-BE | 이유 |
|-------|-------|------|
| "칠레의 대표 와이너리, 타라파카입니다" | "맛은 ~한 느낌이에요" | 원산지/브랜드 → 맛/경험으로 |
| "80여 가지 와인을 탭으로" | "탭에서 한 잔 골라서" | 숫자 스펙 → 행동 묘사로 |
| "가격 대비 만족도가 높은 와인" | "한 잔 X,XXX원인데 이 맛이면" | 추상적 가치 → 구체적 가격+맛 |
| "찾아주셨습니다 ✨" | "한 잔 했어요" | 접대 톤 → 당연히 오는 곳 |
| "압도적인 규모와 인테리어!" | (삭제) | 자기 자랑은 비주얼로 보여주면 됨 |

### 캡션 체크리스트
- [ ] 첫 줄에서 타겟의 상황/감정을 건드리는가?
- [ ] 와인을 "설명"이 아니라 "매력적으로" 만들고 있는가?
- [ ] 숫자 스펙(80종, 1000종)이 아니라 경험을 말하는가?
- [ ] "이걸 본 타겟이 어떤 행동을 할까?"에 답이 있는가?
- [ ] 탭샵바가 아닌 다른 와인바에서도 쓸 수 있는 말이면 → 빼기
