## 팀 미팅 아젠다 제안

```
1. [2분] 오늘 주간미팅 결과 공유 (우성민)
2. [5분] 5개 태스크 설명 & 담당 확인
   - 명함 → 이수민 + 한울님 정보 취합
   - 안내 템플릿 → 이수민 (Figma→PPT)
   - 인스타 → 우성민 (콘텐츠 선별 & 배치)
   - 카플친 → 우성민(세팅) + 이수민(디자인)
   - 매장 방문 → 목요일 14시, 이수민 합류
3. [3분] 이수민 작업 우선순위 확인
   - 화: 도산 긴급 안내 (완료했으면 스킵)
   - 수~목: 명함 + 안내 템플릿
   - 목~금: 카플친 배너
4. [2분] 한울님 명함 정보 취합 기한 확인 (목요일까지)
5. [3분] 목요일 매장 방문 동선 & 준비물 확인
6. [2분] 다음 주 주간미팅 아젠다 프리뷰
─────────────────────────────────
총 약 17분
```
