## 1. 무드보드: 비주얼 톤 방향

### 우리가 지향하는 무드

> **"따뜻한 조명 아래, 누군가의 손에 든 와인 한 잔"**

| 키워드 | 설명 |
|--------|------|
| **Warm & Cozy** | 따뜻한 조명, 캔들, 우드톤. 형광등 절대 X |
| **Human-first** | 사람이 주인공. 와인 보틀 단독 클로즈업 X |
| **Casual & Real** | 연출된 모델컷보다 자연스러운 순간 포착 |
| **Close-up POV** | 1인칭 시점, 가까운 거리감. 내가 그 자리에 앉아있는 느낌 |
| **Dark but Inviting** | 어두운 배경이지만 차갑지 않고 들어가고 싶은 느낌 |

### 컬러 무드

```
[메인 톤]
██ Black (#000000)     - 배경, 텍스트
██ Warm Amber          - 매장 조명, 와인 색감
██ White (#FFFFFF)     - 텍스트, 하이라이트

[포인트 컬러 - BI 기반]
██ TSB Blue (#156DFF)  - CTA, 강조
██ TSB Yellow (#FFC100)- 프로모션, 혜택
██ TSB Red (#F43131)   - 한정, 마감

[사진 보정 방향]
→ 웜톤 +15~20%
→ 채도 살짝 UP
→ 그림자 살리되 디테일 유지
→ 필터: VSCO HB2 또는 C1 계열 참고
```

### DO vs DON'T (비주얼)

| DO | DON'T |
|----|-------|
| 손에 와인잔이 들려있는 컷 | 와인 보틀만 놓인 카탈로그 컷 |
| 매장 조명이 만드는 자연스러운 분위기 | 스튜디오 조명, 플래시 직사 |
| 사람들이 대화하며 마시는 장면 | 빈 테이블, 무인 공간 |
| 가까운 거리에서 찍은 듯한 느낌 | 멀리서 찍은 전경 사진 |
| 안주와 와인이 함께 있는 테이블 전체 | 와인잔만 단독으로 |
| 캐주얼한 복장, 자연스러운 표정 | 정장, 과한 포즈 |

## 4. 3월 콘텐츠 4건 상세 기획

### 제안 방향

첫 달이니까 **"탭샵바가 어떤 곳인지 보여주는 것"**에 집중.
기존 소재/저공수 소재 위주로, 브랜드팀과 톤을 맞추는 데 목적을 둔다.

### 콘텐츠 #1: 유튜브 강승윤 편 활용

> **"연예인도 탭 앞에 서면 그냥 사람" 느낌의 자연스러운 장소 등장 활용**

| 항목 | 내용 |
|------|------|
| **포맷** | 피드 이미지 또는 릴스 (15초 클립) |
| **필러** | 매장 & 브랜드 |
| **소재** | 유튜브 영상에서 합정점 등장 장면 캡처/클립 |
| **공수** | ★☆☆☆☆ (기존 소재 활용) |
| **업로드 목표** | 3월 1~2주 차 |

**캡션 시안**:
```
합정점에 손님이 오셨더라고요 🍷

(유튜브 채널명) 에서 탭샵바 합정점이 등장했어요.
셀프 탭 앞에 서면 누구나 그냥 와인 좋아하는 사람.

전체 영상 보러 가기 👆 (스토리 링크)

#탭샵바 #합정점 #tapshopbar
```

**제작 TO-DO**:
- [ ] 브랜드팀에서 유튜브 영상 링크 수령
- [ ] 영상에서 합정점 등장 구간 캡처 (3-5장) 또는 15초 클립 편집
- [ ] 캡션 확정 (출연자 언급 범위 브랜드팀 확인)
- [ ] 스토리에 영상 링크 연동

**포인트**: 장소 협찬이라 과하게 밀지 않고, "여기 탭샵바네?" 하는 자연스러운 발견 느낌으로. 팬덤 유입 효과도 기대 가능.

### 콘텐츠 #2: 상암점 오픈 공지

> **"6번째 탭이 열립니다" - 신규 매장 오픈 안내**

| 항목 | 내용 |
|------|------|
| **포맷** | 피드 이미지 (1장) 또는 캐러셀 (3장) |
| **필러** | 매장 & 브랜드 |
| **소재** | 상암점 매장 사진 (오픈 전 촬영 or 시공 현장) |
| **공수** | ★★☆☆☆ (사진 1~3장 + 디자인 간단) |
| **업로드 목표** | 3/12~13 (오픈 2~3일 전) |

**캡션 시안 A (간결형)**:
```
6번째 탭이 열립니다.

📍 탭샵바 상암점
🗓 3월 15일 (토) 오픈
📍 서울 마포구 상암동 [주소]

상암에서도 가볍게 탭, 오늘의 와인.

#탭샵바 #상암점 #신규오픈 #tapshopbar #상암맛집
```

**캡션 시안 B (스토리텔링형)**:
```
동대문에서 시작해서,
청계천, 도산, 여의도, 합정을 거쳐
이번엔 상암.

6번째 탭이 열립니다.

📍 탭샵바 상암점
🗓 3.15 (토) OPEN

#탭샵바 #상암점 #6호점 #tapshopbar
```

**제작 TO-DO**:
- [ ] 상암점 매장 사진 확보 (인테리어 완성 후, or 시공 현장 "coming soon" 느낌)
- [ ] 디자인: TSB Black 배경 + 간결한 텍스트 오버레이
- [ ] 캡션 확정
- [ ] 스토리에도 카운트다운 스티커로 D-3, D-2, D-1 운영

**제안**: 오픈 전 매장 사진이 없으면 **"시공 현장 스니크피크"**도 콘텐츠가 됨. "곧 여기서 와인 마실 수 있어요" 같은 티저.

### 콘텐츠 #3: 신메뉴 소개

> **"이번 달 탭샵바 테이블에 새로 올라온 것들"**

| 항목 | 내용 |
|------|------|
| **포맷** | 캐러셀 (3~5장) |
| **필러** | 일상 와인 (메뉴 중심) |
| **소재** | 메뉴 촬영 최종본 (구글 드라이브 폴더 소재 활용) |
| **공수** | ★★☆☆☆ (촬영본 있음, 디자인/편집만) |
| **업로드 목표** | 3월 3~4주 차 |

**캐러셀 구성안**:

| 장 | 내용 | 비주얼 |
|---|------|--------|
| 1 (표지) | "새로 왔어요 🍽" | 메뉴 전체 테이블 샷 or 대표 1품 클로즈업 |
| 2 | 신메뉴 A 소개 | 메뉴 사진 + 이름 + 한 줄 설명 |
| 3 | 신메뉴 B 소개 | 메뉴 사진 + 이름 + 한 줄 설명 |
| 4 | 신메뉴 C 소개 | 메뉴 사진 + 이름 + 한 줄 설명 |
| 5 (마지막) | 와인 페어링 추천 + CTA | "이 메뉴엔 이 와인" + "직접 맛보러 오세요" |

**캡션 시안**:
```
이번 달 탭샵바 테이블에 새로 올라온 것들.

저장해두고 다음에 시켜보세요 👆

어떤 메뉴가 가장 궁금해요?
댓글로 알려주세요!

#탭샵바 #신메뉴 #와인안주 #tapshopbar
```

**제작 TO-DO**:
- [ ] 메뉴 촬영본에서 신메뉴 사진 선별 (파일명 매칭 완료 후)
- [ ] 캐러셀 디자인 (TSB 톤, Pretendard 서체)
- [ ] 메뉴명(한/영) + 한 줄 설명 카피 작성
- [ ] 페어링 와인 추천 (박창규 확인)

### 콘텐츠 #4: 매장 무드 (브랜드 톤 확립용)

> **"사진 한 장으로 '여기 가보고 싶다' 만들기"**

| 항목 | 내용 |
|------|------|
| **포맷** | 피드 이미지 (1장, 무드 중심) |
| **필러** | 일상 와인 |
| **소재** | 3/6 매장 방문 시 촬영 |
| **공수** | ★☆☆☆☆ (촬영 후 보정만) |
| **업로드 목표** | 3월 2주 차 |

**비주얼 방향**:
- 매장 조명 아래 누군가의 손에 와인잔이 들려있는 컷
- 또는 셀프 탭 앞에서 와인을 따르는 순간
- 또는 테이블 위 안주+와인이 함께 있는 장면
- **1인칭 시점** (내가 그 자리에 앉아있는 느낌)

**캡션 시안**:
```
퇴근하고 한 잔.
```
(짧을수록 좋다. 사진이 말하게.)

**대안 캡션**:
```
수요일엔 와인.
```
```
오늘 탭에서 제일 맛있었던 거.
```

**제작 TO-DO**:
- [ ] 3/6 매장 방문 시 무드 컷 촬영 (최소 20장)
- [ ] 베스트 1장 선정 → 웜톤 보정
- [ ] 짧은 캡션 확정
- [ ] 이 콘텐츠가 피드 첫 인상을 잡는 역할 → **가장 신경 써야 할 1장**

**제안**: 이 콘텐츠가 사실상 **피드의 톤세팅**이에요. 브랜드팀과 "이 사진이 탭샵바 피드의 기준이다"를 합의하는 것이 목표. 나머지 3개는 이 톤에 맞춰 갈 수 있도록.

> **이번 달 목표**: 4개 콘텐츠로 "탭샵바 인스타는 이런 느낌이야"를 확립한다. 숫자(팔로워, 참여율)보다 **톤 합의**가 더 중요한 달. 브랜드팀과 "이 사진, 이 캡션이 맞다/아니다"를 하나씩 맞춰가는 과정 자체가 이번 달의 성과다.
