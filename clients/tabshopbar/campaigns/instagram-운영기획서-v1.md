## 1. 채널 컨셉

### 한 줄 정의

> **"와인 초보도 편하게 들어오는 일상 와인 라이프 채널"**

### 채널 성격

| 항목 | 우리는 | 우리는 아니다 |
|------|--------|------------|
| 역할 | 와인과 노는 법을 보여주는 친구 | 가르치는 선생님, 추천하는 소믈리에 |
| 톤 | "같이 마시자" | "추천드립니다", "엄선했습니다" |
| 주인공 | 와인을 즐기는 사람 | 와인 보틀 자체 |
| 벤치마크 | 스타벅스 인스타, 블루보틀 | 와인 전문 채널 |
| 성공 기준 | "나도 저렇게 마시고 싶다" | "와인 공부가 됐다" |

### 포지셔닝

```
            전문적/교육
               |
     와인21     |    와인폴리
               |
 매거진 ───────┼─────── 리얼/일상
               |
         ★ 탭샵바 ★
               |    블루보틀, 온도
               |
            캐주얼/감성
```

벤치마크는 **와인 전문 채널이 아니라 라이프스타일 카페 채널**. 블루보틀이 커피를 '일상의 특별함'으로 보여주듯, 탭샵바는 와인을 '일상의 특별함'으로 보여준다.

## 2. 타겟

| 순위 | 페르소나 | 나이 | 키워드 | 인스타 행동 |
|------|---------|------|--------|-----------|
| 1순위 | 탐험가 지은 | 25-32 여성 | 새로운 경험, 맛집 탐방 | 저장 많이 함, 스토리 공유, 태그 |
| 2순위 | 여유파 민준 | 28-35 남성 | 퇴근 후 한잔, 데이트 | 릴스 시청, 저장 위주 |
| 3순위 | 커플 수아&재현 | 24-30 커플 | 데이트 장소, 분위기 | 위치 태그 확인, 피드 참고 |

### 타겟 마음 → 콘텐츠 연결

| 타겟이 드는 생각 | 우리가 만들 콘텐츠 |
|----------------|----------------|
| "와인 마셔보고 싶은데 어디 가지?" | 매장 분위기 + 셀프 탭 체험 릴스 |
| "이번 주말 데이트 어디 가지?" | 매장별 분위기 + 페어링 추천 |
| "퇴근하고 뭐 마시지?" | "오늘의 한 잔" 시리즈 |
| "와인 잘 모르는데 창피하지 않을까?" | "처음이라도 괜찮아" 시리즈 |
| "가격이 얼마나 할까?" | 구체적 가격 표기 ("잔당 3,900원부터") |

## 3. 콘텐츠 전략

### 3-1. 콘텐츠 카테고리 (4대 필러)

| 필러 | 비중 | 핵심 메시지 | 콘텐츠 예시 | 월 빈도 |
|------|------|-----------|---------|--------|
| **일상 와인** | 40% | "와인은 특별한 게 아니야" | 퇴근 후 한 잔, 요일별 추천, 날씨별 와인, 페어링 | 3-4건 |
| **탭 체험** | 25% | "직접 눌러봐, 그게 제일 빨라" | 셀프 탭 체험 릴스, 신규 탭 와인, 비교 시음 | 2건 |
| **와인 쉽게** | 20% | "몰라도 괜찮아, 같이 알아가자" | 와인 용어, 취향 찾기, 초보 가이드 | 1-2건 |
| **매장 & 브랜드** | 15% | "여기가 그런 곳이야" | 매장 분위기, 프로모션, 신메뉴, 스태프 소개 | 1건 |

### 3-2. 포맷별 전략

| 포맷 | 월 빈도 | 목적 | 핵심 가이드 |
|------|--------|------|-----------|
| **피드 (이미지)** | 4건 | 브랜드 이미지, 무드 유지 | 고품질 비주얼, BI 톤 준수, 1:1 기본 |
| **피드 (캐러셀)** | 2건 | 정보 전달, 저장 유도 | 와인 가이드, 프로모션 상세, 3-7장 |
| **릴스** | 2건 | 도달 확대, 신규 유입 | 15-30초, 첫 3초 훅, 세로 9:16 |
| **스토리** | 주 3회+ | 일상 소통, 참여 유도 | 투표, Q&A, 실시간 매장 |
| **합계** | **피드 8건 + 스토리 수시** | | 프로모션 월에는 +2건 추가 |

### 3-3. 시리즈 콘텐츠

정기적으로 반복하는 시리즈로 채널 정체성을 만든다.

| 시리즈명 | 포맷 | 빈도 | 설명 |
|---------|------|------|------|
| **"오늘의 한 잔"** | 피드 이미지 | 주 1회 | 요일/날씨/기분별 와인 추천. 무드 중심 |
| **"탭 앞에서 30초"** | 릴스 | 격주 | 셀프 탭 시음 리액션. 짧고 임팩트 |
| **"1만원 챌린지"** | 릴스 | 월 1회 | 1만원으로 몇 잔, 어떤 조합이 가능한지 |
| **"이거 몰랐지?"** | 캐러셀 | 격주 | 와인 입문 상식, 숫자로 보는 와인 |
| **"페어링 실험실"** | 피드/릴스 | 월 1회 | 의외의 안주 + 와인 조합 테스트 |
| **"스태프 Pick"** | 스토리 | 주 1회 | 각 매장 스태프의 이번 주 추천 와인 |

## 4. 카피 톤앤매너

### 브랜드 보이스: "와인 잘 아는 재미있는 친구"

| 원칙 | O | X |
|------|---|---|
| 친구 말투 | "이거 진짜 맛있어" | "추천드립니다" |
| 쉬운 말 | "새콤한 맛이 좋아서" | "산미가 우수한" |
| 짧은 문장 | "퇴근하고 한 잔" | "바쁜 하루를 보내신 후 특별한 한 잔으로" |
| 공감 톤 | "이런 날 있지?" | "정말 놀라운 맛입니다!" |
| 행동 유도 | 모든 게시물에 CTA | CTA 없는 정보 나열 |

**금지어**: 엄선, 프리미엄, 럭셔리, 격조, 소믈리에 추천, 빈티지 셀렉션, ~해드립니다, ~모시겠습니다, 고객님

### 해시태그 전략

**필수 (모든 게시물)**:
```
#탭샵바 #tapshopbar #와인한잔 #셀프탭 #와인바
```

**콘텐츠별 추가**:
- 일상 와인: #퇴근와인 #오늘의와인 #와인일상 #와인스타그램
- 탭 체험: #와인시음 #와인입문 #잔와인 #와인추천
- 와인 쉽게: #와인초보 #와인상식 #와인팁
- 매장별: #{매장명} + 지역 태그 (예: #여의도맛집 #합정데이트)

**매장별 위치 태그 (필수)**:
- 동대문: #동대문맛집 #두타 #동대문데이트
- 청계천: #종로맛집 #을지로맛집
- 도산대로: #도산공원 #신사동맛집
- 여의도: #여의도맛집 #여의도직장인
- 합정: #합정맛집 #합정데이트
- 상암: #상암맛집 (오픈 후)

### 댓글/DM 응대

| 유형 | 원칙 | 예시 |
|------|------|------|
| 칭찬/후기 | 감사 + 다음 방문 유도 | "와줘서 고마워요! 다음엔 화이트도 한번" |
| 질문 (메뉴/가격) | 친절 + 구체적 | "탭 와인 잔당 3,900원부터예요! 80종 중에 고르는 재미가 있어요" |
| 질문 (예약/위치) | 빠르게 + 링크 | "합정점이요! 예약은 캐치테이블에서 가능해요" |
| 부정 피드백 | 공감 + DM 전환 | "불편하셨을 텐데 알려주셔서 감사해요. DM으로 자세히 말씀해주시면 확인해볼게요!" |
| 태그/UGC | 리그램/댓글 반응 | "이 조합 진짜 맛있죠?! 스토리에 공유해도 될까요?" |

**응대 시간 목표**: 댓글 12시간 내, DM 24시간 내

## 5. 비주얼 톤앤매너

### 컬러 (TSB BI 기반)

| 용도 | 컬러 | HEX | 사용처 |
|------|------|-----|-------|
| 메인 배경 | TSB Black | #000000 | 텍스트 오버레이, 메인 피드 |

| 포인트 1 | TSB Blue | #156DFF | 하이라이트, CTA, 강조 |
| 포인트 2 | TSB Yellow | #FFC100 | 프로모션, 혜택, 주목 |
| 포인트 3 | TSB Red | #F43131 | 한정, 마감 |

**원칙**: 사진 위 텍스트는 Black 또는 White만. TSB 3색 동시 사용은 로고 영역에서만.

### 타이포그래피

| 용도 | 서체 | 웨이트 |
|------|------|--------|
| 영문 타이틀 | P22 Underground | Heavy |
| 한글 타이틀 | Pretendard | Extra Bold / Bold |
| 한글 본문 | Pretendard | Semi Bold / Medium |

### 사진 톤

| 요소 | DO | DON'T |
|------|-----|-------|
| 조명 | 따뜻한 매장 조명, 캔들, 자연광 | 형광등, 차가운 화이트, 플래시 |
| 색감 | 웜톤 보정, 채도 살짝 높게 | 차가운 블루톤, 과한 필터 |
| 구도 | 가까운 거리감, 1인칭 시점, 손 보이는 컷 | 먼 전경, 무인 테이블 |
| 인물 | 자연스러운 표정, 캐주얼 복장 | 정장, 포즈 과한 모델컷 |
| 주인공 | **사람 > 와인**. 와인을 즐기는 사람이 중심 | 와인 보틀만 클로즈업 |

### 피드 그리드 (9칸 통일감)

```
┌────────────┬────────────┬────────────┐
│ [사진/무드] │ [캐러셀]   │ [릴스 커버] │
├────────────┼────────────┼────────────┤
│ [사진/무드] │ [사진/인물] │ [캐러셀]   │
├────────────┼────────────┼────────────┤
│ [릴스 커버] │ [사진/무드] │ [사진/인물] │

**규칙**:
- 텍스트 위주 콘텐츠 연속 배치 금지 (최소 1칸 간격)
- 릴스 커버도 피드 톤앤매너에 맞춰 별도 제작
- 캐러셀 표지는 비주얼 중심 (텍스트 최소화)

## 7. 운영 프로세스

### 7-1. 월간 사이클

```
[매월 마지막 주]                    [해당 월]
     │                                │
     ├─ 다음 달 테마 확정              ├─ 콘텐츠 발행 (주 2-3건)
     ├─ 월간 콘텐츠 캘린더 수립         ├─ 스토리 발행 (주 3회+)
     ├─ 프로모션 연계 확인              ├─ 댓글/DM 응대
     ├─ 소재 기획 (카피+레퍼런스)       ├─ UGC 리그램
     └─ 촬영 스케줄 확정               └─ 인플루언서 방문 관리
                                        │
[해당 월 1주 차]                    [해당 월 마지막 주]
     │                                │
     ├─ 집중 촬영 (월 1회)             ├─ 월간 성과 리뷰
     ├─ 디자인 작업 (캐러셀/배너)       ├─ 고성과 콘텐츠 분석
     └─ Week 1-2 콘텐츠 완성           └─ 다음 달 기획 시작
```

### 7-2. 주간 루틴

| 요일 | 할 일 | 비고 |
|------|-------|------|
| **월** | 이번 주 콘텐츠 최종 확인, 캡션 완성 | 발행 전 검수 |
| **화** | 주간미팅에서 콘텐츠 방향 공유 | 탭샵바 주간미팅 10시 |
| **화~목** | 피드 콘텐츠 발행 (주 2-3건) | 최적 시간대에 발행 |
| **금** | 스토리 활성화 (주말 방문 유도) | 투표, Q&A, 매장 분위기 |
| **토~일** | 스토리 매장 현장감 (스태프 촬영) | 실시간 분위기 공유 |
| **일** | 주간 수치 체크 | 인사이트 확인, 다음 주 조정 |

```

레퍼런스 수집    편집/보정       수정 반영       댓글/DM 응대    인사이트 분석

담당: 우성민     우성민+외부      박창규/김희종    우성민          우성민
기간: D-7       D-5~D-3        D-2~D-1       발행일          발행 후 3일
