## 4월 월간 테마

> **"봄이니까, 한 잔"**

4월은 봄의 본격 시작. 날씨가 풀리고 외출이 늘어나는 시기.
"꽃놀이 끝나고 한 잔", "테라스에서 로제 한 잔", "봄바람에 와인 한 잔" — 와인과 봄의 자연스러운 연결.

**시즌 키워드**: 로제 와인, 테라스, 봄 와인, 야외 페어링, 가벼운 화이트
**전략 포인트**: 3월에 톤세팅 + 상암점 오픈으로 채널 기반을 잡았다면, 4월은 **시리즈 안착 + 시즌 콘텐츠로 팔로워 확장**

### 1. 이달의 탭 — "4월 탭에 꽂아둔 와인"

| 항목 | 내용 |
|------|------|
| **시리즈** | #이달의탭 |
| **해시태그** | #이달의탭 #탭샵바 #tapshopbar #와인추천 #4월와인 #로제 |
| **포맷** | 카드뉴스 4장 |
| **업로드** | 4/1 (수) |
| **필러** | 와인 쉽게 (20%) |
| **목적** | **인지 + 전환** — 4월 프로모션 와인 소개, 방문 유도 |
| **공수** | ★★☆☆☆ |

**콘셉트 한 줄**: 봄이니까 로제. 이번 달 탭에 꽂아둔 봄 와인을 소개한다.

**뾰족한 포인트**: "로제 = 봄"이라는 시즌 공식을 활용. 와인 이름은 어려워도 "봄에 마시기 좋은 핑크빛 와인"이면 누구나 끌린다. 핵심은 "이름 몰라도 돼, 색깔이 예쁘고 맛이 좋아"

**카드뉴스 구성**:
```
[1장] "4월 탭에 꽂아둔 와인" + 로제 와인잔 비주얼 (핑크빛 강조)
[2장] 맛 한 줄 — "딸기 아이스크림 녹인 것 같은 맛" + 추천 상황 ("테라스에서, 퇴근 후 가볍게")
[3장] 프로모션 안내 — 가격, 기간, 혜택
[4장] "봄이니까, 한 잔. 탭에서 직접 따라보세요." + CTA
```

**캡션 시안**:

시안 A (시즌 후킹):
```
벚꽃 보고 와인 생각 안 나는 사람 있어요?

이번 달 탭에 꽂아둔 와인은
딸기 아이스크림 녹인 것 같은 맛이에요.

이름은 어려운데
색깔이 예뻐서 일단 따라보게 돼요.

봄이니까, 한 잔.

#이달의탭 #탭샵바 #tapshopbar #로제와인 #4월와인
```

시안 B (감각 묘사):
```
핑크빛 와인이 잔에 따라지는 거 보면
봄이 왔구나 싶어요.

맛은요,
딸기 + 약간의 새콤함.

이번 달 탭에 있어요.

#이달의탭 #tapshopbar #로제 #봄와인
```

시안 C (질문형):
```
로제 마셔본 적 있어요?

없으면 이번 달이 기회.
탭에서 한 잔만 따라보세요.

#이달의탭 #tapshopbar
```

**비주얼 디렉션**:
- 로제의 핑크빛을 살리는 자연광 or 매장 조명
- 1장: 로제잔 클로즈업 — 핑크빛이 투명하게 비치는 앵글
- 배경에 봄 소품 (꽃 한 송이, 밝은 테이블) 과하지 않게
- 4장: TSB Blue(#156dff) 배경 + White 텍스트

**연계 스토리**:
- "이번 달 탭에 뭐가 꽂혀 있나" 탭 존 실시간 영상
- "로제 마셔본 사람?" 투표 스토리

**선행 확인**: 박창규 — 4월 프로모션 와인 확정 (로제 추천), 잔당 가격

**"와인 모르는 친구" 검증**: 통과. "핑크빛 와인", "딸기 아이스크림 맛"은 와인 지식 없어도 예쁘고 맛있어 보임.

### 2. 탭샵바 페어링 런칭 1 — "이 와인엔 이거지" (릴스)

| 항목 | 내용 |
|------|------|
| **시리즈** | #탭샵바페어링 |
| **해시태그** | #탭샵바페어링 #탭샵바 #tapshopbar #와인페어링 #와인안주 |
| **포맷** | 릴스 15~20초 |
| **업로드** | 4/3 (목) |
| **필러** | 와인 쉽게 (20%) |
| **목적** | **관심 + 전환** — 페어링 시리즈 첫 런칭, 저장 유도 |
| **공수** | ★★★☆☆ |

**콘셉트 한 줄**: "뭐 시킬지 모르겠을 때, 이 조합이면 됩니다." 와인+안주 페어링의 정답을 15초로.

**뾰족한 포인트**: #탭샵바페어링 시리즈의 첫 콘텐츠. 3월에 빠졌던 시리즈를 본격 런칭. 이후 월 2회 정기 발행의 기준점이 되므로, 포맷과 톤을 여기서 확립한다. 핵심은 "어려운 페어링 지식"이 아니라 "그냥 이거 같이 시키면 맛있어".

**릴스 구성**:
```
[0~3초] 테이블에 빈 접시 + 빈 와인잔 → "뭐 시키지?"
[3~8초] 안주 등장 (슬라이드 인) → "이거"
[8~13초] 와인 따르기 (탭 클로즈업) → "여기에"
[13~18초] 한 입 + 한 모금 리액션 → "이 조합 실화?"
[18~20초] 자막 + 로고
```

**자막 오버레이**:
```
"뭐 시키지?"
"이거."
"여기에 이거."
"실화...?"
📍 탭샵바
```

**캡션 시안**:

시안 A (추천형) ✦ 추천:
```
와인 안주 뭐 시킬지
매번 고민이잖아요.

그냥 이거 시키세요.
[안주명] + [와인명]

한 잔 더 하고 싶어질 수 있어요.

저장해두고 다음에 써먹으세요.

#탭샵바페어링 #탭샵바 #tapshopbar #와인안주
```

시안 B (도발형):
```
이 조합 모르면
탭샵바 절반만 즐기는 거예요.

#탭샵바페어링 #tapshopbar
```

시안 C (리스트형):
```
탭샵바 1년 다닌 사람의
페어링 정답지:

[안주명] + [와인명]

#탭샵바페어링 #tapshopbar #와인페어링
```

**비주얼 디렉션**:
- 톱뷰(위에서) + 45도 앵글 혼합
- 음식과 와인잔이 한 프레임에 — "이 조합" 느낌
- 탭에서 따르는 장면 반드시 포함 (USP 연결)
- BGM: 경쾌한 재즈 or 팝 (BPM 100~120, 봄 느낌)
- 보정: 음식 채도 +10%, 와인 색감 살리기

**연계 스토리**:
- "이 조합 먹어본 사람?" 투표
- "다음 페어링은 뭘 소개해줄까요?" 질문 스토리
- 페어링 시리즈 런칭 알림 ("매주 새로운 조합이 올라와요")

**선행 확인**: 박창규 — 추천 페어링 조합 2~3개 (인기 안주 + 맞는 와인)

**"와인 모르는 친구" 검증**: 통과. 음식 + "이거 같이 먹으면 맛있어"는 와인 지식 불필요. 릴스 리액션이 핵심.
