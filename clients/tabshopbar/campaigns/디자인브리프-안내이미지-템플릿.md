# 디자인 브리프: 매장 안내 이미지 템플릿

> 탭샵바 (TAP SHOP BAR) | 작성: 2026-03-02
> 디자이너 요청용

---

## 1. 요청 배경

탭샵바 매장에서 **대관 행사, 운영 변경, 임시 공지** 등이 발생할 때 고객에게 안내하는 이미지가 필요합니다. 이런 상황이 반복적으로 발생하므로, **매번 새로 만들지 않고 텍스트만 교체해서 사용할 수 있는 템플릿 시스템**을 요청합니다.

---

## 2. 필요한 템플릿 종류

### 템플릿 A: 대관/행사 양해 안내

**사용 상황**: 대관 행사로 인해 일반 고객의 이용에 제한이 있을 때
**게시 위치**: 매장 입구, 테이블, 카카오 채널, 인스타그램 스토리
**핵심 메시지**: 양해 부탁 + 대안 안내

**텍스트 예시** (상황에 따라 변경):
```
오늘 ○○점에서 대관 행사가 진행됩니다.

일부 좌석 이용에 제한이 있을 수 있어요.
불편을 드려 죄송합니다.

📍 가까운 다른 매장: ○○점 (도보 ○분)
📞 문의: 02-3398-4095
```

### 템플릿 B: 운영시간 변경 안내

**사용 상황**: 임시 휴무, 영업시간 변경
**텍스트 예시**:
```
○월 ○일(○) ○○점 운영시간이 변경됩니다.

변경: 11:00 → 14:00 오픈
사유: 매장 점검

다른 매장은 정상 운영합니다!
```

### 템플릿 C: 일반 공지 안내

**사용 상황**: 메뉴 변경, 시스템 점검, 기타 알림
**텍스트 예시**:
```
[안내]
○월 ○일부터 ○○이 변경됩니다.

변경 내용: ○○○
문의: 02-3398-4095
```

## 3. 디자인 가이드라인

### BI 준수 사항 (TSB BI System v1.0)

| 항목 | 규정 |
|------|------|
| **컬러** | TSB Black (#000000) 배경 + TSB White (#FFFFFF) 텍스트 기본. 강조 시 TSB Yellow (#FFC100) 사용 가능 |
| **서체 (한글)** | Pretendard - 타이틀 Bold, 본문 Medium |
| **서체 (영문)** | P22 Underground |
| **로고** | Symbol 1 (Line Type, White) - 하단 우측 또는 좌측 고정 |
| **로고 금지** | 색상 혼용, 비례 변경, 요소 추가/삭제 |

### 디자인 톤

| 요소 | 방향 |
|------|------|
| **전체 톤** | 깔끔하고 심플. 정보 전달이 최우선 |
| **분위기** | 불편을 주는 상황이므로 **정중하지만 딱딱하지 않게** |
| **여백** | 충분히. 정보를 빽빽하게 넣지 않는다 |
| **아이콘** | 필요 시 심플한 라인 아이콘 사용 가능 (복잡한 일러스트 X) |

### 레이아웃 제안

```
┌─────────────────────────────────┐
│                                 │
│  [TSB Black 배경]               │
│                                 │
│     ⚠ (심플 아이콘)              │
│                                 │
│   안내 드립니다                   │
│   (Pretendard Bold / White)     │
│                                 │
│   본문 내용                      │
│   (Pretendard Medium / White)   │
│                                 │
│   대안/문의 안내                  │
│   (Pretendard Medium / Yellow)  │
│                                 │
│                  TSB Symbol 1 ──│
│                                 │
└─────────────────────────────────┘
```

## 4. 사이즈 규격

동일 디자인을 **사이즈별로 5종** 제작 요청:

| 용도 | 사이즈 | 비율 | 비고 |
|------|--------|------|------|
| **매장 내 (테이블 텐트/POP)** | A5 (148x210mm) 세로 | - | 인쇄용 300dpi, CMYK |
| **인스타 스토리 / 카카오 채널** | 1080x1920px | 9:16 | 웹용 72dpi, sRGB |
| **인스타 피드** | 1080x1350px | 4:5 | 웹용 72dpi, sRGB |
| **캐치테이블 공지** | 1080x1080px | 1:1 | 웹용 72dpi, sRGB |
| **네이버 플레이스 소식** | 1080x1080px | 1:1 | 웹용 72dpi, sRGB |

> 캐치테이블/네이버 플레이스는 동일 1:1 사이즈이지만, 플랫폼별 텍스트 미세 조정이 필요할 수 있으므로 별도 프레임으로 관리

## 5. 납품 요청

| 항목 | 요청 |
|------|------|
| **작업 도구** | Figma |
| **납품 형태** | Figma 파일 링크 (편집 권한 포함) |
| **템플릿 수** | 3종 (A: 대관 양해 / B: 운영 변경 / C: 일반 공지) |
| **사이즈** | 각 템플릿당 5종 사이즈 = 총 15개 프레임 |
| **컬러 변형** | Black 배경 기본 1종만 (추후 필요 시 White 배경 추가 가능) |

### Figma 파일 구조 요청

```
TSB_안내이미지_템플릿.fig
│
├── Page: A_대관양해
│   ├── [Frame] 매장POP_A5
│   ├── [Frame] 스토리_1080x1920
│   ├── [Frame] 피드_1080x1350
│   ├── [Frame] 캐치테이블_1080x1080
│   └── [Frame] 네이버플레이스_1080x1080
│
├── Page: B_운영변경
│   └── (동일 5종)
│
├── Page: C_일반공지
│   └── (동일 5종)
│
└── Page: 컴포넌트
    ├── [Component] TSB 로고 (Symbol 1, Line/Solid)
    ├── [Component] 안내 아이콘
    └── [Text Style] 타이틀 / 본문 / 강조
```

→ 총 3종 x 5사이즈 = **15개 프레임**

### 사용 방식

**매장 배포용: PPT 템플릿** (매장 스태프가 직접 수정)
- 각 매장에 PPT 파일로 공유
- 텍스트 상자만 수정 → 이미지 내보내기 또는 인쇄
- PPT 내 서체/컬러 사전 세팅 (Pretendard, TSB 컬러)
- 매장 스태프 누구나 수정 가능

**원본 관리: Figma** (디자인팀 관리)
- Figma에서 마스터 템플릿 관리
- 수정/업데이트 시 Figma → PPT로 재변환
- 컬러, 서체는 Figma 스타일로 등록하여 일관성 유지

## 7. 참고

- 이 템플릿은 앞으로 **매장별로 텍스트만 바꿔서 반복 사용**할 예정이므로, 텍스트 교체가 쉬운 구조가 가장 중요합니다.
- 탭샵바의 톤은 **정중하되 격식적이지 않은, 친근한** 느낌입니다. "고객님께 안내 말씀 드립니다" 보다는 "안내 드립니다" 정도의 톤.
- BI 가이드 파일이 필요하면 별도로 전달하겠습니다.
