## PART A: 디자인 영역 (이수민)

> 아래 3건을 3/6(금)까지 제작해주세요.
> BI 가이드: TSB BI System v1.0 기반

### A-1. 프로필 이미지

| 항목 | 내용 |
|------|------|
| **사이즈** | 640 x 640px |
| **포맷** | PNG (투명 배경 X, 배경색 포함) |
| **디자인** | TSB Symbol 1 (컬러: T빨강/S파랑/B노랑) |
| **배경** | TSB Black (#000000) |
| **여백** | 아이콘 주변 충분한 여백 (모바일에서 작게 보임) |
| **참고** | BI 가이드 p.12 Symbol 1 규격 준수 |

**주의**: 카카오톡에서 프로필 이미지는 원형 마스크로 잘림. 핵심 요소가 원형 안에 들어오는지 확인.

```
┌──────────────┐
│  Black 배경    │
│               │
│    (○)        │  ← 원형 마스크 영역
│   T S B       │
│   Symbol 1    │
│               │
└──────────────┘
```

### A-2. 채널 홈 기본 배너

| 항목 | 내용 |
|------|------|
| **사이즈** | 720 x 360px |
| **포맷** | JPG 또는 PNG |
| **카피** | "가볍게 탭, 오늘의 와인" |
| **비주얼** | 셀프 탭 앞에서 와인을 따르는 손 + 매장 분위기 (사진 소재 없으면 텍스트 온리도 OK) |
| **배경** | TSB Black (#000000) |
| **메인 텍스트** | White (#FFFFFF), Pretendard Bold (한글) / P22 Underground (영문) |
| **로고** | TSB Symbol 1 (Line Type, White) 하단 우측 |

**레이아웃 가이드**:

```
┌──────────────────────────────────────┐
│  [TSB Black 배경]                     │  720px
│                                      │
│   가볍게 탭, 오늘의 와인               │
│   (Pretendard Bold / White)          │  360px
│                                      │
│                       TSB Symbol 1 ──│
└──────────────────────────────────────┘
```

> 이 배너는 프로모션 없을 때 **상시 노출**되는 기본 배너.
> 프로모션 시 별도 배너로 교체하고, 종료 시 이 배너로 복귀.

## PART B: 세팅 영역 (우성민)

> 카카오톡 채널 관리자에서 직접 세팅.
> 디자인 도착 전에 텍스트 기반 작업은 먼저 진행 가능.

### B-1. 채널 소개글 변경

**현재**: 기본 소개글 (브랜드 메시지 부재)

**변경 후**:
```
"오늘 뭐 마시지?" 의 새로운 답.
아메리카노 대신, 탭에서 시작하는 오늘의 와인 한 잔.

한 잔 3,900원부터 | 80종 셀프 탭 | 6개 매장
채널 추가하면 첫 방문 혜택을 드려요!
```

**세팅 위치**: 카카오 채널 관리자 > 채널 관리 > 채널 정보 > 소개

---

### B-2. 웰컴 메시지 교체

**현재**: "안녕하세요 TAP SHOP BAR입니다!" + 매장 정보 나열

**변경 후**:
```
어서 와요, 탭샵바! 🍷

"오늘 뭐 마시지?" 고민될 때,
아메리카노 대신 탭에서 시작해 보세요.

🎁 채널 추가 감사 혜택
→ 첫 방문 시 이 메시지 캡처해서 보여주세요!
  셀프 탭 1잔 무료!

📍 가까운 매장 찾기
[매장 보기 링크]
```

> 사업자 인증 전이므로 쿠폰 대신 "캡처해서 보여주세요" 형태로 임시 운영

**세팅 위치**: 카카오 채널 관리자 > 1:1 채팅 > 채팅 설정 > 인사말

### B-3. 채팅방 메뉴 6개 재구성

**현재 4개 → 6개로 변경**:

| 순서 | 메뉴명 | 연결 | 타입 | 비고 |
|------|--------|------|------|------|
| 1 | **이달의 추천** | 카카오 포스트 or 웹링크 | 링크 | 3월: "이달의 추천 와인 TOP 3" |
| 2 | **매장 찾기 & 예약** | 매장 리스트 페이지 | 링크 | 주소, 운영시간, 예약 링크 |
| 3 | **셀프 탭 이용 방법** | 이미지 카드 or 웹링크 | 링크 | 3-4단계 가이드 |
| 4 | **충전 & 잔액 확인** | 기존 유지 | 기존 | - |
| 5 | **쿠폰함** | "준비 중" 안내 | 링크 | 사업자 인증 후 정식 오픈 |
| 6 | **1:1 문의** | 1:1 채팅 연결 | 채팅 | - |

**메뉴 연결 URL/콘텐츠 사전 준비**:

| 메뉴 | 연결할 콘텐츠 | 준비 상태 |
|------|------------|---------|
| 이달의 추천 | 카카오 포스트 작성 필요 (추천 와인 3종) | ⬜ 작성 필요 |
| 매장 찾기 | 캐치테이블 or 공식 웹페이지 | ✅ 링크 확보 |
| 셀프 탭 이용 | 이미지 가이드 or 웹링크 | ⬜ 콘텐츠 필요 |
| 충전 & 잔액 | 기존 | ✅ |
| 쿠폰함 | "준비 중" 텍스트 | ✅ 바로 가능 |
| 1:1 문의 | 채팅 연결 | ✅ 바로 가능 |

**세팅 위치**: 카카오 채널 관리자 > 1:1 채팅 > 채팅 설정 > 채팅방 메뉴
