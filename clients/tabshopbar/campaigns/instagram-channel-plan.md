## 1. 채널 컨셉 & 방향성

### 한 줄 정의
> **"와인 초보도 편하게 들어오는 일상 와인 라이프 채널"**

### 채널 성격 규정

| 항목 | 규정 | 이유 |
|------|------|------|
| **우리는** | 와인과 노는 법을 보여주는 친구 | 가르치는 선생님 X, 추천하는 소믈리에 X |
| **우리의 톤은** | "같이 마시자" | "추천드립니다" X, "엄선했습니다" X |
| **우리의 주인공은** | 와인을 즐기는 사람 | 와인 보틀 자체 X |
| **우리의 경쟁자는** | 스타벅스 인스타, 라이프스타일 카페 채널 | 와인 전문 채널 X |
| **성공 기준은** | "나도 저렇게 마시고 싶다" | "와인 공부가 됐다" X |

### 채널 포지셔닝

```
            전문적/교육
               |
     와인21     |    와인폴리
     와인나라    |    비노파라다이스
               |
 매거진 ───────┼─────── 리얼/일상
               |
         ★탭샵바★
               |    무드 카페 채널들
     와인바     |    (온도, 블루보틀 등)
     공식 채널들  |
               |
            캐주얼/감성
```

탭샵바 인스타그램의 벤치마크는 **와인 전문 채널이 아니라 라이프스타일 카페 채널**이다. 블루보틀이 커피를 '일상의 특별함'으로 보여주듯, 탭샵바는 와인을 '일상의 특별함'으로 보여준다.

## 2. 타겟

### 핵심 타겟 정의

| 순위 | 페르소나 | 나이 | 키워드 | 인스타 행동 패턴 |
|------|---------|------|--------|-------------|
| 1순위 | 탐험가 지은 | 25-32 여성 | 새로운 경험, 맛집 탐방, SNS 활발 | 저장 많이 함, 스토리 공유, 태그 |
| 2순위 | 여유파 민준 | 28-35 남성 | 퇴근 후 한잔, 데이트, 혼술 | 릴스 시청 많음, 저장 위주 |
| 3순위 | 커플 수아&재현 | 24-30 커플 | 데이트 장소, 분위기 | 위치 태그 확인, 피드 참고 |

### 타겟 인사이트 → 콘텐츠 연결

| 타겟의 마음 | 콘텐츠 방향 |
|-----------|---------|
| "와인 마셔보고 싶은데 어디 가지?" | → 매장 분위기 + 셀프 탭 체험 릴스 |
| "이번 주말 데이트 어디 가지?" | → 도산/여의도점 분위기 + 페어링 추천 |
| "퇴근하고 뭐 마시지?" | → "오늘의 한 잔" 시리즈 + 위치 태그 |
| "와인 잘 모르는데 창피하지 않을까?" | → "처음이라도 괜찮아" 시리즈 |
| "가격이 얼마나 할까?" | → 구체적 가격 표기 ("잔당 3,900원부터") |

## 4. 소셜 커뮤니케이션 방향

### 톤앤매너 (카피)

**브랜드 보이스: "와인 잘 아는 재미있는 친구"**

| 원칙 | 설명 | O 예시 | X 예시 |
|------|------|--------|--------|
| 친구 말투 | 해요체/반말 혼용. 격식 없이 | "이거 진짜 맛있어" | "추천드립니다" |
| 쉬운 말 | 와인 전문용어 최소화 | "새콤한 맛이 좋아서" | "산미가 우수한" |
| 짧은 문장 | 한 줄에 한 메시지 | "퇴근하고 한 잔" | "바쁜 하루를 보내신 후 특별한 한 잔으로" |
| 공감 톤 | 감탄보다 공감 | "이런 날 있지?" | "정말 놀라운 맛입니다!" |
| 행동 유도 | 모든 게시물에 CTA | "오늘 탭 한번 눌러봐" | (CTA 없는 정보만 나열) |

**금지어**: 엄선, 프리미엄, 럭셔리, 격조, 소믈리에 추천, 빈티지 셀렉션, ~해드립니다, ~모시겠습니다, 고객님

### 해시태그 전략

## 5. 비주얼 톤앤매너 (BI 시스템 기반)

### 컬러 팔레트

TSB BI System v1.0에 따른 인스타 채널 전용 컬러 가이드:

| 용도 | 컬러 | HEX | 사용처 |
|------|------|-----|-------|
| **메인 배경** | TSB Black | #000000 | 텍스트 오버레이 배경, 메인 피드 |
| **메인 배경 (밝은)** | TSB White | #FFFFFF | 캐러셀 내부, 정보형 콘텐츠 |
| **포인트 1** | TSB Blue | #156DFF | 하이라이트, CTA, 강조 |
| **포인트 2** | TSB Yellow | #FFC100 | 프로모션, 혜택, 주목 |
| **포인트 3** | TSB Red | #F43131 | 한정, 마감, 경고 |
| **서브 (틴트)** | Sub Blue 1-4 | 80%~20% | 배경, 그래프, 보조 |
| **서브 (틴트)** | Sub Yellow 1-4 | 80%~20% | 배경, 보조 |
| **서브 (틴트)** | Sub Red 1-4 | 80%~20% | 배경, 보조 |

### 컬러 사용 원칙
1. **동일 매체 내에서 배경 컬러는 최소화** (1-2색)
2. **TSB 3색(Blue/Yellow/Red) 동시 사용은 로고 영역에서만**
3. **사진 위 텍스트**: Black 또는 White만 (컬러 텍스트 금지)
4. **컬러 배경 위 로고**: Black 또는 White만 (BI p.19 규정)

### 타이포그래피

| 용도 | 서체 | 웨이트 | 사이즈 기준 |
|------|------|--------|---------|
| **영문 타이틀** | P22 Underground | Heavy | 대문자 또는 소문자 |
| **영문 서브** | P22 Underground | Demi Bold | - |
| **한글 타이틀** | Pretendard | Extra Bold / Bold | H 100% |
| **한글 서브** | Pretendard | Bold | H 75% |
| **한글 본문** | Pretendard | Semi Bold / Medium | H 50% |

### 사진 톤앤매너

| 요소 | DO | DON'T |
|------|-----|-------|
| **조명** | 따뜻한 매장 조명, 캔들 느낌, 자연광 | 형광등, 차가운 화이트, 플래시 직사 |
| **색감** | 웜톤 보정, 약간의 빈티지, 채도 살짝 높게 | 차가운 블루톤, 과한 필터, 무채색 |
| **구도** | 가까운 거리감, 1인칭 시점, 손이 보이는 컷 | 먼 전경, 풍경 위주, 무인 테이블 |
| **인물** | 자연스러운 표정, 웃는 얼굴, 캐주얼 복장 | 정장, 포즈 과한 모델컷, 긴장한 표정 |
| **소품** | 와인잔(손에 들린), 안주, 셀프 탭, 따뜻한 인테리어 | 와인잔만 단독, 빈 테이블, 격식 세팅 |
| **주인공** | 사람 > 와인. "와인을 즐기는 사람"이 중심 | 와인 보틀만 클로즈업 (카탈로그 느낌) |

### 피드 그리드 전략 (9칸 통일감)

```
┌────────────┬────────────┬────────────┐
│ [사진/무드] │ [캐러셀]   │ [릴스 커버] │  ← 1행: 다양한 포맷 믹스
├────────────┼────────────┼────────────┤
│ [사진/무드] │ [사진/인물] │ [캐러셀]   │  ← 2행
├────────────┼────────────┼────────────┤
│ [릴스 커버] │ [사진/무드] │ [사진/인물] │  ← 3행
└────────────┴────────────┴────────────┘

컬러 톤: 전체적으로 웜톤 + Black/White 텍스트
→ 9칸을 펼쳤을 때 "따뜻하고 캐주얼한 와인바" 무드가 느껴져야 함
```

**그리드 규칙**:
- 텍스트 위주 콘텐츠는 **연속 배치 금지** (최소 1칸 간격)
- 릴스 커버도 피드 톤앤매너에 맞춰 제작 (자동 썸네일 X)
- 캐러셀 표지는 항상 비주얼 중심 (텍스트 최소화)

## 8. 콘텐츠 제작 가이드

### 피드 이미지 제작 규격

| 항목 | 규격 |

| 항목 | 규격 |
|------|------|

| 자막 | 필수 (무음 시청 대비). Pretendard Bold, White + Black 그림자 |
| 커버 | 별도 제작 (피드 그리드 톤 유지) |
| BGM | 트렌드 오디오 활용, 저작권 확인 필수 |

| 항목 | 규격 |
|------|------|

| 장수 | 3-7장 (최적 5장) |

| 내부 | 정보 전달, 1장 1메시지 원칙 |

| 서체 | 타이틀 - Pretendard Extra Bold, 본문 - Pretendard Medium |
| 배경 | TSB White 또는 Sub 틴트 (밝은 톤) |
