## 1. 프로필 (브랜드 첫 인상)

### 프로필 이미지

```
"오늘 뭐 마시지?" 의 새로운 답.
아메리카노 대신, 탭에서 시작하는 오늘의 와인 한 잔.

한 잔 3,900원부터 | 80종 셀프 탭 | 6개 매장
채널 추가하면 첫 방문 혜택을 드려요!
```

→ 혜택 강조형. 신규 친구 추가 유인 효과.

## 3. 채팅방 메뉴 버튼 (6개)

### 현재 → 변경

| 순서 | 현재 | 변경 후 | 변경 이유 |
|------|------|---------|---------|
| 1 | 와인 테이스팅 QR | **이달의 추천** | 시의성 높은 정보 (프로모션 없을 때는 "이달의 추천 와인 TOP 3") |
| 2 | 매장정보 | **매장 찾기 & 예약** | 예약까지 원스톱 |
| 3 | 인스타그램 | **셀프 탭 이용 방법** | 신규 고객 핵심 FAQ |
| 4 | 충전&잔액 | **충전 & 잔액 확인** | 유지 (기존 이용자) |
| - | 없음 | **쿠폰함** | 신규 (혜택 확인) |
| - | 없음 | **1:1 문의** | 신규 (CS 채널) |

### 메뉴별 연결

**1) 이달의 추천**
- 프로모션 있는 달: 프로모션 상세 페이지 링크
- 프로모션 없는 달: "이달의 추천 와인 TOP 3" 카카오 포스트
- 갱신 주기: 월 1회

**2) 매장 찾기 & 예약**
- 매장 리스트 페이지 (주소, 전화번호, 운영시간, 예약 링크)

| 매장 | 운영시간 | 키친 L.O |
|------|---------|---------|
| 동대문 두타점 | 10:30 - 23:30 | 22:00 |
| 청계천 삼일빌딩점 | 11:00 - 24:00 | 22:00 |
| 도산대로점 | 11:00 - 24:00 | 22:00 |
| 여의도점 | 11:00 - 24:00 | 22:00 |
| 합정점 | 11:00 - 24:00 | 22:00 |
| 상암점 | TBD | TBD |

**3) 셀프 탭 이용 방법**
- 이미지 카드 3-4단계 가이드:
  1. 카운터에서 와인 카드 충전 (1만원 / 3만원 / 5만원)
  2. 탭 존에서 와인 선택 → 카드 태그 → 원하는 양만큼 따르기
  3. 마음에 드는 와인 → SHOP에서 보틀 구매!
  4. 카운터에서 잔액 환불 OK
- 톤: "처음이라 긴장되죠? 걱정 마세요, 30초면 됩니다"

**4) 충전 & 잔액 확인**
- 기존 기능 유지

**5) 쿠폰함**
- 사업자 인증 완료 전: "준비 중" 안내 또는 웹링크 대체
- 인증 후: 상시 쿠폰 "첫 방문 셀프 탭 1잔 무료" 운영

**6) 1:1 문의**
- 자동 응답 키워드 설정:

| 키워드 | 자동 응답 |
|--------|---------|
| "예약" | 매장별 예약 링크 안내 |
| "주차" | 매장별 주차 정보 |
| "메뉴" | 메뉴 페이지 링크 |
| "프로모션" / "이벤트" | 이달의 프로모션 안내 |
| "위치" / "주소" | 매장 찾기 안내 |
| 미매칭 | "담당자가 확인 후 답변드릴게요! (영업시간 내 답변)" |

## 4. 웰컴 메시지

### 현재 (AS-IS)
```
안녕하세요 TAP SHOP BAR입니다!
매장정보, 운영시간, 주차, 예약 안내...
```
→ 정보만 나열. 브랜드 경험 없음.

### 변경 (TO-BE)

**친구 추가 직후 즉시 발송**:
```
어서 와요, 탭샵바! 🍷

"오늘 뭐 마시지?" 고민될 때,
아메리카노 대신 탭에서 시작해 보세요.

🎁 채널 추가 감사 혜택
→ 첫 방문 시 셀프 탭 1잔 무료!
[쿠폰 받기]

📍 가까운 매장 찾기
[매장 보기]
```

**포인트**:
- 첫 줄에 브랜드 톤 ("어서 와요, 탭샵바!")
- 브랜드 슬로건 자연스럽게 녹임 ("아메리카노 대신")
- 즉시 혜택 제공 → 매장 방문 전환 유도
- 쿠폰은 사업자 인증 전까지 "첫 방문 시 이 메시지 캡처해서 보여주세요" 형태로 임시 운영

### 추후 자동 메시지 (사업자 인증 후 세팅)

| 시점 | 내용 | 목적 |
|------|------|------|
| D+1 | 셀프 탭 이용 가이드 (3단계) | 진입 장벽 낮추기 |
| D+7 | "아직 탭 안 눌러봤어요?" + 이번 주 추천 와인 | 미방문 리마인드 |
| D+14~ | 일반 CRM 편입 (프로모션/소식 정기 수신) | 리텐션 |

→ D+1, D+7은 "스마트 메시지" 또는 외부 CRM 연동 필요. 기본 세팅에서는 웰컴 메시지(즉시)만 먼저 세팅.

## 6. 브랜딩 포인트 정리

카카오 채널에서 **"탭샵바다움"**이 느껴지는 터치포인트:

| 터치포인트 | 브랜딩 포인트 | 현재 상태 |
|-----------|------------|---------|
| **프로필 이미지** | TSB Symbol 1 컬러, Black 배경 | 확인 필요 |
| **채널 소개글** | "아메리카노 대신" 슬로건 + 혜택 | 기본 텍스트 |
| **배너** | TSB 톤 (Black + White, BI 서체) | 미설정 또는 기본 |
| **웰컴 메시지** | 친구 톤 + 혜택 + 매장 안내 | "안녕하세요" 기본 |
| **채팅방 메뉴** | 고객 행동 기반 6개 구성 | 4개 (기능 위주) |
| **FAQ** | 친근한 톤, "모르는 게 당연한 곳" | 미설정 |
| **자동 응답** | 즉각적이고 친근한 톤 | 기본 봇 메시지 |

→ 모든 터치포인트에서 **"몰라도 괜찮아, 여기서 시작하면 돼"** 메시지가 일관되어야 한다.
