## 시리즈 1: 탭샵바 플레이스

> **"6개의 탭, 6개의 공간"** - 지점별 공간 소개

### 콘셉트
각 지점만의 분위기와 특색을 보여주는 공간 소개 시리즈. 같은 탭샵바인데 지점마다 다른 매력이 있다는 것을 전달.

### 포맷별 기획

| 포맷 | 연출 방향 | 레퍼런스 |
|------|----------|---------|
| **릴스** (15~30초) | 입구 → 내부 → 탭 존 → 좌석 순서로 공간을 훑는 시네마틱 워크스루. BGM + 자막 오버레이. | [참고 릴스](https://www.instagram.com/reels/DAIyAESSInG/) |
| **단일 이미지** | 각 지점의 시그니처 앵글 1장. 통일된 보정 톤으로 시리즈감 형성. | @cafe.onion 공간 무드샷 |

### 해시태그 구성
```
#탭샵바플레이스 #탭샵바 #tapshopbar #[지점명]
예: #탭샵바플레이스 #탭샵바합정 #합정와인바 #tapshopbar
```

### 6개 지점 촬영 순서 (안)
1. 합정점 (플래그십 느낌)
2. 도산점 (프리미엄 무드)
3. 여의도점 (직장인 퇴근 후)
4. 청계점 (도심 속 한 잔)
5. 동대문점 (숨은 공간)
6. 상암점 (NEW - 6번째 탭)

### 제작 포인트
- 릴스: 핸드헬드 or 짐벌, 웜톤 조명 살리기, 15~30초 내 마무리
- 단일 이미지: 와이드 앵글 + 따뜻한 조명, 사람 1~2명 실루엣 포함 권장
- 각 지점 촬영 시 **릴스 + 단일 이미지 동시 확보** (효율)

## 시리즈 2: 탭샵바 페어링

> **"이 와인엔 이거지"** - 와인 + 안주 페어링

### 콘셉트
와인과 음식의 조합을 보여주는 시리즈. "뭐 시킬지 모르겠을 때" 참고할 수 있는 실용 콘텐츠.

### 포맷별 기획

| 포맷 | 연출 방향 | 참고 |
|------|----------|------|
| **릴스** (15~30초) | 블루보틀 스타일 모션그래픽. 음식이 하나씩 등장 → 와인과 매칭되는 모션. 깔끔한 스톱모션 or 모션그래픽 전환. | @bluebottlekorea 제품 소개 릴스 |
| **카드뉴스** (3~5장) | 1장: 페어링 타이틀 / 2~3장: 메뉴+와인 조합 / 마지막: CTA ("다음엔 이렇게 시켜보세요") | @winefolly 인포그래픽 |
| **단일 이미지** | 접시 위 안주 + 와인잔이 함께 있는 테이블 컷. 음식이 하나둘 등장하는 스톱모션 GIF도 가능. | 매장 실제 촬영 |

### 해시태그 구성
```
#탭샵바페어링 #탭샵바 #와인페어링 #tapshopbar
예: #탭샵바페어링 #레드와인안주 #와인안주추천 #tapshopbar
```

### 콘텐츠 아이디어
- 레드 와인 + 스테이크/치즈 조합
- 화이트 와인 + 해산물/샐러드 조합
- "직원 추천 페어링" (스태프 픽)
- 시즌별 특별 페어링

### 제작 포인트
- 릴스: 모션그래픽 필요 → 디자이너/모션 작업 필요 (공수 중간~높음)
- 카드뉴스: 메뉴 촬영본 + Pretendard 서체 + TSB 톤
- 저장(북마크) 유도 콘텐츠 → CTA 명확하게

## 시리즈 3: 탭샵바 씬

> **"탭샵바에서 벌어지는 일들"** - 현장 스케치

### 콘셉트
대관 행사, 특별 이벤트, 유명인 방문 등 탭샵바에서 벌어지는 현장의 생생한 장면을 보여주는 시리즈. "여기 이런 곳이야?" 하는 발견의 재미.

### 포맷별 기획

| 포맷 | 연출 방향 | 참고 |
|------|----------|------|
| **릴스** (15~60초) | 현장 분위기 중심 하이라이트 편집. BGM + 간단 자막. 행사 전/중/후 타임랩스. | @speedboatbar 현장감 릴스 |
| **단일 이미지** | 행사 베스트 컷 1장. 사람들이 즐기는 모습 중심. 유튜브 등장 컷 편집도 포함. | 강승윤 편 등 |

### 해시태그 구성
```
#탭샵바씬 #탭샵바 #tapshopbar #[이벤트명]
예: #탭샵바씬 #탭샵바대관 #와인파티 #tapshopbar
```

### 콘텐츠 소스
- 대관 행사 현장 촬영 (릴스 + 사진)
- 유튜브 등장 편 캡처/클립 (강승윤 편 등)
- VIP/셀럽 방문 기록 (허락 받은 것만)
- 시즌 이벤트 (크리스마스, 발렌타인 등)

### 제작 포인트
- 행사 시 **반드시 영상/사진 동시 촬영** (놓치면 소재 없음)
- 유튜브 편집 클립: 15초 하이라이트로 재편집
- 현장감 > 퀄리티. 약간 떨리는 핸드헬드도 OK
- 출연자/참석자 초상권 확인 필수

## 시리즈 4: 탭샵바 뉴

> **"새로 왔어요"** - 신메뉴 소개

### 콘셉트
신메뉴 출시 때마다 발행하는 소개 시리즈. 메뉴 사진 + 짧은 설명 + 페어링 추천까지.

### 포맷별 기획

| 포맷 | 연출 방향 | 참고 |
|------|----------|------|
| **카드뉴스** (3~5장) | 1장: "새로 왔어요" 표지 / 2~4장: 메뉴 개별 소개 (사진+이름+한줄설명) / 마지막: 페어링 추천+CTA | @cafeknotted_kr 시즌 메뉴 |
| **단일 이미지** | 신메뉴 전체를 테이블에 펼친 "풀세팅" 컷 1장. 또는 대표 메뉴 1개 클로즈업. | - |

### 해시태그 구성
```
#탭샵바뉴 #탭샵바 #신메뉴 #tapshopbar
예: #탭샵바뉴 #탭샵바신메뉴 #와인안주 #tapshopbar
```

### 카드뉴스 구성 예시
```
[1장] 이번 달 탭샵바 테이블에 새로 올라온 것들. (메뉴 전체 테이블샷)
[2장] 트러플 감자튀김 — 바삭한 튀김 위 트러플 오일 (사진+설명)
[3장] 부라타 카프레제 — 신선한 부라타에 발사믹 글레이즈 (사진+설명)
[4장] 이 메뉴엔 이 와인 🍷 — 페어링 추천 + "직접 맛보러 오세요"
```

### 제작 포인트
- 메뉴 촬영은 메뉴 출시 전에 미리 진행 (출시 당일 바로 발행)
- 카드뉴스 = 단일 이미지 소스가 겹침 → 동시 제작
- **저장 유도** 문구 필수 ("저장해두고 다음에 시켜보세요")
- 페어링 추천은 박창규 확인 후 확정

## 시리즈 5: 이달의 탭

> **"이번 달은 이 와인"** - 월간 프로모션 & 와인 소개

### 콘셉트
매달 진행하는 프로모션과 연계하여 와인을 매력적이고 재밌게 소개하는 시리즈. 와인 정보를 "교육"이 아닌 "발견"의 느낌으로.

### 포맷별 기획

| 포맷 | 연출 방향 | 참고 |
|------|----------|------|
| **카드뉴스** (3~5장) | 와인 정보성 콘텐츠. 1장: 와인 비주얼 / 2~3장: 맛/향/스토리를 쉽게 / 마지막: 프로모션 안내+CTA | @winefolly 인포그래픽 |
| **단일 이미지** | 이달의 프로모션 와인 1병 클로즈업 + 가격/혜택 텍스트 오버레이. | 타라파카 프로모션 등 |

### 해시태그 구성
```
#이달의탭 #탭샵바 #와인추천 #tapshopbar
예: #이달의탭 #3월의와인 #타라파카 #와인추천 #tapshopbar
```

### 와인 소개 톤 방향
와인을 "진지하게 알려주는" 게 아니라 **재밌게 발견하게** 만든다.

**DO:**
- "산미가 높아서 치킨이랑 찰떡" (일상 연결)
- "이름은 어려운데 맛은 쉬워요" (허들 낮추기)
- "3만원대에 이 맛이면 사기" (가성비 어필)

**DON'T:**
- "이 와인은 부르고뉴 피노 누아로..." (교과서)
- "바디감이 미디엄 풀이며 탄닌이..." (전문용어 남발)

### 제작 포인트
- 매달 초에 그 달 프로모션 와인 확정 → 바로 콘텐츠 제작
- 와인 정보는 **3줄 이내**로 핵심만 (이름, 맛 한줄, 추천 상황)
- 프로모션 가격/혜택은 명확하게 표기
