## 2. 오프라인 매장 전략 (컨설턴트 관점)

### 2-1. 고객 동선별 터치포인트 설계

고객이 매장에 들어와서 나갈 때까지, 타라파카 프로모션을 **최소 5번** 접하게 설계합니다.

```
[입장] → 스탠드 배너 / 입구 TV (1차 인지)
   ↓
[착석] → 테이블 미니 POP / 메뉴판 삽입물 (2차 상세 확인)
   ↓
[탭 이동] → 탭 디스펜서 LCD / 탭 존 추천 표시 (3차 시음 유도)
   ↓
[주문] → 포스 듀얼 모니터 / 스태프 추천 멘트 (4차 페어링 제안)
   ↓
[식사 중] → 매장 TV 슬라이드 (5차 리마인드 / 보틀 구매 유도)
   ↓
[퇴장] → "마음에 드셨으면 보틀로도 가져가세요" (SHOP 연결)
```

### 2-3. 가격 심리학 활용 (3단계 앵커링)

오프라인 매장에서 3단계 가격 구조를 의도적으로 활용합니다.

```
             ₩62,100 에티케타 네그라
                  ↑ 앵커 (이걸 보면 아래가 합리적으로 느껴짐)

        ★ ₩33,300 그란 리제르바 ← 가장 밀고 싶은 구간
              "조금만 더 내면 확 다른 맛"

           ₩18,900 리제르바 ← 진입 장벽 낮추기
              "일단 이것부터 시작"
```

**POP/메뉴판 배치 원칙**:
- 에티케타 네그라(₩62,100)를 **가장 위/왼쪽에** 배치 → 앵커 효과
- 그란 리제르바(₩33,300)를 **중앙/강조** → "BEST" 표시
- 리제르바(₩18,900)를 **"처음이라면"** 문구와 함께 → 입문자 안심

**스태프 업셀 스크립트**:
```
[리제르바 주문 시]
"리제르바도 좋은데, 3만 원대 그란 리제르바가 확 다르거든요.
한 잔만 드셔보실래요? 페어링하면 10% 할인이라 가격 차이 별로 안 나요."

[와인만 주문 시]
"타라파카 드시는 거면, 안주 하나 같이 시키시면 와인도 10% 할인돼요.
하몽 멜론이나 그릴드 오이스터 추천드려요!"
```

### 2-5. TAP → SHOP 연결 (보틀 테이크아웃 유도)

탭샵바의 가장 큰 강점인 TAP → SHOP 구조를 프로모션에 적극 활용합니다.

| 전략 | 실행 방법 |
|------|---------|
| **탭 존 옆 보틀 디스플레이** | 타라파카 7종 보틀을 탭 기기 바로 옆에 진열, "마음에 드시면 한 병 가져가세요" |
| **스태프 추천 멘트** | "이 와인 마음에 드셨으면, SHOP에서 보틀로도 사 가실 수 있어요. 매장 가격이 시중보다 저렴합니다" |
| **보틀 구매 안내 POP** | 탭 존에 "잔 와인이 마음에 들면 → SHOP에서 보틀로!" 안내 부착 |
| **보틀 세트 할인** | 보틀 + 안주 세트 구성 시 추가 할인 검토 (수입사 협의) |
