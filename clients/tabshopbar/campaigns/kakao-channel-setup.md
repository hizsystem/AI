## 1. 현황 분석 & 방향성

### 현재 상태
- **채널명**: TAP SHOP BAR
- **친구 수**: 21,749명 (유의미한 모수 이미 확보)
- **채팅방 메뉴**: 와인 테이스팅 QR / 매장정보 / 인스타그램 / 충전&잔액
- **웰컴 메시지**: 기본 매장 안내 (운영시간, 주차, 예약 링크)
- **사업자 인증**: 1차 반려, 재신청 중 (인증 완료 시 메시지 발송량 확대 가능)

### 문제점
1. **채널 홈이 브랜드를 대변하지 못함** - 방문 시 "탭샵바가 뭔지" 바로 와닿지 않음
2. **채팅방 메뉴가 비활성화 상태** - 실제 고객이 자주 쓰는 메뉴 부재
3. **웰컴 메시지가 기능적** - 환영 + 혜택 제공의 기회를 놓치고 있음
4. **프로모션/시즌 연동 없음** - 3월 타라파카 같은 프로모션과 채널이 연결되지 않음

### 리뉴얼 방향
> **"카카오 채널 = 탭샵바의 디지털 카운터"**
> 매장에 가기 전 모든 정보를 얻고, 혜택을 받고, 궁금한 걸 해결하는 곳

## 7. 쿠폰 탭 운영

### 쿠폰 운영 원칙
1. **상시 1개 + 시즌 1개** - 쿠폰이 너무 많으면 가치가 떨어진다
2. **사용 조건은 단순하게** - 복잡한 조건은 사용률을 떨어뜨린다
3. **유효기간은 넉넉하게** - 30일 이상. 급한 마감은 탭샵바 톤이 아니다
4. **프로모션 연계 필수** - 쿠폰 단독이 아닌 진행 중인 프로모션과 연결

### 3월 쿠폰 운영안

| 쿠폰명 | 혜택 | 유효기간 | 조건 | 비고 |
|-------|------|---------|------|------|
| **[상시] 탭 1잔 무료** | 셀프 탭 1잔 (리제르바급) 무료 | 채널 추가 후 30일 | 첫 방문 시 | 신규 친구 확보용 |
| **[3월] 타라파카 보너스** | 타라파카 와인+푸드 세트 주문 시 디저트 서비스 | 3/1 ~ 3/31 | 페어링 세트 주문 | 프로모션 부스터 |

### 쿠폰 운영 사전 조건
- **사업자 인증 완료 필수**: 현재 재신청 중 → 인증 완료 후 쿠폰 발급 가능
- **인증 전 대안**: 소식 탭에서 "이 게시물 캡처해서 보여주시면 ~혜택" 형태로 임시 운영
- **담당**: 박창규 (사업자 인증), 우성민 (쿠폰 기획/등록)

## 8. 웰커밍 시나리오

### 현재 웰컴 메시지 (AS-IS)
```
안녕하세요 TAP SHOP BAR입니다!
매장정보, 운영시간, 주차, 예약 안내...
```
→ 정보만 나열. 브랜드 경험 없음.

### 리뉴얼 웰컴 메시지 (TO-BE)

#### STEP 1: 즉시 발송 (친구 추가 직후)

**메시지 시안 A (간결형)**:
```
반가워요! 탭샵바에 오신 걸 환영해요 🍷

여기선 와인 몰라도 괜찮아요.
셀프 탭 앞에 서면, 그냥 눌러보면 돼요.
잔 단위 수천 원부터. 80종 중에 분명 내 취향이 있을 거예요.

첫 방문이라면? 👇
[셀프 탭 이용법 보기]

가까운 매장 찾기 👇
[매장 찾기]
```

**메시지 시안 B (혜택 강조형)**:
```
어서 와요, 탭샵바! 🍷

"오늘 뭐 마시지?" 고민될 때,
아메리카노 대신 탭에서 시작해 보세요.

🎁 채널 추가 감사 혜택
→ 첫 방문 시 셀프 탭 1잔 무료!
[쿠폰 받기]

📍 가까운 매장 찾기
[매장 보기]
```

> **추천**: B안 (혜택 강조형) - 신규 친구 → 매장 방문 전환율 극대화

#### STEP 2: D+1 (다음 날) - 자동 발송

```
어제 탭샵바 채널 추가해주셔서 고마워요!

혹시 셀프 탭 처음이시라면, 이것만 알면 돼요:
1️⃣ 카운터에서 카드 충전
2️⃣ 마음에 드는 와인 앞에서 탭!
3️⃣ 안 맞으면 다른 거. 80종이니까!

👉 [셀프 탭 이용 가이드 보기]
```

#### STEP 3: D+7 (일주일 후) - 미방문 시

```
혹시 아직 탭 안 눌러봤어요? 😊

이번 주 탭에 새로 올라온 와인들 소개할게요.
[이번 주 추천 와인 3 보기]

📍 가까운 매장: [매장 찾기]
🎁 첫 방문 쿠폰 아직 살아있어요!
```

### 웰커밍 시나리오 플로우

```
친구 추가
  │
  ├─→ [즉시] STEP 1: 환영 + 혜택 + 매장 안내
  │
  ├─→ [D+1] STEP 2: 셀프 탭 이용 가이드
  │
  ├─→ [D+7] STEP 3: 미방문 시 리마인드
  │         (방문 완료 시 → 발송 안 함)
  │
  └─→ [D+14~] 일반 CRM 세그먼트 편입
              → 프로모션/소식 정기 수신
```

> **참고**: D+1, D+7 자동 발송은 카카오 채널의 "스마트 메시지" 또는 외부 CRM 툴 연동 필요. 사업자 인증 완료 후 설정 가능.

## 9. 시즌/프로모션 연동 리뉴얼 가이드

### 프로모션 시 채널 업데이트 체크리스트

프로모션이 시작될 때마다 아래 항목을 일괄 업데이트:

- [ ] **쿠폰 탭** → 프로모션 연계 쿠폰 등록
- [ ] **FAQ** → 프로모션 관련 FAQ 추가 (필요 시)
- [ ] **웰컴 메시지** → 프로모션 기간 한정 혜택 반영 (선택)
