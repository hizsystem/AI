# 디자인 브리프: 매장 운영 디자인 1~3순위

> 탭샵바 (TAP SHOP BAR) | 작성: 2026-03-19
> 디자이너 요청용 | 우선순위: 1~3순위 선작업

---

## 요청 배경

운영 미팅(3/16) 결과, 프로모션과 별개로 **매장 운영에 상시 필요한 디자인물** 3종이 도출됨.
프로모션 내용은 매달 바뀌지만, 이 3종은 **내용만 교체해서 반복 사용**하는 템플릿 포맷이 필요.

**핵심 요구**: 프로모션이 바뀌어도 텍스트/이미지만 교체하면 되는 **공통 포맷** 설계

---

## 공통 BI 준수사항

| 항목 | 규정 |
|------|------|
| **컬러** | TSB Black(#000000) 배경 기본 / TSB White(#ffffff) 텍스트 / 강조: TSB Blue(#156dff) 또는 TSB Yellow(#ffc100) |
| **한글 서체** | Pretendard — 타이틀 Bold~Extra Bold, 본문 Medium |
| **영문 서체** | P22 Underground |
| **로고** | Symbol 1 (TSB 원형 3개) 또는 Primary (TAP SHOP BAR) — 배경 대비에 따라 Positive/Negative 선택 |
| **톤** | 깔끔, 심플, 정보 전달 최우선. 와인바답되 격식적이지 않게 |

## 1순위: 카운터 스탠드

### 현황 & 문제

카운터에 네이버 이벤트, 탭 충전 방법, 프로모션 등 **여러 내용이 각각 다른 디자인으로 노출**되고 있음.
종이를 끼워 넣은 형태라 보기에도 퀄리티가 떨어지고, 통일감이 없음.

### 목표

**1개 스탠드에 여러 장을 넣되, 모든 장이 같은 포맷**으로 보이게.
프로모션이 바뀌면 해당 장만 교체하면 되는 구조.

### 콘텐츠 유형 (장별)

스탠드에 들어갈 내용을 **유형별로 정리**. 각 유형이 1장씩.

| 유형 | 상시/변동 | 내용 예시 |
|------|----------|----------|
| **A. 탭 이용 가이드** | 상시 | 탭 충전 방법, 잔 단위 시음 안내, 가격대 안내 |
| **B. 이달의 프로모션** | 월별 변동 | 수입사 프로모션 (예: 타라파카 페어링 10%), 시즌 이벤트 |
| **C. 멤버십/포인트 안내** | 상시 | 탭 와인 포인트 적립, 충전 혜택 |
| **D. 네이버/SNS 안내** | 상시 | 네이버 플레이스 리뷰 이벤트, 인스타 팔로우 혜택, QR코드 |

### 템플릿 포맷 설계

**공통 레이아웃** (모든 장 동일 구조):

```
┌─────────────────────────────────┐
│  [TSB Black 배경]               │
│                                 │
│  ── 상단 띠 (TSB Blue) ───────  │
│  유형 라벨                      │
│  (예: "TAP GUIDE" / "MONTHLY"  │
│   / "MEMBERSHIP" / "FOLLOW")   │
│  ────────────────────────────── │
│                                 │
│   메인 타이틀                    │
│   (Pretendard Extra Bold/White) │
│                                 │
│   본문 내용                      │
│   (Pretendard Medium/White)     │
│   - 핵심 정보 2~3줄             │
│   - 숫자/가격은 Yellow 강조      │
│                                 │
│   [QR코드 or 아이콘]  (해당 시)  │
│                                 │
│  ── 하단 ─────── TSB Symbol ──  │
│                                 │
└─────────────────────────────────┘
```

**디자인 포인트**:
- 상단 컬러 띠로 **유형 구분** → 어떤 장이든 한눈에 "이건 프로모션", "이건 가이드" 인식
- 띠 컬러 규칙: A(가이드)=Blue / B(프로모션)=Yellow / C(멤버십)=Blue / D(SNS)=Blue
- 본문 영역은 텍스트 교체만 하면 되는 **고정 그리드**
- QR코드 들어갈 자리 사전 확보

### 각 장별 콘텐츠 시안

**A. 탭 이용 가이드**

```
[ TAP GUIDE ]

탭에서 직접 따라 마셔보세요

1. 카운터에서 탭카드 충전
2. 탭 기기에 카드 터치
3. 원하는 와인 선택 → 잔에 따르기

80여 종 와인 | 잔당 ₩3,000~

처음이어도 괜찮아요.
한 잔씩 시음하면서 취향을 찾아보세요.
```

**B. 이달의 프로모션** (4월 예시 — 매월 교체)

```
[ MONTHLY ]

4월 프로모션

[와인명/브랜드] + [페어링 메뉴]
함께 주문 시 각각 10% 할인

추천 페어링:
[안주명] + [와인명] = ₩XX,XXX

~ 4/30까지 | 전 매장
```

**C. 멤버십/포인트 안내**

```
[ MEMBERSHIP ]

바틀 구매 금액의 10%
탭 와인 포인트로 적립

바틀로 가져가시면
다음에 탭에서 쓸 수 있는 포인트가 쌓여요.

TAP에서 시음 → SHOP에서 구매 → 포인트 적립
```

**D. 네이버/SNS 안내**

```
[ FOLLOW US ]

네이버 플레이스 리뷰 남기면
[혜택 내용]

[QR코드]

@tabshopbar
인스타 팔로우하고 매달 새로운 소식 받기
```

### 규격 & 납품

| 항목 | 내용 |
|------|------|
| **규격** | A5 세로 (148x210mm) — 종이 스탠드/아크릴 스탠드 삽입용 |
| **인쇄** | 300dpi, CMYK |
| **장수** | 4종 (A/B/C/D) |
| **납품** | Figma 원본 + PDF (인쇄용) |
| **교체 빈도** | B만 매월 교체, 나머지 상시 |

## 2순위: 와인 태그

### 현황 & 문제

프로모션 대상 바틀, 라스트 바틀 등을 표시하는 태그가 필요한데:
- 현재 태그가 노후되고 분실돼서 수량 부족
- 디자인이 눈에 잘 안 띔
- 프로모션마다 새로 만들기엔 비효율

### 목표

**2가지 태그 유형**을 템플릿으로 만들어서, 내용만 바꿔 반복 사용.
바틀 넥에 걸거나 선반에 꽂는 방식.

### 태그 유형

| 유형 | 용도 | 빈도 | 색상 코드 |
|------|------|------|----------|
| **프로모션 태그** | 이달의 프로모션 대상 와인 표시 | 월별 교체 | TSB Yellow 배경 |
| **라스트 바틀 태그** | 마지막 1병 남은 와인 표시 | 수시 | TSB Red 배경 |
| **스태프 추천 태그** | 스태프 픽, 추천 와인 | 주별 교체 | TSB Blue 배경 |

### 템플릿 포맷 설계

**공통 레이아웃** (넥 태그):

```
    ┌──── (구멍 — 넥에 걸기) ────┐
    │                            │
    │  [컬러 배경]                │
    │                            │
    │   유형 라벨 (영문)          │
    │   "PROMO" / "LAST ONE"    │
    │   / "STAFF PICK"          │
    │                            │
    │   ─────────────────        │
    │                            │
    │   설명 1줄                  │
    │   (프로모션: 할인율/혜택)    │
    │   (라스트: "마지막 1병")    │
    │   (추천: 추천 이유 한 줄)   │
    │                            │
    │              TSB Symbol    │
    └────────────────────────────┘
```

### 각 유형별 콘텐츠 시안

**프로모션 태그** (Yellow 배경 + Black 텍스트)

```
PROMO
────
페어링 주문 시
10% OFF

~3/31
```

**라스트 바틀 태그** (Red 배경 + White 텍스트)

```
THE LAST ONE
────
마지막 1병

놓치면 다음 입고까지
기다려야 해요
```

**스태프 추천 태그** (Blue 배경 + White 텍스트)

```
STAFF PICK
────
이번 주 추천

[추천 이유 한 줄 — 손글씨 or 인쇄]
```

### 규격 & 납품

| 항목 | 내용 |
|------|------|
| **규격** | 60 x 90mm (세로형, 넥 태그) |
| **재질** | 두꺼운 종이 (300g 이상) 또는 코팅지 — 습기에 강해야 함 |
| **구멍** | 상단 중앙 타공 (끈/고리용) |
| **인쇄** | 양면 (앞면: 정보 / 뒷면: TSB 로고 + 매장 와이파이 등 활용 가능) |
| **수량 기준** | 프로모션 50장 + 라스트 30장 + 추천 30장 (초기 발주) |
| **납품** | Figma 원본 + 인쇄용 PDF (CMYK, 300dpi, 재단선/여백 포함) |

### 운영 방식

- **프로모션 태그**: 매월 프로모션 확정 시 내용 교체 → 인쇄 → 전 매장 배포
- **라스트 바틀 태그**: 1회 대량 인쇄 후 재고로 보유 (내용 고정)
- **스태프 추천 태그**: "추천 이유" 란을 빈칸으로 두고, 스태프가 직접 손글씨 기입하는 방식도 고려

## 3순위: 탭 프로모션 표시 부착물

### 현황 & 문제

탭 디스펜서 LCD에 디지털 이미지를 넣고 있지만, **눈에 잘 띄지 않음**.
고객이 탭 앞에 서서 와인을 고를 때, 어떤 와인이 프로모션 중인지 즉시 인식해야 하는데
디지털 이미지가 다른 정보(와인 리스트, 가격)에 묻혀버림.

### 목표

탭 기기 **외부에 물리적으로 부착**하는 프로모션 표시물.
탭 앞에 서는 순간 "이 와인이 프로모션 중"이라는 걸 1초 만에 인식하게.

### 부착 방식 옵션

| 방식 | 장점 | 단점 | 추천 |
|------|------|------|------|
| **A. 클립형 POP** | 탭 노즐 옆에 집게로 고정, 교체 쉬움 | 떨어질 수 있음 | -- |
| **B. 마그넷 부착** | 탭 기기 금속 부분에 자석 부착, 깔끔 | 기기에 따라 부착 불가 | -- |
| **C. 아크릴 꽂이** | 탭 기기 하단/상단에 아크릴 홀더 고정, 카드 교체 | 설치 필요 | 추천 |
| **D. 행잉형** | 탭 상단에서 매달기, 시선 끌림 | 와인 따를 때 방해 가능 | -- |

> 디자이너분이 합정점 방문하셔서 탭 기기 실물 확인 후 방식 확정 권장 (김희종 슬랙 코멘트 참고)

### 템플릿 포맷 설계 (C. 아크릴 꽂이 기준)

**카드 레이아웃**:

```
┌──────────────────────┐
│ [컬러 배경]           │
│                      │
│  유형 아이콘          │
│  ★ PROMO             │
│                      │
│  혜택 한 줄           │
│  "페어링 시 10% OFF"  │
│                      │
│  기간                 │
│  "~3/31"             │
│                      │
└──────────────────────┘
```

**디자인 포인트**:
- **크기가 작으므로 정보는 최소화** — 유형 + 혜택 + 기간만
- 컬러로 즉시 인식: Yellow = 프로모션 / Red = 라스트 / Blue = 추천
- 와인 태그(2순위)와 **같은 컬러 코드 체계** 사용 → 태그+탭 표시가 연동되는 느낌
- 글씨 크기: 1m 거리에서 읽을 수 있어야 함

### 콘텐츠 시안

**프로모션 카드** (Yellow 배경)

```
★ PROMO
페어링 주문 시 10% OFF
~3/31
```

**라스트 원 카드** (Red 배경)

```
! LAST ONE
마지막 수량
```

**신규 입고 카드** (Blue 배경)

```
NEW
이번 주 새로 들어왔어요
```

### 규격 & 납품

| 항목 | 내용 |
|------|------|
| **카드 규격** | 70 x 50mm (가로형) — 탭 기기 크기에 따라 조정 필요 |
| **아크릴 홀더** | 별도 발주 (디자인팀 규격 확정 후) |
| **재질** | 코팅 카드지 (내구성, 방수) |
| **인쇄** | 양면 컬러 |
| **수량 기준** | 프로모션 80장 + 라스트 40장 + 신규 40장 |
| **납품** | Figma 원본 + 인쇄용 PDF |

## 3종 통합: 컬러 코드 시스템

3가지 디자인물이 **같은 컬러 언어**를 쓰면, 매장 전체에서 통일된 비주얼 시스템이 됨.

| 컬러 | 의미 | 적용처 |
|------|------|--------|
| **TSB Yellow (#ffc100)** | 프로모션 / 할인 / 혜택 | 스탠드 B장, 프로모션 태그, 탭 프로모션 카드 |
| **TSB Red (#f43131)** | 긴급 / 라스트 / 한정 | 라스트 바틀 태그, 탭 라스트 카드 |
| **TSB Blue (#156dff)** | 정보 / 가이드 / 추천 | 스탠드 A/C/D장, 스태프 추천 태그, 탭 신규 카드 |
| **TSB Black (#000000)** | 기본 배경 | 스탠드 전체 배경 |

> 고객 입장에서: Yellow 보이면 "할인이네", Red 보이면 "마지막이네", Blue 보이면 "추천이네" — 학습 효과
