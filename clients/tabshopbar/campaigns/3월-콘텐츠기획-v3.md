## v2 리뷰: 뾰족함 체크

### 1. 4대 필러 비율 — 불균형

v2 6건을 필러별로 분류하면:
- **일상 와인 (목표 40%)**: #1 매장 무드 = 1건 (17%) -- **심각하게 부족**
- **탭 체험 (목표 25%)**: #2 상암점 오픈, #6 합정점 릴스 = 2건 (33%) -- 약간 과다
- **와인 쉽게 (목표 20%)**: #4 신메뉴, #5 타라파카 = 2건 (33%) -- 과다
- **매장/브랜드 (목표 15%)**: #3 강승윤 편 = 1건 (17%) -- 적정

문제: "일상 와인"이 탭샵바의 핵심 필러인데, 이걸 담당하는 콘텐츠가 #1 매장 무드 1건뿐이다. "퇴근하고 한 잔"이라는 감성을 한 달에 1번만 보여주면, 팔로워 머릿속에 탭샵바가 "일상적으로 가는 곳"으로 각인되지 않는다.

### 2. "한 잔 트라이" USP 연결 — 느슨함

| 콘텐츠 | USP 연결 | 판정 |
|--------|---------|------|
| #1 매장 무드 | "퇴근하고 한 잔" → 간접 연결 | 약함. "한 잔"이 셀프 탭의 잔 단위 구매인지, 그냥 분위기인지 구분 안 됨 |
| #2 상암점 오픈 | "가볍게 탭, 오늘의 와인" → 약한 연결 | 보통. 오픈 공지 자체가 목적이라 USP는 부차적 |
| #3 강승윤 편 | "셀프 탭 앞에 서면 누구나" → 좋은 연결 | 좋음. 셀프 탭 체험을 자연스럽게 보여줌 |
| #4 신메뉴 | 페어링 추천으로 우회 연결 | 약함. 신메뉴에 집중하다 보면 "한 잔 트라이"가 빠짐 |
| #5 타라파카 | "비싼 와인도 여기선 한 잔만" → 직접 연결 | 좋음. USP 정타 |
| #6 합정점 릴스 | "10만원짜리도 한 잔만" → 직접 연결 | 좋음 |

6건 중 USP 직접 연결이 2건뿐. 나머지 4건은 "한 잔 트라이"가 없어도 성립하는 콘텐츠. 모든 콘텐츠의 바닥에 USP가 깔려야 하는데, 지금은 절반이 빠져 있다.

### 3. 캡션 후킹력 — 부족

| 콘텐츠 | 현재 캡션 첫 줄 | 후킹 판정 |
|--------|---------------|----------|
| #1 | "퇴근하고 한 잔." | 무드는 있는데 뾰족함이 없음. 어떤 바에서도 쓸 수 있는 말 |
| #2 A | "6번째 탭이 열립니다." | 기존 팬에겐 의미 있지만 신규 유입엔 약함 |
| #2 B | "동대문에서 시작해서..." | 스토리텔링 있지만 첫 줄에서 "이게 뭔데?"가 없음 |
| #3 | "합정점에 손님이 오셨더라고요" | 궁금증 유발 OK. 이모지가 톤에 안 맞음(🍷 삭제 권장) |
| #4 | "이번 달 탭샵바 테이블에 새로 올라온 것들." | 정보 전달용. 스크롤 멈출 이유 없음 |
| #5 | 시안 없음 (카드뉴스 내 텍스트만) | 캡션 시안 부재 |

2030이 스크롤하다 멈추려면, 첫 줄에서 "이거 나 얘기인데?" 또는 "뭐지?" 반응이 나와야 한다. 현재 캡션은 대부분 "정보 전달"에 머물러 있다.

### 4. 시리즈(꼭지) 체계 — 실행 가능성 OK, 차별화 부족

- 5개 시리즈 체계는 잘 짜여 있고, 해시태그도 깔끔하다.
- 다만 시리즈 간 **톤 차이**가 명확하지 않다. #탭샵바플레이스와 #탭샵바씬이 피드에서 구분되려면, 비주얼 필터나 자막 스타일에 차이가 있어야 한다.
- #탭샵바페어링이 3월 기획에 **독립 콘텐츠 0건**. 가장 저장률 높을 시리즈인데 3월에 시작하지 않으면 런칭 모멘텀을 놓친다. (#4 신메뉴에 병합되어 있으나, 이것은 #탭샵바뉴가 메인)

### 5. 빠진 것, 약한 것, 바꿔야 할 것

| 구분 | 내용 |
|------|------|
| **빠진 것** | "일상 와인" 필러 콘텐츠 (퇴근 후 한 잔, 혼술, 주중 와인 등) |
| **빠진 것** | 스토리 콘텐츠 기획 (피드만 6건이고 스토리 전략이 "수시"로만 되어 있음) |
| **약한 것** | 캡션 후킹. 첫 줄에서 타겟의 상황/감정을 건드리는 카피가 부족 |
| **약한 것** | USP 연결. "한 잔 트라이"가 모든 콘텐츠의 바닥에 깔리지 않고 있음 |
| **약한 것** | 각 콘텐츠의 목적(인지/관심/전환)이 명시되지 않아 성과 측정 기준 없음 |
| **바꿔야 할 것** | #1 매장 무드를 단일 이미지 1장에서 캐러셀 3장으로 확대 (톤세팅 효과 증대) |
| **바꿔야 할 것** | #3 강승윤 편을 단일 이미지가 아닌 릴스 15초로 확정 (영상 소재 낭비 방지) |
| **바꿔야 할 것** | #5 타라파카에 캡션 시안 추가 (현재 부재) |

### "와인 모르는 친구한테 보여줘도 재미있을까?" 검증

| 콘텐츠 | 판정 | 이유 |
|--------|------|------|
| #1 매장 무드 | 반반 | 예쁜 사진이면 좋아할 수 있지만, "그래서 뭐?" 될 가능성 |
| #2 상암점 오픈 | X | 탭샵바를 모르면 "6번째 탭"이 무슨 의미인지 모름 |
| #3 강승윤 편 | O | 연예인 + 와인 = 와인 몰라도 호기심 |
| #4 신메뉴 | 반반 | 음식 사진은 범용적이지만, 와인바 메뉴라 진입장벽 |
| #5 타라파카 | X | 와인 이름 자체가 장벽 |
| #6 합정점 릴스 | O | 공간 영상은 누구나 좋아함 + "10만원짜리도 한 잔만"이 궁금증 유발 |

**결론**: 6건 중 2건만 "와인 모르는 친구" 테스트 통과. 4건은 기존 팔로워/와인 관심층에게만 작동. 이 비율을 뒤집어야 한다.
