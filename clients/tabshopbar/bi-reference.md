## Brand Concept

TAP에서 시작된 작은 시도는 SHOP에서 취향이 되고, BAR에서 하나의 문화로 확장된다.

| 단어 | 의미 | 키워드 |
|------|------|--------|
| **TAP** | Easy-Access | 누구나 시작할 수 있으며 |
| **SHOP** | Wine as a Culture | 하나의 문화로 자리잡을 |
| **BAR** | Daily Experience | 와인의 일상적 경험 |

## Logo System

### 로고 유형 & 사용처

| 로고 | 용도 | 우선순위 |
|------|------|---------|
| **Primary** (TAP SHOP BAR 워드마크) | 기본 적용, 가로형 우선 | 1순위 |
| **Symbol 1** (TSB 원형 3개) | 소형/디지털 — 앱 아이콘, SNS 프로필, 스티커 | 2순위 |
| **Symbol 2** (역삼각형) | 사인물 등 제한적 상황만 | 보조 |

### 각 로고 타입

- **Solid Type**: 면 채움 (기본)
- **Line Type**: 외곽선만 (배경 대비에 따라 선택)

### Positive / Negative

- **Positive**: 밝은 배경에 검정 로고
- **Negative**: 어두운 배경에 흰색 로고

### 로고 사용 규칙

- **Clear Space**: 로고 외곽 최소 0.5x 확보
- **최소 크기**:
  - Primary 가로: Print 6mm / Web 24px
  - Primary 세로: Print 10mm / Web 36px
  - Symbol 1: Print 6mm / Web 24px
  - Symbol 2: Print 12mm / Web 40px
- **비율**: 정비례 확대/축소만 허용
- **등록상표 (R)**: Primary와 Symbol 2에만 적용, 매체 내 1회 표기

### 사용 금지 사항 (12가지)

1. 대비가 낮은 배경 색상
2. 로고 색상 혼용
3. 저해상도 활용
4. 아웃라인 처리
5. 최소 공간 규정 위반
6. 구성요소 추가
7. 회전
8. 형태의 반전
9. 일부의 생략
10. 비례의 임의 조정
11. 요소의 임의 형태 조정
12. 기본형의 배치 조정

## Color System

### 메인 컬러

| 컬러명 | HEX | RGB | CMYK | Pantone |
|--------|-----|-----|------|---------|
| **TSB Blue** | `#156dff` | 21, 109, 255 | 88, 55, 0, 0 | 2382 C |
| **TSB Yellow** | `#ffc100` | 255, 193, 0 | 0, 22, 100, 0 | 7548 C |
| **TSB Red** | `#f43131` | 244, 49, 49 | 0, 95, 86, 0 | Red 032 C |
| **TSB Black** | `#000000` | 0, 0, 0 | 20, 20, 0, 100 | - |
| **TSB White** | `#ffffff` | 255, 255, 255 | 0, 0, 0, 0 | - |

### 틴트 컬러 (보조 팔레트)

각 메인 컬러의 80%, 60%, 40%, 20% 단계.
가이드에 정의된 단계만 사용, 임의 농도 생성 금지.
배경, 구획, 정보 강조 등 보조 영역에 활용.

### 컬러 적용 (매체별)

| 컬러 | Paint | Paper |
|------|-------|-------|
| Blue | SH 2180 A (SAMHWA) / VV 8102 (NOROO) | Colorplan CC35 (Adriatic) |
| Yellow | SH 2167 C (SAMHWA) / BT 7091 (NOROO) | Colorplan CC22 (Citrine) |
| Red | SH 2173 C (SAMHWA) / VV 8025 (NOROO) | Colorplan CC11 (Bright Red) |

### 배경 컬러 규정

- 로고 배경에는 가급적 **Black 또는 White** 중 하나로만 적용
- 배경 명도 0~60%: Black 로고
- 배경 명도 30~100%: White 로고
- 중간 명도(30~60%): 대비가 더 높은 컬러 선택

### 배경 이미지 규정

- 밝은 톤 이미지: Black 로고
- 어두운 톤 이미지: White 로고
- 패턴/텍스처가 강한 배경: 로고와 겹침 피하거나 배경 단순화

## Typography

### 영문 서체: P22 Underground (Customized)

| 용도 | Weight | Size | Leading | Tracking |
|------|--------|------|---------|----------|
| Title | Heavy | H 100% | ENG 130% / KOR 140% | 0 |
| Sub Title | Demi Bold | H 75% | 130% | 0 |
| Body Text | *(Pretendard)* Semi Bold / Medium | H ~50% | 130% | -10 |

- 소문자 적용 시, i dot을 원형으로 통일한 커스터마이징 버전 사용
- 동일 지면에서는 한 가지 스타일로 정리

### 한글 서체: Pretendard

| 용도 | Weight | Size | Leading | Tracking |
|------|--------|------|---------|----------|
| Title | Extra Bold / Bold | H 100% | ENG 130% / KOR 140% | 0 |
| Sub Title | Bold | H 75% | 130% | 0 |
| Body Text | Semi Bold / Medium | H ~50% | 130% | -10 |

## 인스타그램 콘텐츠 적용 가이드 (내부 정리)

### 프로필 이미지
- **Symbol 1 Solid Type** 사용 (빨-파-노 TSB 원형)
- 배경: TSB Black (`#000000`) 또는 TSB White (`#ffffff`)

### 피드/릴스 워터마크
- **Primary Logo** (TAP SHOP BAR) 또는 **Symbol 1 Line Type**
- 밝은 이미지: Black 로고 / 어두운 이미지: White 로고
- 최소 크기 24px 이상 유지

### 카드뉴스/캐러셀
- 제목: Pretendard Extra Bold
- 본문: Pretendard Medium
- 포인트 컬러: TSB Blue `#156dff` 또는 TSB Yellow `#ffc100`
- 배경: Black/White 기본, 컬러 배경 시 1색만 사용

### 스토리
- 텍스트 오버레이: Pretendard Bold
- 해시태그/위치: TSB Blue `#156dff`
