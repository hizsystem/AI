# 탭샵바 시그니처 메뉴 스타일링컷 레퍼런스

## 현재 촬영 스타일 분석

### 핵심 3축
1. **컬러 블로킹**: 머스타드 옐로우 단색 배경 (브랜드 컬러)
2. **하드 라이트**: 강한 방향성 조명 → 선명한 그림자 → 음식의 입체감 극대화
3. **다이너 감성**: 그린 림 접시, 테라조 플레이트, 캐스트아이언 → 뉴욕 다이너 + 바 무드

### 스타일링컷 전환 포인트
- 현재: 접시 1개 + 소스컵 정도 (미니멀 프로덕트컷)
- 스타일링컷: 원재료 + 집기(커틀러리) + 텍스처 소품 추가 → "요리가 만들어지는 맥락" 연출
- 레퍼런스: Image #6 스타일 (오버헤드 플랫레이, 복수 접시, 포크, 재료 볼)

---

## 시그니처 메뉴 8종

1. 삼배체 생굴
2. 프라임 채끝 스테이크 (250g)
3. 항정살 구이
4. 연어 스테이크 (200g)
5. 루꼴라 치즈 떡볶이 + 순대 튀김
6. 엔쵸비 쉬림프 파스타
7. 바질 크림 파스타
8. 토마토 블루베리 부라타

## 촬영 시 판단 포인트

### 배경 유지 vs 변경
- 옐로우 배경 유지 → 브랜드 통일감
- 화이트/그레이 톤 전환 → 소품이 더 돋보임
- 추천: 두 가지 다 촬영해서 비교

### 소품 밀도
- 탭샵바 강점인 "미니멀 + 임팩트" 유지
- 원재료 3-4종 + 커틀러리 1개가 적정선
- 너무 많으면 잡지 에디토리얼이 되어 브랜드 톤에서 벗어남

### 그림자 활용
- 하드 라이트 특유의 선명한 그림자가 탭샵바 시그니처
- 소품도 그림자가 예쁘게 떨어지는 것 선택 (커틀러리, 레몬 웨지 등)
