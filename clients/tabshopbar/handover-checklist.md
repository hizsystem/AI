## 6. 인수인계 전 최종 체크리스트

- [ ] 산출물 전체 리스트 → 인수인계 시트에 정리
- [ ] 미완료 항목 → 마무리 or "미완료" 명시
- [ ] 견적서 최종 → 누락/감액 반영
- [ ] 디자인 파일 → 홍민아 대표 리스트와 크로스체크
- [ ] VMD 피그마 → 마무리/인수인계 판단
- [ ] 계정 접근권한 → 정리 (인스타, 카카오, Sheets, 피그마)
- [ ] 나기정 대표 메일링 → 견적서 + 리스트 + 아웃풋
