# 탭샵바 주간미팅 준비 템플릿

> 매주 월요일 | 팀: 우성민(마케팅 리드), 김희종(운영/외부협업), 박창규(사업 운영)
> Google Sheets: https://docs.google.com/spreadsheets/d/1RgZKTI-3py6Myc58NUo4Dhf0Ev-NMaHOA7Mfxqlf7J4

---

## 사용법

매주 월요일 미팅 전, 아래 요청으로 준비 문서를 자동 생성:

```
탭샵바 주간미팅 준비해줘
```

자동으로 수행하는 작업:
1. Google Sheets에서 지난주 태스크 진행 현황 fetch
2. 미완료 태스크 이월 여부 판단
3. 콘텐츠 캘린더에서 이번주 발행 예정 콘텐츠 확인
4. 확정 행사 일정 중 이번주~다음 2주 해당 건 추출
5. 아젠다 생성

---

## 준비 문서 구조

### 1. 지난주 리뷰

| 구분 | 내용 | 담당 | 결과 | 비고 |
|------|------|------|------|------|
| {구분} | {태스크} | {이름} | 완료/미완/이월 | {사유} |

**완료율**: X/Y건 (Z%)

### 2. 이번주 태스크

| # | 구분 | 내용 | 담당 | 마감 | 선행조건 |
|---|------|------|------|------|---------|
| 1 | {구분} | {태스크} | {이름} | {날짜} | {있으면} |

**이월 태스크**: 지난주 미완료 중 이번주로 넘어온 건

### 3. 콘텐츠 캘린더 체크

- 이번주 발행 예정: {콘텐츠 캘린더에서 해당 주 아이템}
- 상태: 기획/컨펌필요/업로드완료
- 필요 액션: {촬영/디자인/카피 등}

### 4. 확정 일정 (이번주 + 2주)

| 날짜 | 구분 | 내용 | 비고 |
|------|------|------|------|
| {날짜} | {수입사/외부협업/플랫폼} | {행사명} | {상태} |

### 5. 아젠다 (20분)

```
1. [3분] 지난주 리뷰 — 완료/미완 빠르게 체크
2. [5분] 이번주 태스크 배분 — 담당자별 확인
3. [5분] 콘텐츠 & 행사 — 발행 예정 + 다가오는 행사
4. [5분] 논의 사항 — 이슈/의사결정 필요한 건
5. [2분] 다음주 프리뷰
```

### 6. 논의 필요 사항

- {미팅에서 결정해야 할 이슈 목록}

## 미팅 후 처리

미팅 끝나면:
```
탭샵바 주간미팅 랩업해줘
```

자동으로:
1. 회의록 정리 (클로바노트 전사본 기반)
2. 액션아이템 추출 → Google Sheets 업데이트 가이드
3. 다음주 태스크 초안 생성
