# 탭샵바 SNS 데이터 시트 설계

> 작성일: 2026-03-16
> 목적: 인스타그램 + 카카오플러스친구 성과 데이터 축적 및 대시보드 연동
> 참조: 휴닉(HUENIC) KPI 시스템 구조 동일 적용

---

## 아키텍처

```
[Google Sheets]          [content-calendar 앱]          [대시보드]
                          (content-calendar-psi.vercel.app)
 ┌──────────────┐        ┌─────────────────┐        ┌──────────────┐
 │ 탭샵바 데이터  │──CSV──→│ API route       │──JSON──→│ /tabshopbar  │
 │ (별도 시트)    │        │ /api/tabshopbar │        │  KPI Tab     │
 └──────────────┘        │ /kpi/[month]    │        │  Weekly Tab  │
                         └─────────────────┘        └──────────────┘
```

**마스터 시트(메뉴 관리)와 별도 시트로 만드는 이유**:
- 메뉴 시트는 운영 데이터 (정적), KPI 시트는 성과 데이터 (주기적 갱신)
- 접근 권한이 다를 수 있음 (KPI는 우리만, 메뉴는 탭샵바팀도)
- content-calendar 앱의 Sheets API 연동은 하나의 SHEET_ID 기준

## Google Sheets 탭 구조 (5개 탭)

### 탭 1: 주간성과

주 1회 업데이트. 인스타그램 인사이트에서 데이터 추출.

| 컬럼 | 예시 | 설명 |
|------|------|------|
| 주차 | 2026-W14 | ISO 주차 |
| 기간 | 3/31~4/6 | 해당 주 기간 |
| 팔로워 | 1,250 | 주말 기준 팔로워 수 |
| 팔로워증감 | +35 | 전주 대비 |
| 게시물수 | 2 | 해당 주 발행 수 |
| 도달 | 5,200 | 주간 총 도달 |
| 도달증감 | +800 | 전주 대비 |
| 노출 | 8,100 | 주간 총 노출 |
| 참여율 | 3.2 | % (좋아요+댓글+저장+공유)/팔로워 |
| 참여율증감 | +0.3 | 전주 대비 (pp) |
| 프로필방문 | 180 | 주간 프로필 방문 수 |
| 링크클릭 | 25 | 바이오 링크 클릭 수 |
| 비고 | 강승윤 릴스 발행 | 특이사항 메모 |

### 탭 2: 게시물별 성과

게시물 발행 후 7일째에 데이터 기록 (성과 안정화 시점).

| 컬럼 | 예시 | 설명 |
|------|------|------|
| 발행일 | 2026-03-11 | 게시물 발행일 |
| 제목 | 추카추카추 강승윤 편 | 콘텐츠 제목 |
| 카테고리 | 탭샵바씬 | 5개 카테고리 중 |
| 포맷 | 릴스 | 릴스/캐러셀/카드뉴스/이미지/스토리 |
| 주력타겟 | 전체 | 지은/민준/커플/전체 |
| 도달 | 3,500 | 게시물 도달 |
| 노출 | 5,200 | 게시물 노출 |
| 좋아요 | 85 | |
| 댓글 | 12 | |
| 저장 | 23 | |
| 공유 | 8 | |
| 참여율 | 4.1 | % |
| 팔로워전환 | 15 | 이 게시물로 팔로우한 수 (인사이트 제공 시) |
| 비고 | 알고리즘 도달 좋았음 | |

### 탭 4: 팔로워추이

| 컬럼 | 예시 | 설명 |

## 데이터 수집 루틴

### 주간 (매주 월요일)
1. 인스타그램 인사이트 → **주간성과** 탭 1행 추가
2. 지난주 발행된 게시물 → **게시물별 성과** 탭 (발행 7일 후 기록)
3. 카카오 채널 관리자 → **카카오플러스친구** 탭 (메시지 발송 있었으면)

### 월간 (매월 1일)
1. 전월 데이터 정리 → **월간KPI** 탭 1행 추가
2. 팔로워 수 기록 → **팔로워추이** 탭 1행 추가
3. 카카오 월간 정리 → **카카오플러스친구** 탭 (월간 유형)

## content-calendar 앱 연동 (개발 범위)

### 필요한 작업

| 항목 | 파일 | 설명 |
|------|------|------|
| 타입 정의 | `src/data/tabshopbar-types.ts` | KpiData, WeeklyReport 등 타입 |
| Sheets 연동 | `src/lib/tabshopbar-sheets.ts` | Google Sheets CSV → 데이터 변환 |
| API route | `src/app/api/tabshopbar/kpi/[month]/route.ts` | KPI 데이터 제공 |
| API route | `src/app/api/tabshopbar/weekly/[week]/route.ts` | 주간 데이터 제공 |
| 대시보드 | `src/app/tabshopbar/page.tsx` 확장 | KPI 탭 추가 |
| 컴포넌트 | `src/components/tabshopbar/` | KPI 카드, 차트, 게시물 성과 |

### 환경변수

```
NEXT_PUBLIC_TSB_SHEET_ID=새로만든시트ID
```

### 휴닉과 차이점

| 항목 | 휴닉 | 탭샵바 |
|------|------|--------|
| 브랜드 | 2개 (베지어트/빙커) | 1개 |
| 매장 | 없음 | 6개 (지점별 분석 가능성) |
| 카카오 | 없음 | **있음** — 별도 탭 필요 |
| 게시물별 성과 | 베스트콘텐츠만 | **전체 게시물** 기록 |
| 타겟 분석 | 없음 | **주력타겟 컬럼** 추가 → 타겟별 성과 비교 가능 |
