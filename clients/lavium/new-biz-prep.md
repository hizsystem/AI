## 니즈 분석 상세

### Step 1: 인테이크 데이터 분석
- 원문: "지속적이고 효율적인 마케팅 방법론을 아직 찾지 못함"
- 키워드: "지속적" (일회성이 아닌 반복 가능한), "효율적" (300만원 예산 인식), "방법론" (체계/시스템), "아직" (시도는 해봤으나 실패)
- 감정 톤: 솔직하고 차분한 인정. 제품에 대한 자신감은 있으나, 마케팅 영역에서의 불확실성
- 마케팅 목표 4개 동시 선택 (Acquisition + Awareness + Conversion + 제품홍보) -- "뭐든 다 해야 할 것 같다"는 막연함

### Step 2: SPIN 기반 니즈 레이어링
| 레이어 | 내용 |
|--------|------|
| **Situation** | 런칭 4개월, 월 1,030만 매출, ROAS 382%. 제품 6종, 유통 8개+ 채널. CEO+CMO 2인 운영, 마케팅 전담 인력 없음. 청창사 마케팅 예산 28백만원 배정. |
| **Problem** | SNS 채널 체계 미구축. 콘텐츠 제작/운영 시스템 부재. 제품 스펙(특허, 성분) 중심 커뮤니케이션 -- 소비자 언어 부재. 채널 다수 분산, 집중 부족. |
| **Implication** | ROAS 382%인데 스케일 불가(시스템 없음). 현대백화점/이마트 바이어 앞에서 "인스타 팔로워 200명"이면 설득력 하락. Seed 라운드에 재현 가능한 성장 엔진 증명 불가. 재구매율 8.8% 방치 시 LTV 정체. |
| **Need-Payoff** | 재현 가능한 콘텐츠 마케팅 시스템 → 월 300만원 내 지속 성장 가능. B2C 브랜드파워가 B2B 영업 레버리지. 투자유치 밸류에이션 상승. |

### Step 3: 고민 계층 분석 (Pain Hierarchy)

    |

### Step 5: 유사 사례 패턴
1. **"제품력 > 브랜드력" 패턴**: 휴닉/베지어트와 동일 구조. R&D 대표가 기술 언어로 콘텐츠를 만들지만 소비자 반응 낮음
2. **"B2C 브랜딩이 B2B 영업의 전제조건" 패턴**: 뉴케어/하이뮨이 요양기관에 입점할 수 있는 이유는 소비자 인지도
3. **"예산 제한 + 다채널 = 분산 실패" 패턴**: 이미 6개+ 채널 운영 중, 월 300만원으로 동시 운영 물리적 불가
4. **"타겟 3개 = 타겟 없음" 패턴**: B2C 3단계 + B2B 3단계 + 글로벌은 IR용이지 실행용이 아님
5. **휴닉과의 직접 유사성**: 푸드테크, 초기 투자단계, 제품력 강/마케팅 체계 미비, SNS 현황 부실 -- 거의 동일 구조

### Step 6: 검증 질문
1. "현재 인스타/SNS 현황이 어떻게 되세요? 팔로워, 게시물, 누가 운영하시는지?" -- 검증: SNS 방치/산발적 운영 가설
2. "콘텐츠 마케팅 희망이시지만, 기대 성과는 '전략 로드맵'이셨어요. 더 시급한 건 당장 콘텐츠인가요, 큰 그림인가요?" -- 검증: 표면 요청 vs 진짜 니즈 갭 인식 여부
3. "현대백화점/아산병원 바이어에게 마음미음 브랜드를 어떻게 소개하고 계세요?" -- 검증: B2B 현장에서 브랜드 인지도 부족 체감 여부
4. "월 300만원으로 가장 먼저 해결하고 싶은 한 가지만 고른다면?" -- 검증: 예산 대비 기대치 현실성, 진짜 페인
