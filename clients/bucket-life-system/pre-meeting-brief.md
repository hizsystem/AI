## 2단계: 디지털 프레즌스 진단

### 웹사이트
- ⚠️ 공식 웹사이트 URL 확인 필요. 웹 검색에서 버킷라이프시스템 공식 사이트가 검색되지 않음.
- **진단**: 웹사이트가 없거나, SEO가 전혀 안 되어 있거나, 아직 구축 전 상태로 추정.

### SNS (소셜 미디어)
- 인테이크에 "상시 채널은 OFF 상태"로 명시.
- 인스타그램, 페이스북, 유튜브 등 **활성 채널 없음** (런칭 후 셋업 예정).
- **진단**: 디지털 프레즌스 사실상 제로 상태. 이는 가장 큰 리스크이자, 동시에 처음부터 제대로 설계할 수 있는 기회.

### 미디어 노출
- 버킷라이프시스템 관련 언론 보도, 인터뷰 ⚠️ 검색 결과 없음.
- **진단**: PR/미디어 노출 전무. 브랜드 인지도 구축이 시급.

### 광고 현황
- Meta 광고 라이브러리: ⚠️ 현재 라이브 광고 없음 추정 (채널 OFF 상태)
- 네이버 검색광고: ⚠️ 확인 필요 (운영 중이지 않을 가능성 높음)
- **향후 계획**: 컨설팅 이후 Meta, GFA 등 신규 셋업 예정

### 디지털 프레즌스 종합 점수: **1/10 (거의 부재)**

## 3단계: 대표자/창업팀 프로필

### 김진규 대표
- **공개 이력**: ⚠️ 웹 검색에서 특정 이력 정보 미확인
- **추정 배경**: 커머스(이커머스) 업계 경험 보유 → 피봇하여 시니어 체험 플랫폼으로 전환
- **행동 특성**: 이탈리아 북부 4대 호수, 프랑스, 인도 등을 **직접 종단**하며 루트 개척 → 실행력과 현장주의가 강한 창업가형
- **철학** (메일 회신에서 추출):
  - "단순 상품 판매(Selling)보다 더 중요한 것은 상품의 본질(Product)" → **Product-First 사고방식**
  - 대행사를 거치지 않고 직접 현지 개척 → **오리지널리티에 대한 강한 집착**
  - 데이터 바우처를 통한 정교한 타겟 데이터 확보 → **데이터 기반 의사결정 지향**

### 의사결정 스타일 추정
- **직접 실행형**: 외주/대행보다 직접 해보는 성향 (현지 답사, 루트 개척 등)
- **품질 중시**: "프리미엄", "독보적", "고부가가치" 등의 키워드를 자주 사용
- **전략적**: 메일 회신이 매우 상세하고 구조적 → 논리적 사고, 체계적 커뮤니케이션 선호
- **마케팅 이해도**: "풀퍼널 마케팅 믹스", "DB 확보", "타겟팅 광고 시나리오" 등 마케팅 용어 활용 → 기본 개념은 이해하나 실행 경험은 제한적

### 상담 시 유의점
- 마케팅 전문용어를 이해하는 수준이므로 쉬운 설명보다는 **구체적 실행안과 수치**로 어필
- "광고효과에 대한 포커싱"이 페인 포인트 → **ROI/ROAS 중심의 성과 기반 제안** 선호
- Product에 대한 자부심이 강하므로, **상품의 우수성을 인정하면서 마케팅 관점의 전환**을 제안

## 4단계: 시장/경쟁 환경

### 시장 트렌드 (2025-2026)

1. **뉴시니어 여행 폭발**: 60대 예약 비중 24.6%, 70대 이상 77% 신장률 (노랑풍선 기준)
2. **프리미엄화**: 시간/금전 여유있는 시니어 → 고급화, 테마화 상품 수요 급증
3. **체험/성장 여행 트렌드**: 단순 관광에서 '현지 깊숙한 삶 체험', '자기 성장' 여행으로 이동
4. **디지털 전환**: 시니어도 모바일 예약, 유튜브/인스타 정보 탐색 활발 (디지털 소외 세대 아님)
5. **초고령사회 진입**: 2025년 65세 이상 인구 20.6% → 시니어 관광 산업 본격 성장기

### 벤치마크 브랜드 (글로벌)
- **Road Scholar** (미국): 55세 이상 교육여행 프로그램, 비영리, 연 25만명 참여. "Learning Adventures"라는 포지셔닝.
- **Overseas Adventure Travel (OAT)**: 소그룹 프리미엄 해외 모험 여행, 50세 이상 타겟.

## 5단계: 초기 가설 수립

### 브랜드 성숙도: **1단계 - 미정립**

| 판단 근거 |
|-----------|
| 디지털 프레즌스 거의 부재 (웹사이트, SNS, 광고 모두 OFF) |
| 브랜드 메시지, 비주얼 아이덴티티, 톤앤매너가 정립되지 않은 상태 |
| "프리미엄 해외 체험/성장 프로그램"이라는 포지셔닝은 있으나 외부 커뮤니케이션으로 전환되지 않음 |
| 커머스 → 체험 플랫폼으로의 피봇 과정에서 브랜드 재정립 필요 |

### 마케팅 성숙도: **1단계 - 미착수**

| 판단 근거 |
|-----------|
| 상시 마케팅 채널 전무 (OFF 상태) |
| 광고 집행 이력 없음 (과거 광고 수치 없음) |
| 마케팅 퍼널 미구축 (DB 확보 → 예약 전환 퍼널 없음) |
| 단, 데이터 바우처 통한 타겟 데이터셋 확보 → 데이터 자산은 보유 |

### 핵심 과제 가설 (우선순위순)

#### 가설 1: "뛰어난 상품을 디지털로 전환하는 브릿지가 없다"
- **근거**: 대표가 직접 개척한 고퀄리티 프로그램 보유 + 중장년 콘텐츠 데이터 보유, 그러나 이를 디지털 메시지로 변환하는 역량/채널이 없음
- **상담 방향**: "상품력은 검증됨. 지금 필요한 것은 이 상품을 5064 타겟이 발견하고, 신뢰하고, 결제까지 가는 디지털 퍼널 설계"

#### 가설 2: "프리미엄 브랜드 포지셔닝이 수립되지 않으면, 대형 여행사와 가격 경쟁에 빠진다"
- **근거**: '독보적 현지 오리지널리티'라는 강점이 있지만, 브랜드로 체계화되지 않으면 소비자는 대형 여행사의 유사 상품과 비교할 수밖에 없음
- **상담 방향**: "가격이 아닌 가치로 선택받는 브랜드 스토리텔링 + 프리미엄 포지셔닝 구축"

#### 가설 3: "6개월 4억 매출 목표는 광고만으로 달성 불가, 풀퍼널 + 오프라인 연계 필요"
- **근거**: 월 마케팅 예산 300만원 이하 × 고단가 상품(해외 프리미엄 프로그램) × 제로 인지도 → 광고만으로 11월~2월 4억 달성은 현실적으로 어려움
- **상담 방향**: "광고 + 콘텐츠(유튜브/블로그) + 커뮤니티 + 제휴(50+ 센터, 시니어 커뮤니티 등) 멀티 채널 전략 제안"

#### 가설 4: "5064 타겟의 디지털 접점은 네이버+유튜브가 핵심, Meta는 보조"
- **근거**: 5064세대의 주 정보 탐색 채널은 네이버 검색 + 유튜브. Meta(페이스북/인스타)는 보조 채널. 대표가 언급한 "Meta, GFA" 중 GFA(네이버)가 실질적으로 더 효과적일 가능성
- **상담 방향**: 채널 우선순위 재설정 제안 (네이버 검색 SEO + 블로그 → 유튜브 스토리텔링 → GFA → Meta 순)

## 상담 전략 요약

### 이 클라이언트의 핵심 매력
1. **상품력이 압도적**: 대표가 직접 현지 개척한 독자 프로그램 (대행사 미경유)
2. **데이터 자산 보유**: 데이터 바우처를 통한 정교한 중장년 타겟 데이터셋
3. **시장 타이밍 우수**: 초고령사회 진입 + 시니어 프리미엄 여행 트렌드 급성장
4. **대표의 마케팅 이해도**: 풀퍼널, DB 확보, 타겟팅 등 개념 이해 → 커뮤니케이션 원활

### 이 클라이언트의 핵심 리스크
1. **예산 제약**: 월 300만원 이하 → 고단가 프리미엄 상품 모객에 충분하지 않을 수 있음
2. **제로 베이스**: 웹사이트, SNS, 광고 모두 없는 상태에서 시작 → 구축 기간 필요
3. **6개월 4억 목표의 현실성**: 11~2월 목표인데, 채널 셋업 + 인지도 구축 시간 고려 시 빠듯
4. **1인(소규모) 운영**: 실행 리소스 제한 → 외주/자동화 병행 필요

### 상담 시 핵심 질문 (확인 필요 사항)
1. 프로그램 1건당 객단가는? (인당 가격 × 인원 = 4억 역산 필요)
2. 기존 커머스 시절의 고객 DB는 있는가? (리마케팅 가능 여부)
3. 현재 팀 구성은? (마케팅 실행 인력 유무)
4. 데이터 바우처로 확보한 데이터셋의 구체적 내용은?
5. 웹사이트/랜딩페이지 구축 계획과 예산은?
6. 프로그램 운영 시작 예정일은? (마케팅 리드타임 역산)
