# SNS 채널 전략 컨설팅 제안서

**Digital Bridge → HD현대오일뱅크 윤활유Mobility팀**
2026. 03

---

## 배경

HD현대오일뱅크 XTeer 브랜드의 SNS 채널 전략 수립 및 디자인 가이드라인 개발 프로젝트에 대한 서비스 범위와 견적을 안내드립니다.

디지털브릿지는 B2B/B2C 브랜드의 디지털 채널 전략 수립 전문 에이전시로, 뷰티, 테크, 라이프스타일 등 다양한 업종의 SNS 채널 컨설팅 경험을 보유하고 있습니다.

---

## 서비스 범위

### Phase 1 — 현황 진단 및 분석 (Week 1-2)

| # | 액션 | 산출물 |
|---|------|--------|
| 1 | SNS 채널 현황 감사 (Audit) | 인스타그램 채널 진단 리포트 (정량/정성) |
| 2 | 경쟁사 벤치마크 분석 | 동종 업계 3사 비교 분석 |
| 3 | 타겟 오디언스 세그멘테이션 | 페르소나 3종 + 오디언스 맵 |

### Phase 2 — 전략 프레임워크 수립 (Week 3-4)

| # | 액션 | 산출물 |
|---|------|--------|
| 4 | 채널 포지셔닝 전략 | 채널별 역할 정의 + 콘텐츠 필러 |
| 5 | 비주얼 무드보드 | 브랜드 비주얼 톤 레퍼런스 (2안) |
| 6 | 콘텐츠 카테고리 설계 | 콘텐츠 유형별 가이드라인 |

### Phase 3 — 디자인 가이드라인 개발 (Week 5-6)

| # | 액션 | 산출물 |
|---|------|--------|
| 7 | SNS 디자인 가이드라인 | 그리드, 컬러, 타이포, 템플릿 2종 |
| 8 | 최종 전달 미팅 | 산출물 핸드오프 + 활용 가이드 (화상 1.5h) |

---

### 별도 견적 항목 (본 프로젝트 미포함)

아래 항목은 본 프로젝트 범위에 포함되지 않으며, 필요 시 별도 제안을 드립니다.

| 항목 | 예상 금액 | 비고 |
|------|-----------|------|
| X(트위터) 채널 런칭 전략 | 200만원 | 채널 세팅 + 운영 가이드 |
| 팔로워 확보 이벤트 기획 | 200만원 | 이벤트 설계 + 운영 시나리오 |
| 콘텐츠 제작 (월간) | 별도 협의 | 운영 대행 전환 시 |

---

## 진행 방식

```
[Week 1-2]           [Week 3-4]              [Week 5-6]
현황 진단 + 분석 →  전략 프레임워크 수립 →  디자인 가이드라인 개발
```

6주 단기 프로젝트로, 산출물 납품 후 종료됩니다.
컨설팅 완료 후 운영 대행 전환을 원하실 경우, 별도 제안서를 준비해 드립니다.

## 견적

| 항목 | 내용 |
|------|------|
| 서비스 | SNS 채널 전략 컨설팅 + 디자인 가이드라인 |
| 금액 | **800만원 (VAT 별도)** |
| 기간 | 약 6주 |
| 결제 | 착수 시 50% / 완료 시 50% |

X 채널 런칭 전략 및 팔로워 이벤트 기획은 본 견적에 포함되지 않으며, 추가 진행 시 별도 견적(각 200만원)이 발생합니다.
