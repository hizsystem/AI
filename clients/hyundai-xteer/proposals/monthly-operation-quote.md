## 배경

SNS 채널 전략 및 디자인 가이드라인 수립(컨설팅) 이후, 가이드라인에 기반한 채널 운영을 BRANDRISE가 월 단위로 제공하는 서비스입니다.

담당자분의 업무 부하를 줄이면서도, 가이드라인이 현장에서 흐트러지지 않도록 콘텐츠 기획부터 제작, 성과 리뷰까지 일관된 파트너십으로 운영합니다. 내부 디자이너와의 협업 구조를 유지하되, 콘텐츠 방향성과 제작은 BRANDRISE가 주도합니다.

| 구분 | 항목 | 월간 수량 |
|------|------|-----------|
| 콘텐츠 제작 | 글로벌 고객 대상 수출 제품 소개 (이미지) | 6건 |
| | 릴스 (제품 이미지 활용 + 레이싱 콘텐츠) | 2건 |

| 기획/관리 | 월간 콘텐츠 캘린더 | 1회 |
| | 콘텐츠 사전 검수 + 피드백 | 매 건 |
| 리포팅 | 월간 성과 리뷰 리포트 | 1회 |
| 미팅 | 정기 화상 미팅 (1h) | 1회 |

**포함 범위**
- 콘텐츠 기획 + 카피라이팅(국문/영문) + 이미지/영상 제작
- 글로벌 고객 대상 수출 제품 소개 콘텐츠 (제품 이미지 소스 활용)
- 레이싱 콘텐츠 (담당자 촬영 소스 제공 → 편집/가공)
- X 계정은 인스타그램 콘텐츠를 플랫폼에 맞게 변환하여 동시 게시
- 댓글/DM 대응 가이드 제공 (직접 대응은 내부 담당)

## 운영 프로세스

```
[매월 초]                [매주]                    [매월 말]
콘텐츠 캘린더 공유  →  제작 + 검수 + 게시  →  성과 리뷰 리포트
       ↓                    ↓                       ↓
 방향성 합의           피드백 반영            다음 달 방향 조정
```

1. 매월 초 — 콘텐츠 캘린더 초안 공유, 담당자 확인 후 확정
2. 매주 — 콘텐츠 제작, 사전 검수(카피 + 비주얼), 승인 후 게시
3. 매월 말 — 성과 리뷰 리포트 + 정기 미팅에서 다음 달 방향 논의
