## Executive Summary

HYUNDAI XTeer는 제품력이 검증된 브랜드입니다. 구매 경험자의 재구매 의향이 높고, 전세계 70개국에 수출하고 있습니다. 문제는 인지도입니다. TOM 0~1%, 보조인지도 24%. 알기만 하면 사는 브랜드인데, 아무도 모릅니다.

본 제안은 XTeer 내부 팀이 인스타그램을 직접 운영할 수 있도록, **전략 기획 + 실행 가이드라인 + 코칭**을 제공하는 구조입니다. 외부에서 대신 해주는 것이 아니라, 내부 역량을 세팅하는 것이 목적입니다.

AI 기반 워크플로우를 통해 3주 내 전략 기획서와 가이드라인을 납품하고, 이후 2주간 실행 코칭을 진행합니다.

1. **인지도 부재** — TOM 0~1%. 소비자 인식에 존재하지 않음
2. **차별점 미전달** — 소비자 10명 중 7명이 XTeer만의 차이를 모름
3. **글로벌 거버넌스 부재** — 해외 딜러가 각자 톤으로 운영, 본사 컨트롤타워 없음

## Part 2. 전략 방향

### 포지셔닝

```
HYUNDAI XTeer = "내 차를 가장 잘 아는 오일"
현대가 설계한 엔진에 최적화된 프리미엄 윤활유
```

경쟁사가 선점하지 못한 "제조사 기반 신뢰" 포지션. ZIC은 "국내 1위", Kixx는 "가성비 프리미엄", S-OIL은 "세련된 이미지". XTeer만의 고유 자산인 "현대 제조사 연결성"을 활용합니다.

### 인스타그램 역할 정의 — "글로벌 브랜드 허브"

| 역할 | 내용 |
|------|------|
| 글로벌 앵커 | 70개국 XTeer 경험의 중심점 |
| 브랜드 컨트롤타워 | 톤, 비주얼, 메시지의 기준 제시 |
| 콘텐츠 큐레이터 | 해외 딜러 우수 콘텐츠 선별, 증폭 |
| R&D 비하인드 | 본사만 할 수 있는 기술/제조 스토리 |

### 타겟

| 구분 | 정의 | 근거 |
|------|------|------|
| Primary | 2535 남녀, 차량 관리에 관심 있는 능동적 소비자 | 오픈서베이 기준 XTeer 상대적 인지도 형성 세그먼트 |
| Secondary | 글로벌 XTeer 딜러/사용자 | 본사 계정 팔로우 동기가 가장 강한 기존 팬베이스 |

### 핵심 메시지

| 타겟 | 메시지 방향 | 근거 |
|------|------------|------|
| 여성 / 30-50대 | "깨끗하고 지속가능한 주행" (친환경) | 오픈서베이 최선호 메시지 52.5% |
| 남성 / 20-30대 | "70개국이 선택한 프리미엄" (글로벌 신뢰) | 오픈서베이 2순위 30.6% |
| 해외 딜러/사용자 | "Premium Lubricant, Made by Hyundai" | 글로벌 인지 자산 활용 |

### 콘텐츠 카테고리

| 카테고리 | 비중 | 포맷 | 예시 |
|----------|------|------|------|
| Brand World | 30% | 피드(이미지/영상) | 제품 비주얼, 브랜드 필름, 시즌 캠페인 |

### 코칭 (실행 지원)

| 회차 | 시점 | 내용 |
|------|------|------|
| 1회 | 납품 시 | 전략 기획서 + 가이드라인 워크스루. 내부 팀이 이해하고 실행할 수 있도록 설명 |
| 2회 | 납품 후 1주 | 내부 팀이 만든 첫 콘텐츠 피드백. 가이드라인 적용 점검 |
| 3회 | 납품 후 2주 | 1주차 운영 리뷰 + 개선점 + 향후 자체 운영 가이드 |

코칭은 화상 1시간 + 서면 피드백으로 진행합니다.

## Part 5. 투자

### 패키지

**인스타그램 전략 기획 + 코칭 패키지**

| 항목 | 내용 |
|------|------|
| 범위 | 전략 기획서 + 디자인 가이드라인 + 콘텐츠 캘린더 + 벤치마크 + 코칭 3회 |
| 기간 | 5주 (기획 3주 + 코칭 2주) |
| **투자** | **500만원 (VAT 별도)** |
| 결제 | 착수 시 100% 또는 착수 50% / 납품 50% 협의 가능 |

### 이 금액으로 확보하는 것

- 내부 팀이 6개월~1년간 활용할 수 있는 인스타그램 운영 설계도
- AI + 내부 직원 체제에 맞춘 실행 가능한 가이드라인
- 오픈서베이 조사 투자의 실행 전략 전환 (조사 ROI 실현)
- 해외 딜러 대비 본사 포지션 명확화

### 투자 관점

- 오픈서베이 조사 투자 (추정 1,000만원+) → 진단은 완료. 실행 전략이 없으면 조사비가 매몰 비용
- SNS 대행사 월 운영비 (월 300~500만원) → 전략 없이 1년 운영하면 4,000만원+ 소진
- 본 패키지는 "전략 설계도"를 500만원에 확보하고, 실행은 내부 AI+직원 체제로 하는 구조

### 옵션: 월간 코칭 리테이너

내부 운영 안정화 후, 월간 코칭이 필요하다면:

| 항목 | 내용 |
|------|------|
| 범위 | 월 2회 코칭(화상 1시간) + 콘텐츠 서면 피드백(월 8건) + 월간 성과 리뷰 |
| **월 투자** | **100만원 (VAT 별도)** |
| 계약 단위 | 3개월 단위 (언제든 종료 가능) |

이 옵션은 필수가 아닙니다. 코칭 3회로 자체 운영이 가능하다면 추가 계약 없이 종료합니다.
