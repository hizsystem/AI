## 크리에이티브 브리프

### 기존 캐러셀("요즘 브랜드, 이렇게 팝업한다") 광고 전환 시 최적화 포인트

| 항목 | 현재 (오가닉) | 광고 최적화 |
|------|-------------|-----------|
| 첫 장(커버) | 브랜딩 톤 타이틀 | 3초 후킹 카피 추가 |
| 장수 | 10장+ | 5장 이내로 압축 |
| 포트폴리오 | 15건+ 나열 | 인지도 높은 3개만 선별 |
| 마지막 장 | 로고/마무리 | CTA 전용 장으로 교체 |

### 소재 시안 3개

#### A안: 문제 제기형 (1순위, 당일 제작 가능)

| 항목 | 내용 |
|------|------|
| 포맷 | 피드 이미지 (1:1) |
| 메인 카피 | "팝업스토어, 기획부터 운영까지 다 직접 하시게요?" |
| 서브 카피 | 장소 섭외, 공간 디자인, 운영 인력, 홍보까지 — 브랜드라이즈가 원스톱으로 해결합니다. 비비고x세븐틴, 네스카페 등 15건+ 실전 경험. |
| CTA | "30분 무료 상담으로 견적부터 받아보세요" |
| 예상 CTR | 1.2~1.8% |

#### B안: 성과 증명형 (1순위, 동시 집행)

| 항목 | 내용 |
|------|------|
| 포맷 | 캐러셀 (1:1, 4장) |
| 메인 카피 | "3일 팝업, 방문객 5,000명. 브랜드라이즈가 만들면 다릅니다." |
| 장 구성 | 1: 메인 카피+숫자 / 2: 비비고x세븐틴 현장+수치 / 3: 네스카페 현장+수치 / 4: CTA |
| CTA | "포트폴리오 보고 상담 신청하기" |
| 예상 CTR | 1.5~2.2% |

#### C안: 도발형 (2순위, 릴스)

| 항목 | 내용 |
|------|------|
| 포맷 | 릴스 (9:16, 15~20초) |
| 메인 카피 | "솔직히, 팝업스토어 대행사 다 비슷해 보이죠?" |
| 영상 구성 | 0~3초: 검은배경+텍스트 / 3~10초: 포트폴리오 하이라이트 / 10~15초: CTA |
| CTA | "포트폴리오 확인하고 비교해보기" |
| 예상 CTR | 0.8~1.5% (릴스는 CPM이 40~60% 저렴) |

### 광고 텍스트 세트

**본문 (A안용)**
```
팝업스토어, 직접 기획하려니 막막하셨죠?

장소 섭외부터 공간 디자인, 운영, 홍보까지
— 한 번에 맡길 수 있는 곳이 있습니다.

브랜드라이즈는 비비고x세븐틴, 네스카페 돌체구스토 등
15건 이상의 팝업을 성공적으로 운영했습니다.

지금 무료 상담 신청하시면,
브랜드에 맞는 팝업 컨셉과 예산 가이드를 드립니다.
```

**본문 (B안용)**
```
"다음 팝업, 어디에 맡기지?"
고민 중인 마케터분들을 위해 준비했습니다.

비비고x세븐틴 / 네스카페 돌체구스토 / 리솔츠
— 결과로 검증된 브랜드라이즈 포트폴리오.

기획부터 운영까지 원스톱.
30분 무료 상담으로 시작하세요.
```

**헤드라인**
- A안: `팝업스토어, 기획부터 운영까지 원스톱`
- B안: `15건+ 팝업 포트폴리오, 직접 확인하세요`
- C안: `팝업 대행사, 아무 데나 맡기지 마세요`

**CTA 버튼**: "더 알아보기" (코어) / "문의하기" (리타게팅)

## A/B 테스트 계획

| 라운드 | 테스트 변수 | 기간 | 판단 기준 |
|--------|-----------|------|----------|
| Round 1 | A안 vs B안 (소재 유형) | 5일 | CTR + 상담 신청 수 |
| Round 2 | 승자 소재의 카피 변형 | 5일 | CTR 차이 30% |
| Round 3 | 프로필 링크 vs 인스턴트 폼 | 5일 | CPL 비교 |
| Round 4 | 릴스(C안) vs 피드 승자 | 선택 | 도달 단가 비교 |

### Phase 1: 테스트 (3/14~3/31)

| 일정 | 액션 |
|------|------|
| D1~D3 | 픽셀 세팅 + 캠페인 세팅 + 광고세트 A/B ON |

- 성과 좋은 광고세트에 예산 60% 집중

- 유사 오디언스 1% Lookalike 테스트
- CBO 전환 검토

### Phase 3: 스케일링 (5월~, 예산 증액 시)

- 예산 100만원+ → 전환 캠페인 전환 (Lead 이벤트 최적화)
- 유사 오디언스 1%/3%/5% 계층화
- 영상 소재 추가 (팝업 현장 15초/30초)
