# 모수 Meta 광고 중간 보고 (4/15, 6일차)

## 핵심 요약

| 항목 | 수치 |
|------|------|
| 총 지출 | ₩27,726 (일예산 ₩50,000 대비 **9.2%**) |
| 총 노출 | 6,778 |
| 총 클릭 → 랜딩뷰 | 169 → 132 |
| CTR / CPC | 2.5% / ₩164 |
| 퍼널 완주 (진단→예약) | **2건** (이메일 폼 제거 후 발생) |
| 전환 (리드) | 0건 (예약 확정 기준) |

## 결론

**광고는 되고 있다. 문제는 랜딩이었고, 고쳤고, 신호가 나왔다.**

1. **소재 성과 양호** — CTR 2.5%는 B2B 기준 상위권. 특히 「왜안돼」 CTR 3.02%, 「솔직히」도 안정적
2. **6일간 전환 0의 원인은 광고가 아니라 랜딩** — 이메일/이름 입력 폼에서 93% 이탈 → 4/13 제거
3. **제거 직후 첫 퍼널 완주 발생** — 결과 페이지 2조회, 상담 예약 2조회. 설문 자체는 정상 작동
4. **예산 소진율 낮음 (9.2%)** — v1은 Advantage+ OFF로 배달 제한, v2는 학습 기간이라 Meta가 천천히 쓰는 중

## 인사이트

### 1) v1 → v2 전환 효과

| | v1 (4/10~12) | v2 (4/13~14) |
|---|---|---|
| 최적화 | 링크 클릭 | **랜딩페이지 조회** |
| Advantage+ | OFF | **ON** |
| 타겟 | 25-50세 | **18-65세** |
| 예산 소진율 | 12% | **38%** (3배 개선) |

v2로 바꾸면서 예산 소진이 개선됐고, Advantage+가 학습 중

### 2) 소재 쏠림 현상

v2에서 Advantage+가 「솔직히」에 예산 99% 집중. 나머지 4개 소재 테스트 불가.
→ 나쁜 건 아님 (Meta가 판단한 최적 소재), 하지만 비교 검증이 안 되는 리스크

### 3) 진짜 병목은 풀렸다

```
[Before] 클릭 134 → 랜딩 88 → 이메일 폼 → 진단 시작 0명 (전멸)
[After]  클릭 35 → 랜딩 32 → 바로 시작 → 결과 2 → 예약 2명
```

이메일 폼 제거 한 가지 변경으로 퍼널이 뚫림. 데이터가 더 쌓이면 전환 캠페인 전환 가능
