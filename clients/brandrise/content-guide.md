## 3. 비주얼 시스템 (정밀)

### 컬러 팔레트

| 용도 | 컬러 | 설명 |
|------|------|------|
| 메인 배경 | 다크 그레이~블랙 | 그라데이션 느낌, 순수 블랙보다 약간 따뜻한 톤 |
| 강조 텍스트 | 브랜드 오렌지 | 핵심 키워드, CTA에 사용 |
| 본문 텍스트 | 화이트 | 헤드라인, 서브카피 |
| 서브 텍스트 | 라이트 그레이 | 부연 설명, 작은 글씨 |
| 데이터 박스 배경 | 반투명 다크 | 둥근 모서리, 안에 텍스트 |
| 비교표 강조 | 오렌지 배경 + 블랙 텍스트 | "최적화한 매장" 등 긍정 항목 |
| 시리즈별 표지 배경 | 오렌지, 블루, 퍼플 | 솔리드 컬러 배경에 화이트/블랙 텍스트 |

### 태그 시스템

태그는 **좌측 상단**에 위치, **둥근 모서리 라벨** 형태:

| 태그 스타일 | 용도 | 예시 |
|-----------|------|------|
| 골드/오렌지 테두리 + 다크 배경 | 시리즈/주제 태그 | `네이버플레이스 마케팅`, `brandrise real talk` |
| 오렌지 배경 + 화이트 텍스트 | 경고/강조 태그 | `경고`, `무료 상담` |
| 블루/퍼플 배경 + 화이트 텍스트 | 스테이지 태그 | `초기 스타트업`, `예비창업` |

**핵심**: 같은 게시물(캐러셀) 안에서도 **슬라이드마다 태그가 바뀜**
- 커버: `네이버플레이스 마케팅`
- 슬라이드2: `등록 vs 최적화`
- 슬라이드3: `체크리스트`
- 슬라이드4: `경고`
- 슬라이드5: `무료 상담`

```

│                              │
│ ──                           │  ← 오렌지 짧은 구분선
│                              │

│                              │
```

## 4. 콘텐츠 시리즈

### 시리즈 A: `brandrise real talk` (메인, 40%)
- **표지 배경**: 다크 그레이 / 블랙
- **태그**: `brandrise real talk` (골드 테두리)
- **목적**: 전문성 어필 + 공감 유도
- **헤드라인 패턴**: 대표들이 고민하지만 물어보기 어려운 질문
- **톤**: 직설적, 약간 도발적
- **기존 게시물**:
  - "키비주얼도 없고, 무드보드도 없고, 그냥 올리고 계신가요?"
  - "30억에서 100억, 구조가 달라야 합니다"
  - "첫 마케터, 누굴 뽑아야 할까?"
  - "대행사에 실망한 스타트업 대표에게"
  - "매출 10억의 벽, 뭐가 막고 있나?"

### 시리즈 B: `대표님, 이거 직접 하고 계세요?` (15%)
- **표지 배경**: 다크 그레이
- **태그**: `대표님, 이거 직접 하고 계세요? #N` (골드 테두리)
- **목적**: 특정 마케팅 업무의 pain point → 대행 필요성 인식
- **기존 게시물**:
  - "메타 광고, 직접 세팅하고 계세요?"

### 시리즈 C: 스테이지별 팁 (15%)
- **표지 배경**: 스테이지별 솔리드 컬러 (예비창업=오렌지, 초기 스타트업=블루)
- **태그**: `예비창업`, `초기 스타트업` 등 (컬러 배경)
- **목적**: 성장 단계별 맞춤 조언
- **기존 게시물**:
  - [예비창업/오렌지] "아이디어가 좋다고 창업하면 안 됩니다"
  - [초기 스타트업/블루] "대표님, 혼자 다 하지 마세요"

### 시리즈 D: 무료 마케팅 진단 / CTA (10%)
- **표지 배경**: 포인트 컬러 또는 다크
- **태그**: `무료 마케팅 진단` (포인트 컬러)
- **목적**: 리드 전환 (성장 전략 진단 폼 유도)
- **기존 게시물**:
  - "스타트업 마케팅, 뭐부터 해야 할지 모르겠다면"

### 시리즈 E: 채널별 실전 가이드 (15%)
- **표지 배경**: 다크 그레이
- **태그**: 채널명 (예: `네이버플레이스 마케팅`)
- **목적**: 구체적인 마케팅 채널 실전 팁 → 저장 유도
- **캐러셀 구조**: 커버 → 비교표 → 체크리스트 → 경고 → CTA (가장 많은 슬라이드)
- **기존 게시물**:
  - "네이버플레이스, 등록만 해놓으셨죠?" (7슬라이드)

### 시리즈 F: 교육형 노하우 가이드 (10%) — NEW
- **표지 배경**: 사진 배경 (라이프/제품 이미지) + 텍스트 오버레이
- **내부 배경**: 라이트 (텍스처/천 느낌) — 기존 다크와 차별화
- **태그**: 주제 카테고리 (예: `상세페이지 설계`, `피드 구조`)
- **목적**: 심화 노하우 전달 → 저장률 극대화 → "이 계정 팔로우해야 한다" 인식
- **캐러셀 구조**: 패턴 C (커버 → 번호별 스텝 + 실제 스크린샷 예시 → CTA)
- **콘텐츠 아이디어**:
  - "미친듯이 성장하는 인스타 피드 구조" (실제 브랜드 피드 캡처 포함)
  - "전환율 높은 랜딩페이지 6가지 요소" (실제 페이지 스크린샷)
  - "Meta 광고 소재, 이렇게 만드세요" (실제 광고 사례)
  - "브랜드 팝업, 성공하는 공식 5단계" (실제 팝업 사례 사진)
  - "스타트업 PR, 기자에게 보내는 보도자료 구조" (실제 보도자료 캡처)

## 5. 캐러셀 구조 패턴

### 패턴 A: 표준형 (5-6슬라이드) — real talk, 스테이지별

```
[1] 커버: 태그 + 헤드라인 + 서브카피 + (데이터 박스)
[2] 문제 제기 / 현실 직시
[3] 핵심 인사이트 1
[4] 핵심 인사이트 2 (또는 비교/사례)
[5] 정리 / 요약
[6] CTA: 무료 상담 or 프로필 링크
```

### 패턴 B: 실전 가이드형 (6-8슬라이드) — 채널별 가이드

```
[1] 커버: 채널 태그 + 도발적 질문 + 데이터 팩트
[2] 비교: 잘못된 방식 vs 올바른 방식 (비교 박스)
[3] 체크리스트 STEP 1: 항목 나열 + DATA
[4] 체크리스트 STEP 2: 항목 나열 + DATA
[5] (필요시) 추가 STEP
[6] 경고: "N개 이상 빠져 있으면..." + 위기감
[7] CTA: "지금 무료 상담 진행중" + 진단 안내
```

### 패턴 C: 교육형 스텝 가이드 (8-10슬라이드) — HOT:HAAN 스타일

**특징**: 라이트 배경 + 번호 스텝 + 실제 사례 스크린샷
**용도**: "~하는 법", "~구조", "~공식" 등 노하우 전달 콘텐츠

```
[1] 커버: 사진 배경 + 태그 + 임팩트 헤드라인
    - 실제 제품/서비스 이미지를 배경으로
    - 태그: 주제 카테고리 (예: "인스타그램 피드 설계")
    - 헤드라인: "미친듯이 OO하는 OO 구조"
[2] 스텝 1: 번호 헤딩 + 설명 + 실제 사례 스크린샷 2-3개
[3] 스텝 2: 번호 헤딩 + 설명 + 스크린샷
[4] 스텝 3: ...
[5] 스텝 4: ...
[6] 스텝 5: ...
[7] (필요시) 스텝 6
[8] CTA: 배경색 전환 + "더 많은 OO가 필요하다면?" + 팔로우/상담 유도
```

```
┌─────────────────────────────┐
│ █ N. 스텝 제목               │  ← 퍼플(또는 오렌지) 사이드바 + 볼드 헤딩
│                              │
│ 설명 텍스트 2-3줄             │  ← 블랙 본문, 자연스러운 톤
│ 포인트를 불릿(·)으로 정리      │
│                              │
│ ┌────────┐ ┌────────┐       │
│ │ 실제    │ │ 실제    │       │  ← 실제 사례 스크린샷 2-3개
│ │ 사례1   │ │ 사례2   │       │     (브랜드 상세페이지, 피드, 광고 등)
│ │ 캡처    │ │ 캡처    │       │
│ └────────┘ └────────┘       │
└─────────────────────────────┘
```

**brandrise 적용 시 주의**:
- 배경: 라이트(텍스처/천 느낌) — 기존 다크 포맷과 **의도적으로 다르게**
- 이유: 교육 콘텐츠는 밝은 배경이 가독성 + 저장률 높음
- 커버만 사진+다크로 가고, 내부는 라이트로 전환
- 실제 스크린샷 예시가 핵심 — 텍스트만으로는 이 포맷의 가치가 절반

---

### 슬라이드별 태그 전환 규칙

같은 캐러셀 안에서 슬라이드마다 태그가 바뀌어 맥락을 안내:
- 커버 → 시리즈/주제 태그
- 비교 → "A vs B" 태그
- 체크리스트 → `체크리스트` 태그
- 경고 → `경고` 태그
- CTA → `무료 상담` 태그

## 6. 카피 작성 규칙

### 헤드라인 (표지)
- **2-3줄**, 최대 15자/줄 (모바일에서 한눈에)
- **질문형** 또는 **단정형** (설명형 X)
- 핵심 키워드는 **오렌지**로 강조 (보통 마지막 줄)
- 대표의 현실적 고민/실수를 직접 짚는 톤

### 헤드라인 공식
```
패턴 1: "OO, [직설적 질문]?" ← 가장 많이 사용
패턴 2: "[숫자]에서 [숫자], [인사이트]"
패턴 3: "[대상]에게" (편지체)
패턴 4: "[대상]님, [하지 마세요/마세요]"
```

### 서브 카피 (표지 하단)
- 1-2줄, 라이트 그레이, 작은 글씨
- 헤드라인이 감성이면 서브는 구체적
- 헤드라인이 질문이면 서브는 현실 묘사

### 데이터 박스 (선택)
- 반투명 다크 배경 + 둥근 모서리
- **볼드 숫자** + 팩트 1-2줄
- 예: "매달 **3,200만 명**이 네이버에서 매장을 검색합니다."

### 비교표 카피
- 좌측 (회색 박스): 잘못된/부족한 상태 → 간결한 리스트
- 우측 (오렌지 박스): 올바른/최적화 상태 → 간결한 리스트
- 하단: 팩트 한 줄로 정리

### 체크리스트 카피
- **STEP 번호** (오렌지, 작은 글씨)
- **제목** (화이트, 볼드, 큰 글씨)
- 각 항목: □ **항목명** (볼드) + 설명 (라이트 그레이)
- 하단 DATA: 수치로 임팩트

### 경고 슬라이드 카피
- 큰 숫자/기호 (오렌지): "3+"
- 헤드라인: "N개 이상 빠져 있으면"
- 서브: 위기감 있는 현실 묘사 (오렌지 강조)
- 하단 팩트 박스: 마무리 인사이트

### CTA 슬라이드 카피
- 중앙 정렬
- "지금" (화이트) + **"무료 상담"** (오렌지, 가장 큰 글씨) + "진행중" (화이트)
- 하단: 구체적 행동 안내 1-2줄

### 캡션
- 첫 줄: 스크롤 멈추게 하는 훅 (표지와 다른 각도)
- 본문: 3-5줄, 개인적 톤
- CTA: "저장해두세요", "DM으로 편하게 물어보세요"
- 해시태그: 15-20개 (하단 분리)

### 해시태그 세트
```
기본: #스타트업마케팅 #스타트업브랜딩 #창업마케팅 #마케팅전략
타겟: #청년창업사관학교 #청창사 #신창사 #스타트업대표
채널: #인스타마케팅 #퍼포먼스마케팅 #SNS마케팅 #메타광고
브랜드: #brandrise #브랜드라이즈
```

## 8. DO / DON'T

| DO | DON'T |
|----|-------|
| 대표의 현실적 고민을 직접 짚기 | 일반적인 마케팅 이론 나열 |
| 직설적이고 자신감 있는 톤 | "~일 수도 있습니다" 같은 모호한 표현 |
| 구체적 숫자/데이터 포함 | 추상적인 조언 |
| 오렌지 강조는 핵심 키워드에만 | 오렌지를 남발하여 강조 효과 희석 |
| 슬라이드마다 태그를 바꿔 맥락 안내 | 전체 캐러셀에 같은 태그 반복 |
| "대표님", "청창사 대표" 직접 호명 | "여러분" 같은 불특정 호칭 |
| CTA는 "무료 상담" 또는 "프로필 링크" | 직접적인 가격/서비스 홍보 |
| 비교표에서 오렌지 박스 = 올바른 방향 | 좌우 구분 없이 비교 나열 |
| 커버에 데이터 팩트 박스 넣기 | 데이터 없이 감성만으로 커버 구성 |
