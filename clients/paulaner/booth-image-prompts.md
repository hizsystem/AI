## 프롬프트 1: 부스 외관 (낮)

```
파울라너(Paulaner) 맥주 브랜드의 옥토버페스트 부스 디자인을 만들어줘.

조건:
- 6m x 3m 컨테이너 7개를 ㄷ자 형태로 배치 (앞 3개 + 중간 연결 1개 + 뒤 3개)
- 컨셉: "파울라너 스테이션" — 독일 기차역을 현대적으로 재해석
- 외관: 컨테이너 골판 메탈은 그대로 노출하되, 원목 프레임으로 포인트
- 사이니지: "PAULANER STATION" 네온 사인 (파울라너 블루 컬러)
- 기차역 느낌: 역 간판 스타일, 플랫폼 지붕, 시계탑 요소
- 디자인 원칙: "나무는 살리되 포인트만 현대적으로" — 전통 원목 + 네온/LED + 그래피티
- 분위기: 힙하고 따뜻한, Heritage를 현대적으로 재해석한 느낌
- 축제 분위기, 사람들이 오가는 모습
- 건축 시각화 스타일, 포토리얼리스틱, 낮 시간대
```

## 프롬프트 2: 부스 외관 (야간)

```
파울라너(Paulaner) 맥주 브랜드의 옥토버페스트 부스, 야간 버전.

조건:
- 컨테이너 7개 ㄷ자 배치, "파울라너 스테이션" 기차역 컨셉
- 야간이라 네온/LED가 핵심 비주얼
- "PAULANER STATION" 블루 네온 사인이 빛나는 모습
- 원목 프레임 사이로 따뜻한 조명이 새어나옴
- 스트링 라이트(전구 줄)가 부스 사이를 연결
- 맥주잔을 든 사람들의 실루엣
- 컨테이너 벽에 그래피티 아트 (파울라너 로고를 스트리트 아트 스타일로)
- 분위기: 따뜻하고 활기찬 축제 야경
- 건축 시각화, 포토리얼리스틱
```

## 프롬프트 3: 부스 내부

```
파울라너(Paulaner) 맥주 부스 내부 디자인.

조건:
- 컨테이너 내부 (6m x 3m, 높이 약 2.6-2.9m)
- 컨셉: 기차 칸 느낌 — 마주보는 4인 좌석 배치
- 원목 카운터에 맥주 탭 (바이스비어 + 헬 라거)
- 따뜻한 톤 조명 (앰버/골드)
- 천장: 컨테이너 골판 메탈 그대로 노출 (인더스트리얼)
- 벽면: 원목 클래딩 + 기차역 모티브 디테일 (출발/도착 시간표 보드, 수하물 선반→맥주잔 디스플레이)
- "날것의 매력" 한글 네온 사인이 벽에
- 파울라너 블루 (#003B7E) 포인트 컬러
- 분위기: 밖은 거친 컨테이너, 안은 따뜻한 뮌헨 비어홀
- 인테리어 시각화, 포토리얼리스틱
```

## 프롬프트 4: 수도승 캐릭터 (콘텐츠용)

```
파울라너(Paulaner) 맥주 로고에 있는 수도승 캐릭터를 현대적인 3D 캐릭터로 재해석해줘.

조건:
- 원본: 갈색 수도복을 입고 맥주잔을 든 중세 수도승
- 재해석: 3D 카툰 스타일, 동글동글하고 귀여운 비율
- 표정: 허당끼 있는 미소, 살짝 취한 느낌
- 옷: 수도복은 유지하되 현대적 디테일 (선글라스를 머리 위에 올려놓거나, 스니커즈를 신거나)
- 한 손에 파울라너 맥주잔, 다른 손은 피스 사인
- 배경: 심플한 파울라너 블루
- 느낌: 진로 두꺼비 캐릭터처럼 전통 캐릭터의 현대적 재해석
- B급 유머, 허당끼, 그런데 사랑스러운
- 3D 렌더링 스타일, Pixar/Disney 느낌
```

## 프롬프트 5: 수도승 시간대별 변화 (시리즈)

```
파울라너 수도승 캐릭터의 하루를 4컷 시리즈로 만들어줘.

1컷 - 오전: 경건하게 기도하는 수도승. 수도복 깔끔하게 입고, 진지한 표정. 맥주잔이 멀리 보임.
2컷 - 점심: 살짝 미소 짓는 수도승. 맥주 한 잔 앞에 두고, 수도복 단추 하나 풀림.
3컷 - 저녁: 양 손에 맥주잔, 소매 걷어올림, 레더호젠(독일 전통 가죽바지) 위에 수도복, 선글라스.
4컷 - 밤: 맥주잔 여러 개 들고 춤추는 수도승. 수도복 완전히 풀어헤침. 최고로 행복한 표정.

스타일: 3D 카툰, 귀엽고 유머러스, 각 컷이 한 줄로 나란히 배치
컨셉: "척하다가 결국 날것이 되는" 서사
```

## 사용 순서

1. **프롬프트 1 (외관 낮)** 먼저 → 전체 방향 확인
2. 방향 맞으면 → **프롬프트 2 (야간)** + **프롬프트 3 (내부)**
3. 콘텐츠 방향 → **프롬프트 4 (수도승)** + **프롬프트 5 (시리즈)**
4. Gemini 결과 중 좋은 거 → 미드저니로 고퀄 재생성
