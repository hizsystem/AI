## 발표 흐름 스크립트 (핵심)

**시장 이해 (5분)**: "시장 상황 빠르게 짚겠습니다" → 숫자 → "파울라너는 지금 위기이자 기회입니다"

**전략 방향 (8분)**: "근데 여기서 질문이 하나 있습니다" → "모든 맥주가 세련되게 가고 있습니다" → "파울라너도 그래야 할까요?" → (잠시 멈춤) → "저희는 다른 길을 제안합니다" → **"멋진 척 하지 않는 맥주"**

**크리에이티브 (20분)**: "그래서 구체적으로 이렇게 합니다" → 인스타 시안 하나씩 → 옥토버페스트 3D → 인플루언서 → "이 모든 게 하나의 방향으로 연결됩니다"

**운영 (5분)**: "체계적으로 관리합니다" → 타임라인 → 예산 → "두 가지 옵션 드립니다"

**클로징 (2분)**: "파울라너는 멋진 척 할 필요가 없습니다. 392년의 진짜니까요."
