## 1. 캠페인 테마: 두 후보로 좁혀짐

### 후보 A: "척하지 않는 맥주" (승희 아이디어, 팀 전원 강한 반응)

- 승희가 회의 중 즉석 제안: "멋진 척 하지 않는다" → **"척하지 않는 맥주"**로 정제
- 석환: "Unfiltered랑 딱 맞고, Diesel Be Stupid 구조와 맞닿아 있다"
- 석환: "다른 맥주들은 전부 '척'하고 있다 — 하이네켄(쿨한 척), 스텔라(고급진 척), 호가든(감각적인 척). 우리만 안 한다는 게 독보적"
- 우성민: "'척'이라는 단어 자체가 한국어. 로컬라이제이션에 딱 맞는 말"
- 석환: "인터넷에서 '척척하지 마라' 유행어도 있다. 가식 떨지 마라, 척하냐 등"

### 후보 B: "정답은 없음" (우성민 + Claude Code 작업)

- 우성민이 이전 세션에서 도출한 테마
- "맥주의 정답은 없다. 따르는 법도, 온도도, 안주도."
- 팀 반응: 방향은 좋으나 단어 자체가 다소 평범 ("정답은 없으면 진짜 구리긴 해요" - 우성민 본인 인정)

### 팀 합의

- **두 테마가 상호배타적이지 않음** — "척하지 않는 맥주"가 브랜드 태도, "정답은 없음"이 실행 화법
- 승희: "척하지 않는"은 인플루언서 미팅/바이럴 콘텐츠에, "정답은 없음"은 콘텐츠 시리즈 화법으로 쓰면 좋겠다
- **다음 회의까지 최종 확정 필요**

## 2. Diesel "Be Stupid" = 핵심 레퍼런스 확정

우성민이 직접 PT 구조의 핵심 참조점으로 제시:

- Diesel: 오래된 패션 브랜드가 기존 관습을 뒤집음
- "Smart vs Stupid" → 우리: **"척하는 맥주 vs 척하지 않는 맥주"**
- "Smart는 뇌가 필요하지만 Stupid는 이야기가 있다" 같은 대비 구조
- RFP의 "Imperfect, not perfect" 화법과 정확히 동일한 구조

## 3. 비주얼 방향: "Heritage를 현대적으로 재해석"

### 문제 인식 (전원 공감)

- 파울라너/옥토버페스트 비주얼을 그대로 풀면 **촌스러움** (목재 테이블, 전통 의상, 알전구)
- 한국에 지역 맥주 축제가 1000개 가까이 → 뻔한 독일 축제 비주얼로는 차별 불가
- 파울라너 블루(짙은 남색)가 눈에 잘 안 띄는 색상 (오렌지 같은 쨍한 색이 아님)
- BUT 헤리티지를 버리면 안 됨 (RFP에서도 강조, 파울라너의 핵심 자산)

### 해결 방향 (합의)

> **"나무는 살리되, 포인트만 현대적으로"**

- 기본 구조(원목, 옥토버페스트 상징물)는 유지
- 포인트 요소를 현대적으로 전환:
  - 전통 알전구 → **네온/LED**
  - 전통 의상 → **힙하게 재해석한 유니폼** (해외 사례 있음)
  - 전통 장식 → **그래피티** (신찬양 같은 아티스트 콜라보 가능)
  - 수도승 캐릭터 → **현대판 캐릭터화** (진로 두꺼비 사례)
- 남중: "결국 옥토버페스트의 정통 비주얼을 갖고 오되, 포인트되는 부분만 현대적으로 — 그래피티, LED, 네온"
- 승희: "뮌헨 복숭아 축제(Oktoberfest)의 비주얼이랑 파울라너가 유사. 헤리티지 버리면 안 된다"

### 진로 두꺼비 = 국내 레퍼런스

- 100년 된 두꺼비 캐릭터를 현대적으로 리브랜딩 → 클럽, 팝업 등에서 성공
- 파울라너 수도승도 같은 방식으로 현대판 캐릭터화 가능
- 대표님: "아예 캐릭터로 해버리면 좋겠다. 동물 캐릭터가 나와서 돌아다니는 거"

## 4. 부스 컨셉: "파울라너 스테이션" (승희 아이디어, 팀 채택)

### 아이디어

- 독일 → 서울을 연결하는 **기차역(Station)** 컨셉
- 컨테이너 = 기차 칸 / 역 플랫폼
- "파울라너가 브릿지가 되어주는 것. 독일의 문화를 우리 것으로 만드는 과정"

### 팀 반응

- 남중: "독일과 한국을 연결하는 공간이 된다는 게 너무 좋다"
- 대표님: "해리포터 9¾ 승강장처럼, 오래된 역에서 출발해서 시공간을 이동하는 느낌"
- 승희: "파울라너 스테이션. 앞에 뭐가 붙을지 — Heritage와 현대의 Heritage가 만나는 것"
- 우성민: "New Heritage... 유리티지... 같은 신조어?" (검토 필요)
- **전원 즉시 합의: "이걸로 가자"**

### 미해결: 물리적 구조 확인 필요

- 7개 연결 컨테이너의 실제 배치 (디귿자? 일렬? 안/밖 방향?)
- 야외 공간 활용 범위
- 텐트 안쪽 / 바깥쪽 소비자 동선
- **대표님이 하이네켄코리아 비딩 담당자에게 직접 연락하기로 함** (메일 or 전화)

## 핵심 Take-away

1. **"척하지 않는 맥주"가 팀 내 가장 강한 공감을 얻은 테마.** "정답은 없음"은 실행 화법으로 병행 가능.
2. **"파울라너 스테이션"** 부스 컨셉이 즉석에서 전원 합의 — Heritage와 현대를 연결하는 기차역.
3. **비주얼의 핵심은 "나무는 살리되 포인트만 현대적으로"** — 그래피티, 네온, LED, 캐릭터.
4. **석환의 맥주캔→옥토버페스트 영상 프로토타입**이 PT 티저 콘텐츠로 매우 유력.
5. **동물 캐릭터**를 대표님이 강하게 밀고 있음 — 캔 디자인 활용 + 시간대별 변화 시리즈.
6. **3D 렌더링은 외부 전문가 필수** (AI 한계 인정, 워킹타임 5일).
7. 부스 물리적 구조는 아직 미확인 — 대표님이 담당자 연락 예정.
