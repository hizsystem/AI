## 1. 문제 인식: 왜 파울라너는 지금 위기이자 기회인가

### 위기

- 한국 소비자 인지도 **1.5%** — "파울라너? 그게 뭐야?"
- 밀맥주 시장 내 **3등** (크로넨버그 35% > 호가든 28% > 파울라너 14%)
- "독일 맥주"의 한국 내 이미지 = 올드, 무거움, 아저씨 맥주
- 하이트진로 → 하이네켄코리아 유통 전환 = 리셋. 기존 인지도마저 초기화

### 기회

- 유통 전환 = **리런칭 명분**. 새 판을 깔 수 있는 타이밍
- 서울 옥토버페스트 2026 (9월, 18만명) = **대규모 경험 플랫폼** 확보
- 하이네켄코리아의 로컬 캠페인 역량 (Trust Bar, Rooftop Revival 실행력)
- 글로벌 캠페인 "Life Unfiltered" = 한국 MZ세대 트렌드와 정합성 높음

## 2. 시장 진단: 모든 맥주가 세련됨을 경쟁하고 있다

### 한국 수입맥주 포지셔닝 맵

```
          세련됨 / 프리미엄
              |
    스텔라 ----+---- 하이네켄
              |
    1664 -----+---- 코로나
              |
 대중적 ------+------ 프리미엄
              |
    카스 ------+---- 삿포로
              |
    테라 ------+---- 아사히
              |
          친근함 / 대중성
```

모든 수입맥주가 "세련된 프리미엄"을 향해 달리고 있다.
하이네켄, 스텔라, 1664, 코로나 — 전부 "쿨하고 감각적인 맥주".
**이 판에서 파울라너가 싸우면 진다.** 인지도 1.5%로는 안 된다.

### 파울라너가 세련되게 가면 안 되는 이유

1. **하이네켄의 열화판이 된다** — 같은 유통사에서 같은 톤으로 나오면, 본사 브랜드에 묻힘
2. **"잘 만든 AI"의 피로** — 소비자는 이미 완벽하게 포장된 콘텐츠에 지쳐있다
3. **파울라너의 진짜 자산은 세련됨이 아니다** — 392년 수도원 양조, 뮌헨 최대 텐트, 무여과 맥주. 이건 "쿨함"이 아니라 "진짜"에 가깝다

## 3. 전략 방향: "멋진 척 하지 않는 맥주"

### 핵심 전략 선언

> **파울라너는 세련됨 대신 솔직함으로 간다.**
>
> 모든 맥주가 멋진 척 할 때,
> 파울라너는 꾸미지 않은 진짜 모습으로 나온다.
>
> 392년을 숨기지 않는다. 올드함을 유쾌하게 드러낸다.
> 그 솔직함이 감정을 만들고, 감정이 밈이 되고, 밈이 인지도가 된다.

### 글로벌-로컬 연결 구조

```
[Global]  "Here's to Life Unfiltered"
              ↕ (번역이 아닌 해석)
[Korea]   "멋진 척 하지 않는 맥주"
              ↕ (실행 언어)
[내부]    "날것의 매력" — 팀 작업 방향 키워드
```

- **Global "Unfiltered"**: 필터를 거치지 않은 삶의 순간
- **Korea 해석**: 필터를 거치지 않은 = 멋진 척 하지 않은 = 날것 그대로의
- **제품 연결**: 파울라너 헤페바이스비어는 문자 그대로 무여과(Unfiltered) 맥주. 효모가 살아있는, 맑지 않은, 흐린 맥주. 브랜드 철학과 제품 속성이 하나로 합쳐짐.

### 왜 이 방향이 하이네켄코리아에게도 좋은가

| 하이네켄 | 파울라너 |
|---------|---------|
| 세련됨, 도시적, 글로벌 | 솔직함, 유쾌함, 정통성 |
| 프리미엄 라거 | 무여과 바이스비어 |
| "Open Your World" | "Life Unfiltered" |
| 새벽 축구, 옥상 문화 | 옥토버페스트, 건배 문화 |

**같은 유통사 안에서 캐릭터가 완전히 다르다.** 파울라너가 하이네켄처럼 하면 자기잠식. 다르게 해야 포트폴리오 가치가 생긴다.

하이트진로 시절에는 "정돈되고 깔끔하게" 했다. 하이네켄코리아로 왔으니 **다르게 가고 싶은 니즈**가 있다. 이 전략이 그 니즈에 대한 답이다.

## 4. 로컬리제이션 근거: 성공한 맥주 브랜드들은 어떻게 했나

### 벤치마크 케이스

**아사히 — "한국 세계 최초" 전략**
- 생맥주캔을 한국에서 세계 최초 출시 → 불매운동 이후 매출 8배 성장, 수입맥주 1위 탈환
- 교훈: 제품 혁신 + "한국이 세계 최초"라는 자존심 자극

**하이네켄 — "한국 문화적 진실" 전략**
- Trust Bar (새벽 무인 축구 바), Rooftop Revival (서울 옥상을 소셜 허브로)
- 글로벌 메시지에서 출발하지 않고, 한국의 문화적 진실에서 출발
- 교훈: 번역이 아닌 해석. 한국 사람이 공감하는 장면에서 시작해야 한다

**삿포로 — "제품 경험" 전략**
- 해외 최초 비어스탠드를 한국(성수동)에 오픈, 셀럽 없이 63% 성장
- 교훈: 체험 > 광고. 제대로 마시게 해주면 제품이 말한다

**호가든/1664블랑 — "카테고리 창출" 전략**
- 호가든이 "벨기에 밀맥주", 1664블랑이 "프렌치 밀맥주" 카테고리를 만듦
- 한국이 세계 1위 소비국/판매국이 되며 글로벌 캠페인까지 바꿈
- 교훈: 기존 판에서 싸우지 말고, 새 판을 깔아라

### 파울라너에 적용

| 벤치마크 전략 | 파울라너 적용 |
|-------------|------------|
| 한국 세계 최초 | 파울라너 브라우하우스 아시아 최초 팝업 or 옥토버페스트 한국 최초 한정판 |
| 문화적 진실 | 한국의 건배 문화 + "멋진 척"에 지친 MZ세대의 결핍 |
| 제품 경험 | 서울 옥토버페스트 9개 부스 = 정통 뮌헨 맥주 체험 공간 |
| 카테고리 창출 | "뮌헨 바이스비어" 또는 "옥토버페스트 비어"로 밀맥주와 다른 판 |

## 5. 타겟 인사이트: 한국 MZ세대가 공감하는 "Unfiltered"

### 타겟 프로필

- **핵심**: 25-35세, 서울/수도권, 맥주를 "경험"으로 소비하는 층
- **확장**: 20대 초 ~ 30대 후반, 축제/팝업/인스타 콘텐츠 소비층

### 타겟의 문화적 진실

> **"잘 꾸며진 것에 지쳤다."**

- 인스타는 필터투성이. 광고도 필터투성이. 사람도 필터투성이.
- 그래서 오히려 **"날것"에 끌린다** — 비하인드, 블로퍼, 자컨(자체 콘텐츠), 밈
- "나혼자산다"가 10년째 인기인 이유: 꾸미지 않은 일상의 매력
- 로이킴, 올데이프로젝트, 코르티즈 — **날것의 감성이 곧 매력인 크리에이터들**

### 비포-에프터로 표현하면

```
[Before = Filtered]          [After = Unfiltered]
잘 차려진 맥주 광고           맥주 따를 때 거품 넘치는 리얼
완벽한 인스타 피드             올리지 않은 사진이 더 좋았던 밤
회식에서 억지 건배             마시고 싶은 사람과 마시고 싶은 맥주
AI가 완벽하게 만든 콘텐츠      대놓고 AI인데 오히려 웃긴 콘텐츠
```

## 6. 크리에이티브 전략: "날것의 매력"을 어떻게 실행하나

### 톤 & 매너

| 파울라너 O | 파울라너 X |
|-----------|-----------|
| 유쾌, 솔직, 반전 | 무겁고 진지한 |
| B급 감성 (일부러 거친) | 촌스러운 |
| 정통성이 자연스럽게 묻어남 | 독일 코스프레 |
| 대놓고 웃김 → 근데 기억남 | 웃기려다 실패한 |
| 밈이 될 수 있는 | 공유 안 되는 |

### PT에서의 언어 전환 (중요)

팀 내부에서는 "B급", "날것"이라 부르지만, **PT 장표에서는 다르게 써야 한다.**
사공차장님은 "웃긴 거 싫어하심" — B급이라는 단어가 나오면 거부감.

| 내부 언어 | PT 장표 언어 |
|----------|------------|
| B급 | **Unfiltered 실행** / 솔직한 콘텐츠 |
| 날것 | **꾸밈없는 매력** / 필터 없는 순간 |
| 밈 | **자발적 확산 콘텐츠** / 소비자 참여형 |
| 일부러 거친 | **의도된 날것** / 리얼리티 연출 |
| 웃김 | **친근함** / 유쾌한 정통성 |

### 설득 로직 (PT 내 필수 장표)

> **"왜 파울라너는 세련됨이 아닌 솔직함으로 가야 하는가"**
>
> 1. 세련된 맥주는 이미 너무 많다 (하이네켄, 1664, 스텔라, 코로나)
> 2. 하이네켄처럼 하면 하이네켄의 열화판이 된다 — 같은 유통사 안에서 자기잠식
> 3. 파울라너의 가장 큰 자산(392년, 뮌헨 최대 텐트, 무여과)은 "세련됨"보다 "진짜"에 가깝다
> 4. "진짜"를 실행하면, 솔직한 콘텐츠가 되고, 이것이 MZ세대 트렌드와 정확히 맞는다
> 5. 솔직함이 밈이 되고 → 밈이 바이럴이 되고 → 바이럴이 인지도(1.5% → 목표치)가 된다
> 6. 하이트진로와 확실히 다른 접근 = 하이네켄코리아로의 전환 가치를 증명

## 7. 실행 체계: 3개 축

### 축 1: 인스타그램 — 밈 팩토리 (연중)

**역할**: 파울라너의 새로운 이미지를 만드는 콘텐츠 공장

**콘텐츠 유형**:

| 유형 | 설명 | 빈도 |
|------|------|------|
| 반전 콘텐츠 | 고전적 맥주 콘텐츠에 반전 (따르기 → 콸콸콸, 한 잔 → 3L) | 주 1-2개 |
| AI 콘텐츠 | 대놓고 AI. 3D 동물 옥토버페스트, B급 AI 영상 | 주 1개 |
| 비포-에프터 | Filter → Paulaner → Unfilter 포맷 | 격주 1개 |
| 날것 시리즈 | 인플루언서/크리에이터의 무대 뒤 + 파울라너 | 월 2개 |

**톤**: "이 계정 뭐야? 맥주 계정인데 왜 웃기지?" → 팔로우

### 축 2: 서울 옥토버페스트 — 경험 폭발 (9월)

**역할**: 온라인에서 쌓은 이미지를 오프라인에서 폭발시키는 플랫폼

**핵심 경험 설계**:
- 9개 컨테이너를 "뮌헨 비어가르텐 마을"로 구성
- 인스타에서 보던 콘텐츠/캐릭터가 현실에 등장
- 참여형 이벤트: 맥주잔 10잔 들기 챌린지, 총 음주량 실시간 카운터
- No Filter Zone: 폰 내려놓고 맥주에 집중하는 공간 (Unfiltered 체험)

**인스타 → 축제 연결**:
- 온라인에서 바이럴된 3D 동물 캐릭터가 축제장에 실물로 등장
- "맥주 따르기 콸콸콸" 콘텐츠 → 축제장에서 실제로 콸콸콸 체험존
- 비포-에프터 포토부스: 정장 차림 → 파울라너 한 잔 후 → 넥타이 풀린 모습

### 축 3: 인플루언서 & PR — 확산 (연중)

**역할**: 파울라너의 "날것" 무드를 신뢰할 수 있는 사람의 입으로 전달

**인플루언서 유형**:

| 유형 | 레퍼런스 | 역할 |
|------|---------|------|
| 날것 크리에이터 | 로이킴, 올데이프로젝트, 코르티즈 | "이 맥주는 멋진 척 안 해서 좋다" |
| 밈 계정 | 유머/밈 특화 계정 | 파울라너 콘텐츠를 밈으로 2차 확산 |
| 푸드/맥주 | 맥주 전문가, 푸드 크리에이터 | 제품력(무여과, 정통성) 전달 |
| 뮌헨 파견 | 실제 뮌헨 옥토버페스트 체험 | "진짜 뮌헨은 이랬다" 콘텐츠 |

## 8. 2026 로드맵: 빌드업 → 폭발 → 리포지셔닝

```
Q1 (1-3월): [준비]
├── 인스타 계정 리런칭 + 콘텐츠 톤 세팅
├── 3D 캐릭터 개발
└── 인플루언서 풀 확보

Q2 (4-6월): [빌드업]
├── 인스타 밈 콘텐츠 본격 가동
├── "이 맥주 계정 뭐야?" 반응 유도
├── 날것 크리에이터 콜라보 시작
└── 서울 옥토버페스트 사전 티징

Q3 (7-9월): [폭발]
├── 옥토버페스트 카운트다운 콘텐츠
├── 뮌헨 옥토버페스트 현지 콘텐츠 (9월 중순)
├── 서울 옥토버페스트 (9월 하순) — 경험의 정점
└── 축제 현장 실시간 콘텐츠 → 바이럴 극대화

Q4 (10-12월): [리포지셔닝]
├── 축제 경험을 일상 콘텐츠로 전환
├── "파울라너 = 멋진 척 안 하는 맥주" 인식 정착
├── 세련된 톤의 콘텐츠 비중 점진적 확대
└── 연말 캠페인: 올해의 Unfiltered 순간 회고
```

### 단계별 톤 변화

| 단계 | 톤 비율 | 목적 |
|------|--------|------|
| Q1-Q2 빌드업 | 유쾌 70% : 정통 30% | 감정 생성. "재밌네, 웃기네" |
