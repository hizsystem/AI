# 파울라너 컨테이너 부스 비주얼 레퍼런스 브리프

> 작성일: 2026-03-26
> 목적: 9개 컨테이너 "Munich Biergarten Village" 3D 렌더링 의뢰용 비주얼 브리프
> 장소: 문화비축기지 (마포구 매봉산로 87)
> 컨셉: "멋진 척 하지 않는 맥주" — 컨테이너의 산업적 날것을 숨기지 않되, 안에 진짜 바이에른의 따뜻함을 채운다

---

## 핵심 디자인 텐션

```
외부 = 날것, 인더스트리얼, 거친 금속
        ↕  이 대비가 "Unfiltered"를 공간으로 번역한 것
내부 = 따뜻함, 원목, 바이에른 진정성, 프리미엄
```

**절대 안 되는 것**: 싸구려 축제 부스, B급 팝업, 독일 코스프레(프라우엔 교회 벽지+플라스틱 맥주잔)
**되어야 하는 것**: "컨테이너인데 안에 들어가면 뮌헨 비어홀이네" — 반전의 감동

## REFERENCE 1. Munich Brauhaus Melbourne (Techne Architecture) — 핵심 디자인 레퍼런스

### 무엇인가
멜버른 사우스워프(South Wharf), 900석 규모 현대식 독일 비어홀. 설계: Techne Architecture + Interior Design (2015). 과거 화물 창고(goods shed)를 개조한 공간.

### 핵심 디자인 요소

**1) 통나무 벽 + 사슴 실루엣 (Front Bar Feature Wall)**
- 수평으로 쌓은 통나무(full logs, 껍질 포함)가 바닥부터 천장까지 벽면 전체를 덮음
- 통나무 사이에 실물 크기 사슴 형상이 수평으로 배치 — 숲속을 걷는 듯한 효과
- 이것이 프론트 바의 포컬 포인트이자 공간 전체의 시그니처
- **파울라너 적용**: 컨테이너 1개의 내부 벽면 전체를 이 기법으로. 단, 사슴뿔 대신 미니멀한 금속 실루엣 사슴으로 현대화. 또는 바이에른 알프스 산맥 실루엣으로 변환

**2) 3개 존 구성 (Zone Separation)**
- Front Bar: 캐주얼 스탠딩+바 좌석, 통나무 벽이 배경
- Main Dining Hall: 전통 커뮤널 장탁자(picnic-style timber tables), 가장 넓은 공간
- Mezzanine: 2층 다이닝, 프라이빗 공간
- 3개 독립 바가 각 존에 배치
- **파울라너 적용**: 9개 컨테이너를 3개 존(Bier/Kultur/Fest)으로 묶어 유사 구조 재현

**3) 바이에른 블루 바 카운터**
- 메인 바의 프론트가 바이에른 블루(#0066B2 계열)로 도색
- 전통적 하를킨(다이아몬드) 패턴이 아닌, 솔리드 블루로 현대적 해석
- 원목+금속 공간에서 블루가 악센트 컬러로 "팝"
- **파울라너 적용**: 메인 바 컨테이너의 바 카운터 전면을 파울라너 블루로. LED 백라이트 추가

**4) 인더스트리얼 x 내추럴 재료 대비**
- 천장: 노출 철골 보(exposed steel beams), 덕트, 산업 조명
- 벽/장식: 통나무, 녹색 식물, 사슴뿔 장식
- 바닥: 콘크리트 + 부분 원목
- 가구: 두꺼운 원목 상판 테이블 + 스틸 프레임 베이스
- **파울라너 적용**: 컨테이너 천장 = 그대로 노출(코루게이트 메탈), 벽 = 원목 클래딩, 바닥 = 원목 데크

## REFERENCE 2. 프리미엄 컨테이너 바 & 페스티벌 활용 사례

- 20ft High Cube 컨테이너 2개를 클러스터로 결합

- 6m 바 카운터, 언더카운터 냉장고 2대, 백바 기본 장착
- **파울라너 적용**: 더블데커(2층) 구조 — 1층 바+2층 루프톱 관람석/포토존

### 2-B. EP1002 Double Deck Brand Activation (ExpoMarketing)
- 30x30ft 더블데커 컨테이너 부스
- 커스텀 랩 그래픽 + 커스텀 바 카운터 + 상층 데크 좌석(우산 차양)

- **파울라너 적용**: 컨테이너 외벽 = 파울라너 브랜디드 래핑, 상층 = 뮌헨 스카이라인 실루엣 그래픽

### 2-C. Party Box (Experiential Vehicles)
- 듀얼 스테이지 커스텀 컨테이너 — 턴키 호스피탈리티 솔루션
- 루프톱 데크 + 조명 + TV + 마감 바닥 기본
- 스포츠 이벤트, 페스티벌에서 접근성 높은 하이 뷰잉 데크 제공
- **파울라너 적용**: 9개 중 2개를 더블데커로 — 문화비축기지 파노라마 뷰

### 핵심 설계 원칙 (싸구려 안 되려면)
1. **벽면 전동 개방**: 낮에는 벽 오픈 → 데크+바 카운터 전환, 개방감
2. **루프톱 데크**: 더블데커 2-3개로 2층 관람석 → 축제 전경 조망 → 프리미엄 느낌
3. **내부 마감**: 단열+창문 커스텀 필수. 코루게이트 메탈 노출이되 내부는 원목 클래딩
4. **논슬립 우드/비닐 데크**: 상층 바닥은 원목 데크 마감, 안전 난간
5. **LED 무드 조명 + 브랜디드 사이니지**: 야간 존재감

## REFERENCE 4. 인더스트리얼 x 바이에른 퓨전 미학

### 핵심 공식

```
코루게이트 메탈(컨테이너 외피) + 원목(내부 클래딩) + 코퍼/브라스(디테일) + 바이에른 블루(악센트)
= "Authentic but Impressive"
```

### 재료별 적용 가이드

**1) 코루게이트 메탈 (Corrugated Metal)**
- 컨테이너 외벽은 그대로 노출하되 깨끗하게 세척/도장
- 내부 천장도 코루게이트 메탈 노출 — "여기가 컨테이너임"을 숨기지 않는다
- 바 카운터 프론트에 코루게이트 메탈 + 원목 상판 조합
- 레퍼런스: 코루게이트 메탈 바 프론트 + 리클레임드 우드 카운터탑 = "farmhouse industrial"

**2) 원목 / 리클레임드 팀버 (Raw Timber)**
- 내부 벽면: 수평 원목 클래딩(Munich Brauhaus 스타일)
- 테이블: 두꺼운(최소 5cm) 원목 상판 + 스틸 프레임 다리
- 바닥: 원목 데크 (컨테이너 바닥 위에 설치)
- 장식: 통나무 단면(tree trunk cross-section) 벽 장식 — 바이에른 숲 느낌
- 바 카운터탑: 리클레임드 우드 또는 두꺼운 부처블록 스타일
- "원목은 따뜻함의 핵심 — 금속 공간에서 인간의 온기를 만드는 유일한 재료"

**3) 코퍼 / 브라스 악센트 (Copper / Brass)**
- 맥주 탭 시스템: 코퍼 파이프 + 브라스 핸들
- 펜던트 조명: 코퍼 셰이드 + 에디슨 벌브
- 바 레일/풋레스트: 브라스 파이프
- Paulaner Brauhaus의 시그니처 코퍼 양조 케틀 연출 — 실물 또는 축소 모형
- "코퍼의 따뜻한 광택은 금속 공간을 '양조장'으로 변환시키는 마법 재료"

**4) 그린/식물 (Greenery)**
- 컨테이너 사이 동선: 홉 덩굴(hop vines) 또는 아이비 늘어뜨리기
- 맥주 원재료(보리, 밀, 홉) 말린 것을 유리병에 넣어 바 장식
- 화분: 허브 포트(민트, 로즈마리) — 바이에른 비어가르텐의 밤나무 아래 분위기 축소 재현

### Modern Beer Hall 디자인 트렌드 (rd+d Magazine 참조)
- 미국/호주의 현대식 비어홀들은 유럽 전통을 "번역이 아닌 해석"
- 하를킨 패턴 → 블루 타일로 탭 월에 적용
- 커뮤널 테이블 → 두꺼운 원목+스틸 베이스로 현대화
- 통나무 단면과 혼합 우드톤 → 독일 숲속 느낌을 도시 공간에 이식

## REFERENCE 5. 한국 맥주 팝업스토어 선행 사례

### 5-A. 삿포로 프리미엄 비어스탠드 (성수동, 2025)
- **무엇**: 해외 최초 삿포로 비어스탠드, 성수동 29평 매장
- **인테리어**: 도쿄 긴자점을 모델로 한 블랙 톤 인테리어, 스탠딩 바 포맷(좌석 없음)
- **특징**: "Perfect Pour"(맥주:거품 7:3)와 "Classic Pour" 2종 체험
- **성과**: 셀럽 없이 제품 경험만으로 63% 매출 성장
- **교훈**: 제품 경험에 집중하면 공간이 작아도 프리미엄 느낌 가능
- **파울라너 적용**: "Paulaner Pour Master" 존 — 헤페바이스비어 회전 따르기 체험. 삿포로가 "Perfect Pour"라면 파울라너는 "Unfiltered Pour"

### 5-B. 아사히 수퍼드라이 "MODERN DRY BAR" (성수동, 2025)
- **무엇**: 아사히 수퍼드라이 "MODERN DRY" 세계관 체험 팝업
- **인테리어**: 입구 "생맥주캔 가든" + 대형 오브제 포토존, 1층 드래프트 바+핑거푸드
- **특징**: 브랜드 세계관을 물리적 공간으로 구현
- **교훈**: 대형 오브제 + 제품 체험 + 포토존 = 팝업 기본 공식

### 5-C. 아사히 트래블 바 (성수동, 2025-2026)
- **무엇**: "술과 함께하는 세계여행" 테마 — 도쿄 아사쿠사 아사히맥주 본사 빌딩 재현
- **특징**: 각 존이 다른 도시를 재현 → 여행 경험
- **교훈**: "한 장소 안에 여러 도시" 컨셉 = 우리의 "9개 컨테이너 = 바이에른 마을 다른 장소"와 정확히 일치

### 5-D. 하이네켄 실버 팝업 (합정동/양양, 2023)
- **무엇**: 하이네켄 실버 런칭 팝업, GS25 프리미엄점
- **교훈**: 같은 하이네켄코리아 포트폴리오 — PT에서 "하이네켄의 팝업 실행력 위에 파울라너 경험을 올린다"로 어필 가능

### 한국 팝업 트렌드 종합

## REFERENCE 6. 컨테이너 빌리지 / 복합 배치 사례

### 6-A. Boxpark Shoreditch, London
- **무엇**: 세계 최초 팝업 몰. 60개 재활용 컨테이너, 2층 구조
- **배치**: 하층 40개 + 상층 20개, 오픈에어 코트야드 중심 배치
- **규모**: 19개 식당+바, 27개 숍. 연간 100만+ 방문
- **설계**: Waugh Thistleton Architects — 과거 철도 화물장을 활용
- **핵심 교훈**: 컨테이너는 "건물"이 아니라 "마을의 블록" — 사이 동선이 곧 거리
- **파울라너 적용**: 9개 컨테이너 사이를 "뮌헨 골목길"로 연출. 자갈/잔디/원목벤치 배치

### 6-B. Downtown Container Park, Las Vegas
- **무엇**: 43개 컨테이너 + 41개 큐브, 38개 테넌트 공간
- **배치**: 남북 방향 배치, 동서로 컨테이너 3층 스택, 중앙 광장에 인터랙티브 놀이터+콘서트장
- **상징물**: 입구에 55ft 금속 사마귀(mantis) — 불꽃 분사 퍼포먼스
- **핵심 교훈**: (1) 중앙 광장이 구심점, (2) 입구 랜드마크가 필수, (3) 숨겨진 구석과 열린 공간의 교차가 탐험 동기 부여
- **파울라너 적용**: 9개 컨테이너 배치 시 중앙 광장(Prost Platz) 확보 + 입구에 파울라너 수사(monk) 대형 오브제 또는 Masskrug 대형 조형물

### 6-C. Wahaca Southbank, London
- **무엇**: 8개 컬러 컨테이너, 2층 구조 멕시칸 레스토랑
- **배치**: 2개씩 짝을 지어 스택, 중앙 유리 아트리움
- **디자인**: 각 컨테이너가 4가지 비비드 컬러(딥 터쿼이즈~스트로 옐로) — 멕시코 거리 + 항구 컨테이너선 색감 참조
- **설계**: Softroom (dezeen 소개)
- **핵심 교훈**: 컨테이너 외벽 컬러가 곧 브랜드 아이덴티티. 각 컨테이너가 다른 역할(1층 레스토랑, 2층 테킬라 바)
- **파울라너 적용**: 9개 컨테이너 각각 다른 "바이에른 마을 장소" 테마 + 외벽은 통일된 파울라너 블루/골드 그라데이션

### 6-D. Smoky Park Supper Club, Asheville
- **무엇**: 19개 컨테이너로 만든 미국 최대 컨테이너 레스토랑
- **배치**: 강변에 2,400sqft 랩어라운드 데크로 연결
- **설계**: SG Blocks — 메인 구조물 3일 만에 설치
- **디자인**: 대형 창문 커팅 → 자연채광+통풍+강 조망 확보
- **핵심 교훈**: 컨테이너에서 "열린 느낌"을 만들려면 벽면 커팅+창문이 핵심. 닫힌 금속 상자 느낌을 깨야 한다
- **파울라너 적용**: 전동 개방 벽면(낮: 오픈 → 밤: 클로즈) + 대형 창문

## REFERENCE 7. Paulaner Brauhaus 실제 인테리어 (글로벌)

### 글로벌 Paulaner Brauhaus 공통 디자인 DNA

**1) 코퍼 양조 케틀 (Copper Brewing Kettles)**
- 모든 Brauhaus의 가장 눈에 띄는 요소
- 광택나는 코퍼 케틀 + 파이프 + 피팅이 바 뒤에 배치
- 실제 마이크로 브루어리로 작동 (현장 양조)
- **파울라너 적용**: 실물 또는 정교한 모형 코퍼 케틀을 메인 바 컨테이너에 배치. "여기서 직접 양조한다" 스토리

**2) 다크 팀버 + 원목 바닥 (Dark Timber Paneling)**
- 다크 팀버 패널링, 미가공 원목 바닥(unpolished timber floors)
- 견고한 원목 테이블과 의자(robust timber tables and chairs)
- "따뜻한 색상, 자연 다크 팀버, 바이에른 테마 장식"이 공간 기본 공식

**3) 마이바움 (Maypole)**
- 싱가포르점: 바닥부터 천장까지 솟아오르는 블루+화이트 줄무늬 마이바움이 장식의 중심
- 전통 바이에른 마을 축제의 상징물
- **파울라너 적용**: 중앙 광장(Prost Platz)에 대형 마이바움 설치 — 랜드마크이자 미팅 포인트

**4) 산악 샬레 느낌 프라이빗 룸 (Mountain Chalet)**
- 싱가포르점: 코지한 반프라이빗 룸이 바이에른 산악 샬레에서 영감
- **파울라너 적용**: 9개 중 1개를 "Stammtisch" 프라이빗 컨테이너로 — 산악 샬레 인테리어

**5) 밴드 스테이지 (Live Music)**
- 1층에 밴드 스테이지 기본 배치 — 바이에른 전통 음악 라이브
- **파울라너 적용**: 중앙 광장에 소규모 스테이지 — "Ein Prosit" 라이브 + DJ

### Paulaner Brauhaus 위치별 특징

| 위치 | 특징 |

## 조명 레퍼런스

### 레이어 1: 스트링 라이트 (String Lights)
- 컨테이너 사이 동선 상부에 교차로 설치
- 에디슨 스타일 벌브 (따뜻한 앰버톤, 2200K~2700K)
- "비어가르텐의 밤나무 아래" 분위기를 도시에서 재현
- 헤비듀티 상업용 등급 (방수, 연중 사용 가능)
- 검색어: "commercial grade Edison string lights beer garden"

### 레이어 2: 펜던트 라이트 (Pendant Lights)
- 컨테이너 내부: 코퍼 셰이드 펜던트 + 에디슨 벌브
- 바 카운터 상부에 3-5개 일렬 배치
- 코퍼 셰이드가 양조장/인더스트리얼 느낌 강화
- 검색어: "copper shade pendant light industrial bar"

### 레이어 3: LED 악센트 (LED Accent)
- 바이에른 블루(#0066B2) LED 스트립: 바 카운터 하부, 선반 하부
- 화이트 LED: 파울라너 로고/사이니지 백라이트
- "Prost Time"에 전 구역 LED 동시 변환(블루+화이트 플래시)
- 검색어: "LED accent lighting bar blue neon"

### 레이어 4: 캔들/랜턴 (Warm Glow)
- 커뮤널 테이블 위: 유리 캔들홀더 or 작은 랜턴
- 가장 친밀한 레이어 — 테이블 단위의 따뜻함
- 검색어: "table candle lantern beer hall"

### 종합 조명 원칙
```
밤 분위기 =
  스트링 라이트(공중, 동선)
  + 펜던트(바 내부, 포커스)
  + LED 블루(브랜드, 악센트)
  + 캔들(테이블, 친밀함)
= 4겹 레이어로 "따뜻하지만 분위기 있는" 야간 연출
```
