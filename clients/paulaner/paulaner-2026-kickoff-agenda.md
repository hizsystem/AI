### 2. 핵심 질문 3개 — 답을 내린다 (30분)

#### Q1. "Unfiltered"를 공간으로 어떻게 만들 거야?

논의 포인트:
- 9개 컨테이너(6m x 3m)를 어떤 컨셉으로 구성할 것인가
- "불완전함, 날것, 진정성"을 공간에서 어떻게 느끼게 할 것인가
- 경품 제공 불법 → 경험 자체가 보상이 되는 구조 필요
- 포토존, 사운드, 향, 촉감 등 오감 활용 방안
- SNS 바이럴이 자연스럽게 나오는 장치 설계

참고 사례:
- 인터랙티브 머그잔 (건배하면 그림이 이어지는 디자인)
- 게이미피케이션 (블라인드 테이스팅, 맥주 퀴즈)
- 캐시리스 텐트 (뮌헨 2025)
- 전통 의상 체험 → 인증샷 유도

#### Q2. 뮌헨 40%를 어떻게 쓸 거야?

논의 포인트:
- 인플루언서 3-4명 + 매거진 1-2명 = 총 5인 파견
- 1.7억 기준 뮌헨 예산 약 6,800만원 안에 항공+숙박+촬영+편집+매거진 타이업
- 캐스팅 전략 = 예산 전략
  - 옵션 A: 메가 1명 + 마이크로 3명 + 매거진 1명
  - 옵션 B: 매크로 2명 + 마이크로 2명 + 매거진 1명
- 콘텐츠 3단계: 사전(기대감) → 현장(체험) → 사후(회상)
- 글로벌 영상 로컬라이제이션 방향

#### Q3. 1.7억 vs 2.5억, 차이를 뭘로 만들 거야?

논의 포인트:
- 단순히 "더 많이"가 아니라, 2.5억에서만 가능한 한 방
- 후보:
  - 뮌헨 파견 규모 확대 (인플루언서 추가, 체류일 증가)
  - 서울 옥토버페스트 파울라너 전용 존 고도화 (시공 포함?)
  - 인스타 Always-on 광고 볼륨 확대
  - 서울 옥토버페스트 사전 버즈 캠페인 추가
- 견적서에서 항목별로 명확하게 차이가 보여야 함

### 3. 차별화 전략 — RFP에 없는 것 (10분)

다른 대행사와 똑같은 걸 내면 진다. 선제 제안 후보:

| 아이디어 | 근거 |
|---------|------|
| **라거(헬) 투트랙 전략** | RFP 브랜드 소개에 헬(Münchner Hell) 포함되어 있으나 SOW에는 빠짐. 국내 시장 80%가 라거 → 볼륨 확장 앵커로 제안 |
| **무알코올 트렌드 활용** | 뮌헨 2025에서도 무알코올 맥주 +6~10% 성장. 소버 큐리어스 트렌드와 연결 |
| **옥토버페스트 서울→뮌헨 연결 내러티브** | 서울에서 체험 → 뮌헨 영상으로 정통성 확인 → 일상에서 파울라너 구매 연결되는 Journey 설계 |
| **UGC 챌린지** | 경품 불가 규제 우회. 유저가 만든 콘텐츠가 파울라너 공식 채널에 올라가는 구조 |
