## 경험적 아이디어

### Idea 1. "프로스트 테이블" (Prost Table)

> 연결 장치 = **1리터 맥주잔(Mass)**

**컨셉**: 뮌헨 비어홀의 장탁자(Stammtisch) 문화를 서울에 이식. 10인 공용 나무 장탁자에 앉으면, 맥주가 동시에 서빙되고, 테이블 전체가 함께 "Prost!(건배)"를 외친다.

**경험 흐름**:
1. 입장 → 빈 자리에 앉는다 (모르는 사람 옆도 OK)
2. 테이블이 꽉 차면 "Prost Bell"이 울린다
3. 1리터 Mass 잔이 동시 서빙 (뮌헨 스타일 한 손 들기)
4. 테이블 전원이 일어서서 "Prost!" — 건배
5. 이 순간이 자동 촬영되어 모니터에 표시 → QR로 다운로드

**왜 파울라너인가**:
- 파울라너 페스트첼트가 실제로 이 방식 (8,450석, 장탁자, 동시 건배)
- "뮌헨에서 온 경험을 그대로" → 정통성
- 1리터 Mass 잔 자체가 SNS 콘텐츠 (한국에서 흔치 않음)

**인스타 연결**: #ShareYourProst + 건배 순간 릴스

---

### Idea 2. "짠 카운터" (Prost Counter)

> 연결 장치 = **건배 횟수 카운터 (물리적 장치)**

**컨셉**: 축제장 중앙에 거대한 기계식 카운터를 설치. 두 사람이 잔을 부딪히면(짠!) 센서가 감지하여 카운터가 올라간다. 축제 기간 중 서울 옥토버페스트의 총 "짠" 횟수를 실시간 집계.

**경험 흐름**:
1. 9개 컨테이너 부스 + 중앙 광장에 "짠 센서" 배치
2. 잔을 부딪히면 "딩!" 소리 + 카운터 +1
3. 실시간 총 건배 횟수가 중앙 대형 스크린에 표시
4. 목표 건배 수 달성 시 이벤트 발동 (ex. 100,000번째 짠 = 뮌헨 여행 추첨)
5. 개인별 "짠 횟수" 기록 카드도 발급 가능

**왜 이 아이디어인가**:
- "짠"은 한국어에서 가장 자연스러운 건배 표현
- 물리적 카운터 = 레트로 + 아날로그 감성 (Unfiltered 철학)
- 경쟁심 유발 → 자발적 참여 → 낯선 사람끼리 "같이 짠 해요!" 연결
- 카스 축제의 EDM/대형 무대와 완전히 다른 접근

**인스타 연결**: "오늘 서울 옥토버페스트 OO만 짠 달성!" 실시간 스토리

### Idea 3. "마이 슈탐티쉬" (My Stammtisch)

> 연결 장치 = **테이블 네임 플레이트 + 짧은 자기소개 카드**

**컨셉**: 뮌헨 비어홀에는 단골 테이블(Stammtisch)이 있다. 서울 옥토버페스트에서도 각 테이블에 이름을 부여하고, 앉는 사람들이 간단한 카드(이름, 한마디)를 테이블에 놓는다. 축제가 끝나면 그 테이블에 앉았던 모든 사람의 카드가 모여 "하루의 기록"이 된다.

**경험 흐름**:
1. 입장 시 빈 카드 수령 (이름 + "오늘 한마디" 기입)
2. 자리에 앉으면 카드를 테이블 홀더에 꽂음
3. 같은 테이블 사람들 카드를 보며 자연스러운 대화 유도
4. 떠날 때 카드는 테이블에 남김
5. 마감 후 그날의 카드들을 모아 사진 촬영 → SNS 공유

**왜 이 아이디어인가**:
- 게스트하우스 분위기(1차 회의 키워드)의 직접 구현
- 디지털이 아닌 아날로그 연결 → "Life Unfiltered"
- 비용 최소, 효과 극대 (인쇄물만 있으면 됨)
- 카드 자체가 수집형 굿즈/기념품이 될 수 있음

### Idea 4. "뮌헨 1분" (One Minute in Munich)

> 연결 장치 = **몰입형 공간 (컨테이너 1개를 뮌헨 텐트로)**

**컨셉**: 9개 컨테이너 중 1개를 "뮌헨 타임캡슐"로 만든다. 들어가면 60초간 뮌헨 옥토버페스트 페스트첼트 내부를 360도 영상+사운드로 체험. 나올 때 파울라너 한 잔이 기다린다.

**경험 흐름**:
1. 컨테이너 외부: 파울라너 26m 타워 미니어처 or 페스트첼트 파사드 재현
2. 내부 진입 (4-6명씩)
3. 360도 프로젝션 + 서라운드 사운드 = 뮌헨 텐트 안에 있는 듯한 경험
4. 클라이맥스: 전원이 함께 "Prost!" → 퇴장
5. 퇴장 시 파울라너 시음 + 포토카드

**왜 이 아이디어인가**:
- "뮌헨에 가지 않아도 뮌헨을 경험한다" → 옥토버페스트 = 파울라너 등식
- 정통성의 가장 강력한 전달 방식 (말이 아닌 체감)
- 대기줄 자체가 콘텐츠 (인스타 스토리)
- 3D 공간 전문가 활용처가 명확 (PT 비주얼의 핵심)

---

### Idea 5. "언필터드 포토부스" (Unfiltered Photo Booth)

> 연결 장치 = **즉석 필름 카메라 (폴라로이드)**

**컨셉**: "Life Unfiltered" 캠페인을 문자 그대로 구현. 디지털 카메라/스마트폰이 아닌 즉석 필름 카메라로 "필터 없는 순간"을 찍는다. 뮌헨 축제 소품(디른들 모자, 프레첼 소품 등)과 함께.

**경험 흐름**:
1. 포토부스 진입 (2-4명)
2. 바이에른 소품 선택 (레더호젠 멜빵, 프레첼 머리띠, Mass 잔 등)
3. 즉석 필름으로 촬영 → 물리적 사진 1장 즉시 수령
4. 같은 사진의 디지털 버전도 QR로 제공 (인스타 공유용)
5. 사진 하단에 "Unfiltered. Seoul Oktoberfest 2026" 각인

**왜 이 아이디어인가**:
- "Unfiltered" = 필터 없는 = 즉석 필름의 물성 그 자체
- 하이네켄 "Boring Phone" 캠페인(디지털 디톡스)과 톤 일치
- 물리적 사진 = 기념품 + 자발적 SNS 공유
- 낯선 사람끼리 "같이 찍어요!" → 연결 장치 기능

## 이미지 레퍼런스

### A. 뮌헨 파울라너 페스트첼트 (정통성 기준점)

**레퍼런스 포인트**:
- 26m 맥주잔 타워 — 옥토버페스트의 아이코닉 랜드마크
- 8,450석 장탁자 — 나무 벤치 + 긴 테이블, 수백 명이 동시에 건배하는 장면
- 바이에른 전통 장식 — 파란색+흰색(바이에른 깃발 색), 화환, 꽃장식
- 지하 맥주 파이프라인 — 기술 혁신의 정통성

**서울 적용 키워드**: 나무 질감, 바이에른 블루&화이트, Mass 잔, 동시 건배

### B. 컨테이너 부스 디자인

**Better Beer Container Bar (호주)**:
- 거대 데칼 래핑 + 라이트업 사인
- 전동 어닝, 커스텀 목재 카운터탑
- 이동식이면서 프리미엄 느낌

**BoxPop / Steel Space Concepts (북미)**:
- 멀티레벨 구조 (2층 옥상 라운지)
- LED 스크린, 터치스크린 인터랙티브
- 팝업이지만 영구 매장급 퀄리티

**서울 적용 키워드**: 인더스트리얼 × 바이에른, 컨테이너 골판 노출 + 나무, 야간 조명

### C. 브랜드 팝업/축제 경험

**하이네켄 Boring Phone / Flipper**:
- "스마트폰 내려놓고 진짜 소셜"
- 디지털 디톡스 = Unfiltered 정신
- 제품이 아닌 철학을 체험하게 만든 사례

**Stella Artois Port de Stella**:
- 프리미엄 맥주 팝업의 교과서
- 바 + 셀럽 + 요트 체험 → "The Life Artois"
- 프리미엄 경험 = 프리미엄 브랜드

**Jagermeister 뮌헨 공항 팝업**:
- 전통 악기 전시 + 현대적 프로모터
- "도착하는 순간부터 옥토버페스트"
- 정통성의 현대적 해석 사례

**서울 적용 키워드**: 철학 체험, 프리미엄 경험 설계, 정통+현대 믹스

### D. 축제 연결 장치 레퍼런스

**워터밤 — 물총**:
- 낯선 사람에게 물총 쏘기 = 허락 없는 상호작용
- 물 = 연결 장치, 축제의 정의 그 자체

**뮌헨 옥토버페스트 — Mass 잔 건배**:
- 1리터 잔을 높이 들어 부딪히는 행위 자체가 연결 장치
- 잔이 무거워서 양손으로 → 옆 사람과 물리적으로 가까워짐

**일본 축제 — 요요이 (YOYO) / 야타이(포장마차)**:
- 같은 포장마차 카운터에 앉으면 자연스럽게 대화
- 카운터 = 연결 장치

**적용 방향**: 파울라너의 연결 장치 = Mass 잔 + Prost(건배) 의식
