## 2. 리셉션 플로우 (접수대 핵심)

### 접수 시 확인 순서

```
참석자 도착
  │
  ├─ 1) 이름 확인 (명단 체크)
  │
  ├─ 2) 결제 상태 확인
  │     ├─ 결제 완료 → ✅ 체크
  │     └─ 미결제 → 포스(POS)로 안내하여 현장 결제
  │
  ├─ 3) 음료 선택 확인
  │     ├─ 샌드위치 + 와인 → 컬러 A 티켓
  │     └─ 샌드위치 + 음료 → 컬러 B 티켓
  │
  ├─ 4) 티켓 증정 (사람마다 반드시)
  │
  └─ 5) 명찰 배부
        └─ 와인/음료 구분 스티커 부착 (또는 컬러 티켓으로 대체)
```

### 와인 충전 안내 (탭샵바 담당)

| 구분 | 리셉션에서 할 것 | 탭샵바에서 할 것 |
|------|-----------------|-----------------|
| **결제 완료 + 와인** | 컬러 A 티켓 지급 → 포스로 안내 | 5,000원 와인 충전 |
| **미결제 + 와인** | 포스로 안내 → 참가비 결제 + 충전 | 참가비 결제 + 5,000원 와인 충전 |
| **결제 완료 + 음료** | 컬러 B 티켓 지급 | 그냥 주문 |
| **미결제 + 음료** | 포스로 안내 → 참가비 결제 | 참가비 결제 후 그냥 주문 |

> 핵심: 와인 선택자는 무조건 포스를 거쳐야 함 (충전 때문에)
> 음료 선택자 중 미결제자도 포스를 거쳐야 함

## 5. 리셉션 스태프 브리핑 요약 (1장)

접수대 담당자가 알아야 할 것:

1. **명단에서 이름 찾기** → 결제 여부 확인 (○/X 표시된 명단)
2. **티켓 나눠주기** — 전원에게 지급
3. **음료 선택 묻기** — "와인 드실 건가요, 음료 드실 건가요?"
4. **와인이면** → 컬러 A 티켓 + "포스에서 충전해주세요" 안내
5. **음료이면** → 컬러 B 티켓
6. **미결제자** → "포스에서 결제 먼저 해주세요" 안내
7. **명찰 배부**
