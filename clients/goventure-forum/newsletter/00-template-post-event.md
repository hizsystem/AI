# GoVenture Forum 뉴스레터 템플릿 — 행사 후 (Post-event)

> 발송 시점: D+3 (행사 3일 후)
> 목적: 감사 + 만족도 조사 + 사진 공유 + 액션 리마인드 + 다음 회차 티저
> 스티비 에디터에 섹션별 복붙

---

## 변수 목록

```
{{회차}}           — 이번 포럼 회차
{{다음회차}}        — 다음 포럼 회차
{{주제}}           — 이번 회차 주제
{{날짜}}           — 행사 날짜 (예: 3월 31일)
{{참석자수}}        — 실참석 인원
{{하이라이트}}      — 현장 한줄 하이라이트
{{연사명}}         — 키노트 연사
{{연사숙제}}        — 연사가 준 액션 아이템 (1~2개)
{{이달의브랜드}}     — 이달의 브랜드명
{{이달의브랜드_인스타}} — 이달의 브랜드 인스타 URL
{{만족도조사_링크}}   — 구글폼 URL

{{다음회차_주제}}    — 다음 회차 주제 (티저)
{{다음회차_날짜}}    — 다음 회차 날짜
```

## 섹션 1: 감사 인사

### 스티비 제목 (메일 subject)

```

```

### 본문

```
제{{회차}}회 고벤처포럼에 참석해 주셔서 감사합니다.

{{날짜}}, {{참석자수}}명이 함께한 자리에서
{{하이라이트}}

{{현장 분위기 묘사 2~3문장}}
```

> 현장 대표 사진 1장 삽입

## 톤앤매너 가이드

- **문체**: 따뜻한 감사 톤, 부드러운 CTA (강압적이지 않게)
- **이모지**: 최소 (사진 섹션에 카메라 정도)
- **구조**: 감사 → 피드백 요청 → 가치 제공(사진) → 리마인드 → 예고
- **섹션 제목**: 영문 대문자 유지
- **핵심**: "강의가 끝나도 아무도 자리를 뜨지 않는" 같은 생생한 현장감 한줄
