# GoVenture Forum 뉴스레터 템플릿 — 행사 전 (Pre-event)

> 발송 시점: D-21 (행사 3주 전)
> 목적: 지난 회차 랩업 + 이번 회차 안내 + 신청 유도
> 스티비 에디터에 섹션별 복붙

---

## 변수 목록

```
{{회차}}           — 이번 포럼 회차 (예: 190)
{{지난회차}}        — 직전 포럼 회차 (예: 189)
{{주제}}           — 이번 회차 주제 (예: 사업계획서도 브랜딩입니다)
{{한줄설명}}        — 주제 한줄 설명
{{날짜_풀}}        — 행사 날짜 (예: 2026년 4월 28일(화))
{{시간}}           — 행사 시간 (예: 17:00~19:30)
{{장소}}           — 행사 장소 (예: 합정 Tap Shop Bar)
{{참가비}}         — 참가비 (예: 10,000원)
{{연사명}}         — 키노트 연사 (예: 권윤정 대표)
{{연사소속}}        — 연사 소속 (예: HIZ)
{{연사주제}}        — 키노트 제목
{{이달의브랜드}}     — 이달의 브랜드명 (없으면 섹션 생략)
{{지난회차_참석자수}} — 직전 회차 참석 인원 (예: 86)
{{지난회차_하이라이트}} — 직전 회차 한줄 하이라이트
{{신청링크}}        — 온오프믹스 or 홈페이지 신청 URL
{{키비주얼_이미지}}   — 키비주얼 이미지 URL
```

## 섹션 1: 헤더 + 후킹

### 스티비 제목 (메일 subject)

```
[GoVenture Forum] {{주제}} — 제{{회차}}회 고벤처포럼
```

### 본문

```
{{후킹_질문}}

{{후킹_설명_2~3문장}}
```

> 패턴: 주제와 연결된 페인포인트 질문으로 시작.
> 예시(189회): "제품은 만들었는데 왜 안 팔릴까요?"
> 예시(190회): "사업계획서, 30페이지나 쓰는데 왜 떨어질까요?"

## 톤앤매너 가이드

- **문체**: 존댓말, 짧은 문장, 볼드로 핵심 강조
- **이모지**: 최소 사용 (CTA 화살표 → 정도만)
- **구조**: 질문 → 공감 → 정보 → 행동 유도
- **섹션 제목**: 영문 대문자 (LAST FORUM RECAP, NEXT FORUM, PROGRAM 등)
- **길이**: 스크롤 3~4회 분량 (모바일 기준)
