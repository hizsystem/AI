## 결제 플로우

```
입장 → 가나다순 리스트에서 이름 체크
     → 미결제자: 탭샵바 포스기로 1만원 결제
     → 컬러 쿠폰 배부 (결제자 vs 게스트, 2색)
     → 게스트 비용 → 부회장님 일괄 결제
     → 와인 선택자: 5천원 추가 충전
     → 커피 선택자: 커피 쿠폰 배부
     → 명찰 + 티켓 배부
     → 입장 후 바로 음료 안내
```
