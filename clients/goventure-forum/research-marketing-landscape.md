- 패스트캠퍼스에서 배운 것을 실제로 적용하려는 사람들이 모이는 곳
- "배움" 다음 단계인 "실행과 연결"을 제공하는 장소

## 6. 고벤처포럼 포지셔닝 맵

### 가격 x 규모 매트릭스

```
참가비 (높음)
  |
  |  DMS (58만)         MAX SUMMIT (18만)
  |
  |  인사이터 (15만)     DMTS (15~20만)
  |
  |  마케티움 (5.5만)    CMS (추정 10만대)
  |                      T.CON (추정 5~10만)
  |
  |  ★ 고벤처포럼 (1만)
  |  위그로스 (무료)
  |
  +--------------------------------------------→ 규모 (큼)
     20명    50명    200명   500명   1000명   2000명
```

### 고벤처포럼의 경쟁 우위

| 요소 | 고벤처포럼 | 대부분의 경쟁자 |
|------|-----------|---------------|
| **가격** | 1만원 (최저 수준) | 5.5만~58만원 |
| **형식** | 토론 + 네트워킹 | 강의 + (부분적) 네트워킹 |
| **규모** | 50명 (밀착형) | 200~2,000명 (대규모) |
| **장소** | 탭샵바 (바 분위기) | 호텔/컨벤션센터 (포멀) |
| **테마** | 스타트업 x 투자 | 범용 마케팅 |
| **참석자** | VC + 창업자 + 전문직 | 마케터 중심 |

### 핵심 차별화 포인트

1. **"단순 강의가 아닌 문제 제기 통한 토론"** - 시장에서 토론 중심 모임은 인사이터 정도. 고벤처포럼은 "투자/스타트업" 도메인에서 유일
2. **VC와 창업자가 같은 테이블에 앉는 포맷** - 다른 마케팅 모임에는 없는 구조
3. **1만원이라는 압도적 접근성** - 인사이터 15만원 대비 1/15. "돈 때문에 못 가는 모임"이 아님
4. **바(Bar) 공간** - 포멀한 호텔/컨벤션 대비 캐주얼한 분위기. 자연스러운 대화 유도
5. **네트워킹 밀도** - 50명이면 모든 참석자와 1:1 대화 가능. 1,000명 컨퍼런스에서는 불가능

### 고벤처포럼만의 킬러 콘텐츠 제안

1. **"이번 달, VC가 가장 많이 본 스타트업 유형은?"** - 투자 트렌드 실시간 공유
2. **"IR 피칭 라이브 피드백"** - 실제 VC 앞에서 피칭하고 즉석 피드백
3. **"0 -> 1 마케팅 전쟁터"** - 초기 스타트업의 첫 고객 확보 실전 경험
4. **"AI 시대, 스타트업이 살아남는 법"** - 가장 뜨거운 주제 + 스타트업 맥락
5. **"투자자가 마케팅을 평가하는 5가지 기준"** - VC 관점 인사이트
