# 30분 브랜드 무료 진단 — 설문지 & 결과지 포맷

> 참고 모델: 태니지먼트 (무료 약식 → 유료 풀버전)
> 목적: 참석자 브랜드 마케팅 역량 진단 + HIZ 컨설팅 전환 퍼널

---

## Part 1. 설문지 (자가진단)

### 콘셉트

> "우리 브랜드의 마케팅 체력, 몇 점인지 아시나요?"

15문항, O/X + 5점 척도, **3분 이내** 작성 완료 목표.
QR코드로 구글폼 접속 → 행사 전 또는 입장 시 작성.

---

### 채점 방식

| 유형 | 채점 |
|------|------|
| O/X | O = 10점, X = 0점 |
| 5점 척도 | 1점=0, 2점=2.5, 3점=5, 4점=7.5, 5점=10 |
| 주관식 | 채점 제외 (질적 데이터로 활용) |

- **총점**: 140점 만점 (14문항 x 10점, 15번 주관식 제외)
- **영역별 점수**: 각 30점 만점 (영역3, 4는 30점 / 영역1, 2, 5는 약간 다름)

> 간소화를 위해: O/X 문항 = 10점, 5점 척도 = 변환점수로 통일

## Part 2. 약식 결과지 (무료 — 현장 즉시 제공)

### 결과지 디자인 콘셉트

> 1장 분량, A4 PDF 또는 모바일 화면. 깔끔하고 시각적.

---

### 결과지 구성

#### 상단: 종합 점수 + 등급

```
┌─────────────────────────────────────────┐
│                                         │
│   🏢 [브랜드명]의 마케팅 체력 진단      │
│                                         │
│        종합 점수: 72 / 140              │
│        등급: 🟡 성장 준비기             │
│                                         │
└─────────────────────────────────────────┘
```

**등급 기준:**

| 점수 | 등급 | 설명 |
|------|------|------|
| 0~35 | 🔴 시작 단계 | 마케팅 기초 체력 만들기가 급선무입니다 |
| 36~70 | 🟠 기초 형성기 | 기본은 있지만, 전략적 방향 정립이 필요합니다 |
| 71~105 | 🟡 성장 준비기 | 실행력은 있지만, 체계적 최적화가 필요합니다 |
| 106~140 | 🟢 성장 가속기 | 탄탄한 기반 위에 스케일업할 준비가 되었습니다 |

#### 중간: 영역별 레이더 차트

```
        브랜드 정체성
             ★★★☆☆
            /      \
  성장전략 /        \ 타겟 고객
  ★★☆☆☆ /    72점   \ ★★★★☆
          \          /
  콘텐츠   \        / 마케팅 실행력
  ★★★☆☆    \      /  ★★☆☆☆
             ------
```

> 5개 영역의 강점/약점을 한눈에 볼 수 있는 오각형 차트
> 구글폼 → 구글시트 연동 → 차트 자동 생성 가능

## Part 3. 풀버전 결과지 (유료 — HIZ 컨설팅 전환)

### 풀버전에만 포함되는 내용 (약식 대비)

| 항목 | 약식 (무료) | 풀버전 (유료) |
|------|------------|-------------|
| 종합 점수 + 등급 | O | O |
| 영역별 점수 | O (숫자만) | O + 상세 해석 |
| 레이더 차트 | O | O + 업종 평균 비교 |
| 핵심 코멘트 | 1줄 | 영역별 3~5줄 |
| 경쟁사 비교 | X | O (동종 업계 벤치마크) |
| 개선 방향 | X | O (영역별 구체적 액션) |
| 6개월 로드맵 | X | O (월별 마케팅 액션플랜) |
| 우선순위 매트릭스 | X | O (긴급도 x 효과 매트릭스) |
| 전문가 코멘트 | X | O (권윤정 대표 직접 코멘트) |
| 분량 | 1페이지 | 5~8페이지 |

### 풀버전 가격 (제안)

| 옵션 | 가격 | 포함 내용 |
|------|------|----------|
| 풀버전 보고서만 | 10~15만원 | 상세 진단 보고서 (5~8페이지) |
| 보고서 + 1:1 컨설팅 30분 | 30만원 | 보고서 + 전문가 화상 미팅 |
| 보고서 + 3개월 마케팅 코칭 | 별도 견적 | HIZ 정규 컨설팅 패키지 |

## Part 4. 운영 플로우

### 행사 당일 흐름

```
[사전]
참가 신청 시 → QR코드로 설문 링크 발송
         → "행사 전 미리 작성해주시면 현장에서 바로 진단 결과를 받을 수 있습니다"

[입장 시] 15:30~
미작성자 → 태블릿 또는 QR로 현장 작성 (3분)

[무료 진단] 15:30~16:30
설문 결과 기반으로 전문가와 1:1 상담
약식 결과지 현장 제공 (출력 또는 모바일)

[키노트 이후]
키노트 내용과 연결하여 "풀버전 진단" 자연스럽게 안내

[네트워킹]
"풀버전 진단 궁금하신 분은 편하게 말씀해주세요"
현장 할인 제안 가능 (행사 참석자 특별가)
```

### 필요 준비물

- [ ] 구글폼 제작 (설문 15문항 + 기본정보 5항목)
- [ ] 구글시트 연동 (자동 점수 계산 수식)
- [ ] 약식 결과지 템플릿 (캔바 또는 피그마)
- [ ] 레이더 차트 자동 생성 방법 결정 (구글시트 차트 / 캔바)
- [ ] 풀버전 결과지 샘플 1부 제작 (시연용)
- [ ] QR코드 제작 + 인쇄
- [ ] 진단 담당자 배정 (권윤정 대표 + 보조 1명)

## 핵심 포인트

> **약식 결과지의 목적**: "아, 우리 브랜드 이 부분이 약하구나" 인식시키기
> **풀버전의 목적**: "그래서 어떻게 해야 하는데?" 궁금증 유발 → HIZ 컨설팅 전환

> 태니지먼트 공식: **무료로 문제를 인식시키고, 유료로 해결책을 제공한다**
