## Brand Spotlight 상세 설계

### 콘셉트

> "매회 하나의 브랜드에 스포트라이트를 비춥니다. 직접 경험하고, 진짜 인사이트를 남겨주세요."

매 회차마다 1개 브랜드를 선정하여 발표 + 제품 제공 + 참석자 피드백 수집

### 진행 흐름 (15분)

| 단계 | 시간 | 내용 |
|------|------|------|
| 1. 브랜드 소개 | 7분 | 브랜드 스토리, 제품/서비스 소개, 마케팅 고민 공유 |
| 2. Q&A | 3분 | 참석자 질문 2~3개 |
| 3. 제품 배포 | 5분 | 참석자 전원에게 제품/샘플 제공 + 설문 QR 안내 |

### 브랜드 입장에서의 가치

| 제공하는 것 | 받는 것 |
|------------|--------|
| 10분 발표 시간 | 50명 타겟 고객 앞에서 브랜드 노출 |
| 제품/샘플 50개 | 50개의 진짜 사용 후기 (설문 데이터) |
| | 마케팅 전문가+VC 앞에서의 피드백 |
| | GoVenture Forum 커뮤니티 노출 |

### 구글폼 설문 설계 (안)

> 제품 수령 시 QR코드로 설문 링크 제공. 네트워킹 시간 중 작성 유도.

**필수 항목 (5문항, 2분 이내)**

| # | 질문 | 유형 |
|---|------|------|
| 1 | 이 제품/서비스를 이전에 알고 있었나요? | 예/아니오 |
| 2 | 브랜드 스포트라이트 발표를 들은 후 인상은? | 5점 척도 |
| 3 | 이 제품/서비스를 직접 사용해볼 의향이 있나요? | 5점 척도 |
| 4 | 가장 인상 깊었던 포인트는? | 객관식 (스토리/디자인/가격/기능/기타) |
| 5 | 브랜드에 하고 싶은 한마디 (자유 피드백) | 단답형 |

**선택 항목 (추가 2문항)**

| # | 질문 | 유형 |
|---|------|------|
| 6 | 이 브랜드를 주변에 추천할 의향은? (NPS) | 0~10점 |
| 7 | 마케팅 관점에서 이 브랜드가 개선할 점은? | 서술형 |

### 브랜드 선정 기준 (제안)

- [ ] 초기 스타트업 (시드~시리즈A) 우선
- [ ] 참석자에게 실물 제공 가능한 제품/서비스
- [ ] 마케팅 고민이 있어 피드백을 진심으로 원하는 팀
- [ ] HIZ 컨설팅 잠재 고객이 될 수 있는 브랜드

### 수익 모델 가능성 (중장기)

| 단계 | 모델 | 설명 |
|------|------|------|
| 초기 (1~3회) | 무료 | 포맷 검증, 사례 축적 |
| 안정기 (4회~) | 유료 스폰서십 | 브랜드가 참가비(50~100만원) 부담, 제품 제공 |
| 확장기 | 프리미엄 패키지 | 발표 + 참석자 DB + 설문 리포트 + SNS 콘텐츠 |

## 토론 세션 운영 방식 (3안)

### 방식 1: 명제 토론 (추천)
- 진행자가 도발적 명제를 던지고 찬반 토론
- 예시 명제: "초기 스타트업에 브랜딩은 사치다" / "마케팅 예산 없이도 성장할 수 있다"
- 참석자 전원 1분 발언 기회

### 방식 2: 테이블 토론 (월드카페)
- 5~6명씩 테이블 배정
- 15분마다 주제 변경, 일부 인원 이동
- 테이블별 결론을 전체 공유

### 방식 3: 라이브 Q&A + 패널
- 키노트 후 실시간 질문 수집 (슬라이도/카톡)
- 패널이 즉석 답변 + 토론
