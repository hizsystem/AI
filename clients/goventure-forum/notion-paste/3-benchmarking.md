## 1. 최우선 벤치마킹 대상 (고벤처포럼과 가장 유사)

### 인사이터 (INSIGHTER) — 토론 중심 소규모 모임

> https://insight-er.com/

| 항목 | 내용 |
|------|------|
| 규모 | 30명 (소규모 밀착형) |
| 가격 | 15만원 (식사+와인+네트워킹 포함) |
| 형식 | 토론 + 네트워킹 + 식사 |
| 프로그램 | 마케터의 밤, 브랜드 마케터의 밤, 퍼포먼스 마케팅의 밤, 그로스해커의 밤 |
| 참석자 | 스타트업/글로벌기업/대기업/컨설팅 등 다양한 직급 |

**핵심 벤치마킹 포인트:**
- 스터디노트 작성 의무 + 벌금 제도로 참여율 유지
- 주제별로 "OO의 밤" 시리즈화 → 고벤처포럼도 "마케팅의 밤" 시리즈 가능
- 가격 15만원인데도 꾸준히 만석 → 콘텐츠 질이 가격을 정당화

**고벤처포럼과의 차이:**
- 가격 15배 (15만 vs 1만) → 우리는 접근성으로 승부
- 참석자: 직장인 마케터 중심 vs 우리는 창업자+VC 믹스

---

### 워스픽 (WARSPEAK) — 드링킹+네트워킹

> https://litt.ly/warspeak

| 항목 | 내용 |
|------|------|
| 운영자 | 디즈니/CJ ENM/GFFG 출신 15년차 마케터 |
| 형식 | 워스픽 살롱(토크), 워스픽샷(드링킹+네트워킹), 워스픽 멘토스(멘토링) |
| 특징 | 시니어 마케터 중심, 브랜드 "성장을 위한 무기" |

**핵심 벤치마킹 포인트:**
- **워스픽샷 = 드링킹+네트워킹** 모델이 탭샵바 콘셉트와 거의 동일
- 시니어가 주도하는 구조 → 고벤처포럼도 "시니어 VC/창업자" 중심 설계 가능
- 브랜딩이 세련됨: 네이밍, 비주얼, 톤앤매너 참고할 것

### 마케티움 소셜세미나 — 50명 규모 동일 포맷

> https://plus.hankyung.com/apps/newsinside.view?aid=202411277141c (한경 기사)

| 항목 | 내용 |
|------|------|
| 규모 | 총 50명 |
| 가격 | 55,000원 (VAT 포함) |
| 역사 | 2015년 시작, 2024년 21회차 |
| 형식 | 전문가 강연 + 참가자 디스커션 + 네트워킹 |
| 주제 예시 | "함께 정의하는 2425 대한민국 마케팅 트렌드" |

**핵심 벤치마킹 포인트:**
- 규모(50명), 형식(강연+토론+네트워킹)이 고벤처포럼과 **거의 동일**
- 9년간 21회 지속 운영 노하우 → 장기 운영 참고
- 매회 테마 변경으로 신선함 유지

**고벤처포럼과의 차이:**
- 가격 5.5배 (5.5만 vs 1만)
- 마케터 전용 vs 우리는 창업자+VC+마케터 크로스

## 5. 고벤처포럼 포지셔닝

> "VC와 창업자가 함께 토론하는 소규모 바(Bar) 포럼" = 한국에 없는 포맷

| | 고벤처포럼 | 경쟁자 평균 |
|---|-----------|-----------|
| 가격 | **1만원** | 5.5만~58만원 |
| 규모 | **50명** 밀착 토론 | 200~2,000명 강연 |
| 장소 | **바(Bar)** 캐주얼 | 호텔/컨벤션 포멀 |
| 참석자 | **VC+창업자** 동석 | 마케터 중심 |
| 형식 | **토론** 중심 | 강의 중심 |
