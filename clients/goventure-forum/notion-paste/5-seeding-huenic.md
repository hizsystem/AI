# 안건 5. 브랜드 씨딩 / 휴닉

## 브랜드 씨딩 (Brand Seeding) = 시딩 마케팅

인플루언서/체험단에게 제품을 무상 제공하여 SNS에서 자연스러운 후기를 유도하는 마케팅 전략

| 항목 | 내용 |
|------|------|
| 핵심 | 광고가 아닌 "진정성 있는 추천"으로 인지도/신뢰 확보 |
| 비용 효율 | 나노/마이크로 인플루언서 활용 시 높은 참여율(25~50%) |
| 적합 대상 | 초기 스타트업, 신제품 론칭 |
| 한국 주요 서비스 | 화해글로벌(브랜디드 시딩), 유커넥, 퍼핀, SimpleSeed |

## 휴닉 (HUENIC) 조사 결과

검색 결과 **(주)휴닉**은 **식물성 대체식품 푸드테크 스타트업**

| 항목 | 내용 |
|------|------|
| 대표 | 박진아 |
| 설립/소재 | 2020년 / 대전 |
| 업종 | 푸드테크 (식물성 대체식품) |
| 투자 | Pre-A 3억원 (JB벤처스) |
| 브랜드 | 와이즈미트, 베지어트, 빈커 |

**마케팅/브랜딩 컨설팅 회사는 아님**

## 추정

| 가능성 | 설명 |
|--------|------|
| 가장 유력 | 휴닉이 발표 사례 기업, 브랜드 씨딩은 마케팅 전략 키워드 |
| 다른 가능성 | 별개 키워드이거나, 다른 회사명의 오기 |
