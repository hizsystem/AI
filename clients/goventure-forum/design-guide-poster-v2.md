### 3-3. 카피 가이드 (확정안)

#### 메인 카피 (키노트 주제)

```
스타트업 브랜드가
올리브영 1위까지
```

> 이것이 포스터에서 가장 크게 보여야 할 문장.
> "올리브영 1위"는 스타트업 대표에게 가장 직관적인 성과 지표.
> 구체적 숫자/성과가 들어간 카피는 추상적 카피 대비 전환율 2~3배 높음.

#### 서브 카피 — 3안 (추천순)

**[안 1] 공감+결과 하이브리드 (최추천)**
```
"제품은 만들었는데, 왜 안 팔릴까?"
매출 정체기를 돌파한 브랜드 전략의 모든 것
```
> 왜 추천: 첫 줄에서 타겟의 **현재 고통**을 정확히 짚고, 둘째 줄에서 **해결의 힌트**를 줌.
> 스타트업 대표는 "나 얘기다"라고 느끼는 순간 클릭함.
> 매출 정체기 → 올리브영 1위라는 비포/애프터가 자연스럽게 연결됨.

**[안 2] 상황 나열형**
```
제품을 런칭했는데 판매가 어려운 분
정부지원사업은 합격했는데 성과가 막막한 분
— 30분이면 답이 보입니다
```
> 장점: 타겟을 구체적으로 특정. 본인 해당 여부를 즉시 판단 가능.
> 단점: 텍스트가 길어서 포스터보다는 바디글에 적합.

**[안 3] 성과/전략 설명형**
```
스타트업을 위한 브랜딩·마케팅 전략
& 올리브영 1위 달성 성공 사례 공유
```
> 이전 HIZ 세션 "스타트업/기업을 위한 브랜딩, 디자인 전략 및 성공 사례 공유"와 유사.
> 단점: 정보 전달은 되지만 감정적 후킹이 약함. "또 세미나네" 느낌.

#### 왜 안 1을 추천하는가

| 비교 기준 | 안 1 (공감+결과) | 안 2 (상황 나열) | 안 3 (설명형) |
|-----------|-----------------|-----------------|-------------|
| 후킹력 | ★★★★★ | ★★★★ | ★★★ |
| 포스터 적합성 | ★★★★★ (짧음) | ★★★ (길음) | ★★★★ |
| 타겟 공감도 | ★★★★★ | ★★★★★ | ★★★ |
| 차별화 | ★★★★★ | ★★★★ | ★★ (기존과 유사) |

> **결론**: 포스터에는 안 1, 바디글/상세페이지에는 안 2의 상황 나열을 함께 사용.

#### HIZ 크레딧 카피

```
GoVenture Forum x HIZ
— 브랜드 마케팅 최고의 전문가들이 리브랜딩하는데 여러분을 모십니다
```

또는 더 짧게:

```
Powered by HIZ
"대한민국 브랜드 마케팅의 최전선에서"
```

> HIZ를 자랑스럽게 드러내되, 포스터 하단에 자연스럽게 배치.
> "최고의 전문가들이 리브랜딩" → 고벤처포럼 자체의 리뉴얼도 HIZ가 이끈다는 신뢰감.

### 3-4. 권윤정 대표 프로필 영역

#### 사진 가이드

- **반신 또는 상반신** 프로필 사진 (전신 X)
- **배경**: 깔끔한 단색 또는 투명 → 포스터 위에 자연스럽게 합성
- **톤**: 전문적이면서 친근한 느낌 (정장보다는 비즈니스 캐주얼)
- **크기**: 포스터 면적의 25~35% 차지 (연사 중심 포스터이므로 존재감 있게)

#### 이력 표기 (제안 — 핵심만)

```
권윤정
HIZ 대표 | 브랜드 마케팅 전문가

• 올리브영 뷰티 카테고리 1위 브랜드 마케팅 총괄
• (주요 이력 1줄 추가)
• (주요 이력 1줄 추가)
```

> 이력은 **3줄 이내**로 압축. 포스터에서 읽히려면 짧아야 함.
> "올리브영 1위"가 메인 카피와 연결되므로 이력에서도 이를 강조.

### v2 바디글 보완 권장사항

**1. 키노트 주제 업데이트 (중요)**

현재 v2의 키노트 주제가 "마케팅에서 실패하는 진짜 이유"로 되어 있는데, 미팅에서 확정된 주제는 **"스타트업 브랜드가 올리브영 1위까지"**입니다.

변경 제안:
```
[키노트] "스타트업 브랜드가 올리브영 1위까지"
어떻게 이름 없는 스타트업 브랜드가 올리브영 뷰티 카테고리 1위에 올랐을까.
권윤정 대표가 직접 경험한 브랜드 마케팅의 전략과 실패, 그리고 돌파구.
```

현재: `키노트 | 권윤정 대표 (HIZ)`
보완:
```

HIZ 대표 · 브랜드 마케팅 전문가
전) 올리브영 뷰티 카테고리 1위 브랜드 마케팅 총괄
"스타트업 브랜드를 올리브영 선반 위에 올린 사람"
```

**3. HIZ 소개 섹션 추가 (신규)**

현재 v2에는 HIZ 소개가 거의 없음. 추가 제안:
