## 3. 월간 마스터시트 운영 방법

### 구글시트 구조 (매월 복제)

하나의 스프레드시트에 **탭(시트)을 월별로** 추가하는 방식:

```
[GoVenture Forum 마스터시트]
├── 탭: 대시보드        ← 전체 현황 한눈에 (자동 집계)
├── 탭: 190회 (4월)     ← 이번 달 신청자 DB + 체크리스트
├── 탭: 189회 (3월)     ← 지난 달 (아카이브)
├── 탭: 188회 (3월)     ← 지지난 달 (아카이브)
├── 탭: 참석자 통합 DB   ← 전체 누적 (중복 제거, 참석 횟수 카운트)
├── 탭: 연사/브랜드 히스토리  ← 회차별 연사, 이달의 브랜드 기록
└── 탭: 운영 템플릿      ← 빈 양식 (매월 복제)
```

### 월별 탭 컬럼 (운영 템플릿)

| 컬럼 | 내용 | 비고 |
|------|------|------|
| A. 이름 | 신청자 이름 | |
| B. 소속 | 회사명 | |
| C. 직함 | 직급/역할 | |
| D. 연락처 | 전화번호 | |
| E. 이메일 | | 뉴스레터 발송용 |
| F. 유형 | 창업자/투자자/전문직/기관/내부 | 명찰 색상 연동 |
| G. 신청 채널 | 온오프믹스/이벤터스/직접/VIP | |
| H. 신청일 | | |
| I. 결제 상태 | 완료/미결제/무료초청 | |

| O. 비고 | | |

### 대시보드 탭 (자동 집계)

```
=COUNTA(신청자) → 총 신청
=COUNTIF(결제상태,"완료") → 결제 완료
=COUNTIF(실참석,"O") → 실참석
=COUNTIF(유형,"창업자") → 유형별 분포
```

### 매월 운영 플로우

1. **D-28**: "운영 템플릿" 탭 복제 → "190회 (4월)"로 이름 변경
2. 신청 들어올 때마다 행 추가
3. 주간보고 시 대시보드 숫자 확인 → 부회장님 보고에 복붙
4. 행사 후 실참석 체크 → "참석자 통합 DB" 탭에 병합

```
├── 현장 물품 준비 (사이니지, 포스터 인쇄)

├── 리마인더 뉴스레터/메시지 발송

[D-2] 참석 확인 + 현장 세팅
├── 전원 참석 확인 연락 (문자/카톡)
├── 커피/와인 선택 취합
├── 미결제자 현장 결제 안내
├── 탭샵바 최종 인원/음식 공유
├── 스태프 역할 배치 + 브리핑
└── 장비 사전 체크 (마이크/프로젝터)

[D-day] 행사 당일
├── 리셉션 → 프로그램 → 네트워킹
├── 현장 촬영 (사진/영상)
└── 참석자 체크인 기록

[D+3] 후속 처리
├── 부회장님 결과 보고
├── 감사 메시지 발송 (만족도 조사 + 브랜드라이즈 CTA)
├── 행사 사진 정리 → 드라이브 업로드
├── 참석자 DB 통합 (마스터시트)
├── 뉴스레터 콘텐츠 작성 (다음 회차 발송용)
└── 다음 달 주제 논의 시작
```
