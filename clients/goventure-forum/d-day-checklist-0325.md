# 제188회 고벤처포럼 D-day 체크리스트

> 행사일: 2026-03-25(화) 15:30~19:10
> 장소: 팁스타운S6 1층 (서울 강남구 역삼로 168)
> 참석자: 106명 (온오프믹스 46 + 이벤터스 46 + 웹사이트 14)
> 작성: 2026-03-23 (D-2)

---

## D-2 (3/23 오늘)

### 1. 신청자 최종 참석 여부 확인 연락
- [ ] 106명 대상 참석 확인 연락 (문자 or 카톡)
  - 만원 현장 결제 안내 (미결제자 대상)
  - 동행 인원 체크 (추가 인원 파악 → 총 인원 확정)
  - 시간 재안내 (15:30 입장 시작, 17:00 오프닝)
- [ ] 미결제자 명단 별도 정리 (현장 결제 준비)
- [ ] 노쇼 예상 비율 감안 실참석 예상 인원 산출

### 2. 탭샵바 상황 공유
- [ ] 최종 참석 예상 인원 공유
- [ ] 장소 대관 진행 상황 확인 (시간, 세팅 가능 시간)
- [ ] 식사(샌드위치) 필요 인원 확정 → 케이터링 발주 (#5 연계)
- [ ] 음료 종류/수량 확인

### 3. 행사 현장 준비물
- [ ] 행사자료 PPT — 필요 여부 확정, 있으면 수합
- [ ] 포스터 출력 (A3 or A2, 입구+접수대용)
- [ ] POP/안내판 (타임테이블 안내판, 접수 안내 사이니지)
- [ ] 명찰 인쇄 (#11 분류 기준 확정 후)
- [ ] 포토/영상 실장님 스케줄 확인 (몇 시 도착, 장비, 납품 일정)

### 4. 행사 프로그램 최종 확인
- [ ] 타임테이블 최종 확정 (아래 기준)
  - 15:30~17:00 입장/체크인 + 무료 상담
  - 17:00~17:10 오프닝 (윤태현 대표)
  - 17:10~17:40 1부 마케팅 세션 키노트 (권윤정 대표)
  - 17:40~17:50 휴식
  - 17:50~18:00 브랜드 3분 스피치 + 제품 배포
  - 18:00~18:30 10초 자기소개 / 토픽 던지기 (윤대현 대표)
  - 18:30~18:40 단체 사진 (윤태현 대표)
  - 18:40~19:10 네트워킹 (샌드위치+음료)
  - 19:10 클로징
- [ ] 키노트 발표자(권윤정 대표) 발표 자료 수합 확인
- [ ] MC/진행자 역할 확정 (윤태현 대표? 별도?)

### 5. 케이터링/다과 확정
- [ ] 샌드위치 업체/메뉴 확정 (네트워킹 시간 18:40~)
- [ ] 수량 확정 (참석 예상 인원 + 여유분 10%)
- [ ] 음료 종류/수량 (휴식 시간 17:40 + 네트워킹 시간)
- [ ] 배달/세팅 시간 (17:30 이전 세팅 완료?)
- [ ] 일회용 컵/냅킨/쓰레기봉투 등 소모품

### 6. 1:1 마케팅 상담 (16:00~17:00) 준비
- [ ] 사전 예약 접수 현황 확인 (예약자 있는지?)
- [ ] 상담자 5명 배치 확정
  - 권윤정 대표 (이커머스)
  - 임애라 실장 (브랜딩)
  - 백승연 대표 (정부지원사업)
  - 류수현 팀장 (소셜마케팅)
  - 우성민 PM (퍼포먼스)
- [ ] 상담 공간/테이블 배치 (5개 테이블 필요)
- [ ] 사전 예약 안내 (참석 확인 연락 시 같이 안내)

### 7. 브랜드 3분 스피치 (17:50~18:00) 확정
- [ ] 발표 브랜드 확정 (어떤 브랜드?)
- [ ] 브랜드 측 발표자 + 발표 내용 확인
- [ ] 제품 배포 수량/방식 확정 (5분간 어떻게 배포?)
- [ ] 제품 현장 반입 시간/방법

## D-1 (3/24 내일)

### 8. 현장 스태프 역할 배치
- [ ] 스태프 명단 + 역할 배정
  - 접수/체크인: 2명 (명찰 배부 + 명단 체크)
  - 안내: 1명 (자리 안내, 동선)
  - 촬영: 1명 (포토/영상 실장님 or 별도)
  - 기술: 1명 (음향/프로젝터/Wi-Fi)
  - 케이터링: 1명 (다과 세팅/정리)
- [ ] 스태프 도착 시간 확정 (행사 시작 최소 1시간 전)
- [ ] 스태프 브리핑 자료 (역할별 할 일 1장 정리)

### 9. 음향/장비 사전 체크
- [ ] 팁스타운 기본 제공 장비 확인 (프로젝터, 스크린, 마이크)
- [ ] 마이크 개수 (핸드마이크 최소 2개 — 발표자+MC)
- [ ] 노트북 → 프로젝터 연결 테스트 (HDMI/USB-C 어댑터)
- [ ] 발표자료 백업 (USB에도 저장)
- [ ] Wi-Fi 접속 정보 확인 (참석자 안내용)
- [ ] 음악 재생 (입장/휴식/네트워킹 BGM)

### 10. 참석자 리마인더 발송
- [ ] 최종 리마인더 메시지 발송 (문자/카톡/이메일)
  - 일시: 3/25(화) 15:30 입장 / 17:00 오프닝
  - 장소: 팁스타운S6 1층 + 약도/교통편
  - 준비물: 명함 (네트워킹용)
  - 미결제자: 현장 결제 만원 안내
  - 1:1 상담 사전 예약 안내 (마감 임박)
- [ ] 발송 채널: 문자/카톡 (이메일은 안 볼 수 있음)

### 11. 명찰 분류 + 최종 인쇄
- [ ] 유형별 색상 기준 확정
  - 창업자/대표: 기본색
  - 투자자/VC: 별도색
  - 전문직(회계/법무): 별도색
  - 연사/VIP: 별도색
  - 스태프/내부: 별도색
- [ ] 106명 명단에 유형 태깅
- [ ] 명찰 디자인 확인 (이름 + 소속 + 직함)
- [ ] 여분 명찰 준비 (빈 명찰 10~15장, 현장 등록용)
- [ ] 명찰 케이스/목걸이줄 수량 확인
