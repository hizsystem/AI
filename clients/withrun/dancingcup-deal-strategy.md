# 댄싱컵(DANCING CUP) 수주 전략

> 생성일: 2026-03-28
> 클라이언트: (주)위드런 / 댄싱컵 브랜드
> 대표: 조기원 / 실무 접점: 변미혜 팀장
> 업종: 저가 커피 프랜차이즈 (130호점, 부산 기반 전국 확장)
> 월 예산: 200만원
> 기존 관계: 미례국밥 계약 체결 (4/1~4/30, 500만원)

## 1-Page 요약 (상담 전 5분 읽기용)

### 상황 한 줄

파워페이지에 매달 200만원을 태우면서 "효과 없는 걸 안다"고 인정한 프랜차이즈 본사가, 미례국밥 계약을 계기로 댄싱컵 마케팅도 맡기고 싶어 상담을 앞당겨 요청한 상태.

### 우리가 풀어야 할 문제

1. 200만원으로 파워페이지보다 눈에 보이는 성과를 만들어야 한다
2. 여름 신메뉴 출시 타이밍에 맞춰 바이럴을 일으켜야 한다
3. 130개 가맹점주에게 "본사가 마케팅하고 있다"는 인식을 심어야 한다

### 솔루션 1줄

**"파워페이지 200만원을 인스타 릴스 + 네이버플레이스 최적화 + 가맹점 공유 패키지로 재배분하면, 같은 돈으로 B2C 바이럴과 B2B(가맹점주) 신뢰를 동시에 잡는다."**

### 핵심 레버리지

- 미례국밥 4월 성과가 나오기 전에 상담이 앞당겨졌다 = 위드런 측 기대감이 이미 높다
- 변미혜 팀장과 2차 대면미팅 후 즉시 계약한 이력 = 의사결정 속도 빠름
- 위드런 통합 월 700만원(미례 500 + 댄싱컵 200) = 파트너 포지셔닝 근거

## Part 2. 3-Tier 패키징

### 설계 원칙
- 200만원 예산이므로 Essential을 실행 가능한 수준으로 설계
- Professional을 "이게 진짜 효과적인 구성"으로 보이게 해서 예산 상향 유도
- 위드런 멀티브랜드 통합 할인을 별도 옵션으로 제시

### Tier 구성

| 구분 | Essential (200만) | Professional (350만) | Premium (500만) |
|------|-------------------|---------------------|-----------------|
| **핵심 콘셉트** | 여름 신메뉴 집중 바이럴 | 본사 마케팅 체계 구축 | 가맹점 동반 성장 시스템 |
| **인스타그램** | 릴스 4건/월 (신메뉴 중심) | 릴스 8건 + 피드 4건/월 | 릴스 12건 + 피드 8건 + 스토리 운영 |
| **네이버플레이스** | 본사 직영점 1개 최적화 | 직영점 + 주요 거점 5개점 가이드 | 전 매장 NP 가이드라인 + 월간 모니터링 |
| **바이럴** | 로컬 인플루언서 3명 시딩 | 인플루언서 6명 + 맘카페/지역카페 | 인플루언서 10명 + 숏폼 크리에이터 협업 |
| **가맹점 공유** | 월간 마케팅 뉴스레터 (PDF) | 뉴스레터 + 매장용 콘텐츠 키트 | 뉴스레터 + 키트 + 분기 가맹점주 웨비나 |
| **리포팅** | 월 1회 성과 리포트 | 격주 리포트 + 주간 슬랙 공유 | 주간 리포트 + 실시간 대시보드 |
| **전략 회의** | 월 1회 | 격주 1회 | 주 1회 |

### 앵커링 전략

Professional(350만)이 가장 합리적으로 보이도록 설계했다. 핵심 포인트:

- Essential은 "최소한 이건 해야 한다"는 바닥선
- Professional은 "가맹점주에게 보여줄 수 있는 수준" = 리얼니즈 충족
- Premium은 "여기까지 가면 완벽"이라는 천장

### 위드런 멀티브랜드 통합 옵션 (별도 제안)

| 구분 | 단독 계약 | 통합 계약 (미례 + 댄싱컵) |
|------|----------|-------------------------|
| 미례국밥 | 500만/월 | 450만/월 (10% 할인) |
| 댄싱컵 | 200만/월 | 200만/월 (동일) |
| 합계 | 700만/월 | 650만/월 |
| **보너스** | - | 월간 위드런 브랜드 통합 리포트 + 크로스 프로모션 기획 1건 |

포인트: 댄싱컵 가격은 건드리지 않고, 미례국밥 할인으로 "묶으면 이득"을 보여준다. 실질적으로 우리 매출 50만원 감소 vs 장기 리텐션 + 싸와디식당 확장 가능성.

## Part 3. Challenger Teaching 포인트

상담에서 조기원 대표(또는 변미혜 팀장)에게 "가르쳐줄" 인사이트 3가지.

### Teaching 1: "파워페이지 200만원의 진짜 문제"

**데이터 포인트:**
- 네이버 파워페이지(블로그 상위노출)의 현재 문제: 네이버가 2025년부터 검색 로직을 대폭 개편하면서, 광고성 블로그 콘텐츠의 노출이 급격히 감소. "블로그를 광고판으로 쓰던 관성, 얕은 최적화에 의존하던 방식은 더 이상 유효하지 않다" (아이보스)
- 실제 사용자 행동 데이터(검색 후 클릭, 지도 보기, 전화 연결, 체류 시간, 재방문)가 네이버플레이스 순위를 결정하는 핵심으로 바뀜
- 결론: 파워페이지 200만원은 "네이버가 가치를 인정하지 않는 콘텐츠"에 돈을 태우고 있는 것

**대안 제시:**
같은 200만원을 아래처럼 재배분하면:
- 인스타 릴스 제작 + 시딩 (100만) → SNS 바이럴 (네이버가 감지 못하는 채널)
- NP 직영점 최적화 + 리뷰 관리 (50만) → 검색 순위 실질 상승
- 로컬 인플루언서 시딩 (50만) → UGC(사용자 생성 콘텐츠) 확보

### Teaching 2: "프랜차이즈 본사 마케팅의 이중 타겟 해법"

**인사이트:**
댄싱컵 본사 마케팅은 타겟이 2개다:
1. **B2C**: 커피 소비자 (매출)
2. **B2B**: 130개 가맹점주 (가맹 유지 + 신규 가맹 모집)

대부분의 프랜차이즈 본사가 이걸 분리하지 못하고 하나의 마케팅으로 둘 다 해결하려다 실패한다. 메가커피가 손흥민에 125억을 쓰는 이유는 소비자용이 아니라 "가맹점주에게 본사가 하고 있다"를 보여주기 위함이 절반.

**해법:**
200만원으로 둘 다 잡는 구조:
- 릴스/시딩 콘텐츠를 제작하면 → 소비자 바이럴 (B2C)
- 같은 콘텐츠를 "월간 마케팅 리포트"로 가맹점주에게 공유 → 본사 신뢰 (B2B)
- 추가 비용 없이 가맹점용 콘텐츠 키트(POP, 매장 내 포스터)로 변환 가능

핵심: **1개의 콘텐츠가 2개의 목적을 수행하는 구조**를 설계하는 것이 관건.

### Teaching 3: "여름 신메뉴, 저가 커피 시장에서의 채널 믹스"

**시장 데이터:**
- 저가 커피 프랜차이즈 시장 10만개 돌파 (2025). 메가커피 3,500개점, 컴포즈커피 3,500개점
- 메가커피 광고비 125억/년, 컴포즈 60억/년 → 댄싱컵이 이들과 광고비로 경쟁하는 건 불가능
- 그러나 댄싱컵의 강점: "부산 기반 로컬 브랜드" + "스페셜티 원두" + "재방문율 82%"

**채널 믹스 제안 (200만 기준):**

| 채널 | 예산 | 역할 | 기대 효과 |
|------|------|------|----------|
| 인스타 릴스 | 80만 | 신메뉴 비주얼 바이럴 | 도달 5만+, 저장 500+ |
| 로컬 인플루언서 | 50만 | 지역 맘카페/대학생 타겟 시딩 | UGC 10건+, 2차 확산 |
| 네이버플레이스 | 40만 | 직영점 검색 순위 최적화 | 플레이스 노출 30%+ 증가 |
| 가맹점 콘텐츠 키트 | 30만 | POP/포스터/SNS 소재 | 가맹점주 만족도 + 매장 내 노출 |

핵심 메시지: "메가커피와 같은 싸움을 하지 마세요. 130개점이 각각 로컬 바이럴의 거점이 되는 구조가 댄싱컵만의 무기입니다."

## Part 4. 예상 반론 & 대응

### 반론 1: "200만원밖에 없는데 뭘 할 수 있어요?"

**대응:**
"파워페이지에 200만원을 3년째 태우면서 효과가 없다고 느끼신 거잖아요. 금액이 아니라 쓰는 방식의 문제입니다. 같은 200만원으로 릴스 4건 + 인플루언서 3명 시딩 + NP 최적화를 하면, 파워페이지와 비교할 수 없는 실질적 도달이 나옵니다. 미례국밥에서도 첫 달에 이 구조로 시작하고 있고요."

### 반론 2: "일단 여름 한 번만 해보자"

**대응:**
"여름 신메뉴 캠페인은 당연히 첫 번째 목표입니다. 다만 한 가지만 말씀드리면, 인스타그램이든 네이버플레이스든 '콘텐츠 자산'이 쌓이는 구조예요. 7월에 만든 콘텐츠가 9월에도 검색되고, 12월 겨울 신메뉴 때 또 활용됩니다. 1개월만 하면 소비되고 끝이지만, 3개월 하면 자산이 됩니다. 그래서 최소 3개월을 권장드리고, 1개월 후 성과 보고 판단하시는 구조를 제안합니다."

### 반론 3: "미례국밥 먼저 해보고 결과 보고 시작할게요"

**대응:**
"그것도 방법입니다. 다만 두 가지를 말씀드리면: 첫째, 여름 신메뉴 타이밍은 6~7월이라 4월 미례국밥 결과를 보고 5월에 시작하면 세팅 기간 포함 타이밍이 빠듯합니다. 둘째, 미례국밥과 댄싱컵은 업종 특성이 완전히 달라서, 국밥집 마케팅 성과가 커피 프랜차이즈에 1:1 적용되지는 않아요. 오히려 동시에 진행하면 위드런 전체 마케팅을 통합 관리하면서 크로스 프로모션도 가능합니다."

### 반론 4: "우리가 직접 해볼까 하는데..."

**대응:**
"인하우스로 하시는 게 장기적으로 맞는 방향이에요. 다만 현재 변미혜 팀장님이 미례국밥 + 댄싱컵 + 싸와디식당 3개 브랜드를 혼자 보시는 구조에서, 130개점 가맹점 관리까지 하면서 릴스 촬영 + 인플루언서 섭외 + NP 관리를 직접 하시기는 현실적으로 어렵습니다. 저희가 초기 3~6개월 구조를 잡아드리고, 이후 인하우스 전환하는 로드맵이 가장 효율적입니다."

## Part 5. 클로징 전략

### 이상적 다음 단계

1. **이번 상담에서**: 200만원 Essential 기본 합의 + Professional 검토 요청
2. **상담 후 당일**: 댄싱컵 맞춤 제안서 + 여름 신메뉴 캠페인 초안 발송
3. **3일 내**: 변미혜 팀장과 실무 미팅 → 세부 스콥 확정
4. **4월 둘째 주**: 계약 → 5월 세팅 → 6월 여름 신메뉴 런칭 캠페인

### 긴급성 요소

- 여름 신메뉴 런칭이 6~7월이라면, 5월에 세팅(NP 최적화, 콘텐츠 촬영, 인플루언서 섭외)이 완료되어야 함
- 지금 시작하면 4월 계약 → 5월 세팅 → 6월 런칭으로 딱 맞음
- 4월 중순 넘기면 세팅 기간 부족으로 7월 런칭, 여름 피크 절반 놓침

### 리스크 제거 장치

- **1개월 트라이**: 미례국밥과 동일하게 1개월 단위 계약. 성과 안 나오면 중단 가능
- **파워페이지 대비 비교 리포트**: 1개월 후 파워페이지 3년 누적 데이터 vs 우리 1개월 성과 비교 리포트 제공
- **무료 선물**: 상담 마무리에 "댄싱컵 인스타그램 진단 리포트 + 직영점 NP 진단" 무료 제공 (실소요 1시간, 신뢰 구축 효과 큼)

### 위드런 멀티브랜드 통합 포지셔닝

상담 마지막에 자연스럽게 꺼낼 한마디:

> "미례국밥, 댄싱컵, 싸와디식당 — 위드런이 가진 3개 브랜드를 하나의 마케팅 파트너가 통합 관리하면, 각 브랜드 개별로 하는 것보다 효율이 훨씬 좋습니다. 크로스 프로모션(댄싱컵 영수증으로 미례국밥 할인), 통합 콘텐츠 제작, 가맹점 마케팅 시스템까지 — 오늘은 댄싱컵만 이야기하지만, 위드런 전체의 마케팅 파트너가 되는 게 저희 목표입니다."

| 여름 집중 (6~8월) | 250만/월 x 3개월 | 인플루언서 +3명, 릴스 +2건 (시즌 부스트) |

| 항목 | 금액 |
|------|------|

## 부록: 시장 벤치마크 데이터

### 저가 커피 프랜차이즈 광고비 규모

| 브랜드 | 연 광고비 | 매장수 | 점당 월 부담 |
|--------|----------|--------|-------------|
| 메가커피 | 125억 (2023) | 3,500+ | 약 30만/점 |
| 컴포즈커피 | 60억 (2023) | 3,500+ | 약 14만/점 |
| 댄싱컵 | 약 2,400만 (추정) | 130 | 약 1.5만/점 |

댄싱컵의 본사 마케팅비는 메가커피의 1/50 수준. 이걸 단점으로 볼 게 아니라, "대형 브랜드와 같은 방식으로 경쟁하면 안 된다"는 전략의 근거로 활용.

### 프랜차이즈 본사 마케팅 이중 구조 벤치마크

- 메가커피: 손흥민 모델비 포함 125억 중 60억을 가맹점주와 50% 분담 → 가맹점주 반발
- 컴포즈커피: BTS 뷔 모델, 60억 중 20억 가맹점주 분담 → 마찬가지로 논란
- 교훈: 본사 마케팅이 가맹점주에게 "부담"이 되면 역효과. 댄싱컵은 200만원이 전액 본사 부담이므로, "가맹점주에게 비용 전가 없이 본사가 직접 투자한다"는 메시지가 가능

### 파워페이지 vs 로컬 바이럴 효과 비교 (추정)

| 지표 | 파워페이지 200만/월 | 로컬 바이럴 200만/월 |
|------|-------------------|---------------------|
| 콘텐츠 생산량 | 블로그 포스트 4~8건 | 릴스 4건 + UGC 10건+ |
| 콘텐츠 수명 | 2~4주 (검색 순위 변동) | 3~6개월 (SNS 알고리즘 재노출) |
| 2차 확산 | 거의 없음 | 공유/저장/리그램 |
| NP 영향 | 간접적 (리뷰 아님) | 직접적 (방문 → 리뷰 → 순위) |
| 가맹점 공유 가능성 | 낮음 (블로그 링크) | 높음 (릴스 리그램, 포스터 활용) |
| 측정 가능성 | 노출수 정도 | 도달/참여/저장/프로필 방문/DM |

### 상담 킬러 질문 5개

1. "파워페이지 3년 하시면서 댄싱컵 검색량이나 플레이스 순위에 변화가 있었나요?" — 의도: 효과 없음을 스스로 확인하게
2. "130개 점주님들이 본사 마케팅에 대해 어떤 피드백을 주시나요?" — 의도: B2B 니즈 깊이 파악
3. "여름 신메뉴가 기존 메뉴 대비 마진이나 객단가에서 어떤 차이가 있나요?" — 의도: 캠페인 ROI 설계 근거
4. "댄싱컵 인스타그램이 현재 어떻게 운영되고 있나요?" — 의도: 현재 콘텐츠 역량 파악
