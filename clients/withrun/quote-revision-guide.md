# 위드런-미례국밥 견적서 보완 가이드

> 작성일: 2026-03-12
> 목적: 기존 3개월 견적서(월 500만원 x 3)에 피드백 반영하여 수주력 강화

---

## 1. 파트너십 개요 (상단) — 월별 차별화

### 현재 (3개월 동일)
```
* 네이버플레이스·인스타그램·블로그 3채널에 집중하여 "노출 → 유입 → 재방문" 마케팅 구조를 3개월간 구축
* 무가 씨딩 인플루언서 30명(릴스 30개)과 매장 붐업 콘텐츠로 인스타그램 노출을 선점하고, 네이버플레이스 최적화와 블로그 리뷰 축적으로 검색 전환까지 연결
* 메타 광고비 월 50만원 포함, 릴스 도달·전환 광고를 병행하여 오가닉 콘텐츠의 도달을 증폭
* 1개월 채널 셋업 및 전략 수립 → 2개월 콘텐츠 확산 및 리뷰 축적 → 3개월 검색 상위 안착 및 퍼포먼스 최적화
```

### 수정안 — 월별 개요 차별화

**Month 1. Set-up 상단:**
```
[미션] "돼지우동" 히어로 메뉴 바이럴 기반 구축

* 돼지우동을 히어로 메뉴로 세우고, 인스타그램·네이버플레이스·블로그 3채널의 기반을 셋업합니다
* 무가 씨딩 1차(10명)로 "돼지우동 릴스" 10개를 확보하고, 메타 광고로 도달을 증폭합니다
* 네이버플레이스 검색 최적화와 블로그 5편으로 "미례국밥" "돼지우동" 키워드 선점을 시작합니다
* 현재 자체 진행 중인 체험단·리뷰이벤트를 전략적으로 구조화하여 효율을 높입니다

[이달의 목표] 돼지우동 릴스 10개 확보 | 네이버플레이스 기본 세팅 완료 | 3채널 전략서 납품
```

**Month 2. Scale-up 상단:**
```
[미션] 콘텐츠 확산 & 검색 순위 상승

* 씨딩 2차(10명)와 매장 붐업 콘텐츠로 인스타그램 노출을 확대하고, 릴스 누적 20개를 달성합니다
* 블로그 체험단 + 리뷰 20편 축적으로 네이버 검색 순위를 끌어올립니다
* 1차 성과 데이터를 분석하여, 어떤 채널·어떤 콘텐츠가 가장 효과적인지 핏을 확정합니다
* 성과 좋은 릴스/리뷰 콘텐츠는 가맹 홍보 소재로 재활용할 수 있도록 정리합니다

[이달의 목표] 릴스 누적 20개 | 블로그 리뷰 20편+ | 네이버플레이스 상위 노출 진입 | 콘텐츠 핏 확정
```

**Month 3. Optimize 상단:**
```
[미션] 검색 상위 안착 & 지속 성장 구조 완성

* 씨딩 3차(10명)로 릴스 누적 30개를 달성하고, 성과 검증된 인플루언서와 재협업합니다
* 블로그 리뷰 40편+ 달성으로 "돼지우동" "부산국밥" 등 핵심 키워드 검색 상위를 안착시킵니다
* 3개월 풀 성과 리포트와 자체 운영 가이드를 납품하여, 4개월차 이후에도 성과가 유지되도록 합니다
* 3개월간 축적된 콘텐츠 자산(릴스 30개, 리뷰 40편, 성과 데이터)을 가맹 마케팅에 활용하는 방안을 제안합니다

[이달의 목표] 릴스 누적 30개 | 블로그 리뷰 40편+ | 3개월 성과 리포트 | 자체 운영 가이드 + 가맹 활용 제안
```

## 3. 차별화 프레이밍 (견적서 전체에 반영할 톤)

### 현재 문제
견적서가 "우리가 해드릴 것 리스트"로만 되어 있음. 위드런은 이미 자체적으로 꽤 많이 하고 있기 때문에, **"이미 하고 있는 것 + 우리가 더하는 것"** 구조가 필요.

### 적용 방법

파트너십 개요 하단에 한 줄 추가:

```
* 현재 위드런 자체 진행 중인 체험단·리뷰이벤트·인플루언서 활동을 전략적으로 구조화하고,
  성과 측정 → 분석 → 개선 사이클을 만들어 효율을 극대화합니다
```

또는 각 월별 항목에서:
- "블로그 체험단" → "블로그 체험단 (자체 진행분 QC 포함, 신규 모집 병행)"
- "무가 씨딩" → "무가 씨딩 (자체 서칭 인플루언서 + 브랜드라이즈 추천 인플루언서 통합 관리)"

## 5. 수정 시 주의사항

1. **가격은 변경하지 않음** — 항목 세부 내용만 보강. 같은 돈에 더 많이 해주는 것처럼 보이게
2. **"돼지우동"을 3회 이상 노출** — 대표님이 직접 말한 키워드이므로 견적서에서 반복해야 "내 이야기를 듣고 만들었구나" 느낌
3. **가맹 관련은 "포함"으로** — 별도 비용 청구하면 스코프 논쟁 발생. "자연스럽게 포함되어 있다"는 느낌이 중요
4. **자체 운영 이관 표현 수정** — "이관"보다 "지속 성장 구조 완성"이 더 적절. 연장 가능성을 열어두는 표현
