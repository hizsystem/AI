# 미례국밥 네이버 파워링크 세팅 가이드

> 작성: 2026-04-13
> 목적: Green 세팅용 + 변팀장님 관리용
> 예상 소요: 세팅 30~40분, 검수 10분

---

## 파워링크가 뭔가요?

네이버에서 "센텀 미례국밥"을 검색하면 **맨 위에 뜨는 텍스트 광고**입니다.

```
[파워링크] ← 이 표시가 붙은 영역
미례국밥 센텀점 - 부산 센텀 돼지국밥 전문점
얼큰돼지우동·돼지국밥·한마리국밥. 센텀역 도보 6분. 주차 가능.
```

**플레이스 광고와 다른 점:**
| 구분 | 파워링크 | 플레이스 광고 |
|------|---------|-------------|
| 노출 위치 | 검색결과 최상단 텍스트 | 지도 영역 내 |
| 과금 | 클릭당 과금 (CPC) | 클릭당 과금 (CPC) |
| 랜딩 | 홈페이지/블로그/플레이스 링크 자유 선택 | 플레이스 페이지 고정 |
| 최소 예산 | 일 제한 없음 (클릭당 70원~) | 일 5,000원~ |
| 관리 | searchad.naver.com | 스마트플레이스 앱 |

→ 둘 다 하는 게 이상적. 플레이스 광고(일 5,000원)는 킥오프에서 이미 합의됨.

## Step 1. 계정 만들기

### 1-1. 네이버 검색광고 가입

1. **searchad.naver.com** 접속
2. **[광고 시작하기]** 클릭
3. 로그인: 변팀장님 네이버 계정 or 미례국밥 사업용 계정
4. **사업자 정보 입력:**
   - 사업자등록번호
   - 상호명: 미례국밥
   - 대표자명
   - 업종: 음식점
5. 가입 완료 → 광고 관리 시스템(광고관리 홈) 진입

> ⚠️ 사업자번호가 필요하므로 변팀장님께 확인 필요.
> 이미 네이버 검색광고 계정이 있을 수도 있음 — 먼저 물어볼 것.

### 1-2. 결제 수단 등록

1. 광고관리 홈 → **[충전]** 메뉴
2. 신용카드 or 계좌이체 등록
3. 선불 충전: **5만원** 먼저 충전 (테스트 기간)

---

## Step 2. 캠페인 구조 만들기

### 구조 설계

```
📁 캠페인: 미례국밥_파워링크
├── 📂 광고그룹: 센텀점
│   ├── 키워드 6~8개
│   └── 소재(광고문구) 2개
└── 📂 광고그룹: 전포점
    ├── 키워드 5~6개
    └── 소재(광고문구) 2개
```

### 2-1. 캠페인 만들기

1. 광고관리 홈 → **[캠페인 만들기]**
2. 캠페인 유형: **파워링크**
3. 캠페인명: `미례국밥_파워링크`
4. 일 예산: **10,000원** (센텀+전포 합산, 시작은 보수적으로)
5. 노출 기간: 제한 없음 (상시)

### 2-2. 광고그룹 — 센텀점

1. 광고그룹명: `센텀점`
2. 하루 예산: 설정 안 해도 됨 (캠페인 예산으로 관리)
3. 입찰가: **150원** (시작가, 이후 조정)
4. 콘텐츠 네트워크: **OFF** (검색에서만 노출, 배너 X)
5. PC/모바일: **모바일 우선** (국밥 검색의 80%+ 모바일)

### 2-3. 광고그룹 — 전포점

1. 위와 동일하게 생성
2. 광고그룹명: `전포점`
3. 입찰가: **120원** (전포는 검색량이 더 높아 단가가 낮을 수 있음)

### 키워드 등록 방법

1. 광고그룹 클릭 → **[키워드]** 탭 → **[키워드 추가]**
2. 위 키워드를 한 줄에 하나씩 입력
3. 매치 타입: **구문 일치** 선택 (정확한 검색에만 노출, 예산 절약)
4. 입찰가: 그룹 기본 입찰가 그대로 (개별 조정은 나중에)

> 💡 **매치 타입 설명:**
> - 정확 일치: "센텀 미례국밥" 정확히 쳐야 노출
> - 구문 일치: "센텀 미례국밥 주차" 등 포함 검색에도 노출 ← **추천**
> - 확장 일치: 유사 검색어까지 자동 노출 (예산 빠르게 소진, 비추)

---

### 파워링크란?

네이버에서 **"센텀 미례국밥"** 같은 키워드를 검색하면 **맨 위에 뜨는 광고**입니다.
고객이 광고를 **클릭할 때만** 비용이 발생하고, 클릭당 약 100~200원 정도입니다.

현재 **일 10,000원 (월 약 30만원)** 으로 세팅되어 있습니다.

### 팀장님이 하실 것 (주 1회, 5분)

**1. 광고 관리 페이지 접속**
- PC: searchad.naver.com → 로그인
- 앱: "네이버 검색광고" 앱 설치

**2. 확인할 것**
- 잔액이 남아있는지 (부족하면 충전)
- 주간 클릭 수 확인 (대시보드에 바로 보임)

**3. 조절할 수 있는 것**
| 상황 | 방법 |
|------|------|
| 예산 아까움, 줄이고 싶음 | 캠페인 → 일 예산을 5,000원으로 변경 |
| 효과 좋아서 늘리고 싶음 | 캠페인 → 일 예산을 20,000원으로 변경 |
| 일시 중단하고 싶음 | 캠페인 → [OFF] 버튼 |
| 다시 켜고 싶음 | 캠페인 → [ON] 버튼 |

**이 외의 키워드 변경, 문구 수정, 입찰가 조정은 저희가 합니다.**

### 월간 리포트

매월 말 성과 리포트에 파워링크 데이터도 포함해서 보내드립니다:
- 총 클릭 수 / 총 비용
- 키워드별 클릭 순위
- 클릭 → 플레이스 전환 추이

## Green 내부 메모

### 세팅 전 변팀장님께 확인할 것

1. **네이버 검색광고 계정 있는지?** → 있으면 계정 정보 받기, 없으면 사업자번호 받아서 개설
2. **플레이스 광고(일 5,000원)와 별도로 파워링크도 진행하는 거 맞는지?** → 예산 겹침 확인
3. **연결할 URL** → 센텀점/전포점 플레이스 링크 각각 확인

### 비용 구조

| 항목 | 월 예상 비용 |
|------|------------|
| 파워링크 | ~30만원 (일 10,000원) |
| 플레이스 광고 | ~15만원 (일 5,000원) |
| **합계** | **~45만원/월** |

500만원 월 예산에서 광고비 45만원은 9% 수준. 적절한 비율.

### 성과 확인 타이밍

- 세팅 후 **3일간**은 데이터 수집 기간 → 건드리지 않기
- **1주 뒤** 첫 성과 체크 (CTR, CPC, 노출량)
- **2주 뒤** 키워드 정리 (클릭 0인 키워드 제거 or 입찰가 조정)
