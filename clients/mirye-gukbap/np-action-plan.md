## 3. N1/N2 올리기 — 유입 채널 + 키워드 몰이

### N1 (유사도/적합성) 올리기 — 키워드 몰이 작업

**원리**: 대표키워드 ↔ 리뷰 키워드 ↔ 블로그 키워드 ↔ 소개문 키워드가 일치할수록 N1 상승

| 액션 | 밀어야 할 키워드 | 담당 | 비고 |
|------|---------------|------|------|
| **영수증 리뷰 가이드** | "센텀 맛집", "재송동 맛집" | BR 제작 → 변팀장 테이블 비치 | 방문목적 4가지 + 키워드 포함 유도 |
| **블로그 체험단 가이드** | "센텀 맛집" (띄어쓰기!) | BR 키워드 가이드 → 체험단 전달 | 제목에 필수 포함 |
| **리뷰 답변 템플릿** | 답변에 "센텀 맛집" 키워드 자연 녹임 | BR 제작 → 변팀장 사용 | 10종 돌려쓰기 |
| **소개문 키워드 추가** | "센텀 맛집", "재송동 맛집" 문맥 포함 | BR | 센텀점 소개문 업데이트 |
| **메뉴명 최적화** | "해장 돼지국밥", "점심특선" 등 목적 키워드 | BR 제안 → 변팀장 적용 | AI가 읽는 영역 |

### N2 (관련성) 올리기 — 유입 채널 다각화

**원리**: 리뷰수, 예약수, 저장수, 외부 유입, 체류시간, 클릭수가 높을수록 N2 상승

| 유입 채널 | 액션 | 비용 | N2 영향 |
|----------|------|------|---------|
| **인스타그램** | 인스타 광고 → 플레이스 링크 랜딩 | 메타 광고비 내 | 외부유입 = N2 직접 상승 (네이버 선호) |
| **블로그 체험단** | 리뷰노트 기반 월 6~10건 | 무료 (식사+포인트) | 블로그 리뷰 수 증가 |
| **당근마켓** | 비즈프로필 + 동네소식 → 플레이스 링크 | 무료 | 지역 외부유입 |
| **네이버플레이스 광고** | 일 **5,000원~** 시작 | 월 ~15만원 | 광고 클릭 자체가 체류시간/클릭수 올림 |
| **영상 소식** | 조리 과정 5~10초 영상 업로드 | 무료 | 플레이스 메인 영상 노출 → 체류시간↑ |
| **네이버 예약** | 예약 기능 활성화 | 무료 | 예약 가능 매장 별도 노출 + 예약수 |
