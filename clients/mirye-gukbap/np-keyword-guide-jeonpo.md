## 2. 영수증 리뷰 가이드 (N1↑)

리뷰 유도 엽서/안내물에 아래 내용을 반영.

### 리뷰 작성 가이드 (고객에게 보여줄 것)

```
리뷰 작성 시 이렇게 써주시면 감사합니다!

1. 방문 목적을 적어주세요
   → "전포 점심으로", "해장하러", "회사 동료들이랑"

2. 드신 메뉴와 느낀 점을 솔직하게
   → "돼지국밥이 진하고 깔끔했어요"

3. 사진 1장 이상 첨부

4. 별점 5점 부탁드립니다!
```

### 핵심: 리뷰에 자연스럽게 포함돼야 할 키워드

| 우선순위 | 키워드 | 유도 방법 |
|---------|--------|----------|
| 1 | **전포 맛집** | 엽서에 "전포 맛집 미례국밥에 방문해주셔서 감사합니다" 문구 |
| 2 | **전포 점심 / 점심** | "어떤 목적으로 방문하셨나요?" 질문으로 유도 |
| 3 | **돼지국밥 / 국밥** | 메뉴명 자체가 포함되므로 자연스럽게 |

## 4. 블로그 체험단 키워드 (N1↑ + N2↑)

체험단 블로거에게 전달할 가이드.

```
[미례국밥 전포본점 체험단 키워드 가이드]

필수 포함 (제목 또는 본문):
- "전포 맛집" (띄어쓰기 주의!)
- "미례국밥"

선택 포함 (1개 이상):
- "전포 돼지국밥"
- "전포 점심"
- "전포동 맛집"

해시태그:
#미례국밥 #전포맛집 #전포점심 #전포돼지국밥 #부산국밥

주의:
- "전포맛집"(붙여쓰기) 아닌 "전포 맛집"(띄어쓰기)으로 써주세요
- 같은 블로거에게 같은 키워드 재의뢰 불가
```
