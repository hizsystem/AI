## 4월 실행 플랜

### Phase 1 (4/7~14): 5~7명 확정

**예산 80만원 내 최적 조합:**

| 타입 | 인원 | 총 비용 | 후보 |
|------|------|---------|------|
| 마이크로 인스타 (10K+) | 2명 | ~50만 | @busan_zzzzzin_ + @broculri13 |
| 나노 인스타 (3~10K) | 3명 | ~30만 | @busan.local.food + @busan_date_ + 1명 추가 서칭 |
| 식사 협찬 | 5명분 | ~15만 | |
| **합계** | **5명** | **~95만원** | 견적 80만 + 식사비 15만 |

> @busan.local(58K)은 단독으로 30~50만이라 5명 조합에 넣으면 예산 초과. 2차 씨딩 or 별도 협의.

### Phase 2 (4/17~30): 릴스 수거 + QC

- 방문 일정 조율 (피크 피해 오후 2~4시)
- 릴스/피드 게시 후 QC
- 2차 활용 동의 확인 (메타 광고 소재)

## 바이럴 가이드 핵심 (인플루언서 공유용)

**필수 촬영 포인트:**
1. 돼지우동 면 뽑기 숏 (비주얼 킥)
2. 국물 클로즈업 (김 모락모락)
3. 매장 외관/내부 (깔끔한 국밥집)
4. 셀프바/밑반찬 (가성비)

**필수 해시태그:**
`#미례국밥` `#돼지우동` `#센텀맛집` `#부산국밥` `#부산맛집`

**필수 키워드 (캡션/본문):**
미례국밥, 돼지우동, 센텀 맛집 중 2개 이상

**법적 필수:**
모든 콘텐츠에 "협찬" 또는 "광고" 표시
