## Step 1: 인테이크 데이터 심층 분석

### 데이터 한계

인테이크 폼 미제출 상태. 대표의 직접 발화가 없으므로 **플레이스 진단 데이터와 상권 특성에서 니즈를 역추론**한다. 상담 시 검증 질문으로 가설을 확인해야 한다.

### 플레이스 진단에서 추출한 핵심 시그널

| 구분 | 시그널 | 해석 |
|------|--------|------|
| 리뷰 1,117개 | 이미 고객이 오는 매장 | 유입 자체는 문제가 아님. **전환 최적화와 재방문**이 과제 |
| 메뉴 14개 전부 사진/설명 완비 | 콘텐츠 기본기 있음 | 대표 또는 담당자가 한 번은 신경 쓴 흔적 |
| 영업시간 미등록 | 디지털 관리에 손이 안 감 | 매장 운영에 바빠서 온라인은 방치 상태 |
| 업주 인증 미완 | 플레이스를 직접 관리하지 않음 | 누군가(대행사 또는 직원)가 초기 등록 후 방치했을 가능성 |
| 예약/쿠폰/소식 전무 | 네이버 솔루션 활용도 0% | 존재 자체를 모르거나, 할 줄 모르거나, 할 시간이 없음 |
| 세트메뉴(술상세트 34,000원) | 객단가 올리려는 메뉴 구조 있음 | 사업적 감각은 있는 대표 |
| 센텀 오피스 상권 | 점심 직장인 수요 상권 | 평일 점심 회전율이 매출의 핵심 |

### 추정 프로파일: "바쁜 국밥집 사장님"

- 매장 운영(주방/홀 관리)에 하루 12시간 이상 투입
- 온라인 마케팅에 쓸 시간 절대 부족
- 네이버 플레이스가 중요하다는 건 알지만, 뭘 어떻게 해야 하는지 모름
- 리뷰 1,117개가 자연스럽게 쌓인 것 = 맛과 운영으로 승부하는 스타일
- "마케팅 해야 하는데..."라는 만성적 고민이 있으나 행동으로 이어지지 못함

## Step 2: SPIN 기반 니즈 레이어링

### S -- Situation (객관적 현재 상태)

| 항목 | 현황 |
|------|------|
| 업종 | F&B, 국밥 전문점 (돼지국밥, 순대국밥, 수육류) |
| 위치 | 부산 해운대구 센텀동로 90 (센텀필 상가 1층) |
| 상권 | 센텀 오피스 밀집지, 센텀역 도보 6분 |
| 리뷰 | 1,117개 (양적 자산 충분) |
| 메뉴 | 14개, 전 메뉴 사진/설명 완비 |
| 가격대 | 국밥 10,000원, 세트 14,000~34,000원 |
| 편의시설 | 주차, 단체 이용, 대기공간 등록 완료 |
| 플레이스 관리 | 업주 인증 미완, 영업시간 누락, 예약/쿠폰/소식 없음 |
| 추정 매출 구조 | 평일 점심(직장인) 70% + 저녁/주말(일반) 30% (검증 필요) |

### P -- Problem (진단에서 발견된 문제)

1. **플레이스 기본 세팅 미비**: 영업시간 누락, 업주 인증 안 됨, 예약/쿠폰 기능 비활성
   - 네이버 D-플레이스 리포트에 따르면, 솔루션 1개 이상 사용 시 미사용 대비 매출 3배 증가
   - 음식점 예약 기능 사용 시 매출 6배 증대 효과 (네이버 공식 데이터)

2. **키워드 전략 부재**: 1,117개 리뷰가 있으나 키워드 포함 리뷰 유도 없음
   - "센텀 국밥", "해운대 돼지국밥" 등 핵심 검색어에 대한 전략적 접근 없음
   - 대표 키워드 세팅 상태 미확인

3. **콘텐츠 운영 공백**: 소식/이벤트 발행 이력 없음
   - 플레이스 알고리즘에서 '활성도'는 핵심 랭킹 팩터
   - 2025-2026 알고리즘 기준, 실질적 상호작용(전화, 길찾기, 저장, 예약) 데이터가 순위 결정의 핵심

4. **상세 소개문 빈약**: 길안내 위주로 작성, 매장 컨셉/메뉴 특징/고객 공감포인트 부족
   - 1,500자 이상, 키워드 밀도 높은 소개문이 필요

### I -- Implication (방치 시 시나리오)

| 현재 문제 | 3개월 후 결과 | 6개월 후 결과 |
|----------|-------------|-------------|
| 영업시간 누락 | 정보 불완전 매장으로 분류, 플레이스 노출 순위 하락 | 경쟁 국밥집에 순위 역전당함 |
| 예약/쿠폰 없음 | 네이버 예약 가능 매장 별도 노출 혜택에서 제외 | 신규 고객 유입 경로 상실, 경쟁 매장으로 유출 |
| 소식/이벤트 없음 | 플레이스 '활성도' 점수 지속 하락 | 검색 결과 2-3페이지로 밀림 |
| 키워드 전략 없음 | "센텀 국밥" 검색 시 상위 노출 기회 상실 | 리뷰 수가 적은 경쟁점이라도 최적화하면 역전 가능 |
| 업주 인증 없음 | 리뷰 답글, 소식 발행 등 핵심 관리 기능 사용 불가 | 고객과의 소통 단절, 부정 리뷰 대응 불가 |

**최악의 시나리오**: 센텀 상권에 신규 국밥집이 오픈하면서 플레이스 최적화를 제대로 세팅 -> 리뷰 200개만으로도 미례국밥(1,117개)보다 상위 노출 -> "리뷰는 많은데 왜 안 보이지?"라는 상황 발생

**핵심 위험**: 리뷰 1,117개라는 자산이 **최적화 없이는 경쟁 우위로 전환되지 않는다**. 양적 자산을 질적 경쟁력으로 바꾸는 작업이 시급하다.

### N -- Need-Payoff (해결 시 구체적 가치)

| 해결 영역 | 기대 가치 | 근거 |
|----------|----------|------|
| 업주 인증 + 기본 세팅 완료 | 플레이스 정보 완성도 90%+ 달성 | 정보 완성도가 알고리즘 기본 조건 |
| 예약/쿠폰 활성화 | 월 신규 유입 +50% 이상 가능 | 네이버 데이터: 예약 사용 시 음식점 매출 6배 |
| 키워드 전략 세팅 | "센텀 국밥" 상위 3위 내 노출 | 리뷰 1,117개 기반으로 키워드 최적화하면 충분히 가능 |
| 소식/이벤트 정기 발행 | 플레이스 활성도 점수 상승, 검색 노출 유지 | 월 2-4회 소식 발행이 업계 기준 |
| 리뷰 키워드 유도 | N1(유사도) 지수 상승, 검색 매칭 강화 | 키워드 포함 리뷰가 알고리즘 핵심 변수 |
| 종합 | 1개월 내 검색 노출 순위 상승, 3개월 내 신규 고객 유입 체감 증가 | 기본 세팅만으로도 빠른 효과 |

```

| "매장 운영에 바빠서 온라인 마케팅에 손을 못 대고 있다"  ★★★★★
| -> 국밥집 사장님의 하루: 새벽 육수 -> 점심 피크 -> 저녁 마감

|

| (1) 디지털 마케팅 역량 부재 ★★★★★

|
| (3) 객단가 및 저녁/주말 매출 과제 ★★★

|
L2 전술적 문제 ────────────────────────────────────────────
| (1) 플레이스 기본 세팅 미비 ★★★★★
|    영업시간, 업주 인증, 예약, 쿠폰 -- 다 안 되어 있음
|    "해야 하는 건 아는데 못 하고 있다"
|
| (2) 리뷰 관리 체계 없음 ★★★★
|    1,117개 리뷰에 답글 관리 상태 미확인
|    키워드 포함 리뷰 유도 가이드 없음
|
| (3) 검색 노출 최적화 안 됨 ★★★
|    대표 키워드, 상세 소개문, 메뉴 설명 내 키워드 삽입 미흡
|
L1 표면 요청 ──────────────────────────────────────────────
  "네이버 플레이스 좀 잘 관리해주세요"                    ★★★★★
  "손님이 더 많이 왔으면 좋겠어요"
```

### 레벨별 해석

**L1 표면 요청** (확신도 ★★★★)
- 인테이크 폼이 없어 직접 발화는 없지만, 마케팅 상담을 받으려 한다는 것 자체가 "뭔가 해야 한다"는 인식의 표현
- 소상공인 F&B에서 가장 흔한 표면 요청: "플레이스 관리", "손님 늘리기"

**L2 전술적 문제** (확신도 ★★★★★)
- 플레이스 진단 데이터가 명확하게 보여주는 영역
- 영업시간 누락, 업주 인증 미완 = 이건 누가 봐도 즉시 고쳐야 할 문제
- 해결 난이도가 낮은 반면 효과는 큼 (Quick Win 영역)

**L3 전략적 문제** (확신도 ★★★★)
- 추정 영역. 상담에서 검증 필요
- 센텀 오피스 상권의 구조적 한계(점심 편중)와 성장 전략은 대표와 대화해봐야 정확히 파악 가능

**L4 근본 원인** (확신도 ★★★★★)
- 소상공인 F&B의 보편적 근본 원인
- 이것이 **우리 서비스의 존재 이유와 정확히 맞닿는 지점**
- "사장님이 직접 하지 않아도 되는 구조"를 만들어주는 것이 핵심 가치

## Step 4: JTBD (Jobs to Be Done) 분석

### Functional Job (기능적 과업)

> "네이버에서 '센텀 국밥' 검색하면 우리 가게가 맨 위에 나오게 하고 싶다. 그래서 점심시간에 빈 자리 없이 꽉 차게 만들고 싶다."

- 플레이스 검색 상위 노출 달성
- 신규 고객(특히 센텀 직장인) 유입 증가
- 예약/쿠폰으로 방문 전환율 높이기
- 점심 외 시간대(저녁, 주말) 매출 확대
- 세트메뉴/술상세트 등 객단가 높은 주문 유도

### Social Job (사회적 과업)

> "주변 가게들도 다 플레이스 하던데, 우리만 뒤처지는 것 같아서 불안하다. 검색하면 우리 가게가 제대로 나왔으면 좋겠다."

- 센텀 상권 내에서 "잘 되는 국밥집"으로 인식되고 싶음
- 고객이 검색했을 때 정보가 허술하면 신뢰가 떨어진다는 인식
- 네이버 예약, 쿠폰 등을 쓰는 주변 가게를 보면서 "우리도 해야 하는데..."
- 단골 손님이 "여기 플레이스에서 봤는데 좋더라"고 주변에 추천해주는 경험

### Emotional Job (감정적 과업)

> "맛에는 자신 있는데, 온라인 마케팅은 뭘 어떻게 해야 할지 모르겠다. 누가 대신 해줬으면 좋겠다."

- 핵심 감정: **답답함 + 불안 + 위임 욕구**
- 매일 새벽부터 육수 끓이고, 점심 피크 치르고, 마감까지 -- 마케팅까지 신경 쓸 에너지가 없음
- "맛은 자신 있다" vs "온라인은 자신 없다"의 역량 불균형에서 오는 스트레스
- 돈을 내고라도 이 불안을 해소해줄 전문가를 찾고 있음
- 원하는 것: "사장님은 매장에만 집중하세요. 온라인은 저희가 합니다"라는 한마디

## Step 5: 유사 사례 패턴 매칭

### 패턴 A: "리뷰 많은데 노출 안 되는 음식점"

2025-2026 네이버 플레이스 알고리즘 핵심 변화:
- 리뷰 수보다 **실질적 상호작용**(전화, 길찾기, 저장, 예약)이 순위 결정에 중요
- 키워드 포함 리뷰의 비중이 단순 리뷰 수보다 N1(유사도) 지수에 영향
- 플레이스 솔루션(예약/쿠폰/소식) 사용 매장에 별도 노출 혜택

**적용점**: 미례국밥은 리뷰 1,117개라는 양적 자산이 있지만, 예약/쿠폰/소식 없이 키워드 전략도 없으면 리뷰 300개짜리 최적화된 경쟁점에 밀릴 수 있다. "리뷰는 많은데 왜 안 보이지?" 시나리오를 예방해야 한다.

### 패턴 B: "부산 F&B 프랜차이즈/본점의 플레이스 성공 사례"

부산 해산물집 사례: 제철 메뉴 업데이트를 즉시 반영 -> 예약률 20% 증가
업계 데이터: 부정 리뷰 진정성 있게 대응 + 재방문 특별 서비스 -> 6개월 만에 별점 3.2→4.7, 월 방문객 38% 증가

**적용점**: 미례국밥도 계절 메뉴, 이벤트를 소식으로 발행하고, 리뷰 답글 관리를 체계화하면 단기간에 가시적 효과를 낼 수 있다.

### 패턴 C: "센텀 오피스 상권의 점심 경쟁"

센텀시티 2,000개+ 입주 기업, 구내식당 1곳당 일 300-350명 이용. 직장인은 가격(5,000-7,000원 구내식당 대비), 접근성, 속도를 중시한다. 국밥 10,000원은 구내식당 대비 프리미엄 가격이므로 "그 돈 낼 가치가 있다"를 플레이스에서 충분히 전달해야 한다.

**적용점**:
- "센텀 점심" 키워드 공략이 핵심 (직장인 점심 검색 수요)
- "가성비" 보다 "제대로 된 한 끼" 포지셔닝이 유효
- 점심 시간 예약 기능은 "대기 없이 빠르게"를 원하는 직장인에게 강력한 셀링 포인트

### 패턴 D: "소상공인이 마케팅 대행을 맡기는 실제 이유"

업계 조사에 따르면 소상공인이 대행사를 찾는 3대 이유:
1. **시간 부족**: 블로거 모집, 리뷰 관리, 플레이스 세팅 등 혼자 하기 번거로움
2. **전문성 부족**: 어디서부터 시작해야 할지 모름
3. **타겟 고객 접근**: 잠재 고객이 검색했을 때 상위 노출되고 싶음

**적용점**: 미례국밥 사장님도 이 3가지가 복합적으로 작용할 가능성이 높다. 상담에서 "시간이 없어서"가 1순위 이유인지, "뭘 해야 할지 몰라서"가 1순위인지 파악해야 한다. 전자라면 대행 범위를, 후자라면 교육+대행 하이브리드를 제안해야 한다.

## Step 6: 검증 질문 도출

### 질문 1: 현재 매출 구조 파악

> "현재 매출에서 점심 vs 저녁/주말 비중이 어떻게 되시나요? 그리고 한 달 매출이 대략 어느 정도 수준인지 여쭤봐도 될까요?"

**검증할 가설**: 센텀 오피스 상권이므로 점심 매출 70% 이상으로 편중되어 있을 것. 만약 그렇다면 저녁/주말 매출 확대가 숨은 니즈이고, 플레이스 소식/이벤트로 "센텀 회식", "센텀 술자리" 키워드 공략이 유효하다.

### 질문 2: 플레이스 관리 현황 확인

> "네이버 플레이스(스마트플레이스)에 직접 들어가서 관리해보신 적 있으세요? 업주 인증은 혹시 시도해보셨나요?"

**검증할 가설**: 대표가 플레이스 관리 자체를 시도한 적이 없는지, 아니면 시도했다가 포기한 것인지 확인. 전자라면 "몰라서 못 한 것"이고, 후자라면 "어려워서 포기한 것". 이에 따라 서비스 범위(풀 대행 vs 세팅+교육)가 달라진다.

### 질문 3: 경쟁 인식 확인

> "센텀 쪽에서 경쟁하시는 국밥집이나, 요즘 손님이 빠지고 있다고 느끼시는 부분이 있으세요?"

**검증할 가설**: 마케팅 상담을 받으려는 트리거가 "성장 욕구"인지 "위기감"인지 확인. 주변에 새로 오픈한 국밥집이 있거나, 최근 매출이 떨어졌다면 긴급도가 높다. 단순히 "좀 더 잘해보자"라면 긴급도는 낮지만 투자 의지는 있는 상태.

### 질문 4: 마케팅 예산과 기대치

> "마케팅에 월 얼마 정도까지 투자할 수 있으신지, 그리고 마케팅을 하면 가장 기대하시는 결과가 뭔지 여쭤봐도 될까요?"

**검증할 가설**: 소상공인 F&B 마케팅 예산은 월 30-100만원이 일반적. 국밥집 사장님의 예산 한도와 기대 ROI를 파악해야 서비스 패키지를 설계할 수 있다. "월 30만원으로 매출 2배" 같은 비현실적 기대가 있다면 기대치 조정이 선행 과제.

### 질문 5: 과거 마케팅 경험

> "이전에 마케팅 대행이나 블로그 체험단 같은 걸 해보신 적 있으세요? 있으시다면 어떠셨나요?"

**검증할 가설**: 과거 대행 경험이 있다면 (1) 어떤 서비스였는지 (2) 왜 중단했는지 (3) 효과는 있었는지 확인. 이전 대행사에 대한 불만이 있다면 그것이 우리 서비스의 차별화 포인트가 된다. 경험이 없다면 "첫 경험"이므로 신뢰 구축이 더 중요하다.

## 핵심 갭 분석: "플레이스 관리 요청" vs "실제 필요한 것"

### 대표의 추정 요청

```
"네이버 플레이스 좀 잘 되게 해주세요"
```

### 실제 상황이 말하는 것

```
"플레이스 기본 세팅부터 안 되어 있고,
1,117개 리뷰라는 자산을 경쟁 우위로 전환할 전략이 없다"
```

### 갭 분석 상세

| 항목 | 대표의 추정 인식 | 실제 상황 | 갭 |
|------|---------------|----------|-----|
| **문제 인식** | "플레이스를 잘 관리하면 되겠지" | 관리 이전에 기본 세팅(업주 인증, 영업시간, 예약, 쿠폰)이 안 됨 | ★★★★ |
| **리뷰 자산** | "리뷰 많으니까 괜찮겠지" | 키워드 전략 없는 리뷰 1,117개는 알고리즘에서 제 힘을 못 씀 | ★★★★★ |
| **시간** | "시간 내서 좀 해봐야지" | 매장 운영만으로 하루가 끝남. 지속적 관리가 물리적으로 불가능 | ★★★★★ |
| **범위** | "플레이스만 하면 되지 않나?" | 플레이스 + 블로그 체험단 + 키워드 광고 + 리뷰 관리가 통합 운영되어야 효과 | ★★★★ |
| **경쟁** | "우리 맛 좋으니까" | 맛만으로는 검색 노출에서 이길 수 없음. 최적화된 경쟁점에 밀릴 수 있음 | ★★★★ |

### 진짜 필요한 것 (우선순위 순)

```
1단계 [즉시 실행] 플레이스 기본 세팅 완성 (1주)
   +-- 업주 인증 완료
   +-- 영업시간 등록
   +-- 예약 기능 활성화
   +-- 알림받기 쿠폰 세팅
   +-- 상세 소개문 재작성 (1,500자+, 키워드 포함)
   --> 비용 최소, 효과 즉각적. 이것만으로도 정보 완성도 90%+

2단계 [2주 내] 키워드 전략 + 리뷰 최적화
   +-- "센텀 국밥", "센텀 점심", "해운대 돼지국밥" 키워드 분석
   +-- 대표 키워드 3-5개 세팅
   +-- 리뷰 키워드 유도 가이드(엽서/카드) 제작
   +-- 기존 리뷰 답글 정비 (키워드+방문목적+편의시설+꿀팁 4요소)
   --> 리뷰 1,117개 자산을 알고리즘 경쟁력으로 전환

3단계 [1개월] 콘텐츠 운영 + 광고 시작
   +-- 소식/이벤트 정기 발행 (월 2-4회)
   +-- 5-10초 영상(국밥 김 올라오는 장면) 업로드
   +-- 플레이스 광고 시작 (일 5,000원~)
   +-- 블로그 체험단 1차 (센텀 국밥, 센텀 점심 키워드)
   --> 검색 노출 순위 본격 상승 시작

4단계 [2-3개월] 확장 + 매출 다변화
   +-- 저녁/주말 매출 확대 전략 ("센텀 회식", "술상세트" 키워드)
   +-- 인스타그램 플레이스 연동
   +-- 당근마켓 비즈프로필 (센텀/재송동 타겟)
   +-- 월간 성과 리포트 + 키워드 순위 모니터링
   --> 단일 채널(플레이스)에서 멀티 채널로 확장
```

### 상담 시 커뮤니케이션 전략

**핵심 메시지**: "사장님 매장은 맛과 리뷰가 이미 검증되어 있습니다. 문제는 네이버가 그걸 못 보고 있다는 겁니다."

**절대 하지 말 것**:
- "플레이스 기본도 안 되어 있네요" 식의 비하
- 전문 용어 남발 (N1, CTR, 알고리즘 등)
- "이것저것 다 해야 합니다" 식의 압도

**대신 할 것**:
> "사장님, 리뷰 1,117개면 센텀 국밥집 중에서 상당히 많은 편이에요. 그런데 지금 네이버에서 '센텀 국밥' 검색하면 사장님 가게가 몇 번째에 나오는지 아세요? (보여주기) 리뷰가 이렇게 많은데 이 정도 순위인 이유가 있어요. 영업시간이 안 들어가 있고, 예약이 안 되어 있고, 쿠폰도 없거든요. 이거 세팅만 잡아도 바로 달라집니다."

이 접근의 효과:
- 리뷰 1,117개를 인정하여 사장님의 자존감을 세워줌
- 현재 순위를 직접 보여주어 "뭔가 해야 한다"는 동기 부여
- 해결책이 쉽다는 것을 보여주어 진입 장벽 낮춤
- "세팅만 잡아도"라는 표현으로 Quick Win 제시

## 종합 요약: 상담 전 핵심 인사이트

### 한 줄 진단

> **미례국밥 센텀점은 "맛과 리뷰로 이미 검증된 매장이지만, 디지털 마케팅 세팅의 공백으로 온라인 잠재력을 전혀 활용하지 못하고 있는" 전형적인 소상공인 F&B 케이스다.**

### 대표의 진짜 니즈 (JTBD 종합)

| 층위 | 니즈 |
|------|------|
| 표면 | 네이버 플레이스 잘 나오게, 손님 더 오게 |
| 실제 | 플레이스 기본 세팅 + 키워드 전략 + 리뷰 최적화 + 지속 관리 |
| 심층 | 매장 운영에만 집중하고 싶은데 온라인도 신경 써야 한다는 부담에서 벗어나고 싶음 |
| 감정 | "맛은 자신 있는데 온라인은 모르겠다"는 답답함 해소, 전문가에게 맡기고 안심하고 싶음 |

### 서비스 설계 방향

이 클라이언트에게는 **코치형 모델보다 풀 대행 모델**이 적합하다.

이유:
- 매장 운영 시간이 절대적으로 길어 학습/실행 시간 확보가 어려움
- 디지털 리터러시가 낮을 가능성 (업주 인증조차 안 된 상태)
- 소상공인은 "결과"를 원하지 "과정"을 배우려 하지 않음
- 월 고정 비용으로 "알아서 해주는" 구조가 가장 매력적

**예상 적합 패키지**:
- 네이버 플레이스 초기 세팅 (1회성) + 월 관리 (리텐션)
- 초기 세팅: 업주 인증, 정보 완성, 키워드 세팅, 소개문 재작성
- 월 관리: 소식 발행 (월 2-4회), 리뷰 답글 관리, 키워드 모니터링, 월간 리포트

### 상담 어젠다 제안

1. **아이스브레이크** (5분): "리뷰 1,117개 대단하십니다" + 센텀 상권 이야기
2. **현재 상태 공유** (10분): 플레이스 진단 결과를 화면으로 보여주기, 검색 순위 직접 확인
3. **Quick Win 제시** (10분): "이것만 세팅해도 이렇게 달라집니다" -- 영업시간, 예약, 쿠폰
4. **로드맵 제시** (10분): 1단계~4단계, 각 단계별 예상 효과
5. **서비스/비용 안내** (10분): 초기 세팅 + 월 관리 패키지
6. **다음 단계 합의** (5분): 업주 인증부터 시작, 구체적 일정 확정
