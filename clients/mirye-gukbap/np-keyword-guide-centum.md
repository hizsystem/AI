## 2. 영수증 리뷰 가이드 (N1↑)

리뷰 유도 엽서/안내물에 아래 내용을 반영.

### 리뷰 작성 가이드 (고객에게 보여줄 것)

```
리뷰 작성 시 이렇게 써주시면 감사합니다!

1. 방문 목적을 적어주세요
   → "센텀 점심으로", "해장하러", "회사 근처라서"

2. 드신 메뉴와 느낀 점을 솔직하게
   → "돼지국밥이 진하고 깔끔했어요"

3. 사진 1장 이상 첨부

4. 별점 5점 부탁드립니다!
```

### 핵심: 리뷰에 자연스럽게 포함돼야 할 키워드

| 우선순위 | 키워드 | 유도 방법 |
|---------|--------|----------|
| 1 | **센텀 맛집** | 엽서에 "센텀 맛집 미례국밥에 방문해주셔서 감사합니다" 문구 |
| 2 | **센텀 점심 / 점심** | "어떤 목적으로 방문하셨나요?" 질문으로 유도 |
| 3 | **돼지국밥 / 국밥** | 메뉴명 자체가 포함되므로 자연스럽게 |

## 4. 블로그 체험단 키워드 (N1↑ + N2↑)

체험단 블로거에게 전달할 가이드.

```
[미례국밥 센텀점 체험단 키워드 가이드]

필수 포함 (제목 또는 본문):
- "센텀 맛집" (띄어쓰기 주의!)
- "미례국밥"

선택 포함 (1개 이상):
- "센텀 돼지국밥"
- "센텀 점심"
- "재송동 맛집"

해시태그:
#미례국밥 #센텀맛집 #센텀점심 #센텀돼지국밥 #부산국밥 #재송동맛집

주의:
- "센텀맛집"(붙여쓰기) 아닌 "센텀 맛집"(띄어쓰기)으로 써주세요
- 같은 블로거에게 같은 키워드 재의뢰 불가
```

## 센텀점 특이사항

센텀점은 전포본점 대비 N2 기반이 약함:

| 지표 | 센텀점 | 전포본점 | 센텀점 해야 할 것 |
|------|-------|---------|----------------|
| 블로그 | 44개 | 259개 | 체험단 확대 (최소 +10/월) |
| 저장 | ~100 | 3,000+ | 쿠폰/이벤트로 저장 유도 |
| N2 | 0.490 | 0.504 | 인스타 광고 → 플레이스 유입 |

센텀점은 N1 세팅 교체와 함께 **N2 축적(체험단+광고)에 더 많은 투자**가 필요.
