# 미례국밥 N2 개선 실행 계획

> 작성: 2026-04-05
> 목적: N2(축적 데이터) 올리기 — 블로그/유입/저장/체류시간

---

## 1. 인스타그램 → 플레이스 유입 동선

### 프로필 링크 세팅 (리틀리 or 링크트리)

```
미례국밥 인스타 프로필 링크
├── 전포본점 네이버플레이스 예약
│   → https://m.place.naver.com/restaurant/1688300738/home
├── 센텀점 네이버플레이스 예약
│   → https://m.place.naver.com/restaurant/2085068967/home
├── 전포본점 네이버 지도
└── 센텀점 네이버 지도
```

### 인스타 콘텐츠 CTA

모든 게시물/릴스에 CTA 포함:

```
피드: "프로필 링크에서 가까운 지점 예약하세요"
릴스: 마지막 프레임에 "예약은 프로필 링크"
스토리: 링크 스티커 → 해당 지점 플레이스 URL
```

### 인스타 광고 → 플레이스 랜딩

| 설정 | 전포본점 | 센텀점 |
|------|---------|-------|
| 타겟 지역 | 전포동/서면/부산진구 반경 3km | 센텀/재송동/해운대 반경 3km |
| 랜딩 URL | 전포 플레이스 URL | 센텀 플레이스 URL |
| 일 예산 | 1~3만원 | 1~3만원 |
| 소재 | 돼지국밥/돼지우동 비주얼 | 돼지국밥/돼지우동 비주얼 |

핵심: 랜딩을 **자사 홈이 아닌 플레이스로** 보내야 외부유입으로 잡혀서 N2가 올라감.

## 2. 블로그 체험단 (N2↑ + N1↑)

### 섭외 기준 (4개 중 3개 충족)

| 지표 | 기준 |
|------|------|
| 블로그 시작 연도 | 2022년 이전 |
| 일일 방문자 | 300명+ |
| 이웃 수 | 500명+ |
| 게시글 수 | 300개+ |

### 월간 목표

| 지점 | 현재 블로그 | 월 목표 | 키워드 |
|------|-----------|--------|--------|
| 전포본점 | 259개 | +5~8편/월 | "전포 맛집" |
| 센텀점 | 44개 | +5~8편/월 | "센텀 맛집" |

센텀점이 더 급함. 44개는 상위권(500~1,000개) 대비 10배 이상 부족.

## 3. 저장수 늘리기

### 방법

| 액션 | 효과 | 어디서 |
|------|------|-------|
| 알림받기 쿠폰 세팅 | 쿠폰 받기 = 자동 저장 | 플레이스 관리자 |
| "저장하면 서비스" 이벤트 | 직접 저장 유도 | 매장 안내물 |
| 소식/이벤트 쿠폰 발행 | 쿠폰 클릭 시 저장 연동 | 플레이스 소식 |

### 현황 vs 목표

| 지점 | 현재 | 1개월 목표 | 3개월 목표 |
|------|------|----------|----------|
| 전포본점 | 3,000+ | 3,500+ | 5,000+ |
| 센텀점 | ~100 | 500+ | 1,500+ |

---

## 4. 체류시간 늘리기

| 액션 | 효과 |
|------|------|
| 영상 소식 업로드 (5~10초) | 플레이스 상세에서 영상 재생 = 체류시간↑ |
| 대표 사진 10장+ | 사진 넘기는 시간 = 체류시간 |
| 소개문 1,500자+ | 읽는 시간 = 체류시간 |
| 메뉴 사진 전체 등록 | 메뉴 탐색 시간 = 체류시간 |

---

## 5. 당근마켓 비즈프로필

| 설정 | 내용 |
|------|------|
| 비즈프로필 개설 | 전포본점/센텀점 각각 |
| 위치 | 매장 주소 기반 |
| 콘텐츠 | 주 1회 소식 발행 (신메뉴, 이벤트) |
| 링크 | 네이버플레이스 URL |

당근에서 플레이스로의 유입도 외부유입 = N2 기여.

---

## 6. 네이버플레이스 광고

| 설정 | 전포본점 | 센텀점 |
|------|---------|-------|
| 일 예산 | 1~3만원 | 1~3만원 |
| 키워드 태그 | 전포 맛집, 전포밥집, 전포점심 | 센텀 맛집, 센텀국밥, 센텀점심 |
| 제외 검색어 | 무관한 키워드 (세팅 후 1주일 뒤 확인) |
| 파트너 매체 | OFF (네이버 검색에만 집중) |
| 소재 | A/B 테스트 2개 이상 |

광고 자체 전환보다, **유입 고객의 체류시간/클릭이 N2를 높이는 것**이 핵심 효과.
