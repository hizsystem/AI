# 미례국밥 x 브랜드라이즈 — 3개월 마케팅 제안 요약

---

## 한 줄 요약

**"돼지우동 하면 미례국밥"** — 3개월 안에 이 공식을 만드는 프로젝트입니다.

---

## 지금 상황

미례국밥 센텀점은 오픈 2개월 만에 리뷰 1,117개를 모았습니다.
맛은 이미 검증됐습니다.

하지만 아직 이런 상태입니다:
- 네이버에서 "돼지우동"을 검색하면 미례국밥이 안 뜹니다
- 인스타그램에서 "돼지우동 릴스"가 거의 없습니다
- 동동국밥이 이미 우동국밥으로 서울에 진출하고 있습니다

**맛은 있는데, 아직 세상이 모릅니다.**

---

## 3개월 후 모습

| 지표 | 지금 | 3개월 후 |
|------|------|---------|
| "돼지우동" 릴스 | 거의 없음 | 30개 (인플루언서 제작) |
| 블로그 리뷰 | 9팀 진행 중 | 40편 이상 |
| 네이버 "돼지우동" 검색 | 미노출 | 상위 노출 |
| 네이버 "센텀 국밥" 검색 | 순위 미확인 | 상위권 안착 |
| 콘텐츠 자산 | 거의 없음 | 릴스 30개 + 블로그 40편 + 바이럴 가이드 |

이 콘텐츠 자산은 3개월이 끝나도 계속 남아서 일합니다.
그리고 가맹 홍보 소재로 바로 재활용할 수 있습니다.

## 3개월 로드맵

### Month 1. 기반 세팅 (Set-up)

**이달의 미션**: 돼지우동을 히어로 메뉴로 세우고, 3개 채널의 기반을 만듭니다.

| 하는 일 | 왜 필요한지 | 결과물 |
|---------|----------|--------|
| 돼지우동 바이럴 전략 수립 | 어떤 채널에서, 어떤 콘텐츠로, 어떤 사람에게 보여줄지 방향을 정해야 합니다. 방향 없이 돈 쓰면 낭비됩니다. | 채널 전략서, 무드보드, 콘텐츠 가이드라인 |
| 네이버 플레이스 셋업 | 네이버에서 "센텀 국밥" "돼지우동" 검색하면 미례국밥이 떠야 합니다. 지금은 기본 세팅이 빠져있어서 순위에서 밀리고 있습니다. | 플레이스 전 항목 최적화 완료 |
| 블로그 세팅 + 5편 | 네이버 검색에서 "미례국밥" "돼지우동" "부산국밥"을 검색하면 우리 글이 나와야 합니다. 블로그가 없으면 검색 점유율 0%입니다. | 블로그 채널 + 콘텐츠 5편 |
| 인플루언서 씨딩 1차 (10명) | 돼지우동 먹방/ASMR/비주얼 릴스 10개를 확보합니다. 자극적이 아닌, 진정성 있는 콘텐츠로. | 릴스 10개 |
| 메타 광고 | 릴스의 도달을 증폭시킵니다. 광고비 50만원 포함. | 광고 리포트 |

### Month 2. 확산 (Scale-up)

**이달의 미션**: 1차 데이터를 보고, 뭐가 먹히는지 확인하고, 2배로 늘립니다.

| 하는 일 | 왜 필요한지 | 결과물 |
|---------|----------|--------|
| 인플루언서 씨딩 2차 (10명) | 1차 성과를 분석해서 잘 된 유형으로 2차를 집행합니다. 성과 좋은 인플루언서는 재협업합니다. | 릴스 누적 20개, 바이럴 가이드 개선안 |
| 인스타 콘텐츠 기획/제작 | 매장 붐업 콘텐츠(줄서기, 매장 활기, 돼지우동 클로즈업), 콘텐츠 캘린더 | 월간 콘텐츠 캘린더 |
| 블로그 체험단 (10~15팀) | 블로그 리뷰 20편 이상 확보 → 네이버 검색 순위 상승 | 블로그 리뷰 20편+ |
| 네이버 플레이스 관리 | 리뷰 유도 가이드, 키워드 순위 추적, 인스타/블로그/당근 교차 링크 | 경쟁 분석 리포트, 키워드 순위 리포트 |
| 경쟁 매장 비교 분석 | 동동국밥 등 같은 상권 + 돼지우동 유사 메뉴 바이럴 성공 사례 벤치마크 | 경쟁 분석 리포트 |
| 메타 광고 운영 | 릴스 광고 증폭 + 타겟 세분화 + 소재별 A/B 테스트 | 광고 리포트 |

### Month 3. 안착 (Optimize)

**이달의 미션**: 검색 상위에 안착시키고, 4개월차 이후에도 성과가 유지되는 구조를 만듭니다.

| 하는 일 | 왜 필요한지 | 결과물 |
|---------|----------|--------|
| 인플루언서 씨딩 3차 (10명) | 릴스 누적 30개 달성. 성과 검증된 인플루언서와 재협업 | 릴스 30개, 콘텐츠 캘린더 |
| 인스타 콘텐츠 + 가맹 점주용 SNS 운영 가이드 | 이제 가맹점에서도 같은 퀄리티로 SNS를 운영할 수 있게 가이드를 만듭니다 | 자체 운영 가이드 |
| 블로그 체험단 (10~15팀) | 블로그 리뷰 누적 40편 → "돼지우동" "부산국밥" 키워드 검색 상위 안착 | 블로그 리뷰 40편+, 리뷰 관리 가이드 |
| 네이버 플레이스 관리 | 리뷰 관리 가이드 납품, 리뷰 유도 POP 제안 | 키워드 순위 리포트 |
| 메타 광고 + 풀 리포트 | 전 채널 성과 분석, 자체 운영 가이드, 가맹 홍보 활용 방안 | **3개월 풀 성과 리포트 + 다음 분기 제안서** |

## 3개월 동안 쌓이는 자산

3개월이 끝나면 위드런이 가져가는 것:

| 자산 | 수량 | 활용처 |
|------|------|--------|
| 돼지우동 릴스 | 30개 | 인스타 피드, 메타 광고, 가맹 소개 자료 |
| 블로그 리뷰 | 40편+ | 네이버 검색 점유, 가맹 홍보 |
| 채널 전략서 | 1식 | 자체 운영 기준, 가맹점 교육 |
| 바이럴 가이드 | 1식 | 인플루언서 협업 매뉴얼 |
| 키워드 전략서 | 1식 | 네이버 플레이스 운영 기준 |
| 콘텐츠 캘린더 | 3개월분 | 자체 운영 템플릿 |
| 자체 운영 가이드 | 1식 | 4개월차부터 자체 운영 가능 |
| 가맹 점주용 SNS 가이드 | 1식 | 가맹 교육 자료 |
| 3개월 풀 성과 리포트 | 1식 | 의사결정 데이터, 투자자 보고 |

**3개월 후에는 외부 도움 없이도 운영할 수 있는 구조가 만들어집니다.**
추가로 필요하시면 그때 다시 논의하면 됩니다.

## 예산 구조 (월 500만원)

### 한눈에 보기

| 영역 | M1 | M2 | M3 | 3개월 합계 |
|------|-----|-----|-----|----------|
| 인스타 콘텐츠 & 씨딩 | 120만 | 220만 | 220만 | 560만 |
| 블로그 & 검색 | 100만 | 130만 | 130만 | 360만 |
| 네이버 플레이스 | 80만 | 150만 | 50만 | 280만 |
| 메타 광고 (광고비 포함) | 80만 | 50만 (광고비) | 100만 (풀리포트) | 230만 (M3는 풀 분석) |
| 전략 수립 | 120만 | - | - | 120만 (1회) |
| **월 합계** | **500만** | **500만** | **500만** | **1,500만** |

### 왜 이 금액인지

- **인플루언서 씨딩 120만/월**: 10명 x 무가(음식 제공만). 유가 씨딩(1인당 30-50만원)보다 훨씬 저렴. 대신 저희가 선정/코디/QC를 해서 퀄리티를 관리합니다.
- **메타 광고 50-80만/월**: 광고비가 포함된 금액입니다. 릴스 도달을 늘리는 데 최소한의 부스트.
- **전략 수립 120만 (1회)**: 첫 달에만 발생. 방향 없이 실행하면 3개월치 예산이 낭비됩니다.
- **블로그 체험단 80만/월**: 10~15팀 모집/관리/QC. 자체 진행 대비 효율과 품질이 올라갑니다.

## 일하는 방식

### 주간 리듬

| 요일 | 내용 |
|------|------|
| **월요일** | 주간 미팅 (30분) — 이번 주 할 일 확인, 지난 주 성과 리뷰 |
| **수요일** | 중간 체크 (카톡) — 진행 상황 공유, 급한 이슈 대응 |
| **금요일** | 주간 리포트 발송 — 숫자 + 인사이트 + 다음 주 계획 |

### 월간 산출물
매달 말에 받으시는 것:
- 월간 성과 리포트 (숫자 + 해석 + 액션)
- 다음 달 콘텐츠 캘린더
- 비용 집행 내역

### 소통 채널
- 카카오톡 단체방 (실무 소통)
- 주간 미팅 (줌 또는 대면)
- 긴급 이슈: 전화

## 경쟁사 동향 — 왜 지금 해야 하는지

**동동국밥**이 이미 우동국밥으로 서울에 진출하고 있습니다.

"돼지우동"이라는 키워드는 아직 누구도 선점하지 않았습니다.
지금 3개월 안에 릴스 30개 + 블로그 40편을 깔아놓으면,
나중에 경쟁자가 "돼지우동"을 검색해도 미례국밥이 먼저 나옵니다.

**선점은 빠를수록 싸고, 늦을수록 비쌉니다.**

## 자주 묻는 질문

**Q. 3개월 후에 효과가 없으면요?**
매주 데이터를 공유하고, 매달 성과 리포트를 드립니다. 안 되는 건 빠르게 끊고, 되는 건 밀어넣는 구조입니다. 3개월 내내 안 되는 채 끝나는 일은 없습니다.

**Q. 기존에 자체 진행하던 체험단/리뷰이벤트는요?**
그대로 유지하시면 됩니다. 저희가 그 활동을 전략적으로 구조화해서 효율을 높여드리는 겁니다. 중복 비용이 아니라 시너지입니다.

**Q. 가맹 마케팅은 별도인가요?**
이번 3개월은 센텀점 "돼지우동 바이럴"에 집중합니다. 다만 Month 3에서 만드는 자산(릴스, 리뷰, 가이드)은 가맹 홍보에 바로 활용할 수 있도록 설계합니다.

**Q. 댄싱컵은요?**
이번 프로젝트는 미례국밥에 집중합니다. 댄싱컵은 별도 논의가 필요합니다.
