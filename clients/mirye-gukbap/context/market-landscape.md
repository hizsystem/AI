## 국밥 시장 개요

| 지표 | 수치 |
|------|------|
| 한국 외식 시장 규모 | 약 110~153조원 (2024) |
| 부산 돼지국밥 등록 매장 | 1,090개+ (다이닝코드) |
| 서울 돼지국밥 등록 매장 | 537곳+ (다이닝코드) |
| 돼지국밥 평균 가격 | 부산 8,000~10,000원, 수도권 10,000~12,000원 |

### 핵심 트렌드

- **뉴웨이브 국밥** (2025 배민트렌드 3대 키워드): 고급 그릇, 이색 토핑, 세련된 인테리어, 2030 여성 타겟
- **로코노미**: 지역명 포함 메뉴 판매 매장 약 20만곳 (배민 2025 Q1)
- **혼밥 비중**: 전체 식사의 47% (2024), 긍정 인식 83%
- **부산 돼지국밥의 전국화**: 미쉐린 6곳 선정, 영화 '변호인' 이후 인지도 급상승
- **국밥 프랜차이즈 경기방어력**: 불황에도 꾸준한 수요, "한 그릇 식사" 트렌드

## 직접 경쟁사

### 1. 동동국밥 (최우선 경쟁자)

| 항목 | 내용 |
|------|------|
| 운영사 | 위포먼스F&B |
| 설립 | 2023년 (울산 언양본점) |
| 매장 수 | 약 40개 (2025.01) |
| 지역 | 울산 중심, 부산 (센텀/장산/하단 등), 서울 마곡나루 1곳 |
| 대표 메뉴 | 우동돼지국밥 9,500원 |
| 브랜드 검색량 | 월 24,600건 |
| 식자재 | 동원홈푸드 MOU 체결 |

**경쟁 분석:**
- 2년 만에 40개 매장 = 매우 빠른 성장
- "우동국밥" 컨셉으로 이미 시장 인지도 확보
- 서울 진출은 아직 초기 (마곡나루 1곳)
- 미례국밥과 동일 경로 (영남 → 수도권)

**미례국밥 대응 전략:**
- 동동의 키워드와 직접 싸우지 않음 (24,600건 vs 미례 1,660건)
- "돼지우동"을 인스타/블로그에서 별도 인지도 구축
- 차별화: 맛/레시피, 브랜딩 감성, 히어로 메뉴 스토리

## 센텀 상권 분석

| 지표 | 수치 |
|------|------|
| 상주 직장인 | 약 32,000명 (지식산업단지 850개+ 업체) |
| 거주 인구 | 약 6,000명 |
| 핵심 시설 | 신세계백화점, 롯데백화점, BEXCO, 메디컬타운 |
| 오피스 공실률 | 18.1% (2024 Q4, 전국 평균의 2배) |
| 소비 특성 | 프리미엄 소비, 비즈니스+라이프스타일 복합 |

**상권 해석:**
- 평일 점심 직장인 수요 = 핵심 매출원
- 오피스 공실률 상승 중이나 여전히 부산 최대 오피스 밀집
- 주말 신세계/BEXCO 방문객이 유동인구 견인
- 10,000원대 국밥은 직장인 점심 가격대 적합
