| 항목 | 내용 |

| 가격 | 11,500원 |
| 구성 | 진한 돼지국밥 육수 + 쫄깃한 우동면 |
| 포지셔닝 | "국밥집이 만든 우동, 이게 진짜 돼지우동이다" |
| 바이럴 요소 | 이름 자체 호기심 유발, 면발 올라가는 비주얼, SNS "신기한 메뉴" |
| 전략적 역할 | 인지도 견인 → 방문 유도 → 국밥 본메뉴 재방문 |

### 돼지우동 키워드 전략 (킥오프 컨펌)

- NP 대표키워드에서는 **제거** (카테고리 불일치로 N1 적합도 하락)
- **인스타 + 블로그에서 인지도용**으로 활용
- 목표: 인스타 바이럴로 검색량 1,000건+ 키운 후 NP 메인 키워드 교체
