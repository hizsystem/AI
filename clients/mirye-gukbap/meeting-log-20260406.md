## Part 1. 미례국밥 (00:00~42:00)

### 데이터 공유 — 컨펌됨

- 월 검색량: 1,600 → **2,690명** (62% 증가, 4/1 기준)
- 콘텐츠 포화도: **2.42%** — 100명 검색하면 2명만 콘텐츠를 봄. 98명은 볼 게 없어서 이탈
- 타겟: 30~40대 71%, 남성 55%
- **전포 미례국밥 1,220건** > 센텀 670건 (전포가 검색량 2배)
- "이치란 국밥" 12,800건 → 인플루언서가 "이치란 같은 국밥"으로 바이럴해서 키워드 급등한 것. 블로그에 "이치란"+"국밥" 동시 노출되면서 수치가 부풀려진 측면도 있음

### 네이버플레이스 — 컨펌됨

**센텀점:**
- 센텀 맛집 키워드: 42위 → **9위** 급상승
- 재성동 맛집: **3위**
- 센텀 국밥, 센텀 돼지국밥: **1~2위**
- 문제: 블로그 콘텐츠 부족 → 체험단으로 채우기

**전포점:**
- 리뷰/저장 자산 풍부 (블로그 257개, 저장 3,000개)
- 문제: 세팅 미비 (소개문 200자, 예약/쿠폰 미설정) → **39점**
- 전포 맛집 키워드: 월 33,850건인데 현재 **8위** → 세팅만 잡으면 올라감

**대표키워드 교체 — 컨펌됨:**
- 센텀점: 돼지우동 → **센텀 맛집** (월 14,230건, 이미 9위까지 올라옴)
- 전포점: 민생회복소비쿠폰 → **전포 맛집** (월 33,850건)

**돼지우동 키워드 전략 — 컨펌됨:**
- 현재 월 검색 310건, 상위 18개가 전부 고기집 → NP 키워드로는 비효율
- N1(적합성) 문제: "우동" 검색 시 국밥집이 나오면 적합도 점수가 깎임
- 전략: **인스타에서 콘텐츠 바이럴 → 검색량 1,000건+ 키운 후 → 그때 메인 키워드로 교체**

**별점 제도 부활:**
- **오늘(4/6) 부활** → 리뷰 시 별점 5개 요청 가이드 필요
- 리뷰 대댓글 템플릿 제공 예정

**플레이스 광고:**
- 일 5,000원 추천 → 월 약 12~15만원
- 점수 확보 목적 (돈을 더 쓴다고 효과가 비례하지 않음)

### 인스타그램 — A안(브랜딩) 컨펌됨

**방향 결정:**
- A안(브랜딩) 확정, B안(가맹)은 추후 가미
- 대표: "브랜딩으로 갈 거면 확실하게 가는 게 맞다. 이도 저도 아닌 건 의미 없다"
- 브랜딩이 갖춰져야 가맹도 매력적으로 보인다는 논리에 동의

**콘텐츠 믹스:**
- 음식 50% / 사람 25% / 매장 15% / 브랜드 10%
- 핸드폰 촬영 중심 (각 잡고 찍는 건 아님)
- 레퍼런스: SÁNG, OUTTA COFFEE 등 커피 브랜드 참고

**톤 & 슬로건 — 컨펌됨:**
- 톤: **"정성이 보이는데 격식은 차리지 않는 곳"**
- 슬로건: **"오늘도 정성 한 그릇."** → 모든 콘텐츠 끝에 삽입

**가맹 요소:**
- 직접적 "가맹문의" 노출 대신 리틀리에 간접 삽입
- 프로필: "함께할 사장님을 찾습니다" 같은 열린 문장으로

**대표 피드백 (중요):**
1. "새벽 5시 출근" 같은 장인 콘텐츠는 **가맹 관점에서 부담** → 선 지키기. 정성을 보여주되 "이걸 나도 해야 해?"가 되면 안 됨
2. 직원/대표 노출: **대표는 노출 꺼림** ("하고 싶은 말이 없다"), 직원도 마찬가지 → 방법 찾기로 협조
3. 리틀리 매장 늘어나면? → 지역별 분류로 확장 가능

### 협의사항 — 후속 필요

| 항목 | 상태 | 후속 |
|------|------|------|
| NP 관리자 계정 공유 | 변팀장과 협의 | 변팀장에게 요청 |
| 인스타 계정 접근 | 변팀장과 협의 | 변팀장에게 요청 |
| 메타 비즈니스 계정 | 확인 필요 | 변팀장에게 확인 |
| 촬영 소스 | 기본 사진만 있음, 영상 없음 | 부산 출장 1일 촬영 필요 |
| 체험단 비용 | 기존 업체 있음, 비용 디테일 미확인 | 변팀장과 별도 협의 |

### KPI — 컨펌됨

## Part 3. 댄싱컵 — 월간 리테이너 (NEW) (49:00~01:00:00)

### 대표 직접 제안 — 핵심 신규 안건

대표가 2년간 마케팅 방향을 못 잡고 있었는데, 최근 내부적으로 방향이 잡혔다고 직접 설명.

### 댄싱컵 포지셔닝 2축

**축 1. "라떼 맛있는 집"**
- 실제로 라떼가 맛있지만, 고객이 "여기 라떼 맛있어"로 인지하게 만들지 못함
- 큰 슬로건으로 가져가는 스테디 전략

**축 2. "유행을 가장 빨리 맛볼 수 있는 곳"**
- 프랜차이즈 규모 120~130개 → 메가/컴포즈(3,000~4,000개) 대비 작아서 **빠르게 움직일 수 있는 게 장점**
- 두쫀쿠, 우배 등 유행 사이클 빠르게 캐치 → 즉시 출시
- 오늘(4/6) 새 메뉴 출시함, 둘째 주에 또 출시 예정
- 큰 프랜차이즈는 수급/의사결정 단계가 많아서 유행 따라가기 어려움 → 댄싱컵의 차별점

### 예산 규모

| 항목 | 금액 |
|------|------|
| 월간 리테이너 | **약 500만원** |
| 별도 마케팅 예산 (인플루언서, 콘텐츠 제작 등) | **3,000~5,000만원** (사용 방법은 제안 받겠다) |

### 대표 발언 핵심
> "2년간 아무것도 안 한 게 하기 싫어서가 아니고 뭘 해야 될지 몰라서. 이번에 방향이 잡혔다."
> "내부 역량으로는 방향성을 끌어가기 힘드니까 도움을 받고 싶다."
> "큰 타이틀은 라떼 + 트렌드 빠른 캐치, 나머지는 다 어떻게든 할 수 있다."

### 우성민 반응
> "딱 듣자마자 — 유행을 가장 빨리 맛볼 수 있는 곳, 댄싱컵. 이거 되잖아요? 유행꽃에 탑승하는 거예요."
> "라떼 맛있는 집 + 트렌드 빠른 캐치, 이 두 축이 정체성을 잘 확립하셨다."

### 후속 액션
- **차주에 댄싱컵 제안 미팅** (변팀장과 일정 조율)
- 대표: 동석할 수도 있고, 못 하면 내용 공유받으면 됨
- 변팀장: "방향은 잡았지만 완전 픽스는 아님. 제안 받고 조정"

## 미팅 톤 메모 (내부 참고)

- 대표는 명쾌하고 빠른 의사결정자. 방향 제시하면 "됐다" 스타일
- 변팀장은 실무 디테일 중심. 일정/비용/운영 관련 질문이 많음
- 대표 우려: "알맹이 없이 톤만 예쁜 인스타"를 가장 경계함 → 실질적 성과 연결 강조
- 대표 노출 완전 기피 → 대표 인터뷰 콘텐츠는 배제
- "장인 콘텐츠 주의": 새벽 출근 등 지나친 정성 강조는 가맹 부담 → 선 지키기
- 댄싱컵은 대표가 직접 방향을 잡아온 만큼 열의가 높음 → 빠르게 제안서 가져가기
