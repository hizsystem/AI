## 주간 리딩 구조

### 주간 루틴 (매주 월요일)

```
[월요일 오전] Green 혼자 15분
├── 지난 주 프록시 지표 스크린샷 수집 (NP 통계, 인스타 인사이트)
├── 이번 주 태스크 확인 + 인턴 배분
└── 슬랙/콜 미팅 어젠다 1줄 정리

[월요일 오후] 인턴 30분 싱크
├── 지난 주 완료 태스크 체크
├── 이번 주 할 일 확인 (구체적 마감일)
└── 막히는 거 있으면 바로 해결

[수요일] 인턴 자율 실행, Green 비동기 QC
├── 인턴이 작업물 슬랙/카톡 공유
└── Green이 코멘트로 피드백

[금요일] 위드런 슬랙 체크인 (격주는 콜 미팅)
├── 이번 주 한 일 3줄 요약
├── 다음 주 예고 1줄
└── 필요한 협조 사항 (사진, 매장 접근 등)
```

### 인턴 업무 가이드

| 산출물 | 템플릿/가이드 | Green이 해줄 것 |
|--------|-------------|---------------|
| 블로그 콘텐츠 | 키워드 + 구조(서론/본론/결론) + 참고 블로그 3개 지정 | 초안 QC + 키워드 체크 |
| NP 소식 | 이미지 사이즈 + 카피 공식 + 예시 3개 | 발행 전 최종 확인 |
| 씨딩 코디 | DM 템플릿 + 협업 제안서 + 가이드 | 최종 인플루언서 선정 |
| 리뷰 답글 | 템플릿 10종 → 위드런 전달 | 템플릿 작성 |
| 데이터 수집 | 스크린샷 매뉴얼 (어디서 뭘 캡처하는지) | 주간 해석 |
