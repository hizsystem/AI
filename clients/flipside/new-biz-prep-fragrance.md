## 1-Page 요약 (상담 5분 전에 읽기)

### 브랜드 현재 상황
라이프 프래그런스 브랜드. 2023년 설립, 멀티퍼퓸(고부향률) 중심 제품 라인업. 무신사/W컨셉 입점 완료, imweb 기반 자사몰 운영. 디지털 존재감 거의 없음.

### 브랜드 현재 고민
- 표면: "0부터 시작하는 초기브랜드가 타겟 고객에게 인지도를 알리는 방안이 가장 큰 고민입니다."
- 추정 근본: 브랜드 포지셔닝 미정립 + Go-to-Market 설계도 부재. 콘텐츠 이전에 '누구에게, 왜 선택받아야 하는지'가 언어화되지 않은 상태.

### 마케팅 행보
SNS 존재감 미미, 미디어 노출 전무. 무신사/W컨셉 입점은 되어 있으나 트래픽 유입 전략 없음. "높은 부향률"이라는 기능적 소구에 머물러 있어 경쟁 브랜드 대비 차별화 약함.

### 필수 질문 Top 5
1. "플립사이드가 기존 헬스/뷰티 브랜드와 다른 점을 한 문장으로 말씀해주신다면?" — 의도: 포지셔닝 언어화 수준 확인
2. "지금 타겟 고객을 떠올리면 어떤 사람이 그려지세요?" — 의도: 타겟 명확도 진단
3. "100만원 예산으로 가장 기대하시는 건, 콘텐츠 대행인지 방향 코칭인지?" — 의도: 기대치와 현실적 모델 갭 확인
4. "현재 인스타그램이나 다른 SNS 계정이 있으신가요? 반응은 어떠셨어요?" — 의도: 현재 운영 상태 파악
5. "지금 이 상태로 6개월 더 가면 어떻게 될 것 같으세요?" — 의도: 긴급성 인식 + 변화 의지 확인

## 니즈 분석 상세

### SPIN 분석
- Situation: 헬스/뷰티 업종 초기 브랜드. 마케팅 경험 거의 없음. 월 예산 100만원.
- Problem: 타겟 고객 도달 방법 모름. 콘텐츠 체계 없음.
- Implication: 타겟과 무관한 콘텐츠 양산, 초기 6개월 골든타임 낭비. 재진입 비용 2-3배.
- Need-Payoff: 명확한 포지셔닝 + 타겟 정의 선행 시, 100만원으로도 입소문 구조 가능.

### Pain Hierarchy
- L1 표면 요청: "SNS 콘텐츠를 만들어서 올려주세요"
- L2 전술적 문제: "무슨 콘텐츠를 만들어야 타겟에게 닿는지 모르겠다"
- L3 전략적 문제: "브랜드가 누구에게, 왜 선택받아야 하는지 정리가 안 됐다"
- L4 근본 원인: "Go-to-Market 설계도 자체가 없다"

### JTBD 분석
- Functional Job: 제한된 예산으로 작동하는 인지도 확보 시스템
- Social Job: 헬스/뷰티 시장에서 '아, 그 브랜드' 하고 떠오르는 존재
- Emotional Job: 이 방향이 맞는지 확신, 혼자 결정하는 불안 해소

### 희망 서비스 vs 진짜 니즈
- 요청: SNS 콘텐츠 제작 + 운영 대행
- 추정 진짜 니즈: 브랜드 포지셔닝 → 타겟 정의 → 콘텐츠 전략 → 실행 코칭
- 갭 크기: 큼 — 콘텐츠 제작은 전략 수립 이후의 실행 단계

## 수주 전략 상세

### 기회 적합도 (3.5/5.0 — Conditional Go)
- 니즈-서비스 핏: 4/5 (초기 브랜드 인지도 확보 = 우리 코어)
- 예산 적합성: 2/5 (월 100만원, 풀 리테이너 불가)
- 의사결정 구조: 5/5 (대표 직접 신청, 즉시 결정 가능)
- 긴급도: 3/5 (시급하나 론칭 시점 미확인)
- 성장 잠재력: 4/5 (성장 시장, 예산 증가 여지)
- 레퍼런스 가치: 3/5 (0→1 성공 케이스 포트폴리오 가치)

### Challenger Teaching 3가지
1. "초기 브랜드 SNS는 팔로워 수가 아니라 '첫 100명의 질'이 전부" — DM 문의 수, 저장 수가 진짜 KPI
2. "헬스/뷰티 초기 브랜드 80%가 똑같은 실수: 제품 스펙 나열" — 사용 장면(Use Scene) 콘텐츠가 저장률 3배
3. "월 100만원이면 대행이 아니라 '시스템'을 사야 한다" — 3개월 코칭 후 자체 운영 가능한 체계

### 핵심 솔루션 1줄
"월 100만원 예산의 초기 헬스/뷰티 브랜드에게 대행을 팔지 말고, 80만원짜리 1회 진단으로 진입한 뒤 코칭형 월 리테이너로 전환하라."

## 상담 코칭 상세

### 60분 타임라인
- Phase 1 (5분): 라포 형성 — 강의 경험 연결, 향 이름 네이밍 소재
- Phase 2 (15분): 상황 확인 — 메일 미회신이므로 확대, SPIN Situation/Problem
- Phase 3 (15분): 깊은 진단 — SPIN Implication/Need-Payoff + 직접 진단
- Phase 4 (10분): 인사이트 공유 — Challenger Teach + SCQA 프레이밍
- Phase 5 (10분): 솔루션 방향 — 3단계 로드맵 스케치
- Phase 6 (5분): 클로징 — 다음 단계 합의

### Quick Tip (Trusted Advisor 마무리)
"인스타 하이라이트에 '향 이야기' 카테고리를 만드세요. 각 향마다 15초 릴스 — '이 향을 만들 때 떠올린 장면'을 한 줄로 쓰고 무드 영상을 깔면 됩니다. 향수는 코가 아니라 상상력으로 먼저 파는 거예요."

### 상담 후 액션
- 당일: 감사 문자 + 내부 판단
- 2일 이내 (4/10): 1장짜리 방향 요약 발송
- 1주 이내 (4/15): 반응 확인 → 정식 제안서 or 관계 유지
