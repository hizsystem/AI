## Step 1: 인테이크 데이터 분석

### 원문 키워드 추출

| 구분 | 원문 | 추출 키워드 |
|------|------|-------------|
| 목표 | "브랜드 인지도를 올리고 싶습니다" | 인지도, 0→1 |
| 수단 | "팔로워를 늘려서 브랜드 제품을 노출" | 팔로워=인지도라는 등식 |
| 단계적 사고 | "이후에는 인지도>구매로 전환" | 퍼널 인식은 있으나 구체화 안 됨 |
| 자기 인식 | "마케팅을 전혀 모르는 상태" | 무경험 자각, 학습 의지 |
| 희망 | "마케팅 방향 설정 + 현실적 실행 방안" | 방향 > 실행, '현실적'이라는 단서 |

### 감정 톤 분석

- 전반적 톤: **솔직한 초보**. 모르는 걸 숨기지 않음.
- 플립사이드 prep에서도 동일 패턴 — "0부터 시작하는 초기브랜드"라는 표현 사용.
- "현실적"이라는 단어에 주목: 과거에 비현실적 제안을 받았거나, 예산 제약을 인지하고 있음.
- 2개 브랜드를 동시에 상담 요청 = 한 번에 정리하고 싶은 급한 마음. 런칭 6월이면 실질 준비 기간 2개월 미만.

### 플립사이드 연결 맥락

정호정 대표는 플립사이드(향수)와 브렐라(바디케어) 2개 브랜드를 운영. 플립사이드 prep 진단 결과:
- 마케팅 성숙도: 1~2/5 (미착수~시행착오)
- 핵심 과제: 브랜드 서사 부재, 타겟 정의 미흡, 콘텐츠/채널 전략 부재
- 예산: 월 100만원 수준

브렐라는 플립사이드보다 더 초기 — 아직 제품 출시도 안 된 프리런칭 단계.

## Step 2: SPIN 기반 니즈 레이어링

### Situation (객관적 현재 상태)

| 항목 | 상태 |
|------|------|
| 브랜드 단계 | 프리런칭 (6월 출시 예정, 약 2개월 남음) |
| 제품 라인업 | 3종 (세라마이드 클렌저, 자일리톨 쿨링 미스트, 레티노이드 바디로션) |
| 판매 채널 | 자사몰 + 스마트스토어 (예정) |
| 마케팅 채널 | 브랜드 인스타 @brela_official (초기), 대표 개인 스레드 |
| 마케팅 경험 | 광고 진행 없음. 완전 제로. |
| 팀 구성 | 확인 필요 — 플립사이드와 겸업 추정 (1인 또는 소규모) |
| 예산 | 미확인 (플립사이드가 월 100만원이므로 유사 수준 추정) |
| 타겟 | 운동하는 30~40대 남녀 (헬스, 러닝, 필라테스, 수영) |
| 포지셔닝 | "운동 후 회복" — 사용 상황 명확 |
| 경쟁 환경 | 국내에 "운동 후 바디케어" 특화 브랜드 거의 부재 (블루오션이자 시장 미검증) |

### Problem (대표가 인식한 문제)

1. **인지도가 0이다** — 브랜드 존재 자체를 아무도 모름
2. **마케팅을 어디서부터 시작해야 할지 모른다** — 방향 자체가 없음
3. **팔로워가 없어서 제품 노출이 안 된다** — 팔로워=노출이라는 공식
4. **인지도에서 구매로 넘어가는 방법을 모른다** — 퍼널 전환 고민

### Implication (방치 시 파급효과)

| 문제 | 방치하면 | 비용 |
|------|---------|------|
| 인지도 0인 채 6월 런칭 | 출시해도 아무도 모름. 초기 매출 0에 수렴. 제품 유통기한 리스크. | 재고 비용 + 기회비용 |
| 마케팅 방향 없이 콘텐츠 시작 | 제품 스펙 나열형 콘텐츠 양산 → 팔로워 무반응 → 의욕 저하 | 3-6개월 골든타임 |
| 프리런칭 기간(2개월)을 허비 | 런칭 후 0부터 인지도 쌓기 시작 = 매출 발생까지 추가 3-6개월 | 캐시번 압박 |
| 플립사이드와 동시 운영 | 한정된 리소스 분산, 두 브랜드 모두 중도 포기 리스크 | 브랜드 2개 동시 실패 |

### Need-Payoff (해결 시 가치)

1. **프리런칭 2개월을 전략적으로 쓰면**: 런칭 시점에 초기 팔로워 500-1,000명 + 시딩 리뷰 확보 가능
2. **"운동 후 바디케어"라는 명확한 사용 상황이 최대 무기**: 콘텐츠 방향이 자동으로 잡힘 — 운동 장면이 곧 콘텐츠
3. **타겟이 커뮤니티성이 강한 집단**: 러닝크루, 필라테스, 크로스핏 등은 자발적 공유 문화 → 시딩 효과 극대화 가능
4. **2개 브랜드 통합 마케팅 설계**: 플립사이드(향)와 브렐라(바디케어)의 크로스셀링 구조 설계 가능

## Step 3: 고민 계층 분석 (Pain Hierarchy)

### L1. 표면 요청
**"팔로워 늘려서 인지도 올리고 싶어요"**

팔로워 수 = 인지도 = 매출이라는 단순 공식. 대부분의 초기 대표가 가진 인식. 이것 자체가 틀린 건 아니지만, 팔로워 숫자보다 "누가 팔로우하느냐"가 핵심.

### L2. 전술적 문제
**"무슨 콘텐츠를 만들어야 하고, 어떤 채널에서, 어떤 순서로 해야 하는지 모른다"**

- 인스타그램을 만들었는데 뭘 올려야 할지 모름
- 광고를 해야 하는지, 시딩을 해야 하는지, 협찬을 해야 하는지 판단 기준이 없음
- 6월 런칭까지 남은 2개월을 어떤 순서로 채워야 하는지 타임라인 부재

### L3. 전략적 문제
**"브렐라가 시장에서 어떤 위치를 잡아야 하는지, 누구의 첫 번째 선택이 되어야 하는지 정리가 안 됐다"**

- "운동 후 바디케어"라는 컨셉은 잡았지만, 이게 기존 바디케어(일리윤, 세타필 등)와 왜 다른지 언어화 안 됨
- 30~40대 운동하는 남녀라는 타겟은 넓음 — 헬스하는 30대 남성과 필라테스하는 40대 여성은 완전히 다른 니즈
- 3종 제품 중 "히어로 제품"이 뭔지 결정 안 됨

### L4. 근본 원인
**"2개 브랜드를 동시에 런칭/운영하면서 각각의 GTM(Go-to-Market) 설계도가 없다"**

정호정 대표의 본질적 상황:
- 플립사이드(향수): 이미 제품 있지만 마케팅 미착수
- 브렐라(바디케어): 제품 출시 전, 마케팅도 미착수
- 1인(또는 극소규모)으로 2개 브랜드 동시 구축
- 어디에 먼저 집중해야 하는지, 리소스 배분 기준이 없음
- "마케팅을 전혀 모르는" 상태에서 2개 브랜드의 마케팅을 동시에 고민

## Step 4: JTBD (Jobs-to-be-Done) 분석

### Functional Job (실질적으로 해결할 업무)

> "6월 런칭 전까지, 한정된 예산으로 브렐라를 타겟 고객이 '아, 그 운동 바디케어' 하고 떠올리게 만들 수 있는 실행 가능한 마케팅 시스템을 세팅하고 싶다."

구체적 하위 Job:
1. 인스타그램 콘텐츠 전략 + 초기 콘텐츠 방향 세팅
2. 6월 런칭까지의 프리런칭 타임라인 수립
3. 시딩/체험단 등 초기 인지도 확보 전술
4. 런칭 후 인지도 → 구매 전환 퍼널 설계
5. 플립사이드와의 리소스 배분 기준

### Social Job (외부에 어떻게 보이고 싶은지)

> "운동하는 사람들 사이에서 '진짜 운동하는 사람이 만든, 운동하는 사람을 위한 브랜드'로 인정받고 싶다."

- 단순 바디케어가 아니라 "운동 라이프스타일 브랜드"로 포지셔닝하고 싶은 욕구
- 대표 개인 스레드(@hohoy_j)를 운영하는 것 자체가 "창업자의 얼굴"을 보여주겠다는 신호
- 운동 커뮤니티에서 "쓸 만한 제품 나왔다"는 입소문을 원함

### Emotional Job (내면적으로 해소할 감정)

> "2개 브랜드를 혼자 끌고 가는데, 이 방향이 맞는 건지 확인받고 싶다. 뭘 해야 할지 알면 실행은 할 수 있는데, 지금은 뭘 해야 할지조차 모르겠다."

- **불확실성에 대한 불안**: 마케팅 전무 경험 + 2개 브랜드 동시 운영
- **시간 압박**: 6월 런칭이 코앞인데 아직 마케팅 방향도 없음
- **컨펌에 대한 갈증**: "이렇게 하면 됩니다"라는 확신을 줄 전문가가 필요
- 플립사이드 상담에 이어 바로 브렐라까지 의뢰한 것 = "한 번에 정리하고 싶다"는 급한 심리

## Step 5: 유사 사례 패턴 매칭

### 패턴 1: "운동 특화" 뷰티/바디케어 — 국내 사례 거의 없는 블루오션

**시장 현황:**
- 국내에 "운동 후 바디케어"를 전면에 내세운 브랜드가 사실상 없음
- LG생활건강 '비클리닉스'가 "부위별 바디케어"로 기능성 접근했지만, 운동 특화는 아님
- 해외에는 Megababe, SweatWellth, Kinfield 등 "액티브 라이프스타일" 스킨케어 존재
- 국내 바디케어 시장은 보습/저자극 중심 (일리윤, 세타필, 피지오겔이 상위권)

**시사점:**
- 경쟁자가 없다 = 기회이자 리스크. "운동 후 바디케어"라는 카테고리를 소비자 머릿속에 새로 만들어야 함
- 기존 바디케어 브랜드와의 비교가 아니라, "운동하는 사람의 루틴"이라는 새로운 프레임 필요
- 카테고리 창출형 브랜드는 초기 교육 비용(콘텐츠)이 많이 듦

### 패턴 2: D2C 뷰티 브랜드 프리런칭 — 흔한 실수와 성공 패턴

**실패 패턴 (80% 이상이 빠지는 함정):**
1. 런칭일에 맞춰 "오픈했습니다!" 게시물 → 아무도 모름 → 매출 0
2. 제품 스펙/성분 나열형 콘텐츠 → 저장 0, 공유 0
3. 팔로워 늘리기 위해 팔로우/언팔 노가다 → 유령 팔로워만 증가
4. 광고비 태우기 전에 랜딩페이지/콘텐츠 미준비 → CPA 폭등

**성공 패턴:**
1. 런칭 8-12주 전부터 "브랜드 스토리" 콘텐츠로 기대감 형성
2. 창업자의 개인 브랜딩이 초기 팬 확보의 핵심 (정호정 대표 스레드가 여기에 해당)
3. 타겟 커뮤니티(러닝크루, 헬스 동호회, 필라테스 스튜디오)에 시딩 → 리뷰 확보
4. "히어로 제품" 1개에 집중 → 나머지는 크로스셀

### 패턴 3: 초기 D2C 바디케어 브랜드의 진짜 니즈 (우리가 봐온 것)

유사 규모/업종 클라이언트에서 반복적으로 발견되는 니즈:

| 대표가 말하는 것 | 진짜 필요한 것 |
|-----------------|---------------|
| "팔로워 늘리고 싶다" | 타겟 고객 정의 + 채널 전략 |
| "콘텐츠 만들어주세요" | 콘텐츠 기획 체계 (뭘 만들지 아는 것) |
| "인지도 올리고 싶다" | 프리런칭 타임라인 + 런칭 캠페인 설계 |
| "광고 해야 하나요?" | 오가닉 기반 확보 후 광고 판단 기준 |
| "마케팅 대행 찾고 있다" | 방향 설계 + 코칭 (대행할 예산이 안 됨) |

### 패턴 4: "운동하는 30~40대" 타겟의 미디어 소비 특성

- 인스타그램 릴스 + 스레드가 주력 채널 (대표가 이미 스레드 운영 중 = 감이 있음)
- 운동 커뮤니티 특성: 루틴 공유, 제품 추천이 자발적으로 일어남
- 러닝크루/필라테스 스튜디오는 로컬 커뮤니티 → 오프라인 시딩 효과적
- "운동 후 피부 고민"은 실제로 존재하지만 아직 콘텐츠화되지 않은 영역

## Step 6: 검증 질문 도출

상담은 이미 진행되었으므로, 후속 미팅 또는 제안서 작성 시 확인해야 할 핵심 질문.

### 질문 1: 히어로 제품 결정

**질문:** "3종 제품 중 하나만 광고한다면 어떤 제품인가요? 그리고 그 이유는?"

**검증 가설:** 대표가 히어로 제품을 아직 결정하지 못했을 가능성이 높다. 3종을 동시에 밀면 메시지가 흩어지고, 초기 예산으로 감당 불가. 런칭 시 1종에 집중하는 전략이 필수인데, 이 결정이 되어있는지 확인.

**우리 관점:** 제품 특성상 "자일리톨 쿨링 미스트"가 히어로 후보. 이유 = 운동 직후 사용 장면이 가장 직관적, 가격대 낮아 진입장벽 저, 미스트는 SNS 비주얼에 유리(분사 장면).

---

### 질문 2: 리소스 배분 기준

**질문:** "플립사이드와 브렐라 중 올해 어디에 더 집중하실 건가요? 마케팅 예산과 시간을 100이라 하면 어떻게 나누실 생각이세요?"

**검증 가설:** 2개 브랜드를 동시에 키우려는 욕심이 있을 것. 현실적으로 1인(또는 극소규모)이 2개 브랜드를 동시에 마케팅하는 것은 불가능에 가까움. 제안서에서 "한 브랜드 집중 → 나머지는 최소 유지" 구조를 제시해야 하므로, 대표의 우선순위를 먼저 확인.

---

### 질문 3: 판매 채널과 가격 전략

**질문:** "자사몰과 스마트스토어 중 어디를 메인 판매 채널로 생각하세요? 그리고 제품 가격대는 어느 정도인가요?"

**검증 가설:** 초기 D2C 브랜드가 자사몰과 스마트스토어를 동시에 운영하면 트래픽이 분산됨. 가격대에 따라 마케팅 전략이 완전히 달라짐 (프리미엄 바디케어 vs. 가성비 바디케어). 올리브영 입점 계획이 있는지도 중요 — 있다면 마케팅 방향이 크게 달라짐.

---

### 질문 4: 대표 개인 브랜딩 의향

**질문:** "스레드를 이미 운영하고 계시는데, 브렐라 마케팅에서 대표님 개인의 얼굴과 이야기를 전면에 내세우는 것에 대해 어떻게 생각하세요?"

**검증 가설:** 대표 개인 스레드(@hohoy_j) 운영 = 퍼스널 브랜딩에 거부감이 적을 것. 초기 D2C 뷰티 브랜드에서 "창업자 스토리"는 가장 강력한 초기 콘텐츠. "운동하는 창업자가 자기 피부 고민을 해결하려고 만든 제품"이라는 서사가 가능한지 확인. 만약 대표 본인이 운동을 하는 사람이 아니라면 서사가 달라짐.

---

### 질문 5: 런칭 목표 수치

**질문:** "6월 런칭 후 첫 3개월 동안 '이 정도면 성공이다'라고 생각하시는 수치가 있으세요? 매출, 팔로워, 리뷰 수 등."

**검증 가설:** 구체적 목표 수치가 없을 가능성이 높다. "인지도 향상"이라는 추상적 목표만 있을 것. 우리가 제안서에서 현실적 KPI를 제시하려면(예: 런칭 3개월 팔로워 1,000명, 첫 달 매출 OO만원), 대표의 기대치 수준을 먼저 파악해야 함. 기대치가 비현실적이면 조정이 필요하고, 너무 낮으면 더 높은 목표를 제시할 수 있음.

## 종합 진단: 희망 서비스 vs 진짜 니즈

| 구분 | 내용 |
|------|------|
| **대표가 요청한 것** | 인지도 향상, 팔로워 확보, 마케팅 방향 컨설팅 |
| **진짜 필요한 것** | (1) 프리런칭 GTM 설계도 (2) 히어로 제품 결정 (3) 타겟 세분화 (4) 2개 브랜드 리소스 배분 (5) 런칭 후 3개월 실행 로드맵 |
| **갭 크기** | 중간~큼. 요청과 니즈가 같은 방향이지만, 요청은 "결과"이고 니즈는 "구조". |
| **갭의 본질** | "팔로워 늘리기"를 원하지만, 그 전에 "누구에게, 무슨 메시지로, 어떤 순서로"가 정리되어야 함 |

### 핵심 솔루션 1줄

> "6월 런칭까지 2개월, 콘텐츠 대행이 아니라 '프리런칭 GTM 설계 + 실행 코칭'을 팔아라. 히어로 제품 1개 + 타겟 커뮤니티 3곳 + 8주 타임라인이 핵심 산출물."

### 제안 방향 (플립사이드 통합)

플립사이드와 브렐라를 별도 프로젝트로 제안하되, 통합 할인 또는 패키지 구조 고려:

| 구분 | 플립사이드 (향수) | 브렐라 (바디케어) |
|------|------------------|------------------|
| 단계 | 제품 있음, 마케팅 미착수 | 프리런칭, 6월 출시 |
| 시급도 | 중간 (언제든 시작 가능) | 높음 (6월 데드라인) |
| 우선순위 제안 | 2순위 (최소 유지) | 1순위 (집중) |
| 서비스 | 브랜드 포지셔닝 + 콘텐츠 로드맵 | 프리런칭 GTM + 런칭 캠페인 |
