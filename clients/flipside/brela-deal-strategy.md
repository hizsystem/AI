## 0. Executive Summary

브렐라는 플립사이드(라이프 프래그런스) 대표 정호정이 새로 런칭하는 운동 후 바디케어/스킨케어 브랜드다. 6월 런칭 예정이며 현재 제품 개발은 완료(세라마이드 올인원 클렌저, 자일리톨 쿨링 미스트, 레티노이드 바디 로션), 마케팅은 완전 미착수 상태. 상담은 이미 완료되었고, 스콥(인스타그램 채널 세팅 + 콘텐츠 기획 + 마케팅 설계 컨설팅)이 어느 정도 합의된 상태다.

핵심 판단: **Strong Go** -- 니즈-서비스 핏이 높고, 런칭 2개월 전 골든타임이라는 긴급성이 명확하며, 플립사이드와 번들 시 객단가 확대 가능성이 크다.

## 1. 기회 적합도 평가

### 점수표

| 항목 | 가중치 | 점수 | 근거 |
|------|--------|------|------|
| 니즈-서비스 핏 | 30% | **5/5** | 인스타 채널 세팅 + 콘텐츠 기획 + 마케팅 설계 = 우리 코어 서비스 그 자체. 마케팅을 전혀 모르는 상태이므로 의존도 높음 |
| 예산 적합성 | 20% | **3/5** | 플립사이드 월 100만원이었으므로 브렐라도 유사 수준 추정. 단, 두 브랜드 번들 시 월 200만원+ 가능. 런칭 시점 일시 투자도 가능성 있음 |
| 의사결정 구조 | 15% | **5/5** | 대표 직접 상담, 즉시 의사결정 가능. 내부 승인 프로세스 없음 |
| 긴급도 | 15% | **5/5** | 6월 런칭까지 2개월. 인스타 채널 세팅, 프리런칭 콘텐츠, 런칭 캠페인 모두 지금 시작해야 함 |
| 성장 잠재력 | 10% | **4/5** | 운동+바디케어 교차점은 성장 카테고리. 바디케어 시장 연평균 7.5% 성장. 제품 확장 시 업무 범위 확대 여지 |
| 레퍼런스 가치 | 10% | **4/5** | 운동/피트니스 x 바디케어라는 명확한 포지셔닝은 포트폴리오 차별화. 0-to-1 런칭 케이스 |

### 종합 점수: **4.4/5.0 -- Strong Go**

가중 계산: (5x0.3) + (3x0.2) + (5x0.15) + (5x0.15) + (4x0.1) + (4x0.1) = 1.5 + 0.6 + 0.75 + 0.75 + 0.4 + 0.4 = **4.4**

플립사이드 단독(3.5, Conditional Go) 대비 브렐라는 훨씬 높은 점수. 차이를 만드는 요인은:
- 명확한 런칭 데드라인 = 긴급도 5점
- 마케팅 완전 미착수 = 니즈-서비스 핏 5점
- 운동 후 바디케어라는 뾰족한 포지셔닝 = 레퍼런스 가치 상승

## 2. 시장 컨텍스트 (상담 후 제안서에 활용)

### 바디케어 시장 성장성
- 글로벌 바디케어: 기초 화장품 내 **연평균 7.5% 성장**으로 가장 높은 성장률 (2021-2026)
- 한국 스킨케어 시장: 2021년 9.7조원 -> 2026년 12.8조원 (연평균 5.6%)
- 바디케어 핵심 트렌드: "몸도 얼굴처럼 스킨케어" -- 세라마이드, 히알루론산, 레티놀 등 고기능 성분이 바디로 확장

### 운동 x 바디케어 교차점
- 2026년 웰니스/뷰티 트렌드: 하이퍼 감각 케어(Hyper Sensory) + 고기능 멀티 제품
- 멘톨/쿨링 성분: 운동 후, 야외 활동 후 빠른 개운함을 원하는 3040 세대 생활 패턴과 맞아떨어지는 성장 카테고리
- 세라마이드 시장: 2025년 3.6억 달러 -> 2033년 6.0억 달러 (CAGR 6.69%)

### 경쟁 환경 초기 판단
- 직접 경쟁: 운동 후 바디케어 전문 브랜드는 한국에 거의 없음 -> 포지셔닝 선점 기회
- 간접 경쟁: 일반 바디케어(아모레퍼시픽, LG생활건강, 글람스), 스포츠 뷰티(해외: 각질관리/데오드란트 중심)
- 벤치마크: Megababe(미국, 운동+바디케어), Sol de Janeiro(브라질, 바디 중심 프리미엄), 국내 인디 바디케어 브랜드

### 브렐라 포지셔닝 기회
"운동 후 피부 회복"이라는 명확한 사용 상황(Use Case)을 가진 국내 유일의 바디케어 브랜드. 이 포지셔닝이 살아 있으면 콘텐츠 기획이 매우 쉬워진다 -- 모든 콘텐츠의 시작점이 "운동 후"라는 장면에서 출발하기 때문.

## 3. 3-Tier 패키징 설계

### 전제
- 확정 스콥: 인스타그램 채널 세팅 + 콘텐츠 기획 + 마케팅 설계 컨설팅
- 기간: 프리런칭(4-5월) + 런칭(6월) = 최소 3개월
- 대표님이 자사몰 직접 운영 -> 자사몰 마케팅은 스콥 외
- 콘텐츠 제작 실행은 대표님 or 인하우스 (우리는 기획/디렉팅)

### Tier 1: Launch Starter (런칭 세팅)

| 항목 | 내용 |
|------|------|
| **대상** | "일단 방향만 잡아주세요" |
| **포함** | 인스타 채널 세팅 (프로필/하이라이트/비주얼 가이드) + 브랜드 포지셔닝 1장 정리 + 3개월 콘텐츠 로드맵 + 런칭 D-day 플랜 |
| **방식** | 1회성 프로젝트 (2-3주) |
| **미팅** | 킥오프 1회 + 중간점검 1회 + 최종 납품 1회 |
| **가격대** | 150-180만원 (1회) |
| **적합** | 예산 제한적이나 방향은 반드시 필요한 경우 |

### Tier 2: Launch Partner (런칭 동반) -- 추천

| 항목 | 내용 |
|------|------|
| **대상** | "런칭까지 같이 가주세요" |
| **포함** | Tier 1 전체 + 월 2회 코칭콜(콘텐츠 디렉팅) + 월간 콘텐츠 기획안 제공(8-10개) + 릴스 기획 2개/월 + 런칭 캠페인 설계 + 마케팅 전체 로드맵 |
| **방식** | 월 리테이너 (최소 3개월: 4-6월) |
| **미팅** | 격주 1회 정기 코칭콜 |
| **가격대** | 월 150-180만원 (3개월 = 450-540만원) |
| **적합** | 런칭 전후 지속적 디렉팅이 필요한 경우 |

### Tier 3: Launch + Growth (런칭 + 성장)

| 항목 | 내용 |
|------|------|
| **대상** | "런칭부터 초기 성장까지 전부" |
| **포함** | Tier 2 전체 + 콘텐츠 일부 제작(월 4개 피드 + 릴스 2개) + 시딩/인플루언서 캠페인 설계 + 퍼포먼스 리포트(월간) |
| **방식** | 월 리테이너 (최소 3개월) |
| **미팅** | 주 1회 정기 미팅 |
| **가격대** | 월 280-350만원 (3개월 = 840만원-1,050만원) |
| **적합** | 인하우스 제작 인력 없이 전면 위탁 원하는 경우 |

### 플립사이드 번들 옵션

| 구성 | 내용 | 가격대 |
|------|------|--------|
| 듀얼 브랜드 코칭 | 플립사이드(향수) + 브렐라(바디케어) 동시 코칭. 공통 전략 세션 + 브랜드별 콘텐츠 디렉팅 | 월 220-260만원 (개별 대비 20-25% 할인) |
| 듀얼 브랜드 풀 | 두 브랜드 모두 Tier 2 수준 | 월 280-320만원 (개별 대비 25-30% 할인) |

번들의 핵심 셀링 포인트: "두 브랜드가 같은 대표 아래 있으므로, 브랜드 간 크로스 프로모션 전략도 설계 가능. 향수(플립사이드) x 바디케어(브렐라) = 라이프스타일 시너지."

## 4. Challenger Teaching 포인트

상담은 이미 완료되었으므로, 이 포인트들은 **후속 제안서/메일에 녹여야 할 인사이트**다.

### Teaching 1: "운동 후 바디케어는 제품이 아니라 '루틴'을 파는 것"

> 세라마이드 클렌저 -> 쿨링 미스트 -> 레티노이드 로션. 이미 3-step 루틴이 완성되어 있다. 개별 제품을 따로 파는 게 아니라 "운동 후 10분 스킨 리커버리 루틴"이라는 패키지로 포지셔닝하면, 콘텐츠 기획이 쉬워지고 객단가도 올라간다.
>
> 벤치마크: CeraVe의 "AM/PM 루틴" 콘텐츠가 틱톡에서 1억+ 조회. 루틴 콘텐츠는 저장/공유율이 개별 제품 소개 대비 2-3배.

### Teaching 2: "런칭 전 2개월이 런칭 후 6개월보다 중요하다"

> 바디케어 인디 브랜드의 가장 흔한 실수: 제품 나온 뒤에야 인스타 시작. 이미 런칭일에 팔로워 0명, 콘텐츠 0개 상태. 프리런칭 기간(4-5월)에 "브랜드 빌딩 콘텐츠"를 쌓아야 런칭일에 첫 구매가 발생한다.
>
> 프리런칭 콘텐츠 예시:
> - 대표님의 운동 루틴 + "왜 이 브랜드를 만들었는지"
> - 제품 개발 비하인드 (세라마이드 선택 이유, 쿨링 미스트 테스트 과정)
> - 운동 후 피부 문제 공감 콘텐츠 (설문, 인터뷰)

### Teaching 3: "3040 운동인의 인스타 행동 패턴은 2030 뷰티 유저와 다르다"

> 3040 운동 라이프스타일 타겟은 "예쁜 피드"보다 "진짜 정보"에 반응한다. 저장률이 높은 콘텐츠 유형:
> - 운동 후 피부 트러블 원인 설명 (교육형)
> - Before/After 루틴 비교 (증거형)
> - 성분 해설 (세라마이드가 뭔데? 왜 운동 후에 필요한데?)
>
> 좋아요 수보다 **저장 수와 DM 문의 수**가 초기 KPI. 팔로워 1,000명이라도 저장률 10%면 성공.

## 5. 예상 반론 & 대응

### "런칭하고 나서 반응 보고 마케팅 시작하려고요"

**대응**: "런칭일에 인스타 팔로워 0명, 콘텐츠 0개면 자사몰 트래픽이 발생할 루트가 없습니다. 제품이 아무리 좋아도 '발견'이 안 되면 판매가 0입니다. 프리런칭 2개월은 런칭일 매출을 결정하는 기간이에요. 지금 시작하면 런칭일까지 최소 20-30개 콘텐츠 + 팔로워 500-1,000명 확보가 가능합니다."

### "예산이 많지 않아서요"

**대응**: "그래서 대행이 아니라 코칭형을 제안드리는 겁니다. 콘텐츠 제작은 대표님이 직접, 기획과 방향은 저희가. 월 150만원이면 매일 카페 라떼 한 잔 값인데, 이걸로 런칭 3개월간의 마케팅 설계도를 얻는 거예요. 런칭 후 직접 운영할 수 있는 체계까지 세워드립니다."

### "직접 해볼게요, 인스타 좀 공부하면 되지 않을까요"

**대응**: "물론 직접 하실 수 있어요. 다만 런칭까지 2개월인데, 마케팅 공부 + 채널 세팅 + 콘텐츠 기획 + 제작을 동시에 하시면 제품 준비에 집중하기 어려워집니다. 저희가 '마케팅 설계도'를 만들어드리면, 대표님은 설계도대로 실행만 하면 됩니다. 3개월 후에는 혼자 운영하실 수 있게 체계를 잡아드리는 게 목표예요."

### "효과가 있을까요? 팔로워가 늘까요?"

**대응**: "솔직히 말씀드리면, 3개월 만에 팔로워 1만 명은 어렵습니다. 하지만 '진짜 운동하는 사람 500-1,000명'이 팔로우하고, 그 중 10%가 런칭일에 구매하면 첫 달 매출이 발생합니다. 초기 브랜드는 팔로워 수가 아니라 '첫 100명의 팬'이 전부예요. 그 100명이 리뷰를 쓰고, 친구에게 추천하고, 재구매하는 구조를 만드는 게 진짜 마케팅입니다."

### "플립사이드도 같이 하면 좀 깎아주세요"

**대응**: "번들 패키지를 준비했습니다. 두 브랜드를 개별로 진행하면 월 300만원+ 인데, 번들이면 월 220-260만원 수준. 공통 전략 세션에서 시간을 절약하고, 브랜드 간 크로스 프로모션도 설계해드립니다. 향수(플립사이드) x 바디케어(브렐라) = '운동하는 사람의 라이프스타일 브랜드'라는 상위 포지셔닝도 가능해요."

## 6. 클로징 전략

### 상황 정리
- 상담 완료 (4/6 대면)
- 스콥 대략 합의: 인스타 채널 세팅 + 콘텐츠 기획 + 마케팅 설계 컨설팅
- 플립사이드 상담(4/8 콜)도 예정되어 있음
- 6월 런칭 데드라인이라는 자연스러운 긴급성 존재

### 액션 플랜

**Day 0 (4/6, 오늘)**: 감사 메시지 발송
- 채널: 문자 or 카톡
- 톤: 가볍고 진정성 있게
- 내용: "오늘 미팅 감사합니다. 브렐라 제품 컨셉이 정말 뾰족해서 인상적이었어요. 운동 후 바디케어라는 포지셔닝, 콘텐츠로 풀어내면 확실히 먹힐 겁니다. 이번 주 중으로 정리해서 제안 드릴게요."

**Day 1-2 (4/7-8)**: 1장짜리 방향 요약 발송
- 형식: PDF 또는 깔끔한 메일
- 내용:
  - 브렐라 포지셔닝 핵심 1줄
  - 프리런칭(4-5월) -> 런칭(6월) 2단계 로드맵 요약
  - 인스타 채널 세팅 체크리스트 (미리보기)
  - "자세한 견적은 플립사이드 상담 후 번들 포함해서 함께 보내드릴게요"

**Day 2-3 (4/8-9)**: 플립사이드 상담 완료 후 통합 제안
- 브렐라 단독 제안 (Tier 2 추천)
- 플립사이드 단독 제안
- 듀얼 브랜드 번들 옵션
- 3가지를 한 제안서에 담아 비교 가능하게

**Day 3-5 (4/9-11)**: 의사결정 촉진
- "4월 중순부터 시작해야 6월 런칭에 맞출 수 있습니다"
- "프리런칭 콘텐츠 쌓는 데 최소 6-8주 필요"
- 데드라인 활용: 자연스럽게 긴급성 환기, 압박은 NO

**Day 7 (4/13)**: 미응답 시 가볍게 팔로업
- "혹시 견적 관련 궁금한 점 있으실까요?"
- "4월 셋째 주부터 시작하면 런칭 맞추기 빠듯할 수 있어서요"

### 의사결정 촉진 레버

1. **시간 레버**: 6월 런칭까지 2개월. 프리런칭 콘텐츠 준비 기간 고려하면 4월 중순이 실질적 데드라인
2. **번들 레버**: 플립사이드와 함께하면 할인 + 시너지. 개별보다 묶는 게 합리적이라는 구조
3. **리스크 레버**: "런칭일에 팔로워 0명이면?" -- 이 질문을 던지되, 겁주는 게 아니라 현실 인식 차원
4. **신뢰 레버**: 상담에서 이미 라포 형성. 대표님이 마케팅을 모른다고 솔직히 말한 상태이므로 전문성 신뢰 기반 확보

### 최적 시나리오

플립사이드 상담(4/8) 완료 -> 통합 제안서 발송(4/9) -> 의사결정(4/11-14) -> 킥오프(4/14-18 주)

**추천 패키지**: 브렐라 Tier 2 (Launch Partner) + 플립사이드 코칭형 번들 = 월 220-260만원 x 3개월

## 7. 제안서 작성 시 반영할 핵심 프레이밍

### SCQA 프레이밍

- **S (Situation)**: 운동 후 바디케어라는 명확한 포지셔닝의 3개 제품이 준비되었고, 6월 런칭 예정
- **C (Complication)**: 마케팅 경험 제로, 인스타 채널 미세팅, 런칭까지 2개월밖에 안 남음
- **Q (Question)**: 2개월 안에 인스타 채널을 세팅하고, 런칭일에 실제 구매가 발생하는 구조를 어떻게 만들 것인가?
- **A (Answer)**: 프리런칭(4-5월) 브랜드 빌딩 + 런칭(6월) 전환 캠페인의 2단계 설계. 코칭형으로 대표님 직접 운영 체계까지 구축

### 제안서에 꼭 들어갈 것
1. 프리런칭 -> 런칭 2단계 타임라인 (비주얼)
2. 인스타 채널 세팅 Before/After 이미지 (포트폴리오)
3. 콘텐츠 기획 샘플 3개 (운동 후 루틴, 성분 교육, 대표 스토리)
4. 팔로워/저장/DM 목표 KPI (현실적 수치)
5. 3개월 후 자체 운영 가능한 상태의 정의
