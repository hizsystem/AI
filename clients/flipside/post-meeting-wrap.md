## 1. 미팅 핵심 요약 (3줄)

1. 정호정 대표는 플립사이드(향수) 외에 **운동 후 피부 컨디션 개선 바디케어 브랜드**를 새로 준비 중이며, 이번 상담의 주제는 향수가 아니라 이 신규 바디케어 브랜드였다.
2. 6월 1차 런칭(쿨링 미스트) 예정이나 자사몰 미완성, 콘텐츠 자산 없음, 인스타 초기 상태 — **런칭 전 2개월간 마케팅 기반을 잡아야 하는 상황**.
3. 마케팅 예산 최대 200만원으로 인스타 채널 세팅 + 콘텐츠 기획 + 전체 설계 컨설팅을 원하며, 수요일(4/8)까지 진단+제안 발송을 약속했다.

## 2. 사전 가설 vs 실제 확인

| 사전 가설 (향수 브랜드 기준) | 상담에서 확인된 내용 | 판정 |
|---|---|---|
| 라이프 프래그런스(향수) 브랜드가 상담 주제 | **바디케어 신규 브랜드**가 상담 주제. 향수는 별도 운영 중 | **수정** |
| 월 예산 100만원 | **최대 200만원** (런칭 전 마케팅 기준) | **수정 (상향)** |
| 브랜드 포지셔닝 미정립 | 타겟(30~40대 운동하는 남녀)과 컨셉(운동 후 피부)은 있으나 **언어화/체계화 미흡** | **부분 일치** |
| Go-to-Market 설계도 부재 | 맞음. 자사몰 미완, 인스타 초기, 오프라인 구상만 있음 | **일치** |
| SNS 존재감 미미 | 맞음. 인스타 개설했으나 지인 팔로워만, 매거진 콘텐츠 4~5개 성과 없음 | **일치** |
| 콘텐츠 제작 대행 요청 추정 | 대행보다는 **기획 디렉팅 + 컨설팅** 원함. 자사몰도 직접 하겠다고 | **수정** |
| 1인 또는 소규모 팀 추정 | 상담에서 팀 구성 미확인. 1인 추정 유지 | **미확인** |

### 신규 발견 사항
- 정호정 대표는 향수(플립사이드)와 바디케어를 **별도 브랜드**로 운영. 바디케어 브랜드명은 상담에서 미확인
- 제품 3종 중 내용물 개발 지연으로 **쿨링 미스트만 6월 1차 런칭**, 나머지(올인원 클렌저, 레티놀 세럼)는 하반기
- 1차 타겟 = **러닝족**, 2차 = 헬스/사이클/아웃도어. 운동 커뮤니티 오프라인 접근 구상 중
- 자사몰 카페24 유료 플랜, 4월말~5월초 직접 제작 예정
- 제품 이미지 AI로 제작한 버전 있으나 쿨링 미스트용은 없음 — 실제 샘플 나오면 촬영 별도 계획

## 3. 대표 온도 평가

- **관심도**: 4/5 — 36분 내내 집중, 구체적 질문 다수, "좋은 정보 얻었다" 반응
- **긴급도**: 4/5 — 6월 런칭 확정, 역산하면 지금 시작해야 함. 본인도 인지하고 있음
- **예산 의지**: 3/5 — 200만원 상한선 명확. 넘길 의사는 아직 없어 보임. 단, 향수 브랜드 별도 운영 중이므로 전체 가용 예산은 더 있을 수 있음
- **수주 확률**: **70%** — 런칭 시점이 가까워 파트너가 필요한 상황, 상담 중 신뢰 형성 양호, 다만 200만원 예산 내 만족할 만한 패키지를 제안할 수 있는지가 관건
- **대표 의사결정 스타일**: **분석형 + 실행형** 혼합 — 질문이 구체적이고 채널/전략을 직접 이해하려 함. 위임보다 참여 선호. 카페24 직접 제작하겠다는 점에서 실행력도 있음
- **핵심 관심사**: "인스타그램을 어떻게 키우지?" — 인스타 성장이 가장 많이 언급되고 반응한 주제

기존 3-Tier 구조가 아닌, 대표가 직접 언급한 스콥에 맞춘 커스텀 패키지가 적절하다.

### 200만원 패키지 구성 방안

| 항목 | 내용 | 비중 |
|------|------|------|
| 브랜드 포지셔닝 정리 | 타겟 정의, 경쟁 차별화, 핵심 메시지 1장 | 20% |
| 인스타그램 채널 설계 | 프로필/하이라이트/피드 그리드 구조, 콘텐츠 카테고리 | 25% |
| 런칭 콘텐츠 기획 | 런칭 전 4주 + 런칭 후 2주 콘텐츠 캘린더, 릴스 기획안 | 30% |
| GTM 로드맵 | 자사몰/스마트스토어/오프라인 채널 우선순위, 런칭 시퀀스 | 15% |
| 자사몰 체크 | 카페24 제작 가이드라인 + 완성 후 리뷰 1회 | 10% |

기간: 4월 중순 ~ 5월 말 (약 6주)
산출물: 브랜드 포지셔닝 1장 + 채널 설계서 + 콘텐츠 캘린더 + GTM 로드맵
코칭콜: 격주 1회 (총 3회)

### 가격 민감도
- 200만원이라는 숫자를 대표가 먼저 제시. 상한선이 명확함
- "200만원 안에서 최선의 그림" 합의 — 이 금액을 넘는 제안은 역효과
- 200만원 일시불이 아닌 2회 분할(착수 100 + 중간 100) 제안이 심리적 부담 줄일 수 있음

## 5. 리스크 & 기회

### 기회 요소
- 6월 런칭이라는 명확한 데드라인 → 의사결정이 빠를 수밖에 없음
- 대표가 직접 실행하는 스타일 → 코칭형 모델이 잘 맞음, 산출물 활용도 높을 것
- 향수 브랜드(플립사이드) 운영 경험 있음 → 완전 초보는 아님, 학습 속도 빠를 것
- 바디케어 시장(운동+스킨케어 교집합)은 성장 세그먼트 → 레퍼런스 가치 있음
- 상담 중 신뢰 형성 양호 — "편하게 연락하세요" 마무리에 긍정 반응

### 리스크 요소
- 200만원 단발 프로젝트로 끝날 가능성 — 리테이너 전환 불확실
- 제품 내용물 개발 지연 이력 → 런칭 일정 밀릴 수 있음, 그러면 우리 작업도 공중에 뜸
- 제품 이미지/비주얼 자산이 거의 없음 → 콘텐츠 기획만 하고 실제 제작은 막히는 상황 우려
- 바디케어 브랜드명 미확인 — 브랜딩 기초가 덜 갖춰진 상태일 수 있음

### 불확실 요소
- 팀 구성: 혼자 하는 건지 팀원이 있는지 미확인 → 실행 역량에 영향
- 향수 브랜드와 바디케어 브랜드의 관계: 같은 법인인지, 별도인지
- 6월 런칭 확정도: 내용물 지연 상황에서 실제로 6월에 나오는 건지
- 오프라인 채널(헬스장, 요가원) 접근은 어느 수준까지 구체화되어 있는지

## 내부 메모

이번 상담의 가장 큰 반전은 **향수가 아니라 바디케어 브랜드가 주제**였다는 점이다. 사전 prep은 향수 기준으로 작성했기 때문에 시장/경쟁 분석은 쓸 수 없지만, 니즈 분석(GTM 부재, 포지셔닝 미정립)과 대표 성향 파악은 거의 그대로 적용 가능했다.

200만원이라는 예산은 리테이너로 보면 작지만, "런칭 전 2개월 준비"라는 프레이밍으로 가면 적절한 프로젝트 단가다. 핵심은 이 단발 프로젝트를 잘 마무리해서 런칭 이후 월 리테이너(콘텐츠 디렉팅/코칭)로 전환하는 것이다.

대표가 "인스타를 어떻게 키우지?"를 핵심 관심사로 두고 있으므로, 제안서에서 인스타 성장 로드맵을 가장 앞에 배치하고, 나머지(포지셔닝, GTM)는 그 토대로 엮는 구조가 설득력 있을 것이다.

플립사이드(향수) 쪽도 별도 기회다. 이번에 바디케어로 좋은 관계를 만들면 향수 브랜드도 자연스럽게 연결될 수 있다. 다만 지금은 바디케어에만 집중하는 게 맞다.
