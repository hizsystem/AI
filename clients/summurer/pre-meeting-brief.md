## 1-Page 요약 (상담 5분 전에 읽기)

### 브랜드 현재 상황
제조/생산 업종의 초기 브랜드. 웹사이트, SNS, 커머스 채널 등 온라인 디지털 프레즌스가 검색 엔진에서 전혀 확인되지 않는 상태. "summurer"라는 브랜드명 자체가 검색 결과에 노출되지 않아, 런칭 전이거나 극초기 단계로 추정.

### 브랜드 현재 고민
- 표면: "촬영" — 제품 촬영(상세페이지, 커머스용 이미지)이 가장 급한 과제로 보임
- 추정 근본: 제품은 있으나 판매 채널에 올릴 비주얼 콘텐츠가 부재. 촬영 이전에 브랜드 아이덴티티, 패키지 디자인, 판매 채널 구축이 선행되어야 할 가능성 높음

### 마케팅 행보
- 디지털 성숙도: 0~1/5 (미존재~미개척) — 웹사이트, SNS, 커머스 채널 모두 미확인
- 월 마케팅 예산: 100만원 (극소규모)
- 희망 서비스: IMC 마케팅 (굿즈, 캠페인, 인플루언서, 리뷰, 광고 제작 등) — 예산 대비 희망 범위가 매우 넓음
- 기대: 파일럿 테스트를 통한 빠른 매출 성과 검증

### 필수 질문 Top 5
1. "서머러가 어떤 제품을 만드는 브랜드인지 직접 소개해주실 수 있나요?" — 의도: 제품 카테고리/타겟 확인 (온라인에서 정보를 전혀 찾을 수 없음)
2. "현재 제품이 어디서 팔리고 있나요? 자사몰, 스마트스토어, 쿠팡 등 채널이 있으신가요?" — 의도: 판매 인프라 현황 파악
3. "'촬영'이 가장 급하다고 하셨는데, 지금 제품 이미지가 전혀 없는 상태인가요?" — 의도: 현재 콘텐츠 자산 레벨 파악
4. "월 100만원 예산에서 가장 먼저 해결하고 싶은 한 가지는 무엇인가요?" — 의도: 예산 대비 우선순위 좁히기
5. "파일럿 테스트로 검증하고 싶은 것이 '이 제품이 팔리는가'인지, '이 채널이 맞는가'인지?" — 의도: PMF 검증 단계인지 채널 최적화 단계인지 구분

### 예상 프로젝트 범위

| 티어 | 범위 | 기간 | 가격대 |
|------|------|------|--------|
| Starter | 제품 촬영 디렉팅 + 상세페이지 1종 | 2주 | 150만~250만 |
| **Essential** | **촬영 + 스마트스토어 세팅 + 상세페이지 + 기본 SNS 세팅** | **4주** | **400만~600만** |
| Growth | 촬영 + 커머스 세팅 + SNS 운영 1개월 + 소액 광고 테스트 | 6주 | 800만~1,200만 |

> 주의: 월 100만원 예산 기준으로는 Starter도 부담될 수 있음. 상담 시 예산 현실성 확인 필수.

**핵심 미확인 사항**: "서머러"가 무엇을 만드는 회사인지 온라인 검색으로 전혀 파악 불가. 브랜드명의 어원도 불명확(summer + 접미사? 조어?). 상담 시 반드시 첫 질문으로 확인해야 함.

|------|------|------|

| 네이버 스마트스토어 | 미확인 | 검색 결과 없음 |

| 블로그/리뷰 | 노출 없음 | 검색 결과 0건 |

**디지털 프레즌스 종합 점수: 0/10**
브랜드명 "summurer"로 구글, 네이버, 인스타그램 등 어떤 플랫폼에서도 관련 결과가 나오지 않음. 이는 두 가지 중 하나를 의미:
- (A) 아직 런칭 전이거나 제품만 있고 온라인 진출을 하지 않은 상태
- (B) 다른 브랜드명/채널명을 사용 중이며 "summurer"는 새 브랜드명

### 3단계: 대표자 프로필

| 항목 | 내용 |
|------|------|
| 이름 | 이윤수 |
| 직책 | 대표/CEO |
| 이력 | 확인 필요 (동명이인 다수 — 가수, 정치인, 교수 등 검색되나 서머러 관련 인물은 미확인) |
| 인터뷰/기고 | 없음 |
| SNS | 확인 필요 |

**주목 포인트**:
- 오프라인 강의(브랜드라이즈)에 직접 참석 → 마케팅/브랜딩에 대한 학습 의지 있음
- 대표가 직접 상담 신청 → 의사결정자가 곧 실무자일 가능성 높음 (1인 또는 극소규모)
- 메일 회신 없음 → 관심도가 낮거나, 메일 확인 습관이 없거나, 아직 구체적 니즈를 정리하지 못한 상태

### 4단계: 시장/경쟁 환경

제품 카테고리가 불명확하여 시장 분석 불가. 상담 시 제품을 확인한 후 후속 리서치 필요.

**상담 시 파악해야 할 시장 정보**:
- 제품 카테고리 (식품? 화장품? 리빙? 패션? 소품?)
- B2C vs B2B
- 온라인 vs 오프라인 판매 비중
- 제조 방식 (자체 생산 vs OEM/ODM)
- 현재 매출 규모 및 주요 판매 채널

### 5단계: 초기 가설 수립

**브랜드 성숙도: 1/5 (태동기)**
- 브랜드명은 존재하나, 온라인에서 인지도/가시성이 전무
- 브랜드 아이덴티티(로고, 톤앤매너, 메시징) 정립 여부 불명
- 제품은 있으나(제조/생산 업종) 브랜드로서의 체계는 미구축 추정

**마케팅 성숙도: 0~1/5 (미착수)**
- 디지털 채널 부재
- 마케팅 예산 월 100만원 (극소)
- 희망 서비스(IMC)와 예산 사이 큰 괴리

**핵심 과제 가설 3가지**:

1. **제품→브랜드 전환이 안 되어 있다**
   - 제조는 하고 있으나 "브랜드"로 포장되지 않은 상태. 촬영이 급하다는 것은 제품 비주얼 자산이 없다는 뜻이며, 이는 곧 커머스 진입 자체가 안 되어 있을 가능성.

2. **판매 채널이 없거나 극초기다**
   - 온라인 어디에서도 판매 흔적이 없음. 오프라인(지인, 소매)으로만 소량 판매 중이거나, 이제 막 온라인 진출을 준비하는 단계.

3. **예산 대비 기대치 조율이 핵심**
   - 월 100만원으로 IMC(굿즈, 캠페인, 인플루언서, 리뷰, 광고)는 현실적으로 불가능. "파일럿 테스트"라는 키워드를 활용해, 가장 임팩트 있는 1가지(예: 스마트스토어 + 상세페이지)에 집중하는 방향으로 가이드 필요.

## 상담 전략 메모

### 리스크 요인
- **예산 미스매치**: 월 100만원 vs IMC 전체 → 기대 관리 실패 시 불만족 가능
- **정보 부족**: 제품이 무엇인지조차 모르는 상태에서 상담 → 준비 부족 느낌 줄 수 있음
- **메일 무응답**: 관여도가 낮을 수 있음 → 유선 연락으로 전환 권장
- **촬영 단건 의뢰 가능성**: 마케팅 전략이 아닌 촬영 대행만 원할 수 있음

### 기회 요인
- **대표가 직접 강의 참석**: 학습 의지 + 직접 의사결정 가능
- **제조 기반**: 제품이 이미 있다면 촬영→상세페이지→커머스→광고 순서로 빠른 매출 검증 가능
- **파일럿 테스트 마인드**: 작게 시작해서 검증하겠다는 태도는 건강한 스타트업 마인드
- **빈 캔버스**: 디지털 프레즌스가 0이므로, 처음부터 제대로 세팅해줄 수 있는 기회

### 상담 시 우선 확인 사항 (처음 10분)
1. 서머러가 만드는 제품이 무엇인지
2. 현재 어디서 팔고 있는지 (판매 채널)
3. "촬영"이라고 쓴 이유 — 구체적으로 무엇을 촬영하고 싶은지
4. 월 100만원이 마케팅 전체 예산인지, 외주 비용인지
5. 언제까지 어떤 결과를 원하는지 (타임라인)

### 추천 상담 톤
- 이 케이스는 "우리가 뭘 해줄 수 있는지" 보다 "대표님이 뭘 하고 계신지" 먼저 충분히 들어야 함
- 정보가 없는 상태이므로, 경청 → 질문 → 정리 → 제안 순서로 진행
- 예산이 적으므로 솔직하게 "이 예산으로 가능한 것/불가능한 것"을 구분해주는 것이 신뢰 구축에 유리
- 코치형 접근: "직접 하실 수 있는 부분은 가이드 드리고, 전문가가 필요한 부분만 저희가" 프레이밍
