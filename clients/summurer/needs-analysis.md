**대상**: 이윤수 대표 / 서머러(summurer)

## Step 1: 인테이크 데이터 분석

### 1-1. 폼 원문 정리

| 항목 | 내용 | 해석 노트 |
|------|------|-----------|
| 신청 경로 | 오프라인 강의 참석 | 직접 대면 경험 → 신뢰도 형성됨 |
| 대표님 성함 | 이윤수 | - |
| 회사/브랜드명 | 서머러(summurer) | 웹 검색 결과 없음 → 온라인 프레즌스 극초기 |
| 직책 | 대표/CEO | 의사결정권자 본인 |
| 업종 | 제조/생산 | 자체 제품 보유, 실물 상품 기반 |
| 마케팅 목표 | 매출 증대 및 전환율 개선 | "전환율"이라는 단어 사용 → 일정 수준 마케팅 용어 인지 |
| 페인 포인트 | "촬영" | 단 한 단어. 아래 상세 분석 |
| 희망 서비스 | IMC 마케팅 | 굿즈, 캠페인, 인플루언서, 리뷰, 광고 제작 등 → "다 해야 할 것 같다" 패턴 |
| 월 예산 | 100만원 | 최저 구간. 현실적으로 1-2개 시책만 가능 |
| 기대 성과 | 파일럿 테스트를 통한 빠른 매출 성과 검증 | 아직 검증 안 된 상태 |
| 메일 회신 | 없음 | 관심은 있으나 적극적 행동은 아직 안 함 |

### 1-2. "촬영" 한 단어의 가능한 해석

페인 포인트 칸에 "촬영"이라고만 적은 것은 다음 중 하나를 의미할 가능성이 높다:

1. **제품 촬영 자체가 안 되어 있다** — 상세페이지, SNS, 광고에 쓸 사진/영상이 없거나 부족
2. **촬영 비용이 부담이다** — 제조업 특성상 전문 촬영 비용(50-200만원/회)이 월 마케팅 예산과 맞먹음
3. **촬영 퀄리티가 낮다** — 셀프 촬영이나 저가 촬영으로 진행했으나 결과물이 전환에 도움 안 됨
4. **촬영이 마케팅의 선결 조건이라고 인식** — "촬영부터 해야 마케팅을 시작할 수 있다"는 병목 인식
5. **어떤 촬영을 해야 하는지 모른다** — 제품컷, 모델컷, 라이프스타일, 영상 중 뭘 해야 할지 판단 불가

**가장 유력한 해석**: 4번 + 1번 조합. "마케팅을 하고 싶은데, 촬영이 안 되어 있어서 시작조차 못 하는 상태."

### 1-3. 감정 톤 분석

- 폼 작성이 전반적으로 **최소한** → 바쁘거나, 마케팅에 익숙하지 않거나, 뭘 적어야 할지 모름
- "파일럿 테스트"라는 표현 → **조심스러움**, 대규모 투자에 대한 리스크 회피 성향
- 메일 회신 없음 → **아직 확신이 없거나**, 단순히 바쁘거나, 우선순위가 낮음
- 오프라인 강의 참석 → **능동적으로 배우려는 의지**는 있음

### 1-4. 웹 프레즌스 분석 결과

"서머러(summurer)" 브랜드로 검색한 결과:
- 공식 웹사이트: 발견되지 않음
- 인스타그램: 발견되지 않음
- 스마트스토어/쿠팡 등 커머스: 발견되지 않음
- 언론/블로그 노출: 없음

**해석**: 온라인 존재감이 제로에 가까운 극초기 브랜드이거나, 지금까지 B2B/오프라인 유통 위주로 운영해왔을 가능성. 이 경우 "촬영"이 병목이라는 인식이 더 설득력을 가짐 — D2C 전환을 시도하려는데 기본 에셋이 없는 상황.

## Step 2: SPIN 기반 니즈 레이어링

### Situation (객관적 현재 상태)
- 제조/생산 기반 자체 브랜드 보유
- 온라인 프레즌스가 거의 없는 상태 (검색 결과 제로)
- 월 마케팅 예산 100만원 (최저 구간)
- 대표가 직접 마케팅 의사결정 (전담 인력 없음 추정)
- 오프라인 강의를 통해 마케팅 학습 중
- 마케팅 실행 경험이 거의 없거나 초기 (파일럿 테스트 언급)

### Problem (대표가 인식한 문제)
- **직접 인식**: "촬영이 안 되어 있다"
- **간접 인식**: 매출이 원하는 수준이 아니다 (매출 증대 목표)
- **간접 인식**: 고객이 제품을 보고도 사지 않는다 (전환율 개선 목표)
- **간접 인식**: 마케팅을 다방면으로 해야 할 것 같다 (IMC 선택)

### Implication (방치 시 파급효과)
- 촬영 없이는 상세페이지/SNS/광고 소재 제작 불가 → 온라인 판매 채널 구축 자체가 불가
- 온라인 전환 지연 → 오프라인/B2B 의존 지속 → 매출 성장 천장
- 100만원 예산으로 IMC를 시도하면 모든 채널에서 중도반단 → "마케팅은 효과 없다" 학습 위험
- 경쟁사가 온라인 먼저 장악하면 후발 진입 비용 급증
- 대표가 직접 마케팅까지 하면 제품/생산에 집중 못함 → 본업 퀄리티 하락

### Need-Payoff (해결 시 가치)
- 촬영 에셋 확보 → 상세페이지/SNS/광고 동시 활용 가능 (1회 촬영, 다채널 활용)
- 핵심 채널 1개 집중 → 100만원으로도 의미 있는 테스트 가능
- 전환율 개선의 핵심은 "촬영 퀄리티 + 상세페이지 구성" → 가장 ROI 높은 투자
- 파일럿 성공 시 예산 증액의 근거 확보 → 성장 루프 시작

## Step 3: 고민 계층 분석 (Pain Hierarchy)

```
L4 근본 원인
    "D2C/온라인 판매로 전환하고 싶은데, 어디서부터 시작해야 할지 모른다"
    └── 제조업 대표의 전형적 디지털 전환 고민

  L3 전략적 문제
      "마케팅 채널, 메시지, 타겟 전략이 없는 상태에서
       촬영-콘텐츠-광고-전환의 전체 그림을 못 그린다"
      └── 마케팅 시스템 부재

    L2 전술적 문제
        "제품 사진/영상이 없어서 상세페이지, SNS, 광고를
         만들 수 없다. 만들어도 퀄리티가 낮아 전환이 안 된다"
        └── 콘텐츠 에셋 부재 → 전환율 저조

      L1 표면 요청
          "촬영"
          └── 가장 눈에 보이는, 즉각적인 병목
```

### 계층 간 연결 해석

**"촬영"은 빙산의 일각이다.**

이윤수 대표가 "촬영"이라고 적은 것은, 마케팅의 전체 프로세스 중 가장 물리적으로 보이는 단계를 지목한 것이다. 실제로는:

1. 촬영 → 상세페이지 → 전환율 개선 → 매출의 연쇄 구조에서
2. 첫 번째 블록(촬영)이 빠져 있어서 나머지가 전부 멈춰 있는 상태
3. IMC를 선택한 것은 "뭘 해야 할지 모르니까 다 선택"한 것
4. 100만원 예산은 "아직 확신이 없어서 최소한만 투자"하겠다는 신호

## Step 4: JTBD (Jobs to Be Done) 분석

### Functional Job (기능적 과업)
> "우리 제품을 온라인에서 매력적으로 보여줘서, 보는 사람이 구매하게 만들고 싶다."

- 제품 촬영 에셋 확보
- 전환율 높은 상세페이지 구축
- 온라인 판매 채널 활성화
- 투입 대비 매출 성과 검증

### Social Job (사회적 과업)
> "제조만 하는 회사가 아니라, 브랜드를 가진 회사로 인정받고 싶다."

- 제조업체 → 브랜드 기업으로 외부 인식 전환
- SNS/온라인에서 '있어 보이는' 브랜드 이미지 구축
- 강의에서 배운 것을 실제로 실행하는 대표로서의 자부심

### Emotional Job (감정적 과업)
> "마케팅을 어떻게 시작해야 할지 막막한 상태에서 벗어나고 싶다."

- 마케팅에 대한 불확실성/막막함 해소
- "일단 이것부터 하면 된다"는 명확한 방향 확보
- 작은 성과라도 눈으로 확인하여 확신 얻기
- 촬영이라는 구체적 단어를 꺼낸 것 자체가 "하나라도 확실한 것을 잡고 싶다"는 심리

## Step 5: 유사 사례 패턴 매칭

### 패턴: "제조업 대표 + 촬영이 병목 + 저예산 + 파일럿"

이 조합은 다음과 같은 전형적 시나리오를 나타낸다:

**[패턴 A] OEM/ODM → 자체 브랜드 전환 초기**
- 지금까지 B2B로 납품만 해왔는데, 자체 브랜드로 D2C 시작하려는 단계
- 제품은 있지만 소비자 대상 마케팅 에셋이 전무
- "촬영"은 B2C 전환의 가장 첫 번째 관문
- 예산이 낮은 이유: 아직 D2C 매출이 없어서 마케팅 투자 여력이 없음

**[패턴 B] 소규모 자체 브랜드, 오프라인/지인 판매 위주**
- 스마트스토어나 자사몰은 있지만 유입/전환이 거의 없음
- 제품 사진이 셀프 촬영 수준이라 상세페이지가 경쟁력 없음
- "촬영만 제대로 하면 팔릴 텐데"라는 가설을 가지고 있음

**[패턴 C] 신규 브랜드 런칭 준비 단계**
- 제품 개발은 완료했지만 아직 시장 출시 전
- 런칭에 필요한 에셋(촬영, 상세페이지, SNS)이 전무
- "파일럿 테스트"가 곧 "시장 반응 테스트"를 의미

### 이 패턴에서 흔한 실수

1. **촬영만 하면 해결된다는 착각** — 촬영 후에도 상세페이지 기획, 채널 세팅, 유입 전략이 없으면 무용지물
2. **100만원으로 IMC를 시도** — 예산 분산 → 모든 채널에서 의미 없는 결과 → "마케팅 안 먹힌다" 결론
3. **촬영 비용에 예산 전부 소진** — 전문 촬영 1회(80-150만원)에 월 예산 전부 사용 → 운영비 제로
4. **전략 없는 촬영** — 어떤 컷이 필요한지(제품컷/모델컷/라이프스타일/영상) 정의 없이 촬영 → 재촬영 필요

### 이 패턴에서 효과적인 접근

1. **촬영 전에 채널 전략 먼저** — 어디서 팔 것인지 정하고, 그 채널에 맞는 촬영 기획
2. **1개 채널 집중** — 스마트스토어 or 자사몰 중 하나에 올인
3. **상세페이지 중심 촬영** — 전환율에 직접 영향을 주는 상세페이지용 촬영 우선
4. **월 100만원은 운영비로, 촬영은 별도 초기 투자로 분리** — 예산 구조 재설계 필요

## Step 6: 상담 검증 질문 (Top 5)

### Q1. "촬영"의 진짜 의미 파악 (최우선)

> "상담 신청서에 페인 포인트로 '촬영'이라고 적어주셨는데, 조금 더 구체적으로 여쭤봐도 될까요? 지금 제품 촬영이 아예 안 되어 있는 건지, 촬영은 했는데 퀄리티가 아쉬운 건지, 아니면 촬영 비용이 부담되시는 건지 — 어떤 상황이신가요?"

**검증 목표**: L1(촬영) 아래의 L2(전술적 문제)가 무엇인지 특정

### Q2. 현재 판매 채널과 매출 구조

> "지금 서머러 제품은 어디서 판매하고 계세요? 온라인이면 어떤 플랫폼인지, 오프라인이면 어떤 형태인지 궁금합니다. 현재 월 매출 규모도 대략적으로 알 수 있을까요?"

**검증 목표**: 패턴 A/B/C 중 어디에 해당하는지 특정. 온라인 전환 단계 진단.

### Q3. 제품 정체성과 타겟 고객

> "서머러는 어떤 제품을 만드시나요? 주요 타겟 고객은 누구이고, 경쟁 제품 대비 가장 차별화되는 점은 뭐라고 보시나요?"

**검증 목표**: 촬영 방향성 결정의 기초 데이터. 브랜드 전략 수준 진단.

### Q4. 마케팅 경험과 기대 수준

> "이전에 마케팅을 직접 해보시거나, 외주를 맡겨보신 경험이 있으세요? '파일럿 테스트'라고 하셨는데, 구체적으로 어떤 결과가 나오면 '이건 된다'고 판단하실 건가요?"

**검증 목표**: 마케팅 성숙도 진단. 성공 기준(KPI) 합의 가능성 확인. "파일럿" 이후 본격 투자 의향 여부.

### Q5. 예산 구조와 투자 의향

> "월 100만원 정도로 말씀해주셨는데, 이건 촬영 비용까지 포함인 건지 운영비 개념인지 궁금합니다. 만약 촬영을 제대로 하려면 초기에 별도 투자가 필요할 수 있는데, 초기 셋업 비용은 어느 정도까지 생각하고 계신가요?"

**검증 목표**: 예산 구조 재설계 가능성. 100만원이 "전부"인지 "월 운영비"인지에 따라 제안 범위가 완전히 달라짐.

## 종합 진단

### 핵심 가설

**이윤수 대표는 자체 제품을 가진 제조업 대표로, D2C/온라인 판매를 시작하거나 강화하고 싶은데 마케팅의 첫 단추(촬영 에셋)가 빠져 있어서 전체 마케팅이 멈춰 있는 상태다. "촬영"이라는 한 단어는 마케팅 전체 프로세스에 대한 막막함이 응축된 표현이다.**

### 진짜 니즈 (표면 → 깊은 순서)

| 레벨 | 니즈 | 확신도 |
|------|------|--------|
| 표면 | 제품 촬영 에셋 확보 | 높음 |
| 중간 | 전환율 높은 상세페이지/콘텐츠 구축 | 높음 |
| 깊은 | 온라인 판매 채널 구축 + 첫 매출 검증 | 중간 |
| 근본 | "마케팅을 어디서부터 어떻게 해야 하는지" 로드맵 | 중간-높음 |

### 상담 전략 제언

1. **"촬영"에서 시작하되, 촬영으로 끝내지 말 것** — 촬영은 공감 포인트로 쓰고, 촬영 이후의 전체 그림(상세페이지 → 채널 → 유입 → 전환)을 보여줘야 함
2. **IMC는 해체해야 함** — 100만원으로 IMC는 불가능. 1개 채널 집중 전략으로 리프레이밍 필요
3. **"파일럿"을 역이용** — "파일럿이라면 더더욱 집중해야 합니다. 3개월, 1개 채널, 1개 KPI"로 구조화
4. **예산 구조를 분리 제안** — 초기 셋업(촬영+상세페이지) 별도 + 월 운영비로 나누는 구조
5. **온라인 프레즌스 제로를 기회로** — "지금이 가장 좋은 타이밍. 처음부터 제대로 세팅하면 재작업 비용이 안 든다"

### 리스크 플래그

- **메일 회신 없음** → 관심도가 낮거나, 단순 정보 수집 목적일 수 있음. 상담 확정 전 온도 체크 필요
- **100만원 예산** → 촬영 + 운영을 동시에 하기엔 부족. 예산 증액 없이는 서비스 범위가 매우 제한적
- **"촬영" 한 단어** → 마케팅에 대한 이해도가 낮을 수 있음. 교육적 접근이 필요할 수 있으나, 코칭 라벨은 사용 금지
