## 2. Professional 견적 상세

대표님이 요청하신 "이 비용이 어디에 쓰이는지"를 투명하게 정리했습니다.

### 월 450만 원의 구성

| 업무 영역 | 월 투입 시간 | 내역 |
|-----------|:-----------:|------|
| **전략 기획** | ~12시간 | 월간 마케팅 전략 수립, 시즌 플래닝, 경쟁사/시장 모니터링 |
| **콘텐츠 기획 + 카피** | ~16시간 | 월 12~16개 콘텐츠 기획안, 캡션/카피 작성, 비주얼 디렉션 |
| **Meta 광고 운영** | ~10시간 | 캠페인 구조 설계, 타겟 세팅, 소재 제작 디렉션, 주간 최적화 |
| **시딩 캠페인 관리** | ~8시간 | 인플루언서 리서치/섭외, 시딩 키트 기획, 콘텐츠 QC |
| **리포팅 + 미팅** | ~6시간 | 주간 리포트, 월간 전략 리뷰, 주 1회 정기 미팅 |
| **총 월 투입** | **~52시간** | 주당 약 13시간, 전담 마케팅 파트너 |

참고 — 이 시간은 최소 기준입니다. FW 시즌 등 집중 시기에는 추가 투입이 이루어지며, 별도 비용은 청구하지 않습니다.

### 같은 업무를 직접 구성하면

| 구분 | 월 비용 |
|------|--------:|
| 마케팅 매니저 정규직 1명 (경력 3~5년) | 350~450만 + 4대보험/퇴직금 (~100만) |
| 콘텐츠 기획 프리랜서 | 200~300만 |
| 퍼포먼스 마케터 프리랜서 | 200~300만 |
| **합계** | **750~1,050만** |

브랜드라이즈 Professional은 이 세 역할을 월 450만 원에 커버합니다.
직접 구성 대비 **40~57% 비용 절감**이면서, 전략-콘텐츠-퍼포먼스가 하나의 팀에서 연결되어 움직입니다.

### 말렌 팀이 세이브하는 업무

지금 대표님과 내부 팀이 직접 고민하거나 시행착오를 겪고 있는 업무들입니다.

| 현재 말렌이 쓰는 시간 | Professional 시작 후 |
|----------------------|---------------------|
| "이번 달 인스타에 뭘 올리지?" 콘텐츠 기획 고민 | 월초에 콘텐츠 캘린더 전달. 말렌은 촬영/업로드만 |
| 광고 카피 직접 작성 (또는 포기) | 피드/광고/시딩 카피 전부 브랜드라이즈에서 작성 |
| Meta 광고 세팅, 타겟 고민, ROAS 해석 | 세팅~최적화~리포팅 전 과정 대행. 주간 요약으로 성과만 확인 |
| 인플루언서 찾기, DM 보내기, 조건 협상 | 리서치~섭외~계약~콘텐츠 QC까지 관리 |
| "마케팅 뭐부터 해야 하지?" 방향 고민 | 월간 전략 + 주간 미팅에서 우선순위 정리. 실행에만 집중 |
| 시즌 기획 (FW 언제부터 뭘 해야 하는지) | FW 시즌 역산 로드맵 제공. 타이밍 놓치지 않도록 관리 |
| 보고서/데이터 정리 | 주간/월간 리포트 자동 전달. 숫자 해석 + 다음 액션까지 |

정리하면 — 대표님은 "제품과 브랜드 방향"에만 집중하시면 됩니다.
마케팅의 기획, 실행, 분석, 개선 사이클은 브랜드라이즈가 돌립니다.

## 3. ROI 분석

### 투자 시나리오별 수익 예측

**기준 전제**
- 현재 연매출: 약 10억 원
- 현재 마케팅 투자: 거의 없음 (Meta 광고 월 150만 수준)
- 투자 기간: 12개월 기준

#### 시나리오 A: Professional + Meta 광고 월 300만

### 투자 회수 시점

**시나리오 A 기준 (보수적)**

- 월 1~2개월: 콘텐츠 기반 세팅 기간. 직접적 매출 증가는 제한적
- 월 3~4개월: 콘텐츠 누적 + 광고 최적화 효과 시작. 월 매출 +500~1,000만 증가 체감
- 월 5~6개월: 시딩 콘텐츠 누적, 리타겟팅 효과 본격화. 월 매출 +1,500~2,500만
- **투자 회수 시점: 4~5개월째** (보수적 기준, 누적 매출 증가분이 누적 투자를 넘는 시점)
- 월 9~11개월 (FW 시즌): 월 매출 2~3억 돌파 구간. 이 시기에 연간 ROI가 결정됨

### 숨어 있는 ROI: CRM

현재 말렌은 자사몰 방문자를 한 번 보내고 끊기는 구조입니다.

- 리타겟팅 미활용으로 인한 추정 손실: 월 2,000만 원 이상
- CRM(카카오 알림톡, 이메일 시퀀스) 구축 시 ROI: 10~20배 (가장 효율 높은 채널)
- 이미 한 번 구매한 고객에게 다시 파는 비용은 신규 고객 획득 비용의 1/5

Professional 패키지 안에서 기본적인 리타겟팅 구조는 세팅합니다.
Premium에서는 CRM 자동화까지 포함됩니다.

### 대표님이 체감하실 비유

> **"말렌에 마케팅 투자를 안 하는 건, 좋은 옷을 만들어놓고 옷장에 넣어두는 것과 같습니다."**
>

>
> 월 450만 원은 고객 1명을 데려오는 데 약 4,500~9,000원입니다.
> 말렌의 평균 객단가가 15~30만 원이라면, 고객 1명당 수익이 투자의 15~30배입니다.

## 4. 파일럿 옵션

7년간 마케팅 투자를 안 하셨던 만큼, 처음부터 장기 계약이 부담스러우실 수 있습니다.
"한 번 같이 일해보고 판단하자"는 방식을 제안드립니다.

### 2주 파일럿 — 150만 원 (VAT 별도)

**목적**: 브랜드라이즈의 일하는 방식과 산출물 퀄리티를 직접 경험

**2주 동안 받으시는 것**:
- 말렌 브랜드 진단 리포트 (현재 SNS, 광고, 시장 포지셔닝 분석)
- 인스타그램 콘텐츠 기획안 4개 (카피 포함, 바로 활용 가능)
- Meta 광고 계정 진단 + 즉시 개선 액션 3가지
- 1회 전략 미팅 (60분, FW 시즌 로드맵 초안 공유)

**파일럿 후 선택지**:
- 만족하시면 → Professional로 전환 (파일럿 비용은 첫 달 차감)
- 아쉬우시면 → 진단 리포트와 콘텐츠 기획안은 그대로 활용. 부담 없이 끝

파일럿 150만 원은 Professional 전환 시 첫 달에서 차감되므로, 실질적으로 추가 비용이 아닙니다.

## 5. 추천 패키지 + 다음 단계

### 추천: Professional (월 450만 원) + 3개월 단위 계약

**왜 Professional인가**

1. **말렌의 현재 상황에 정확히 맞습니다**
   - 내부 마케팅 인력 0명, 콘텐츠 기반 0에 가까운 상태
   - 전략만 주는 Essential로는 실행할 사람이 없습니다
   - CRM/해외까지 포함된 Premium은 지금 단계에서 과합니다

2. **대표님이 말씀하신 400~500만 범위와 일치합니다**
   - 부담 없이 시작할 수 있는 금액대
   - 마케팅 매니저 정규직 1명보다 저렴하면서 커버 범위는 넓습니다

3. **FW 시즌 역산 시, 지금 시작해야 합니다**
   - FW 시즌(9~11월) 매출 극대화를 위해 최소 4~5개월 사전 준비 필요
   - 5월 시작 시: 5~6월 콘텐츠 기반 세팅 → 7~8월 노출 확대 → 9월 FW 캠페인 런칭
   - 6월 시작 시: 준비 기간 1개월 부족, FW 피크 효과 감소

4. **3개월 단위 계약으로 리스크를 줄입니다**
   - 12개월 장기 계약이 아닙니다
   - 3개월 후 성과를 보고 연장/조정/종료를 판단하실 수 있습니다
   - 첫 3개월 목표: 콘텐츠 기반 완성 + 광고 효율 개선 + FW 준비 착수
