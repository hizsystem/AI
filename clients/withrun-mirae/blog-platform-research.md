## Tier 1 — 사장님들이 가장 많이 쓰는 플랫폼

### 리뷰노트 (reviewnote.co.kr) — 1순위 추천

- **운영 방식**: 공모형 (캠페인 등록 → 블로거 신청 → 광고주가 직접 선정)
- **비용**: 플랫폼 수수료 0원. 식사 제공만 하면 됨. 포인트 2~5만P 추가 시 지원자 3배 증가
- **규모**: 국내 체험단 수 1위 (캠페인 수 최다)
- **부산 커버리지**: 전국 (지역 필터, 부산 캠페인 활발)
- **장점**: 비용 0원, 블로거 선정 시 일방문자/맛집 포스팅 이력 직접 확인 가능, 키워드 가이드 전달 가능
- **단점**: 무료라 노쇼/저품질 리뷰 리스크, 직접 관리 필요
- **월 30만원 배분**: 식사 원가 ~10만 + 포인트 ~15만 + 여유 5만

### 레뷰 REVU (revu.net)

- **운영 방식**: 공모형 + 프리미엄 매칭
- **비용**: 건당 약 15~25만원 (대행 포함)
- **규모**: 국내 최대 인플루언서 플랫폼 (블로그+인스타+유튜브)
- **장점**: 블로거 풀 최대, 콘텐츠 품질 관리 체계적
- **단점**: 월 30만원이면 1~2건 수준. 소규모 식당에 과함
- **판단**: 예산 초과. 향후 예산 늘면 검토

### 디너의여왕 (dinnerqueen.net)

- **운영 방식**: 공모형 + 프리미엄 직접 섭외 (푸드 특화)
- **비용**: 별도 상담 (비공개)
- **규모**: 푸드/맛집 분야 국내 1위 특화
- **장점**: 맛집 블로거 밀도 높음, 디너패스(할인권) 등 추가 모객
- **단점**: 비용 비공개, 프리미엄은 비쌈
- **판단**: 비용 확인 후 검토

## 미례국밥 운영 전략

### 월 30만원 시뮬레이션

**리뷰노트 직접 운영 기준:**

| 항목 | 금액 | 비고 |
|------|------|------|
| 식사 제공 (원가) | ~10만원 | 돼지국밥+돼지우동 셋 x 6~8명 |
| 포인트 지급 | ~15만원 | 건당 2~3만P, 지원자 확보용 |
| 예비 | ~5만원 | 추가 지원자 대응 |
| **합계** | **30만원** | **월 6~10건 블로그 콘텐츠** |

### 키워드 가이드 운영 포인트

1. 체험단 가이드에 필수 키워드 3~5개 명시 (이미 만들어둔 `experience-team-guide.md` 활용)
2. 블로그 제목에 키워드 1개 이상 포함 요청
3. 사진 5장 이상 + 2,000자 이상 조건
4. **뒷광고 표시 필수** — "업체로부터 제품/서비스를 제공받았습니다"

### 운영 팁

- 블로거 선정 시 기존 포스팅 품질(사진, 글 분량) 반드시 확인
- 방문 시간은 피크 피해 오후 2~4시로 유도 (일반 손님에게 자연스러운 홍보)
- 예약 확인 문자 필수 (노쇼 방지)
- 센텀점/전포점 교차 운영하면 키워드 분산 효과
