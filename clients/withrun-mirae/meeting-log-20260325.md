## 미팅 흐름

### 1. 데이터 브리핑 (우성민)

블랙키위 데이터를 화면 공유하며 설명.

- "미례국밥" 월 검색량 1,660 — 매장 2개로 이 수치는 큰 편
- 콘텐츠 포화도 4.28% "매우 낮음" — 검색하는 사람은 있는데 볼 콘텐츠가 없는 상태
- 동동국밥은 월 24,600 검색, 15배 차이 — 40개 매장의 물리적 차이
- 동동국밥도 포화도 10%로 아직 "낮음" — 시장 자체가 크게 플레이하는 곳이 없음
- "돼지우동" 키워드 — 특정 브랜드가 선점하지 않은 상태
- "돼지우동서울" — 블로그 2건뿐. 서울 준비 시점에 빨리 쌓아야

> 변팀장 반응: "1660이면 엄청 작은 거죠?" → 우성민: "2개 매장에 1,600은 큰 수치. 중요한 건 포화도가 매우 낮다는 것"

### 2. 견적서 항목별 설명

**Part 1: 채널 전략 & 검색 기반 (270만)**

- 채널 운영 전략 100만: 인스타 컨셉/무드보드/후킹포인트/콘텐츠 가이드라인/KPI
- 네이버플레이스 키워드 최적화 100만: 소개문 재작성, 키워드 세팅, 별점 부활 대비, 소식 발행
- 블로그 세팅 + 5편 70만: 채널 개설, SEO 최적화 콘텐츠, 키워드 전략

**Part 2: 인플루언서 씨딩 & 광고 (230만)**

- 인플루언서 씨딩 80만: 5~7명, 바이럴 가이드, 방문 코디, 릴스 QC
- 메타 광고 80만: 광고비 50만 포함, 초기 세팅, A/B 테스트
- 리뷰 구조화 70만: 리뷰 답변 템플릿, 영수증 리뷰 안내물, 운영 가이드

### 3. 변팀장 주요 질문 + 합의 사항

#### 인스타 콘텐츠 방향 (가장 오래 논의됨)

**변팀장 핵심 의견:**
- "너무 무겁게 가지 않았으면 좋겠다"
- "포토가 붙어서 전문 카메라로 찍으면 오히려 차갑게 느껴진다"
- "댓글을 못 달 것 같은 분위기는 피하고 싶다"
- "스타벅스는 고급인데도 댓글이 잘 달리는데, 안목(국밥 브랜드)은 너무 고급져서 댓글을 못 달겠다"
- "동동국밥처럼은 절대 안 간다. 저 죽어도 그렇게는 못한다"
- "적당한 무게감을 유지했으면 좋겠다. 제일 어려운 말이지만"
- 핸드폰으로 퀄리티 있게 찍는 쪽을 선호
- 궁극적으로는 고객 참여/티키타카가 이루어지는 채널 원함

**합의:**
- 톤앤매너를 여러 레퍼런스와 함께 제안해서 맞춰보기로
- 초반에는 브랜드 메시지 전달, 팬덤 형성 후 쌍방향 콘텐츠로 전환
- 포토 촬영은 한 번은 각 잡고 찍되(포토+핸드폰 총동원), 일상 콘텐츠는 핸드폰 기반
- 서울 신규 매장 오픈 시 촬영하는 게 더 깨끗할 수 있음
- 기존 촬영 소스를 먼저 공유받아서 당장 한 달 운영 가능한지 체크

#### 콘텐츠 제작 역할

- 가이드/전략은 BR이 만들고, 실제 제작은 위드런 팀이 하는 구조
- 추가 콘텐츠 제작 시 별도 견적 (건당 or 월 단위)
- 촬영이 필요하면 BR 포토그래퍼 또는 콘텐츠 매니저가 부산 당일치기 가능

#### 네이버플레이스

- 변팀장: "센텀 돼지국밥으로 검색하면 이미 첫 페이지에 나오는데 광고가 필요한가?"
- 우성민: "지금 잘 돼 있으면 광고비 안 써도 됨. 진단 결과 보고 판단"
- 변팀장: "진단은 계약 후에 하는 거죠?" → "맞습니다"
- 플레이스 광고 집행 여부는 진단 후 판단하기로

#### 네이버플레이스 관리 체계

- 우성민이 소상공인 대상 네이버플레이스 코칭 시스템을 설명
- 체크리스트 기반 점수 진단 → 주간 체크 → 코칭 모델
- "새로오픈" 태그는 최적화가 70% 이상 됐을 때 켜는 게 효과적 (비장의 카드)
- 별점 부활 (4~6월 예정) 대비 준비

#### 블로그

- 체험단 블로그와 자체 블로그의 차이: "우리가 최적화되게 쓸 수 있다"
- 상위 노출 시 인기 게시물 유지 기간 평균 71일 → 3개월 확보 가능
- 체험단 가이드에도 우리가 밀어가는 키워드를 반영해서 드리겠다

#### 인플루언서 씨딩

- 변팀장: "부산 로컬 맛집 계정을 보유하고 계신 게 있나?"
- 우성민: "더 찾아야 하지만, 있으시면 그걸 기반으로 디벨롭 가능"
- 변팀장: "아는 사람들은 80만원에 안 되더라" → 이번 씨딩은 파워페이지가 아닌 소규모
- 핵심 목적: 콘텐츠 확보 + 2차 활용 (협업 광고, 광고 소재)

#### 메타 광고

- 메타 비즈니스 계정: 미례국밥은 아직 없음 (댄싱컵은 있었음). 새로 만들기로
- 월 50만원 광고비 — 최소 비용이지만 조절 가능
- 인스타 콘텐츠 + 인플루언서 2차 활용 콘텐츠를 소재로
- 파트너 권한 설정해서 인플루언서 콘텐츠를 동시 노출하는 방법도 설명

#### 리뷰 시스템

- 리뷰 답변이 네이버플레이스 점수에 크게 영향
- 똑같은 답변 반복하면 네이버 AI가 걸러냄 → 템플릿 10종으로 다채롭게
- 현재는 매장 담당자에게 넘긴 상태 → 가이드 제공 필요

### 5. 추가 논의 — 댄싱컵

변팀장이 자연스럽게 댄싱컵 이야기를 꺼냄.

**댄싱컵 방향성 발견:**
- 데이터 보니까 시즌 메뉴보다 기존 라떼류(딥낫떼, 블랙글레이즈드 라떼)가 많이 나감
- "라떼가 맛있는 집"으로 포지셔닝 가능성
- 단, "라떼 맛집 댄싱컵"은 너무 넓음 → 특정 메뉴까지 좁혀야
- 트렌드 탑승 전략: 우베라떼(미국 스타벅스 유행) 빠르게 출시 중
- "이 브랜드는 유행에 빨리 따라가는구나" 인식 만들기

**호기심깍까 사례:**
- 유명 디저트 인플루언서가 댄싱컵을 자발적으로 올려줌 (광고 아님)
- "라떼도 맛집이라서 디카페인으로 시켜봤는데 맛있어요" — 이미 원하는 메시지가 다 담김
- 변팀장: 여력이 없어서 DM 감사만 보내고 2차 활용 못함
- → 이런 인플루언서에게 지속적으로 제품 제공 + 관계 유지 필요

**로컬 마케팅 논의:**
- 변팀장: "센텀점 주변 아파트 주민들도 아직 모른다. 어떻게 알리나?"
- 우성민 사례: 천안 카페 — 전단지+베네핏으로 아파트 관리사무실/게시판/우편함 공략 → 효과 좋았음
- 아파트 커뮤니티(카톡방) 접근, 당근 비즈니스 광고는 이미 하고 있음
- 로컬 맛집 블로거 댓글에 브랜드 계정으로 먼저 다가가기

**창업 DB 광고 관련:**
- 변팀장이 프랜차이즈 창업 DB 전문 광고 업체를 찾고 있음
- 기존 업체와 소통 문제로 교체 원함
- BR 측: "검증 후 소개 가능. 알아보겠다"
- 변팀장: "브랜드가 좋으면 가만히 있어도 창업 문의가 들어온다. 일단 브랜딩이 먼저"

## 변팀장 인사이트 (다음에 반영할 것)

1. **"적당한 무게감"** — 너무 고급지지 않으면서 브랜드가 관리하는 느낌. 동동국밥처럼 너무 가볍지도 않게
2. **콘텐츠 제작 역량 부족** — "혼자서는 이런 거 여력이 안 된다"를 여러 번 언급. 아이디어는 있지만 실행할 손발이 없음
3. **로컬 매출이 급한 상황** — 오픈 3개월인데 주변 아파트/오피스에서도 인지도 부족
4. **댄싱컵도 관리가 필요** — 미례국밥 이후 자연스럽게 댄싱컵 논의로 이어짐. 향후 추가 수주 가능성
5. **창업 DB 업체 니즈** — 프랜차이즈 가맹모집 마케팅 업체를 찾고 있음. 소개해주면 관계 강화
