## 탭 구조 (6개)

### 탭 1: 대시보드

**용도**: 한눈에 현황 파악. 변팀장님에게도 공유 가능한 첫 화면.

| 섹션 | 내용 | 데이터 소스 |
|------|------|-----------|
| 4월 미션 | "검색하면 볼 게 있는 상태" | 고정 |
| KPI 현황 | 체험단 발행 N/6건, 가이드라인 미확정/확정 | 탭2에서 자동 |
| 이번주 핵심 액션 | 주차별 태스크 중 이번주 것 3~5개 | 탭2에서 수동 발췌 |
| 키워드 순위 요약 | 센텀 맛집 9위, 재송동 맛집 3위 등 | 탭4에서 수동 |
| 다음 미팅 | 날짜 + 안건 | 수동 |

**레이아웃**:
- A1~F1: 헤더 (미례국밥 x 브랜드라이즈 | 4월)
- A3~F8: KPI 카드 2개 (체험단 / 가이드라인)
- A10~F15: 이번주 액션 (체크박스 + 담당 + 상태)
- A17~F22: 키워드 순위 TOP 5
- A24~F26: 다음 미팅 일정

---

### 탭 2: 주차별 태스크

**용도**: 주차별 할 일 관리. 내부 실무용.

| 컬럼 | 내용 |
|------|------|
| A | 주차 (W1/W2/W3/W4) |
| B | 채널 (플레이스/인스타/체험단/메타/공통) |
| C | 태스크 |
| D | 담당 (BR/변팀장/지은) |
| E | 상태 (대기/진행중/완료/보류) |
| F | 마감일 |
| G | 비고 |

**초기 데이터** (kickoff-deck-v2 기준):

W1 — 분석 (4/1~4/4)
- 공통 | 블랙키위 데이터 분석 | BR
- 플레이스 | 에드로그 히든 키워드 분석 | BR
- 플레이스 | 네이버플레이스 진단 | BR
- 공통 | KPI 기준선 측정 | BR
- 공통 | 기존 촬영 소스 검토 | BR

W2 — 킥오프 (4/7~4/11)
- 공통 | 킥오프 미팅 (4/7) | BR+변팀장
- 인스타 | 톤 방향 확정 | BR+변팀장
- 플레이스 | 대표키워드 5개 교체 | BR
- 플레이스 | 소개문 교체 (센텀) | BR
- 플레이스 | 소개문에 "센텀 맛집/재송동 맛집" 추가 | BR
- 체험단 | 리뷰노트 캠페인 등록 | BR
- 체험단 | 키워드 가이드 전달 | 변팀장
- 메타 | 비즈니스 계정 생성 | BR
- 인스타 | 인플루언서 후보 확정 (5~7명) | BR

W3 — 실행 (4/14~4/18)
- 인스타 | 콘텐츠 가이드라인 확정 | BR
- 체험단 | 첫 체험단 방문 시작 | 변팀장+BR
- 인스타 | 인플루언서 컨택 DM | BR
- 플레이스 | 소식 발행 시작 | BR
- 플레이스 | 리뷰 답변 템플릿 적용 시작 | 변팀장
- 메타 | 광고 세팅 (픽셀, 타겟) | BR

W4 — 마무리 (4/21~4/30)
- 인스타 | 씨딩 릴스 수거 + QC | BR
- 메타 | 광고 시작 (릴스 소재) | BR
- 체험단 | 체험단 콘텐츠 수거 + 키워드 체크 | BR
- 공통 | 월간 리포트 작성 | BR
- 공통 | Month 2 방향 제안 | BR
- 플레이스 | 에드로그 순위 변동 체크 | BR

### 탭 3: KPI 트래킹

**용도**: 월간 KPI 추적. 월간 리포트의 근거 데이터.

| 컬럼 | 내용 |
|------|------|
| A | 지표 |
| B | 4/1 기준선 |
| C | W1 (4/4) |
| D | W2 (4/11) |
| E | W3 (4/18) |
| F | W4 (4/30) |
| G | 목표 |
| H | 달성 여부 |

**지표 목록**:
- 체험단 콘텐츠 발행 수 (기준: 0 / 목표: 6건+)
- 인스타 가이드라인 (기준: 미확정 / 목표: 확정)
- 센텀 맛집 순위 (기준: 9위)
- 재송동 맛집 순위 (기준: 3위)
- 블로그 리뷰 수 (기준: 44개)
- 플레이스 방문자 리뷰 (기준: 1,296개)
- 인플루언서 릴스 수거 (기준: 0 / 목표: 5~7건)
