## 편 1. "센텀 미례국밥, 돼지우동이 뭐길래 줄 서서 먹는다는 그 메뉴"

### 키워드

| 구분 | 키워드 |
|------|--------|
| 메인 | 센텀 미례국밥 |
| 서브 | 돼지우동, 센텀 국밥, 미례국밥센텀점, 센텀시티 맛집 |
| 의도 | 미례국밥 주차, 미례국밥 웨이팅 |

### 글 구조

1. **도입 — "국밥집에서 우동?"** : 돼지우동이라는 메뉴명이 생소한 이유, 호기심 유발. "센텀 미례국밥에 가면 국밥 말고 이걸 먼저 시킨다"는 훅
2. **미례국밥 센텀점 소개** : 위치(센텀필 상가), 매장 분위기, 첫인상. "깔끔한 국밥집"이라는 포인트
3. **돼지우동 상세 리뷰** : 비법 육수 + 쫄깃한 우동면 + 고기 토핑. 비주얼 묘사. 국밥 육수 베이스라 깊은 맛, 일반 우동과의 차이
4. **함께 먹으면 좋은 조합** : 돼지우동 + 공기밥, 반찬 구성, 셀프바 있으면 언급
5. **가격 및 메뉴판** : 돼지국밥 10,000원 / 돼지우동 11,500원 / 한마리국밥 13,000원. 가성비 코멘트
6. **주차 & 웨이팅 정보** : 센텀필 상가 1,2관 주차장, 1시간 주차권 제공. 웨이팅 팁(피크 시간 피하기)
7. **마무리 — 다음엔 한마리국밥** : 재방문 의사 + 다른 메뉴 예고로 시리즈 연결

### 핵심 내용 포인트

- 돼지우동이 시그니처라는 메시지를 반복 각인
- "국밥집인데 우동?"이라는 반전 요소로 흥미 유발
- 변팀장 인사이트 반영: 브랜드가 관리하는 느낌이되 너무 딱딱하지 않은 톤
- "맛은 정성으로 예는 마음으로" 슬로건 자연스럽게 1회 포함

## 편 2. "센텀 돼지국밥 미례국밥 방문기 — 주차부터 웨이팅까지 솔직 후기"

| 의도 | 미례국밥 주차, 미례국밥 웨이팅 |

### 글 구조

1. **도입 — 방문 계기** : "센텀에서 돼지국밥 어디 가냐"는 질문에 추천받은 곳, 또는 SNS에서 보고 찾아간 경험
2. **찾아가는 길 & 주차** : 센텀필 상가 위치 상세, 주차장 진입 방법, 1시간 주차권 받는 법. 대중교통 안내
3. **웨이팅 & 입장** : 피크 시간대(점심 12~1시) 웨이팅 상황, 오픈런 팁, 대기 시 메뉴 미리 보기
4. **주문 — 돼지국밥 + 돼지우동** : 2인 방문 기준 추천 조합. 메뉴판 사진 + 가격 정리
5. **돼지국밥 솔직 후기** : 진한 육수, 고기 양, 밥과의 조합. 부산 돼지국밥 특유의 맛
6. **돼지우동도 먹어봄** : 국밥집에서 우동? 의외의 조합이 왜 되는지. 편 1과 다른 각도(동행인 시점)
7. **총평 & 재방문 의사** : 가격 대비 만족도, 분위기, 서비스. "센텀 국밥 고민이면 여기"로 마무리

- 실제 방문기 형태로 현장감 살리기 (시간 순서대로 서술)

- 2인 방문 시 추천 조합 제안으로 객단가 상승 유도
- 사진 배치 가이드: 외관 → 메뉴판 → 음식 클로즈업 → 먹는 장면 → 주차장

## 편 3. "전포 미례국밥, 부산 현지인이 추천하는 돼지국밥집"

### 키워드

| 구분 | 키워드 |
|------|--------|
| 메인 | 전포 미례국밥 |
| 서브 | 부산 돼지국밥, 전포 맛집, 서면 미례국밥, 부산 국밥 |
| 의도 | 미례국밥 주차, 미례국밥 메뉴 |

### 글 구조

1. **도입 — 전포동에 국밥집이?** : 카페거리로 유명한 전포에 국밥 맛집이 있다는 의외성. "전포 맛집 하면 카페만 떠올리는 사람, 여기 한번 가보세요"
2. **전포점 위치 & 접근성** : 전포카페거리 근처, 서면역 도보 거리. 주차 정보
3. **매장 분위기** : 깔끔하고 정돈된 인테리어. "요즘 국밥집" 느낌. 기존 국밥집 이미지와의 차이
4. **메뉴 추천 — 돼지국밥 중심** : 전포점 방문이면 기본인 돼지국밥부터. 진한 육수의 비결, 고기 퀄리티
5. **돼지우동도 빠질 수 없다** : 미례국밥의 시그니처. 전포점에서도 동일한 맛
6. **현지인 추천 포인트 3가지** : 맛(진한 육수), 양(든든함), 가격(합리적). 부산 돼지국밥 맛집으로 꼽는 이유
7. **정보 정리 & 마무리** : 메뉴 가격표, 영업시간, 주차, "부산 국밥 찾는다면 전포 미례국밥 추천"

### 핵심 내용 포인트

- 전포 = 카페거리라는 기존 인식을 역이용해서 흥미 유발
- "현지인 추천"이라는 프레임으로 신뢰도 확보
- 서면 미례국밥 키워드도 자연스럽게 포함 (서면역 근처라는 맥락)
- 센텀점과의 차별점보다는 "어디서든 같은 맛"이라는 일관성 강조

## 편 4. "부산 미례국밥 전 메뉴 솔직 리뷰 — 돼지국밥부터 순대국밥까지"

| 의도 | 미례국밥 가격, 미례국밥 메뉴 |

### 글 구조

1. **도입 — 미례국밥, 뭘 먹어야 하나?** : 처음 가면 메뉴 고르기 고민. "전 메뉴 먹어본 사람이 정리해 드립니다"
2. **돼지국밥 (10,000원)** : 기본이자 시작. 진한 육수, 부산 돼지국밥의 정석. 이것만 먹어도 만족
3. **돼지우동 (11,500원)** : 미례국밥의 시그니처. 국밥 육수 + 우동면의 조합. "이거 때문에 재방문하는 사람 많다"
4. **한마리국밥 (13,000원)** : 고기를 듬뿍 먹고 싶을 때. 양과 퀄리티. 가성비 평가
5. **순대국밥** : 순대 좋아하는 사람을 위한 선택지. 순대 퀄리티, 국물과의 궁합
6. **메뉴별 추천 상황** : 혼밥이면 돼지국밥, 데이트면 돼지우동, 든든하게면 한마리국밥. 상황별 가이드
7. **가격 총정리 & 마무리** : 전 메뉴 가격표 정리. "부산 미례국밥, 결론은 돼지우동 꼭 드세요"

### 핵심 내용 포인트

- 전 메뉴 리뷰라는 포맷으로 정보성 콘텐츠 극대화
- 각 메뉴별 사진 필수 (메뉴판 + 개별 음식 사진)
- 돼지우동을 시그니처로 자연스럽게 포지셔닝 (마지막 결론에서 강조)
- 상황별 추천으로 실용성 높이기
- "부산 미례국밥" 키워드로 지역 대표 국밥집 포지셔닝

## 편 5. "미례국밥 돼지우동 — 국밥집에서 우동을 시킨다고?"

### 키워드

| 구분 | 키워드 |
|------|--------|
| 메인 | 미례국밥 돼지우동 |
| 서브 | 돼지우동, 부산 국밥, 부산 돼지우동 |
| 의도 | 미례국밥 메뉴 |

### 글 구조

1. **도입 — "돼지우동이 뭔데?"** : 메뉴명만 듣고 갸우뚱하는 사람들을 위한 설명. 국밥 육수에 우동면을 넣는다는 개념
2. **돼지우동의 탄생 배경** : 미례국밥이 돼지우동을 만든 이유. 국밥 육수의 깊은 맛을 다른 방식으로 즐기는 메뉴. 브랜드 스토리 녹이기
3. **맛 상세 리뷰** : 육수(돼지 뼈를 오래 우린 진한 국물) + 면(쫄깃한 우동면) + 토핑(고기, 파 등). 한 입 먹었을 때의 경험을 생생하게
4. **일반 우동 vs 돼지우동** : 가쓰오부시 베이스 우동과의 차이. 국밥 육수 베이스라 더 깊고 진한 맛. 매운맛 조절 가능 여부
5. **이렇게 먹으면 더 맛있다** : 공기밥 추가, 김치와 함께, 고추 넣기 등 꿀팁. 단골들의 먹는 법
6. **가격 & 다른 메뉴와 비교** : 돼지우동 11,500원. 돼지국밥(10,000원)보다 1,500원 더인데 그 값어치를 하는 이유
7. **마무리 — "부산 가면 미례국밥 돼지우동"** : 부산 여행 맛집으로, 부산 현지인 추천 메뉴로 포지셔닝. 방문 정보 간단 정리

### 핵심 내용 포인트

- "돼지우동" 키워드를 제목부터 본문까지 자연스럽게 반복 (선점 목적)
- 메뉴 하나에 집중하는 딥다이브 콘텐츠
- "국밥집에서 우동?"이라는 호기심을 풀어주는 구조
- 돼지우동 = 미례국밥 공식을 각인시키는 것이 최종 목적
- 블랙키위 데이터: "돼지우동" 330검색인데 특정 브랜드 선점 없음 → 선점 기회

## 공통 작성 가이드

### 톤앤매너
- 너무 딱딱한 정보 나열 X, 너무 가벼운 맛집 블로그 X
- "직접 가보고 정리한 사람"의 톤. 신뢰감 있되 친근하게
- 변팀장 요청: "적당한 무게감. 브랜드가 관리하는 느낌이되 댓글 달고 싶은 분위기"

### SEO 규칙
- 제목에 메인 키워드 필수 포함
- 본문 첫 2문장 안에 메인 키워드 1회
- 소제목(H2/H3)에 서브 키워드 자연 포함
- 본문 내 메인 키워드 3~5회, 서브 키워드 각 1~2회
- 사진 ALT 태그에 키워드 포함
- 마지막 문단에 메인 키워드 1회 마무리

### 사진 배치 (편당 10~15장)
- 매장 외관 1장
- 메뉴판/가격표 1장
- 음식 클로즈업 3~5장
- 먹는 장면/분위기 2~3장
- 주차장/입구 1장
- 지도/위치 캡처 1장

### 필수 정보 블록 (매 편 하단에 동일 포맷)

```
--- 방문 정보 ---
매장명: 미례국밥 센텀점 (or 전포점)
주소: [상세 주소]
영업시간: [확인 후 기재]
주차: 센텀필 상가 1,2관 주차장 (1시간 주차권 제공)
대표 메뉴: 돼지국밥 10,000 / 돼지우동 11,500 / 한마리국밥 13,000
```
