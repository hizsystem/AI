## 왜 지금, 왜 미례국밥인가

**시장이 미례국밥을 기다리고 있다.**

2025년 배달의민족이 선정한 외식 3대 트렌드 중 하나가 **"뉴웨이브 국밥"**이다. 투박한 서민 음식이던 국밥이 세련된 경험 + 차별화된 메뉴로 MZ 소비자를 끌어들이는 흐름. 미례국밥의 **"돼지우동"**은 이 트렌드의 한가운데 있다.

동시에 혼밥 비중이 전체 식사의 47%를 차지하고, "부산에서 온" 로컬 브랜드에 대한 선호가 역대 최고다. 서울에 돼지국밥집은 537곳이 넘지만, **"돼지우동"이라는 키워드를 자기 브랜드와 연결시킨 곳은 아직 없다.**

### 경쟁 환경: 동동국밥이 움직이고 있다

가장 주의해야 할 경쟁사는 **동동국밥**(위포먼스F&B)이다.

| 항목 | 동동국밥 | 미례국밥 |
|------|---------|---------|
| 시작 | 2023년 울산 | 부산 |
| 매장 수 | 40개 (2년 만에) | 2개 |
| 유사 메뉴 | 우동돼지국밥 9,500원 | 돼지우동 |
| 서울 진출 | 마곡나루 1곳 | 5~6월 예정 |
| 식자재 | 동원홈푸드 MOU | — |
| 온라인 | 공식 홈페이지 있음 | 거의 없음 |

동동국밥이 "우동국밥"으로 부르고, 미례국밥이 "돼지우동"으로 부르는 차이. **아직 어느 브랜드도 "돼지우동"이라는 키워드를 선점하지 못했다.** 이 기회는 지금뿐이다.

## 구체적 진행 내용

### Phase 0. 브랜드 진단 + 전략 설계 (Week 1~2)

이건 조금 생소한 과정일 수 있다. 쉽게 말하면 **"우리 브랜드가 지금 어디에 있는지 체크하고, 어디로 갈지 방향을 정하는 것"**이다.

**하는 일:**
1. 기존 마케팅 액션 전수 점검 — 체험단, 인플루언서, 플레이스 광고 등 지금까지 한 것들이 효과가 있었는지 숫자로 체크
2. 내부 워크숍 1회 (대표님 + 팀장님 + 브랜드라이즈, 콜 미팅 60분)
   - 미례국밥이 경쟁사(동동국밥 등) 대비 진짜 다른 점은 뭔지
   - 돼지우동 외에 히어로 후보 메뉴가 더 있는지
   - B2C(소비자 매출) vs B2B(가맹 문의) 중 1순위 확정
3. 경쟁사 벤치마킹 — 동동국밥, 열정국밥 등의 온라인 마케팅 분석
4. "미례국밥" / "돼지우동" 네이버 검색량 기준선 측정

**산출물:**
- 브랜드 진단 리포트 (현재 상태 + 경쟁 환경 + 기회/리스크)
- Month 1~3 로드맵 (이달 할 것 / 다음 달 할 것 / 서울 오픈 시 할 것)

### Phase 1. 네이버 플레이스 최적화 (Week 1~2, 진단과 동시 진행)

네이버 플레이스는 **지도에서 "돼지국밥" 검색했을 때 미례국밥이 상위에 뜨게 만드는 것**이다. 매장 근처 고객이 "점심 뭐 먹지?" 검색할 때 가장 먼저 보이게 하는 작업.

**현재 단계:** 기본 등록은 되어 있으나, 최적화 항목이 남아 있다.

**하는 일:**

| 항목 | 구체적으로 하는 것 | 왜 필요한지 |
|------|------------------|------------|
| S등급: 영수증 리뷰 이벤트 | 매장 내 안내물 제작 (포스기 앞 + 테이블). 리뷰 작성 가이드 포함 — 방문 목적 + 키워드 + 사진 + 별점5 | 네이버 알고리즘이 리뷰 속 키워드를 읽어서 순위를 매김. 4월 6일 별점 제도 부활 대비 |
| S등급: 플레이스 광고 세팅 | 일 예산 5,000~10,000원으로 시작. 키워드 태그 설정 + A/B 테스트 | 광고 자체보다 유입 고객의 체류시간이 알고리즘 점수를 올림 |
| S등급: 예약 + 쿠폰 | 네이버 예약 활성화 + 알림받기 쿠폰 세팅 | 예약 가능 매장은 별도 노출. 쿠폰 자체가 지수에 영향 |
| A등급: 대표 사진/영상 | 메뉴 사진 재촬영 (돼지우동 비주얼 강조) + 5~10초 영상 업로드 | 사진이 클릭을 결정. 영상은 체류시간 직접 증가 |
| A등급: 대표 키워드 | "부산 돼지국밥", "돼지우동", 지역명 포함 키워드 세팅 | 알고리즘이 읽는 메타태그. 경쟁사 키워드 분석 후 설정 |
| A등급: 소식 발행 | 주 1회 소식 게시 (신메뉴/이벤트) | 2026년부터 소식이 알고리즘 점수에 더 많이 반영 |
| B등급: 상세 소개문 | 1,500자 이상 작성 (키워드 자연 포함) | 검색 매칭 + 체류시간 증가 |
| 모니터링 | 에드로그 가입 (월 2만원) → N1/N2/N3 히든 지수 주간 추적 | 숫자로 효과 측정. 어떤 액션이 순위를 올렸는지 판단 |

**이건 "한번 세팅하고 끝"이 아니라 계속 관리가 들어가야 한다.** 매주 리뷰 답변 (키워드 포함), 소식 업데이트, 사진 교체, 키워드 성과 체크. 이 루틴은 팀장님이 주간 30분 투자하시면 유지되는 수준으로 가이드를 드린다.

### Phase 2. "돼지우동" 바이럴 콘텐츠 (Week 2~4)

**목표:** 서울 사람이 "돼지우동"을 검색하면 미례국밥이 나오게 만든다.

**인플루언서 전략:**

| 구분 | 대상 | 인원 | 예상 비용 | 하는 일 |
|------|------|------|----------|--------|
| 부산 맛집 계정 | 팔로워 1만~10만 인스타/블로그 | 3~5명 | 60~150만원 | "부산 현지인이 인정한 돼지우동" 콘텐츠 — 정통성 확보 |
| 서울 맛집 계정 | 팔로워 1천~5만 나노~마이크로 | 5~8명 | 50~120만원 | "부산 가면 꼭 먹어야 한다는 돼지우동" — 서울 소비자 호기심 유발 |
| 블로그 체험단 | 네이버 블로거 | 15~20건 | 30~50만원 | 네이버 검색 "부산 돼지국밥" / "돼지우동" 노출 확보 |

**왜 마이크로 인플루언서인가:**
- 팔로워 10만+ 대형 인플루언서 1명보다 마이크로 10명이 참여율(좋아요/댓글) 2.4배, ROI 60% 높음 (HubSpot 2024)
- 500만원 예산에서 대형 인플루언서는 1명도 쓰기 어려움 (건당 100만원+)
- 마이크로 인플루언서는 "진짜 맛집 추천" 느낌이 나서 신뢰도가 높음

**콘텐츠 방향:**
- "면발 올리는 장면" — 돼지우동의 비주얼 포인트 (SNS 후킹)
- "부산 현지인 국밥집의 비밀 메뉴" — 스토리텔링
- "국밥집에서 우동을 시켰다" — 호기심/반전 코드
- 릴스/숏폼 중심 (15~30초)

## 왜 브랜드라이즈인가

다른 마케팅 업체도 메타 광고, 네이버 플레이스, 인플루언서를 할 수 있다. **방법은 같다.**

차이는 **"왜 미례국밥이어야 하는지"를 찾아내는 것**이다.

- 돼지국밥집은 서울에만 537곳이 넘는다. 그 안에서 미례국밥이 돋보여야 하는 이유.
- 동동국밥이 이미 "우동국밥"으로 40개 매장을 운영 중이다. 여기서 미례국밥이 다른 점.
- "뉴웨이브 국밥" 트렌드가 지금 이 순간 일어나고 있다. 이 흐름에 어떻게 올라탈지.

이 **"뾰족함"**을 찾고, 검색량이라는 숫자로 증명하는 것이 우리가 하는 일이다.

매월 AB 테스트로 "이건 효과 있었다 / 없었다"를 투명하게 공유하고, 효과 없는 건 바로 커팅한다. 3개월이면 미례국밥의 첫 번째 성공 공식이 만들어진다.
