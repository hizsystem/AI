# 미례국밥(위드런) 시장 딥리서치

> 작성: 2026-03-18
> 목적: 미례국밥 프랜차이즈 마케팅 전략 수립을 위한 시장/경쟁/소비자 종합 리서치
> 참고: 미팅록(2026-03-18) 기반, 웹서치 데이터 활용

## 1. 돼지국밥 시장 분석

### 1-1. 한국 외식 시장 전체 규모

| 연도 | 시장 규모 | 비고 |
|------|----------|------|
| 2024 | 약 110.8조~153.2조원 | 유로모니터/ATFIS 추정치 차이 존재 |
| 2025 | 성장 지속 전망 | 배달 비중 37%(2029년 예측)까지 확대 |

- 매장 취식 45%, 배달 35%, 포장 18% (2024 기준)
- 1인 가구 증가 + 혼밥 비중 전체 식사의 47% 차지 (2024)
- 혼밥 긍정 인식: 2014년 56% -> 2025년 83%로 급증

### 1-2. 국밥/탕류 카테고리 포지션

돼지국밥만 분리된 시장 규모 데이터는 공식 통계로 존재하지 않는다. 다만 아래 지표로 카테고리의 성장세를 추정할 수 있다.

**국밥 프랜차이즈 주요 지표:**
- 큰맘할매순대국: 가맹점 399개 (2022)
- 일품양평해장국: 가맹점 400개+ (2025)
- 국밥생각: 가맹점 222개+ (2025)
- 열정국밥: 가맹점 234~247개, 월 100만 그릇 판매
- 이화수전통육개장: 164개, 연평균 매출 약 4.15억원 (가맹점 기준)
- 현대옥(콩나물국밥): 146개 (2020)

국밥 프랜차이즈는 외식업 중에서도 경기 방어력이 높은 카테고리로, 불황에도 꾸준한 수요가 있다. 특히 "한 그릇 식사" 중심으로 외식 구조가 재편되면서 국밥, 찌개, 덮밥, 면류 전문점이 성장 중이다.

### 1-3. "뉴웨이브 국밥" 트렌드 (2025 배민트렌드 Vol.1 핵심 키워드)

배달의민족이 2025년 외식업 3대 트렌드로 선정한 키워드:
1. **시즌리스 아이스** (4계절 아이스)
2. **뉴웨이브 국밥**
3. **저속노화**

**뉴웨이브 국밥의 특징:**
- 투박한 옹기 그릇 대신 고급 유기/도자기에 담아냄
- 물냉이, 유자, 아루굴라 페스토 등 이색 토핑
- 밝고 세련된 인테리어 (구식 노포 탈피)
- 서민 음식에서 "경험 + 인테리어 + 맛" 3박자를 갖춘 고급 외식으로 진화
- 2030 여성 소비자가 핵심 타겟

**대표 사례: 안암(安庵)**
- 서울 재동 소재 돼지국밥 전문점
- 뉴웨이브 국밥의 선두주자로 평가
- MZ 소비자 감성에 맞춘 공간 + 메뉴 구성

**미례국밥 시사점:**
- 서울 진출 시 "뉴웨이브 국밥" 트렌드에 올라타는 것이 포지셔닝의 핵심
- 단순히 "부산 돼지국밥 그대로"가 아니라 "부산 정통성 + 서울 감성"의 접점을 찾아야 함
- 인테리어/플레이팅/메뉴 구성에서 MZ 소비자가 사진 찍고 싶은 요소 필수

### 1-4. 부산 돼지국밥의 전국화 트렌드

**역사적 배경:**
- 돼지국밥의 기원: 밀양(1938년 무안장터) + 부산(6.25 피난민 시절)
- 스타일 3가지: 대구식(향신료+내장 강조), 밀양식(소뼈 육수, 맑은 국물), 부산식(돼지뼈 육수, 탁한 국물)
- 2013년 영화 '변호인'이 부산 돼지국밥의 전국적 인지도 상승 계기

**서울 시장 현황:**
- 서울 돼지국밥 맛집 537곳+ (다이닝코드 기준)
- 미쉐린 가이드에서 돼지국밥 맛집 6곳 별도 선정
- 광화문국밥(박찬일 셰프): 미쉐린 빕 구르망, 서울식 돼지국밥의 대표주자
- "서울식 돼지국밥"이라는 별도 카테고리 형성 중 (장국밥에 가까운 형태)

**가격 동향:**
- 부산역 부근: 대부분 9,000원대 (2022년~)
- 수도권: 10,000~12,000원
- 원가 부담: 돼지 앞다리 사용, 생각보다 높은 원가
- "서민 음식"이라는 인식과 실제 판매가 사이의 갭이 존재

**지역명 포함 메뉴 매장 수:**
- "부산식 수육국밥", "제주 청귤 에이드", "광주식 오리탕" 등 지역명 포함 메뉴 판매 매장 약 20만곳 (2025 Q1, 배민 기준)
- 로코노미(지역+경제) 트렌드가 외식업에서 매우 강력하게 작동 중

**동동국밥 분석:**
- 2023년 시작 후 2년 만에 40개 매장 확보 = 매우 빠른 성장
- "우동국밥"이라는 메뉴명 자체가 차별화 포인트 (돼지국밥+우동)
- 동원홈푸드와의 MOU는 식자재 품질 안정화 + 스케일업 준비의 신호
- 서울 진출은 마곡나루 1곳으로 아직 초기 단계
- 영남권 중심 성장 후 수도권 확장 패턴 = 미례국밥과 동일 경로

**미례국밥과의 직접 비교:**
- 동동국밥은 "우동국밥"으로 이미 시장에서 인지도 확보 중
- 미례국밥의 "돼지우동"과 메뉴 컨셉이 유사 (돼지국밥+우동)
- 동동국밥이 40개 매장으로 선점 중이므로, 미례국밥은 차별화가 더 절실
- 미례국밥의 차별화 후보: 맛/레시피, 브랜딩 감성, 서울 진출 속도, 히어로 메뉴 스토리

## 3. 미례국밥 현재 온라인 존재감

### 3-1. 웹서치 결과 기반 온라인 존재감

"미례국밥" 키워드로 웹서치 시:
- **네이버/구글 검색**: "미례국밥"에 대한 구체적인 검색 결과 거의 없음
- **맛집 플랫폼(다이닝코드, 식신 등)**: 미례국밥 단독 프로필 미확인
- **블로그/카페**: 언급량 극히 제한적
- **인스타그램**: 블랙 계정(본사 공식) 운영 중이나 팔로워/콘텐츠 상세 미확인

**진단:**
- 온라인 존재감이 현재 "거의 0"에 가까운 상태
- 이는 역설적으로 "깨끗한 캔버스"라는 장점 = 브랜딩 방향을 처음부터 설정 가능
- 검색량 기준선(baseline) 측정이 첫 번째 액션 → 이후 모든 마케팅 성과를 여기서 측정

### 3-2. "돼지우동" 키워드 분석

웹서치 결과 기반:
- "돼지우동"은 부산/울산 지역 일부 국밥집에서 사용하는 메뉴명
- 동동국밥이 "우동돼지국밥", "우동순대국밥"으로 이미 메뉴화
- 나무위키 돼지국밥 문서에서도 "돼지우동" 언급: "국밥에 밀가루 면 대신 우동을 넣어 파는 형태"
- 독립 키워드로서의 검색량은 아직 미미할 것으로 추정

**기회:**
- "돼지우동"이라는 키워드를 미례국밥이 선점할 수 있는 시장 공백 존재
- 동동국밥은 "우동국밥"/"우동돼지국밥"으로 부르고, 미례국밥은 "돼지우동"으로 브랜딩 → 차별화
- "돼지국밥" 검색량은 높지만 경쟁 치열 / "돼지우동"은 검색량 낮지만 독점 가능

## 4. 돼지우동 포지셔닝 분석

### 4-1. "돼지우동"의 시장 내 위치

| 메뉴명 | 포지션 | 시장 상황 |
|--------|--------|----------|
| 돼지국밥 | 부산 대표 음식, 전국적 인지도 | 경쟁 과포화 (서울만 537곳+) |
| 돼지국수 | 돼지 육수 + 소면/칼국수 | 일부 지역 존재, 프랜차이즈 없음 |
| 우동 | 일본식 면요리 | 완전 다른 카테고리 |
| **돼지우동** | 돼지 육수 + 우동 면 | **니치 포지션, 선점 가능** |

### 4-2. 히어로 제품으로서의 "돼지우동" 장단점

**강점:**
- 이름 자체가 호기심을 유발함 (돼지 + 우동 = "뭐지?")
- 기존 돼지국밥과 명확한 차별화 (밥 대신 우동면)
- 사진/영상 콘텐츠 제작 시 비주얼 차별화 용이 (면발이 올라가는 장면 등)
- "한 번은 먹어봐야 하는 메뉴" = 시도율(trial) 유발에 유리
- SNS 바이럴에 적합한 "신기한 메뉴" 요소

**약점/리스크:**
- "돼지우동"이라는 이름이 카테고리 혼란을 줄 수 있음 (일본 우동? 국밥?)
- 동동국밥이 "우동국밥"으로 이미 유사 컨셉 운영 중 (40개 매장)
- 돼지국밥의 정통성을 기대하는 소비자에게는 거부감 가능
- "면" 중심이면 국밥 카테고리에서 이탈할 수 있음

**전략 제안:**

## 5. 부산 -> 서울 F&B 브랜드 진출 성공/실패 사례

### 5-1. 성공 사례 상세

#### (1) 삼진어묵 — 프리미엄 리포지셔닝의 교과서

| 항목 | 내용 |
|------|------|
| 설립 | 1953년 (부산 봉래동) |
| 전환점 | 2010년, 3세대 박용준 대표가 영업 전면 개편 |
| 핵심 전략 | 어묵을 "길거리 음식"에서 "프리미엄 베이커리/선물"로 리포지셔닝 |
| 매출 추이 | 83억(2013) -> 850억(2023) -> 1,007억(2024) = 10년 만에 12배 성장 |
| 직원 수 | 25명(2011) -> 250명+(현재) |
| 유통 채널 | 백화점 식품관, 인천공항 T2, 온라인몰, 신세계 센텀시티 등 |
| IPO | 코스닥 상장 도전 (2025년 기사) |

**성공 요인:**
1. "어묵 베이커리" = 기존에 없던 카테고리 창출
2. 프리미엄 선물세트 = 매해 명절 완판
3. 깔끔한 매장 인테리어 = 갓 튀긴 어묵의 맛+경험 차별화
4. SNS 바이럴 = 부산 여행 필수 코스로 자리매김

**미례국밥 적용:**
- 삼진어묵의 핵심은 "기존 카테고리의 재정의"
- 미례국밥도 돼지국밥을 재정의해야 함 → "돼지우동"이 그 역할
- 프리미엄화가 아닌 "경험의 차별화"에 초점

#### (2) 컴포즈커피 — 부산발 저가 커피의 전국 제패

| 항목 | 내용 |
|------|------|
| 설립 | 2014년 (부산) |
| 매장 수 추이 | 395개(2019) -> 725개(2020) -> 1,285개(2021) -> 1,901개(2022) -> 2,262개(2023.07) -> 3,000개 목전(2024) |
| 서울 진출 | 2015년, 2022년에 서울 매장(312개)이 부산(248개)을 추월 |
| 해외 | 싱가포르 매장 오픈 |

**성공 요인:**
1. 명확한 가성비 포지셔닝 (저가 커피 = 메가커피, 컴포즈, 더벤티)
2. 코로나 기간 오히려 매장 급증 (비대면 + 테이크아웃 수요)
3. 지방에서 검증된 모델을 수도권에 이식

**미례국밥 적용:**
- 가성비 포지셔닝보다는 "부산 정통성 + 차별화 메뉴" 전략이 적합
- 컴포즈처럼 양적 확장보다, 삼진어묵처럼 질적 차별화가 미례국밥에 맞는 모델

#### (3) 정직유부 — 소형 매장에서 200개+ 프랜차이즈로

**성공 요인:**

2. 한 입에 재료가 모두 어우러지는 레시피
3. 유분기 제거 기술로 느끼함 해소
4. 5년 만에 8평 -> 200개+ 프랜차이즈

**미례국밥 적용:**
- 작은 매장에서 시작해 빠르게 프랜차이즈로 확장한 모델
- "1초에 2개" 같은 숫자 기반 마케팅 메시지가 가맹 모집에 효과적
- 미례국밥도 "월 OO그릇 판매" 같은 숫자 마케팅 준비 필요

### 6-1. 경쟁 지형 요약

```
                    |  미례국밥(2, 돼지우동) ← HERE
                    |
                낮은 매장수
```

미례국밥의 현재 위치: 매장 수 적지만 차별화 잠재력 높음

### 6-2. 미례국밥의 핵심 기회

1. **"돼지우동" 키워드 선점**: 아직 누구도 "돼지우동"이라는 키워드를 브랜드와 연결시키지 못함
2. **뉴웨이브 국밥 트렌드 탑승**: 2025년 배민 선정 3대 트렌드, 타이밍 정확
3. **서울 진출 = 뉴스**: "부산 돼지국밥의 서울 상륙"은 그 자체로 바이럴 소재
4. **혼밥 트렌드**: 국밥은 혼밥 최적화 메뉴, 1인 가구 증가와 맞물림
5. **로코노미**: "부산에서 온" 스토리는 서울 소비자에게 강력한 후킹

### 6-3. 미례국밥의 핵심 리스크

1. **동동국밥의 선행**: 유사 컨셉(우동+국밥)으로 이미 40개 매장, 동원홈푸드 MOU
2. **온라인 존재감 제로**: 검색해도 나오지 않는 상태에서 시작
3. **가격 저항**: 서울 소비자의 국밥 가격 인식 (10,000원 초과 시 저항)
4. **맛 표준화**: 프랜차이즈 확장 시 맛 품질 일관성 유지 과제
5. **소규모 매장(2개)의 신뢰도**: 가맹 희망자에게 "검증된 모델인가?" 의문

### 6-4. 우선순위 액션

| 순위 | 액션 | 목적 |
|------|------|------|
| 1 | "미례국밥" + "돼지우동" 네이버 검색량 기준선 측정 | KPI baseline |
| 2 | 동동국밥 대비 차별화 포인트 정의 | 경쟁 전략 |
| 3 | 서울 오픈 전 "부산 돼지우동" 바이럴 콘텐츠 제작 | 인지도 선행 |
| 4 | 인스타 블랙 계정 리브랜딩 (뉴웨이브 국밥 감성) | 브랜드 이미지 |
| 5 | 네이버 플레이스 부산 매장 최적화 | 검색 기반 |
