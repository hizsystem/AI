## 지금 상황

청우메디칼은 25년간 의료장비를 만들어온 기술력이 있고, 홈뷰티 디바이스 브랜드 **인셀라(INSELLA)**를 론칭합니다.

이미 준비된 것들:
- 큐티프로 EMS 패치 + 리프팅밴드 — "캡슐 구조"로 반복 구매 모델 확보
- 콘텐츠 80개 촬영 진행 중
- 4/20 홈페이지 오픈 확정
- 5/29 네이버 라이브 확정
- 권윤정 이사(뷰티 30년차)의 업계 네트워크

하지만 아직 이런 상태입니다:
- 인스타그램이 "바이어/MD가 봤을 때 세련된" 수준으로 세팅되지 않았습니다
- 시딩으로 초기 버즈를 만들 인프라가 없습니다
- 공동구매 셀러 연결이 안 되어 있습니다
- "25년 의료기기 기술"이라는 스토리가 콘텐츠로 전환되지 않았습니다

**기술은 있는데, 아직 세상에 인셀라를 보여줄 무대가 없습니다.**

## 파트너십 구조 — 4개 영역

브랜드라이즈가 담당하는 영역과, 이사님 측에서 직접 운영하시는 영역을 명확히 구분합니다.

### 영역 1. 인스타그램 브랜딩 & 운영 (브랜드라이즈 대행)

> "바이어가 봤을 때 세련된 브랜드"를 만드는 것

| 하는 일 | 구체적 내용 |
|---------|----------|
| 톤앤매너 세팅 | 피드 그리드 설계, 컬러 팔레트, 타이포 가이드, 무드보드 |
| 콘텐츠 기획 | 월간 콘텐츠 캘린더, 릴스/세로형 중심 기획 |
| 콘텐츠 제작 | 피드 디자인, 릴스 편집, 카피라이팅, 스토리 템플릿 |
| 채널 운영 | 게시, 해시태그 전략, 댓글 관리, DM 응대 가이드 |
| 성과 리포팅 | 주간 인사이트, 월간 성과 리포트 |

**월간 산출물**: 피드 12~15개 + 릴스 8~10개 + 스토리 20개+

### 영역 2. 인플루언서 시딩 (기획/매칭/QC)

> "톰뷰티 벤치마크하되, 인셀라만의 차별화"

| 하는 일 | 구체적 내용 |
|---------|----------|
| 시딩 전략 수립 | 물량 배분(뷰티에디터 / 마이크로 / 일반), 웨이브 설계 |
| 인플루언서 선정 | 카테고리별 리스트업, 적합도 스코어링, 섭외 |
| 가이드 제작 | 바이럴 포인트 가이드, 촬영 레퍼런스, 필수 멘션 |
| QC & 리포팅 | 콘텐츠 사전 검수, 성과 트래킹, 효율 분석 |

**월 목표**: 시딩 물량 300~500개/월, 인플루언서 콘텐츠 15~20개/월

### 영역 3. 공동구매 셀러 매칭

> "200억 하는 탑셀러" 연결 — 다른 에이전시가 못 주는 밸류

| 하는 일 | 구체적 내용 |
|---------|----------|
| 셀러 리스트업 | 뷰티 디바이스 카테고리 탑셀러 4인 프로필 + 실적 |
| 매칭 & 협상 | 셀러 미팅, 조건 협상 (착수금 + 수익 쉐어) |
| 공구 기획 | 셀러별 공구 타임라인, 특가 구조, 리뷰 이벤트 설계 |
| 실행 지원 | 셀러 전용 콘텐츠 제작, 상세페이지 감수, 성과 리포트 |

### 영역 4. Heyday 미국 시딩 루트

> "미국 에스테틱 샵에서 쓰는 디바이스" — PR 스토리 확보

| 하는 일 | 구체적 내용 |
|---------|----------|
| 루트 확인 | Heyday 시딩 가능 여부, 조건, 일정 확인 |
| 시딩 기획 | 미국 에스테틱 시장 맞춤 소재, 사용 후기 수집 설계 |
| PR 연결 | 시딩 결과물 → 국내 PR 스토리("미국 에스테틱 샵이 선택한 디바이스") |

## 경쟁사 포지셔닝 — 왜 지금, 이 방식인가

| 브랜드 | 전략 | 규모 | 시사점 |
|--------|------|------|--------|
| **메디큐브 (APR)** | 자체 콘텐츠 + 대규모 광고비 | 연매출 수천억 | 같은 판에서 싸우면 안 됨. 콘텐츠 퀄리티로 차별화 |
| **톰뷰티** | 시딩 → 공구 → 입소문 | 급성장 | **인셀라의 롤모델**. 시딩 물량으로 초기 버즈 → 공구로 전환 |
| **듀얼소닉** | 의료기기 기술 마케팅 | 중견 | 기술 스토리는 비슷. 하지만 "세련된 브랜딩"이 부족 |

**인셀라의 포지션**: "25년 의료기기 기술 + 세련된 뷰티 브랜딩" = 기술은 듀얼소닉급, 브랜딩은 톰뷰티급

## Quick Win — 계약 전에 먼저 드리는 것

큐티프로 EMS 패치+리프팅밴드의 "캡슐 구조"를 릴스 한 편으로 보여주는 스크립트:

**컨셉: "뷰티 디바이스계의 프린터 잉크 모델"**

```
[Hook] "이 디바이스, 한 번 사면 계속 돈 쓰게 됩니다"
[반전] "근데 이게 칭찬입니다"
[설명] 큐티프로 본체 → 캡슐 교체 장면 (ASMR 클로즈업)
[포인트] "프린터는 잉크로 돈 벌고, 큐티프로는 캡슐로 피부 관리"
[CTA] "캡슐 하나로 전문 에스테틱 한 번"
```

이 스크립트를 투자자/파트너 미팅에서도 쓸 수 있고, 실제 릴스로도 바로 제작 가능합니다.

## 자주 묻는 질문

**Q. 2억 중에 브랜드라이즈 비용이 5,300만원이면, 나머지는요?**
나머지 약 1.47억은 이사님이 직접 집행하시는 실비입니다 — 메타 광고비, 시딩 제품/물류, 공구 셀러 착수금 등. 브랜드라이즈는 그 돈이 최대 효율로 쓰이도록 설계하고 관리합니다.

**Q. 공구 매출이 안 나오면요?**
공구 셀러 매칭은 실적 기반으로 선정합니다. 셀러가 "이 제품은 안 된다"고 판단하면 매칭이 안 됩니다 — 매칭이 된다는 건 셀러도 "되겠다"고 본 겁니다. 그래도 리스크 분산을 위해 4명을 동시에 접근합니다.

**Q. 인스타 운영을 나중에 인하우스로 가져갈 수 있나요?**
네. 6개월간 브랜드 가이드, 콘텐츠 캘린더 템플릿, 해시태그 전략서를 만들어 드리기 때문에, 인하우스 마케터를 채용하시면 바로 인수인계 가능합니다.

**Q. Phase 단위로 끊을 수 있나요?**
Phase 1(2개월) 완료 후 성과를 보고 Phase 2 진행 여부를 결정하실 수 있습니다. 다만 6개월 풀 계약 시 전략의 연속성과 가격 효율이 훨씬 좋습니다.
