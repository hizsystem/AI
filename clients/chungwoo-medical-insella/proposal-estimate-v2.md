## 지금 상황

청우메디칼은 25년간 의료장비를 만들어온 기술력이 있고, 홈뷰티 디바이스 브랜드 **인셀라(INCELLA)**를 론칭합니다.

이미 준비된 것들:
- 큐티프로 EMS 패치 + 리프팅밴드 — "캡슐 구조"로 반복 구매 모델 확보
- 콘텐츠 80개 촬영 진행 중
- 4/20 홈페이지 오픈 확정
- 5/29 네이버 라이브 확정
- 권윤정 이사(뷰티 30년차)의 업계 네트워크

하지만 아직 이런 상태입니다:
- 인스타그램이 "바이어/MD가 봤을 때 세련된" 수준으로 세팅되지 않았습니다
- 인플루언서/시딩으로 초기 버즈를 만들 체계가 없습니다
- 공동구매 셀러 연결이 안 되어 있습니다
- "25년 의료기기 기술"이라는 스토리가 콘텐츠로 전환되지 않았습니다

**기술은 있는데, 아직 세상에 인셀라를 보여줄 무대가 없습니다.**

### 1. 인스타그램 — 5,500만원

> "바이어가 봤을 때 세련된 브랜드"를 만드는 것

| 하는 일 | 구체적 내용 |
|---------|----------|
| 톤앤매너 세팅 | 피드 그리드 설계, 컬러 팔레트, 타이포 가이드, 무드보드 |
| 콘텐츠 기획 | 월간 콘텐츠 캘린더, 릴스/세로형 중심 기획 |
| 콘텐츠 제작 | 피드 디자인, 릴스 편집, 카피라이팅, 스토리 템플릿 |
| 채널 운영 | 게시, 해시태그 전략, 댓글 관리, DM 응대 가이드 |
| 성과 리포팅 | 주간 인사이트, 월간 성과 리포트 |

## 일하는 방식

### 주간 리듬

| 요일 | 내용 |
|------|------|
| **월요일** | 주간 미팅 (30분) — 이번 주 할 일, 지난 주 성과 |
| **수요일** | 중간 체크 (카톡/슬랙) — 콘텐츠 시안 확인, 진행 상황 |
| **금요일** | 주간 리포트 — 숫자 + 인사이트 + 다음 주 계획 |

### 월간 산출물
- 월간 성과 리포트 (인스타 + 인플루언서 + 공구 + 광고 통합)
- 다음 달 콘텐츠 캘린더
- 인플루언서 효율 분석
- 비용 집행 내역

### 소통 채널
- 카카오톡 단체방 (실무 소통)
- 주간 미팅 (줌 또는 대면)
- 콘텐츠 승인: 피그마 또는 구글 드라이브

## 경쟁사 포지셔닝 — 왜 지금, 이 방식인가

| 브랜드 | 전략 | 시사점 |
|--------|------|--------|
| **메디큐브 (APR)** | 자체 콘텐츠 + 대규모 광고비 | 같은 판에서 싸우면 안 됨. 콘텐츠 퀄리티로 차별화 |
| **톰뷰티** | 시딩 → 공구 → 입소문 | **인셀라의 롤모델**. 시딩+공구로 초기 진입 → 입소문 전환 |
| **듀얼소닉** | 의료기기 기술 마케팅 | 기술 스토리는 비슷. "세련된 브랜딩"이 부족 |

**인셀라의 포지션**: "25년 의료기기 기술 + 세련된 뷰티 브랜딩" = 기술은 듀얼소닉급, 브랜딩은 톰뷰티급

## Quick Win — 계약 전에 먼저 드리는 것

큐티프로 EMS 패치+리프팅밴드의 "캡슐 구조"를 릴스 한 편으로 보여주는 스크립트:

**컨셉: "뷰티 디바이스계의 프린터 잉크 모델"**

```
[Hook] "이 디바이스, 한 번 사면 계속 돈 쓰게 됩니다"
[반전] "근데 이게 칭찬입니다"
[설명] 큐티프로 본체 → 캡슐 교체 장면 (ASMR 클로즈업)
[포인트] "프린터는 잉크로 돈 벌고, 큐티프로는 캡슐로 피부 관리"
[CTA] "캡슐 하나로 전문 에스테틱 한 번"
```

## 자주 묻는 질문

**Q. 공구 매출이 안 나오면요?**
공구 셀러 매칭은 실적 기반으로 선정합니다. 셀러가 "이 제품은 안 된다"고 판단하면 매칭이 안 됩니다 — 매칭이 된다는 건 셀러도 "되겠다"고 본 겁니다. 리스크 분산을 위해 5명을 동시에 접근합니다.

**Q. 인스타 운영을 나중에 인하우스로 가져갈 수 있나요?**
네. 6개월간 브랜드 가이드, 콘텐츠 캘린더 템플릿, 해시태그 전략서를 만들어 드리기 때문에, 인하우스 마케터를 채용하시면 바로 인수인계 가능합니다.

**Q. Phase 단위로 끊을 수 있나요?**
Phase 1(2개월) 완료 후 성과를 보고 Phase 2 진행 여부를 결정하실 수 있습니다. 다만 6개월 풀 계약 시 전략의 연속성과 가격 효율이 훨씬 좋습니다.
