## 운영 철학 (Operating Philosophy)

> "우리가 못 보는 것들을 제대로 코칭해주는구나"

| 원칙 | 의미 | 실행 |
|------|------|------|
| **코치, not 플레이어** | 제작/관리는 휴닉팀, 기획/디렉션/코칭은 HIZ | 모든 산출물은 휴닉팀이 1차 제작 → HIZ가 피드백 |
| **자생력 구축** | 파트너십 종료 후에도 휴닉팀이 독립 운영 가능하게 | 프레임워크/에셋/프로세스를 넘기고 교육 |
| **힘 빼기** | 매주 불필요한 작업에 시간 쓰지 않기 | 자동화 + 시스템 + 주간 체크로 관리 비용 최소화 |
| **주간 리듬** | 월간 보고가 아닌 주간 단위 체크 & 리포팅 | 주 1회 15분 체크인 + 자동 주간 보고서 |

## 1. 킥오프 전 사전 질문 (슬랙으로 미리 보내기)

> 목적: 킥오프 시간을 "정보 수집"이 아닌 "방향 합의"에 쓰기 위해

### 브랜드 & 비주얼
- 베지어트의 현재 브랜드 가이드라인(컬러, 폰트, 로고 파일 등)이 정리된 자료가 있나요? 있다면 공유 부탁드립니다.
- 베지어트 인스타그램 계정 로그인 정보 / Meta Business Suite 권한 공유 가능한가요?
- 현재 베지어트 인스타그램에서 "이건 잘 됐다"고 느꼈던 콘텐츠 3개, "이건 아쉬웠다"고 느꼈던 콘텐츠 3개를 각각 골라주실 수 있나요?
- 베지어트가 참고하고 싶은 (벤치마크) 인스타그램 계정이 있다면 알려주세요 (동종업계 아니어도 OK)

### 운영 현황
- 현재 콘텐츠 제작 프로세스가 어떻게 되나요? (기획 → 제작 → 승인 → 업로드 흐름)
- 인국님, 예랑님의 주간 업무 중 베지어트에 투입 가능한 시간은 대략 어느 정도인가요?
- Meta 광고 현재 운영 현황 (월 예산, 주요 캠페인 목적, ROAS 등)
- 올영(올리브영) 채널의 현재 매출 규모와 프로틴 카테고리 내 순위

### 빈커 관련
- 빈커 4월 이후 리테일 입점/데모 일정이 잡힌 것이 있다면 미리 공유 부탁드립니다
- 빈커 인스타그램 계정 현재 상태 (팔로워 수, 최근 포스팅 날짜 등)

### 업무 도구
- 현재 휴닉팀이 사용하는 업무 도구는? (노션, 슬랙, 피그마, 캔바 등)
- 콘텐츠 캘린더를 어디서 관리하고 계신가요?

## 5. 예산 관리

### 원칙
- BC3팀 마스터시트에서 통합 관리
- 월별 실제 집행 금액 vs 계획 금액 비교 추적
- 올영 체험단은 선급 차감 방식이므로 별도 잔액 관리 필요

### 예산 트래킹 시트 구조

```
[마스터시트 - HUENIC 탭]
├── 0) 전략 파트너십 Fee (월 100만 × 12 = 2,400만)
│   └── 월별 자동 차감
├── 1) 브랜딩 & 패키지 (1,250만)
│   └── 건별 완료 시 차감 (브랜드덱, 각 패키지별)
├── 2) 인스타그램 VEGGIET (3,000만)
│   ├── 리서치 (1,000만) — 상/하반기 각 500만
│   ├── 월간 기획보드 (2,000만) — 월 200만 × 10
│   └── 촬영 (잔액 소진) — 회차별
├── 3) 서울웰니스/캐나다 (3,000만) — 전환 확인 후 세부 항목
├── 4) VINKER (800만) — 월 80만 × 10
├── 5) 올영 체험단 (1,069만) — 선급 잔액 차감
└── 6) 기타 디자인/IMC — 발생 시 추가
```

### 주간 예산 체크 포인트
- 매주 금요일: 이번 주 발생 비용 기록
- 매월 말: 월간 정산 (실 집행 vs 계획)
- 분기별: 잔여 예산 리뷰 & 후반기 재배분

## 6~7. 프로젝트 관리 도구 선택

### 추천: 하이브리드 접근

| 용도 | 도구 | 이유 |
|------|------|------|
| **예산/데이터 관리** | 구글 시트 (마스터시트) | BC3 통합 관리, 수식/자동화 용이 |
| **프로젝트/태스크 관리** | 노션 | 휴닉팀과 공유 가능, 칸반/타임라인 뷰 |
| **디자인 에셋/무드보드** | 피그마 | 인스타 비주얼 작업의 표준 도구 |
| **콘텐츠 기획 & 보고서** | 웹 대시보드 (커스텀) | 아래 8~9번 참조 |
| **일상 소통** | 슬랙 | 이미 사용 중 |

### 시스템 구축 vs 기존 도구 판단

**단계적 접근을 추천합니다:**
1. **Phase 0 (3~4월)**: 노션 + 구글시트 + 피그마로 빠르게 시작
2. **Phase 1 (5월~)**: 반복 업무 패턴이 잡히면, 그때 웹 대시보드로 통합
3. 이유: 처음부터 시스템 구축하면 실제 운영 패턴을 모른 채 만들게 됨

## 9. 콘텐츠 기획 & 보고서 — 웹 대시보드

### 핵심 니즈: "탭샵바처럼 효율적으로"

### 웹 대시보드에 포함할 기능

**A. 콘텐츠 캘린더**
- 월간/주간 뷰
- 각 콘텐츠: 카테고리(브랜딩/활동성/시딩), 담당자, 상태(기획/제작/리뷰/완료)
- 드래그앤드롭으로 일정 조정
- 베지어트/빈커 필터

**B. 콘텐츠 카드**
- 콘텐츠 1개 = 카드 1개
- 기획 의도, 캡션 초안, 해시태그, 이미지/영상 첨부
- 리뷰 코멘트 (HIZ → 휴닉)
- 성과 데이터 자동 연결

**C. 주간 리포트 (자동 생성)**
- 이번 주 발행 콘텐츠 목록 + 성과
- 팔로워 변화, 도달, engagement rate
- Top 콘텐츠 / 저성과 콘텐츠
- 다음 주 플랜 요약
- PM 코멘트 (한 줄 인사이트)

**D. 대시보드 (KPI 트래킹)**
- 팔로워 추이 (목표선 대비)
- 월별/주별 콘텐츠 수량
- engagement rate 추이
- 올영 카테고리 순위
- 예산 소진율

### 구현 방식 옵션

## 11~12. 코치형 운영 모델

### "코치" vs "선수" 역할 구분

```
[HIZ = 코치]                    [휴닉팀 = 선수]

 전략 방향 설정          ───→    실행
 프레임워크 제공         ───→    프레임워크로 기획
 피드백 & 코칭          ←───    콘텐츠 초안 공유
 성과 분석 & 인사이트    ───→    다음 액션 반영
 디자인 에셋 시스템      ───→    에셋 활용하여 제작
 교육 & 레슨            ───→    학습 & 자생력 구축
```

### 구체적 코칭 방식

**1. 기획 레슨 (초기 집중)**
- 첫 기획회의에서 "콘텐츠 기획 프레임워크" 전수
- 왜 이 콘텐츠를 만드는지 (목적), 누구에게 (타겟), 어떤 반응을 원하는지 (CTA)
- 이후 휴닉팀이 프레임워크로 기획 → HIZ가 피드백

**2. 피드백 루프 (매주)**
- 휴닉팀이 콘텐츠 기획안/시안 공유 (수요일까지)
- HIZ가 "코치 코멘트"로 피드백 (목요일까지)
- 코멘트 형식: "잘한 점 1개 + 개선 포인트 1개 + 다음에 시도해볼 것 1개"
- 절대 "이렇게 바꿔라"가 아닌 "이런 관점에서 보면 어떨까요?"

**3. 자생력 지표 (3개월 단위 체크)**
- 기획안 수정 횟수가 줄어들고 있는가?
- 디자인 에셋을 자체적으로 활용하고 있는가?
- 데이터를 보고 스스로 인사이트를 뽑고 있는가?
- → 이 지표가 좋아지면 우리 비용 대비 가치가 올라가는 것

### 힘 빼기 전략

| 하지 않을 것 | 대신 할 것 |
|-------------|----------|
| 콘텐츠 직접 제작 | 제작 가이드 & 피드백 |
| 매일 슬랙 모니터링 | 주 3회 정해진 시간에 체크 |
| 상세한 기획서 작성 | 기획 프레임워크 + 월간 방향만 제시 |
| 수치 일일 트래킹 | 자동 대시보드로 주간 확인 |
| 빈커 콘텐츠 고퀄 제작 | 최소 퀄리티로 유지 (에셋 재활용) |
