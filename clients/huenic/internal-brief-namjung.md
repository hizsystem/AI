## Part 2. 인플루언서 & 시딩 방향

### 킥오프 합의 사항
- 현재 러너 3명 확보 상태, 추가 섭외 필요
- 유가 시딩: 대용량 본품 1개 + 파우치 1박스
- 서리태: 즉시 발송 가능 / 딸기: 3/27 이후

### 베러픽 기획에서 확정된 방향 (진아 대표 지시)

**서포터즈 X → 웰니스 피플 협업으로 전환**

| 기존 방향 | 변경 방향 |
|---------|---------|
| 서포터즈 (방문형) | 요가, 바레, 필라테스 웰니스 피플 협업 |
| 러너 중심 | 러너 + 웰니스 피플 (헬스, 러닝, 요가, 바레 전부) |
| 방문형 필수 | 방문형 아니어도 OK |

### 인플루언서 분류

| 분류 | 역할 | 콘텐츠 CTA | 예산 |
|------|------|----------|------|
| **앰버서더** (유민/혜림/정민/현지) | 베러픽 트래픽 콘텐츠 업로드 | 올리브베러 방문 유도 | 베러픽 예산 (예랑 관리) |
| **유가 시딩** (러너/웰니스 피플) | 자연스러운 제품 통합 콘텐츠 | "운동 후 속 편한 식물성 단백질" | 제품 제공 |
| **콜라보** (요가 피플 등) | 올리브베러픽 CTA 포함 협업 콘텐츠 | "올리브베러에서 만나보세요" | 협찬 500만 (조정) |

### 남중 액션 아이템

| # | 할 일 | 기한 |
|---|------|------|
| 1 | **유가 시딩 리스트 정리** — 러너 + 요가/바레/필라테스 인플루언서 후보 10~15명 | 3/24 |
| 2 | **시딩 가이드라인 초안** — 베지어트 브리프 템플릿 (카피 톤, 금지어, CTA 가이드) | 3/25 |
| 3 | **서리태 시딩 발송 준비** — 4/1 이후 순차 발송, 발송 리스트 + 트래킹 시트 | 3/31 |
| 4 | **러너 추가 섭외** — 현재 3명 → 목표 5~7명 | 4/7 |
| 5 | 웰니스 피플 콜라보 후보 리스트 — 예랑님과 공유 (베러픽 협찬 예산 연계) | 3/25 |

### 시딩 → 콘텐츠 → 메타 광고 플로우

```
제품 발송 (서리태/딸기)
  → 인플루언서 콘텐츠 업로드 (자연스러운 제품 통합)
    → 베지어트 계정 리포스트 (스토리)
      → 공동 마케팅 신청
        → META 광고 소재로 활용
```

이 플로우에서 남중 역할: 발송 → 콘텐츠 수급 → 리포스트 큐레이션까지.
메타 광고 연결은 인국님 담당.

### 시딩 콘텐츠 브리프 핵심 (인플루언서에게 전달할 것)

- "운동 후 챙겨 먹는 식물성 단백질" 맥락으로 자연스럽게
- CTA: "올리브베러에서 만나보세요" (4월 한정)
- 금지어: 대체육, 대체식품, 비건(단독), 최고/최강
- 촬영 가이드: 자연광, 일상 속 장면, 스튜디오 느낌 X
- 해시태그: #베지어트 #VEGGIET #식물성단백질 #올리브베러픽
