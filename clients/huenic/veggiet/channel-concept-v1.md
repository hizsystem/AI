## 1. 채널 포지셔닝 스테이트먼트

**"식물성 단백질의 기준을 만드는 브랜드, 베지어트의 세계관을 보여주는 공간."**

- 제품 정보 채널이 아니라 **식물성 웰니스 라이프스타일을 정의하는 브랜드 채널**
- 팔로워가 베지어트를 따라가는 게 아니라, 베지어트가 제안하는 루틴에 자연스럽게 합류하는 느낌
- 목표: "이 브랜드 감도 뭐야?" -> 프로필 클릭 -> 팔로우

---

## 2. 브랜드 보이스 키워드 3가지

| 키워드 | 정의 | 콘텐츠 적용 |
|--------|------|------------|
| **진정성(Authenticity)** | 식물성 단백질을 진심으로 만드는 팀의 이야기. 꾸미지 않되 감도는 높게 | 대표/팀 비하인드, 원료 스토리, 제조 과정 |
| **기준(Standard)** | "식물성 단백질은 베지어트"라는 카테고리 리더십. 1등의 톤 | 제품 USP, 성분 비교, 브랜드 철학 |
| **일상(Everyday)** | 특별한 날이 아닌 매일의 루틴 속 베지어트. 가볍고 자연스러운 접점 | TPO 콘텐츠, 레시피, 모먼트 |

## 3. 톤앤매너 가이드

### 3-1. 말투

- **기본 톤**: 담백하고 자신감 있는 1인칭 브랜드 보이스. 친근하되 가볍지 않다.
- **문체**: 짧은 문장 선호. 군더더기 없이. 여운을 남기는 마침표.
- **호칭**: "당신" 또는 직접 호칭 없이 상황 묘사 ("아침에 눈 뜨고 제일 먼저 하는 일")

**톤 스펙트럼**
```
[차갑다] --------- [베지어트] --- [따뜻하다]
                       ^
              담백하고 건조하되
              진심이 느껴지는 온도
```

**예시 카피**
- O: "속이 편한 아침. 오늘도 식물성으로."
- O: "서리태 한 스쿱이면 충분합니다."
- X: "여러분~ 오늘도 건강한 하루 보내세요!"
- X: "대체육의 혁명! 놀라운 맛!"

### 3-2. 금지어 / 주의어

| 금지 | 대체 표현 |
|------|----------|
| 대체육, 대체식품 | 식물성 식품, 식물성 단백질 |
| 비건(단독 사용) | 100% 식물성, 식물성 기반 |
| 혁명, 혁신 | 기준, 새로운 선택지 |
| ~해요체 과다 | 해요체 최소화, 문어체/명사형 종결 선호 |
| 최고, 최강, 넘버원 | "식물성 단백질의 기준" (자연스러운 1등 암시) |

### 3-3. 해시태그 체계

- 브랜드 고정: #베지어트 #VEGGIET #식물성단백질
- 시리즈별: #이달의베지어터 #베지어트랩 #베지어트모먼트 (각 시리즈 참조)
- 확장 키워드: #프로틴 #웰니스 #식단관리 #아침루틴 #운동후단백질
- 국문/영문 혼용: 캡션 하단에 영문 1~2줄 병기 (해외 영업 활용)

## 4. 비주얼 디렉션

### 4-1. 컬러 팔레트 방향

```
Primary
-- 베지어트 그린 (브랜드 키컬러, 기존 BI 연장)
-- 크림 화이트 (배경, 여백)
-- 웜 베이지 (서브, 텍스트 배경)

Secondary
-- 딥 브라운 (텍스트, 타이포)
-- 소프트 테라코타 (포인트, 따뜻한 감성)
-- 라이트 세이지 (신선함, 원재료 연상)

금지 컬러
-- 네온/형광 계열 (B급 느낌 유발)
-- 순수 블랙 배경 (브랜드 무드와 불일치)
-- 과채색 컬러 (시장, 마트 느낌 유발)
```

### 4-2. 사진 스타일

**지향점**: AG1의 정돈된 프리미엄감 + Sweetgreen의 자연스러운 일상감

| 항목 | 가이드 |
|------|--------|
| 라이팅 | 자연광 기본. 소프트하고 따뜻한 톤. 플래시/인공 조명의 날카로운 그림자 지양 |
| 구도 | 여백 충분히. 오브제 최소화. 제품이 화면의 40~60% |
| 배경 | 크림/베이지/리넨/우드/대리석. 깨끗하되 차갑지 않은 소재 |
| 인물 | 얼굴 클로즈업보다 실루엣, 손, 상반신 크롭. 자연스러운 동작 |
| 식물성 원료 | 서리태, 딸기 등 원재료를 스타일링 소품처럼 활용 (산지 느낌 X, 테이블 세팅 O) |
| 텍스트 오버레이 | 산세리프 서체. 좌측 또는 하단 정렬. 화면의 20% 이하 |

**피드 그리드 무드**
- 3열 그리드에서 전체적으로 **크림~그린 톤의 통일감**
- 제품 컷 / 라이프 컷 / 타이포 컷을 번갈아 배치하여 리듬감
- 한 줄(3개)을 봤을 때 "하나의 브랜드"로 읽히는 것이 기준

### 4-3. 타이포그래피 방향

| 용도 | 방향 |
|------|------|
| 타이틀 | 산세리프 계열, 웨이트 Medium~Bold. 영문은 대문자 활용 |
| 본문 | 가독성 우선. 세리프 or 산세리프 Light~Regular |
| 포인트 | 손글씨체 지양. 감도 = 정돈된 타이포에서 나온다 |
| 국/영문 혼용 | 한글 타이틀 아래 영문 서브타이틀 배치 (해외 영업 활용 겸) |

## 5. 피드 구성 비율

킥오프 합의 기준: **브랜딩 70% + 활동성(참여/소통) 30%**

### 상세 배분 (월 10~12개 기준)

| 카테고리 | 비율 | 월 개수 | 콘텐츠 성격 |
|---------|------|--------|------------|
| 브랜딩 -- 제품 | 30% | 3~4개 | 제품 USP, 신제품, 레시피, 성분 |
| 브랜딩 -- 브랜드 스토리 | 20% | 2~3개 | 철학, 비하인드, 대표 스토리, 원료 |
| 브랜딩 -- 라이프스타일 | 20% | 2~3개 | TPO 모먼트, 웰니스 루틴, 감성 |
| 활동성 -- 사람/커뮤니티 | 20% | 2~3개 | 이달의 베지어터, 러닝크루, 인플루언서 |
| 활동성 -- 프로모션/참여 | 10% | 1~2개 | 올영 연계, 챌린지, 이벤트 |

### 릴스 vs 피드

| 포맷 | 비중 | 용도 |
|------|------|------|
| 피드 (이미지/캐러셀) | 60% | 브랜드 감도, 제품, 정보 전달 |
| 릴스 (숏폼 영상) | 30% | 레시피, 비하인드, 러닝크루, 바이럴 |
| 스토리 | 상시 | 일상 공유, 투표, Q&A, 리포스트 (KPI 별도) |

## 6. 경쟁사 대비 차별 전략

### 경쟁 지형 분석

| 경쟁사 | 포지셔닝 | 인스타 감도 | 베지어트와의 차이 |
|--------|---------|-----------|-----------------|
| 프로티원 | 올영 가성비 프로틴 1위 | 중 (프로모션 중심) | 브랜드 감도 부재, 가격 전쟁 |
| 플라이밀 | 간편식 + 프로틴 | 중 (기능 소구) | 식물성 전문 아님 |
| 홀블렌딩 | 약사가 만든 클린 프로틴 | 중상 (전문성 어필) | 전문가 권위 vs 브랜드 세계관 |
| 슈퍼말차 | 프리미엄 감도, 힙한 라이프 | 상 (비주얼 최상급) | 말차 카테고리, 식물성 단백질 아님 |
| CJ 얼티브 | 대기업 자본, 유통 장악 | 중 (대기업 톤) | 스타트업 진정성 부재 |

### 베지어트의 무기: 진정성 + 감도

대기업이 못 하는 것 = 베지어트가 해야 하는 것

1. **창업자의 얼굴과 목소리** -- CJ는 "누가 만들었는지" 보여줄 수 없다. 베지어트는 박진아 대표의 철학, 인국님의 제작 과정을 있는 그대로 보여줄 수 있다. 단, B급이 아니라 감도 높은 포맷으로.

2. **카테고리 순수성** -- 프로티원은 유청 단백질도 판다. 베지어트는 100% 식물성만 한다. 이 "하나만 한다"는 것 자체가 브랜드다.

3. **커뮤니티 리얼리티** -- 대기업 체험단은 스크립트가 보인다. 베지어트의 러닝크루, 실제 유저 인터뷰는 진짜 경험이다.

4. **해외 감도의 한국화** -- AG1, Sweetgreen 수준의 비주얼을 한국 시장에서 식물성 프로틴으로 구현하는 브랜드는 아직 없다.

### 경쟁 우위 실행 원칙

| 원칙 | 구체 행동 |
|------|----------|
| 1등은 1등답게 말한다 | "식물성 단백질의 기준"이라는 프레이밍. 비교하지 않고 정의한다 |
| 진정성은 편집에서 나온다 | 날것을 보여주되, 감도 높은 컬러그레이딩과 구도로. 대충 찍은 것과 자연스러운 것은 다르다 |
| 반복이 브랜드를 만든다 | 시리즈 꼭지를 정하고, 같은 포맷을 반복. 3개월 뒤 "아 그 시리즈" 하면 성공 |
| 숫자보다 장면을 남긴다 | "단백질 20g" 대신 "운동 끝나고 한 잔, 속이 편했다" |

## 7. 운영 프로세스 & R&R

### 역할 분담

| 단계 | HIZ | 휴닉 |
|------|-----|------|
| 월간 기획 | 콘텐츠 캘린더 + 시리즈별 브리프 제공 | 기획안 검토 + 피드백 |
| 레퍼런스 | 시리즈별 레퍼런스 3~5개 큐레이션 | 촬영 전 참고 |
| 제작 | 카피 초안 + 디자인 에셋(수민) | 촬영, 편집, 디자인 완성(인국, 예랑) |
| 리뷰 | 업로드 전 최종 검수 | 수정 반영 |
| 업로드 | - | 계정 운영, 업로드, DM 관리 |
| 성과 분석 | 주간/월간 리포트, KPI 트래킹 | 데이터 공유 (Meta Business Suite) |

### 주간 리듬

```
월요일: 금주 콘텐츠 최종 확인 (슬랙)
화/목: 업로드 (주 2~3회)
금요일: 차주 콘텐츠 브리프 전달 (HIZ -> 휴닉)
월말:  다음 달 캘린더 공유 + 월간 리뷰
```
