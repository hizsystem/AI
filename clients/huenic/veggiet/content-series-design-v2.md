# [VEGGIET] 콘텐츠 시리즈 설계 v2 — Concern-First

> 작성: 2026-03-21 | HIZ x HUENIC
> 변경점: v1은 "브랜드가 할 말"에서 출발. v2는 "타겟의 진짜 고민"에서 출발.
> 원칙: 프로틴을 먹는 25~40 여성의 concern에 답하는 콘텐츠 = 시리즈

---

## 설계 원칙

```
v1: 브랜드가 할 말 → 시리즈 → 콘텐츠
v2: 타겟의 진짜 고민 → 그 고민에 답하는 콘텐츠 → 시리즈
```

사람들은 "베지어트의 철학"이 궁금해서 인스타를 보지 않는다.
**자기 고민을 해결해줄 것 같아서** 멈춘다.
그래서 콘텐츠의 출발점은 항상 "이 사람이 지금 무슨 고민을 하고 있는가?"여야 한다.

## 타겟 프로필

| 항목 | 내용 |
|------|------|
| 코어 타겟 | 30대 여성 |
| 확장 | 25~45세 여성 |
| 공통점 | 건강은 챙기고 싶지만, 먹는 걸로 스트레스 받고 싶지 않음 |
| 프로틴 경험 | 안 먹어봄 ~ 먹다 말았음 ~ 지금 먹고 있음 (전 단계 포함) |

## 5가지 Concern → 5개 콘텐츠 필라

### Concern 1. "이게 나한테 맞아?"

> 머릿속: "프로틴은 운동하는 사람이 먹는 거 아니야?"
> "뭘 골라야 하지? 유청? 식물성? 너무 많아"
> "나 같은 사람이 먹어도 되는 건가?"

**해결**: 프로틴의 문턱을 낮춰주는 콘텐츠. "당신도 먹어도 됩니다"가 아니라, **당신 같은 사람이 이미 먹고 있는 장면**을 보여준다.

#### 시리즈: 이달의 베지어터 (구 B-1)

| 항목 | 내용 |
|------|------|
| 한 줄 | 나와 비슷한 사람이 베지어트를 먹는 이유 |
| 포맷 | 캐러셀 (5~7장, 인터뷰+라이프 사진) or 릴스 (60~90초) |
| 빈도 | 월 1회 |
| 톤 | 29CM 하우스 레퍼런스. 일상 공간, 자연광, 진짜 사람 |

**핵심**: 러너, 직장인, 엄마, 요가 하는 사람 — 각기 다른 사람이 "왜 베지어트인지"를 자기 언어로 말함. 이걸 본 잠재 고객이 "아 나랑 비슷한 사람이 먹네" → 진입 장벽이 낮아짐.

**예시 주제**
1. "러너 지은 씨의 아침 루틴" — 러닝 전후 식단
2. "두 아이 엄마 현정 씨의 간식 혁명" — 가족 건강
3. "필라테스 강사 수진 씨가 프로틴을 고르는 기준" — 식물성을 선택한 이유

---

### Concern 2. "진짜 괜찮은 거야?"

> 머릿속: "첨가물 잔뜩 들어간 거 아냐?"
> "식물성이라면서 맛이 있을 수 있어?"
> "이 브랜드 믿을 만해?"

**해결**: 성분과 제조 과정을 **설명하지 않고 보여준다.** 말로 "우리 깨끗해요" 하면 광고. 서리태 원료를 시네마틱하게 보여주면 신뢰.

#### 시리즈: 베지어트 랩 (구 D-1)

| 항목 | 내용 |
|------|------|
| 한 줄 | 이 한 스쿱이 만들어지기까지 |
| 포맷 | 캐러셀 (비하인드 사진 3~5장) or 릴스 (제조 공정 영상) |
| 빈도 | 월 1회 |
| 톤 | Sweetgreen의 "식재료가 히어로". 원재료를 가장 매력적으로 |

**핵심**: 성분표를 나열하는 게 아니라, **서리태가 어디서 왔는지, 어떻게 프로틴이 되는지**를 장면으로 보여줌. 홀블렌딩이 "약사가 설명"하는 걸 베지어트는 "보여주는 것"으로 대체.

**예시 주제**
1. "서리태는 어디서 오는가" — 원료 산지, 계약 농가
2. "한 스쿱에 담긴 것들" — 원료 → 배합 → 품질검사 → 패키지
3. "딸기 프로틴이 탄생하기까지" — 개발 237일 비하인드

#### 시리즈: 베지레터 (신규 — 인국님 아이디어)

| 항목 | 내용 |
|------|------|
| 한 줄 | 실제 고객이 하는 말 = 가장 강력한 신뢰 |
| 포맷 | 릴스 (고객 리뷰 읽기, 딜라이틴 무드) or 캐러셀 |
| 빈도 | 월 1~2회 |
| 톤 | 고객의 말을 그대로. 재밌는 리뷰, 감동적인 리뷰, 날카로운 질문 |

**핵심**: "찐후기가 클릭률이 높다"는 인국님의 데이터 인사이트가 정확한 이유 — 고객의 concern에 고객이 답하고 있기 때문.

**예시 주제**
1. "한 통 다 비우고 재구매한 단백질은 처음이에요" — 재구매 후기
2. "다른 단백질은 당이 0.1g인데 여기는 4g이네요" — 날카로운 질문 + 솔직한 답변
3. "비린맛 없이 고소한 미숫가루 맛" — 맛 concern 해소

### Concern 3. "맛있어? 어떻게 먹어?"

> 머릿속: "전에 프로틴 먹어봤는데 비린내 나서 못 먹겠더라"
> "맨날 물에 타먹으니까 질렸어"
> "아침에 뭘 먹지? 빵은 죄책감, 굶으면 점심에 폭식"

**해결**: 맛과 활용법을 **직접 보여준다.** "맛있어요"라고 말하면 광고. 만들어서 먹는 장면을 보여주면 "나도 저렇게 먹어볼까" 가 된다.

#### 시리즈: 베지어트 한 끼 (구 A-2)

| 항목 | 내용 |
|------|------|
| 한 줄 | "이렇게 먹으면 맛있어요"가 아니라 "이렇게 먹고 있어요" |
| 포맷 | 릴스 (30~60초, 레시피/활용법) or 캐러셀 |
| 빈도 | 월 1~2회 |
| 톤 | 짧고 감각적. 식욕 자극 비주얼 |

**예시 주제**
1. "아침 5분 프로틴 볼" — 오트밀 + 서리태 + 과일
2. "얼죽아: 얼음 넣으면 미숫가루 라떼" — 인국님 아이디어, 바이럴 가능
3. "스파우트 파우치 얼먹" — 잘라서 아이스크림처럼 (전환형 바이럴)

#### 시리즈: 쉐이크 유형 (신규 — 인국님 아이디어, 구 B-3 편입)

| 항목 | 내용 |
|------|------|
| 한 줄 | 베지어트 팀이 실제로 먹는 방법 = 리얼 + 재미 |
| 포맷 | 릴스 (밈 포맷, 60~90초) |
| 빈도 | 월 1회 |
| 톤 | 밈/유형 테스트 느낌. 유쾌하되 감도 유지 |

**예시 주제**
1. "베지어트 팀의 쉐이크 먹는 유형 6가지" — 얼죽아형, 전완근폭주형, 내새끼형...
2. "모닝루틴 유형 2탄" — 원샷형, 홍보대사형, 인간휘핑기형...

**이 시리즈가 Concern 3에 답하는 이유**: "맛있어? 어떻게 먹어?"에 대해 브랜드가 "이렇게 드세요"라고 말하는 게 아니라, **만드는 사람들이 실제로 이렇게 먹고 있다**는 걸 보여줌. 리얼함이 설득력.

---

### Concern 4. "먹으면 뭐가 달라져?"

> 머릿속: "한 달째 먹는데 뭐가 달라진 건지 모르겠어"
> "배고픈데 건강하게 배부르고 싶어"
> "운동 후에 뭘 먹어야 할지 모르겠어"

**해결**: 효과를 "성분 숫자"로 증명하지 않고, **일상 속 변화의 장면**으로 보여준다.

#### 시리즈: 베지어트 모먼트 (구 C-1)

| 항목 | 내용 |
|------|------|
| 한 줄 | 베지어트가 있는 일상의 한 장면 |
| 포맷 | 피드 (무드 사진 1장 + 짧은 카피) |
| 빈도 | 월 2회 |
| 톤 | 담백하고 건조하되 진심이 느껴지는 온도 |

**핵심**: "단백질 20g"이 아니라 **"운동 끝나고 한 잔, 속이 편했다"**. 숫자가 아니라 장면이 설득한다.

**예시 주제**
1. "토요일 아침, 늦잠 자고 일어나서 첫 한 잔"
2. "회의 끝나고 3시의 리셋"
3. "운동 가방에 항상 들어있는 것"

**예시 카피**
- "아침에 한 잔 마시면 점심때까지 배가 안 고프다. 그게 전부예요."
- "운동 후에 떡볶이 생각이 줄었다. 속이 편하니까 자극적인 게 안 땡기더라."

---

### Concern 5. "이 브랜드는 뭐가 달라?"

> 머릿속: "프로티원이랑 뭐가 다른 거야?"
> "CJ 얼티브가 더 싸던데"
> "홀블렌딩은 약사가 만들었다는데"

**해결**: 비교하지 않고 **자기 기준을 선언한다.** 1등은 1등답게 말한다.

#### 시리즈: 식물성의 기준 (구 D-2)

| 항목 | 내용 |
|------|------|
| 한 줄 | 베지어트가 정의하는 "좋은 프로틴"의 기준 |
| 포맷 | 피드 (타이포그래피 중심 + 본문 카피) |
| 빈도 | 월 1회 |
| 톤 | 선언적. 짧고 자신감 있게. |

**핵심**: "우리가 1위예요"가 아니라, **"식물성 단백질의 기준은 이것이다"**라고 정의하면 자연스럽게 1위 포지션이 됨. 비교 없이 선점.

**예시 카피**
1. "우리는 '대체'하지 않습니다. 새로운 기준을 만듭니다."
2. "100% 식물성만 합니다. 그게 우리의 기준입니다."
3. "속이 편한 단백질이 좋은 단백질입니다."

#### 시리즈: 만드는 사람들 (구 B-3)

| 항목 | 내용 |
|------|------|
| 한 줄 | 대기업이 보여줄 수 없는 것 = 만드는 사람의 얼굴과 태도 |
| 포맷 | 피드 (사진 + 짧은 텍스트) or 릴스 (무음 자막) |
| 빈도 | 격월 1회 |
| 톤 | 따뜻하되 정돈된. 파타고니아/마일스톤커피 레퍼런스 (예랑님 제시) |

**핵심**: CJ가 "누가 만들었는지" 보여줄 수 있어? 없어. 베지어트는 할 수 있어. 이게 스타트업의 무기. 단, B급이 아니라 **감도 높은 포맷**으로.

**B급 방지 가이드**
- 대표님 등장: 정면 인터뷰 X → 손/뒷모습/사이드 앵글 + 자막
- 예랑님 나이 문제: 예랑은 "전달자"가 아니라 "일상을 보여주는 사람"으로. 설득은 대표님 or 고객 목소리가.
- 조명, 구도, 컬러그레이딩에 신경. 날것이되 정돈된 날것.

## 신제품 론칭 (Concern 횡단)

신제품 론칭은 특정 concern 하나에 속하지 않고, **여러 concern을 동시에 건드리는 캠페인**.

#### 시리즈: New Drop (구 A-3)

| 항목 | 내용 |
|------|------|
| 한 줄 | 신제품 출시를 브랜드 이벤트처럼 연출 |
| 구조 | 티저 → 론칭 → 리뷰 3단계 |
| 빈도 | 출시 시 |

**4월 딸기 론칭 Concern 매핑**:
- 티저 (3/26, 3/28): Concern 5 "이 브랜드는 뭐가 달라?" → 기대감
- 론칭 (4/1): Concern 1 "이게 나한테 맞아?" → 새로운 선택지
- 리뷰 (4/14): Concern 3 "맛있어?" → 딸기 레시피 릴스

## v1 → v2 변경 대조표

| v1 (브랜드 기준) | v2 (Concern 기준) | 바뀐 점 |
|---------------|-----------------|--------|
| A축. 제품 | Concern 3. 맛/활용 | 출발점이 "USP 전달"에서 "맛 concern 해소"로 |
| B축. 사람 | Concern 1. 나한테 맞아? + Concern 5. 뭐가 달라? | "사람을 보여주자"가 아니라 "이 사람도 먹네 = 나도 OK" |
| C축. 라이프 | Concern 4. 뭐가 달라져? | "라이프스타일 무드"가 아니라 "먹고 나서 달라진 일상" |
| D축. 브랜드 | Concern 2. 괜찮은 거야? + Concern 5. 뭐가 달라? | "철학 전달"이 아니라 "신뢰 구축" + "포지션 선언" |

## 이 설계를 기획안에서 어떻게 말할까

3/25 슬라이드에서:

> "프로틴을 먹는 사람들에게는 5가지 고민이 있습니다.
>
> 1. 이게 나한테 맞아?
> 2. 진짜 괜찮은 거야?
> 3. 맛있어? 어떻게 먹어?
> 4. 먹으면 뭐가 달라져?
> 5. 이 브랜드는 뭐가 달라?
>
> 베지어트 인스타의 모든 콘텐츠는 이 5가지 중 하나에 답합니다.
> 그것이 시리즈가 되고, 시리즈가 반복되면 브랜드가 됩니다."

## 부록: 콘텐츠 브리프 템플릿 (v2)

각 콘텐츠 제작 시 HIZ가 휴닉에 전달하는 브리프:

