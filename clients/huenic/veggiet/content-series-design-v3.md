# [VEGGIET] 콘텐츠 시리즈 설계 v3 — 최종 정리

> 작성: 2026-03-24 | HIZ x HUENIC
> v2 → v3 변경: 시리즈 8개 → 6+1로 정리. 인터뷰 소스 연결. 비주얼 방향 구체화.
> 용도: 3/25 기획 공유 + 팀 내부 미팅 기반

---

## 브랜드 메시지 (확정)

> **"매일 먹어도 속 편한, 식물성 프로틴의 기준"**

이 메시지의 뿌리 — 대표 인터뷰에서 확인:
- 남편 궤양성대장염 8년 → 식물성 식단 1년 만에 염증 소멸
- "속 편함"은 마케팅 카피가 아니라 **창업 동기 그 자체**

**채널별 메시지 변환:**
- 올영에서 사게 만드는 말 = "속 편하고, 맛있고, 한 끼 되는" (실용)
- 인스타를 팔로우하게 만드는 말 = "이 브랜드를 먹는 나, 괜찮다" (감정/정체성)

## 시리즈 전체 구조

| # | 시리즈 | 역할 | 포맷 | 빈도 |
|---|--------|------|------|------|
| 1 | 이달의 베지어터 | 발견+설득 | 캐러셀/릴스 | 월 1 |
| 2 | 베지어트 랩 | 신뢰(유지) | 캐러셀/릴스 | 월 1 |
| 3 | 베지레터 | 발견+확산 | 릴스/캐러셀 | 월 1~2 |
| 4 | 베지어트 한 끼 | 발견 | 릴스/캐러셀 | 월 1~2 |
| 5 | 베지어트 모먼트 | 설득(쇼룸) | 피드 | 월 2 |
| 6 | 만드는 사람들 | 유지(팬덤) | 피드/릴스 | 격월 1 |
| + | 이벤트/프로모션/소식 | 전환 | 피드/릴스 | 수시 |

**월간 총: 약 10~12개** (이벤트 별도)

### v2 → v3 변경점
- **쉐이크 유형** → 베지어트 한 끼에 통합 (레시피+유형 테스트 하나로)
- **식물성의 기준** → 베지어트 모먼트 카피에 녹임 (독립 시리즈 필요 없음, 모먼트 카피가 그 역할)
- **모먼트 + 만드는사람 일상** 합체 → 모먼트가 팀 일상까지 커버
- **만드는 사람들** → 대표님 인터뷰/팀 스토리 중심, 격월 특별 콘텐츠로 분리

## 1. 이달의 베지어터

> Concern: "이게 나한테 맞아?"
> 한 줄: 나와 비슷한 사람이 베지어트를 먹는 이유

| 항목 | 내용 |
|------|------|
| 포맷 | 캐러셀 (5~7장, 인터뷰+라이프 사진) or 릴스 (60~90초) |
| 빈도 | 월 1회 |
| 톤 | 29CM 하우스. 일상 공간, 자연광, 진짜 사람 |
| 역할 | 발견 + 설득. "나랑 비슷한 사람이 먹네" → 진입 장벽 해소 |

### 인터뷰 기반 소스

대표 Q4에서 이상적 고객 정의: **"뭘 먹어야 건강한지 의식 있는 30대~"**
예랑 Q4 추천 화법: "물이랑만 타 먹어도 진한 미숫가루 맛 → 매일 먹게 된다"

**4월 1호 주제 후보:**
- 요가 하는 30대 직장인 (예랑님 고객 네트워크 활용, 요가원 콜라보 연계)
- 러닝 하는 30대 (대표님 이상적 고객상과 일치)
- 자취생 예랑님 본인 (내부 인물로 0호 테스트 가능)

### 인트로 예시

캐러셀 첫 장:
```
"아침에 뭘 먹지?"
→ 이 질문이 스트레스인 사람을 위한 시리즈.
```

릴스 인트로 (0~3초):
```
[쉐이크를 흔드는 손 클로즈업]
자막: "지은 씨의 아침은 매일 같다."
```

### 인플루언서 연계 (2차 활용)

1차: 시딩 인플루언서에게 제품 제공
2차: 그 중 브랜드핏 좋은 분을 "이달의 베지어터" 정식 출연 제안
→ 인플루언서 입장에서도 브랜드 공식 콘텐츠 출연 = 가치 있음

### 비주얼 방향

**핵심 무드: "편안한 사람의 일상에 베지어트가 자연스럽게 있다"**

- 레퍼런스: 29CM 하우스 인터뷰, 오드아이 매거진 인물 화보
- 촬영: 자연광, 집/카페/운동 후 — 스튜디오 X
- 구도: 누끼(제품 단독 컷) X → 손에 들고 있는 장면, 테이블 위 장면
- 쉐이크 흔드는 장면은 인트로/전환 컷으로 활용 (소리 ASMR 효과)

## 2. 베지어트 랩

> Concern: "진짜 괜찮은 거야?"
> 한 줄: 이 한 스쿱이 만들어지기까지

| 항목 | 내용 |
|------|------|
| 포맷 | 캐러셀 (비하인드 사진 3~5장) or 릴스 (제조 공정 영상) |
| 빈도 | 월 1회 |
| 톤 | Sweetgreen "식재료가 히어로". 원재료를 가장 매력적으로 |
| 역할 | 신뢰 구축. "성분표를 나열하지 않고 보여준다" |

### 인터뷰 기반 소스 — 보물급

**서리태 스토리 (대표 Q5):**
- 7살 새치 → 할머니 매일 아침 검은콩미숫가루 → 할머니 돌아가신 후 그 맛을 재현
- Whole bean 직접 농가 구매 → 로스팅 → 프로틴 파우더
- ISP(분리대두단백) 대신 이 방식을 고집: 번거롭고 비용 2배지만 더 신선하고 건강

**딸기 스토리 (대표 Q6):**
- 시중 딸기맛 = 딸기향+크리머+색소+인공감미료 조합. "우리 혀를 속이고 있다"
- 생딸기 동결건조 → 파우더 → 23% 함량
- 제조사가 "단가/수급 감당 불가" 경고 → 2024년부터 개발 → "미션임파서블을 미션파서블로"
- "전 세계에 이런 프로틴은 없을 것. 원가가 너무 비싸서 보통은 이렇게 만들지 않음"

**가루 텍스처 (대표 Q2):**
- 다른 식물성보다 가루가 고운 편 → 목넘김 좋음 → 제조 단계 기술 도입 + 정성

### 4월 콘텐츠 예시

1. **"딸기 프로틴이 탄생하기까지"** — 4월 딸기 론칭과 연계. 동결건조 과정, 23%의 의미
2. **"서리태는 어디서 오는가"** — 농가 Whole bean → 로스팅 → 파우더 과정

### 비주얼 방향: 원재료를 매력적으로

**"설명하지 않고 보여준다"를 어떻게?**

방법 A — 식재료 히어로 샷 (@seok.goout 스타일):
- 서리태 콩알을 손바닥 위에 올린 클로즈업
- 동결건조 딸기가 부서지는 슬로모션
- 파우더가 물에 풀리는 순간 (탑뷰)
- 조명: 자연광 사이드, 배경 심플 (흰색 or 원목)

방법 B — 제조 공정 다큐 (릴스):
- [15초] 콩알 → 로스팅 → 분쇄 → 배합 → 패키지 타임랩스
- 자막: 숫자만. "서리태 100% / 로스팅 180°C / 48시간"
- BGM: 미니멀, 공장 ASMR (기계 소리 + 콩 부딪히는 소리)

방법 C — 성분 비교 캐러셀 (인국님 요청 반영):
- 1장: "시중 딸기맛 프로틴의 성분표" (딸기향, 색소, 감미료)
- 2장: "베지어트 딸기의 성분표" (동결건조딸기 23%, 완두단백, 끝)
- 3장: 동결건조 딸기 클로즈업
- 4장: "우리가 비싼 이유가 아니라, 다른 게 싼 이유입니다."
- → 인국님 concern "가격이 비싸다는 반응을 콘텐츠로 풀어내고 싶다" 직접 해결

## 3. 베지레터

> Concern: "진짜 괜찮은 거야?"
> 한 줄: 실제 고객이 하는 말 = 가장 강력한 신뢰

| 항목 | 내용 |
|------|------|
| 포맷 | 릴스 (고객 리뷰 읽기) or 캐러셀 |
| 빈도 | 월 1~2회 |
| 톤 | B급과 감도 사이. 고객 말은 날것, 비주얼은 정돈 |
| 역할 | 발견 + 확산. 찐후기 = 공유 유도 |

### "B급과 감도 사이" — 구체적으로

**문제**: 기존 베지레터는 광고 안 돌리면 조회수 1,000 미만 (예랑 인터뷰 확인)
**원인 가설**: 얼굴 미공개 + 나레이션 방식 → 설득력 저하 (예랑 Q2 직접 언급)
**해결 방향**: 포맷을 바꾼다

| 기존 베지레터 | 리뉴얼 베지레터 |
|-------------|-------------|
| 나레이션 + 배경 영상 | 텍스트 오버레이 + 무드 B-roll |
| 후기를 "읽어주는" 느낌 | 후기가 "화면에 뜨는" 느낌 |
| 예랑 얼굴 미공개 등장 | 사람 없이 제품+공간만 |
| 1개 후기를 길게 | 여러 후기를 빠르게 (3~5개) |

**리뉴얼 포맷:**
```
[릴스 30~45초]
0초: 제품 클로즈업 + 자막 "이 리뷰 실화?"
3초: 올영 리뷰 스크린샷 확대 → 핵심 문장 하이라이트
8초: 다음 리뷰 → "한 통 다 비우고 재구매한 단백질은 처음"
13초: 다음 → "비린맛 없이 진짜 미숫가루"
18초: 제품 흔드는 장면 (전환 컷)
23초: 마지막 리뷰 → 가장 임팩트 있는 것
28초: 제품 + 자막 "4,818명이 먹고 있습니다" (팔로워 수 활용)
```

**레퍼런스 방향:**
- Glossier 리뷰 콘텐츠: 고객 리뷰를 디자인 에셋처럼 활용
- 스킨1004 인스타: 리뷰 스크린샷을 브랜드 컬러로 감싸서 캐러셀
- 핵심: **리뷰 텍스트 자체가 비주얼** → 별도 촬영 최소화, 제작 효율 높음

### 인터뷰 기반 소스

인국 Q3에서 확인된 "기분 좋았던 반응":
- "성분도 좋은데 맛있다!" → 맛 concern 해소
- "재구매하고 싶은 제품" → 신뢰 concern 해소
- "먹고 문제들이 해결되었다" → 효과 concern 해소

인국 Q3 "아쉬웠던 피드백" 활용 (날카로운 질문 시리즈):
- "비싸다" → "g당 가격 비교해봤더니..." 자막 + 비교 인포그래픽
- "맛없다" → "맛 취향은 다르지만, 이것만은 확실합니다" + 성분표 비교

## 4. 베지어트 한 끼

> Concern: "맛있어? 어떻게 먹어?"
> 한 줄: "이렇게 먹으면 맛있어요"가 아니라 "이렇게 먹고 있어요"

| 항목 | 내용 |
|------|------|
| 포맷 | 릴스 (30~60초, 레시피/활용법) or 캐러셀 |
| 빈도 | 월 1~2회 |
| 톤 | 짧고 감각적. 식욕 자극 비주얼 |
| 역할 | 발견. 레시피 = 저장 유도 = 성장 엔진 |

### 시리즈 명칭 — 제안

"베지어트 한 끼"를 유지하되, 썸네일/커버에 쓸 **타이틀 포맷**이 필요:

| 후보 | 느낌 | 비고 |
|------|------|------|
| **한 끼의 방법** | 29CM "~의 방법" 시리즈 감성 | 매회 "한 끼의 방법 vol.4" |
| **오늘의 한 끼** | 심플, 일상적 | 월간지 느낌 가능 |
| **VEGGIET KITCHEN** | 영문, 감도 높음 | 해외 계정 톤 |
| **한 스쿱 레시피** | 베지어트 고유 단위 활용 | 짧고 직관적 |

→ 추천: **"한 끼의 방법"** — 29CM 톤과 일관성 있고, 매회 넘버링으로 시리즈감

### 인터뷰 기반 소스

예랑 Q4: "얼음 넣어서 아이스 미숫가루 라떼처럼" — 이미 검증된 먹는 방법
대표 Q8: "물 연하게, 마지막 싹싹" — 가장 심플한 먹는 방법 (리얼)

**4월 콘텐츠 예시:**
1. "얼죽아 미숫가루 라떼" — 예랑님 실제 루틴. 얼음 + 서리태 + 쉐이킹 (바이럴 가능)
2. "아침 5분 프로틴 볼" — 오트밀 + 서리태 + 과일 (저장 유도)
3. "딸기 프로틴 스무디" — 신제품 연계 레시피

### 쉐이크 유형 (v2에서 통합)

밈/유형 테스트 포맷은 "베지어트 한 끼" 안에서 비정기 특별편으로:
- "베지어트 팀의 쉐이크 유형 6가지" — 직원 맛 추천 릴스가 ROAS 300% (예랑 Q2 확인)
- 빈도: 분기 1회 정도. 너무 자주 하면 밈 채널이 됨.

### 비주얼 방향

- 탑뷰 + 손 동작 중심 (얼굴 없이도 사람 느낌)
- 색감: 음식이 맛있어 보이는 따뜻한 톤 (쿨톤 X)
- BGM: 경쾌, 짧은 루프 (트렌딩 오디오 활용)
- 자막: 재료+양만 간결하게. 설명 최소화.

## 5. 베지어트 모먼트

> Concern: "먹으면 뭐가 달라져?"
> 한 줄: 베지어트가 있는 일상의 한 장면

| 항목 | 내용 |
|------|------|
| 포맷 | 피드 (무드 사진 1장 + 짧은 카피) |
| 빈도 | 월 2회 |
| 톤 | 담백하고 건조하되 진심이 느껴지는 온도 |
| 역할 | 설득 (쇼룸). 피드 그리드의 톤을 지키는 앵커 |

### v2 → v3 변경: 만드는 사람 일상 합체

v2에서 "만드는 사람들"의 일상 부분을 모먼트에 통합.
**모먼트 = 고객의 일상 + 팀의 일상, 둘 다 포함.**

| 유형 | 예시 |
|------|------|
| 고객 모먼트 | "토요일 아침, 늦잠 자고 일어나서 첫 한 잔" |
| 팀 모먼트 | "월요일 오전, 사무실 도착하자마자 얼음부터" (예랑 실제 루틴) |
| 대표 모먼트 | "출장 가방에 파우치 3개" (대표 Q8 기반) |

### 인터뷰 기반 카피 소스

대표 Q8: "미팅 & 출장이 많아서 밥 챙기기 어려울 때, 베지어트는 저에게도 매우 고마운 존재"

## 6. 만드는 사람들

> Concern: "이 브랜드는 뭐가 달라?"
> 한 줄: 대기업이 보여줄 수 없는 것 = 만드는 사람의 얼굴과 태도

| 항목 | 내용 |
|------|------|
| 포맷 | 피드 (사진+짧은 텍스트) or 릴스 (무음 자막) |
| 빈도 | 격월 1회 |
| 톤 | 따뜻하되 정돈된. 파타고니아/마일스톤커피 (예랑 제시) |
| 역할 | 유지 (팬덤). 이미 팔로우한 사람이 "이 브랜드 진짜 좋다"고 느끼는 콘텐츠 |

### 인터뷰에서 얻은 스토리 소스 — 이게 진짜 보물

| 스토리 | 출처 | 콘텐츠화 방향 |
|--------|------|------------|
| 남편 궤양성대장염 8년 → 식물성 식단 1년 만에 염증 소멸 | 대표 Q1 | "베지어트가 시작된 이유" — 가장 강력한 원점 스토리 |
| 할머니 검은콩미숫가루 → 서리태 쉐이크 | 대표 Q5 | "서리태에 담긴 이야기" — 정서적 연결 |
| 동물성 섞어 팔라는 제안 거절 | 대표 Q2 | "우리가 100% 식물성을 고집하는 이유" |
| 딸기 동결건조 미션임파서블 | 대표 Q6 | "딸기 프로틴 237일의 기록" — 4월 론칭 연계 |
| 5명 모두 따뜻하고 수평적 | 예랑 Q1 | "5명이 만드는 브랜드" — 팀 소개 |
| "영업이익을 줄이고 장인정신으로" | 대표 Q2 | "우리가 비싼 게 아니라, 다른 게 싼 거다" |

### B급 방지 가이드 (구체화)

| 요소 | 하지 않는 것 | 하는 것 |
|------|-----------|--------|
| 대표 등장 | 정면 인터뷰, 긴 독백 | 손/뒷모습/사이드 앵글 + 자막. 행동 중심 |
| 팀 등장 | "안녕하세요 저는~" 인사 | 일하는 장면, 제품 만지는 장면 (무음+자막) |
| 대사 | 대본 느낌 | 실제 말투 그대로 (음슴체 OK). 자막이 보정 |
| 편집 | 빠른 컷 전환, 효과음 과다 | 느린 호흡, 자연음, 여백 |
| 공간 | 깔끔한 세트 | 실제 사무실, 실제 작업장 (정돈은 하되 꾸미지 않음) |
| 조명 | 형광등 그대로 | 자연광 or 따뜻한 조명 보정 (컬러그레이딩 필수) |

**레퍼런스:**
- 파타고니아: 직원이 서핑하러 가는 장면 (일과 삶의 경계 없음)
- 마일스톤커피: 원두 로스팅하는 손 클로즈업 (예랑 Q2 직접 제시)
- @seok.goout: 식재료 매력적 소개

### 캠페인성 여부 — 결론

격월 1회 시리즈로 충분. 별도 캠페인까지 갈 필요 없음.
단, **대표님 원점 스토리(Q1)는 첫 회에 쓰되, 3~5분 브랜드 필름 수준으로 제작하면 자산이 됨.**
→ 예산/리소스 여부에 따라 영상 제작사 협업 or 내부 제작(릴스) 선택.

## 7. 이벤트/프로모션/소식

| 항목 | 내용 |
|------|------|
| 한 줄 | 브랜드 뉴스를 브랜드 톤으로 전달 |
| 포맷 | 피드 or 릴스 (이벤트 성격에 따라) |
| 빈도 | 수시 (신제품, 올영 프로모션, 콜라보 등) |
| 톤 | 브랜드 톤 유지. 할인 강조 X, 가치 강조 O |

### 핵심 원칙 — 예랑 인터뷰에서 배운 것

예랑 Q2: "블프 할인 릴스는 전환 저조. 제품이 싸다/할인한다는 베지어트와 거리감."
예랑 Q2: 반면 **파우치 팔로워 이벤트는 600명+ 유입 성공.**

**차이점**: 가격 할인 소구 vs 경험 제공 소구

| 하지 않는 것 | 하는 것 |
|-----------|--------|
| "50% 할인!" | "처음 만나보세요" |
| 가격 중심 | 경험 중심 |
| 올영세일 배너 그대로 | 베지어트 톤으로 재해석 |
| 급하게 만든 프로모션 피드 | 미리 만들어둔 브랜드 톤 템플릿에 정보만 교체 |

### 예랑 concern 해결: "갑작스러운 프로모션 대응"

올영 프로모션은 예측 불가 → **프로모션 피드 템플릿 3종**을 미리 만들어 둔다:
1. 올리브베러 / 올영세일용
2. 신제품 런칭용
3. 콜라보/이벤트용

템플릿 = 디자인 에셋(컬러, 폰트, 레이아웃) 고정, 텍스트만 교체.
→ 갑작스러운 프로모션에도 브랜드 무드 유지 가능.

### 4월 이벤트 예시

- **딸기 프로틴 런칭** (4/1): New Drop 3단계 (티저→론칭→리뷰)
- **올영 파우치 이벤트**: 팔로워 확보 검증된 포맷 재활용
- 인터랙션 유도: 댓글 이벤트 ("딸기 먹어본 사람 손") > 추첨 파우치 증정
