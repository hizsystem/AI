## Part 1. 콘텐츠 시리즈 꼭지 설계

### A축. 제품 (Product)

---

#### A-1. 속이 편한 한 잔 / A Gentle Scoop

**한 줄 컨셉**: 베지어트 제품의 핵심 USP를 감도 높은 비주얼로 직관 전달

| 항목 | 내용 |
|------|------|
| 포맷 | 피드 (단일 이미지 or 캐러셀 2~3장) |
| 빈도 | 월 2회 |
| 제작 난이도 | 보통 (촬영 + 카피) |
| 해시태그 | #속이편한프로틴 #베지어트 |

**예시 주제**
1. "유청 프로틴에 속이 불편했다면" -- 식물성 프로틴이 속이 편한 이유 (캐러셀 3장)
2. "서리태 프로틴, 성분표 한 줄이면 충분합니다" -- 성분표 클로즈업 + 한 줄 카피
3. "운동 후 30분, 속 편한 단백질 보충" -- TPO 장면 + 제품 컷

---

#### A-2. 베지어트 한 끼 / One Meal with VEGGIET

**한 줄 컨셉**: TPO별 베지어트 활용법을 레시피/스타일링으로 보여주는 실용 콘텐츠

| 항목 | 내용 |
|------|------|
| 포맷 | 릴스 (30~60초) or 캐러셀 (스텝별 이미지) |
| 빈도 | 월 1~2회 |
| 제작 난이도 | 보통 (촬영 + 편집) |
| 해시태그 | #베지어트한끼 #프로틴레시피 |

**예시 주제**
1. "아침 5분 프로틴 볼" -- 오트밀 + 베지어트 서리태 + 과일 토핑
2. "운동 끝나고 바로 한 잔" -- 셰이커에 넣고 흔들기 (릴스, 러닝 후 장면)
3. "오후 3시, 간식 대신 베지어트" -- 오피스/카페 세팅, 파우치 제품 활용

---

#### A-3. 신제품 론칭 / New Drop

**한 줄 컨셉**: 신제품 출시를 브랜드 이벤트처럼 연출. 티저-론칭-리뷰 3단계 구조

| 항목 | 내용 |
|------|------|
| 포맷 | 피드 + 릴스 + 스토리 (복합) |
| 빈도 | 신제품 출시 시 (4월: 딸기 프로틴) |
| 제작 난이도 | 어려움 (기획 + 촬영 + 시리즈 운영) |
| 해시태그 | #베지어트딸기 #NEWVEGGIET |

**예시 주제 (딸기 론칭 기준)**
1. 티저: "4월, 새로운 맛이 옵니다" -- 딸기 실루엣/컬러 힌트만 노출
2. 론칭: "베지어트 스트로베리 프로틴" -- 제품 풀샷 + 올영 베러 단독 안내
3. 리뷰: 첫 시음 릴스 -- 팀 내부 or 인플루언서 첫 반응 (감도 높은 편집)

### B축. 사람 (People)

---

#### B-1. 이달의 베지어터 / Veggieter of the Month

**한 줄 컨셉**: 실제 유저의 일상 속 베지어트를 인터뷰 형식으로 기록 (레퍼런스: 29CM 하우스)

| 항목 | 내용 |
|------|------|
| 포맷 | 캐러셀 (5~7장, 인터뷰+라이프 사진) or 릴스 (60~90초) |
| 빈도 | 월 1회 |
| 제작 난이도 | 어려움 (섭외 + 촬영 + 인터뷰 정리) |
| 해시태그 | #이달의베지어터 #베지어트피플 |

**예시 주제**
1. "러너 지은 씨의 아침 루틴" -- 러닝 전후 식단, 베지어트 활용법, 공간 사진
2. "요가 강사 수진 씨가 프로틴을 고르는 기준" -- 식물성을 선택한 이유, 일상 컷
3. "두 아이 엄마 현정 씨의 간식 혁명" -- 가족 건강, 아이와 함께 먹는 프로틴

**제작 가이드**
- 서면 인터뷰 가능 (대면 촬영이 어려울 경우 셀프 사진 + 서면 Q&A)
- 질문 3~4개로 짧게: "프로틴을 챙기게 된 계기", "베지어트를 고른 이유", "당신의 루틴 한 줄"
- 사진은 자연광, 일상 공간에서. 스튜디오 느낌 X

---

#### B-2. 러닝크루 x 베지어트 / Run with VEGGIET

**한 줄 컨셉**: 러닝크루 협업 콘텐츠. 운동하는 사람들의 리얼 시딩 & 브랜드 접점

| 항목 | 내용 |
|------|------|
| 포맷 | 릴스 (러닝 장면 + 프로틴 보충 장면) or 피드 (그룹샷) |
| 빈도 | 월 1~2회 (시딩 연계) |
| 제작 난이도 | 보통 (러너 촬영분 활용 가능) |
| 해시태그 | #러닝크루베지어트 #RunWithVEGGIET |

**예시 주제**
1. "한강 10km 러닝 후, 서리태 한 잔" -- 러닝크루 러너 3명과의 공동 릴스
2. "러너의 가방 속에는" -- 러닝 장비 + 베지어트 파우치 플랫레이
3. "러닝 끝나고 뭐 먹어?" -- 러너 인터뷰 숏폼

**운영 포인트**
- 현재 확보된 러너 3명 활용, 추가 섭외 병행
- 서리태 시딩과 연계: 시딩 -> 콘텐츠 수집 -> 리포스트/공동 제작
- 러너 계정에서도 동시 업로드 (공동 마케팅 -> META 광고 연결)

---

#### B-3. 만드는 사람들 / The Makers

**한 줄 컨셉**: 창업자와 팀의 진정성을 감도 높은 포맷으로 전달 (킥오프 "창업자의 한 끼" 발전형)

| 항목 | 내용 |
|------|------|
| 포맷 | 피드 (사진 + 짧은 텍스트) or 릴스 (무음 자막 스타일) |
| 빈도 | 격월 1회 (과하면 B급 우려) |
| 제작 난이도 | 보통 (촬영 디렉션이 핵심) |
| 해시태그 | #베지어트팀 #만드는사람들 |

**예시 주제**
1. "대표의 점심" -- 박진아 대표의 점심 한 끼. 대화 없이 장면만. 마지막에 베지어트 한 잔
2. "서리태를 고르는 기준" -- 원료 선정 과정 비하인드, 산지 방문 or 테스트 장면
3. "신제품이 나오기까지 237일" -- 딸기 프로틴 개발 타임라인 캐러셀

**B급 방지 가이드**
- 대표님 등장 시: 정면 인터뷰 X -> 손/뒷모습/사이드 앵글 + 자막
- "대표가 직접 만들었어요!" 톤 X -> 장면이 이야기하게
- 조명, 구도, 컬러그레이딩에 신경. 날것이되 정돈된 날것

### C축. 라이프스타일 (Lifestyle)

---

#### C-1. 베지어트 모먼트 / VEGGIET Moment

**한 줄 컨셉**: 일상 속 식물성 단백질이 자연스럽게 녹아드는 순간을 포착

| 항목 | 내용 |
|------|------|
| 포맷 | 피드 (무드 사진 1장 + 짧은 카피) |
| 빈도 | 월 2회 |
| 제작 난이도 | 쉬움 (기존 촬영분 활용 가능) |
| 해시태그 | #베지어트모먼트 #VEGGIETMoment |

**예시 주제**
1. "토요일 아침, 늦잠 자고 일어나서 첫 한 잔" -- 침대 옆 베지어트 + 자연광
2. "회의 끝나고 3시의 리셋" -- 오피스 데스크 위 파우치 제품
3. "운동 가방에 항상 들어있는 것" -- 체육관 로커/가방 플랫레이

---

#### C-2. 식단 루틴 챌린지 / Routine Challenge

**한 줄 컨셉**: 팔로워 참여형 챌린지. 베지어트와 함께하는 식단 관리 루틴

| 항목 | 내용 |
|------|------|
| 포맷 | 스토리 (투표/Q&A) + 피드 (챌린지 안내/결과) |
| 빈도 | 분기 1회 |
| 제작 난이도 | 보통 (기획 + 참여 관리) |
| 해시태그 | #베지어트루틴 #7일챌린지 |

**예시 주제**
1. "7일 아침 프로틴 챌린지" -- 매일 아침 베지어트 + 인증 스토리 리포스트
2. "나의 프로틴 TPO" -- 팔로워가 자신의 베지어트 타이밍 공유
3. "식단 관리 Before/After 4주" -- 참여자 경험담 취합 (UGC)

---

### D축. 브랜드 (Brand)

---

#### D-1. 베지어트 랩 / VEGGIET Lab

**한 줄 컨셉**: 제조/개발 비하인드를 보여주는 브랜드 신뢰 콘텐츠

| 항목 | 내용 |
|------|------|
| 포맷 | 캐러셀 (비하인드 사진 3~5장) or 릴스 (공정 영상) |
| 빈도 | 월 1회 |
| 제작 난이도 | 보통 (기존 소스 활용 가능, 추가 촬영 시 어려움) |
| 해시태그 | #베지어트랩 #VEGGIETLab |

**예시 주제**
1. "서리태는 어디서 오는가" -- 원료 산지 이야기, 계약 농가 사진
2. "한 스쿱에 담긴 것들" -- 제조 공정 요약 캐러셀 (원료 -> 배합 -> 품질검사 -> 패키지)
3. "딸기 프로틴이 탄생하기까지" -- 신제품 개발 비하인드 스토리

---

#### D-2. 식물성의 기준 / The Standard

**한 줄 컨셉**: 베지어트의 브랜드 철학과 식물성 단백질에 대한 관점을 정립하는 선언적 콘텐츠

| 항목 | 내용 |
|------|------|
| 포맷 | 피드 (타이포그래피 중심 이미지 + 본문 카피) |
| 빈도 | 월 1회 |
| 제작 난이도 | 쉬움 (카피 + 디자인 에셋) |
| 해시태그 | #식물성의기준 #TheStandard |

**예시 주제**
1. "우리는 '대체'하지 않습니다. 새로운 기준을 만듭니다." -- 브랜드 매니페스토 1장
2. "100% 식물성만 하는 이유" -- 카테고리 순수성 메시지
3. "속이 편한 단백질이 좋은 단백질입니다" -- USP를 철학으로 승화

촬영 가이드:
레퍼런스 이미지: (링크 or 첨부)

[ ] 금지어 확인 (대체육/대체식품 X)
