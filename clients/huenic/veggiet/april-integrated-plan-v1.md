## 1. 4월 전체 구조

### 두 트랙의 통합

4월은 **올리브베러픽 캠페인**(세일즈)과 **채널 리뉴얼**(브랜딩)이 동시에 시작되는 달이다.
두 트랙을 분리하면 피드가 프로모션 일색이 되고, 합치면 브랜딩이 묻힌다.
해법: **피드 노출 콘텐츠는 브랜딩 감도로, 릴스/스토리는 세일즈 CTA로** 역할을 나눈다.

| 레이어 | 역할 | 감도 | CTA |
|--------|------|------|-----|
| 피드 (이미지/캐러셀) | 브랜드 세계관 구축, 피드 그리드 톤 유지 | 높음 (AG1/Sweetgreen급) | 간접 — 바이오 링크 |
| 릴스 | 바이럴 + 세일즈 전환 | 중~상 (리얼감 + 감도) | 직접 — "올리브베러에서 만나보세요" |
| 스토리 | 실시간 소통 + 프로모션 직접 CTA | 자유 | 직접 — 올영 링크, 라이브 |
| 메타 광고 | 퍼포먼스 전환 | 소재별 상이 | 올리브베러 랜딩 |

## 2. 프리론칭 전략 (3/24~3/31)

> 인국님 요청: "4/1 딸기맛 런칭 전 인스타를 런칭 관련 콘텐츠로 채우고 싶다 (슈퍼말차 슈퍼보리처럼)"

### 레퍼런스 분석: 슈퍼말차 슈퍼보리 출시전 플레이

슈퍼말차가 슈퍼보리를 런칭할 때 한 것:
- 출시 2~3주 전부터 원재료(보리) 비주얼로 피드 무드를 서서히 전환
- "COMING SOON" → 날짜 공개 → 제품 실루엣 → 풀 론칭
- 기존 말차 콘텐츠 사이사이에 보리 컬러(골든)를 섞어 피드 그리드에 리듬감

### 베지어트 딸기 프리론칭 플레이 제안

**목표**: 3/24~3/31 동안 딸기 론칭 기대감 형성 + 피드 톤 전환의 시작점

#### 3/24(월) — 이미 기획된 참여형 릴스

- 오니스트 리뷰형 릴스 + 이벤트 (인국님 기획, 확정)
- "지속가능한 루틴을 가진 사람들의 습관 3가지"
- 이건 예정대로 진행. 딸기와 직접 관계없지만, 채널 활성화 역할

#### 3/26(수) — 티저 #1: 컬러 힌트

- **포맷**: 피드 (1장 이미지)
- **비주얼**: 딸기 핑크 컬러 그라데이션 배경 + 베지어트 로고만 (제품 미노출)
- **캡션**: "4월, 새로운 계절이 옵니다."
- **포인트**: 제품을 보여주지 않는다. 컬러만으로 기대감. "뭐지?" 반응 유도
- **시리즈 태그**: A-3. 신제품 론칭 (티저)

#### 3/28(금) — 티저 #2: 원재료 히어로

- **포맷**: 릴스 (15~20초) — 딸기 런칭 콘텐츠2.mp4 활용 (동결건조 딸기 비주얼)
- **비주얼**: 동결건조 딸기 클로즈업 → 파우더로 변하는 장면 → "Strawberry. 26.04.01"
- **캡션**: "곧 만나요."
- **포인트**: 딸기 런칭 콘텐츠3.mp4의 "Seasonal Flavor Drop 26.04.01" 스타일 활용
- **시리즈 태그**: A-3. 신제품 론칭 (티저)

#### 3/30(일) — 티저 #3: 제품 실루엣 or 카운트다운

- **포맷**: 스토리 (카운트다운 스티커)
- **비주얼**: "D-2" + 딸기 프로틴 패키지 실루엣 (반투명)
- **CTA**: 4/1 라이브 알림 설정 유도

#### 3/31(월) — 베러픽 런칭 이미지 제작 (확정)

- 베러픽 런칭 피드 이미지 제작 완료 → 4/1 업로드

### 프리론칭 피드 그리드 시뮬레이션

```
기존 피드 | 기존 피드 | 기존 피드
기존 피드 | 3/24 릴스  | 기존 피드
3/26 핑크  | 3/28 딸기  | 3/31 런칭이미지  ← 핑크 톤으로 전환 시작
4/1 베러픽 | 4/3 릴스   | 4/9 딸기+베러픽
```

→ 3열 하단이 "핑크 → 딸기 → 런칭"으로 자연스럽게 연결되면서 피드 톤이 전환됨

## 7. 인국님 요청 응답 정리 (슬랙 회신용)

### 4/1 전 프리론칭 플레이 제안

인국님이 원하는 "슈퍼보리 출시전처럼 피드를 런칭 관련으로 채우기"에 대한 답:

**3가지 콘텐츠로 프리론칭 무드를 만든다:**

1. **3/26(수) 피드**: 딸기 핑크 컬러 힌트 (제품 미노출, "4월, 새로운 계절이 옵니다")
2. **3/28(금) 릴스**: 딸기 런칭 콘텐츠2+3 편집 활용 ("Strawberry. 26.04.01. Flavor Drop")
3. **3/30(일) 스토리**: D-2 카운트다운 + 4/1 라이브 알림 설정

이미 만들어둔 딸기 런칭 릴스 4개가 퀄리티 좋으니, 2+3번 소스를 프리론칭용으로 먼저 활용하고, 1+4번은 론칭 이후 메타 광고 소재로 쓰는 구조.

### 콘텐츠 감도 피드백

현재 기획안 전반적으로 좋은 점:
- 브레인스토밍 아이디어 풀이 풍부하고 리얼함 (특히 유형 밈, 찐후기)
- 해외 레퍼런스를 잘 소화하고 있음 (슈퍼보리, 오니스트, 트루바니)
- "찐후기 클릭률 높다"는 데이터 기반 인사이트가 정확

강화하면 좋을 점:
- 카피 톤앤매너 통일 (채널 컨셉 v1 가이드 적용)
- 피드 노출 콘텐츠의 비주얼 감도 관리 (무드보드 기반)
- 시리즈 꼭지 태깅 → 반복이 브랜드를 만든다
