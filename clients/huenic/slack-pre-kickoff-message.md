### 1) 베지어트 브랜드 키워드 (대표님께)

베지어트 인스타그램을 기획하려면, **대표님이 중요하게 생각하시는 가치**를 먼저 알아야 합니다.
아래 질문에 키워드 or 짧은 문장으로 편하게 답변 주세요!

- 베지어트를 한 문장으로 소개한다면? (고객에게 말하듯이)
- 베지어트가 경쟁사와 **절대 다른 점** 3가지는?
- 대표님이 베지어트를 만들 때 **가장 양보할 수 없었던 것**은? (재료? 맛? 가격? 철학?)
- 고객에게 **"이것만은 꼭 알아줬으면"** 하는 것이 있다면?
- 베지어트 고객 중 **가장 기억에 남는 반응/피드백**이 있다면 공유해주세요

> 이 답변들이 앞으로 인스타그램 콘텐츠의 **뿌리 키워드**가 됩니다.
> 길게 쓰실 필요 없어요 — 떠오르는 단어 몇 개면 충분합니다!

### 2) 콘텐츠 소스 체크 (인국님, 예랑님)

우리가 콘텐츠를 만들 재료를 파악하고 싶습니다!

**a. 기존 콘텐츠 셀프 체크**
- 베지어트 인스타에서 **"이건 잘 됐다"** 콘텐츠 3개 (링크 or 스크린샷)
- **"이건 아쉬웠다"** 콘텐츠 3개
- 각각 **왜** 그렇게 느꼈는지 한 줄씩 달아주시면 더 좋습니다

**b. 콘텐츠 소스 재고**
아래 중 현재 있는 것에 체크해주세요:
- [ ] 제품 개발/제조 과정 영상이나 사진
- [ ] 대표님 인터뷰 or 브랜드 스토리 영상
- [ ] 고객 후기/UGC (인스타 태그, 블로그 리뷰 등)
- [ ] 원재료 관련 비하인드 (산지 방문, 원료 선정 과정 등)
- [ ] 레시피/활용법 콘텐츠 (기촬영분)
- [ ] 팀원들의 일상/회사 비하인드 소스

> 없는 것은 앞으로 같이 만들어가면 됩니다. 지금 뭐가 있는지 파악이 목적입니다!

### 3) 톤 & 무드 레퍼런스 (인국님, 예랑님)

- 베지어트가 참고하고 싶은 인스타 계정 (동종업계 아니어도 OK)
- 특히 **"톤이 좋다"**, **"이 계정의 이 느낌"** 하는 포인트가 있으면 알려주세요
- 최근에 본 릴스/피드 중 **"우리도 이런 거 해보고 싶다"** 하는 것이 있다면 링크!

### 4) 운영 & 데이터 환경

**운영 현황**
- 현재 콘텐츠 기획 → 제작 → 업로드 프로세스 간단히 알려주세요
- 인국님, 예랑님의 주간 베지어트 투입 가능 시간은 대략?
- META 광고 현재 운영 현황 (월 예산, 주요 목적) 간단히

**데이터 접근 — 이건 중요합니다!**
- 베지어트 인스타그램 **Meta Business Suite** 접근 권한을 저희 팀에 공유해주실 수 있나요?
- 베지어트 브랜드 가이드라인 (로고 파일, 컬러 코드, 폰트 등) 정리된 자료가 있다면 공유 부탁드립니다 (없어도 OK!)
- 현재 사용하시는 업무 도구 (노션, 캔바, 피그마 등) & 콘텐츠 캘린더 관리 도구

> 데이터 권한이 있어야 콘텐츠 반응을 트래킹하고, **뭐가 되는지/안 되는지** 근거를 가지고 운영할 수 있습니다.

## 내부 메모 (발송 안 함)

### 이 질문 구조의 의도

```
질문 1) 브랜드 DNA 키워드 추출
        → 대표의 진정성, 양보 불가 가치, 핵심 차별점
        → 이게 모든 콘텐츠의 뿌리가 됨

질문 2) 콘텐츠 소스 재고 확인
        → 현재 뭘 가지고 있는지 파악
        → 촬영/제작 우선순위 결정의 근거

질문 3) 톤 & 문법 레퍼런스
        → "감도 있게, 요즘 문법에 맞춰" 의 기준점 수립
        → 단순 비주얼이 아닌 톤/화법/문법까지 레퍼런스

질문 4) 데이터 트래킹 환경
        → Meta Business Suite 권한 = 반응 측정의 전제조건
        → "콘텐츠 → 데이터로 반응 확인 → 방향 수정" 루프의 기반

결과적으로:
키워드 → 콘텐츠 기획 → 감도 있는 실행 → 데이터 트래킹 → 반응 기반 개선
이 순환 구조를 만들기 위한 사전 질문
```

### 킥오프에서 이 답변을 활용하는 법

1. **대표님 답변 → 키워드 추출 워크숍** (킥오프 Part 1에서 15분)
   - 대표님이 쓴 단어들을 화이트보드에 나열
   - 3개 핵심 키워드로 압축 (ex: "진정성", "식물성의 새로운 맛", "건강한 간편식")
   - 이 키워드가 4월 콘텐츠 기획의 축이 됨

2. **콘텐츠 소스 재고 → 4월 콘텐츠 캘린더 초안** (킥오프 Part 3에서)
   - 있는 소스로 바로 만들 수 있는 콘텐츠 먼저 기획
   - 없는 소스 중 필요한 것 → 촬영/제작 일정 잡기

3. **레퍼런스 → 무드보드 v1 기반** (킥오프 후 Week 1에서)
   - 그들이 고른 레퍼런스의 공통점 분석
   - 베지어트 톤앤매너 가이드의 시작점

### 발송 전 체크리스트

- [ ] 질문이 5개로 적절한지 (기존 6개 → 5개로 줄임, 단 깊이는 더 깊게)
- [ ] 대표님 질문(1번)은 따로 분리 — 실무진과 다른 관점 필요
- [ ] 실무진 질문(2,3번)에 체크리스트 형태로 답하기 쉽게
- [ ] 데이터 권한(4번)의 중요성을 톤은 가볍게, 의미는 명확하게
- [ ] 슬랙 포맷 확인 (볼드, 체크박스, 인용 블록)
