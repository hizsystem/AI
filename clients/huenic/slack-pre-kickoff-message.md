### 1) 베지어트 브랜드 키워드 (대표님께)

저희가 작업한 **베지어트 브랜드 가이드라인**을 기반으로 인스타그램 콘텐츠를 기획하려 합니다.
가이드라인의 비주얼/톤 방향은 킥오프에서 같이 리뷰할 예정이고, 그 전에 **대표님의 언어**로 된 키워드를 미리 받고 싶습니다!

- 베지어트를 한 문장으로 소개한다면? (고객에게 말하듯이)
- 베지어트가 경쟁사와 **절대 다른 점** 3가지는?
- 대표님이 베지어트를 만들 때 **가장 양보할 수 없었던 것**은? (재료? 맛? 가격? 철학?)
- 고객에게 **"이것만은 꼭 알아줬으면"** 하는 것이 있다면?
- 베지어트 고객 중 **가장 기억에 남는 반응/피드백**이 있다면 공유해주세요

> 이 답변들이 가이드라인 위에 얹혀서 **콘텐츠의 뿌리 키워드**가 됩니다.
> 길게 쓰실 필요 없어요 — 떠오르는 단어 몇 개면 충분합니다!

### 2) 콘텐츠 소스 체크 (인국님, 예랑님)

우리가 콘텐츠를 만들 재료를 파악하고 싶습니다!

**a. 기존 콘텐츠 셀프 체크**
- 베지어트 인스타에서 **"이건 잘 됐다"** 콘텐츠 3개 (링크 or 스크린샷)
- **"이건 아쉬웠다"** 콘텐츠 3개
- 각각 **왜** 그렇게 느꼈는지 한 줄씩 달아주시면 더 좋습니다

**b. 콘텐츠 소스 재고**
아래 중 현재 있는 것에 체크해주세요:
- [ ] 제품 개발/제조 과정 영상이나 사진
- [ ] 대표님 인터뷰 or 브랜드 스토리 영상
- [ ] 고객 후기/UGC (인스타 태그, 블로그 리뷰 등)
- [ ] 원재료 관련 비하인드 (산지 방문, 원료 선정 과정 등)
- [ ] 레시피/활용법 콘텐츠 (기촬영분)
- [ ] 팀원들의 일상/회사 비하인드 소스

> 없는 것은 앞으로 같이 만들어가면 됩니다. 지금 뭐가 있는지 파악이 목적입니다!

### 3) 톤 & 무드 레퍼런스 (인국님, 예랑님)

저희가 **베지어트 브랜드 가이드라인에서 잡은 비주얼 톤**이 있는데, 인스타그램에서는 좀 더 구체적인 무드가 필요합니다.

- 베지어트가 참고하고 싶은 인스타 계정 (동종업계 아니어도 OK)
- 특히 **"톤이 좋다"**, **"이 계정의 이 느낌"** 하는 포인트가 있으면 알려주세요
- 최근에 본 릴스/피드 중 **"우리도 이런 거 해보고 싶다"** 하는 것이 있다면 링크!

> 킥오프에서 가이드라인 비주얼 + 여러분의 레퍼런스를 합쳐서 인스타 무드보드 방향을 잡을 예정입니다.

## 내부 메모 (발송 안 함)

### 이 질문 구조의 의도

```
[전제] 베지어트/빙커 브랜드 가이드라인을 이미 보유
       → "자료 달라" 포지션이 아닌 "기준을 제시하고 리드" 포지션

질문 1) 브랜드 DNA 키워드 추출
        → 가이드라인(비주얼) 위에 대표의 언어(메시지)를 얹는 구조
        → 이게 모든 콘텐츠의 뿌리가 됨

질문 2) 콘텐츠 소스 재고 확인
        → 현재 뭘 가지고 있는지 파악
        → 촬영/제작 우선순위 결정의 근거

질문 3) 톤 & 무드 레퍼런스
        → 가이드라인 비주얼 톤 + 휴닉팀 레퍼런스 = 인스타 무드보드
        → 단순 비주얼이 아닌 톤/화법/문법까지 레퍼런스

질문 4) 데이터 트래킹 환경
        → Meta Business Suite 권한 = 반응 측정의 전제조건
        → 가이드라인은 이미 보유 → 요청 삭제

결과적으로:
가이드라인(기준) → 키워드(메시지) → 콘텐츠 기획 → 데이터 트래킹 → 반응 기반 개선
가이드라인이 있으니 "기준 세우기"가 아닌 "실행 설계"부터 시작
```

### 킥오프에서 이 답변을 활용하는 법

1. **가이드라인 리뷰 + 키워드 매핑** (킥오프 Part 1에서 20분)
   - 베지어트 브랜드 가이드라인을 화면에 띄우고 핵심 비주얼 방향 공유
   - 대표님 키워드 답변을 가이드라인 위에 매핑 → "이 비주얼에 이 메시지"
   - 애로사항 1번(비주얼 체계화) 해결 완료를 자연스럽게 보여줌
   - "이 기준으로 갑니다" 선언 → 리드 포지션 확보

2. **콘텐츠 소스 재고 → 4월 콘텐츠 캘린더 초안** (킥오프 Part 2에서)
   - 가이드라인 기준이 있으니 "뭘 만들까"가 아닌 "이 기준으로 뭘 먼저 만들까"
   - 있는 소스로 바로 만들 수 있는 콘텐츠 먼저 기획
   - 없는 소스 중 필요한 것 → 촬영/제작 일정 잡기

3. **레퍼런스 + 가이드라인 → 인스타 무드보드** (킥오프 Part 3 또는 후속 기획회의)
   - 가이드라인 비주얼 톤 × 휴닉팀 레퍼런스 = 인스타 전용 무드보드
   - 톤앤매너를 새로 만드는 게 아니라, 가이드라인에서 인스타 버전으로 확장

### 발송 전 체크리스트

- [ ] 질문이 5개로 적절한지 (기존 6개 → 5개로 줄임, 단 깊이는 더 깊게)
- [ ] 대표님 질문(1번)은 따로 분리 — 실무진과 다른 관점 필요
- [ ] 실무진 질문(2,3번)에 체크리스트 형태로 답하기 쉽게
- [ ] 데이터 권한(4번)의 중요성을 톤은 가볍게, 의미는 명확하게
- [ ] 슬랙 포맷 확인 (볼드, 체크박스, 인용 블록)
