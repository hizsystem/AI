## 2. 사전 질문 (3/20 발송)

> 목적: 3/25 기획안에 반영할 브랜드 DNA + 콘텐츠 소스 확보
> 대상: 박진아 대표 / 인국 / 예랑

### Part A. 대표님 인터뷰 질문 (브랜드 진정성 — 대기업과의 차별화 핵심)

1. 베지어트를 처음 만들 때, **"이건 꼭 지키겠다"고 결심한 것**이 있다면?
   - 맛? 원재료? 가격? 제조 방식? 아니면 다른 무언가?

2. CJ 얼티브, 프로티원 같은 대기업 제품과 베지어트를 나란히 놓았을 때,
   **대표님이 "우리는 절대 다르다"고 확신하는 지점**은?

3. 베지어트를 만들면서 **가장 힘들었지만 타협하지 않은 순간**이 있다면 하나만 공유해주세요.
   (원료 선정, 제조 과정, 맛 개발 등 어떤 영역이든)

4. 고객 중 **"이 사람이 우리 이상적인 고객이다"** 하는 분이 떠오르시나요?
   그 분은 왜 베지어트를 선택했을까요?

5. 5년 후 베지어트가 **어떤 브랜드로 기억되면** 좋겠나요? (한 줄로)

> 이 답변들이 콘텐츠의 뿌리가 됩니다.
> 길게 쓰실 필요 없어요 — 떠오르는 단어나 에피소드면 충분합니다.

### Part B. 콘텐츠 소스 & 운영 (인국, 예랑)

1. 현재 보유한 콘텐츠 소스 체크 (해당하는 것 모두):
   - [ ] 제조 과정 영상/사진 (공장, 원료 등)
   - [ ] 대표님 인터뷰 or 브랜드 스토리 영상
   - [ ] 고객 후기/UGC (인스타 태그, 블로그, 올영 리뷰)
   - [ ] 원재료 비하인드 (서리태 산지, 원료 선정 과정)
   - [ ] 레시피/활용법 촬영분
   - [ ] 팀 비하인드 (사무실, 개발 과정, 일상)
   - [ ] 러너/운동 관련 촬영분
   - [ ] 기존 인플루언서 시딩 콘텐츠

2. 현재 콘텐츠 기획 → 제작 → 업로드 프로세스를 간단히 알려주세요.

3. META 광고 현황:
   - 월 예산 대략?
   - 서리태 광고 주요 타겟/메시지는?
   - 현재 ROAS or 주요 성과 지표?

4. 베지어트 인스타에서 **"이건 잘 됐다"** 콘텐츠 3개 + **"아쉬웠다"** 3개
   (링크 or 스크린샷 + 이유 한 줄)

### Part C. 빙커 관련 (간단히)

1. 작년 빙커 인스타그램 운영에 **만족하셨나요?** (솔직하게)
   - 잘 됐던 점 / 아쉬웠던 점 각 1개씩

2. 빙커를 한 문장으로 소구한다면? (해외 바이어/소비자에게)

3. 4월 이후 리테일 입점/데모 일정 공유 (잡힌 것 있으면)

## 4. 휴닉 서버(대시보드) 오픈 계획

### 단계별 접근

**Phase 1: 데이터 아카이빙 (3/18~3/31)**
- 콘텐츠 캘린더 웹앱(content-calendar/) 활성화
- 기존 CSV 데이터 임포트 (팔로워추이, 주간성과, 월간KPI, ER추이, 베스트콘텐츠)
- 베지어트 인스타그램 기본 데이터 수집 시작
  - Meta Business Suite 권한 확보 후 자동 연동
- 구글시트 마스터시트 셋업 (setup-huenic-master.js 활용)

**Phase 2: 대시보드 MVP (4월 1~2주)**
- KPI 트래킹 (팔로워 목표선 대비 추이)
- 주간 콘텐츠 성과 자동 집계
- 주간 리포트 자동 생성 기반 마련

**Phase 3: 콘텐츠 관리 통합 (4월 3주~)**
- 콘텐츠 카드 (기획 → 제작 → 리뷰 → 완료)
- 휴닉팀 접근 권한 오픈
- 인플루언서 시딩 트래킹

### 즉시 실행 가능한 것
1. content-calendar 앱 Vercel 배포 확인 & URL 공유
2. 구글시트 마스터시트 생성 (스크립트 준비 완료)
3. 베지어트 인스타 현재 데이터 스냅샷 저장
