> **받는 사람**: 인국 (휴닉)
> **채널**: 슬랙 `#huenic` 혹은 DM
> **발송 시점**: 2026-04-19 (일) 저녁 ~ 2026-04-20 (월) 오전
> **목적**: 주간 리포트·KPI 운영 정식 런칭 + W16부터 시트 입력 요청

이후 제가 대시보드에서 코멘트·계획 이어 붙여서 지나님께 공유하는 흐름입니다.

## 보내기 전 Green 체크리스트

- [ ] `시트-입력-가이드.md` 링크를 **노션 or Github 공개 URL**로 치환 (슬랙에서 바로 열리게)
- [ ] 인국님이 시트 편집 권한 있는지 확인 (링크 접근 후 편집 가능한지)
- [ ] 예랑님께도 cc — 릴스 기획자로서 베스트콘텐츠 기준 공유 필요
- [ ] 지나님께는 발송 후 "주간 리포트 런칭했습니다" 1줄 따로 DM

## 발송 후 처리

1. 인국님 첫 입력 완료 시 → 대시보드 확인 → 5분 내 반영 안 되면 시트 CSV 공개 여부 체크
2. 주간 리포트 탭에서 코멘트·다음주 계획 작성 → 스크린샷 `#huenic` 공유
3. 주간 리듬이 안정되면 (약 3주차부터) 월간 루틴 리마인드 준비
