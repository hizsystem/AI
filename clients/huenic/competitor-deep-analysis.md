## 1. 홀블렌딩 (Whole Blending)

### 1-1. 브랜드 소개

| 항목 | 내용 |
|------|------|
| 운영사 | (주)홀블렌딩 |
| 대표 | 이보희 (약사 출신) |
| 공식 사이트 | wholeblending.com |
| 인스타그램 | @wholeblending |
| 핵심 카테고리 | 식물성 단백질 쉐이크 |
| 설립 배경 | 약사로서 "약과 음식은 달라야 한다"는 철학에서 출발 |

### 1-2. 핵심 제품 & 가격대

- 100% 식물성 프로틴 파우더 4종: 곡물(Grain), 카카오(Cacao), 검은콩(Black Bean), 쑥(Mugwort)
- 완두콩 단백질 기반, BCAA 풍부, 현미와의 조합으로 아미노산 프로필 보완
- 식이섬유, 프로바이오틱스, 효소 배합 — 소화 편의성 강조
- 블렌더보틀 등 액세서리 라인업 보유
- 가격대: 프리미엄 포지셔닝 (커뮤니티 후기에서 "성분은 좋지만 가격이 높다"는 평가 반복)
- V-LABEL(이탈리아) 비건 인증 획득

### 1-3. USP / 핵심 소구 포인트

| 소구 포인트 | 설명 |
|------------|------|
| **약사가 만든 프로틴** | 전문가 권위 + 신뢰감. "약사가 설계했다"는 메시지가 강력한 신뢰 앵커 |
| **클린 프로틴** | 인공첨가물 배제, 정제 설탕/포도당/수크랄로스/스테비아 미사용 |
| **소화 편의** | 식이섬유 + 프로바이오틱스 + 효소 → 속 편한 프로틴 |
| **비건 인증** | V-LABEL 국제 인증으로 비건 소비자 공략 |

### 1-4. 타겟 오디언스

- 30~40대 여성, 건강 관리에 관심 높은 소비자
- 기존 프로틴 제품에서 소화 불편을 경험한 사람
- "첨가물 없는 깨끗한 프로틴"을 찾는 클린 이팅 추구 소비자
- 비건/식물성 식단에 관심 있지만 맛/편의성도 포기하지 않으려는 층

### 1-5. 인스타그램 운영 현황

| 항목 | 현황 |
|------|------|
| 팔로워 | 약 1만 내외 (소규모) |
| 콘텐츠 유형 | 제품 소개, 성분 교육, 약사 대표의 전문 지식 콘텐츠 |
| 피드 톤 | 깔끔하고 미니멀한 톤, 화이트/베이지 계열 |
| 업로드 빈도 | 주 1~2회 수준 (비정기적) |
| 특이사항 | 매거진 섹션(wholeblending.com/article)을 통한 교육 콘텐츠 병행 |

### 1-6. 브랜드 비주얼 톤

- 컬러: 화이트, 베이지, 라이트 그린 — 자연스럽고 깨끗한 이미지
- 타이포: 깔끔한 산세리프 위주
- 사진 스타일: 제품 중심의 스튜디오 컷, 성분 강조형 플랫레이
- 전체적으로 "청결함"과 "전문성"을 강조하는 톤

### 1-7. 베지어트 관점에서의 위협 분석

**위협 요인:**
- "약사가 만든"이라는 전문가 권위 포지셔닝은 "누가 만들었느냐"에 민감한 한국 소비자에게 강력한 무기
- 클린 프로틴 포지셔닝이 베지어트의 "속 편한 식물성 단백질"과 직접 겹침
- 비건 국제 인증(V-LABEL)은 베지어트도 아직 갖추지 못한 차별 자산

**베지어트 차별화 포인트:**
- 베지어트는 "식단 관리의 시작"이라는 라이프스타일 관점에서 접근 가능 — 홀블렌딩의 "전문가가 설계한 성분" 관점과 다른 축
- 베지어트는 TPO(운동 후/아침 대용/간식) 중심의 다양한 섭취 맥락 제안 가능
- 베지어트는 서리태(모발 건강) 등 한국 원재료 기반의 독자 소구 축 보유
- 브랜드 감도에서 승부 가능 — 홀블렌딩은 비주얼 투자가 상대적으로 약함

- 20~30대, 건강한 라이프스타일에 관심 있는 MZ세대

- 비건/클린 이팅에 관심 있는 여성 소비자

- 사진 스타일: 매장 공간 + 음료의 감도 높은 촬영, 자연광 활용
- 공간 디자인: 로봇팔이 음료를 만드는 독특한 매장 경험 — 브랜드 스토리텔링 소재
- Behance에 브랜딩 프로젝트 공개 — 디자인 커뮤니티에서도 주목받는 수준
- 전체적으로 "프리미엄이면서 힙한" 톤 — 성수 감성의 정석

## 3. AG1 (Athletic Greens)

### 3-1. 브랜드 소개

| 인스타그램 | @athleticgreens (223K 팔로워) |
| 핵심 전략 | Single-SKU → DTC 구독 모델 |

### 3-2. 핵심 제품 & 가격대

| 제품 | 가격 | 비고 |

| AG1 파우더 (30일분) | 월 $79 (약 10.5만원) | 구독 기본가 |

### 3-3. USP / 핵심 소구 포인트

| 소구 포인트 | 설명 |
|------------|------|
| **올인원 솔루션** | 75가지 비타민/미네랄/프로바이오틱스를 한 잔에 — "이것 하나면 된다" |
| **과학 기반** | 11년간의 포뮬러 개선, 연구 데이터 기반 소구 |
| **라이프스타일 통합** | "최고의 하루를 위한 첫 번째 습관" — 아침 루틴으로 포지셔닝 |
| **심플한 선택** | 복잡한 영양제 조합 대신 하나만 선택하면 되는 편의성 |
| **구독 = 커뮤니티** | DTC 구독 모델을 통한 브랜드 로열티 구축 |

### 3-4. 타겟 오디언스

- 핵심: 30~50대 도시 거주 성인, 고소득층
- 운동 선수/피트니스 애호가 → 퍼포먼스 최적화 소구
- 바쁜 직장인/창업가 → 집중력/에너지 소구
- 부모(아빠/엄마 세그먼트) → 가족 건강 소구
- 미국, 호주, 캐나다 중심 — 건강 보충제 시장 성숙 국가 타겟

### 3-5. 인스타그램 운영 현황

| 항목 | 현황 |
|------|------|
| 계정 | @athleticgreens |
| 팔로워 | 약 223,000명 |
| 콘텐츠 유형 | UGC 리포스팅, 크리에이터 협업, 경품 이벤트, 라이프스타일 무드, 과학/성분 교육 |
| 업로드 빈도 | 주 5~7회 (거의 매일) |

**콘텐츠 전략 핵심:**

1. **크리에이터 파트너십 중심**: 전통 광고 대신 인플루언서의 자연스러운 제품 통합. 크리에이티브 브리프 + 콘텐츠 가이드라인 + 스토리텔링 프레임워크를 구조화하여 제공하되, 크리에이터 고유의 톤을 존중
2. **UGC 기반 커뮤니티**: 고객이 만든 콘텐츠를 적극 리포스팅 — 소속감과 커뮤니티 문화 형성
3. **경품 이벤트**: 상위 EMV(미디어 가치) 게시물 10개 중 8개가 경품 이벤트 — 인게이지먼트 극대화
4. **멀티 세그먼트 접근**: 러너, 아빠, 엄마, 인플루언서 등 세그먼트별 맞춤 콘텐츠 제작
5. **경험 기반 마케팅**: 1 Hotels, True Food Kitchen 등과 협업 — "빌보드가 아닌 직접 경험"

### 3-6. 브랜드 비주얼 톤

- 컬러: AG1 시그니처 그린(진한 녹색) + 화이트 + 블랙 — 과학적이면서 프리미엄

## 6. 베지어트 포지셔닝 시사점

### 6-1. 현재 베지어트의 포지셔닝 공백

킥오프 미팅에서 진단한 바와 같이, 베지어트는 **"제품력은 있지만 브랜드가 뾰족하지 않다"**는 상태다. 4개 브랜드 분석을 통해 이 공백을 구체적으로 정의할 수 있다.

| 브랜드 | 뾰족한 것 | 베지어트에 없는 것 |
|--------|----------|------------------|
| 홀블렌딩 | "약사가 만든" — 전문가 권위 | 명확한 신뢰 앵커(누가/왜 만들었나) |
| 슈퍼말차 | 성수 감성의 브랜드 감도 | 일관된 비주얼 시스템과 감도 높은 톤 |
| AG1 | "이것 하나면 된다" — 극도의 단순함 | 한 줄로 설명되는 핵심 메시지 |
| Sweetgreen | 미션과 디자인이 하나인 브랜드 | 브랜드 철학이 녹아든 비주얼 에셋 |

### 6-2. 베지어트만의 뾰족한 포지셔닝 방향

**제안 포지셔닝: "매일 먹어도 속 편한, 식물성 프로틴의 기준"**

이 방향은 4개 브랜드 분석에서 도출된 다음의 전략적 판단에 근거한다:

**[1] 홀블렌딩과의 차별화 — "전문가 vs 경험"**
- 홀블렌딩의 "약사가 만든"은 강력하지만, "처방전 같은" 차가운 느낌이 될 수 있다
- 베지어트는 "매일 먹어본 사람이 아는 편안함" — 경험 기반 소구로 대응
- 선도주자(퍼스트무버)임을 강조하되, "가장 오래 고민한 브랜드"로 포지셔닝

**[2] 슈퍼말차 수준의 브랜드 감도 확보 — "건강한 것이 아름다워야 한다"**
- 비주얼 시스템 정비가 급선무: 키컬러, 서체, 사진 톤 확정
- 슈퍼말차가 "말차=힙"으로 인식 전환에 성공한 것처럼, "식물성 프로틴=감각적"으로 인식 전환 필요
- 최소 요건: 피그마 기반 인스타 무드보드 + 촬영 가이드라인

**[3] AG1의 심플한 메시지 구조 차용 — "한 줄로 끝내기"**
- AG1이 75가지 성분을 "이것 하나면 된다"로 압축한 것처럼
- 베지어트도 "속 편한 식물성 프로틴"이라는 한 문장에 모든 소구를 담아야 한다
- 인스타그램에서는 이 한 문장이 바이오, 캡션, 해시태그에 반복적으로 등장해야 함

**[4] Sweetgreen의 "식재료 히어로" 비주얼 전략 적용**
- 완두콩, 서리태, 현미 등 베지어트 원재료를 시네마틱하게 촬영
- "무엇으로 만들었는가"를 보여주는 것 자체가 클린/자연의 소구
- 홀블렌딩이 성분표로 설명하는 것을 베지어트는 비주얼로 보여줘야 한다
