## 1. 국내 식물성 단백질 시장

### 시장 구조
- **카테고리**: 식물성 프로틴 파우더 / 식물성 대체식품
- **성장 단계**: 초기 성장기 (인지도 확산 중)
- **주요 유통 채널**: 자사몰, 올리브영, 쿠팡, 카카오 선물하기

### 소비자 세그먼트

| 세그먼트 | 비중 | 특성 |
|---------|------|------|
| 비건/식물성 확신층 | 소수 | 가치관 기반, 가격 민감도 낮음 |
| 플렉시테리언 | 성장 중 | "가끔 식물성", 맛/편의성 중시 |
| 건강 의식 소비자 | 다수 | 30~50대, "뭘 먹어야 건강한지" 의식, 베지어트 핵심 타겟 |
| 프로틴 일반 소비자 | 최대 | 유청 단백질 위주, 식물성에 대한 인지도 낮음 |

### 시장 맥락 (2026년 초)
- 글로벌: 대체육 시장 둔화 (Beyond Meat, Impossible Foods 부진)
- 국내: 대기업(CJ 얼티브 등) 진입 → 유통 경쟁 심화
- 기회: "대체육"이 아닌 **"식물성 웰니스"** 프레이밍으로 차별화 가능
- 박진아 대표 LA Expo West 소감: "대체육 회사들은 다 죽어가는데, 식물성 프로틴은 다른 축"

## 3. 직접 경쟁사 프로필

### 홀블렌딩 (Whole Blending)
| 항목 | 내용 |
|------|------|
| 대표 | 이보희 (약사 출신) |
| 포지셔닝 | "약사가 만든 클린 프로틴" |
| 제품 | 식물성 프로틴 4종 (곡물, 카카오, 검은콩, 쑥) |
| 가격 | 프리미엄 (베지어트 대비 비슷하거나 높음) |
| 팔로워 | ~1만 |
| 강점 | 전문가 권위, V-LABEL 비건 인증, 클린 성분 |
| 약점 | 비주얼 투자 부족, 콘텐츠 비정기적 |
| **베지어트 차별점** | 라이프스타일 관점 vs 전문가 성분 관점, 브랜드 감도 우위 |

### 프로티원
| 항목 | 내용 |
|------|------|
| 포지셔닝 | 올영 가성비 프로틴 1위 |
| 특성 | 가격 전쟁, 프로모션 중심 |
| **베지어트 차별점** | 브랜드 감도 부재 → 감도로 이긴다. 식물성 전문 아님 |

### CJ 얼티브
| 항목 | 내용 |
|------|------|
| 포지셔닝 | 대기업 자본, 유통 장악 |
| 특성 | 대기업 톤, 다양한 제품 라인업 |
| **베지어트 차별점** | 스타트업 진정성/창업자 스토리 부재, 카테고리 순수성 없음 |

## 4. 글로벌 벤치마크

### AG1 (Athletic Greens) — 인스타 전략 벤치마크
- 크리에이터 퍼스트 전략
- 정돈된 미니멀리즘 + 과학적 감성
- 시그니처 그린 컬러 일관성
- **베지어트 적용**: 비주얼 감도의 기준점, 타이포 기반 피드 포스트 참고

### Sweetgreen — 비주얼 벤치마크
- 시네마틱 푸드 포토그래피
- 자연스러운 일상감 + 라이프스타일 통합
- 커스텀 타이포그래피
- **베지어트 적용**: 사진 스타일의 따뜻한 톤 참고

### 슈퍼말차 (SUPER MATCHA) — 국내 감도 벤치마크
- 힛더티(HIT THE TEA) 운영, 2017년 설립, 성수동
- 프리미엄 감도 + 힙한 라이프스타일
- 국내 F&B 중 인스타 비주얼 최상급
- 키컬러 일관성 유지
- **베지어트 적용**: 국내에서 구현 가능한 브랜드 감도 수준의 벤치마크

## 5. 경쟁 전략 요약

### 베지어트의 4가지 무기

1. **창업자의 얼굴과 목소리** — 박진아 대표의 개인 스토리, 팀의 진정성
2. **카테고리 순수성** — 100% 식물성만 (유청 단백질 안 팜)
3. **커뮤니티 리얼리티** — 러닝크루, 실제 유저 인터뷰 (스크립트 없는 진짜)
4. **해외 감도의 한국화** — AG1/Sweetgreen 수준을 한국 식물성 프로틴으로

### 전략 원칙
- "식물성 단백질의 기준" 프레이밍 → 비교하지 않고 정의한다
- 가격 경쟁 참여하지 않음 (블프 할인 콘텐츠 = 전환 저조 + 브랜드 훼손)
- 브랜드 감도에서 승부 (비주얼, 톤앤매너, 시리즈 일관성)

## 6. 해외 시장 (빙커 관련)

### 캐나다 식물성 프로틴 시장
- 플렉시테리언 인구 증가 추세
- "Korean Beauty from Inside" — K-뷰티에서 K-웰니스로의 확장 포지셔닝

### 글로벌 경쟁 환경
- Beyond Meat, Impossible Foods 등 대체육 기업 부진

- 베지어트/빙커의 포지셔닝: "대체육"이 아닌 "식물성 웰니스 프로틴"
