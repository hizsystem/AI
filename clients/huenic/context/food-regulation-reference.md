# 식품 규제 레퍼런스 — 휴닉(HUENIC) 전용

> 이 문서는 food-regulation-advisor 에이전트가 콘텐츠/카피 법적 리스크 점검 시 참조하는 레퍼런스입니다.
> 베지어트/빙커의 마케팅 카피에 직접 영향을 미치는 규제만 수록합니다.
> 최종 업데이트: 2026-04-13

---

## 1. 적용 법률 개요

| 법률 | 관할 | 휴닉 적용 포인트 |
|------|------|-----------------|
| **식품등의 표시·광고에 관한 법률** (식품표시광고법) | 식약처 | 모든 마케팅 카피의 1차 검증 기준 |
| **건강기능식품에 관한 법률** | 식약처 | 베지어트가 건기식이 아닌 '일반식품'임을 명확히 |
| **전자상거래법** | 공정위 | 자사몰/올영 판매 시 표시 의무 |
| **표시·광고의 공정화에 관한 법률** | 공정위 | 인플루언서 시딩, 체험단 뒷광고 규제 |

---

## 2. 휴닉 제품 분류: 일반식품

**중요**: 베지어트/빙커는 **일반식품(단백질 쉐이크)**이지, 건강기능식품이 아님.

### 일반식품이 할 수 있는 표현
- 원재료/성분 사실 기술 ("서리태 30% 함유", "완두콩 단백질 기반")
- 영양성분 표시 (단백질 Xg, 칼로리 등)
- 맛/식감 묘사 ("진한 미숫가루 맛", "속이 편한")
- 제조 과정/원료 스토리 (농가 직구매, 로스팅 과정)
- 라이프스타일 연결 ("아침 루틴", "운동 후 한 잔")

### 일반식품이 할 수 없는 표현 (위반 시 과태료/영업정지)

| 금지 유형 | 예시 (X) | 대체 표현 (O) |
|----------|---------|-------------|
| **질병 예방/치료 암시** | "탈모 예방", "대장염에 좋은" | "모발 건강에 도움이 되는 성분 함유" (주의: 이것도 건기식 인정 기능 표현에 해당할 수 있음) |
| **건기식 기능성 표현** | "면역력 강화", "혈당 조절", "체지방 감소" | 사용 불가. 일반식품에서는 기능성 표현 자체가 금지 |
| **의약품 오인 표현** | "처방", "복용", "치료" | "섭취", "드시기", "즐기기" |
| **비교 광고 (입증 불가)** | "경쟁사보다 단백질 2배" | 자사 제품 사실만 기술 |
| **과대/허위 광고** | "세계 최초", "국내 유일" | 입증 자료 확보 시에만 사용 가능 |

## 3. 휴닉 카피에서 자주 나오는 위험 표현 점검

### "모발 건강" 관련
- **현재 광고 메시지**: "식단+모발 관리 한 번에 / 검은콩 30%, 맥주효모, 비오틴"
- **위험도**: 중~높음
  - "모발 관리"는 건기식 인정 기능성 표현에 가까움
  - 일반식품에서 "모발 건강에 좋다"고 직접 표현하면 식품표시광고법 위반 소지
- **안전한 대체**:
  - "검은콩(서리태) 30% 함유" — 원재료 사실만 기술
  - "이소플라본, 안토시아닌이 풍부한 서리태" — 성분명만 나열
  - "할머니의 검은콩미숫가루를 현대적으로" — 스토리 전환
  - 소비자 후기로 간접 전달 (단, "개인 경험"임을 명시)

### "속이 편한" 관련
- **위험도**: 낮음
  - "속이 편한"은 식감/소화 편의성 묘사로 해석 가능
  - 단, "소화 기능 개선"으로 확대하면 건기식 기능성 표현
- **안전한 표현**: "속이 편한 식물성 단백질" (O) / "소화 기능을 개선하는" (X)

### "지속 가능한 웰니스" 관련
- **위험도**: 낮음
  - 라이프스타일/가치관 표현으로 규제 대상 아님
  - 단, "건강을 지켜준다", "질병을 예방한다"로 연결하면 위반

## 4. 인플루언서/시딩 규제

### 뒷광고 표시 의무 (공정위 추천·보증 심사지침)

| 요건 | 내용 |
|------|------|
| 경제적 이해관계 표시 | 제품 무상 제공, 원고료 지급 시 반드시 표시 |
| 표시 위치 | 게시물 **첫 줄** 또는 **썸네일에 포함** (하단 해시태그만으로 불충분) |
| 표시 문구 예시 | "제품을 제공받았습니다", "광고", "유료 파트너십" |
| 인스타그램 | Instagram 공식 "유료 파트너십" 라벨 사용 권장 |
| 스토리/릴스 | 영상 시작 3초 내 "광고" 자막 또는 음성 고지 |

### 휴닉 시딩 캠페인 체크리스트
- [ ] 시딩 제품 발송 시 뒷광고 표시 가이드 동봉
- [ ] 인플루언서에게 표시 의무 사전 고지 (DM/이메일)
- [ ] 게시물 업로드 후 표시 여부 모니터링
- [ ] 올영 체험단도 동일 기준 적용

---

## 5. 올리브영 체험단 특이사항

- 체험단 리뷰 = 광고로 간주될 수 있음
- 체험단 참여자에게 "솔직한 리뷰" 요청 + "제품 제공" 표시 의무
- 올영 자체 체험단 프로그램은 올영 가이드라인 준수 (별도 확인 필요)
- 리뷰 조작/허위 리뷰 작성 유도 시 전자상거래법 위반

---

## 6. 식물성 식품 특화 규제

### "대체육" 표시 관련
- 식약처는 "대체육"이라는 용어에 대한 명확한 법적 정의를 아직 확정하지 않음
- 그러나 "고기를 대체한다"는 표현은 축산물 관련 오인 유발 가능
- **휴닉 가이드**: 브랜드 전략상 이미 "대체육" 표현을 금지하고 있으므로 규제 리스크 낮음

### 영양성분 표시
- "고단백" 표시: 100g당 단백질 12g 이상 또는 100kcal당 6g 이상 시 가능
- "저칼로리" 표시: 100ml당 20kcal 이하 시 가능
- 영양성분 강조 표시 기준 충족 여부 확인 필요

---

## 7. 해외(캐나다) 규제 주의사항 — 빙커

| 기관 | 규제 |
|------|------|
| CFIA (Canadian Food Inspection Agency) | 식품 라벨링 규정 |
| Health Canada | 건강 강조 표시(Health Claims) 규제 |

### 주요 차이점
- 캐나다는 "plant-based" 표현 자체는 자유롭게 사용 가능
- 그러나 건강 관련 주장(Health Claims)은 Health Canada 승인 필요
- 영양소 함량 표시(Nutrient Content Claims)는 CFIA 기준 충족 필요
- 알레르기 표시 의무: 대두(soy) 포함 시 반드시 표시

## 8. 카피 검수 프로세스

```
콘텐츠 카피 초안 작성
    ↓
food-regulation-advisor 자동 검수
    ↓
위험 표현 발견 시 → 대체 표현 제안 + 위험 등급 표시
    ↓
  [녹색] 안전    → 그대로 진행
  [노란색] 주의  → 표현 수정 권장
  [빨간색] 금지  → 반드시 수정 후 진행
```
