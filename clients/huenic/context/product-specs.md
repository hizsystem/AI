## 2. 공통 제조 철학

### 원칙
1. **100% 식물성 원료만 사용** — 동물성 원료 절대 배제 (투자자/바이어의 "동물성 섞으라" 제안 모두 거절)
2. **클린 레이블** — 인공첨가물, 정제 설탕, 포도당, 수크랄로스, 스테비아 미사용
3. **원가 > 영업이익** — 좋은 원료 = 단가 상승 → 판매가 올리지 않고 영업이익 줄이는 방식 선택
4. **가루 품질** — 제조 단계 기술 투입으로 고운 입자 (목넘김 좋음). 단, 서리태는 콩 입자 특성상 약간 텁텁

### 서리태(검은콩) 제조 특이사항
- 농가에서 Whole bean 직접 구매 → 자체 로스팅 → 프로틴 파우더
- ISP(분리대두단백) 대비 번거롭고 비용 2배이지만 더 신선하고 건강
- 기계장비부터 새로 세팅하여 개발 (업계 최초)

### 딸기 프로틴 제조 특이사항
- 3~4년간 고객 요청 축적 → 2024년 개발 착수
- 생딸기 동결건조 → 파우더 → 23% 함량 (시중 제품은 딸기향+크리머+색소+인공감미료)
- 제조사 경고: "단가/수급 감당 불가" → 그래도 강행
- 대표 코멘트: "전 세계에 이런 원료/밸런스의 프로틴은 없을 것"

## 4. 핵심 소구 포인트 (콘텐츠 활용)

### 기능적 USP
- **속 편한 식물성 단백질** — 유당 불내증 걱정 없음
- **서리태 = 모발건강** — 이소플라본, 안토시아닌 (식단+모발 관리 한 번에)
- **완두콩단백 PURIS** — 탄소 발자국 1kg당 0.2kg (가장 지속 가능한 단백질)
- **클린 레이블** — 뒷면 표시사항만 봐도 차이를 알 수 있음

### 감성적 USP
- 할머니의 검은콩미숫가루 맛 → 서리태 쉐이크로 구현 (대표 개인 스토리)
- "가장 쉽고 간편하지만 지속 가능한 방법"
- 매일 마시는 루틴 (특별한 날이 아닌 일상)

### 가격 허들 대응 화법
- "매일 마셔야 하고 내 몸에 들어가는 거라면 좋은 음식으로 나를 채워주는 게 더 행복한 선택"
- "실제 성분/원재료 따지면 비싸지 않다" → 콘텐츠로 풀어내기
- 올영 파우치 1포씩 추천 (진입장벽 낮추기)

## 5. 고객 반응 데이터

### 긍정 반응 키워드
- "성분도 좋은데 맛있다"
- "재구매하고 싶다"
- "먹고 (건강) 문제들이 해결되었다"
- "물이랑만 타 먹어도 진한 미숫가루 맛"

### 부정 반응 키워드
- "맛이 없다" (주관적)
- "가격이 비싸다" → 성분 대비 합리적임을 콘텐츠로 설명 필요

### 잘 된 콘텐츠 패턴
- 직원이 가장 좋아하는 맛 릴스 → ROAS 300%+, 조회수 4.5만
- 파우치 팔로워 이벤트 → 팔로워 600명+ 유입

### 안 된 콘텐츠 패턴
- 베지레터(뉴스레터형) → 조회수 1,000 미만
- 블프 할인 프로모션 → 전환 저조 ("싸다는 메시지 = 베지어트와 거리감")
- 대표 감사인사 릴스 → 조회수 매우 저조
