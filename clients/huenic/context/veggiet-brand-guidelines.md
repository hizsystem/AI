## 1. 브랜드 아이덴티티

### 포지셔닝 스테이트먼트
**"식물성 단백질의 기준을 만드는 브랜드"**

- 제품 판매 채널이 아니라 **식물성 웰니스 라이프스타일을 정의하는 브랜드**
- 카테고리 리더십: "식물성 단백질 = 베지어트"라는 인식 구축
- 파타고니아가 의류에서 지속 가능성을 대표하듯, 베지어트가 식품에서 그 역할

### 브랜드 보이스 키워드 (3가지)

| 키워드 | 정의 | 적용 |
|--------|------|------|
| **진정성(Authenticity)** | 식물성 단백질을 진심으로 만드는 팀의 이야기. 꾸미지 않되 감도는 높게 | 대표/팀 비하인드, 원료 스토리, 제조 과정 |
| **기준(Standard)** | "식물성 단백질은 베지어트"라는 카테고리 리더십. 1등의 톤 | 제품 USP, 성분 비교, 브랜드 철학 |
| **일상(Everyday)** | 특별한 날이 아닌 매일의 루틴 속 베지어트. 가볍고 자연스러운 접점 | TPO 콘텐츠, 레시피, 모먼트 |

### 창업 스토리 (콘텐츠 원천)
- **시작**: 대표 남편의 궤양성대장염(8년 희귀난치질환) → 유당 소화 불가, 첨가물 가득한 보충제 먹으면 배탈
- **전환점**: 아마존 목축업 다큐 → 식물성 식단 시작 → 1년 만에 염증 소멸
- **확신**: "인간도 건강, 환경도 건강 = 하지 않을 이유가 없다"
- **서리태 비하인드**: 대표 선천적 새치(7살부터) → 할머니가 매일 검은콩미숫가루 → 그 맛을 서리태 쉐이크로 구현
- **5년 후 비전**: "식물성 단백질의 대명사"

## 2. 톤앤매너

### 말투
- **기본 톤**: 담백하고 자신감 있는 1인칭 브랜드 보이스. 친근하되 가볍지 않다.
- **문체**: 짧은 문장. 군더더기 없이. 여운을 남기는 마침표.
- **호칭**: "당신" 또는 직접 호칭 없이 상황 묘사 ("아침에 눈 뜨고 제일 먼저 하는 일")

**톤 스펙트럼**
```
[차갑다] --------- [베지어트] --- [따뜻하다]
                       ^
              담백하고 건조하되
              진심이 느껴지는 온도
```

### 카피 예시
- O: "속이 편한 아침. 오늘도 식물성으로."
- O: "서리태 한 스쿱이면 충분합니다."
- O: "매일 마시는 사람은 압니다."
- X: "여러분~ 오늘도 건강한 하루 보내세요!"
- X: "대체육의 혁명! 놀라운 맛!"
- X: "식물성 단백질계의 넘버원!"

### 금지어 / 대체 표현

| 금지 | 대체 표현 |
|------|----------|
| 대체육, 대체식품 | 식물성 식품, 식물성 단백질 |
| 비건 (단독 사용) | 100% 식물성, 식물성 기반 |
| 혁명, 혁신 | 기준, 새로운 선택지 |
| ~해요체 과다 | 해요체 최소화, 문어체/명사형 종결 선호 |
| 최고, 최강, 넘버원 | "식물성 단백질의 기준" (자연스러운 1등 암시) |
| 싸다, 저렴하다 | 가치 중심 표현 ("매일 먹어도 좋은 선택") |

- 브랜드 고정: #베지어트 #VEGGIET #식물성단백질
- 시리즈별: #이달의베지어터 #베지어트랩 #베지어트모먼트
- 확장 키워드: #프로틴 #웰니스 #식단관리 #아침루틴 #운동후단백질
- 국문/영문 혼용: 캡션 하단에 영문 1~2줄 병기 (해외 영업 활용)

## 3. 비주얼 가이드라인

### 컬러 팔레트

```
Primary (메인 3색)
  베지어트 그린 — 브랜드 키컬러 (기존 BI 연장)
  크림 화이트   — 배경, 여백
  웜 베이지    — 서브, 텍스트 배경

Secondary (서브 3색)
  딥 브라운      — 텍스트, 타이포
  소프트 테라코타 — 포인트, 따뜻한 감성
  라이트 세이지   — 신선함, 원재료 연상

시즌 컬러
  딸기 핑크 — 딸기 프로틴 론칭용 (시즌 한정 포인트)

금지 컬러
  네온/형광 계열  → B급 느낌 유발
  순수 블랙 배경  → 브랜드 무드와 불일치
  과채색 컬러    → 시장/마트 느낌 유발
```

### 사진 스타일

**지향점**: AG1의 정돈된 프리미엄감 + Sweetgreen의 자연스러운 일상감

| 항목 | 가이드 |
|------|--------|
| 라이팅 | 자연광 기본. 소프트하고 따뜻한 톤. 플래시/인공 조명의 날카로운 그림자 지양 |
| 구도 | 여백 충분히. 오브제 최소화. 제품이 화면의 40~60% |
| 배경 | 크림/베이지/리넨/우드/대리석. 깨끗하되 차갑지 않은 소재 |
| 인물 | 얼굴 클로즈업보다 실루엣, 손, 상반신 크롭. 자연스러운 동작 |
| 원재료 | 서리태, 딸기 등을 스타일링 소품처럼 (산지 느낌 X, 테이블 세팅 O) |
| 텍스트 오버레이 | 산세리프 서체. 좌측 또는 하단 정렬. 화면의 20% 이하 |

### 인물 촬영 가이드 (B급 방지)
- 대표/팀 등장 시: 실루엣, 손, 상반신 크롭 우선
- 자연스러운 동작 (포즈 X) — "찍히고 있다"는 느낌 배제
- 감도 = 촬영 디렉션에서 나온다 (앵글, 라이팅, 크롭으로 해결)
- 프로모션에도 B급 안 되게: 브랜드 톤 안에서 프로모션 에셋 사용

### 타이포그래피

| 용도 | 방향 |
|------|------|
| 타이틀 | 산세리프, Medium~Bold. 영문은 대문자 활용 (VEGGIET, THE STANDARD) |
| 본문 | 가독성 우선. 세리프 or 산세리프 Light~Regular |
| 포인트 | 손글씨체 절대 지양. 감도 = 정돈된 타이포에서 나온다 |
| 국/영문 혼용 | 한글 타이틀 아래 영문 서브타이틀 배치 |

### 피드 그리드
- 3열 그리드에서 전체적으로 **크림~그린 톤의 통일감**
- 제품 컷 / 라이프 컷 / 타이포 컷을 번갈아 배치하여 리듬감
- 한 줄(3개)을 봤을 때 "하나의 브랜드"로 읽히는 것이 기준

## 6. 경쟁 포지셔닝

### 직접 경쟁사

| 경쟁사 | 포지셔닝 | 베지어트 차별점 |
|--------|---------|---------------|
| 프로티원 | 올영 가성비 프로틴 1위 | 브랜드 감도 부재 → 감도로 이긴다 |
| 홀블렌딩 | 약사 만든 클린 프로틴 | 전문가 권위 vs 브랜드 세계관 |
| 슈퍼말차 | 프리미엄 F&B 감도 최상급 | 말차 ≠ 식물성 단백질, 카테고리 다름 |
| CJ 얼티브 | 대기업, 유통 장악 | 스타트업 진정성/창업자 스토리 부재 |

### 베지어트의 무기 (대기업이 못 하는 것)
1. **창업자의 얼굴과 목소리** — 박진아 대표의 철학이 브랜드에 묻어남
2. **카테고리 순수성** — 100% 식물성만 한다. "하나만 한다"가 곧 브랜드
3. **커뮤니티 리얼리티** — 러닝크루, 실제 유저 인터뷰 = 진짜 경험
4. **해외 감도의 한국화** — AG1/Sweetgreen 수준 비주얼을 한국 식물성 프로틴으로

### 경쟁 우위 실행 원칙
- 1등은 1등답게 말한다: 비교하지 않고 정의한다
- 진정성은 편집에서 나온다: 날것 + 감도 높은 컬러그레이딩/구도
- 반복이 브랜드를 만든다: 시리즈 포맷 일관 유지
- 숫자보다 장면을 남긴다: "단백질 20g" 대신 "운동 끝나고 한 잔, 속이 편했다"

## 7. 타겟 오디언스

### META 광고 현황 (인국님 기준)
- 월 예산: 약 2,000만원
- 주요 타겟: 30~50대 여성 (연령만 제한, 어드밴티지 타겟)
- 메시지: 식단+모발 관리 한 번에 / 검은콩 30%, 맥주효모, 비오틴
- ROAS: 약 200%
