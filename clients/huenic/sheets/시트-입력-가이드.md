## 탭별 입력 방법

### 1. 주간성과 (매주 금요일)

Meta Business Suite → 인사이트 → 개요에서 확인할 수 있는 수치입니다.

| 열 | 뭘 입력하나요? | 어디서 보나요? | 예시 |
|----|--------------|-------------|------|
| 브랜드 | 베지어트 또는 빙커 | - | 베지어트 |
| 주차 | 해당 주차 (형식: 2026-W13) | - | 2026-W13 |
| 기간 | 해당 주 월~일 | - | 2026-03-23 ~ 2026-03-29 |
| 팔로워 | 금요일 기준 총 팔로워 수 | 인스타 프로필 | 12800 |
| 팔로워증감 | 지난주 대비 늘거나 줄은 수 | 전주 팔로워에서 빼기 | 220 |
| 게시물수 | 이번 주에 올린 게시물 수 | 직접 카운트 | 3 |
| 참여율 | 참여율(%) | 인사이트 → 참여 | 4.3 |
| 참여율증감 | 지난주 대비 변동 | 전주 참여율에서 빼기 | 0.1 |
| 최고좋아요 | 이번 주 가장 좋아요 많은 콘텐츠의 좋아요 수 | 각 게시물 인사이트 | 380 |
| 도달 | 이번 주 총 도달 수 | 인사이트 → 도달한 계정 | 48000 |
| 도달증감 | 지난주 대비 변동 | 전주 도달에서 빼기 | 3000 |

**입력 팁**:
- 숫자만 입력하세요 (12800, 콤마 없이)
- 참여율은 소수점 한 자리까지 (4.3)
- 줄어든 경우 마이너스 붙이세요 (-200)

| 열 | 뭘 입력하나요? | 예시 |

### 3. 월간KPI (매월 말)

한 달 전체를 정리하는 시트입니다. 전월과 비교한 변동 수치도 입력합니다.

| 열 | 뭘 입력하나요? | 예시 |
|----|--------------|------|
| 브랜드 | 베지어트 또는 빙커 | 베지어트 |
| 연도 | 연도 | 2026 |
| 월 | 월 (숫자) | 4 |
| 팔로워 | 월말 기준 팔로워 | 13200 |
| 팔로워증감 | 전월 대비 증감 | 860 |
| 팔로워증감률 | 증감률 (%) | 7.0 |
| 월간게시물 | 이번 달 총 게시물 수 | 14 |
| 게시물증감 | 전월 대비 증감 | 2 |
| 게시물증감률 | 증감률 (%) | 16.7 |
| 평균참여율 | 월 평균 참여율 (%) | 4.4 |
| 참여율증감 | 전월 대비 변동 | 0.2 |
| 월간도달 | 월 총 도달 | 205000 |
| 도달증감 | 전월 대비 증감 | 17500 |
| 도달증감률 | 증감률 (%) | 9.3 |

**팁**: 증감률은 (이번달 - 지난달) / 지난달 × 100으로 계산하면 됩니다.
시트에 수식을 걸어두면 자동 계산됩니다.

### 4. 팔로워추이 (매월 말)

| 열 | 뭘 입력하나요? | 예시 |
|----|--------------|------|

### 5. 참여율추이 (매주 금요일)

주별 참여율 breakdown입니다. KPI 차트에 쓰입니다.

| 열 | 뭘 입력하나요? | 예시 |
|----|--------------|------|
| 브랜드 | 베지어트 또는 빙커 | 베지어트 |
| 주차 | 표시용 (W몇인지) | W13 |
| 전체 | 전체 참여율 (%) | 4.3 |
| 피드 | 피드 게시물 참여율 | 3.9 |
| 릴스 | 릴스 참여율 | 6.2 |
| 스토리 | 스토리 참여율 | 2.4 |

**참여율 계산**: (좋아요 + 댓글 + 저장) / 팔로워 × 100
Meta Business Suite에서 콘텐츠 유형별로 필터링해서 확인할 수 있습니다.

| 실수 | 해결 |

## 참여율이 뭔가요?

**참여율(Engagement Rate)** = 팔로워가 콘텐츠에 얼마나 반응하는지 보는 지표

```
참여율(%) = (좋아요 + 댓글 + 저장) / 팔로워 수 × 100
```

- **3% 이상**: 좋은 편 (식품/라이프스타일 계정 기준)
- **5% 이상**: 매우 좋음
- **1% 이하**: 개선 필요

릴스가 보통 피드보다 참여율이 높고, 스토리는 낮습니다.
이 차이를 보면서 "어떤 유형의 콘텐츠에 반응이 좋은지" 판단합니다.
