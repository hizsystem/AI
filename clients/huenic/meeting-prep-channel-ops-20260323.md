## 자료 1. 서두 — 셋이 같이 보는 것 (화이트보드용)

### v1 → v2: 뭐가 달라졌나

```
v1: 브랜드가 할 말 → 시리즈 → 콘텐츠
v2: 타겟의 진짜 고민 → 그 고민에 답하는 콘텐츠 → 시리즈
```

사람들은 베지어트의 철학이 궁금해서 인스타를 보지 않는다.
**자기 고민을 해결해줄 것 같아서 멈추는 거다.**

### 프로틴 먹는 25~40 여성의 5가지 고민

| # | Concern (타겟의 머릿속) | 한 줄 |
|---|----------------------|-------|
| 1 | **"이게 나한테 맞아?"** | 프로틴은 운동하는 사람 거 아니야? |
| 2 | **"진짜 괜찮은 거야?"** | 첨가물 잔뜩 아냐? 이 브랜드 믿을 만해? |
| 3 | **"맛있어? 어떻게 먹어?"** | 전에 프로틴 먹어봤는데 비린내... |
| 4 | **"먹으면 뭐가 달라져?"** | 한 달째 먹는데 뭐가 달라진 건지 모르겠어 |
| 5 | **"이 브랜드는 뭐가 달라?"** | 프로티원이랑 뭐가 다른 거야? |

### 핵심 원칙 2개

1. **포맷은 우리가 지정하지 않는다** — concern과 역할만 정하고, 릴스/캐러셀/피드는 인국/예랑 재량
2. **모든 콘텐츠에는 역할이 있다** — 발견 / 설득 / 유지 / 확산 중 하나

## 자료 1-B. 브랜드 메시지 — 아직 확정 아님 (중요)

### 베지어트의 한 줄 — 작업 기준은 "속 편함", 인터뷰 후 미세 조정

**작업 기준 (거의 확정):**
> "매일 먹어도 속 편한, 식물성 프로틴의 기준"

**왜 이게 유력한가:**
- 고객이 실제로 하는 말: "속이 편해서 매일 먹는다", "12시까지 배가 안 고프다"
- 경쟁사가 점유하지 않은 포지션: 홀블렌딩=약사 권위, 슈퍼말차=힙 감도
- **뾰족함 테스트 통과**: 이 한 줄에서 "베지어트"를 지우고 오필리/프로티원을 넣으면 안 맞음. 베지어트에만 성립.

**검토했다가 탈락한 후보:**
- "매일 한 끼, 좋은 걸로 채우는 습관" → 오필리(한끼대용 잔소리 쉐이크), 플라이밀 등 누구나 쓸 수 있는 말. 카테고리 설명이지 뾰족함이 아님.
- "100% 식물성 단백질" → 카테고리 설명
- "웰니스 식습관의 시작" → 너무 넓음

**인터뷰 답변에 따른 미세 조정 가능성:**
대표 Q2 "타협하지 않은 것"이 원료/성분/제조에 가까우면, "속 편한" 위에 성분 뾰족함이 추가될 수 있음 (ex: "서리태만 고집하는 이유"가 메시지에 녹을 수 있음). 하지만 "속 편한"을 버리는 게 아니라 보강하는 방향.

### 뾰족함 테스트 — 이 기준으로 판단한다

> **"이 한 줄에서 브랜드 이름을 지우고 경쟁사 이름을 넣어도 성립하면, 뾰족한 게 아니다."**

| 후보 | 오필리에 넣으면? | 홀블렌딩에 넣으면? | 판정 |
|------|---------------|----------------|------|
| 속 편한 식물성 프로틴의 기준 | 안 맞음 (식물성 전문 X) | 안 맞음 (약사 권위지 속 편함 아님) | **통과** |
| 좋은 걸로 채우는 습관 | 성립 | 성립 | **탈락** |

### 왜 이게 중요한가 — 메시지가 모든 걸 결정한다

```
메시지 (한 줄) → 무드 (감각) → 비주얼 룰 (실행 기준) → 피드 그리드
```

"속 편한"이 메시지면: 부드러운 크림톤, 여백, 자연광 = "속이 편한" 느낌의 시각화
여기에 성분 뾰족함이 추가되면: 원재료(서리태)의 시네마틱한 비주얼이 더해짐

**수민에게**: "속 편한"을 기준으로 무드보드 잡되, 원재료 비주얼도 레퍼런스에 포함
**남중에게**: concern 리서치할 때 "속 편함" 관련 리뷰를 특히 주목

### 메시지 결정 이전에도 확실한 것들

어떤 방향이든 변하지 않는 것:

- **5가지 concern 구조**: 타겟의 고민에서 콘텐츠가 나온다
- **콘텐츠 역할 4가지**: 발견/설득/유지/확산
- **R&R**: HIZ = concern+역할, 휴닉 = 포맷+실행
- **톤앤매너**: 담백하고 자신감 있는 1인칭. 비교하지 않고 정의한다
- **금지어**: 대체육, 대체식품, 비건(단독), 최고/최강

## 자료 2. v2 시리즈 요약표 (1장)

### 5 Concern x 8 시리즈 x 4 역할

| Concern | 시리즈 | 월 빈도 | 역할 | 성장/수비 |
|---------|--------|--------|------|----------|
| 1. 나한테 맞아? | **이달의 베지어터** | 1회 | 설득+유지 | 수비 |
| 2. 괜찮은 거야? | **베지어트 랩** | 1회 | 유지 | 수비 |
| 2. 괜찮은 거야? | **베지레터** (신규) | 1~2회 | **발견+확산** | **성장 엔진** |
| 3. 맛있어? 어떻게? | **베지어트 한 끼** | 1~2회 | **발견** | **성장 엔진** |
| 3. 맛있어? 어떻게? | **쉐이크 유형** (신규) | 1회 | **발견+확산** | **성장 엔진** |
| 4. 뭐가 달라져? | **베지어트 모먼트** | 2회 | 설득 | 수비 |
| 5. 뭐가 달라? | **식물성의 기준** | 1회 | 설득 | 수비 |
| 5. 뭐가 달라? | **만드는 사람들** | 격월 1회 | 유지(팬덤) | 수비 |
| (횡단) | **New Drop** (론칭 시) | 출시 시 | 복합 | 캠페인 |

> **월간 총 10~12개.** 성장 엔진 3개(베지레터, 한 끼, 쉐이크 유형)가 계정을 키우고, 수비 4개가 피드를 지킨다.

### 콘텐츠 역할 정의

```
[발견]                    [확산]
 비팔로워가 처음 만남       친구한테 "이거 봐" 공유
 → 조회수, 비팔 도달        → DM 공유수, 태그
       \                    /
        +--→ [설득] ←------+
             프로필 와서 "괜찮네" 판단
             → 팔로우 전환
                  |
              [유지]
              계속 보고, 신뢰 쌓임
              → ER, 저장수
```

**핵심**: 발견/확산이 사람을 데려오고, 설득이 팔로워로 만들고, 유지가 팬으로 만든다.

### 시리즈별 역할 매핑 한눈에

```
[발견/확산 = 성장 엔진]
  베지레터 ........... 찐후기 = 공유 유도
  베지어트 한 끼 ..... 레시피 = 저장 유도
  쉐이크 유형 ........ 밈/공감 = DM 공유

[설득 = 쇼룸]
  베지어트 모먼트 .... 피드 그리드 톤 유지
  식물성의 기준 ...... 브랜드 포지셔닝 선언
  이달의 베지어터 .... "이런 사람이 먹는 브랜드"

[유지 = 팬덤]
  베지어트 랩 ........ 성분/제조 신뢰 깊게
  만드는 사람들 ...... 대기업이 못 하는 진정성
```

## R&R 재정리 (v2 기준)

### HIZ가 주는 것 vs 휴닉이 결정하는 것

| HIZ (전략/방향) | 휴닉 (실행/포맷) |
|--------------|-------------|
| 이 콘텐츠가 답하는 concern | 릴스로 할지 캐러셀로 할지 |
