## 4단계: 시장/경쟁 환경

### K-뷰티 시장 현황 (2025-2026)

- 2025년 한국 화장품 수출액 104억 달러 돌파 — 프랑스에 이어 세계 2위
- 기초화장품(스킨케어)이 전체 수출의 41.7%로 최대 비중
- 글로벌 스킨케어 시장 1,800억 달러 규모 (2025)
- 인디 브랜드 중심 성장세: 토리덴, VT, 퓌, 티르티르 등 급성장

### 코큐텐(CoQ10) 스킨케어 포지션

| 구분 | 현황 |
|------|------|
| 이너뷰티(영양제) | 인지도 높음 — 항산화, 에너지, 심혈관 건강 |
| 스킨케어(화장품) | 니치 성분 — 레티놀, 비타민C, 나이아신아마이드 대비 인지도 낮음 |
| 시장 기회 | "성분 스토리텔링"이 가능한 차별화 포인트 |
| 시장 리스크 | 소비자 인지도 부족 → 교육 비용 높음 |

### 직접 경쟁 구도

에센셜리의 정확한 가격대/타겟이 불분명하나, 코큐텐 성분 스킨케어 + D2C + 소규모 인디 브랜드 기준으로:

| 경쟁 유형 | 브랜드 예시 | 특징 |
|-----------|------------|------|
| 성분 경쟁 | 닥터오라클 (CoQ10 라인 보유) | 코큐텐 성분 화장품 제조사, 전문 피부과 브랜드 |
| 카테고리 경쟁 | 에센시엘, 에센느, 에센허브 | 유사 네이밍 + 스킨케어 인디 브랜드 |
| 포지션 경쟁 | 라운드랩, 아누아, 달바 등 | 성분 중심 인디 브랜드, D2C → 올리브영 확장 성공 사례 |
| 가격대 경쟁 | 중저가 스킨케어 인디 브랜드 전체 | 올리브영 입점 브랜드 다수 |

### 시장 진입 장벽

1. **발견 가능성(Discoverability)**: 올리브영 미입점 + 검색 노출 약함 → 소비자가 브랜드를 찾을 경로 없음
2. **성분 인지도**: CoQ10은 화장품 성분으로서 대중적 인지도 부족
3. **브랜드 네이밍 혼동**: "에센셜", "에센시엘", "에센느" 등 유사 이름 다수 → 검색/인지 차별화 어려움
4. **대전 기반**: 서울 중심 뷰티 업계 네트워크/유통 접근 제한

### 핵심 과제 가설 (상담 시 검증 필요)

**가설 1: 광고 효율 문제 = 퍼널 하단 문제가 아니라 퍼널 전체 문제**
- 월 100만원으로 퍼포먼스 광고를 돌려봤으나 ROAS가 안 나옴
- 원인 추정: 브랜드 인지도 0 상태에서 전환 광고 → 당연히 비효율
- 광고 문제가 아니라, 광고를 받쳐줄 브랜드/콘텐츠/채널 인프라 부재

**가설 2: "빠른 매출 성과 검증" 기대 vs 현실 갭**
- 파일럿 테스트 희망 → 단기 ROI 기대
- 현실: 인지도 0 브랜드의 퍼포먼스 마케팅은 최소 3개월 데이터 축적 필요
- 월 100만원으로는 학습 데이터조차 충분히 모이지 않을 수 있음

**가설 3: 채널 다변화 부재**
- 자사몰 단일 채널 → 트래픽 소스가 유료 광고에 100% 의존
- 네이버 스마트스토어, 쿠팡 등 검색 기반 채널 미활용
- 오가닉 트래픽(블로그, SNS, 리뷰) 거의 0
