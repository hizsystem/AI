## 📋 내부 랩업 (사내 공유용)

### 1. 미팅 핵심 요약

- OEM 공장 보유 → 자체 브랜드 런칭한 1인 제조/판매 사업자. 비누 월 800만원(광고비 50만원)으로 수익 구조는 건강하나, **5년간 월 1,500만원 천장**에 갇혀 있는 상태.
- 네이버 검색 상위노출(90%)에 매출이 편중되어 있고, CRM·재구매 전략이 전무. 화장품 확장을 시도 중이나 아직 매출 견인 실패.
- 월 100~200만원 마케팅 예산 투입 의향 있으며, **히어로 제품 집중 → 채널 확장** 순서에 대해 공감대 형성됨.

### 2. 사전 가설 vs 실제 확인

| 사전 가설 | 상담에서 확인된 내용 | 판정 |
|-----------|---------------------|------|
| 표면 니즈: 퍼포먼스 마케팅으로 매출 올리기 | 퍼포먼스보다는 **"매출이 나오는 채널에 집중"**이 실제 표현. 광고 확대보다 효율 집중 성향 | **수정** — 채널 최적화 욕구가 더 정확 |
| 히어로 제품 부재가 근본 문제 | 대표 스스로도 "설거지 비누로 먼저 키우고 → 화장품" 순서 인식. 히어로 선정 필요성 동의 | **일치** |
| 마케팅 분산이 L2 Pain | 네이버 90% + 맘카페 10%로 사실상 **편중**. 쿠팡은 수수료 후 이익 100만원 | **수정** — 분산보다 **편중 + 채널 다변화 실패**가 정확 |
| 브랜드 전략 부재 (L4) | 브랜드 전략 언급 자체 없음. 제품·채널·매출 중심 사고 | **일치** — 다만 대표가 문제로 인식 않으므로 접근법 조정 필요 |
| 스마트스토어 찜 8,000명 미활용 | 고객관리 안 하고 있음 확인. 재구매율 20%인데 CRM·쿠폰 없음 | **일치** — 미활용 자산 확정 |
| "돈 투자했지만 이익 없었다" 감정 | 쿠팡 이익 100만원, 인플루언서/체험단 화장품 효과 X | **일치** — ROI 민감성 높음 |

**신규 발견:**
- **핫딜 판매 고려 중** — 1KG 대용량 단위 구매 패턴 → 번들/세트 전략 가능성
- **아기침대 판매 이력** — 블로그 체험단으로 매출 올린 성공 경험 → 콘텐츠 마케팅에 기본 이해와 신뢰
- **비누 단일 제품 3,000만원 목표** — 구체적 숫자 목표 → 제안서에 역산 로드맵 넣으면 설득력 높음

### 3. 대표 온도 평가

| 항목 | 평가 | 근거 |
|------|------|------|
| **관심도** | ★★★☆☆ (3/5) | 상담 참여 + 예산 언급했지만, "지금 적은 고민이 전부"는 탐색 단계 시그널 |
| **긴급도** | ★★☆☆☆ (2/5) | 5년간 1,500만원 유지 = 급하지 않음. "불편하지만 생존 가능"한 구간 |
| **예산 의지** | ★★★☆☆ (3/5) | 월 100~200만원 언급, 그러나 쿠팡 이익 100만원에 민감 → 실제 집행은 보수적 가능성 |

**수주 확률: 40~50%**
- (+) 직접 제조 기반, 히어로 전략 공감, 예산 범위 언급
- (-) 긴급도 낮음, "고민이 전부" 태도 = 외부 도움 필요성 크게 안 느낌, 참석자 불일치 미해결

**의사결정 스타일**: 실용주의·데이터형. "이거 하면 매출 얼마나 오르나"에 반응하는 타입.
**핵심 관심사**: 설거지 비누 매출 3,000만원 돌파 → 화장품 라인 안착. 최소 비용 최대 효율.

### 4. 수주 전략 업데이트

| 항목 | 사전 | 업데이트 |
|------|------|----------|
| 진입 프레임 | "브랜드 전략 부재" 지적 | **금지**. "설거지 비누 3,000만원 로드맵"으로 진입 |
| Tier 추천 | Tier1 진단(80~100만원) | **Tier1.5 — 진단+실행 (월 120~150만원, 2개월)** |
| 핵심 훅 | 투자 3.3% 임계점 | 유지 + **"비누 3,000만원 달성 시 연매출 시나리오"** 수치 제시 |

**추천 솔루션 1줄**: "설거지 비누 히어로 세트 기획 + 스마트스토어 CRM 세팅 + 네이버 광고 최적화" 2개월 파일럿
**가격 민감도**: 중상 — ROI 명확하지 않으면 집행 안 할 타입. 첫 달 성과 지표 약속 필요.

### 5. 리스크 & 기회

🟢 **기회**
- 자체 공장 보유 → 세트/번들/핫딜 자유, 마진 높음
- 비누 월 800 / 광고비 50만원 → ROAS 이미 양호, 구조화하면 레버리지 큼
- 재구매율 20% + CRM 제로 → 가장 쉬운 Quick Win
- 찜 8,000명 즉시 활용 가능
- 오가닉 상위노출 (#입욕제 #세안비누) → SEO 기반 존재

🔴 **리스크**
- 참석자 불일치 미해결 → 제안서 발송 대상 불명확
- 낮은 긴급도 → 의사결정 지연·자연 소멸 가능성
- 1인 운영 → 전략 줘도 실행 병목 발생 가능
- 화장품 스코프 넓어지면 리소스 분산 → 비누 집중 명확히

🟡 **불확실**
- 실제 마진율 미확인
- "카톡방" 유입 경로 상세 불명
- 대용량 핫딜이 브랜드 가치와 양립 가능한지
