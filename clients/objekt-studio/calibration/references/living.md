### 1. HAY (@haydesign)

**Brand Overview**
Danish design brand (founded 2002, now owned by Herman Miller). Accessible Scandinavian design. "Democratic design" — high-design at mid-range prices.

**Visual Characteristics**

| Element | Description |
|---------|-------------|
| Lighting | Flat, even, diffused studio light. Almost shadowless. Occasional hard directional light for editorial content. |
| Composition | Centered single object or tight grid arrangements. Strong geometric order. Products as graphic elements. |
| Color Palette | Saturated pastels + bold primaries against neutral backgrounds. Signature: candy-like pops against white/light grey. |
| Backgrounds | Pure white (#FFFFFF), warm off-white (#F5F2ED), or single saturated color block matching or contrasting the product. |
| Textures | Smooth, clean. Almost no organic textures — everything reads as crisp and graphic. |

**Key Colors (Approximate)**
- Background whites: #FFFFFF, #F5F2ED, #EDEDED
- Brand yellow: #FFD700 → #F2C94C
- Signature pastels: lavender (#C3B1E1), soft green (#A8D8B9), dusty pink (#E8C4C4), sky blue (#87CEEB)
- Accent primaries: red (#E63946), cobalt (#2D5BFF)

**What Makes It Distinctive**
- "Color blocking" 접근 — 제품 컬러와 배경 컬러의 의도적 조합이 핵심
- 오브제를 그래픽 요소처럼 다룸. 사진이라기보다 포스터에 가까운 느낌
- 그림자를 최소화하여 flatness를 극대화
- 제품 군집 사진: 다수 오브제를 컬러/형태 기준으로 정렬하여 그리드 구성

**Objekt Interpretation Notes**
- HAY 해석 시: 배경색과 오브제 컬러의 "조합 자체"가 크리에이티브. 단순히 제품을 찍는 게 아니라 컬러 palette를 디자인하는 감각.
- Lighting: 완전 diffused, 그림자 최소. Almost flat lay graphic.
- 프롬프트 키워드: `flat even lighting, color-blocked background, centered composition, graphic minimalism, no shadows, Scandinavian design photography`

### 2. Diptyque (@diptyque)

**Brand Overview**
Parisian luxury maison (founded 1961). Candles, fragrances, home scents. Art-meets-parfumerie. Signature oval label, hand-drawn botanical illustrations.

**Visual Characteristics**

| Element | Description |
|---------|-------------|
| Lighting | Warm, painterly light. Directional (often from upper-left), creating soft chiaroscuro. Candle glow frequently used as practical light source. |
| Composition | Tableau vivant / still-life painting approach. Products as part of a scene, never isolated. Deliberate "staged casualness." |
| Color Palette | Deep, muted, old-world. Olive, burgundy, ivory, aged gold. Black-and-white label as constant anchor. |
| Backgrounds | Dark/moody (charcoal, deep green, midnight blue) or marble/stone surfaces. Vintage furniture, draped fabrics. |
| Textures | Rich and layered — aged wood, linen, dried botanicals, wax drips, glass reflections. |

**Key Colors (Approximate)**
- Signature black: #1A1A1A (label)
- Ivory/cream: #F4EDE4, #EBE4D8
- Olive/sage: #6B7B5E, #8A9A7B
- Burgundy: #6B1D2A, #8C3A4B
- Aged gold: #C5A55A, #B8963E
- Background darks: #2C2C2C, #1E2D2B

**What Makes It Distinctive**
- 17~18세기 네덜란드/프랑스 정물화(vanitas) 직접 레퍼런스
- "Candle as light source" — 촛불 자체가 조명이 되어 분위기를 만드는 접근
- 텍스처 레이어링: 매 이미지에 최소 3-4가지 소재가 공존 (유리, 왁스, 나무, 직물, 도자기)
- 계절감이 강함. 봄엔 꽃, 가을엔 dried leaves, 겨울엔 velvet + warm light
- 오브제가 "사용되는 중"인 느낌 — 왁스가 흐르고, 뚜껑이 옆에 놓여있고

**Objekt Interpretation Notes**
- Diptyque 해석 시: 단순히 "어두운 배경에 캔들"이 아니라, 회화적 조명(painterly light)과 텍스처 층위가 핵심.
- 정물화 구도 참조: Dutch Golden Age still life, Chardin, Morandi
- 프롬프트 키워드: `warm directional light, Dutch still life composition, dark moody background, layered textures, candlelight glow, wax drips, vintage fabric, painterly photography`

### 3. Granhand (@granhand_official)

**Brand Overview**
Korean lifestyle fragrance brand. "Writing a scent, reading a space." 문방구 + 향 = 독특한 카테고리 포지셔닝. Typography-driven branding, literary sensibility.

**Visual Characteristics**

| Element | Description |
|---------|-------------|
| Lighting | Soft, warm, slightly amber-toned natural light. Window light 느낌. 부드럽지만 방향성 있음. |
| Composition | Off-center, editorial. Negative space 적극 활용. 여백이 "향이 퍼지는 공간"을 시사. |
| Color Palette | Muted warm neutrals. Beige-ivory 계열 중심. 색 대비가 거의 없는 ton-on-ton. |
| Backgrounds | Warm plaster walls, linen, kraft paper, aged wood. 한국적 공간감 — 한옥의 벽면 같은 질감. |
| Textures | Paper, fabric (linen, cotton), glass, brass. 촉감이 느껴지는 소재 조합. |

**Key Colors (Approximate)**
- Signature beige: #D4C5B0, #C9BAA3
- Warm ivory: #F0E8DC, #EDE4D4
- Paper/kraft: #BFB09A, #A89880
- Brass accent: #C5A55A, #B49748
- Typography black: #2C2419
- Glass green (bottles): #C4CDB8, #A8B89B

**What Makes It Distinctive**
- "활자 냄새" — 타이포그래피가 비주얼의 핵심. 레이블, 편지, 책 페이지가 소품으로 자주 등장
- Monochromatic warmth: 거의 한 톤(warm beige)으로 전체 피드 통일
- 한국적 미니멀리즘: 일본(무인양품)도 아니고 북유럽도 아닌, 한국 특유의 따뜻한 절제
- 촉각적 사진 — 보는 사람이 종이 질감, 리넨 촉감을 느끼게 하는 텍스처 강조
- 여백 = 향기. 빈 공간이 제품의 "보이지 않는 가치(향)"를 전달

**Objekt Interpretation Notes**
- Granhand 해석 시: 컬러를 절제하고, 텍스처로 이야기하는 접근. 배경과 오브제의 색 차이가 거의 없는 ton-on-ton이 핵심.
- 여백이 있어야 함. 빽빽하게 채우면 Granhand가 아님.
- 프롬프트 키워드: `warm natural window light, muted beige tones, linen texture, negative space, Korean minimalism, typography detail, soft amber warmth, ton-on-ton palette`

### 4. Cire Trudon (@ciretrudon)

**Brand Overview**
세계에서 가장 오래된 캔들 메이커 (1643, 프랑스). 베르사유 궁전 공급. Heritage luxury. 그린 글라스 jar + 금박 라벨이 시그니처.

**Visual Characteristics**

| Element | Description |
|---------|-------------|
| Lighting | Dramatic, high-contrast. Rembrandt lighting 또는 single-source side light. Deep shadows. 촛불 실제 광원 활용. |
| Composition | Symmetrical and monumental. 제품을 "왕관" 또는 "초상화 주인공"처럼 중앙에 배치. Regal framing. |
| Color Palette | Deep emerald green + gold + ivory. 풍성하고 opulent한 컬러. |
| Backgrounds | Velvet (deep burgundy, forest green, midnight blue), marble, gilded frames, dark stone. |
| Textures | Wax, gilt, glass, velvet, marble, patina. Heritage materials only. |

**Key Colors (Approximate)**
- Signature green (jar): #2D5F3B, #1E4D2E
- Gold foil: #D4AF37, #C9A84C
- Ivory wax: #F5EFE0, #EDE4D0
- Velvet burgundy: #5B1A2A, #722F3F
- Background black: #0F0F0F, #1A1A18
- Marble white: #E8E0D5

**What Makes It Distinctive**
- "Royal portraiture" 접근 — 캔들 하나를 마치 왕의 초상화를 그리듯 촬영
- 역사적 오브제와의 병치: 앤티크 거울, 금박 프레임, 고전 조각상과 함께 배치
- Green glass + gold label이 만드는 "빛의 상호작용"이 핵심 비주얼 디바이스
- Maximalism within constraint — 소품은 풍성하지만 컬러는 2-3색으로 통제
- 시간의 깊이: 모든 이미지에 "수백 년 된 메종"이라는 시간성이 배어있음

**Objekt Interpretation Notes**
- Cire Trudon 해석 시: 단순한 dark moody가 아니라 "regal" — 군주적 위엄. 조명의 드라마가 핵심.
- Green glass jar를 통과하는 빛, 금박의 반사를 어떻게 렌더링하느냐가 난이도 포인트.
- 프롬프트 키워드: `dramatic Rembrandt lighting, deep emerald green, gold gilt details, symmetrical regal composition, velvet background, candlelight, baroque luxury, high contrast`

### 5. Menu (by Audo) (@audocopenhagen)

**Brand Overview**
Danish design. 2023년 Menu와 Norm Architects의 Audo가 합병하여 Audo Copenhagen으로 리브랜딩. Architectural minimalism. "Soft minimalism" 무브먼트의 선두.

**Visual Characteristics**

| Element | Description |
|---------|-------------|
| Lighting | Natural, architectural light. 큰 창문에서 들어오는 빛. Soft but directional. 그림자가 있되, 부드러운 gradient. |
| Composition | Architectural — 오브제를 공간 안에 배치. Negative space가 넓음. 수평/수직 라인 강조. |
| Color Palette | Earth tones only. Greige, taupe, sand, warm grey, muted terracotta. 무채색 + 따뜻한 회색. |
| Backgrounds | Raw plaster walls, concrete, travertine stone, light oak wood, linen. |
| Textures | Tactile but restrained — stone, ceramic, wood, bouclé fabric. Matte finishes dominant. |

**Key Colors (Approximate)**
- Greige: #B8AFA6, #A69E94
- Warm sand: #D5C8B8, #C9BFAC
- Plaster white: #EDE8E2, #E5DED5
- Muted terracotta: #B8886F, #A67B65
- Warm grey: #8B8580, #9A9490
- Dark accent: #3E3B38, #4A4643

**What Makes It Distinctive**
- "Architecture of objects" — 제품이 건축적 공간 안의 요소로 존재. 사진 자체가 인테리어 씬.
- Soft minimalism: 차갑지 않은 미니멀리즘. Warm grey > Cool grey.
- Matte everything: 광택이 거의 없음. 모든 표면이 무광.
- In-situ photography: 실제 공간(Audo House, Copenhagen)에서 촬영한 느낌 강조.
- 수평선(horizontal line)이 이미지를 지배 — 선반, 테이블 엣지, 벽-바닥 경계선.

**Objekt Interpretation Notes**
- Audo 해석 시: 배경이 "공간"이어야 함. 스튜디오 cut-out이 아니라 architectural context.
- 컬러는 무조건 warm neutral. 파란색, 검은색, 순백색 사용 안 함.
- 프롬프트 키워드: `soft natural architectural light, warm greige palette, matte surfaces, travertine stone, raw plaster wall, negative space, horizontal lines, Scandinavian soft minimalism`

## Part 3: Cross-Brand Visual Patterns

---

### Lighting Approaches

| Pattern | Brands | Technical Description |
|---------|--------|----------------------|
| **Flat Diffused** | HAY, Cereal | 대형 소프트박스 또는 overcast daylight. 그림자 거의 없음. 오브제의 컬러/형태 자체를 강조. |
| **Warm Directional** | Diptyque, Cire Trudon, Granhand, Frama | Single source, window 또는 spot. 45-60도 각도. Warm color temperature (3200-4000K). 분위기와 깊이감. |
| **Architectural Natural** | Audo, Kinfolk | 큰 창문 자연광. Soft but readable shadow. 공간감 부여. Neutral-to-cool temperature (5000-5500K). |
| **Candlelight Practical** | Diptyque, Cire Trudon | 촛불 자체가 광원. Extremely warm (2200K). 강한 falloff. 고급 향수/캔들 브랜드 시그니처. |

**Objekt Takeaway**: 카테고리별 라이팅 프리셋을 만들어야 함.
- Living Object (HAY, Audo 해석): Soft diffused 또는 architectural natural
- Home Fragrance (Diptyque, Cire Trudon, Granhand 해석): Warm directional
- 모든 경우에 단일 광원 원칙(single light source) 유지 — 다중 조명은 상업적/카탈로그 느낌

---

### Composition Trends

| Pattern | Description | Representative |
|---------|-------------|----------------|
| **Centered Monumental** | 정중앙 배치, 좌우 대칭, "초상화" 느낌 | Cire Trudon |
| **Off-center + Negative Space** | 1/3 지점 배치, 넓은 여백 | Granhand, Cereal |
| **Graphic Grid** | 다수 오브제를 기하학적으로 배열 | HAY |
| **Tableau/Scene** | 여러 오브제가 "장면"을 구성 | Diptyque, Frama |
| **Architectural In-Situ** | 실제 공간 맥락 안의 오브제 | Audo, Kinfolk |
| **Flat Lay (Top-down)** | 직부감 / overhead 구도 | Kinfolk, HAY |

**Objekt Takeaway**: Brand Set(캐러셀)에서는 하나의 구도만 반복하지 말 것. 3-5장 구성 시:
1. Hero: Centered Monumental 또는 Off-center
2. Detail: Macro/close-up (텍스처, 라벨, 소재)
3. Angle: 3/4 view 또는 low angle
4. Scene: Tableau with context objects
5. Closing: Graphic/minimal (피드 그리드용)

### Color Palettes by Mood

#### Warm Neutral (가장 보편적)

```
#F0E8DC  #D4C5B0  #B8AFA6  #A69E94  #8B8580
 ivory    beige    greige    taupe    warm grey
```

사용 브랜드: Granhand, Audo, Cereal, Kinfolk, Frama
특성: 안전하고 통일감 있음. 피드 전체의 기본 톤으로 적합.

#### Deep Moody

```
#0F0F0F  #1E2D2B  #2D5F3B  #5B1A2A  #D4AF37
 black    forest   emerald  burgundy  gold
```

사용 브랜드: Cire Trudon, Diptyque
특성: 럭셔리/헤리티지. 촛불이나 warm directional light와 함께.

#### Color-Pop on Neutral

```
#FFFFFF  #F5F2ED  #E63946  #2D5BFF  #FFD700
 white    offwhite  red      cobalt   yellow
```

사용 브랜드: HAY
특성: 제품 컬러 자체가 주인공. 배경은 순백 또는 matching color block.

#### Apothecary Warm

```
#2C1F14  #4A3728  #8B6914  #D4BC8A  #E8DCC8
 espresso  walnut  amber    sand     cream
```

사용 브랜드: Frama, (partially) Granhand
특성: Dark wood + amber glass + dried botanicals. 약재상 분위기.

### Background & Surface Treatments

| Surface | Visual Quality | Best For |
|---------|---------------|----------|
| **Raw plaster / lime wash** | Uneven, chalky, organic texture. Warm or cool depending on pigment. | Audo, Granhand 스타일. 가장 트렌디한 현재 배경. |
| **Natural stone (travertine, marble)** | Veined, cool-warm depending on stone type. Weight and permanence 부여. | Audo, Kinfolk, Cire Trudon. 고급감. |
| **Linen / cotton fabric** | Soft, wrinkled, tactile. Warm and domestic. | Granhand, Kinfolk. Casual elegance. |
| **Solid color seamless** | Flat, graphic, no texture. Color = statement. | HAY. Clean and bold. |
| **Dark velvet / fabric** | Deep, light-absorbing, rich. | Cire Trudon, Diptyque. Dramatic/regal. |
| **Aged wood** | Patina, grain, warmth. History. | Frama, Diptyque. Heritage/artisanal. |
| **Concrete / raw cement** | Industrial, cool, modern. | Audo, Frama. Architectural/raw. |

**Objekt Takeaway**: 프롬프트에서 배경 surface를 매우 구체적으로 지정해야 함. "neutral background"는 너무 모호 — "raw lime-washed plaster wall in warm ivory (#EDE8E2)" 수준으로.

### Category-to-Style Mapping for Living/Home Objects

| Product Type | Primary Reference | Lighting | Palette | Background | Composition |
|-------------|-------------------|----------|---------|------------|-------------|
| Candles (luxury) | Cire Trudon, Diptyque | Warm directional + candlelight | Deep Moody | Velvet, dark stone, aged wood | Centered monumental |

### Visual Do's and Don'ts for @objekt.studio Living/Home

**Do**
- 단일 광원 원칙 유지 (single key light, no fill or very subtle)
- 배경 surface를 구체적으로 지정 (material + color + texture level)
- 오브제의 "소재감"을 강조 — glass의 투과, ceramic의 matte, metal의 반사
- 브랜드별 컬러 palette를 사전에 리서치하고 matching background 결정
- 캐러셀 내 구도 변화 (hero → detail → angle → scene)

**Don't**
- 범용적 "studio white background"로 모든 브랜드를 동일 처리
- 여러 방향의 조명 사용 (상업 카탈로그 느낌)
- 실제 존재하지 않는 과도한 질감/디테일 (AI 아티팩트 원인)
- 브랜드 로고가 변형되거나 읽히지 않는 각도로 촬영
- 생활감 없는 "떠있는 오브제" — 반드시 표면(surface) 위에 놓여야 함

### Feed Grid Harmony for Living/Home Category

Living/Home 콘텐츠가 피드에서 다른 카테고리(Beauty, F&B)와 섞일 때 통일감을 유지하려면:

1. **Anchor tone**: Warm neutral (#D4C5B0 ~ #EDE8E2) 계열을 피드의 60% 이상 유지
2. **Accent rhythm**: Moody/dark 이미지는 3열 그리드에서 대각선 배치 (1-5-9 또는 3-5-7)
3. **Color-pop**: HAY 같은 bold color 이미지는 피드당 1/9 이하로 제한
4. **Texture consistency**: 모든 이미지에 최소 하나의 organic texture 포함
