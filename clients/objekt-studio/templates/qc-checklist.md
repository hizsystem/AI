# QC Checklist v0

> Phase 0 캘리브레이션 후 실제 NG 패턴을 반영하여 v1으로 업그레이드.

---

## Hard NG (즉시 탈락)

- [ ] 손/손가락 형태 이상
- [ ] 텍스트/로고 왜곡 또는 의미 없는 글자 생성
- [ ] 제품 형태 비현실적 변형 (병 모양, 뚜껑, 패키지 구조)
- [ ] 명백한 AI 패턴 (과도한 매끄러움, 비현실적 대칭, 플라스틱 질감)
- [ ] 해부학적 오류 (사람이 포함된 경우)

## Soft NG (보정 가능 여부 판단)

- [ ] 반사/그림자 물리적 부자연스러움
- [ ] 배경 디테일 이상 (녹아든 사물, 반복 패턴, 공간 왜곡)
- [ ] 질감 뭉개짐 (천, 나무, 금속, 유리 등 소재별)
- [ ] 컬러 톤 불일치 (브랜드 무드와 안 맞음)
- [ ] 깊이감/보케 부자연스러움
- [ ] 소품 간 스케일 비율 이상

## OK 기준

- [ ] **3초 테스트**: 3초 안에 "사진이다"라고 느껴지는가?
- [ ] **브랜드 일치**: 브랜드 톤과 시각적으로 일치하는가?
- [ ] **피드 통합**: 인스타 피드에 올렸을 때 기존 포스트와 이질감 없는가?
- [ ] **확대 테스트**: 이미지를 확대했을 때 디테일이 자연스러운가?
- [ ] **해석력**: 이 이미지가 브랜드에 대한 "해석"을 담고 있는가? (예쁘기만 한 건 NG)

## Review Process

1. 생성된 후보 이미지 전체를 빠르게 스캔 (Hard NG 탈락)
2. 남은 이미지에 Soft NG 체크
3. OK 기준 5개 항목 통과 여부 확인
4. 최종 셀렉 (Brand Set: 3-5장, Daily Object: 1장)
5. 필요 시 재생성 지시 (NG 사유 + 개선 방향 명시)
