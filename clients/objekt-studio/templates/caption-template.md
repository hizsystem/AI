# Caption Templates

## Format Rule

모든 캡션은 국문 → 영문 → 해시태그 순서.

---

## Brand Set Caption

```
[브랜드명]을 [톤 키워드]로 해석했다.
[1-2문장: 왜 이 톤인지, 브랜드의 어떤 면을 포착했는지]

[Brand name] through the lens of [tone keyword].
[1-2 sentences: what we saw in this brand, why this visual direction]

#objekt #visualdirection #stilllife #[brandname] #[categorytag]
```

### Example — Aesop

```
이솝을 '조용한 의식'으로 해석했다.
매일 반복하는 손 씻기라는 행위 안에 이솝이 담고 싶은 미학이 있다.

Aesop through the lens of quiet ritual.
There's an aesthetic intention hidden in the act of washing your hands.

#objekt #visualdirection #stilllife #aesop #beauty #skincare
```

### Example — HAY

```
헤이의 오브제는 쓰임 이전에 형태로 존재한다.
선반 위에 놓인 것만으로 이미 완성된 장면.

HAY objects exist as form before function.
Already a complete scene, just sitting on a shelf.

#objekt #visualdirection #stilllife #hay #living #design
```

## Daily Object Caption

```
[제품명], [브랜드명].
[감성 한 줄 — 빛, 질감, 순간에 대한 관찰]

[Product name], [Brand name].
[One-line observation — light, texture, moment]

#objekt #stilllife #[brandname]
```

### Example — Le Labo

```
Santal 26, Le Labo.
오후 네 시의 빛이 왁스 위에 머무는 순간.

Santal 26, Le Labo.
The moment afternoon light rests on wax.

#objekt #stilllife #lelabo #candle #fragrance
```

### Example — Blue Bottle

```
Blue Bottle single origin.
크래프트 종이 위의 무게감.

Blue Bottle single origin.
The weight of craft on paper.

#objekt #stilllife #bluebottle #coffee #packaging
```
