## 실행 로드맵

### Phase 0. 세팅 (1~2주)

| 항목 | 담당 |
|------|------|
| 메인 카피 확정 | BR 제안 → 위드런 결정 |
| 딥라떼 라인업 확정 (3~5종) | 위드런 R&D + BR 네이밍 |
| 브랜드 톤 가이드 | BR |
| 인스타 계정 진단 + 리셋 플랜 | BR |

### Phase 1. 오프라인 기반 (3~4주)

- 딥라떼 홍보물 (포스터/배너/테이블 텐트)
- 메뉴판/키오스크 재배치
- 매장 셋팅 가이드 (점장용 1장)
- 시그니처 우유 옵션 검토

### Phase 2. 온라인 점화 (4~8주)

- 인스타 리셋 + 딥라떼 중심 콘텐츠 운영
- 네이버 체험단 (블로그 20편+)
- 인플루언서 씨딩 (릴스 5~8건, 도달 10만+)
- 2차 가공 바이럴 (리그램/편집)
- 블라인드 테스트 콘텐츠
- 메타 광고 (딥라떼 릴스 → 매장 반경 타겟)

### Phase 3. 트렌드 캐치 루틴화 (상시)

- 트렌드 모니터링 → 빠른 출시 → 즉시 콘텐츠
- 전 매장 동시 론칭 키트
- 시즈널 딥라떼 한정판
- 신메뉴별 판매량 + 콘텐츠 성과 리포트

## 가맹점주 보여주기 전략 (이중 타겟 해법)

| 시점 | 액션 | 효과 |
|------|------|------|
| 출시 전 | 가맹점 단톡 사전 공지 "이렇게 마케팅합니다" | 본사가 일하고 있다는 인식 |
| 출시 주 | 마케팅 키트 배포 (POP + 스토리 템플릿 + 촬영 가이드) | "본사에서 이런 걸 보내줬다" 체감 |
| 출시 후 2주 | 1차 성과 리포트 PDF (도달/인게이지먼트/인플루언서 콘텐츠 모음) | 점주들에게 숫자로 증명 |
| 월간 | 가맹점 공유용 요약 리포트 | 본사 마케팅 지속성 증명 |

핵심: 1개 콘텐츠가 B2C 바이럴 + B2B(가맹점주) 신뢰를 동시에 수행하는 구조

## 일하는 방식

| 주기 | 내용 |
|------|------|
| 월요일 | 주간 싱크 30분 — 이번 주 콘텐츠/액션 확인 |
| 수요일 | 콘텐츠 공유 + 피드백 (비동기) |
| 금요일 | 주간 리포트 — 숫자 + 인사이트 + 다음 주 계획 |
| 월간 | 성과 리포트 + 콘텐츠 캘린더 + 트렌드 후보 + 비용 내역 + 가맹점 공유 리포트 |

소통 채널: 카카오톡 단체방 (실무) / 줌 (주간 미팅) / 전화 (긴급)
