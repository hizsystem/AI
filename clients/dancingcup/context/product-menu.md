## 핵심 메뉴 전략 — 딥라떼 (Deep Latte)

### 컨셉

- **Deep Latte = DANCING CUP** 등식 확립
- "얼음이 녹아도 마지막까지 진한" 라떼 경험
- 2,000원대 라떼에서 이 묵직함은 없었음 → 차별화 핵심
- 딥라떼 라인업 3~5종으로 **카테고리 자체를 만들기**

### 딥라떼 서브 카피 후보

| # | 카피 | 비고 |
|---|------|------|
| 1 | 얼음이 녹아도 마지막까지 진한 이유 | **BR 추천** — 아이스 라떼 최대 불만 직접 타격 |
| 2 | 첫 모금에 바로 느껴지는 묵직함 | |
| 3 | 가벼운 라떼에 지친 당신을 위해 딥하게 춤추다 | |
| 4 | 끝까지 진한 댄싱컵 딥 라떼 시리즈 | |

### 오프라인 매장 적용

- 메뉴판/키오스크에서 딥라떼 카테고리 **최전면 배치**
- 딥라떼 전용 포스터, 배너, 테이블 텐트 제작
- 라떼류 추가 옵션으로 시그니처 우유 변경 검토
- 레퍼런스: 스타벅스 에어로카노 시리즈
