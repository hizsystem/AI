# 댄싱컵 x 브랜드라이즈 — 월간 브랜딩 파트너십 제안

---

## 한 줄 요약

**"라떼 맛있는 집, 유행을 가장 빨리 맛볼 수 있는 곳" — 이 두 문장이 댄싱컵의 전부가 되게 만드는 프로젝트입니다.**

---

## 왜 지금인가

댄싱컵은 12년 동안 "좋은 커피를 싸게"라는 약속으로 130호점까지 왔습니다.
그런데 세상은 아직 모릅니다.

- 인스타 팔로워 2,500명. 130개 매장에 비해 극히 적습니다
- 월 200만원 파워페이지에 "왜 쓰는지도 모르겠는데 관성으로" 쓰고 있었습니다
- 2년간 마케팅 방향을 못 잡았습니다

지금이 다른 이유는 하나입니다. **방향이 잡혔습니다.**

대표님이 직접 가져오신 두 축:
1. 라떼가 진짜 맛있는 집
2. 유행하는 건 어디보다 빨리 내놓는 곳

변팀장님이 이미 오프라인/온라인 실행안까지 정리해 오셨습니다.
방향도 있고, 의지도 있고, 예산도 있습니다.
남은 건 **속도**입니다.

---

## 전략 프레임워크 — 두 바퀴

댄싱컵의 브랜드를 자동차에 비유하면, 두 바퀴로 굴러갑니다.

### 바퀴 1. 딥라떼 — 정체성 (스테디)

> "라떼가 맛있는 집" = 변하지 않는 축

이 바퀴의 역할은 **"댄싱컵은 라떼가 다르다"**를 인지시키는 것입니다.

변팀장님이 정리하신 딥라떼 전략이 여기에 해당합니다:
- Deep Latte = DANCING CUP이라는 등식 확립
- 딥라떼 라인업 3~5종으로 카테고리 자체를 만들기
- 매장 내 키오스크/메뉴판에서 딥라떼를 최전면 배치
- "얼음이 녹아도 마지막까지 진한" — 2,000원대 라떼에서 이 경험은 없었음

**이 바퀴는 한번 만들면 계속 굴러갑니다.** 매장에 오면 보이고, 검색하면 나오고, 인스타에서 만나면 기억됩니다.

### 바퀴 2. 트렌드 캐치 — 화제성 (시즈널)

> "유행을 가장 빨리 맛볼 수 있는 곳" = 계속 돌아오게 만드는 축

이 바퀴의 역할은 **"댄싱컵에 또 뭐 나왔대"**를 만드는 것입니다.

메가커피 4,000개, 컴포즈 3,000개는 빨리 못 움직입니다.
수급, 의사결정, 전 매장 배포까지 최소 2~3개월.
댄싱컵 130개는 **2주면 됩니다.**

- 두쫀쿠 유행? → 2주 안에 댄싱컵에서 맛볼 수 있음
- 우배 뜬다? → 다음 주 출시
- 본사에서 완제품 납품하면 전 매장, 레시피만 뿌리면 일부 의욕적 매장

**이 바퀴는 돌 때마다 소식이 됩니다.** SNS에서 화제가 되고, 점주들이 신나고, 손님이 "또 뭐 나왔어?" 하고 재방문합니다.

### 두 바퀴의 관계

```
[딥라떼 = 정체성]          [트렌드 캐치 = 화제성]
  "여기 라떼 진짜 맛있어"    "여기 또 뭐 나왔대"
         ↓                         ↓
     첫 방문 이유               재방문 이유
         ↓                         ↓
              → 단골 형성 ←
              → 가맹 매력 ←
```

## 실행 로드맵

### Phase 0. 세팅 (1~2주)

지금 당장 해야 할 것들입니다.

| 항목 | 내용 | 담당 |
|------|------|------|
| 메인 카피 확정 | 후보 4개 중 택 1 (또는 새로운 안) | BR 제안 → 위드런 결정 |
| 딥라떼 라인업 확정 | 댄싱 딥라떼 포함 3~5종, 네이밍/가격 | 위드런 R&D + BR 네이밍 |
| 브랜드 톤 가이드 | 색상, 폰트, 촬영 톤, 카피 톤 1장 정리 | BR |
| 인스타 계정 진단 + 리셋 플랜 | 현재 @dancingcup.official 분석 → 리셋 방향 | BR |

### Phase 1. 오프라인 기반 (3~4주)

매장에 오는 사람이 "어? 여기 라떼가 메인이네?"를 느끼게 만듭니다.

| 항목 | 내용 | 산출물 |
|------|------|--------|
| 딥라떼 홍보물 제작 | 포스터, 배너, 테이블 텐트 | 디자인 파일 + 인쇄 가이드 |
| 메뉴판/키오스크 재배치 | 딥라떼 카테고리 최전면 배치 | 키오스크 화면 설계안 |
| 매장 셋팅 가이드 | 전 매장 적용 가능한 표준 셋팅 가이드 | 점장용 1장 가이드 |
| 시그니처 우유 옵션 검토 | 라떼류 추가 옵션으로 시그니처 우유 변경 | 메뉴 구조 제안 |

### Phase 2. 온라인 점화 (4~8주)

매장 밖에서 "댄싱컵 라떼가 다르다던데?"가 시작됩니다.

| 항목 | 내용 | 예상 효과 |
|------|------|----------|
| 인스타 리셋 + 콘텐츠 운영 | 피드 톤 통일, 딥라떼 중심 콘텐츠 발행 시작 | 팔로워 2,500 → 5,000+ |
| 체험단 진행 | 네이버 블로그 콘텐츠 → "댄싱컵 딥라떼" 연관검색어 생성 | 블로그 20편+, 검색 노출 |
| 인플루언서 씨딩 | "인생 라떼를 찾았다" "저가 커피에서 이런 묵직함이?" 콘셉트 | 릴스 5~8건, 도달 10만+ |
| 2차 가공 바이럴 | 인플루언서 후기를 리그램/편집 → 자체 콘텐츠화 | 콘텐츠 자산 축적 |
| 블라인드 테스트 | 유명 라떼 맛집 vs 댄싱컵 비교 콘텐츠 (유튜버/인플루언서 협업) | 바이럴 킬러 콘텐츠 |
| 메타 광고 | 딥라떼 릴스 → 매장 반경 타겟 광고 | 매장별 유입 증가 |

### Phase 3. 트렌드 캐치 루틴화 (상시)

"댄싱컵에 또 뭐 나왔대" 사이클이 돌기 시작합니다.

| 항목 | 내용 |
|------|------|
| 트렌드 모니터링 | 유행 음료 사이클 주간 체크 (BR) |
| 빠른 출시 → 즉시 콘텐츠 | 출시 당일 릴스 촬영 → 24시간 내 업로드 |
| 전 매장 동시 론칭 키트 | 점장용 "오늘 이거 올려주세요" 1장 가이드 |
| 시즈널 딥라떼 | 계절별 딥라떼 한정판 (봄 벚꽃, 여름 망고, 가을 고구마, 겨울 초코) |
| 성과 트래킹 | 신메뉴별 판매량 + 콘텐츠 성과 리포트 |

## 메인 카피 제안

변팀장님 후보 4개를 검토하고, 미팅에서 확인된 방향을 반영한 제안입니다.

| # | 후보 | 평가 |
|---|------|------|
| 1 | The latte master | 영문이라 매장 고객 체감 약함 |
| 2 | 라떼가 가장 맛있는 농도, 댄싱컵 | 농도에 포커싱 — 딥라떼와 연결 좋음 |
| 3 | Latte house DANCING CUP | 영문, 서브 카피로는 OK |
| 4 | 인생 라떼를 만나는 가장 가까운 방법 | 접근성 강조 — 130호점과 맞닿음 |

### BR 추천안

**메인**: "라떼가 가장 맛있는 농도, 댄싱컵"
- 딥라떼 전략과 직결
- "농도"라는 단어가 곧 Deep Latte의 핵심
- 다른 저가 커피와 즉시 차별화

**서브 (인스타/광고용)**: "인생 라떼를 만나는 가장 가까운 방법"
- 130호점 접근성과 연결
- 인플루언서 콘텐츠 후킹에 적합

**영문 (키오스크/패키지)**: Latte house DANCING CUP
- 영문이 필요한 곳에만 사용

### 딥라떼 서브 카피 추천

변팀장님 후보 중 **"얼음이 녹아도 마지막까지 진한 이유"**가 가장 강합니다.
- 아이스 라떼의 가장 큰 불만(얼음 녹으면 맛 없어짐)을 직접 타격
- "이유"로 끝나서 궁금증 유발 → 콘텐츠로 풀기 좋음

## 일하는 방식

### 주간 리듬

| 요일 | 내용 |
|------|------|
| 월요일 | 주간 싱크 (30분) — 이번 주 콘텐츠/액션 확인 |
| 수요일 | 콘텐츠 공유 + 피드백 (비동기) |
| 금요일 | 주간 리포트 — 숫자 + 인사이트 + 다음 주 계획 |

### 월간 산출물

- 월간 성과 리포트 (콘텐츠 성과, 팔로워 추이, 광고 ROI)
- 다음 달 콘텐츠 캘린더
- 트렌드 캐치 후보 리스트
- 비용 집행 내역
- 가맹점 공유용 요약 리포트 (점주들에게 "본사가 마케팅 잘하고 있다" 증명)

### 소통 채널

- 카카오톡 단체방 (실무)
- 주간 미팅 (줌)
- 긴급 이슈: 전화

## 왜 브랜드라이즈인가

미례국밥과 같은 본사 아래에서 이미 일하고 있습니다.
위드런의 사업 구조, 의사결정 방식, 팀 역량을 이미 알고 있습니다.

- 미례국밥: 검색 기반 구축 → 콘텐츠 바이럴 → 가맹 인지도
- 댄싱컵: 브랜드 정체성 확립 → 트렌드 화제성 → 재방문 + 가맹 매력

**같은 팀이 두 브랜드를 보면, 시너지가 나옵니다.**
촬영 출장, 인플루언서 네트워크, 체험단 풀, 메타 광고 노하우를 공유할 수 있습니다.
