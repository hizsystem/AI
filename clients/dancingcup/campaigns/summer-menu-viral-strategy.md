## 2. 히어로 메뉴 전략 — 컵빙수 올인

### 왜 컵빙수인가

**타겟**: 20대 여성. 카페에서 디저트를 사는 이유 1위 = **기분전환(42.5%)**. "나를 위한 작은 사치".

2025년 메가커피 컵빙수가 증명한 것:
- 팥빙젤라또 파르페 4,400원 → 3개월 850만개 판매
- 빙수가 "이벤트 음식"(설빙 1.2만원, 2인)에서 → **"일상 음식"(컵빙수 4,500원, 1인)**으로 전환됨
- 소비자 말: "음료보다 컵빙수", "스벅 한 잔 가격으로 커피+빙수 둘 다"

댄싱컵 아메리카노 2,000원 + 컵빙수 4,500원 = **6,500원. 스타벅스 프라푸치노 한 잔 가격.**

### 히어로: 컵빙수 — 카피 5안

> 타겟 심리: "오늘 하루 수고했으니까, 음료 대신 빙수 하나." / "이 가격이면 부담 없이 나한테 사줄 수 있어."

| # | 카피 | 앵글 | 타겟 심리 |
|---|------|------|----------|
| 1 | **"오늘 좀 힘들었으면, 음료 말고 빙수 하나 해"** | 자기 보상 + 업그레이드 | "나를 위한 작은 사치" (디저트 구매 동기 1위) |
| 2 | **"아메 값 더하면 빙수까지 되는 가격"** | 가격 프레이밍 | "2,000+4,500=6,500원, 스벅 한 잔 가격" |
| 3 | **"혼자 먹기 딱 좋은 빙수가 4,500원"** | 1인 사이즈 + 가격 | "혼자 빙수 한 그릇은 부담, 컵빙수는 양도 맛도 적당" |
| 4 | **"팥, 그래놀라, 인절미 떡. 이 가격에 다 넣었다고?"** | 기대 초과 서프라이즈 | "가격치고 내용물이 좀 괜찮더라" |
| 5 | **"올여름은 음료 대신 빙수 한다"** | 새로운 소비 패턴 | "음료보다 컵빙수" (실제 트렌드 키워드) |

### 서브 히어로: 오미자 에이드 — 카피 5안

> 타겟 인식: 오미자 = "할머니가 담가주시는 것" → **비주얼(핑크)과 맛(새콤달콤)을 먼저, 오미자는 나중에 알려주는 구조**
> 기회: 약과/흑임자/쑥 다음, "코리안 힙" 다음 주자로 선점 가능
> 강점: 분홍~붉은 천연 컬러 = 인스타 비주얼 강점 + 레몬/자몽 에이드와 확실히 다른 색

| # | 카피 | 앵글 | 타겟 심리 |
|---|------|------|----------|
| 1 | **"이 핑크색 에이드 뭐야? — 오미자래"** | 비주얼 호기심 → 정체 공개 | "먹어본 적 없는데 예쁘고 궁금한 맛" |
| 2 | **"새콤달콤한데 레몬은 아니야. 맞혀봐"** | 미스터리 + 맛 호기심 | 에이드 선택 기준: 색감 > 맛 > 새로움 |
| 3 | **"요즘 약과, 흑임자 다음은 이거래"** | 코리안 힙 트렌드 | 전통 소재의 힙한 재해석 트렌드 |
| 4 | **"달지만 않은 에이드가 있다. 4,300원"** | 기존 에이드의 과한 단맛 탈피 | "지나치게 달지 않은데다 새콤한 맛" (실제 후기) |
| 5 | **"이 색깔, 이 새콤함. 문경에서 왔다"** | 경험 먼저 → 산지 보조 | 색감+맛 경험 후 산지가 신뢰 요소로 작동 |

## 3. 캠페인 구조 — 인플루언서 시딩 → 메타 광고

### 핵심 엔진: "인플루언서가 만든 콘텐츠를, 메타 광고로 확산한다"

```
인플루언서 시딩 1~2명 (컵빙수 중심)
→ UGC 2건 확보
→ 반응 좋은 1건 선별
→ 메타 "브랜디드 콘텐츠 광고"로 부스팅
→ 인플루언서 이름으로 노출 = 광고처럼 안 보이는 광고
→ 도달 + 매장 방문 전환
```

**왜 이 구조인가:**
- 파워페이지 = 브랜드가 만든 콘텐츠를 돈 주고 노출 → 사람들이 광고로 인식 → 스킵
- 우리 방식 = 인플루언서의 자연스러운 후기를 소재로 → 신뢰도 높은 광고
- 메타 "브랜디드 콘텐츠 광고" 기능: 인플루언서 게시물을 브랜드 계정에서 광고로 부스팅. 인플루언서 이름으로 뜨니까 전환율이 높음.

### + BR 자체 제작 광고 소재

인플루언서 콘텐츠 외에, BR이 직접 만드는 광고 소재도 병행:
- 컵빙수 비주얼 릴스 (매장 촬영)
- 메뉴 한 줄 카피 + 가격 강조 이미지
- 이것도 메타 광고로 부스팅 → A/B 테스트 (인플루언서 소재 vs BR 소재)

## 7. 추가 제안 사항 (본사 판단 영역)

아래는 BR 캠페인 범위 밖이지만, 효과가 클 것으로 판단되어 제안하는 항목들.

### 7-1. "아포가토 빙수" 시크릿 메뉴 이벤트

컵빙수에 에스프레소 샷을 추가하면 "아포가토 빙수"가 된다.

| 항목 | 내용 |
|------|------|
| 기간 | 출시 첫 주 (5/12~18) |
| 방식 | 인스타에서만 공개: "컵빙수 주문할 때 '아포가토로 해주세요'라고 하면 에스프레소 샷 무료 추가" |
| 비용 | 에스프레소 샷 원가 (~300원) × 주문 수. 본사/가맹점 부담 |
| 효과 | "아는 사람만 아는" 시크릿 메뉴 → 매장 방문 트리거 + 자발적 UGC ("나 이거 알아") |
| 왜 좋은가 | 추가 비용 거의 0 + 인스타에서 봤으면 매장 가야 하는 이유가 생김 + "아포가토 빙수"라는 이름 자체가 바이럴 소재 |

> 실행 여부는 본사 R&D/운영팀과 확인 필요. 맛 테스트도 선행되어야 함.

### 7-2. "호기심깍까" 인플루언서 관계 구축

미팅(3/25)에서 확인된 팩트:
- 유명 디저트 인플루언서가 댄싱컵 라떼를 자발적으로 올려줌 (광고 아님)
- "라떼도 맛집이라서 디카페인으로 시켜봤는데 맛있어요"
- 변팀장: 여력 없어서 DM 감사만 보내고 2차 활용 못함

| 제안 | 내용 |
|------|------|
| 딜 | 신메뉴 출시 때마다 무료 시음 초대 (제품 원가만 발생) |
| 조건 | 포스팅 의무 없음. 맛있으면 올리고 아니면 안 올려도 OK |
| 비용 | 메뉴 원가 ~3,000원 |
| 효과 | 광고가 아닌 진짜 후기. 손해 볼 건 전혀 없음 |

> 호기심깍까 이후 접촉 상태 확인 필요 (변팀장에게 문의).

### 7-3. 130개 매장 동시 오픈 공지

출시일(5/12)에 130개 매장이 동시에 여름 신메뉴를 시작한다는 것 자체를 마케팅 메시지로 활용.

| 방법 | 내용 |
|------|------|
| 가맹점 단톡 사전 공지 | 출시 1~2주 전: "5/12에 이렇게 마케팅합니다" |
| D-Day 공지 | "오늘 전국 130개 매장에서 동시 출시" |
| 가맹점 스토리 권장 | 참여 의무 아닌 권장. 템플릿 제공하면 참여율 상승 가능 |
| 성과 리포트 | 출시 2주 후 성과 요약 → 가맹점 단톡 공유 |

> 가맹점 키트 제작/배포는 본사 영역. BR은 "이렇게 하면 효과적이다"는 가이드만 제공.
