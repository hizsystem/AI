# 댄싱컵 여름 신메뉴 바이럴 캠페인

**파워페이지 200만원, 이렇게 바꿉니다.**

---

## 1. 뭘 하나

컵빙수를 메인으로 밀어서 바이럴을 만든다.

- 히어로 메뉴: **컵빙수 4,500원** (팥+그래놀라+인절미 떡)
- 서브: 문경 오미자 에이드 4,300원
- 타겟: 20대 여성, 카페에서 음료 대신 디저트로 기분전환하는 소비 패턴

참고 — 메가커피 컵빙수 4,400원이 3개월 만에 850만개 팔렸다. 시장은 이미 검증됨.

---

## 2. 어떻게

**인플루언서가 만든 콘텐츠를, 메타 광고로 확산한다.**

| 파워페이지 (기존) | 우리 방식 (제안) |
|-----------------|----------------|
| 브랜드가 만든 광고성 콘텐츠 → 스킵 | 인플루언서 후기 → 광고처럼 안 보이는 광고 |
| 끄면 노출 0 | 인플루언서 콘텐츠 = 영구 자산 |
| 효과 측정 불가 | 도달/클릭/전환 전부 숫자로 추적 |

구조:
```
부산 로컬 인플루언서 1~2명이 컵빙수 촬영
→ 자연스러운 후기 콘텐츠 확보
→ 메타(인스타) 협업 광고로 부스팅
→ + BR 자체 제작 소재도 메타 광고로 병행
```

## 5. 어떻게 판단하나

캠페인 종료 후, **"파워페이지 3년 vs 우리 1회"** 비교 리포트를 드린다.

| 지표 | 파워페이지 (3년) | 우리 (1회) |
|------|----------------|-----------|
| 도달 | 측정 불가 | 메타 광고 도달 수 |
| 클릭 | 측정 불가 | 메타 광고 클릭/CTR |
| 콘텐츠 자산 | 0건 | 인플루언서 UGC + 릴스 |
| 팔로워 변화 | 정체 | 순증 수 |

## + 추가 제안 (해보시면 좋을 것들)

**아포가토 빙수 이벤트**: 출시 첫 주, 컵빙수에 에스프레소 샷 무료 추가. 인스타에서만 공개 → 매장 방문 트리거. (비용: 샷 원가 ~300원)

**호기심깍까 인플루언서**: 이미 댄싱컵 라떼를 자발적으로 올려준 분. 신메뉴 출시 때 무료 시음 초대 — 포스팅 의무 없음, 맛있으면 올리고 아니면 말고. (비용: 메뉴 원가만)

**130개 매장 동시 오픈 공지**: 출시일에 가맹점 단톡으로 "오늘 전국 130개 매장 동시 출시" 공지 + 스토리 권장. 점주들에게 "본사가 마케팅을 하고 있다"는 체감.
