## 1. 합의된 사항

### 전략 방향
- **"딥라떼(정체성) + 트렌드 캐치(화제성)" 투트랙 전략** 합의 — 조 대표 "그렇게 해보고 또 상의하면 되지 않을까"로 승인
- **콘텐츠 바이럴**이 핵심 실행 수단이라는 점에 양측 동의

### 채널 전략
- **인스타그램**: 기존 감도 높은 이미지 유지 + AI 영상화로 릴스 알고리즘 공략
- **틱톡**: 일상 맞닿은 영상 (인플루언서, 알바생 등 활용)
- **X(트위터)**: 실시간 현장감 있는 라이브 콘텐츠 (품절 소식, 아침 출근길 등)
- 댄싱컵 **공식 틱톡/X 계정** 개설 필요 — 합의

### 콘텐츠 소스 수급
- 변 팀장이 **현장 콘텐츠 촬영 협조** — "10개 정도 기획안 주시면 날 잡아서 찍어서 보내드릴게요"
- 조 대표 "더 확장할 수 있다. 그럼 할 수 있어요" — 콘텐츠 소스 공급 가능 확인
- 신메뉴 출시 일정은 **변 팀장 → 브랜드라이즈** 직접 공유하기로 합의

### 가맹점 활용
- 130개 전체 동시 움직이는 것은 **비현실적** — 양측 솔직하게 합의
- 대신 **운영팀이 적극적인 매장을 선별** → 인센티브 제공 → 콘텐츠 생산 유도 (조 대표 제안)
- "알바생 추천 콘텐츠" 같은 가이드를 주면 해당 매장에서 뽑아내는 것은 가능

### 메타 광고
- **경상권(부산/경남) 타겟팅**으로 지역 한정 집행 — 합의
- 콘텐츠 반응 테스트 → 워킹하는 패턴 발견 → 확대 적용 순서

### 일정
- **5/12 여름 신메뉴 출시**에 맞춰 첫 바이럴 캠페인 세팅
- **딥라떼 라인업은 6월 초순까지 준비** (조 대표가 직접 일정 확인)
- **다음 주 월요일 내부 회의** — 조 대표 "이 내용 가지고 월요일 날 회의할 때 교체시키는 사업도 우리가 좀 해야 되니까 그렇게 회의합시다"

### 계약
- 계약서 진행 확인 — "계약서 써야 되는 거죠?" → "써야 되는 거 다 끝났어?" (진행 중 확인)

## 2. 대표/변팀장 주요 발언

### 조기원 대표 (발화자 3)

**브랜드 정체성에 대해:**
- "처음에 이걸 기획을 할 때는 이 정도 레벨의 커피를 2,500원에 마시면 진짜 싸다. 근데 그거를... 안 되더라고." — 품질 대비 가격 가치가 소비자에게 전달이 안 되는 것이 핵심 고민
- 텐퍼센트커피와의 비교에 대해: "2년 먼저 했는데 저렇게 차이가 난다면 제가 억울해서" — 경쟁 의식 및 위기감 있음

**가맹점 현실에 대해 (매우 솔직):**
- "가맹 점주님들이... 큰 돈이 들어갔는데 그 돈이 들어간 만큼의 열정을 가지고 일하지는 않아요"
- "회사원의 마인드로 장사를 하시는 분들이 꽤 많거든요"
- "자유도를 주면 막 미쳐 날뛰면 좋겠는데 아예 가동이 안 되니까"
- "하라고 해도 안 해요... 심지어 신메뉴 발주도 안 해요"
- → **이 현실을 디폴트로 깔고 계획을 수립해야 한다**고 직접 조언

**마케팅 → 영업 연결에 대해:**
- "브랜드 인지도가 없는 상황에서 매출이 얼마네 지원을 해주네 쏟아부어 봐야 내가 하고 싶은 마음이 들진 않잖아"
- "부산 경남에 한정돼 있지만 점주들이 외출에 도움이 되게끔 작업들을 해 나가면 그게 영업 쪽으로 훨씬 더 도움이 될 것 같다"
- → **마케팅이 가맹 영업의 도구가 되어야 한다**는 관점 (전국 바이럴 → 가보고 싶다 → 창업 전환)

**PPL에 대해:**
- "PPL을 제가 생각하고 있었나요?" — PPL 특정이 아니라 **예산을 뾰족하게 활용할 방법을 열어둔 것**
- "TP(수단)는 뭐든 상관없어요"

**댄싱커피 메뉴에 대해:**
- "시그니처 댄싱컵커피 잡고 검색도 많이 되고... 했던 단계가 있었는데 그 시기가 지났어요"
- "네이밍을 살려서 리뉴얼을 하든지 딥라떼로 가니까 거기에 구성으로 어떻게 잡든지 방법을 좀 찾아야 될 것 같다"
- 현재 커스터드 크림 구성은 **활용 어렵다**고 직접 언급

### 변미혜 팀장 (발화자 4)

**SNS 운영 현실:**
- "SNS 이미지 배포해도 올리지 않고, 스토리 리그램하는 매장이 몇 %도 안 돼요"
- "인스타 활발히 운영하는 매장이 얼마 안 돼요... 오픈하면 정신이 없거든요"
- 영상 촬영에는 **전담 인력 2명이 붙어야** 돌아간다

**알고리즘 관련 질문:**
- 자신의 계정에서 편의점 콘텐츠가 100만 뷰 터졌는데, 다른 주제는 안 터지고 편의점만 터진다 → 알고리즘이 카테고리별로 밀어주는지 궁금해함
- → 실무적 이해도가 있고, 직접 콘텐츠 운영 경험이 있음

**커스터마이징 가능성:**
- "빌드업을 할 수 있나요? 그것조차" — R&D 확인 필요 표현
- "저희도 생각해볼게요. 따로 할 수 있는 선에서" — 레시피 커스터마이징 검토 의지

## 5. 새로운 인사이트 (미팅에서 처음 알게 된 것)

### 댄싱컵 내부 상황
- **설립 연도 정정**: 텐퍼센트커피보다 2년 먼저 시작 (2019년 vs 2021년) — 검색량 격차가 더 뼈아픈 상황
- **시즌빈(시즈널 원두)** 운영 중 — 주기적으로 원두를 바꿔서 제공, 호불호 있는 스페셜티급
- **커스터드 크림 메뉴 현재 활용 어려움** — 조 대표가 직접 "활용 못 할 것 같다" 언급
- **우베크림** 존재 확인 — 커스터마이징 소재로 활용 가능성
- **가맹점 SNS 활성 비율 극히 낮음** — 이미지 배포해도 리그램조차 10개 매장 정도
- **신메뉴 발주조차 안 하는 점주 다수** — 가맹점 참여 기대치를 대폭 낮춰야 함

### 콘텐츠 자산
- **유튜브에 이미 알바생 자발적 콘텐츠 존재** — 발화자 8이 직접 언급 ("메가커피 알바생처럼 이런 거 하는 매장이 있다")
- **버터떡 메뉴**: 블로그 글 9개인데 검색량 550 → 시의성 메뉴가 검색량 끌어올리는 효과 실증
- **변 팀장 개인 계정에서 100만 뷰 경험** — 편의점 콘텐츠로 바이럴 경험 보유, 실무 감각 있음

### 비즈니스 맥락
- 조 대표는 마케팅을 **가맹 영업의 전 단계**로 봄 — "브랜드 인지 → 가보고 싶다 → 창업 전환"
- **전국 바이럴이 부산/경남 한정 매장과 모순되지 않는다**는 논리를 조 대표가 직접 제시
- 댄싱컵은 **미립국밥** 프로젝트도 병행 중 — 조 대표가 "미립국밥하고 그건 그거대로 이건 이거대로" 언급
- 매장 방문 제안 받음 — 조 대표 "매장 한번 일체해 보시고" → 현장 파악 기회
