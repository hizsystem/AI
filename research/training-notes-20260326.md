## 0. 핵심 요약 — 1차 대비 추가/심화 내용

### 에드로그 실전 활용 (6개 기능 실습)

3. **히든 지수**: N1(유사도), N2(관련성), N3(랭킹) 확인 및 개선 방향 도출
4. **추세 분석**: 경쟁사 순위 변동 히스토리 추적 → 어떤 행동으로 올라갔는지 추론
5. **히든 키워드**: 현재 상위 노출 키워드 + 쿼리 높은데 순위 낮은 기회 키워드 발견
6. **키워드 분석**: 특정 키워드 상위 업체들의 대표 키워드 확인, 롱테일 전략 벤치마킹

### 대행사 관점 핵심 프레임

- "뭐가 부족한지"를 N1/N2 지수로 진단 → 구체적 액션 도출
- 히든 키워드로 기회 키워드 발굴 → 클라이언트 매출 기여

## 1. 띄어쓰기와 검색 의도

네이버는 띄어쓰기 유무로 검색 의도를 구분한다.

| 검색어 | 네이버 인식 | 검색 결과 |
|--------|-----------|----------|
| "부산국밥" (붙여쓰기) | **브랜드명** 검색 | "부산국밥"이라는 이름의 매장 노출 |
| "부산 국밥" (띄어쓰기) | **지역+업종** 검색 | 부산에 있는 국밥 맛집 순위 노출 |

→ **타겟 키워드는 띄어쓰기 버전**이 핵심 (예: "부산 국밥", "홍대 와인바")
→ 블로그 체험단 가이드에도 띄어쓰기 방식 반드시 지정

## 2. 에드로그 히든 지수 (N1/N2/N3) 실전 해석

### N1 (유사도) — 키워드 정합성
- 대표 키워드 ↔ 영수증 리뷰 ↔ 블로그 키워드 ↔ 메뉴/소개 키워드 일치도
- **1점 만점, 0.5 이상이면 상위 노출 가능권**
- N1이 높은데 순위가 낮다 → N2가 문제

### N2 (관련성) — 축적 데이터
- 블로그 리뷰 수, 예약 수, 저장 수, 외부 유입 트래픽, 체류시간, 클릭 행동
- N2가 낮다 → 리뷰/저장/외부유입 부족
- **해결**: 인스타 광고(CPC) → 플레이스 유입 / 체험단 확대 / 소식·이벤트 활성화

### N3 (랭킹) — 결과값
- N1 + N2의 종합 결과
- 개별적으로 개선하기보다 N1, N2를 올리면 자연히 따라옴

### 실전 진단 흐름
```
1. 에드로그에서 목표 키워드의 N1/N2/N3 확인
2. 같은 키워드 상위 1~3위 업체의 N1/N2/N3 비교
3. 우리가 부족한 지수 파악 (N1? N2?)
4. 부족한 영역에 맞는 액션 실행
5. 주 1회 순위/지수 변동 모니터링
```

## 3. 히든 키워드 — 기회 키워드 발굴

에드로그 "히든 키워드" 기능에서 플레이스 링크를 등록하면:
- 현재 10위/20위 이내 상위 노출되고 있는 키워드 목록
- 각 키워드별 쿼리(검색량) + 현재 순위

### 분석 포인트
1. **잘 잡히고 있는 키워드**: 방문 전환이 높은 키워드가 잘 잡혀 있는지 확인
2. **기회 키워드**: 쿼리 높은데 순위 낮은 키워드 → 잡아야 할 타겟
3. **적합성 필터**: 우리 브랜드와 관련 없는 키워드는 제외 (와인바 키워드에 국밥집이 올라가도 전환 안 됨)

### 사례 (미래국밥 센텀점)
- "부산 국밥" 키워드: 월 6,000건 검색, 업체 2,076개, 현재 105위 → 핵심 개선 대상
- "센텀 돼지국밥" 키워드: 쿼리 430건, 현재 3위 → 이미 잘 잡혀 있음 (지역 전환률 높음)
- "재송동 맛집": 쿼리 높은데 순위 낮음 → 잡아야 할 기회

## 5. 키워드 분석 — 경쟁사 대표 키워드 벤치마킹

에드로그 "키워드 분석"에서 특정 키워드 검색 시:
- 상위 노출 업체들의 **대표 키워드 5개**가 전부 노출됨
- 롱테일 전략 벤치마킹 가능

### 롱테일 키워드 전략
- 5개 대표 키워드 중 **2개 정도만 롱테일** 사용 권장
- 롱테일 예시: "서울 망원동 청첩장 모임" → 서울, 망원동, 청첩장, 모임 4개 키워드 한 번에 커버
- **여러 지역 커버 시 유리** (합정+홍대+망원 모두 노출 가능)
- 과도한 롱테일 = 네이버가 반려시킴 (어그로성 판단)
- 롱테일 ≠ 무조건 상위 노출. 지수가 좋아야 함. 부가적 영역.

## 6. 유입 통계 — 채널별 유입 해석

에드로그 유입 통계 기능 (유료 멤버십):
- 채널별 유입수 확인: 네이버 지도, 네이버 검색, 블로그, 인스타그램, 구글 등
- 키워드별 유입수 확인
- 월별/일별 데이터 비교

### 채널별 유입 의미 (핵심)

| 유입 채널 | 고객 유형 | 의미 |
|----------|----------|------|
| 네이버 **지도** | 재방문/단골 | 이미 인지하고 길찾기로 방문. 업력 오래된 브랜드에 높음 |
| 네이버 **검색** | 신규 잠재고객 | SNS/광고에서 보고 검색해서 유입. 키워드 마케팅 효과 반영 |
| 블로그 | 탐색 고객 | 체험단/리뷰 콘텐츠 통한 유입 |
| 인스타그램 | 외부 유입 | 메타 광고 or 오가닉 → 플레이스 유입 (N2 상승) |

→ **지도 비율 높음** = 단골 많음, 브랜드 인지도 높음
→ **검색 비율 높음** = 신규 유입 활발, 마케팅 효과 있음
→ 브랜드 체험단 중단 후 블로그 유입 급감 → 체험단 효과 증명 가능

### 단골 고객 정확한 수치는 톡톡에서
- 네이버 예약 기능 → 전월 방문 고객 수 확인 가능
- 톡톡으로 무료 푸시 알림 발송 → 재방문 유도

| 기존 메뉴명 | 최적화 메뉴명 | 추가 키워드 |
|-----------|-------------|-----------|
| 얼큰 돼지국밥 | **해장** 얼큰 돼지국밥 | 목적 (해장) |

| 원기회복 찬물어 | 국내 유일 **한우 원플러스플러스** 원기회복 찬물어 | 재료 차별화 |

→ AI에게 키워드 추천 받는 것도 유효
→ 메뉴 사진과 메뉴명 키워드가 일치해야 효과 극대화

## 9. 대표 사진·영상 전략 — FMB 업종

FMB(식당/카페/바) 대표 사진 전략 3가지 중 택1:

| 유형 | 적합 케이스 | 예시 |
|------|-----------|------|
| 시즌 이벤트 | 계절감 있는 매장 | "딸기 축제" 이미지 |
| 대표 메뉴 | 메뉴가 강점인 매장 | 스테이크 클로즈업 |
| 분위기 | 공간이 강점인 매장 | 인테리어/조명 사진 |

→ **소개문과 대표사진 방향 일치** 필수 (스테이크 사진 + 스테이크 관련 소개문)
→ 서비스업(필라테스 등): 가격 어필 < 가치 어필 (예: "변화 없으면 100% 환불")

### 영상 세팅
- 소식 → 영상 업로드 → 공지/알림으로 등록 → 플레이스 내 노출
- 검색 결과에는 미노출, 플레이스 상세 페이지에서만 보임
- **효과**: 체류시간 증가 → 지수 상승

## 10. 메타태그 개념 (보충)

네이버 알고리즘 봇이 플레이스를 평가할 때 가장 주의 깊게 보는 영역:

1. **대표 키워드** (사업자가 직접 설정) → 검색 결과에 직접 영향
2. **상호명** (지역 기반 검색 결과에 영향)
3. **메뉴 영역** (업종-메뉴 일치도 확인. 한식인데 와인 메뉴 → 정합성 하락)

→ 대표 키워드에 넣은 키워드로 노출 → 고객 반응 좋으면 → 네이버가 "맞네" 판단 → 순위 상승
→ 유입 키워드 통계에서 해당 키워드 유입 없으면 → 다른 키워드로 교체

## 11. 블로그 체험단 — 실전 보충

### 같은 키워드 2회 상위 노출 불가
- 이미 "홍대 와인바" 1위 한 블로거에게 같은 키워드로 재의뢰 = 낭비
- 한 블로거가 같은 키워드로 블로그 탭에 2개 노출되는 일은 없음
- 다른 키워드로 작업하거나, 더 높은 지수의 블로거 섭외

### 사장님(기업) 블로그의 한계
- 기업형 블로그 = 블로그 지수 낮음 + 상위 노출 어려움
- 일반 잠재고객 전환에는 비효율적
- **유효한 사용처**: DB 마케팅 (가맹점주 설득, 사업 히스토리 전달)
- 상위 노출 목적이라면 체험단/인플루언서가 훨씬 효과적

## 12. 인플루언서 전략 (보충)

### 지역 채널 > 전국 채널
- 팔로워 수보다 **지역 기반 팔로워** 구성이 중요
- 지역 맛집 인플루언서 팔로워 = 해당 지역 거주/관심 → 방문 전환률 높음
- 전국 맛집 채널: 노출은 되지만 "거기까지 가기는..." → 전환 약함

### CTA 설계 필수
- 10만 조회수 콘텐츠도 CTA 없으면 매출 전환 0 가능
- 필수 동선: 콘텐츠 → "프로필 링크 확인" → 플레이스 예약
- CTA 예시: "첫 방문 무료 체험 25분", "서비스 순대 드려요"
- 쿠폰+예약 세팅이 먼저 되어 있어야 CTA가 작동

## 13. AI 소개문 (마이크로 리뷰) 관리

- 네이버가 영수증 리뷰에서 키워드를 추출해 자동 생성
- **격주 단위**로 데이터 수집하여 문구 변경
- 사장님이 직접 쓰는 게 아니라 알고리즘이 리뷰 기반으로 조합
- 성별 비율 확인: 여성 많으면 여성 타겟 문구, 남성 많으면 남성 타겟 문구 유지
- 잘 맞는 문구가 나왔다면 → 영수증 리뷰 키워드 방향을 유지해서 변경되지 않게 관리

## 14. 플레이스 광고 세팅 (보충)

### 파트너 매체 비활성화
- 네이버 기본 설정: 다음/네이트 등 파트너 매체 노출 ON
- **추천**: 파트너 매체 OFF → 네이버 검색에만 집중 → 예산 효율화
- PC/모바일: 기본 1:1이나 모바일 검색 중요한 매장은 모바일 가중치 높이기

### 태그(키워드) 관리
- 광고 설정 > 태그에서 노출 키워드 선택
- **제외 검색어**: 예산 낭비 키워드 제거 (예: 경복궁 맛집에 노출 불필요한 광화문 매장)
- **역발견**: 의외로 클릭률 높은 키워드 발견 → 대표 키워드로 변경 검토

### A/B 테스트
- 소재 2개 이상 등록 → 성과 높은 소재만 유지

- **결론**: 리뷰 수 적은 신규 업체도 로직만 잘 알면 상위 노출 가능 → 어뷰징 불필요

## 17. 핵심 실전 교훈

### 대행사 보고서 작성 팁
- 에드로그 데이터 기반: "부산 국밥 키워드 월 6,000건 검색, 현재 105위 → 이것이 문제"
- 경쟁사 비교: "1위 업체 N2 = 0.8, 우리 N2 = 0.3 → 외부 유입 강화 필요"
- 채널 분석: "블로그 체험단 중단 후 블로그 유입 0 → 체험단 재개 필요"

### 키워드 전략 흐름
```
1. 에드로그에서 목표 키워드 등록
2. 히든 키워드로 기회 키워드 발굴
3. 순위 비교 분석으로 경쟁사 대비 약점 파악
4. N1 낮으면 → 키워드 정합성 개선 (대표키워드/리뷰/블로그)
5. N2 낮으면 → 외부 유입/체류시간/리뷰수 개선
6. 주 1회 순위/지수 모니터링 → 반복
```

### 키워드 선택 기준
- "맛집" 같은 빅키워드: 전환률 낮음 (어디 갈지 정하지 않은 사람)
- "홍대 와인바" 같은 세부키워드: 전환률 높음 (와인바를 가고 싶은 사람)
- 센텀 돼지국밥 (지역 밀착): 검색량 적지만 방문 전환 높음 (거기 있는 사람)
- 부산 국밥 (광역): 검색량 많지만 방문 불확실 (관광객 등)
- **추천**: 지역 밀착 키워드 먼저 잡고 → 광역 키워드 도전
