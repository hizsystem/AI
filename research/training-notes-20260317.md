## 0. 핵심 요약 — 중요도 등급 체계

### S등급 — 반드시 먼저 세팅 (매출에 직접 연결)

- 네이버 플레이스 광고: 일 예산 최소 5,000원~3만원. 광고 자체보다 유입 고객의 체류시간/클릭이 지수를 높임
- 예약 & 쿠폰 기능: 켜두는 것 자체가 지수에 영향. 예약 매장 별도 노출 혜택

### A등급 — 필수 세팅 (노출 순위 결정)
- 대표 사진: 유튜브 썸네일 원리. 영상 소식 업로드 시 체류시간 증가
- 대표 키워드: 알고리즘 봇이 읽는 메타태그. 지역명 필수. 유입 키워드 통계로 성과 확인 후 교체
- 소식 & 이벤트: 2026년부터 지수 상승 효과 증가
- 블로그 체험단: 4개 지표로 직접 섭외 (시작연도, 발행수, 일일방문자, 이웃수)

### B등급 — 있으면 좋은 것
- 상세 소개문: 1,500자 이상 (최대 2,000자)
- AI 브리핑: 노출 켜기. 격주 단위 데이터 수집
- 주차 & 편의시설: 타겟 고객에게 유효한 정보

### 절대 하지 말 것
- 길찾기 트래픽 조작
- AI 블로그 배포 (실방문 없이 키워드만 넣은 대량 발행)
- 저장하기 트래픽 작업

### 보너스 — 매출 부스터
- 네이버 톡톡 메시지: 매월 1일 무료 발송 (최대 ~2,900명)
- 메타 & 당근 광고: 외부 → 플레이스 유입을 네이버가 선호
- 네이버 커넥트 (포스): 영수증 없이 카드 결제 후 즉시 리뷰 가능
- 자사 블로그 연동

## 1. 영수증 리뷰 & 이벤트 전략

### 원리
알고리즘 봇이 키워드 단위로 영수증 리뷰를 읽음. 원하는 키워드가 포함된 리뷰가 쌓여야 해당 키워드로 매장 인식.

### 영수증 리뷰 유도 4단계
1. 방문 목적 적기 ("OO하러 왔어요" — 데이트, 회식 등)
2. 방문 후 느낀 점 솔직하고 구체적으로
3. 추천 / 상위노출 원하는 내용
4. 별점 5점 (4월 6일부터 별점 제도 부활)

### 리뷰 이벤트 세팅
- 사진 필수 유도
- 방문 목적 포함 유도
- 핵심 키워드 포함 유도
- 포스기 앞 + 테이블마다 안내물 배치
- 영기판(QR코드) 활용 참여형 이벤트 연계

### 답글 4가지 필수 요소 (답글도 알고리즘이 읽음)
1. 핵심 키워드 언급
2. 방문 목적 언급
3. 주차 및 편의시설 PR
4. 매장 이용 꿀팁 포함

## 2. 플레이스 광고 집행

### 광고 역할
광고 → 신규 고객 유입 → 체류시간/클릭 증가 → 알고리즘 지수 상향 → 자연 상위 노출

### 핵심 설정값
- 일 예산: 최소 5,000원 / 보통 1~3만원
- 키워드: 리뷰 기반 자동 생성 + 추가/제외 가능
- PC/모바일 가중치 조정
- A/B 테스트: 이미지 2가지로 소재 테스트

## 3. 예약 & 쿠폰 기능

- 예약 가능 매장만 모아서 별도 노출 (2026년 변경)
- 알림받기 쿠폰 → 재방문 유도
- 매월 1일 무료 마케팅 메시지 (알림받기 + 예약 방문 고객, 최대 ~2,900명)

## 4. 대표 키워드 발굴 3가지 방법

### 1) 경쟁사 소스코드 확인
- 경쟁사 플레이스 → 마우스 우클릭 → 프레임 소스보기 → "keyword" 검색

### 2) 마피아넷 (무료)
- 키워드 분석 → 연관 키워드 30개 → 적합한 것 선택

### 3) 자동완성 키워드
- 네이버 검색창에 주요 키워드 입력 → 자동완성 = 고객 궁금증
- 이 질문의 정답을 소개글/체험단 원고에 반영

## 5. 플레이스 알고리즘 — 히든 지수 (N1/N2/N3)

에드로그 툴에서 확인 가능.

| 지수 | 이름 | 측정 기준 |
|------|------|---------|
| N1 | 유사도 | 대표키워드 ↔ 영수증리뷰 ↔ 블로거키워드 ↔ 소개/메뉴 키워드 일치도 |
| N2 | 관련성 | 리뷰수, 예약수, 저장수, 외부 유입 트래픽, 체류시간, 클릭수 |
| N3 | 랭킹 | 해당 키워드 플레이스 순위 (0.5 이상이면 상위 노출 가능권) |

N1 + N2 = 원인 → N3 = 결과.

## 6. 에드로그 툴
- 월 2만원, 본사 계정 1개로 전 지점 관리
- 키워드별 순위 트래킹, 히든 지수 분석, 경쟁사 비교, 체류시간 확인

## 7. 블로그 체험단

### 블로거 선정 4개 지표 (4개 중 3개 이상 충족)
1. 블로그 시작 연도 (오래될수록 유리)
2. 일일 방문자 수
3. 이웃 수 (나를 추가한 이웃)
4. 발행 게시글 수

### 키워드 전략
- 큰 키워드 1개 + 목적형 키워드 (데이트, 회식, 가족모임 등)
- 세부 목적형 키워드가 전환율 높음
- 띄어쓰기 방식 지정해서 체험단에 전달

## 8. 인플루언서 & CTA

- 오픈 시 필수: 알림받기 쿠폰 + 영수증 리뷰 이벤트 + 대표 이미지/영상
