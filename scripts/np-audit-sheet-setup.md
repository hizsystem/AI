# NP 코칭 마스터시트 세팅 가이드

## 1. 새 구글 스프레드시트 생성

이름: "NP 코칭 마스터시트"

## 2. [가격표] 탭 수동 생성

이 탭만 직접 만든다. 나머지 4개 탭은 GAS가 첫 데이터 수신 시 자동 생성.

| A열 (항목코드) | B열 (금액) |
|---------------|-----------|
| PKG-SET | (세팅 패키지 가격) |
| PKG-MNG | (관리 패키지 가격) |
| PKG-PRM | (프리미엄 패키지 가격) |
| OPT-BLOG | (체험단 5팀) |
| OPT-AD-SET | (광고 세팅) |
| OPT-AD-MNG | (광고 월 운영) |
| OPT-PHOTO | (사진 촬영) |
| OPT-INTRO | (소개문 작성) |

- A1셀에 "항목코드", B1셀에 "금액" 헤더 추가
- 1행 고정 (행 고정)
- 금액은 운영하면서 확정. 숫자만 바꾸면 [견적] 탭이 자동 재계산

## 3. GAS 배포

1. [확장 프로그램 > Apps Script] 열기
2. `scripts/np-audit-gas-v2.js` 코드 전체 복사 → 붙여넣기
3. [배포 > 새 배포] 클릭
4. 유형: **웹 앱**
5. 실행 주체: **본인**
6. 액세스: **누구나**
7. [배포] 클릭 → URL 복사

## 6. 자동 생성되는 탭 (4개)

첫 `/np-audit` 실행 시 아래 탭이 자동 생성:

| 탭 | 용도 |
|----|------|
| [매장 현황] | 전 매장 점수/등급 대시보드 |
| [진단 상세] | 13개 항목별 점수 원본 |
| [태스크] | 매장별 코칭 미션 + 완료 체크 (필터: 매장별/주차별/미완료/담당별) |
| [견적] | 패키지 추천 + VLOOKUP 자동 산출 금액 |
